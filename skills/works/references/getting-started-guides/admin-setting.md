# 도메인 확인 및 상품 활성화

네이버웍스를 사용하기 위해 도메인 소유권을 확인하고 사용할 상품을 활성화합니다.

# 도메인 소유권 확인

기존 도메인의 소유권을 확인합니다.

네이버웍스에서 제공하는 기본 도메인(by-works.com 또는 by-works.net)으로 가입하지 않고 기존 도메인을 사용하는 경우, 네이버웍스 사용을 위해 반드시 도메인 소유권을 확인해야 합니다.

도메인 소유권은 PC웹에서만 확인할 수 있습니다.

### PC 웹

1. Admin에 접근합니다.
2. Admin 상단 배너에서 ‘상태 확인하기’를 클릭합니다.
3. '도메인 소유권 확인 가이드'를 클릭합니다.
4. 가이드를 따라 도메인 소유권 확인을 진행합니다.

더 자세한 내용은 [도메인 소유권 확인 FAQ](/ko/faqs/#how-to-verify-domain/)를 참고해주세요.

# 경영지원 상품 활성화

네이버웍스 결재 상품을 신청한 경우, 추가 항목을 입력해 상품을 활성화합니다.

### PC 웹

1. Admin에 접근합니다.
2. '접속 URL' 항목에 경영지원 상품으로 바로 이동할 수 있는 URL을 입력합니다.
   - 3~32자 사이여야 합니다.
   - 숫자, 영소문자, 하이픈(-)만 입력 가능하며, 하이픈(-)은 문자열의 첫 번째 또는 마지막에 입력할 수 없습니다.
3. 비즈니스 국가를 선택합니다.
4. 사용하는 기준 통화를 선택하고 '저장'을 클릭합니다.
5. '확인'을 클릭하여 경영지원 활성화를 완료합니다.

  

기타 회사 정보에 대한 설정은 [회사 정보](/ko/admin-guides/settings/company-information/) 가이드를 참고해주세요.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
