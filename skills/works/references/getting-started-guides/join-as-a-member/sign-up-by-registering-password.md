# 비밀번호 등록해 로그인

가입 메일에서 비밀번호를 직접 등록해 회사 도메인에 로그인할 수 있습니다.

단, 메일을 받은 후 30일 내에 비밀번호를 등록하지 않았거나, 새 메일을 받았거나, 관리자가 설정을 바꾼 경우 비밀번호를 등록할 수 없습니다. 이 경우 관리자가 새 메일을 보내야 합니다.

// start of content for ko

일부 메일 서비스(예: NATE)에서 '비밀번호 등록하기'링크가 동작하지 않을 수 있습니다. 비밀번호 등록에 실패하는 경우, 다른 이메일 주소를 사용하세요.

// end of content for ko

### 모바일 앱

1. 가입 메일에서 '비밀번호 등록하기' 버튼을 탭합니다.
2. 새로운 비밀번호를 등록합니다.
3. 가입 완료 화면에서 ID를 확인합니다.
4. 앱 첫 화면에서 '로그인' 버튼을 탭한 후 전달받은 ID와 직접 등록한 비밀번호로 로그인합니다.
5. 휴대폰 번호를 인증합니다.

### PC 웹

1. 가입 메일에서 '비밀번호 등록하기' 버튼을 클릭합니다.
2. 새로운 비밀번호를 등록한 후 ‘확인’ 버튼을 클릭합니다.
3. 가입 완료 화면에서 ID를 확인하고 ‘NAVER WORKS 서비스 바로가기’ 버튼을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
