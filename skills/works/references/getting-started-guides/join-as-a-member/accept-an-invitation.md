# 초대 링크로 가입

네이버웍스 구성원이 보낸 초대 링크로 회사 도메인에 가입할 수 있습니다.

### 모바일 앱

1. 초대 메시지에서 초대 링크를 탭합니다.  
   QR 코드로 초대 받은 경우, 모바일 기기로 직접 QR 코드를 인식합니다. 메일로 초대받은 경우라면, 메일에서 '가입하기' 버튼을 탭합니다.
2. '구성원으로 가입하기' 버튼을 탭합니다.
3. '네이버로 간편 가입' 또는 '가입하기' 버튼을 탭합니다.  
   네이버로 가입 시 [네이버 ID로 쉽고 빠르게 가입하고 로그인](/ko/use-guides/settings/security/login-security-settings/#i-3)할 수 있습니다.
4. 성과 이름을 입력한 후 '다음' 버튼을 탭합니다.
5. 국가, 휴대폰 번호를 입력한 후 '인증번호 받기' 버튼을 탭합니다.
6. SMS로 수신한 인증번호 4자리를 입력합니다.

### PC 웹

1. 가입 메일을 열고 '가입하기' 버튼을 클릭합니다.
2. '네이버로 간편 가입 또는 '가입하기' 버튼을 클릭합니다.  
   네이버로 가입 시 [네이버 ID로 쉽고 빠르게 가입하고 로그인](/ko/use-guides/settings/security/login-security-settings/#i-3)할 수 있습니다.
3. 성과 이름을 입력한 후 '다음' 버튼을 탭합니다.
4. 국가, 휴대폰 번호를 입력한 후 '인증번호 받기' 버튼을 탭합니다.
5. SMS로 수신한 인증번호 4자리를 입력합니다.

인증 시 사용한 휴대폰 번호는 로그인 시에도 사용됩니다. 다시 로그인할 경우 '휴대폰 번호로 로그인'을 선택해 로그인할 수 있습니다.

인증 시 이미 사용 중인 휴대폰 번호라고 뜨는 경우 'ID/비밀번호로 가입하기' 버튼을 탭하여 ID와 비밀번호를 생성해 가입하거나, '기존 계정과 연결을 해제'를 선택하고 휴대폰 번호를 인증해 가입을 이어서 진행할 수 있습니다.

단, 가입을 완료하더라도 가입 승인이 대기 중인 경우 관리자가 최종 승인을 해야 서비스를 시작할 수 있습니다.

휴대폰 번호 인증 시 SMS가 오지 않는 경우, 아래 방법을 시도해보세요.

1. 입력한 휴대폰 번호를 다시 확인하고, 수신 거부 설정을 하지 않았는지 또는 스팸 문자함에 문자가 없는지 확인합니다.
2. 휴대폰의 전파 수신 상태를 확인하고 휴대폰 전원을 껐다가 다시 켜서 SMS가 도착했는지 확인합니다.
3. 네이버웍스에서 과거에 한번이라도 SMS를 받은 경우, 받은 SMS 메시지를 삭제한 후 다시 인증합니다.

위의 방법으로 해결되지 않는다면 [관리자가 직접 ID와 비밀번호를 생성해 전달](/ko/getting-started-guides/join-as-a-member/sign-in-with-created-account/)하도록 요청해주세요.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
