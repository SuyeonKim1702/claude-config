# 구성원 추가

네이버웍스를 함께 사용할 사내 구성원을 추가합니다.

사내 구성원 정보를 직접 입력하여 추가하거나, 링크나 QR 코드를 공유해 사내 구성원으로 초대할 수 있습니다.

# 구성원 정보를 직접 입력하여 추가

Admin에서 구성원을 직접 추가합니다.

### 모바일 앱

1. 앱 하단 '홈'을 누른 후 'Admin 메뉴'를 선택합니다.
2. '구성원 관리'를 탭합니다.
3. 오른쪽 상단의 '구성원 추가' 버튼을 탭합니다.
4. 구성원의 성, 이름, ID를 입력하고 비밀번호 생성 방법을 선택합니다. '명함 스캔하기' 버튼을 탭한 후 명함을 촬영해 정보를 바로 등록할 수도 있습니다.
   - 자동으로 생성: 임시 비밀번호가 생성됩니다. 구성원이 처음 로그인할 때 새 비밀번호를 등록하도록 요청할 수 있습니다.
   - 관리자가 등록: 비밀번호를 직접 등록한 후 구성원에게 전달할 수 있습니다. 구성원이 처음 로그인할 때 새 비밀번호를 등록하도록 요청할 수 있습니다.
   - 구성원이 등록: 발송된 초대 메일을 통해 구성원이 직접 비밀번호를 등록하게 합니다.
5. 계정 활성화 일자를 선택합니다. 해당 일자부터 구성원이 네이버웍스를 사용할 수 있습니다.  
   '바로 활성화'를 선택하면 오늘이 해당 구성원의 입사일이 되며, 구성원이 로그인해 네이버웍스를 바로 사용할 수 있습니다.
6. 오른쪽 상단의 '추가' 버튼을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘Admin’을 클릭합니다.
3. 왼쪽 메뉴에서 '구성원'을 클릭합니다.
4. '구성원 추가' 버튼을 클릭합니다.
5. 구성원의 성, 이름, ID를 입력하고 비밀번호 생성 방법을 선택합니다.
   - 자동으로 생성: 임시 비밀번호가 생성됩니다. 구성원이 처음 로그인할 때 새 비밀번호를 등록하도록 요청할 수 있습니다.
   - 관리자가 등록: 비밀번호를 직접 등록한 후 구성원에게 전달할 수 있습니다. 구성원이 처음 로그인할 때 새 비밀번호를 등록하도록 요청할 수 있습니다.
   - 구성원이 등록: 발송된 초대 메일을 통해 구성원이 직접 비밀번호를 등록하게 합니다.
6. 계정 활성화 일자를 선택합니다. 해당 일자부터 구성원이 네이버웍스를 사용할 수 있습니다.  
   '바로 활성화'를 선택하면 오늘이 해당 구성원의 입사일이 되며, 구성원이 로그인해 네이버웍스를 바로 사용할 수 있습니다.
7. 오른쪽 하단의 '추가' 버튼을 클릭합니다.

# 구성원 초대

초대 링크 또는 QR 코드를 공유해 구성원을 초대합니다.

### 모바일 앱

1. '주소록'을 탭합니다.
2. ![](/wp-content/uploads/2024/02/icon_plus.png) 아이콘을 탭한 후 '구성원 추가'를 탭합니다.
3. '링크 공유하기' 또는 'QR 코드 보여주기'를 선택합니다.
   - 링크 공유하기: 네이버웍스의 구성원으로 가입할 수 있는 링크를 SMS나 다른 앱으로 공유할 수 있습니다.
   - QR 코드 보여주기: 네이버웍스의 구성원으로 가입할 수 있는 QR 코드를 동료에게 보여주거나, SMS나 다른 앱으로 공유할 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '주소록'을 클릭합니다.
3. 왼쪽 메뉴에서 '구성원 초대'를 클릭합니다.
4. 동료로 초대할 구성원의 메일 정보를 입력합니다.
5. '초대 메일 보내기' 버튼을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
