# 위젯

네이버웍스에서 업무에 중요한 정보 또는 자주 사용하는 기능을 위젯으로 추가해서 홈 화면에서 볼 수 있습니다.

| 나의 연차 | (근태 서비스 사용 시) 나의 잔여 연차(아르바이트는 나의 남은 휴가) 정보 표시  - 탭하여 '부재 일정 서식' 작성, 아르바이트 사원은 '아르바이트의 휴가 신청' 화면으로 이동 |

# 위젯 설정

위젯의 사용 여부와 순서를 설정할 수 있습니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '위젯 설정'을 탭합니다.
4. '홈'에 노출할 위젯을 설정합니다.
   - 홈에 추가한 위젯: 현재 노출 중인 위젯입니다. '숨긴 위젯'에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/03/list_18_addlist_fill.png) 아이콘을 탭하여 노출할 위젯을 추가할 수 있습니다.
   - 숨긴 위젯: 노출하지 않는 위젯입니다.
     - Android: 노출을 원하지 않는 위젯은 '홈에 추가한 위젯'에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/list_18_deletelist_fill.png) 아이콘을 탭합니다.
     - iOS: 노출을 원하지 않는 위젯은 '홈에 추가한 위젯'에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/list_18_deletelist_fill.png) 아이콘을 탭한 뒤 '삭제'를 탭합니다.
5. 순서를 조정하려면 위젯 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/02/icon-switch-location-1.png) 아이콘을 길게 탭한 후 원하는 위치에 놓습니다.
6. '저장'을 탭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
