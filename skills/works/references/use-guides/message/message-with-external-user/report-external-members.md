# 외부 대화 상대 신고

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
외부 네이버웍스 사용자로부터 스팸 메시지가 온 경우 해당 대화 내용, 메시지방, 대화 상대를 신고할 수 있습니다.

신고 접수된 내용을 조사하여 이용 약관에 위배되는 행위가 확인될 경우 그에 따라 처리되며 자세한 조사 내용은 알려드릴 수 없는 점 양해 바랍니다.

신고 사실은 상대방에게 알려지지 않습니다.

단, LINE 사용자 또는 LINE 사용자가 포함된 메시지방은 신고할 수 없습니다.

# **대화 내용 신고하기**

외부 네이버웍스 사용자가 보낸 메시지 내용을 신고합니다.

### 모바일 앱

1. 신고할 메시지를 길게 누릅니다.
2. ‘신고’를 탭합니다.
3. 신고 사유를 선택하거나 직접 입력합니다.
4. '동의 및 신고'를 탭합니다.

### PC 웹

1. 신고할 메시지를 마우스 오른쪽 버튼으로 클릭합니다.  
   또는 메시지 옆에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
2. ‘신고’를 클릭합니다.
3. 신고 사유를 선택하거나 직접 입력합니다.
4. '동의 및 신고'를 클릭합니다.

### PC 앱

1. 신고할 메시지를 마우스 오른쪽 버튼으로 클릭합니다.  
   또는 메시지 옆에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
2. ‘신고’를 클릭합니다.
3. 신고 사유를 선택하거나 직접 입력합니다.
4. '동의 및 신고'를 클릭합니다.

# **메시지방 신고하기**

외부 네이버웍스 사용자가 만든 메시지방을 신고합니다.

### 모바일 앱

1. 메시지방에 들어가서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ‘신고’를 탭합니다.
3. 신고 사유를 선택하거나 직접 입력합니다.
4. '동의 및 신고'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
5. ‘신고’를 클릭합니다.
6. 신고 사유를 선택하거나 직접 입력합니다.
7. '동의 및 신고'를 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. ‘신고’를 클릭합니다.
5. 신고 사유를 선택하거나 직접 입력합니다.
6. '동의 및 신고'를 클릭합니다.

# **대화 상대 신고하기**

외부 네이버웍스 대화 상대를 신고합니다.

### 모바일 앱

1. 신고할 대화 상대의 프로필 이미지를 탭합니다.
2. ‘더보기’를 탭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
4. ‘신고’를 탭합니다.
5. 신고 사유를 선택하거나 직접 입력합니다.
6. '동의 및 신고'를 탭합니다.

### PC 웹

1. 신고할 대화 상대의 프로필 이미지를 클릭합니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
3. ‘신고’를 클릭합니다.
4. 신고 사유를 선택하거나 직접 입력합니다.
5. '동의 및 신고'를 클릭합니다.

### PC 앱

1. 신고할 대화 상대의 프로필 이미지를 클릭합니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
3. ‘신고’를 클릭합니다.
