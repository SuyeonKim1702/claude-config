# 노트 열기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
여러 명이 참여하는 메시지방에서 노트를 열 수 있습니다.

### 모바일 앱

1. 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '노트'를 탭해서 노트를 엽니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 선택합니다.  
   오른쪽에 화면에 노트 게시글 목록이 표시됩니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 오른쪽 상단 ![](/wp-content/uploads/2024/04/icon-message-pc-app-note.png) 아이콘을 클릭합니다.

# 게시판별로 게시글 보기

게시글에 설정된 게시판별로 게시글을 모아서 볼 수 있습니다.

### 모바일 앱

1. 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '노트'를 탭해서 노트를 엽니다.
3. 화면 상단의 파란색으로 표시된 게시판 중 원하는 게시판을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 선택합니다.
4. 화면 오른쪽 '노트' 탭에서 '전체 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-8-arrow-down-222222.png)'를 클릭합니다.
5. 원하는 게시판을 선택합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 오른쪽 상단 ![](/wp-content/uploads/2024/04/icon-message-pc-app-note.png) 아이콘을 클릭합니다.
4. PC웹 화면으로 이동하여 화면 오른쪽 '노트' 탭에서 '전체 ~~![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-8-arrow-down-222222.png)~~'를 클릭합니다.
5. 원하는 게시판을 선택합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
