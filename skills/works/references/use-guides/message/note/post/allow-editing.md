# 편집 허용

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
하나의 노트 게시글을 여러 멤버가 편집할 수 있습니다.

게시글의 수정 내역을 확인할 수 있고, 복원도 가능합니다.

단, 하나의 글을 동시에 여러 명이 편집할 수는 없습니다.

# 편집 허용 설정

새 게시글을 작성할 때 또는 이미 글을 등록한 이후에 편집 허용 설정을 할 수 있습니다.

게시글의 편집 허용 여부는 게시글 내 작성자 이름 하단에 '편집 허용 : ON' , '편집 허용 : OFF'로 표시됩니다.

편집 허용의 기본 설정 값은 '편집 허용 : ON'입니다.

## 새 게시글 작성 시 편집 허용 설정

새 게시글을 작성할 때 편집 허용을 설정할 수 있습니다.

### 모바일 앱

1. 글쓰기에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_fill.png) 아이콘을 탭합니다.
2. '글쓰기 설정'을 탭합니다.
3. '편집 허용'을 켜거나 끕니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 화면 오른쪽의 '글쓰기'를 클릭합니다.
5. '편집 허용'을 켜거나 끕니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단의![](/wp-content/uploads/2024/04/icon-message-pc-app-note.png) 아이콘을 클릭합니다.
4. PC웹 화면으로 이동하여 화면 오른쪽의 '글쓰기'를 클릭합니다.
5. '편집 허용'을 켜거나 끕니다.

## 등록한 게시글의 편집 허용 설정

작성자는 작성한 게시글에 대해 편집 허용을 설정할 수 있습니다.

### 모바일 앱

1. 게시글을 열어 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
2. '수정'을 탭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '글쓰기 설정'을 탭합니다.
4. '편집 허용'을 켜거나 끕니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 '노트' 탭의 게시글 목록에서 공동 편집을 설정할 게시글을 클릭합니다.
5. 게시글 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '수정'을 클릭합니다.
7. 화면 상단의 '편집 허용'을 선택하거나 선택 해제합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 오른쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-note.png) 아이콘을 클릭합니다.
4. PC웹 화면으로 이동하여 화면 오른쪽 '노트' 탭의 게시글 목록에서 공동 편집을 설정할 게시글을 클릭합니다.
5. 게시글 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '수정'을 클릭합니다.
7. 화면 상단의 '편집 허용'을 선택하거나 선택 해제합니다.

조직장 및 그룹 마스터는 관리 목적상 그룹 멤버가 작성한 게시글의 편집 허용 설정을 할 수 있습니다.

# 게시글 수정

멤버는 누구나 편집이 허용된 게시글을 수정할 수 있습니다.

조직장 및 그룹 마스터는 모든 게시글을 수정할 수 있습니다.

### 모바일 앱

1. 수정할 게시글을 엽니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
3. '수정'을 탭해서 게시글을 수정합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 '노트' 탭의 게시글 목록에서 수정할 게시글을 클릭합니다.
5. 게시글 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '수정'을 클릭합니다.
7. 본문 내용을 수정 후 '등록'을 클릭하여 게시글을 저장합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 오른쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-note.png) 아이콘을 클릭합니다.
4. PC웹 화면으로 이동하여 화면 오른쪽 '노트' 탭의 게시글 목록에서 수정할 게시글을 클릭합니다.
5. 게시글 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '수정'을 클릭합니다.
7. 본문 내용을 수정 후 '등록'을 클릭하여 게시글을 저장합니다.

작성자 정보는 마지막에 수정한 작성자명과 수정 시간으로 표시됩니다.

편집 허용이 제한된 게시글을 열었을 때는 작성자와 이름 하단에 '편집 허용 : OFF'로 표시됩니다.

# 게시글 수정 내역 확인

편집이 허용된 게시글에서는 게시글을 수정한 멤버와 수정 시간, 복원 여부 등을 확인할 수 있습니다.

수정 및 복원 내역은 최신 30건만 조회가 가능합니다.

### 모바일 앱

게시글을 연 후 작성자 하단의 '최근 수정'을 탭해 수정 내역을 확인합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 '노트' 탭의 게시글 목록에서 수정된 게시글을 클릭합니다.
5. 작성자 하단의 '최근 수정'을 클릭해 수정 내역을 확인합니다.

# 게시글 복원

수정 내역 중 특정 버전의 게시글을 복원할 수 있습니다.

### 모바일 앱

1. 게시글을 연 후 작성자 하단의 '최근 수정'을 탭합니다.
2. 수정 내역 중 복원하고자 하는 내역의 '복원'을 탭해 게시글을 복원합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 '노트' 탭의 게시글 목록에서 수정된 게시글을 클릭합니다.
5. 작성자 하단의 '최근 수정'을 클릭해 수정 내역을 확인합니다.
6. 수정 내역 중 복원하고자 하는 내역의 '복원'을 클릭해 게시글을 복원합니다.

복원 기록도 수정 내역에 기록됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
