# 게시글 임시 저장

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
새 게시글 작성 시 임시 저장할 수 있고, 임시 저장한 게시글을 이어서 쓸 수 있습니다.

# 게시글 임시 저장하기

새 게시글을 등록하지 않고 임시 저장할 수 있습니다.

### 모바일 앱

1. '새 글 쓰기' 화면 상단의 ![](/wp-content/uploads/2024/05/common_24_close_line.png) 아이콘을 탭합니다.
2. '예'를 탭하여 임시 저장합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 화면 오른쪽 '글쓰기'를 클릭합니다.
5. '임시 저장'을 클릭하여 임시 저장합니다.

# 임시 저장한 내 게시글을 이어서 작성

'전체' 목록에서 임시 저장한 게시글을 확인할 수 있고, 이어서 작성할 수도 있습니다.

임시 저장한 게시글은 '작성중' 표시가 있습니다.

### 모바일 앱

1. 노트 목록 화면으로 이동합니다.
2. 목록 상단에 '임시보관함' 혹은 임시 보관된 게시글을 선택합니다.  
   '임시보관함'을 선택하면 임시 보관된 게시글 목록으로 이동합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 오른쪽 '노트' 탭을 클릭하여, 목록 상단에 '임시보관함' 혹은 임시 보관된 게시글을 선택합니다.  
   '임시보관함'을 선택하면 임시 보관된 게시글 목록으로 이동합니다.

# 임시 저장한 게시글 삭제

임시 보관된 게시글을 삭제할 수 있습니다.

단, 임시 저장한 게시글은 삭제 시 복구할 수 없습니다.

### 모바일 앱

1. 삭제할 임시 보관된 게시글을 엽니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_fill.png) 아이콘을 탭합니다.
3. '삭제'를 탭해서 삭제합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 오른쪽 '노트' 탭을 클릭하여, 목록에서 삭제할 임시 보관된 게시글을 클릭합니다.
5. '삭제'를 선택하여 삭제합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
