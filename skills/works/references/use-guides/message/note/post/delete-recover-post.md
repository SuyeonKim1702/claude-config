# 게시글 삭제 및 복구, 영구삭제

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

# 게시글 삭제

본인이 작성한 게시글을 삭제할 수 있습니다.

게시글을 삭제하면 휴지통으로 이동하고, 관리자가 설정한 휴지통 보관 기간이 지나면 자동으로 영구 삭제됩니다.

### 모바일 앱

1. 삭제할 게시글을 엽니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
3. '삭제'를 탭해서 게시글을 삭제합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록에서 삭제할 게시글을 클릭합니다.
5. 게시글 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '삭제'를 클릭합니다.

조직장 및 그룹 마스터는 관리 목적상 다른 멤버가 작성한 게시글을 삭제할 수 있습니다.

# 삭제한 게시글 복구

삭제한 게시글은 관리자가 설정한 휴지통 보관 기간 이내에 휴지통에서 복구할 수 있습니다.

삭제한 게시글을 복구할 때 특정 게시판으로 이동할 수 있습니다.

### 모바일 앱

1. 노트에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
2. '휴지통'을 탭합니다.
3. 휴지통에서 복원할 게시글을 탭합니다.
4. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
5. '이동'을 탭합니다.
6. 이동할 게시판을 선택하고 ![](/wp-content/uploads/2024/02/groups_note_trash_folder_select-1.jpg) 아이콘을 탭해서 게시글을 복구합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록 하단의 '휴지통'을 클릭합니다.
5. 복구하고 싶은 게시글을 선택합니다.
6. 게시글 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
7. '이동'을 클릭합니다.
8. 이동할 게시판을 선택하여 '확인'을 클릭합니다.

조직장 및 그룹 마스터는 관리 목적상 다른 그룹 멤버가 삭제한 게시글을 복구할 수 있습니다.

# 삭제한 게시글 영구 삭제

삭제한 게시글은 관리자가 설정한 휴지통 보관 기간이 지나면 자동으로 영구 삭제됩니다.

자동으로 삭제되기 전에 본인이 삭제한 게시글은 수동으로 영구 삭제할 수 있습니다.

### 모바일 앱

1. 노트에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
2. '휴지통'을 탭합니다.
3. 휴지통에서 영구 삭제할 게시글을 탭합니다.
4. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
5. '영구 삭제'를 탭해서 게시글을 휴지통에서 영구 삭제합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록 하단의 '휴지통'을 클릭합니다.
5. 영구 삭제하고 싶은 게시글을 선택합니다.
6. 게시글 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
7. '영구 삭제'를 클릭합니다.
8. 주의사항을 읽고 '삭제'를 클릭합니다.

조직장 및 그룹 마스터는 그룹 멤버가 삭제한 게시글을 영구 삭제할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
