# 노트 작성하는 동안 다른 내용 보기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
노트를 작성하다가 다른 서비스의 내용을 확인하고 이어서 작성할 수 있습니다.

### 모바일 앱

1. 작성 중인 화면 상단 중앙의 회색 바를 아래로 쓸어 내립니다.  

2. 다시 작성을 하고 싶다면 ![](/wp-content/uploads/2024/05/common_24_pencilwrite_fill.png) 아이콘 위에 ![](/wp-content/uploads/2024/05/common_24_draft_line.png) 아이콘을 탭합니다.
3. 작성 중인 화면이 여러 개일 경우 원하는 화면을 탭하여 다시 작성합니다.

작성창은 최대 5개까지 띄워놓을 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
