# 게시글 쓰기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
노트 게시판별로 게시글을 쓸 수 있고, 게시글에는 텍스트뿐만 아니라 사진, 동영상, 템플릿, 파일, URL, 스티커 등을 첨부할 수 있습니다.

# 새 게시글 쓰기

새 게시글은 메시지방의 [노트를 열어](/ko/use-guides/message/note/open-note/) 쓸 수 있습니다.

### 모바일 앱

1. 노트에서 ![](/wp-content/uploads/2024/05/common_24_pencilwrite_fill.png) 아이콘을 탭합니다.
2. 필요한 경우 게시글을 올릴 게시판을 선택합니다.  
   게시판을 선택하지 않아도 게시글을 올릴 수 있습니다. 게시판을 선택하면 노트에서 게시판별로 필터링해서 볼 수 있습니다.
3. 제목과 본문을 입력합니다.
4. 필요한 경우 사진이나 드라이브에 있는 파일을 첨부를 합니다.
5. 공지로 등록하거나 [편집 허용을 설정](/ko/use-guides/message/note/post/allow-editing/)하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '글쓰기 설정'을 탭합니다.
6. '저장'을 탭해 노트에 게시글을 올립니다. 메시지방에 알림을 보내려면 '예'를 선택합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 화면 오른쪽 '글쓰기'를 클릭합니다.
5. '전체 ▼'를 클릭하여 원하는 게시판을 선택합니다.  
   게시판을 선택하지 않아도 게시글을 올릴 수 있습니다. 게시판을 선택하면 노트에서 게시판별로 필터링해서 볼 수 있습니다.
6. 제목과 본문을 입력합니다.
7. 필요한 경우 사진이나 드라이브에 있는 파일을 첨부를 합니다.
8. 상단에서 공지로 등록 여부, [편집 허용 여부](/ko/use-guides/message/note/post/allow-editing/)를 설정합니다.
9. '등록'을 클릭합니다. 메시지방에 알림을 보내려면 '메시지방에 알림 보내기'를 선택하여 '확인'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. PC 웹 화면으로 이동하여 화면 오른쪽 '글쓰기'를 클릭합니다.
4. '전체▼'를 클릭하여 원하는 게시판을 선택합니다.  
   게시판을 선택하지 않아도 게시글을 올릴 수 있습니다. 게시판을 선택하면 노트에서 게시판별로 필터링해서 볼 수 있습니다.
5. 제목과 본문을 입력합니다.
6. 필요한 경우 사진이나 드라이브에 있는 파일을 첨부를 합니다.
7. 상단에서 공지로 등록 여부, [편집 허용 여부](/ko/use-guides/message/note/post/allow-editing/)를 설정합니다.
8. '등록'을 클릭합니다. 메시지방에 알림을 보내려면 '메시지방에 알림 보내기'를 선택하여 '확인'을 클릭합니다.

## 동영상 삽입

카메라로 촬영하거나 기기에 있는 동영상을 본문에 삽입할 수 있습니다.

단, .mp4, .mov가 확장자명인 동영상만 삽입할 수 있습니다.

### 모바일 앱

1. ![](/wp-content/uploads/2024/04/icon-n-act_com_img-1.png) 아이콘을 탭합니다.
2. 기기의 동영상을 선택하거나 카메라 촬영을 하여 삽입합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 화면 오른쪽 '글쓰기'를 클릭합니다.
5. 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/add-video.png) 아이콘을 클릭합니다.
6. '내 PC', '드라이브' 중 동영상을 불러올 곳을 선택합니다.
7. 동영상을 선택합니다.
8. '열기'를 클릭합니다.
9. 삽입된 동영상의 이름과 용량을 확인합니다.

h.264코덱에서 재생 가능한 .mp4, .mov가 확장자명인 동영상만 삽입이 가능합니다.

본문에 삽입된 동영상은 노출된 동영상 제목으로 검색이 가능합니다.

## 동영상 제목 수정하기

게시글 본문에 삽입된 동영상 제목을 수정할 수 있습니다.

### 모바일 앱

1. 삽입된 동영상을 탭합니다.
2. '동영상 제목 편집'을 탭합니다.
3. 동영상 제목을 입력합니다.
4. '확인'을 탭합니다.

### PC 웹

1. 삽입된 동영상을 클릭합니다.
2. 동영상 위에 노출되는 아이콘에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/img-edit-video-rename.png) 아이콘을 클릭합니다.
3. 동영상 제목을 입력합니다.
4. '확인'을 클릭합니다.

## 동영상 URL 삽입

게시글 본문에 동영상 플랫폼에서 제공하는 동영상을 URL로 삽입할 수 있습니다.

- 지원 동영상 플랫폼 : YouTube, Naver TV, Vimeo

동영상 플랫폼에 업로드한 콘텐츠 제공자의 동영상 공유 허용 여부에 따라서, 동영상이 노출되지 않을 수 있습니다.

이 외에도 동영상 URL을 제공하는 플랫폼의 제약사항을 따릅니다.

### 모바일 앱

1. 동영상 플랫폼에서 URL을 복사합니다.
2. 글쓰기 본문에, 동영상 URL을 붙여넣기 합니다.

### PC 웹

1. 동영상 플랫폼에서 URL을 복사합니다.
2. 글쓰기 본문에, 동영상 URL을 붙여넣기 합니다.

# 본문 번역 삽입

작성중인 본문을 다른 언어로 번역해서 본문에 삽입할 수 있습니다.

현재 번역 지원 언어는 영어, 일본어, 중국어(간체/번체), 프랑스어, 베트남어, 태국어, 인도네시아어, 스페인어, 포르투갈어입니다.

### 모바일 앱

1. ![](/wp-content/uploads/2024/05/add-translation.png) 아이콘을 탭합니다.
2. 번역할 언어를 선택하고 '확인'을 탭합니다.
3. 번역 결과를 확인하고 '삽입'을 탭해서 본문에 삽입합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 화면 오른쪽 '글쓰기'를 클릭합니다.
5. 본문 작성 후 편집할 게시글의 영역을 선택합니다.
6. ![](/wp-content/uploads/2024/05/add-translation.png) 아이콘을 클릭합니다.
7. 번역할 언어를 선택합니다.

# 사진, 동영상, 파일 첨부

기기에 있는 사진 및 동영상 또는 파일을 첨부할 수 있습니다.

### 모바일 앱

1. ![](/wp-content/uploads/2024/05/common_24_attach_line.png) 아이콘을 탭합니다.
2. '사진/동영상 선택' 또는 '파일 선택'을 탭해서 기기의 사진, 동영상 또는 파일을 첨부합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.

6. PC에서 첨부할 사진, 동영상 또는 파일을 선택하여 첨부합니다.

## 메시지방 폴더와 드라이브에 있는 사진, 동영상, 파일 첨부

메시지방 폴더와 드라이브에 있는 사진, 동영상, 파일을 첨부할 수 있습니다.

### 모바일 앱

3. 파일을 선택해서 첨부합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.

## 첨부 파일 최대 개수와 최대 용량

게시글 한 개에 첨부할 수 있는 파일 개수의 제한은 없고, 최대 용량은 관리자가 설정한 최대 용량을 따릅니다. 사용 가능한 공용용량을 초과하는 첨부파일은 업로드할 수 없습니다.

# 멤버 멘션하기

노트 게시글 작성 시 특정 사용자를 멘션할 수 있습니다.

노트 게시글 등록 시 ‘메시지방에 알림 보내기’를 선택하면 메시지방으로 알림이 전송됩니다.

이때 게시글 본문에서 멘션된 멤버는 파란색으로 강조 표시되며, 멘션 대상에 본인이 포함된 경우 노란색으로 강조되어 구분됩니다.

또한 멘션이 포함된 노트가 메시지방으로 전송되면, 본인이 멘션 대상일 경우 PC 웹 팝업 또는 모바일 푸시 알림에서 ‘멘션되었습니다’라는 안내 문구가 함께 표시됩니다.

단, 게시글 등록 시 ‘메시지방에 알림 보내기’를 선택하지 않으면 해당 알림은 전송되지 않습니다.

### 모바일 앱

1. 노트 게시글을 작성할 때 '@'을 입력합니다.
2. 멘션할 조직/그룹의 멤버를 선택합니다.  
   전체 멤버를 멘션하려면 '@All'을 입력하거나 '모든 멤버 멘션하기'를 탭합니다.
