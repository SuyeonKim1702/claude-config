# 안 읽은 게시글 모아보기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
노트의 게시글 중 아직 안 읽은 게시글만 모아서 볼 수도 있습니다.

### 모바일 앱

1. [노트를 열어](/ko/use-guides/message/note/open-note/) ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
2. '안읽은 글'을 탭해서 안 읽은 게시글 목록을 확인합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 화면 오른쪽 '노트' 탭에서 '전체 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-8-arrow-down-222222.png)' 를 클릭합니다.
5. '안읽음'을 클릭하여 안 읽은 게시글 목록을 확인합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
