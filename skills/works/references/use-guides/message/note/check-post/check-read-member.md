# 게시글 읽음 확인

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
노트의 게시글을 읽은 멤버 수를 확인할 수 있고, 읽은 멤버와 안 읽은 멤버의 명단을 확인할 수 있습니다.

멤버는 누구나 읽은 멤버 명단을 확인할 수 있습니다.

# 읽은 멤버 수 확인

게시글을 읽은 멤버의 수를 확인할 수 있습니다.

### 모바일 앱

1. 게시글을 열어 '읽음' 옆의 숫자를 확인합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록에서 게시글을 클릭합니다.
5. '읽음' 옆의 숫자를 확인합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 오른쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-note.png) 아이콘을 클릭합니다.
4. PC웹 화면으로 이동하여 화면 오른쪽 '노트' 탭의 게시글 목록에서 게시글을 클릭합니다.
5. '읽음' 옆의 숫자를 확인합니다.

조회 수가 아니라 읽은 멤버의 수입니다.

# 읽은 멤버와 안 읽은 멤버 명단 확인

그룹 멤버는 누구나 게시글을 읽은 멤버와 안 읽은 멤버의 명단을 확인할 수 있습니다.

### 모바일 앱

1. 게시글을 열고 '읽음'을 탭합니다.
2. 게시글을 읽은 멤버와 안 읽은 멤버의 명단을 확인합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록에서 게시글을 클릭합니다.
5. 게시글을 열고 '읽음'을 클릭합니다.
6. '읽음'을 클릭하여 게시글을 읽은 멤버의 명단을 확인하거나, '안 읽음'을 클릭하여 안 읽은 멤버의 명단을 확인합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 오른쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-note.png) 아이콘을 클릭합니다.
4. PC웹 화면으로 이동하여 화면 오른쪽 '노트' 탭의 게시글 목록에서 게시글을 클릭합니다.
5. 게시글을 열고 '읽음'을 클릭합니다.
6. '읽음'을 클릭하여 게시글을 읽은 멤버의 명단을 확인하거나, '안 읽음'을 클릭하여 안 읽은 멤버의 명단을 확인합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
