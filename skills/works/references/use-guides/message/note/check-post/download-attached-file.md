# 사진 및 파일 보기, 다운로드

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
'게시글'에 첨부된 사진이나 동영상, 파일을 기기로 다운로드할 수 있습니다.

# 사진 보기 및 다운로드

게시글에 첨부된 사진을 크게 볼 수 있고, 기기나 드라이브 또는 폴더에 다운로드할 수 있습니다.

### 모바일 앱

1. 게시글에서 다운로드할 사진을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2018/05/common_24_up_share_line.png)아이콘을 탭합니다.  
   파일을 전부 일괄 다운로드하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2018/05/common_24_up_shareall_line.png) 아이콘을 탭합니다.
3. '기기에 저장', '드라이브' 또는 '폴더'를 선택합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록에서 게시글을 클릭합니다.
5. 다운로드할 사진의 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_download_pc_767676.png) 아이콘을 클릭하여 다운로드하거나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_download_drive_767676.png) 아이콘을 클릭하여 드라이브 또는 폴더에 저장합니다.  
   파일을 전부 일괄 다운로드하려면 '모두 저장'을 클릭합니다. 이후 'PC에 저장'을 선택하면 ZIP 형식으로 압축되어 저장됩니다.

# 사진 편집하기

Admin에서 '모바일앱에서 파일 다운로드 제한'이 '제한 안 함'으로 설정되어 있을 경우, 사진을 편집하여 저장할 수 있습니다.

### 모바일 앱

1. 편집할 사진을 탭합니다.
2. '편집'을 탭합니다.
3. 편집이 필요한 사진일 경우 아래 기능을 사용하여 편집한 후 '적용'을 탭합니다.  

   |  |  |
   | --- | --- |
   |  | 사진 회전 |
   |  | 사진 자르기 |
   |  | 사진에 그림 그리기 |

   사진 편집 후 원본으로 돌리고 싶다면 '원본으로'를 탭합니다.
4. ![](/wp-content/uploads/2024/05/common_24_save_line.png) 아이콘을 탭합니다.
5. 기기에 저장하려면 '저장'을 탭하고, 드라이브에 저장하려면 '드라이브 저장' 또는 '폴더 저장'을 탭해서 저장할 위치를 선택합니다.

# 동영상 보기 및 다운로드

첨부된 동영상을 재생할 수 있고, 기기나 드라이브 또는 폴더에 다운로드할 수 있습니다.

### 모바일 앱

### iOS 기기에 동영상 다운로드

2. ![](/wp-content/uploads/2024/05/common_24_save_line.png) 아이콘을 탭합니다.
3. 기기에 저장하려면 '저장'을 탭하고, 드라이브에 저장하려면 '드라이브 저장' 또는 '폴더 저장'을 탭해서 저장할 위치를 선택합니다.

### 안드로이드 기기에 동영상 다운로드

1. 게시글에서 다운로드할 동영상을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_save_line.png) 아이콘을 탭하여 기기에 저장합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.

5. 다운로드할 동영상의 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_download_pc_767676.png) 아이콘을 클릭하여 다운로드하거나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_download_drive_767676.png) 아이콘을 클릭하여 드라이브 또는 폴더에 저장합니다.

# 첨부 파일 보기 및 다운로드

첨부된 엑셀, 파워포인트, 워드, .txt 등의 문서 파일을 확인할 수 있고, 기기나 드라이브 또는 폴더에 다운로드할 수 있습니다.

### 모바일 앱

### 첨부 파일 보기

1. '첨부'를 탭하여 목록을 엽니다.
2. 확인할 파일을 탭합니다.

### iOS 기기에 첨부 파일 다운로드

iOS 11 및 이후 버전에서만 기기에 첨부 파일 다운로드가 가능합니다. iOS 11 이전 버전에서는 기기의 특성상 첨부 파일을 기기에 저장할 수 없습니다.

1. '첨부'를 탭하여 목록을 엽니다.
2. 드라이브 혹은 폴더에 다운로드 하려면 파일 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_download_drive_767676.png) 아이콘을 탭합니다.  
   기기에 다운로드 하려면 파일을 탭하여 ![](/wp-content/uploads/2024/04/icon_ios_action-1.png) 아이콘을 탭한 후, ![](/wp-content/uploads/2024/04/icon_ios_file.png) 아이콘을 선택해서 저장합니다.

### 안드로이드 기기에 첨부 파일 다운로드

1. '첨부'를 탭하여 목록을 엽니다.
2. 다운로드 할 파일의 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_download_pc_767676.png) 아이콘을 탭합니다.  
   기기에 저장하려면 '저장'을 탭하고, 드라이브에 저장하려면 '드라이브 저장' 또는 '폴더 저장'을 탭해서 저장할 위치를 선택합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록에서 게시글을 클릭합니다.
5. 파일을 확인하거나 다운로드합니다.
   - 파일 확인: 파일 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-fileviewer.png) 아이콘을 클릭합니다.
   - 파일 다운로드: 파일 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_download_pc_767676.png) 아이콘을 클릭하거나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_download_drive_767676.png) 아이콘을 클릭합니다.

# 첨부 파일 모두 저장

PC 웹에서는 게시글에 첨부된 파일을 한번에 다운로드할 수 있습니다.

내 PC에 파일을 저장할 때는 ZIP 형식으로 압축되어 저장됩니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록에서 게시글을 클릭합니다.
5. 첨부 파일 텍스트 오른쪽의 '모두 저장'을 클릭하여 'PC에 저장', '드라이브로 보내기', 또는 '폴더로 보내기'를 선택합니다.

# 첨부 파일 요약

이 기능은 AI 스튜디오 상품 신청 후 사용 가능합니다.

AI로 [문서 내용을 자동으로 요약](/ko/use-guides/drive/ai/ai-summary/)해 한눈에 확인할 수 있습니다.

PC 웹에서는 최대 5개, 모바일에서는 1개의 문서를 한 번에 요약할 수 있습니다.

요약문의 길이와 형식을 선택할 수 있습니다.

### 모바일 앱

1. 파일이 첨부된 게시글을 엽니다.
2. 첨부된 파일 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-fileviewer.png) 아이콘을 탭합니다.
3. 문서 파일 오른쪽의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
4. 'AI 요약'을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록에서 게시글을 클릭합니다.
5. 첨부된 파일 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-fileviewer.png) 아이콘을 클릭합니다.
6. 문서 파일 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2018/05/ic24_more_normal.png) 아이콘을 클릭합니다.
7. 'AI 요약'을 클릭합니다.

# 첨부 파일 보관기간

노트에 첨부된 파일은 보관기간 제한이 없습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
