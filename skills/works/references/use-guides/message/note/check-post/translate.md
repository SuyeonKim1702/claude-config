# 게시글 번역

노트의 게시글을 자동으로 번역해서 볼 수 있습니다.

### 모바일 앱

1. 번역해서 볼 게시글을 열고 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
2. '번역'을 탭합니다.
3. 번역할 언어를 선택하고 '확인'을 탭해서 번역된 게시글을 봅니다.  
   다시 원문을 보려면 '원문보기'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 '노트' 탭의 게시글 목록에서 번역할 게시글을 클릭합니다.
5. 게시글 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '번역'을 클릭합니다.
7. 작성자 하단에서 번역할 언어를 선택하여 번역된 게시글을 봅니다.  
   다시 원문을 보려면 '원문보기'를 클릭합니다.

현재 번역 지원 언어는 영어, 일본어, 중국어(간체/번체), 프랑스어, 베트남어, 태국어, 인도네시아어, 스페인어, 포르투갈어입니다.

# 특정 댓글 번역

### 모바일 앱

1. 번역해서 볼 댓글의  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
2. '번역'을 탭합니다. '언어선택'을 탭해 번역 언어를 변경할 수도 있습니다.

### PC 웹

1. 번역해서 볼 게시글을 클릭합니다.
2. 번역해서 볼 댓글의 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
3. '번역'을 클릭합니다. '언어선택'을 클릭해 번역 언어를 변경할 수도 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
