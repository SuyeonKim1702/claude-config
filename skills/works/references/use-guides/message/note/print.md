# 게시글 인쇄

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
노트 게시글 본문을 인쇄할 수 있습니다.

### PC 웹

1. 인쇄할 노트 게시글을 클릭합니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭하고 '인쇄'를 클릭합니다.

## 템플릿 인쇄

[노트 게시글 본문에 삽입된 템플릿](/ko/use-guides/message/note/post/write-post/#i-2)을 작성 내용을 포함해 인쇄할 수 있습니다.

### PC 웹

1. 노트 게시글 본문에 삽입된 템플릿 중 인쇄할 템플릿 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-pc-template-more.png) 아이콘을 클릭합니다.
2. 템플릿 본문 오른쪽 상단의  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭하고 '인쇄'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
