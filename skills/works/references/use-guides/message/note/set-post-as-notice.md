# 공지 등록 및 해제

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
게시글을 공지로 등록하면 노트 '공지사항' 영역에서 모아볼 수 있고, 메시지방 상단에 공지를 등록할 수 있습니다.

메시지방 멤버는 누구나 본인의 게시글뿐만 아니라 다른 사람의 게시글도 노트 공지로 등록하거나 등록을 해제할 수 있습니다.

메시지방의 상단에 표시된 공지는 멤버가 접거나 다시 보지 않게 할 수 있습니다.

# 공지 등록

## 새 게시글 작성 시 공지로 등록

새 게시글을 올릴 때 공지로 바로 등록할 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ‘노트’를 탭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_plus.png) 아이콘을 탭합니다.
4. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭하고 ‘글쓰기 설정’을 탭합니다.
5. ‘노트 공지 등록’을 켭니다.  
   게시글이 등록될 때 '메시지방에 알림을 보내고 메시지방 공지로 등록하시겠습니까?'의 '예'를 누르면 메시지방 공지로도 등록됩니다. '아니오'를 누르면 노트 공지로만 등록됩니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 화면 오른쪽의 '글쓰기'를 클릭합니다.
5. 상단의 '노트 공지 등록'을 선택합니다.
6. 게시글 저장 시 '메시지방에 알림 보내고 메시지방 공지로 등록' 체크 시 메시지방 공지로 등록됩니다. 체크하지 않을 시, 노트 공지로만 등록됩니다.

## 기존 게시글을 공지로 등록

### 모바일 앱

1. 공지로 등록할 게시글을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
3. '노트 공지 등록'을 탭합니다.
4. '게시글을 노트 공지로 등록합니다. 메시지방 공지로도 등록하시겠습니까? 등록 시 메시지방에 알림이 발송됩니다.'의 '예'를 누르면 메시지방 공지로도 등록됩니다. '아니오'를 누르면 노트 공지로만 등록됩니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 오른쪽 '노트' 탭을 클릭하여, 게시글 목록에서 공지로 등록할 게시글을 클릭합니다.
5. 게시글 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '노트 공지 등록'을 클릭합니다.
7. '확인'을 클릭합니다.  
   '메시지방에 알림 보내고 메시지방 공지로 등록' 선택 시 메시지방 공지로 등록됩니다. 선택하지 않을 시, 노트 공지로만 등록됩니다.

공지로 등록하면 자동으로 메시지방에 공유됩니다.

# 노트 공지 해제

멤버는 본인이 등록한 게시글뿐만 아니라 다른 사람이 등록한 공지를 해제할 수 있습니다.

메시지방에 노출된 노트 공지는 메시지방에서 해제할 수 있습니다.

### 모바일 앱

1. 공지를 해제할 게시글을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
3. '노트 공지 내리기'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 오른쪽 '노트' 탭을 클릭하여, 게시글 목록에서 공지를 해제할 게시글을 클릭합니다.
5. 게시글 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '노트 공지 내림'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
