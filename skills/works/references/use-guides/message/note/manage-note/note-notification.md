# 노트 알림 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
노트에 새 게시글이나 공지가 등록되거나 댓글이 달리면 자동으로 메시지방에서 메시지로 알림을 받을 수 있습니다.

다음은 노트의 메시지 알림 종류입니다.

# 새 게시글 메시지 알림

다른 멤버가 노트에 게시글을 등록하면 자동으로 메시지방으로 게시글이 등록되었다는 메시지를 받을 수 있습니다. 새 게시글 등록 메시지를 탭하면 노트의 내용을 확인할 수 있습니다.

새 게시글이 등록될 때 메시지 알림을 끄는 방법은 없습니다.

# 공지 메시지 알림

게시글을 공지로 등록하면 메시지방에 공지로 등록한 멤버의 이름으로 메시지 알림이 발송됩니다.

공지로 등록될 때 메시지 알림을 끄는 방법은 없습니다.

# 댓글 메시지 알림

노트에 댓글이 달리면 메시지방에서 메시지 알림을 받을 수 있습니다. 설정에서 [노트 댓글 알림을 설정](/ko/use-guides/message/note/manage-note/set-comment-notification/)할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
