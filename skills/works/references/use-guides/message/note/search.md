# 검색

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 노트 게시글의 제목이나 본문뿐만 아니라 작성자, 파일명, 댓글 내용을 검색할 수 있습니다.

# 게시글 제목, 본문, 작성자, 파일명, 댓글 내용 검색

노트의 게시글과 첨부된 파일 그리고 댓글 내용을 검색할 수 있습니다.

# 작성자명 검색

멤버 이름으로 검색하고 '작성자'를 선택하면, 해당 멤버가 작성한 게시글 목록을 볼 수 있습니다.

### 모바일 앱

1. 노트에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 탭합니다.
2. 작성자명을 입력합니다.
3. '작성자' 탭을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_search_3f4247.png) 아이콘을 클릭합니다.
5. 작성자명을 입력합니다.
6. '노트' 탭을 클릭합니다.
7. '작성자'를 클릭합니다.

# 파일명 검색

게시글 첨부 파일의 파일명과 파일 본문 내용에서 검색한 키워드가 포함된 파일 목록을 볼 수 있습니다.

### 모바일 앱

1. 노트에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 탭합니다.
2. 파일명을 입력합니다.
3. '파일' 탭을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_search_3f4247.png) 아이콘을 클릭합니다.
5. 파일명을 입력합니다.
6. '노트' 탭을 클릭합니다.
7. '파일'을 클릭합니다.

# 댓글 내용 검색

댓글에서 찾으려는 내용 일부를 입력하고 '댓글'을 선택하면, 입력 내용이 포함된 댓글 목록을 볼 수 있습니다.

### 모바일 앱

1. 노트에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 탭합니다.
2. 댓글 내용을 입력합니다.
3. '댓글' 탭을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_search_3f4247.png) 아이콘을 클릭합니다.
5. 댓글 내용을 입력합니다.
6. '노트' 탭을 클릭합니다.
7. '댓글'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
