# 게시글 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
게시글을 다른 메시지방 노트 혹은 게시판으로 보낼 수 있습니다.

# 게시글을 다른 메시지방의 노트로 보내기

### 모바일 앱

1. 다른 메시지방 노트로 보낼 게시글을 엽니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_up_share_line.png) 아이콘을 탭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/more_34_note_line.png) 아이콘을 탭합니다.
4. 게시글을 보낼 노트와 게시판을 선택합니다.
5. 필요한 경우 게시글 제목, 내용, 첨부파일을 수정합니다.
6. '저장'을 탭하여 게시글을 다른 메시지방의 노트로 보냅니다. 메시지방에 알림을 보내려면 '예'를 선택합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 '노트' 탭에서 다른 노트로 보내고 싶은 메시지방 노트를 클릭합니다.
5. 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '보내기'를 클릭합니다.
7. '다른 노트로 보내기'를 클릭합니다.
8. 보내고 싶은 메시지방을 클릭합니다.
9. '확인'을 클릭합니다.
10. 필요에 따라 내용 수정 후 화면 상단의 '등록'을 클릭합니다. 메시지방에 알림을 보내려면 '메시지방에 알림 보내기'를 선택하여 '확인'을 클릭합니다.

# 게시글을 게시판으로 보내기

### 모바일 앱

1. 게시판으로 보낼 게시글을 엽니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_up_share_line.png) 아이콘을 탭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/more_34_note_line.png) 아이콘을 탭합니다.
4. 게시글을 보낼 게시판을 선택합니다.
5. 필요한 경우 게시글 제목, 내용, 첨부파일을 수정합니다.
6. '저장'을 탭하여 게시글을 게시판으로 보냅니다. 게시글 등록 알림을 보내려면 '예'를 선택하여 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 게시판으로 보낼 노트가 있는 메시지방을 클릭합니다.
4. 화면 오른쪽 '노트' 탭에서 게시판으로 보낼 노트를 클릭합니다.
5. 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
6. '보내기'를 클릭합니다.
7. '게시판 서비스로 보내기'를 클릭합니다.
8. 게시글을 보낼 게시판을 선택하여 '확인'을 클릭합니다.
9. 필요한 경우 내용을 수정하고 '등록'을 클릭합니다. 게시글 등록 알림을 보내려면 '게시물 등록 알림 보내기'를 선택하여 '확인'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
