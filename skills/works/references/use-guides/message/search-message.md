# 메시지 검색

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 이름, 메시지방의 멤버, 메시지 내용, 파일명, 파일 본문을 검색할 수 있습니다.

단, 검색은 사내 관리자가 설정한 조회 가능 기간에 해당하는 메시지만 가능합니다.

# 전체 메시지 검색

전체 메시지방의 메시지를 검색할 수 있습니다.

## 메시지 검색

### 모바일 앱

1. 메시지 목록에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 탭합니다.
2. 찾으려는 메시지방 이름, 멤버 이름, 메시지 내용 또는 파일명을 입력해서 검색합니다.

최근 찾은 메시지방과 검색어는 자동 저장되어, 다시 검색할 때 검색 기록에서 빠르게 선택할 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 상단의 검색창을 클릭합니다.
4. 찾으려는 메시지방 이름, 멤버 이름, 메시지 내용 또는 파일명을 입력해서 검색합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록 상단의 검색창을 클릭합니다.
3. 찾으려는 메시지방 이름, 멤버 이름, 메시지 내용 또는 파일명을 입력해서 검색합니다.

## 메시지 검색 결과 정렬

### 모바일 앱

1. 검색 결과 화면에서 ![](/wp-content/uploads/2024/05/common_24_align_line.png) 아이콘을 탭합니다.
2. 정렬 옵션을 선택해서 검색결과를 정렬합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 검색 결과에서 메시지방/멤버 이름 탭 오른쪽의 정렬 옵션을 클릭합니다.
4. 검색 결과를 정렬합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록 화면의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-talk-list-sort.png) 아이콘을 클릭합니다.
3. 정렬 옵션을 선택해서 검색 결과를 정렬합니다.

## 메시지 검색 후 메시지방 숨김

### 모바일 앱

1. 메시지방 목록에서 숨기고 싶은 메시지방을 왼쪽으로 쓸어 넘깁니다.
2. '숨김'을 탭합니다.
3. 대화창에서 '숨김'을 탭합니다.

메시지방 하나를 길게 누른 후 여러 개의 메시지방을 선택하여 한꺼번에 숨길 수도 있습니다.

모바일에서 메시지방을 숨겨도 PC웹과 PC앱에는 반영되지 않습니다.

### PC 웹

1. 메시지방 목록에서 숨길 메시지방 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
2. '숨김'을 클릭합니다.
3. 숨길 다른 메시지방이 있다면 함께 선택한 후 '숨김'을 클릭합니다.

PC웹에서 메시지방을 숨겨도 모바일과 PC앱에는 반영되지 않습니다.

### PC 앱

1. 메시지방 목록에서 숨길 메시지방을 마우스 오른쪽 클릭합니다.
2. '숨김'을 클릭합니다.
3. 숨길 다른 메시지방이 있다면 함께 선택한 후 '숨김'을 클릭합니다.

PC앱에서 메시지방을 숨겨도 PC웹과 모바일에는 반영되지 않습니다.

## 메시지 검색 후 메시지방 나가기

### 모바일 앱

1. 메시지방 목록에서 나가고 싶은 메시지방을 왼쪽으로 쓸어넘깁니다.
2. '나가기'를 탭합니다.
3. 대화창에서 '확인'을 탭합니다.

메시지방 하나를 길게 누른 후 여러 개의 메시지방을 선택하여 한꺼번에 나갈 수도 있습니다.

### PC 웹

1. 메시지방 목록에서 나가고 싶은 메시지방 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
2. '나가기'를 클릭합니다.
3. 나가고 싶은 다른 메시지방이 있다면 함께 선택한 후 '나가기'를 클릭합니다.

### PC 앱

1. 메시지방 목록에서 나가고 싶은 메시지방을 마우스 오른쪽 클릭합니다.
2. '나가기'를 클릭합니다.
3. 나가고 싶은 다른 메시지방이 있다면 함께 선택한 후 '나가기'를 클릭합니다.

# 메시지방 내부 검색

메시지방 안에서 메시지를 검색하거나, 특정 날짜의 메시지를 확인할 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 탭합니다.
2. 찾으려는 메시지 내용, 멤버 이름, 파일명을 입력해서 검색합니다.  
   특정 날짜의 메시지 내용을 확인하고 싶다면 ![](/wp-content/uploads/2024/04/icon-search-date.png) 아이콘을 탭한 후 원하는 날짜를 선택합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-12-search-767676.png) 아이콘을 클릭합니다.
5. 찾으려는 메시지 내용, 파일명을 입력해서 검색합니다.  
   특정 날짜의 메시지 내용을 확인하고 싶다면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_schedule_search@3x.png) 아이콘을 클릭한 후 원하는 날짜를 선택합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
3. ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '메시지 검색'을 클릭합니다.
5. 찾으려는 메시지 내용, 멤버 이름, 파일명을 입력해서 검색합니다.  
   특정 날짜의 메시지 내용을 확인하고 싶다면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_schedule_search@3x.png) 아이콘을 클릭한 후 원하는 날짜를 선택합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
