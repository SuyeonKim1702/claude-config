# 할 일 수정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

> 내가 작성한 할 일 뿐만 아니라 다른 메시지방 멤버가 작성한 할 일도 수정할 수 있습니다.
>
> ### 모바일 앱
>
> 1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 버튼을 탭하고 '할 일'을 탭합니다.
> 2. 할 일 목록에서 수정할 할 일을 탭합니다.
> 3. 오른쪽 상단의  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png)  아이콘을 탭합니다.
> 4. '수정'을 탭합니다.
> 5. 할 일을 수정한 뒤 '저장'을 탭합니다.
>
> ### PC 웹
>
> 1. 메시지방에서 화면 오른쪽의 '할 일' 탭을 클릭합니다.
> 2. 할 일 목록에서 수정할 할 일을 클릭합니다.
> 3. 할 일 상세 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-task-edit.png) 아이콘을 클릭합니다.
> 4. 할 일을 수정한 뒤 '저장'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
