# 할 일에서 보기 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
자주 사용하는 메시지방 할 일을 목록에 추가할 수 있습니다.

할 일에서 보기 설정은 개인 설정이므로, 다른 멤버들의 목록에는 추가되지 않습니다.

기본 설정은 '추가 안 함'입니다.

- 추가 안 함: 각 메시지방에서만 메시지방 할 일을 확인할 수 있습니다. 할 일의 목록에는 표시되지 않습니다.
- 할 일 목록에 추가: 할 일 서비스 화면 왼쪽의 ['메시지방 할 일' 목록](/ko/use-guides/task/task-list/message-room-task/)과 [할 일 쓰기](/ko/use-guides/task/task-writing/new-task/) 시 메시지방 선택 목록에 이 메시지방이 표시됩니다. 모바일앱, PC웹 할 일에서 할 일을 확인할 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '할 일'을 탭해 할 일을 엽니다.
3. 할 일에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
4. '할 일에서 보기'를 탭합니다.
5. '할 일 목록에 추가'를 탭해 메시지방 할 일을 할 일 목록에 추가합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 '할 일' 탭을 클릭합니다.
5. 할 일에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_more_3f4247.png) 아이콘을 클릭합니다.
6. '할 일에서 보기'를 선택해 메시지방 할 일을 할 일 목록에 추가합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
