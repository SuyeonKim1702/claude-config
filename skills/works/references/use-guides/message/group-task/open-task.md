# 할 일 열기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
여러 명이 참여하는 메시지방에서 할 일을 열 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '할 일'을 탭해 할 일을 엽니다.

일정에서 화면을 왼쪽으로 쓸어넘겨 할 일을 열 수도 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방을 클릭합니다.
4. 메시지방 오른쪽의 '할 일' 탭을 클릭합니다.

### PC 앱

1. 왼쪽의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/gnb-task-bk@3x.png) 아이콘을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
