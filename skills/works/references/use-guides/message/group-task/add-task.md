# 할 일 쓰기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 멤버라면 누구나 할 일을 등록할 수 있습니다.

할 일의 내용을 입력한 뒤 기한, 요청자, 담당자, 완료 조건을 선택하고 파일을 첨부해 할 일을 작성할 수 있습니다.

본인이 담당자, 요청자로 지정된 할 일은 [담당 할 일, 요청 할 일](/ko/use-guides/task/task-list/my-task) 또는 [내 캘린더의 할 일 캘린더](/ko/use-guides/calendar/calendar/calendar-type/#i)에서 볼 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭하고 '할 일'을 탭합니다.
2. 할 일에서 ![](/wp-content/uploads/2024/05/common_24_pencilwrite_fill.png) 아이콘을 탭합니다.
3. 할 일의 내용을 입력하고 기한, 요청자, 담당자, 완료 조건을 선택한 뒤 필요할 경우 파일을 첨부합니다.  
   단, 담당자를 복수로 지정한 경우에만 완료 조건을 선택할 수 있습니다.
4. '저장'을 탭해 메시지방에 할 일을 등록합니다. 메시지방에 알림을 보내려면 '예'를 선택합니다.

### PC 웹

1. 메시지방에서 화면 오른쪽의 '할 일' 탭을 클릭합니다.
2. 오른쪽 상단의 '할 일 쓰기'를 클릭합니다.
3. 할 일의 내용을 입력하고 기한, 요청자, 담당자, 완료 조건을 선택한 뒤 필요할 경우 파일을 첨부합니다.  
   단, 담당자를 복수로 지정한 경우에만 완료 조건을 선택할 수 있습니다.
4. '저장'을 클릭하여 메시지방에 할 일을 등록합니다.메시지방에 알림을 보내려면 '메시지방에 알림 보내기'를 선택하여 '확인'을 클릭합니다.

# 제목 입력

할 일의 제목은 최대 200자까지 입력할 수 있습니다.

입력한 제목은 할 일 목록에 노출됩니다.

# 내용 입력

할 일의 내용은 5,000자까지 입력할 수 있습니다.

# 기한 선택

할 일의 기한을 '오늘', '내일', '다음 주' 중에서 선택하거나, 캘린더에서 날짜를 선택할 수 있습니다.

기본 설정은 '없음'입니다.

# 카테고리 선택

메시지방 할 일이 위치할 카테고리를 선택할 수 있습니다.

기본 설정은 해당 메시지방의 '미분류 할 일' 카테고리며, 카테고리를 선택해 할 일을 저장하면 해당 카테고리 상단에 할 일이 생성됩니다.

### 모바일 앱

1. 할 일 쓰기 화면에서 '카테고리' 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-arrow-line-next-767676.png) 아이콘을 탭합니다.
2. 원하는 카테고리를 선택합니다.

### PC 웹

할 일 쓰기 화면에서 '카테고리' 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-8-arrow-down-222222.png) 아이콘을 클릭해 원하는 카테고리를 선택합니다.

# 요청자 및 담당자 선택

메시지방 멤버를 검색하거나 멤버 목록에서 선택해 요청자 및 담당자로 지정할 수 있습니다.

요청자는 1명, 담당자는 0명부터 100명까지 선택할 수 있습니다.

기본 설정은 본인으로 되어있습니다.

# 완료 조건 선택

담당자를 복수로 지정한 경우, 할 일의 완료 조건을 선택할 수 있습니다.

- 담당자 중 1명 완료: 담당자 중 1명만 완료해도 할 일이 완료됩니다.
- 담당자 모두 완료: 모든 담당자가 완료해야 할 일이 완료됩니다.

기본 설정은 '담당자 중 1명 완료'입니다.

# 파일 첨부

할 일에 파일을 최대 10개, 총 2GB까지 첨부할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
