# 할 일 보기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
할 일을 등록순, 기한순으로 정렬하거나 완료된 할 일을 볼지 설정할 수 있습니다.

# 할 일 목록 정렬

아래 기준으로 할 일 목록을 정렬해 볼 수 있습니다.

- 등록순: 최근에 등록된 할 일부터 정렬됩니다.
- 기한순: 지난, 오늘, 내일, 7일 이내, 이후, 기한 없음 순으로 정렬됩니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭하고 '할 일'을 탭합니다.
2. 할 일 목록에서 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_align.png) 아이콘을 탭합니다.
3. 원하는 정렬 기준을 탭합니다.

### PC 웹

1. 메시지방에서 화면 오른쪽의 '할 일' 탭을 클릭합니다.
2. 할 일 목록에서 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_align.png) 아이콘을 클릭합니다.
3. 원하는 정렬 기준을 클릭합니다.

# 완료된 할 일 보기 설정

완료된 할 일의 노출 여부를 설정할 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭하고 '할 일'을 탭합니다.
2. 할 일 목록에서 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_align.png) 아이콘을 탭합니다.
3. '완료된 할 일 보기'를 켜거나 끕니다.

### PC 웹

1. 메시지방에서 화면 오른쪽의 '할 일' 탭을 클릭합니다.
2. 할 일 목록에서 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_align.png) 아이콘을 클릭합니다.
3. 완료된 할 일을 목록에서 보려면 '완료된 할 일 보기'를 선택하고, 보기를 원하지 않는다면 선택 해제합니다.

# 할 일 완료 또는 미완료로 변경

할 일 목록에서 할 일을 완료 또는 미완료로 변경할 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭하고 '할 일'을 탭합니다.
2. 할 일 목록에서 할 일 왼쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_18_checkbox_normal_fill.png) 아이콘을 탭해 할 일을 완료하거나, ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-ico24TodoCheck.png) 아이콘을 탭해 미완료로 변경합니다.

### PC 웹

1. 메시지방에서 화면 오른쪽의 '할 일' 탭을 클릭합니다.
2. 할 일 목록에서 할 일 왼쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_18_checkbox_normal_fill.png) 아이콘을 클릭해 할 일을 완료 하거나, ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-ico24TodoCheck.png) 아이콘을 클릭해 미완료로 변경합니다.

할 일 상세 화면에서도 할 일을 완료 또는 미완료로 변경할 수 있습니다.

# 나만 완료 또는 미완료로 변경

담당자가 여러 명이면서 완료 조건이 '담당자 모두 완료'인 할 일의 경우, 할 일 목록에서 내 할 일을 완료하거나 미완료로 변경할 수 있습니다.

할 일을 완료하거나 미완료로 변경하더라도 다른 담당자의 완료 상태는 유지됩니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭하고 '할 일'을 탭합니다.
2. 할 일 목록에서 할 일 왼쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_18_checkbox_normal_fill.png) 아이콘을 탭하면 나만 완료로 변경되고, ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic16_membertodo_157efb.png) 아이콘이 표시됩니다. 한 번 더 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_18_checkbox_normal_fill.png) 아이콘을 탭하면 나만 미완료로 변경되고, ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic16_membertodo_157efb.png) 아이콘이 사라집니다.

### PC 웹

1. 메시지방에서 화면 오른쪽의 '할 일' 탭을 클릭합니다.
2. 할 일 목록에서 할 일 왼쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_18_checkbox_normal_fill.png) 아이콘을 클릭하면 나만 완료로 변경되고, ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic16_membertodo_157efb.png) 아이콘이 표시됩니다. 한 번 더  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_18_checkbox_normal_fill.png) 아이콘을 클릭하면 나만 미완료로 변경되고, ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic16_membertodo_157efb.png) 아이콘이 사라집니다.

[할 일 상세 화면](/ko/use-guides/task/mark-complete-or-incomplete/#i)에서도 '나만 완료하기' 또는 '나만 미완료로 변경'을 눌러 할 일의 상태를 변경할 수 있습니다.

# 할 일 상세 보기

내용, 기한, 요청자, 담당자, 완료 조건, 첨부 파일 등 할 일의 상세 정보를 확인할 수 있습니다.

할 일에 첨부된 사진이나 동영상, 파일을 기기에 다운로드하거나 드라이브에 저장할 수 있습니다.

# 작업 내역 확인

할 일의 작업 시간 및 요청자, 담당자를 확인할 수 있습니다.

작업 내역은 다음 경우에 추가됩니다.

- 할 일 생성
- 내용, 기한,  요청자, 담당자, 완료 조건 변경
- 첨부 추가, 삭제
- 메시지방에 공유
- 할 일 완료, 미완료로 변경
- 담당자가 나만 완료, 미완료로 변경
- 할 일 자동 완료, 미완료로 변경
- 할 일 삭제

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭하고 '할 일'을 탭합니다.
2. 할 일 목록에서 확인할 할 일을 탭하고 하단의 작업 내역을 확인합니다.

### PC 웹

1. 메시지방에서 화면 오른쪽의 '할 일' 탭을 클릭합니다.
2. 할 일 목록에서 확인할 할 일을 클릭하고 하단의 작업 내역을 확인합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
