# 메시지방 목록

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
카테고리, 정렬, 핀 고정, 숨기기, 나가기 기능을 사용하여 메시지방 목록을 관리할 수 있습니다.

# 메시지방 카테고리

전체 메시지방 외에도, 1:1, 여러 명 등 카테고리별로 메시지방이 자동으로 분류되며, 카테고리를 선택해 원하는 메시지방을 쉽게 찾을 수 있습니다.

직접 개인 카테고리를 만들어 원하는 메시지방만 모아보거나, 카테고리의 순서 이동 및 숨김 여부도 설정할 수 있습니다.

메시지방 카테고리 기능은 모바일 앱과 PC 앱에서만 제공됩니다.

## 카테고리 추가

### 모바일 앱

관리화면에서 추가하기

1. 메시지방 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png) 아이콘을 탭합니다.
2. '새 카테고리 만들기'를 탭합니다.
3. 카테고리 이름을 입력하고 '메시지방 추가'를 눌러 포함할 메시지방을 선택합니다.
4. '저장'을 탭합니다.

메시지방 목록에서 추가하기

1. 메시지방 목록에서 메시지방을 길게 눌러 포함할 메시지방을 선택합니다. 여러 개를 선택할 수도 있습니다.
2. 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/10/icon-n-act_com_folder_move.png) 아이콘을 선택 후,  '새 카테고리 만들기'를 탭합니다.
3. 카테고리 이름을 입력한 후 '저장'을 탭합니다.

### PC 앱

관리화면에서 추가하기

1. 메시지방 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png)  아이콘을 클릭합니다.
2. '새 카테고리 만들기'를 클릭합니다.
3. 카테고리 이름을 입력하고 '메시지방 추가'를 눌러 포함할 메시지방을 선택합니다.
4. '저장'을 클릭합니다.

메시지방 목록에서 추가하기

1. 메시지방 목록에서 추가할 카테고리에 포함할 메시지방을 마우스 오른쪽 버튼으로 클릭합니다.
2. '새 카테고리 만들기'를 클릭합니다.
3. 카테고리 이름을 입력하고 '메시지방 추가'를 클릭합니다.
4. '저장'을 클릭합니다.

## 카테고리 수정

직접 추가한 개인 카테고리만 수정할 수 있습니다.

### 모바일 앱

1. 메시지방 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png)  아이콘을 탭합니다.
2. 수정할 카테고리를 선택합니다.
3. 카테고리 이름 또는 포함할 메시지방을 수정하고 '저장'을 탭합니다.

### PC 앱

1. 메시지방 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png) 아이콘을 클릭합니다.
2. 수정할 카테고리를 클릭합니다.
3. 카테고리 이름 또는 포함할 메시지방을 수정하고 '저장'을 클릭합니다.

## 카테고리 삭제

직접 추가한 개인 카테고리만 삭제할 수 있습니다.

### 모바일 앱

1. 메시지방 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png) 아이콘을 탭합니다.
2. 수정할 카테고리를  선택합니다.
3. 카테고리 수정 화면에서 하단의 '카테고리 삭제'를 탭합니다.
4. '삭제'를 탭합니다.

### PC 앱

1. 메시지방 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png) 아이콘을 클릭합니다.
2. 수정할 카테고리를 클릭합니다.
3. 카테고리 수정 화면에서 하단의 '카테고리 삭제'를 클릭합니다.
4. '삭제'를 클릭합니다.

## 다른 카테고리로 이동

직접 추가한 개인 카테고리만 이동할 수 있습니다.

### 모바일 앱

1. 개인 카테고리 목록에서 메시지방 하나를 길게 눌러, 메시지방을 선택합니다.
2. 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/10/icon-n-act_com_folder_move.png) 아이콘을 탭합니다.
3. '다른 카테고리에 이동'을 탭합니다.
4. '저장'을 클릭합니다.

### PC 앱

1. 개인 카테고리 목록에서 이동할 메시지방을 마우스 오른쪽 버튼으로 클릭합니다.
2. '카테고리에 추가' 를 클릭합니다.
3. 이동할 카테고리를 클릭합니다.

## 카테고리 해제

직접 추가한 개인 카테고리만 해제할 수 있습니다.

### 모바일 앱

1. 개인 카테고리의 메시지방 목록에서 메시지방 하나를 길게 눌러, 메시지방을 선택합니다.
2. 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/10/icon-n-act_com_folder_move.png) 아이콘을 탭합니다.
3. '카테고리 해제'를 탭합니다.

### PC 앱

1. 개인 카테고리의 메시지방 목록에서 해제할 메시지방을 마우스 오른쪽 버튼으로 클릭합니다.
2. '카테고리 변경'을 클릭합니다.
3. '카테고리 해제'를 클릭합니다.

## 카테고리 순서 변경

전체를 제외한 모든 카테고리의 노출 순서는 카테고리 관리 화면에서 조정할 수 있습니다.

### 모바일 앱

1. 메시지방 목록 상단에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png) 아이콘을 탭합니다.
2. '목록 편집'을 선택합니다.
3. 이동할 카테고리를 눌러 원하는 위치로 끌어다 놓습니다.
4. '저장'을 탭합니다.

### PC 앱

1. 메시지방 목록 상단에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png) 아이콘을 클릭합니다.
2. '목록 편집'을 선택합니다.
3. 이동할 카테고리를 눌러 원하는 위치로 끌어다 놓습니다.
4. '저장'을 클릭합니다.

## **카테고리 숨김 설정**

자주 사용하지 않는 카테고리는 메시지방 목록에서 보이지 않도록 숨겨둘 수 있습니다.

숨겨진 카테고리는 하단의 표시 안함 영역에서 확인할 수 있으며, + 아이콘을 눌러 다시 표시할 수 있습니다.

### 모바일 앱

1. 메시지방 목록 상단에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png) 아이콘을 탭합니다.
2. '목록 편집'을 선택합니다.
3. 숨기려는 카테고리 앞의 – 아이콘을 탭합니다.
4. '저장'을 탭합니다.

### PC 앱

1. 메시지방 목록 상단에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic24_categorysetting_line.png) 아이콘을 클릭합니다.
2. '목록 편집'을 선택합니다.
3. 숨기려는 카테고리 앞의 – 아이콘을 클릭합니다.
4. '저장'을 클릭합니다.

특정 카테고리가 선택된 상태에서 ![](/wp-content/uploads/2024/05/ServiceMessage.png) 아이콘을 눌러 전체 카테고리로 빠르게 돌아갈 수도 있습니다.

# 메시지방 정렬

메시지방 목록은 선택된 기준에 따라 자동으로 정렬됩니다.

기본 설정은 시간 순이며, 원하는 대로 설정을 변경할 수 있습니다.

- 시간순: 최근 메시지가 있는 메시지방부터 정렬됩니다.
- 안 읽은 순: 메시지방 종류와 상관없이 안 읽은 메시지가 있는 메시지방부터 정렬됩니다.
- 종류순: 공식/Bot, 조직/그룹, 일반 메시지방의 순서로 분류되며, 종류별로 목록을 접거나 펼쳐 볼 수 있습니다. 종류순 정렬은 PC웹에서만 제공됩니다.

### 모바일 앱

1. 메시지 목록에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-align-767676.png) 아이콘을 탭합니다.
2. 메뉴에서 원하는 정렬 기준을 탭해 정렬합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-align-767676.png) 아이콘을 클릭합니다.
4. 메뉴에서 원하는 정렬 기준을 클릭해 정렬합니다.

### PC 앱

1. 화면 왼쪽 ![](/wp-content/uploads/2024/05/ServiceMessage.png) 아이콘을 클릭합니다.
2. 메시지방 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-align-767676.png) 아이콘을 클릭합니다.
3. 메뉴에서 원하는 정렬 기준을 클릭해 정렬합니다.

# 메시지방 핀 고정 및 해제

자주 사용하는 메시지방을 목록 상단에 고정하거나, 고정 상태를 해제할 수 있습니다.

### 모바일 앱

1. 메시지방 목록에서 상단에 고정하고 싶은 메시지방을 왼쪽으로 쓸어 넘깁니다.
2. '핀 고정' 또는 '핀 해제'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 핀 고정 또는 핀 고정 해제할 메시지방 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
4. '핀 고정'을 클릭하여 상단 고정하거나 '핀 해제'를 클릭하여 상단 고정을 해제합니다.

### PC 앱

1. 화면 왼쪽의 ![](/wp-content/uploads/2024/05/ServiceMessage.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 핀 고정 또는 핀 고정 해제할 메시지방을 마우스 오른쪽 버튼으로 클릭합니다.
3. '핀 고정'을 클릭하여 상단 고정하거나 '핀 해제'를 클릭하여 상단 고정을 해제합니다.

# 메시지방 숨기기

목록에서 특정 메시지방을 숨길 수 있습니다.

### 모바일 앱

1. 메시지방 목록에서 숨기고 싶은 메시지방을 왼쪽으로 쓸어 넘깁니다.
2. '숨김'을 탭합니다.
3. 대화창에서 '숨김'을 탭해 메시지방을 숨깁니다.

메시지방 하나를 길게 누른 후 여러 개의 메시지방을 선택하여 한꺼번에 숨길 수 있습니다.

모바일에서 메시지방을 숨겨도 PC웹과 PC앱에는 반영되지 않습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 숨길 메시지방 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
4. '숨김'을 클릭합니다.
5. 숨길 다른 메시지방이 있다면 함께 선택한 후 '숨김'을 클릭하여 메시지방을 숨깁니다.

PC웹에서 메시지방을 숨겨도 모바일과 PC앱에는 반영되지 않습니다.

### PC 앱

1. 화면 왼쪽의 ![](/wp-content/uploads/2024/05/ServiceMessage.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 숨길 메시지방을 마우스 오른쪽 버튼으로 클릭합니다.
3. '숨김'을 클릭합니다.
4. 숨길 다른 메시지방이 있다면 함께 선택한 후 '숨김'을 클릭하여 메시지방을 숨깁니다.

PC앱에서 메시지방을 숨겨도 모바일과 PC웹에는 반영되지 않습니다.

목록에서 숨긴 메시지방은 새로운 메시지가 도착하면 목록에 다시 나타납니다. 메시지방을 다시 보고 싶지 않은 경우에는 메시지방을 나가면 됩니다.

# 메시지방 나가기

메시지방 목록에서 메시지방을 나갈 수 있습니다. 일반 메시지방에서만 나갈 수 있고, [조직 메시지방](/ko/use-guides/message/message-room/team-message-room/#i-5) 및 [그룹 메시지방](/ko/use-guides/message/message-room/group-message-room/group-message-room/#i-4)에서는 스스로 나갈 수 없습니다.

### 모바일 앱

1. 메시지방 목록에서 나가고 싶은 메시지방을 왼쪽으로 쓸어 넘깁니다.
2. '나가기'를 탭합니다.
3. 대화창에서 '확인'을 탭합니다.

메시지방 하나를 길게 누른 후 여러 개의 메시지방을 선택하여 한꺼번에 나갈 수도 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 나가고 싶은 메시지방 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
4. '나가기'를 클릭합니다.
5. 나가고 싶은 다른 메시지방이 있다면 함께 선택한 후 '나가기'를 클릭합니다.

### PC 앱

1. 화면 왼쪽의 ![](/wp-content/uploads/2024/05/ServiceMessage.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 나가고 싶은 메시지방을 마우스 오른쪽 버튼으로 클릭합니다.
3. '나가기'를 클릭합니다.
4. 나가고 싶은 다른 메시지방이 있다면 함께 선택한 후 '나가기'를 클릭합니다.

메시지방을 나가면 해당 메시지방에서 나눈 대화 내용을 더 이상 볼 수 없습니다. 내가 보낸 메시지는 그 방에 남아 있어 다른 멤버는 볼 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
