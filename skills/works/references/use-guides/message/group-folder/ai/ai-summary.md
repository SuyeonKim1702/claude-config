# AI 요약하기

이 기능은 AI 스튜디오 상품 신청 후 사용 가능합니다.

AI 요약 기능을 사용해 문서의 핵심 내용을 한눈에 확인할 수 있습니다.

요약문의 길이(짧게/중간/길게)와 형식(목록형/문장형)을 원하는 옵션으로 선택해 요약할 수 있습니다.

요약을 지원하는 파일의 확장자는 아래와 같습니다.

- MS Office 파일(DOC, DOCX, PPT, PPTX, XLS, XSLX)
- PDF 파일(PDF)
- 한글 파일(HWP, HWPX)
- 텍스트 파일(TXT, RTF, XML)

# 목록에서 요약하기

### PC 웹

1. 파일 목록에서 요약할 파일 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_more_3f4247.png) 아이콘을 클릭합니다.
2. 'AI 요약'을 클릭합니다.

# 미리보기(문서 뷰어)에서 요약하기

### PC 웹

1. 파일 목록에서 미리보기로 열어볼 파일을 더블 클릭하거나, 파일 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_more_3f4247.png) 아이콘을 클릭한 후 '미리보기'를 클릭하여 문서 뷰어를 엽니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/06/ic_ai_shorten_b.png) 아이콘을 클릭합니다.

# **추가 기능**

### PC 웹

- 오른쪽 하단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic18_copy_3f4247.png)  아이콘을 클릭하여 요약문을 복사할 수 있습니다.
- 오른쪽 하단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/10/ic18_refresh_3f4247.png) 아이콘을 클릭하여 요약문을 다시 생성할 수 있습니다.
- 왼쪽 하단 '요약 방식 변경' 버튼을 클릭한 후, 길이 또는 형식 옵션을 선택하여 요약할 수 있습니다.

- 최대 5개의 파일을 한 번에 요약할 수 있습니다.
- 최대 100MB의 문서 파일에 대해 요약 기능을 지원합니다.
- 문서 본문이 10만 자를 초과하는 경우, 처음 10만 자까지만 요약됩니다.
- 문서 내 이미지에 포함된 텍스트는 요약되지 않습니다.
- 암호가 설정된 문서는 요약되지 않습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
