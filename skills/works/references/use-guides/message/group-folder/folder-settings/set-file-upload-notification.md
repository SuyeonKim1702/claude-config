# 파일 업로드 시 메시지 알림 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방의 멤버는 폴더 설정에서 파일 업로드 알림 기능의 사용 여부를 설정할 수 있습니다.

업로드 알림을 사용하면 멤버가 폴더에 파일을 업로드할 때 해당 메시지방에 자동으로 알림이 발송됩니다.

단, [접근 멤버가 제한된 폴더](/ko/use-guides/message/group-folder/folder-settings/permission/)에서는 파일 업로드 알림이 발송되지 않습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. 폴더에서 '파일 업로드 알림'을 켜거나 끕니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽  '폴더' 탭을 클릭합니다.
5. 화면 오른쪽 하단의 '설정'을 클릭합니다.
6. '새 파일 알림'을 설정합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
