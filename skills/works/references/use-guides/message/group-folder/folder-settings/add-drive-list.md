# 드라이브에서 보기 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
자주 사용하는 메시지방 폴더를 드라이브의 목록에 추가할 수 있습니다.

드라이브에서 보기 설정은 개인 설정으로, 다른 멤버의 목록은 변경되지 않습니다.

이 설정은 기본적으로 선택 해제되어 있습니다.

선택한 메시지방 폴더는 모바일 앱, PC 웹 드라이브에서 확인할 수 있으며, 드라이브 탐색기에서 동기화할 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '폴더'를 탭합니다.
3. 폴더에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
4. '드라이브에서 보기'를 탭합니다.
5. '드라이브 목록에 추가'를 탭해 메시지방 폴더를 드라이브 목록에 추가합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 '폴더' 탭을 클릭합니다.
5. 폴더 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_more_3f4247.png) 아이콘을 클릭합니다.
6. '드라이브에서 보기'를 선택해 메시지방 폴더를 드라이브 목록에 표시합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
