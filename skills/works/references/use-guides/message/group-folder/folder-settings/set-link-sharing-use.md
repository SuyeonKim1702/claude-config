# 링크 공유 기능 사용 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방의 멤버는 폴더 설정에서 [링크로 보내기](/ko/use-guides/message/group-folder/send-file-or-link/send-as-link/share-link/) 기능의 사용 여부를 설정할 수 있습니다.

조직/그룹 폴더의 경우 폴더 마스터(조직장 또는 그룹 마스터)만 링크로 보내기 설정을 변경할 수 있습니다.

단, 관리자가 [메시지방 폴더에서 링크 공유를 제한](/ko/admin-guides/manage-service/drive/general-settings/#i-2)한 경우 이 설정이 노출되지 않습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '링크 공유 기능'을 켜거나 끕니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 '폴더' 탭을 클릭합니다.
5. 화면 오른쪽 하단의 '설정'을 클릭합니다.
6. '링크 공유'에서 링크 공유 기능 사용 여부를 설정합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
