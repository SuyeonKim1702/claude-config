# 파일 및 폴더 검색

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
파일 및 폴더를 검색할 수 있습니다.

자료의 이름과 확장자(.pptx 등)로 검색을 할 수도 있고, 파일의 본문을 검색하거나 검색 범위와 기간, 확장자를 선택해 검색하는 상세검색을 이용할 수도 있습니다.

파일은 업로드 1분 후부터 검색이 가능하며, 검색 언어로 한국어, 영어, 일본어, 중국어 간체, 중국어 번체를 지원합니다.

# 파일 및 폴더 이름 검색

파일 및 폴더의 이름으로 자료를 검색합니다. 엑셀 파일이나 파워포인트 파일을 찾을 때에는 확장자명을 입력하면 빠르게 찾을 수 있습니다.

### 모바일 앱

1. 폴더에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 탭합니다.
2. 검색할 파일이나 폴더 이름을 입력합니다.  
   키워드를 한글자씩 입력할때 마다 그 글자가 포함된 파일이나 이름이 검색됩니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/pc-icon-contact-search-button.png) 아이콘을 클릭합니다.
5. 검색할 키워드를 입력합니다.
6. '폴더' 탭을 클릭하여 이름에 키워드가 포함된 파일이나 폴더를 검색합니다.

# 폴더 내 파일의 본문 검색

폴더에 저장된 문서파일의 본문 내용 속 단어를 검색해 파일을 찾을 수 있습니다.

본문 검색을 지원하는 파일의 확장자는 다음과 같습니다.

- MS Office 파일 (.doc, .docx, .ppt, .pptx, .xls, .xlsx)
- PDF 파일 (.pdf)
- 한글 파일 (.hwp, hwpx)
- 텍스트 파일 (.txt, .rtf, .xml)

검색을 할 수 있는 언어는 한국어, 영어, 일본어, 중국어 간체, 중국어 번체입니다.

### 모바일 앱

1. 폴더에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 탭합니다.
2. 검색할 키워드를 입력합니다.
3. '본문'을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/pc-icon-contact-search-button.png) 아이콘을 클릭합니다.
5. 검색할 키워드를 입력합니다.
6. '폴더' 탭을 클릭하여 본문에 키워드가 포함된 파일을 검색합니다.

단, PC웹에서는 이름 검색 결과와 본문 검색 결과가 함께 노출됩니다.

# 상세 검색

특정 폴더, 파일 유형, 파일 수정일 등 검색 범위를 지정해서 검색할 수 있습니다.

### 모바일 앱

1. 폴더에서  ![](/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 탭합니다.
2. '상세'를 탭합니다.
3. 검색할 폴더를 선택합니다. 폴더를 선택하지 않으면 디폴트로 전체 폴더를 대상으로 합니다.
4. 파일 유형을 선택합니다.
5. 파일 수정일의 기간을 선택합니다.
6. 검색할 키워드를 입력해 검색합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
