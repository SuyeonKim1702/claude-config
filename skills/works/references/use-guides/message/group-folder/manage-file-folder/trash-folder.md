# 삭제한 파일 및 폴더의 복원, 영구 삭제

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
[메시지방 폴더에서 삭제](/ko/use-guides/message/group-folder/manage-file-folder/file-folder-manage/#i-5)된 파일 및 폴더는 각 메시지방 폴더의 휴지통으로 보관됩니다.

휴지통에서는 삭제된 파일 및 폴더를 확인할 수 있고, 복원하거나 영구 삭제할 수 있습니다.

# 삭제한 파일 또는 폴더 복원

폴더 휴지통에서 파일이나 폴더를 복원할 수 있습니다.

복원된 파일 및 폴더는 삭제 전 위치에 복원됩니다.

### 모바일 앱

1. 폴더에서  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png)  아이콘을 탭합니다.
2. '휴지통'을 탭합니다.
3. 복원할 파일 또는 폴더의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
4. '복원'을 탭해 복원합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽  '폴더' 탭을 클릭합니다.
5. 폴더 목록 하단의 '휴지통'을 클릭합니다.
6. 복원할 파일 또는 폴더의 오른쪽   ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
7. '복원'을 클릭하여 복원합니다.

# 파일 또는 폴더 영구 삭제

삭제한 파일 또는 폴더는 해당 폴더의 휴지통으로 이동하며, 영구 삭제될 때까지 보관됩니다.

파일을 영구 삭제하면 파일 크기만큼 사용할 수 있는 공용용량이 확보됩니다.

아래 두 가지 방법으로 영구 삭제할 수 있습니다.

- 개별 파일 또는 폴더의 영구 삭제
- 휴지통 비우기

## 개별 파일 또는 폴더의 영구 삭제

삭제한 파일 및 폴더를 선택해서 영구 삭제할 수 있습니다. 영구 삭제한 파일 또는 폴더는 복원할 수 없습니다.

조직/그룹 폴더에서는 멤버 본인이 삭제한 파일만 영구 삭제할 수 있습니다. 조직장 또는 그룹 마스터는 다른 멤버가 삭제한 파일들도 영구 삭제할 수 있습니다.

### 모바일 앱

1. 폴더에서  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
2. '휴지통'을 탭합니다.
3. 영구 삭제할 파일 또는 폴더의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
4. '영구 삭제'를 탭해 파일 또는 폴더를 영구 삭제합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 폴더 목록 하단의 '휴지통'을 클릭합니다.
6. 영구 삭제할 파일 또는 폴더의 오른쪽   ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
7. '영구 삭제'를 클릭하여 파일 또는 폴더를 영구 삭제합니다.

## 휴지통 비우기

휴지통으로 이동한 파일이나 폴더 전체 항목을 한 번에 영구 삭제할 수 있습니다. 영구 삭제한 파일 또는 폴더는 복원할 수 없습니다.

조직/그룹 폴더에서는 조직장 또는 그룹 마스터만 휴지통을 비울 수 있습니다.

휴지통을 비우면 사용 중이던 [휴지통 용량](/ko/use-guides/message/group-folder/folder-settings/check-fold-and-trash-usage/#i-2)만큼 사용할 수 있는 공용용량이 확보됩니다.

### 모바일 앱

1. 폴더에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭합니다.
2. '휴지통'을 탭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_trash_all.png) 아이콘을 탭해서 휴지통의 전체 항목을 영구 삭제합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 폴더 목록 하단의 '휴지통'을 클릭합니다.
6. 목록 왼쪽 상단의 '휴지통 비우기'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
