# 파일 잠금

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 폴더에서 다른 멤버가 파일을 편집하길 원하지 않을 경우 '파일 잠금' 기능을 사용할 수 있습니다.

파일을 잠그면 다른 멤버에게는 읽기 전용으로 열리고, 같은 파일명으로 편집할 수 없습니다.

목록에서 잠근 파일에는 ![](/wp-content/uploads/2024/04/icon_lock-2.png) 아이콘이 표시됩니다.

# 파일 잠그기

메시지방 폴더에서 편집 권한이 있는 멤버는 파일 잠금을 할 수 있습니다.

### 모바일 앱

1. 잠글 파일의![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '상세 정보'를 탭합니다.
3. '파일 잠금' 설정을 켭니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 잠글 파일의 오른쪽   ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
6. '파일 잠금'을 클릭하여 파일 잠금을 설정합니다.

# 파일 잠금 정보 확인하기

잠긴 파일의 잠금 날짜, 잠근 사람의 정보를 확인할 수 있습니다.

### 모바일 앱

1. 잠금 정보를 확인할 파일 옆의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '상세 정보'를 탭합니다.
3. 잠금 날짜와 잠근 사람을 확인합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 잠금 정보를 확인할 파일의 오른쪽   ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
6. '파일 잠금 정보'를 클릭하여 '파일 잠금 정보' 화면에서 '잠근 날짜'와 '잠근 사람'을 확인합니다.

# 파일 잠금 해제하기

메시지방 폴더에서 편집 권한이 있는 멤버는 파일 잠금 해제를 할 수 있습니다.

잠금 해제 시 잠근 사용자에게 알림 메시지가 발송됩니다.

### 모바일 앱

1. 잠금을 해제할 파일 옆의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '상세 정보'를 탭합니다.
3. '파일 잠금' 설정을 끕니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 잠금을 해제할 파일의 오른쪽   ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
6. '잠금 해제'를 클릭합니다.
7. '파일 잠금 정보' 화면 하단의 '잠금 해제'를 클릭하여 파일 잠금을 해제합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
