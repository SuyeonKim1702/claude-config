# 문서 편집하기

이 기능은 네이버웍스 코어 상품의 Standard Plus 플랜 또는 웍스 드라이브 상품의 Standard Plus 플랜 신청 후 사용 가능합니다.

외부 프로그램 없이도 메시지방 폴더에서 문서 파일을 편집할 수 있습니다.

문서 편집기에서는 300MB 이하 파일만 편집할 수 있습니다.

문서 파일에 접근 권한이 있는 구성원끼리 문서를 실시간으로 공동 편집할 수 있습니다.

# 문서 편집기 지원 파일

문서 편집기가 지원하는 파일의 확장자는 다음과 같습니다.

- MS Office 파일 (.doc, .docx, .ppt, .pptx, .xls, .xlsx)
- 기타 파일 (.csv, .txt)

# 문서 편집기 사용

메시지방 폴더에서 문서를 편집기로 열어 실시간으로 공동 편집하고 저장할 수 있습니다.

### PC 웹

1. 파일 목록에서 문서 편집기로 열어볼 파일의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_more_3f4247.png) 아이콘을 클릭합니다.
2. '문서 편집'을 클릭합니다.
