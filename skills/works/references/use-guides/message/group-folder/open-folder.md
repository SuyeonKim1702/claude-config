# 폴더 열기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
여러 명이 참여하는 메시지방에서 폴더를 열 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '폴더'를 탭해서 폴더를 엽니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_align_line.png) : 폴더 또는 파일을 수정일, 종류, 이름순으로 정렬합니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) : 폴더 또는 파일을 검색합니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) : '드라이브에서 보기'를 설정하거나 휴지통을 확인합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/02/icon-n-ic18_refresh_3f4247.png) : 최신 상태로 갱신합니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/02/icon-n-ic18_align_3f4247.png) : 폴더 또는 파일을 다음 정렬 기준 및 순서(오름차순/내림차순)로 정렬합니다.
     - 정렬 기준: 이름순, 크기순, 수정한 날짜순, 수정한 사람순, 생성한 날짜순
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_more_3f4247.png): '드라이브에서 보기'를 설정합니다. 설정을 켠 폴더는 드라이브 서비스 목록에서도 볼 수 있습니다.
   - '+ 새로 만들기' : '새 폴더 만들기' 또는 '파일 올리기'를 선택합니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/10/ic20_info.png): '상세 정보 및 활동'이 열립니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/04/pc-icon-contact-folder-button.png) 아이콘을 클릭합니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/02/icon-n-ic18_refresh_3f4247.png) : 최신 상태로 갱신합니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/02/icon-n-ic18_align_3f4247.png) : 폴더 또는 파일을 다음 정렬 기준 및 순서(오름차순/내림차순)로 정렬합니다.
     - 정렬 기준: 이름순, 크기순, 수정한 날짜순, 수정한 사람순, 생성한 날짜순
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_more_3f4247.png): '드라이브에서 보기'를 설정합니다. 설정을 켠 폴더는 드라이브 서비스 목록에서도 볼 수 있습니다.
   - '+ 새로 만들기' : '새 폴더 만들기' 또는 '파일 올리기'를 선택합니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/10/ic20_info.png): '상세 정보 및 활동'이 열립니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
