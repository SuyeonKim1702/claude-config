# 링크 관리 및 히스토리 조회

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
[공유 중인 링크](/ko/use-guides/message/group-folder/send-file-or-link/send-as-link/share-link/)별로 설정을 변경하거나 링크를 삭제할 수 있습니다.

링크를 공유 받은 사람이 링크로 연 페이지에서 어떤 활동을 했는지 히스토리를 조회할 수 있습니다.

관리자는 Admin에서 [링크 공유 기능의 허용 여부와 링크 유효 기간을 설정하거나 링크의 접근 권한을 제한](/ko/admin-guides/manage-service/drive/general-settings/)할 수 있습니다.

# 링크 설정

## 접근 권한

링크를 공유할 때는 보안을 위해 다음의 접근 권한 중 하나를 설정할 수 있습니다.

[관리자의 설정](/ko/admin-guides/manage-service/drive/general-settings/#i-3)에 따라 일부 접근 권한은 표시되지 않을 수 있습니다.

- 모든 사람 (비밀번호 설정 가능)
- 사내 구성원
- OTP 인증을 통해 접근 가능

접근 권한 설정 외의 이유로 링크에 접근할 수 없는 경우는 아래와 같습니다.

- 공유자가 링크를 삭제한 경우
- 링크의 유효기간이 만료되어 링크가 삭제된 경우
- 링크의 접근 권한이 변경되어 접근 권한이 없는 경우
- (보안링크) 공유 멤버에서 제외된 경우
- (보안링크) 최초 인증한 브라우저가 아닌 다른 브라우저에서 접근한 경우

## 유효 기간

유효 기간은 제한없음, 7일, 30일, 90일, 180일 중 지정할 수 있습니다.

[관리자의 설정](/ko/admin-guides/manage-service/drive/general-settings/#i-2-2)에 따라 일부 유효 기간은 설정하지 못할 수 있습니다.

## 편집 권한 범위

기본적으로 링크는 편집 권한으로 생성되고 보안과 사용성을 고려하여 권한을 설정할 수 있습니다.

링크로 공유하는 대상이 폴더인지 파일인지에 따라 제공되는 권한 옵션이 달라집니다.

|  |  |  |
| --- | --- | --- |
|  | **폴더** | **파일** |
| 편집 | 올리기, 내려받기, 미리보기, 편집, 이동, 삭제 | 내려받기, 미리보기, 편집, 삭제 |
| 올리기 및 내려받기 | 올리기, 내려받기, 미리보기 | 미제공 |
| 올리기 | 올리기, 미리보기 | 미제공 |
| 내려받기 | 내려받기, 미리보기 | 내려받기, 미리보기 |
| 미리보기 | 미리보기 | 미리보기 |

# 링크 설정 확인 및 변경

공유한 링크에 대해 접근 권한, 유효기간, 편집 권한을 변경할 수 있습니다.

일반 링크를 OTP 링크로 변경하거나, OTP 링크를 일반 링크로 변경하기 위해서는 링크 삭제 후 재생성이 필요합니다.

### 모바일 앱

1. 링크가 생성되어 있는 파일 및 폴더 옆의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '보내기'를 탭합니다.
3. '링크 관리 및 보내기'를 탭해 상세 정보를 확인합니다.
4. 접근 권한, 유효기간, 편집 권한을 변경합니다.
5. '확인'을 탭하여 링크 옵션을 변경합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 '폴더' 탭을 클릭합니다.
5. 파일 및 폴더의 오른쪽   ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
6. '링크 설정'을 클릭합니다.
7. 접근 권한, 유효기간, 편집 권한을 변경합니다.

# 링크 삭제

링크를 통해 공유하고 있던 파일과 폴더를 더 이상 공유하고 싶지 않을 경우, 해당 파일 및 폴더의 링크를 삭제하여 공유 상태를 해제할 수 있습니다.

### 모바일 앱

1. 링크가 생성되어 있는 파일 및 폴더 옆의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '보내기'를 탭합니다.
3. '링크 관리 및 보내기'를 탭합니다.
4. '링크 삭제'를 탭해서 공유를 해제합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 '폴더' 탭을 클릭합니다.
5. 파일 및 폴더의 오른쪽   ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
6. '링크 설정'을 클릭합니다.
7. '링크 삭제'를 클릭합니다.

# 링크 히스토리 확인

공유한 링크 내 최근 6개월 간의 활동 기록을 확인할 수 있습니다.

링크 유효기간 만료 전까지 기록된 히스토리를 확인할 수 있습니다.

링크가 삭제되면 링크 공유 히스토리도 삭제됩니다.

- 링크 접근
- 미리보기
- 이름 바꾸기
- 내려 받기
- 새 폴더 만들기
- 올리기
- 덮어쓰기
- 이동
- 복사
- 삭제

## 링크 히스토리 확인하기

### 모바일 앱

1. 공유한 폴더나 파일 옆의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '보내기'를 탭합니다.
3. '링크 관리 및 보내기'를 탭해서 상세 정보를 확인합니다
4. '링크 히스토리'를 탭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
