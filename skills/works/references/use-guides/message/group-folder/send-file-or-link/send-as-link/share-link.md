# 링크 공유

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 멤버 이외의 다른 구성원에게 메시지방 폴더 내 파일 및 폴더를 메일이나 메시지의 첨부파일로 보내는 대신 링크를 만들어 보낼 수 있습니다.

접근 권한과 유효 기간, 편집 권한을 설정하여 보안을 강화할 수 있습니다.

링크 공유는 공용 용량을 추가로 사용하지 않아 대용량 파일 공유에 효과적입니다.

링크로 공유된 폴더, 파일은 이름 오른쪽에 ![](/wp-content/uploads/2024/04/icon_share.png) 아이콘이 표시됩니다.

링크를 보내려면 해당 폴더의 [링크 공유 사용 설정이 허용](/ko/use-guides/message/group-folder/folder-settings/set-link-sharing-use/)되어 있어야 합니다.

[외부 그룹](/ko/use-guides/message/message-room/group-message-room/external-group-message-room/)의 폴더에선 링크를 보낼 수 없습니다.

# 링크 보내기

메시지방 폴더 내 파일이나 폴더를 링크로 공유할 수 있습니다.

폴더와 파일 링크 모두 특정 대상만 열람할 수 있도록 접근 권한을 설정하여 공유할 수 있습니다.

링크로 공유하는 대상이 폴더인지 파일인지에 따라 제공되는 권한 옵션이 달라집니다.

|  |  |  |
| --- | --- | --- |
|  | **폴더** | **파일** |
| 편집 | 올리기, 내려받기, 미리보기, 편집, 이동, 삭제 | 내려받기, 미리보기, 편집, 삭제 |
| 올리기 및 내려받기 | 올리기, 내려받기, 미리보기 | 미제공 |
| 올리기 | 올리기, 미리보기 | 미제공 |
| 내려받기 | 내려받기, 미리보기 | 내려받기, 미리보기 |
| 미리보기 | 미리보기 | 미리보기 |

### 모바일 앱

1. 공유하려는 파일 및 폴더 오른쪽의![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '보내기'를 탭합니다.
3. '링크 공유'를 탭합니다.
4. 화면 하단의 '링크 보내기'를 탭합니다. 필요 시 [링크 공유 설정을 변경](/ko/use-guides/message/group-folder/send-file-or-link/send-as-link/manage-link/)할 수도 있습니다.
5. 링크를 보낼 대상을 선택합니다. 다른 메시지방을 선택하려면 '더보기'를 탭합니다.  
   연락처를 선택하려면 '연락처'를 탭합니다. 메일, 노트를 선택해 파일 링크를 바로 첨부할 수도 있습니다.
6. '확인'을 탭하여 링크를 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 '폴더' 탭을 클릭합니다.
5. 공유하려는 파일 및 폴더의 오른쪽    ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
6. '링크 공유'를 클릭합니다. 이때 링크의 공유 범위를 설정할 수 있으며, [보안 OTP 인증을 해야만 접근할 수 있는 링크](https://help.worksmobile.com/ko/use-guides/drive/send-file-or-link/send-as-link/otp-link-authenticate/)를 만들 수도 있습니다.
7. '복사'를 클릭해 링크를 원하는 곳에 공유합니다. 권한이나 유효기간과 같은 [링크 공유 설정을 변경](/ko/use-guides/message/group-folder/send-file-or-link/send-as-link/manage-link/)할 수도 있습니다.

## 보안 OTP로 인증하는 링크 보내기

OTP 인증번호를 입력해야 접근할 수 있는 공유 링크를 생성합니다.

링크를 공유받은 사용자는 [메일로 OTP 인증을 받은 후](https://help.worksmobile.com/ko/use-guides/drive/send-file-or-link/send-as-link/otp-link-authenticate/) 링크에 접근할 수 있습니다.

회사 구성원이 아닌 외부인과 안전하게 공유하고자 할 경우 사용할 수 있습니다.

### 모바일 앱

1. 공유하려는 파일 및 폴더 오른쪽의![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '보내기'를 탭합니다.
3. '링크 공유'를 탭합니다.
4. '접근 권한'을 탭합니다.
5. '지정한 사람만(보안 OTP 인증)'을 탭하고 '예'를 탭합니다.
6. ![](/wp-content/uploads/2024/04/icon_drive_otp_add_member.png) 아이콘을 탭하고 공유할 구성원의 이메일 주소를 입력한 후 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_check.png) 아이콘을 탭합니다.  
   최초 인증한 브라우저에서만 접근하게 하려면 '보안 강화'를 탭한 후 '사용함'을 선택합니다.
7. '확인'을 탭해서 링크를 메일로 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 '폴더' 탭을 클릭합니다.
5. 공유하려는 파일 및 폴더의 오른쪽    ![](/wp-content/uploads/2024/05/ic16_more_767676.png)    아이콘을 클릭합니다.
6. '링크 공유'를 클릭합니다.
7. 링크 공유 범위에서 '지정한 사람만(보안 OTP 인증)'을 선택합니다.
8. '링크 생성'을 클릭합니다.
9. 메일 주소 입력란에 링크를 공유할 구성원의 이메일 주소를 입력한 후, '메일 발송'을 클릭합니다.
10. '확인'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
