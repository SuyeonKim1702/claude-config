# 메시지방에 폴더 또는 파일의 바로가기 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 폴더에 있는 파일과 폴더는 메시지방에 언제든 손쉽게 공유할 수 있습니다.

파일 또는 폴더에 바로 접근할 수 있도록 바로가기 버튼이 포함된 메시지를 보내, 공용용량 제한 걱정 없이 업무에 필요한 자료를 주고받을 수 있습니다.

'폴더 바로가기' 버튼을 누르면 해당 폴더 또는 파일이 저장된 폴더로 이동합니다.

조직장 혹은 그룹 마스터가 [멤버별 접근 멤버를 설정](/ko/use-guides/message/group-folder/folder-settings/set-authority/#i)한 폴더에서는 이 기능을 사용할 수 없습니다.

# 메시지방으로 폴더 또는 파일의 바로가기 보내기

### 모바일 앱

1. 위치를 보낼 파일 또는 폴더 오른쪽의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '보내기'를 탭합니다.
3. '이 메시지방으로 위치 보내기'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 바로가기로 보낼 폴더나 파일의 오른쪽    ![](/wp-content/uploads/2024/05/ic16_more_767676.png)    아이콘을 클릭합니다.
6. '메시지방에 바로가기 보내기'를 클릭합니다.

# 메시지 알림으로 폴더 진입하기

### 모바일 앱

- 메시지방으로 발송된 메시지 알림에서 '폴더 바로가기'를 탭하면 해당 폴더로 이동할 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방으로 발송된 메시지 알림에서 '폴더 바로가기'를 클릭하면 해당 폴더로 이동할 수 있습니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방으로 발송된 메시지 알림에서 '폴더 바로가기'를 클릭합니다. PC웹 화면으로 이동하여 오른쪽 화면에서 해당 폴더를 확인할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
