# 메시지로 파일 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 폴더에 있는 파일들을 메시지방으로 보내 손쉽게 공유할 수 있습니다.

# 메시지로 파일 보내기

### 모바일 앱

1. 보내고자 하는 파일 오른쪽의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '보내기'를 탭합니다.
3. '파일로 보내기'를 탭합니다.
4. 파일을 보낼 대상을 선택합니다. 다른 메시지방을 선택하려면 '더보기'를 탭합니다.  
   연락처를 선택하려면 '연락처'를 탭합니다.
5. '확인'을 탭하여 파일을 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 메시지로 보낼 파일의 오른쪽   ![](/wp-content/uploads/2024/05/ic16_more_767676.png)   아이콘을 클릭합니다.
6. '메시지로 보내기'를 클릭합니다.
7. 파일을 전달할 연락처나 메시지방을 선택하고 '전달'을 클릭합니다.

# 메시지로 여러 개의 파일 보내기

### 모바일 앱

1. 보내고자 하는 파일을 길게 누릅니다.
2. 보낼 파일들을 선택합니다.
3. 왼쪽 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_up_share_line.png) 아이콘을 탭합니다.
4. '파일로 보내기'를 탭합니다.
5. 파일을 보낼 대상을 선택합니다. 다른 메시지방을 선택하려면 '더보기'를 탭합니다.  
   연락처를 선택하려면 '연락처'를 탭합니다.
6. '확인'을 탭하여 파일을 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 메시지로 보낼 파일들을 선택한 후, 마우스 오른쪽 버튼을 클릭합니다.
6. '메시지로 보내기'를 클릭합니다.
7. 파일을 전달할 연락처나 메시지방을 선택하고 '전달'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
