# 노트로 파일 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 폴더에 있는 파일들을 첨부해 바로 노트를 작성할 수 있습니다.

파일을 노트로 보내면 해당 파일이 첨부된 노트 작성 화면이 열립니다. 게시판을 선택하고, 제목과 본문을 작성해 노트를 등록할 수도 있습니다.

관리자에 의해 파일 전송이 제한된 외부 그룹 폴더는 이 기능을 사용할 수 없습니다.

# 노트로 보내기

### 모바일 앱

1. 폴더에서 보낼 파일의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '보내기'를 탭합니다.
3. '파일로 보내기'를 탭합니다.
4. '노트'를 탭합니다.
5. 파일을 첨부할 메시지방 노트를 선택합니다.
6. '확인'을 탭합니다.

# 노트로 여러 개의 파일 보내기

### 모바일 앱

1. 보내고자 하는 파일을 길게 누릅니다.
2. 보낼 파일들을 선택합니다.
3. 왼쪽 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_up_share_line.png) 아이콘을 탭합니다.
4. '파일로 보내기'를 탭합니다.
5. '노트'를 탭합니다.
6. 파일을 첨부할 메시지방 노트를 선택합니다.
7. '확인'을 탭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
