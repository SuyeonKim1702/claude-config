- [메시지로 파일 보내기](https://help.worksmobile.com/ko/use-guides/message/group-folder/send-file-or-link/send-as-file/message/)
- [노트로 파일 보내기](https://help.worksmobile.com/ko/use-guides/message/group-folder/send-file-or-link/send-as-file/note/)
- [메일로 파일 보내기](https://help.worksmobile.com/ko/use-guides/message/group-folder/send-file-or-link/send-as-file/mail/)

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
