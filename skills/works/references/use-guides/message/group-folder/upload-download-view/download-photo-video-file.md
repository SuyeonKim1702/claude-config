# 사진 및 파일 보기, 다운로드

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방 폴더에 있는 파일들을 확인하고, 다운로드할 수 있습니다.

모바일 기기에서는 [관리자 설정](/ko/admin-guides/security/file-security/#i)에 따라 파일 다운로드가 제한될 수 있습니다.

# 사진, 동영상 보기 및 다운로드

첨부된 사진이나 동영상을 크게 볼 수 있고, 기기에 다운로드할 수 있습니다.

### 모바일 앱

### 사진, 동영상 확인 후 다운로드

1. 폴더에서 사진 또는 동영상을 탭합니다.
2. 오른쪽 하단의 ![](/wp-content/uploads/2024/05/common_24_save_line.png) 아이콘을 탭해서 기기에 사진 또는 동영상을 저장합니다.

### 목록에서 다운로드

1. 폴더의 파일 목록에서 다운로드하려는 사진 또는 동영상 오른쪽의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
2. '내려받기'를 탭합니다.

### 여러 개의 사진 또는 동영상 다운로드

1. 폴더의 파일 목록에서 다운로드하려는 사진 또는 동영상 한 개를 길게 누릅니다.
2. 다운로드할 사진 또는 동영상들을 차례대로 선택합니다.
3. ![](/wp-content/uploads/2024/04/icon_download_file.png) 아이콘을 탭합니다.

다운로드 불가능한 사진 또는 동영상이 함께 선택되었을 경우, ![](/wp-content/uploads/2024/04/icon_download_file.png) 아이콘이 표시되지 않습니다.

### PC 웹

### 사진, 동영상 확인 후 다운로드

1. 상단의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 확인하고 싶은 사진 또는 동영상의 미리보기 이미지 혹은 이름을 클릭합니다.
6. 오른쪽 상단의 'PC 저장' 혹은 '드라이브 저장'을 클릭합니다.

뷰어 지원이 되지 않는 파일 유형의 경우, 더블 클릭 시 파일이 바로 다운로드됩니다.

### 목록에서 다운로드

1. 상단의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 폴더의 파일 목록에서 다운로드하려는 사진 또는 동영상 오른쪽의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 클릭합니다.
6. '내려받기'를 클릭합니다.

# 파일 보기 및 다운로드

폴더에 저장된 엑셀, 파워포인트, 워드 등의 파일을 확인할 수 있고, 기기에 다운로드할 수 있습니다.

PC웹의 경우, 연결 프로그램 없이도 [네이버웍스 문서 뷰어](/ko/use-guides/general/docviewer/basic/)로 파일들을 열람할 수 있습니다.

단, 모바일에서는 기기의 기본 뷰어 혹은 연결 프로그램이 우선적으로 열리며, 관리자가 'Admin' > '보안' > '파일 보안'에서 '[모바일앱에서 파일 다운로드 제한](https://help.worksmobile.com//ko/admin-guides/security/file-security/#i)' 또는 ['워터마크'](https://help.worksmobile.com//ko/admin-guides/security/file-security/#i-3) 설정을 사용했을 때만 네이버웍스 문서 뷰어가 열립니다.

### 모바일 앱

### iOS 기기에서 파일 확인 및 다운로드

iOS 11 및 이후 버전에서만 기기에 파일 다운로드가 가능합니다. iOS 11 이전 버전에서는 기기의 특성상 파일을 기기에 저장할 수 없습니다.

단, 파일을 탭했을 때 [관리자의 설정](https://help.worksmobile.com//ko/admin-guides/security/file-security/)에 따라 모바일 [문서 뷰어](/ko/use-guides/general/docviewer/)가 열리며 다운로드가 제한될 수 있습니다.

1. 폴더에서 확인 및 다운로드할 파일을 탭합니다.
2. 오른쪽 상단의  ![](/wp-content/uploads/2024/04/icon_ios_action-1.png) 아이콘을 탭한 후 '파일에 저장'을 선택해서 저장합니다.

### Android 기기에서 파일 확인 및 다운로드

Android 기기에서는 첨부파일을 열 수 있는 연결 프로그램을 선택할 수 있습니다.

단, 파일을 탭했을 때 [관리자의 설정](https://help.worksmobile.com//ko/admin-guides/security/file-security/)에 따라 모바일 [문서 뷰어](/ko/use-guides/general/docviewer/)가 열리며 다운로드가 제한될 수 있습니다.

1. 폴더에서 확인 및 다운로드할 파일을 탭합니다.
2. 해당 파일을 열 때 사용할 연결 프로그램을 선택합니다.

### 여러 개의 파일 다운로드

1. 폴더의 파일 목록에서 다운로드하려는 파일 한 개를 길게 누릅니다.
2. 다운로드할 파일들을 차례대로 선택합니다.
3. ![](/wp-content/uploads/2024/04/icon_download_file.png) 아이콘을 탭합니다.

다운로드 불가능한 파일이 함께 선택되었을 경우 ![](/wp-content/uploads/2024/04/icon_download_file.png) 아이콘이 표시되지 않습니다.

### PC 웹

### 파일 확인 후 다운로드

1. 상단의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 확인하고 싶은 파일의 미리보기 이미지 혹은 이름을 클릭합니다.
6. 화면 오른쪽 상단의 '내려받기'를 클릭하여 파일을 저장합니다.

파일의 확장자에 따라 더블 클릭 시 바로 다운로드될 수도 있습니다.

### 목록에서 다운로드

1. 상단의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 화면 왼쪽 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '폴더' 탭을 클릭합니다.
5. 폴더의 파일 목록에서 다운로드하려는 파일 오른쪽의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 클릭합니다.
6. '내려받기'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
