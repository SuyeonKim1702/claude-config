# 사진 및 파일 업로드

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
기기에 있는 사진, 동영상, 파일을 업로드할 수 있습니다.

관리자가 [파일 전송을 제한한 외부 그룹 폴더](/ko/admin-guides/security/file-security/#i-5)의 경우나, 관리자가 [모바일 기기 파일 업로드를 제한한](/ko/admin-guides/security/file-security/#i-2) 경우 이 기능을 사용할 수 없습니다.

한번에 업로드할 수 있는 파일의 최대 갯수는 제한이 없고, 파일당 최대 크기는 아래와 같습니다.

- Lite: 10GB
- Standard: 50GB
- Standard Plus: 200GB

# 사진 및 동영상 업로드

기기에 저장된 사진이나 동영상을 업로드하거나, 모바일에서는 바로 사진이나 동영상을 촬영해서 업로드할 수 있습니다.

### 모바일 앱

1. 메시지방 폴더에서 ![](/wp-content/uploads/2024/05/common_24_pencilwrite_fill.png) 아이콘을 탭합니다.
2. 기기에 있는 사진이나 동영상을 업로드하려면 '사진/동영상 선택'을 탭하고, 바로 사진이나 동영상을 촬영해서 업로드하려면 '사진/동영상 촬영'을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 오른쪽 상단의 '폴더' 탭을 클릭합니다.
4. 왼쪽 상단의 '새로 만들기' 버튼을 클릭합니다.
5. 원하는 항목을 클릭합니다.
   - 새 폴더 만들기
   - 파일 올리기
   - 폴더 올리기

PC의 파일 또는 폴더를 끌어다 놓는 방법으로 업로드할 수도 있습니다.

# 파일 업로드

### 모바일 앱

1. 메시지방 폴더에서 ![](/wp-content/uploads/2024/05/common_24_pencilwrite_fill.png) 아이콘을 탭합니다.
2. 기기에 있는 파일을 업로드하려면 '파일 올리기'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 오른쪽 상단의 '폴더' 탭을 클릭합니다.
4. 왼쪽 상단의 '새로 만들기' 버튼을 클릭합니다.
5. 원하는 항목을 클릭합니다.
   - 새 폴더 만들기
   - 파일 올리기
   - 폴더 올리기

PC의 파일을 끌어다 놓는 방법으로 업로드할 수도 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
