# 메시지 소개

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
네이버웍스 메시지는 LINE과 유사한 화면을 제공하여 사용하기 편리할 뿐 아니라, 업무에 최적화된 기능을 제공하는 비즈니스 메신저입니다.

# 메시지방의 종류

메시지방은 사용 용도에 따라 [일반 메시지방](/ko/use-guides/message/message-room/general-message-room/general-message-room/), [그룹 메시지방](/ko/use-guides/message/message-room/group-message-room/group-message-room/), [조직 메시지방](/ko/use-guides/message/message-room/team-message-room/), [나와의 메시지](/ko/use-guides/message/message-room/my-message-room/)의 총 4가지로 나뉩니다.

# 메시지에서 무엇을 할 수 있나요?

**# 메시지 보내기**

메시지방에서는 텍스트는 물론 [사진과 동영상, 파일, 설문, 위치, 템플릿, 할 일](/ko/use-guides/message/send-message/)과 같은 다양한 콘텐츠도 메시지로 보낼 수 있습니다. 누가 내 메시지를 읽었는지 확인할 수 있는 [읽음 확인](/ko/use-guides/message/check-message/read-receipt/), 잘못 보낸 메시지를 전송 취소하는 [회수](/ko/use-guides/message/send-message/delete-recall-message/) 등 업무에 특화된 기능 또한 제공합니다.

**# 노트, 일정, 할 일, 폴더 활용**

여러 명이 대화하는 메시지방에서 [노트](/ko/use-guides/message/note/)와 [일정](/ko/use-guides/message/group-schedule/), [할 일](/ko/use-guides/message/group-task/), [폴더](/ko/use-guides/message/group-folder/)의 4가지 부가 기능을 사용할 수 있습니다. 멤버들과 긴밀하게 협업하는 상황에 유용합니다.

[조직 메시지방](/ko/use-guides/message/message-room/team-message-room/)은 조직도와 멤버가 연동되는 메시지방입니다.  조직의 구성원이 변경될 때 메시지방 멤버에도 자동 반영되어 편리합니다.

특정인이 관리하는 프로젝트는 [그룹 메시지방](/ko/use-guides/message/message-room/group-message-room/group-message-room/)으로 만들 수 있습니다. 그룹은 그룹 마스터가 멤버와 기능 설정을 관리하며, 주소록의 그룹 연락처 정보와 연동됩니다.

**# 음성 및 영상 통화**

텍스트로 표현하기 어려울 때는 [음성 메시지](/ko/use-guides/message/send-message/voice-message/)로 녹음해 보내거나, [화상 회의](/ko/use-guides/message/voice-video-call/)로 이야기할 수 있습니다. 재택근무 중에도 [화면 공유](/ko/use-guides/message/screen-share/)를 활용하여 회의를 원활히 진행할 수 있습니다.

**# LINE 또는 네이버웍스 연동**

사내 구성원뿐 아니라, 네이버웍스나 LINE을 사용하는 외부 사용자와도 메시지방에서 대화할 수 있습니다.

외부 사람과의 단체 협업은 '외부 사용자와 대화'로 시작할 수 있습니다.

[외부 네이버웍스 사용자](/ko/use-guides/message/message-with-external-user/message-with-external-line-works-user/)와 여러 명이 대화하는 메시지방은 그룹 마스터가 관리할 수 있는 외부 그룹으로 만들어집니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
