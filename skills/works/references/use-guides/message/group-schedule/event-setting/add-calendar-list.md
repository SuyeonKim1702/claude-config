# 캘린더에서 보기 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
자주 사용하는 메시지방 일정을 캘린더의 목록에 추가할 수 있습니다.

캘린더에서 보기 설정은 개인 설정으로, 다른 멤버의 목록은 변경되지 않습니다.

기본 설정은 '추가 안 함'입니다.

- 추가 안 함: 각 메시지방에서만 일정을 확인할 수 있습니다. 캘린더 목록에는 표시되지 않습니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '일정'을 탭합니다.
3. 일정에서  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png)  아이콘을 탭합니다.
4. '캘린더에서 보기'를 탭합니다.
5. 메시지방 일정을 캘린더 목록에 추가하거나 제외합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 '일정' 탭을 클릭합니다.
5. 일정에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic18_more_3f4247.png) 아이콘을 클릭합니다.
6. '캘린더에서 보기'를 클릭해서 메시지방 일정을 캘린더 목록에 추가하거나 제외합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
