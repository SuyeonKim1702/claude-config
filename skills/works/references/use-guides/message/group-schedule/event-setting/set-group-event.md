# 일정 사용 여부 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
조직장 및 [그룹 마스터](/ko/use-guides/message/message-room/group-message-room/manage-group-master-and-member/)는 조직/그룹 일정 사용 여부를 설정할 수 있습니다.

조직/그룹이 아닌 일반 메시지방 일정은 사용 여부를 변경할 수 없습니다.

# 조직 일정 사용 여부 설정

조직장은 일정 사용 여부를 설정할 수 있습니다.

### 모바일 앱

1. 조직 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png)  아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '조직 기능 관리'를 탭합니다.
4. '메시지방 기능'을 탭합니다.
5. 일정을 켜거나 끕니다.
6. '저장'을 탭해서 사용 여부를 설정합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다
3. 메시지방 목록에서 조직 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 ~~![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png)~~ 아이콘을 클릭합니다.
5. '조직 관리'를 클릭합니다.
6. '메시지방 기능'에서 '일정'의 '사용'을 클릭하여 사용을 켜거나 '사용 안 함'을 클릭하여 사용을 끕니다.
7. '저장'을 클릭하여 사용 여부를 설정합니다.

# 그룹 일정 사용 여부 설정

그룹 마스터는 일정 사용 여부를 설정할 수 있습니다.

### 모바일 앱

1. 그룹 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png)  아이콘을 탭합니다.
2. '그룹 관리'를 탭합니다.
3. '메시지방 기능'을 탭합니다.
4. 일정을 켜거나 끕니다.
5. '저장' 을 탭해서 사용 여부를 설정합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의  ~~![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png)~~ 아이콘을 클릭합니다.
5. '그룹 관리'를 클릭합니다.
6. '메시지방 기능'에서 '일정'의 '사용'을 클릭하여 사용을 켜거나 '사용 안 함'을 클릭하여 사용을 끕니다.
7. '저장'을 클릭하여 사용 여부를 설정합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
