# 일정 작성

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
멤버라면 누구나 일정을 등록할 수 있습니다.

일정 등록 시, 메시지방 알림 발송 여부를 선택할 수 있습니다.

### 모바일 앱

1. 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 버튼을 탭하고 '일정'을 탭합니다.
2. 일정에서 ![](/wp-content/uploads/2024/05/common_24_pencilwrite_fill.png) 아이콘을 탭합니다.
3. 일정을 작성합니다.
4. '저장'을 탭합니다. 메시지방에 알림을 보내려면 '예'를 선택합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 화면 오른쪽의 '일정' 탭을 클릭합니다.
5. '새 일정'을 클릭합니다.
6. 일정을 작성합니다.
7. '저장'을 클릭합니다. 메시지방에 알림을 보내려면 '메시지방에 알림 보내기'를 선택하여 '확인'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
