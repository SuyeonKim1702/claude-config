# 일정 삭제

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
내가 작성한 일정뿐만 아니라 다른 메시지방 멤버가 작성한 일정도 삭제할 수 있습니다.

일정 삭제 시 메시지방 알림 발송 여부를 선택할 수 있습니다.

### 모바일 앱

1. 삭제하려는 일정을 탭합니다.  
   일정 정보 하단에 일정 생성자 및 최근에 해당 일정을 업데이트한 사람과 업데이트 일시를 볼 수 있습니다.
2. 일정 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 탭합니다.
3. '삭제'를 탭해서 일정을 삭제합니다.

### PC 웹

1. 삭제하려는 일정을 하단의 일정 목록에서 클릭합니다.  
   일정 정보 하단에 일정 생성자 및 최근에 해당 일정을 업데이트한 사람과 업데이트 일시를 볼 수 있습니다.
2. 일정 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
3. '삭제'를 클릭합니다.
   - 해당 일정만 삭제할 경우, '이 일정만 삭제'를 클릭하고 '확인'을 클릭하여 일정을 삭제합니다.
   - 해당 일정 이후 반복되는 일정을 모두 삭제할 경우, '이 일정부터 이후 일정 모두 삭제'를 클릭하고 '확인'을 클릭하여 일정을 삭제합니다.
   - 해당 일정 이전부터 이후 반복되는 일정을 모두 삭제할 경우, '전체 반복일정 삭제'를 클릭하고 '확인'을 클릭하여 일정을 삭제합니다.
4. '확인' 을 클릭합니다.

삭제 후 복원할 수 없으니 삭제할 때에는 주의해야 합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
