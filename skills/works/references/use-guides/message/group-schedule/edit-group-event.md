# 일정 수정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
내가 작성한 일정 뿐 아니라 다른 메시지방 멤버가 작성한 일정도 수정할 수 있습니다.

일정 수정 시 메시지방 알림 발송 여부를 선택할 수 있습니다.

### 모바일 앱

1. 수정하려는 일정을 탭합니다.  
   일정 정보 하단에 일정 생성자 및 최근에 해당 일정을 업데이트한 사람과 업데이트 일시를 볼 수 있습니다.
2. 일정을 수정합니다.
3. '저장'을 탭해서 수정한 일정을 저장합니다.

### PC 웹

1. 수정하려는 일정을 하단의 일정 목록에서 클릭합니다.  
   일정 정보 하단에 일정 생성자 및 최근에 해당 일정을 업데이트한 사람과 업데이트 일시를 볼 수 있습니다.
2. 일정 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
3. '수정'을 클릭합니다.
   - 해당 일정만 수정할 경우, '이 일정만 수정'을 클릭합니다.
   - 해당 일정 이후 반복되는 일정을 모두 수정할 경우, '이 일정부터 이후 일정 모두 수정'을 클릭합니다.
   - 해당 일정 이전부터 이후 반복되는 일정을 모두 수정할 경우, '전체 반복일정 수정'을 클릭합니다.
4. '확인'을 클릭합니다.
5. 일정을 수정한 후 '저장'을 클릭하여 일정을 저장합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
