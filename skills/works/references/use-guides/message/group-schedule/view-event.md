# 일정 보기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
일정을 월 단위, 하루 단위 시간대, 하루 단위 목록으로 볼 수 있습니다.

# 월 단위 일정 보기

일정은 기본적으로 월간 달력 형태로 볼 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '일정'을 탭합니다.

### PC 웹

1. 메시지방에서 화면 오른쪽의 '일정'을 클릭합니다.
2. 화면 상단의 월간, 하단의 일간 목록을 확인합니다.

# 하루 단위 시간대로 보기

특정일의 시간대별 일정을 볼 수 있습니다.

### 모바일 앱

1. 월 단위 보기에서 시간대로 일정을 볼 날짜를 탭한 후 다시 탭합니다.

# 하루 단위 목록으로 보기

특정일의 일정을 목록으로 확인할 수 있습니다.

### 모바일 앱

1. 월 단위 보기에서 시간대로 일정을 볼 날짜를 탭한 후 다시 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2018/05/common_24_calendartype_line.png) 아이콘을 탭해서 해당 일자의 일정을 목록으로 봅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
