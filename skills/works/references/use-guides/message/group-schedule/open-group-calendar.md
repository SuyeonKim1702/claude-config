# 일정 열기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
여러 명이 참여하는 메시지방에서 일정을 열 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '일정'을 탭해서 일정을 엽니다.

노트에서 화면을 왼쪽으로 쓸어넘겨 일정을 열 수도 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다
3. 메시지방을 클릭합니다.
4. 메시지방 오른쪽 '일정' 탭을 클릭하여 일정을 엽니다.

### PC 앱

1. 메시지방을 클릭합니다.
2. 메시지방 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-25-gnb-dark-calendar-000000-45-1.png) 아이콘을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
