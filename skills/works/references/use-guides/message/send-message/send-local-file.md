# 기기에 있는 파일 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
모바일 기기에 있는 문서나 사진, 동영상을 선택하여 메시지로 보낼 수 있습니다.

### 모바일 앱

1. 메시지 입력창에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_add-1.png) 아이콘을 탭합니다.
2. '파일'을 탭합니다.
3. 기기에서 보내려고 하는 문서나 이미지 파일을 선택하고 '전송'을 탭해 파일을 보냅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
