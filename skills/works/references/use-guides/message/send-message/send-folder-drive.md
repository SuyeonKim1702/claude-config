# 폴더/드라이브에 있는 파일 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

폴더 또는 드라이브에 있는 문서나 이미지를 메시지로 보낼 수 있습니다.

### 모바일 앱

1. 메시지 입력창에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_add-1.png) 아이콘을 탭합니다.
2. '폴더' 또는 '드라이브'를 탭합니다.
3. 메시지방으로 보낼 문서나 이미지 파일을 선택하고 '확인'을 탭해 파일을 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-attach-767676@3x.png) 아이콘을 클릭합니다.
5. '폴더' 또는 '드라이브'를 클릭합니다.
6. 메시지방으로 보낼 문서나 이미지 파일을 선택하고 '확인'을 클릭하여 메시지로 보냅니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
3. ![](/wp-content/uploads/2024/04/icon-message-pc-web-clip.png) 아이콘을 클릭합니다.
4. '내 PC' 또는 '드라이브'를 클릭합니다.
5. 메시지방으로 보낼 문서나 이미지 파일을 선택하고 '확인'을 클릭하여 메시지로 보냅니다.

## 전송할 수 있는 파일의 최대 개수와 용량

한 개 파일당 전송 가능한 최대 용량은 [관리자의 설정](/ko/admin-guides/manage-service/message/general-settings/#i-2)에 따릅니다.

사진, 동영상 파일 등의 전송 시 비정상적인 사용(어뷰징) 방지를 위해 1분당 보낼 수 있는 개수 및 용량이 제한됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
