# 스티커 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
업무용 LINE 스티커를 이용하여 간결하지만 빠르고 효과적으로 대화할 수 있습니다.

다양한 스티커를 통해 생각이나 의견을 전달해보세요. 대화를 풍성하고 활기차게 만들어주는 원동력이 됩니다.

기본적으로 LINE의 기본 스티커와 업무용 LINE 스티커가 제공됩니다.

### 모바일 앱

1. 메시지 입력창에서 ![](/wp-content/uploads/2024/05/ic20_sticker_767676.png) 아이콘을 탭합니다.
2. 스티커를 선택합니다.  
   오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭하면 새로운 스티커를 다운로드하거나 사용하지 않는 스티커를 정리할 수 있습니다.
3. 보내려는 스티커를 탭합니다.
4. 스티커만 보내려면 스티커를 한번 더 탭합니다. 메시지와 같이 보내려면 메시지 입력창을 한번 탭해서 메시지 입력 후 ![](/wp-content/uploads/2024/04/icon_message_send.png) 아이콘을 탭해서 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지 입력창의 ![](/wp-content/uploads/2024/05/ic20_sticker_767676.png) 아이콘을 클릭합니다.
5. 보내려는 스티커를 클릭하여 메시지로 보냅니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
3. 메시지 입력창의 ![](/wp-content/uploads/2024/05/ic20_sticker_767676.png) 아이콘을 클릭합니다.
4. 보내려는 스티커를 클릭하여 메시지로 보냅니다.

LINE에서 개인적으로 구입한 스티커는 네이버웍스에서 사용할 수 없습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
