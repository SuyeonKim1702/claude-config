# 음성 메시지 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지를 입력하기 번거로울 때, 음성 메시지 기능을 사용하여 내용을 음성으로 녹음해서 바로 보낼 수 있습니다.

단, 모바일에서만 전송 가능합니다.

받은 음성 메시지는 모바일과 PC 어디에서나 즉시 재생할 수 있습니다.

## 음성 녹음해서 보내기

### 모바일 앱

1. 텍스트 입력창 옆에 있는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_voice-1.png) 아이콘을 탭합니다.
2. 아래의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_voice-1.png) 아이콘을 길게 누르면서 녹음을 시작합니다.  
   누르고 있는 동안 목소리가 녹음됩니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_voice-1.png) 아이콘에서 손가락을 떼면, 녹음된 메시지가 전송됩니다.  
   녹음 중 취소하고 싶으면, 누르고 있는 손가락을 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_voice-1.png) 버튼 바깥쪽으로 밀어 옮긴 후, 놓아서 취소합니다.

## 음성 메시지 재생하기

### 모바일 앱

1. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic22_play.png) 아이콘을 탭합니다.
2. 재생 도중 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-pauseFill.png) 아이콘을 탭하여 일시정지 할 수 있습니다. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic22_play.png) 아이콘을 탭하면 이어서 재생됩니다.

### PC 웹

1. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic22_play.png) 아이콘을 클릭합니다.
2. 재생 도중 ~~![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-pauseFill.png)~~ 아이콘을 탭하여 일시정지 할 수 있습니다.![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic22_play.png) 아이콘을 클릭하면 이어서 재생됩니다.

### PC 앱

1. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic22_play.png) 아이콘을 클릭합니다.
2. 재생 도중 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-pauseFill.png) 아이콘을 클릭하여 일시정지 할 수 있습니다.![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic22_play.png) 아이콘을 클릭하면 이어서 재생됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
