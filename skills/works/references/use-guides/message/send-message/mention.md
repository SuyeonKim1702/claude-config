# 멘션

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방에서 특정 사용자를 지정해 메시지를 보내는 기능입니다. 해당 멤버에게 알림이 가고, 메시지방에서는 이름이 하이라이트 되어 본인이 멘션된 메시지를 쉽게 확인할 수 있습니다.

# 멤버 멘션하기

### 모바일 앱

1. 메시지방의 입력창에서 '@이름'을 입력합니다. '@'을 입력하면 메시지방 내 멤버 목록이 나옵니다.  
   메시지방의 모든 사용자를 멘션하려면 @All을 선택합니다.
2. 멘션할 사람을 선택하고, 메시지를 입력하고 '전송'을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방의 입력창에서 '@이름'으로 입력하거나 '@'을 입력하면 메시지방내의 멤버 목록이 나옵니다.  
   메시지방의 모든 사용자를 멘션하려면 @All을 선택합니다.
5. 멘션할 사람을 선택하고, 메시지를 입력하고 키보드의 Enter를 누릅니다.

### PC 앱

1. 화면 왼쪽 상단의  ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방의 입력창에서 '@이름'으로 입력하거나 '@'을 입력하면 메시지방내의 멤버 목록이 나옵니다.  
   메시지방의 모든 사용자를 멘션하려면 @All을 선택합니다.
4. 멘션할 사람을 선택하고, 메시지를 입력하고 키보드의 Enter를 누릅니다.

# 멘션된 메시지 알림

내가 멘션된 메시지는 팝업 알림과 메시지방 목록에 “멘션되었습니다.”가 표시되고, 메시지방 안에서는 이름이 하이라이트되어 본인이 멘션된 메시지를 쉽게 확인할 수 있습니다.

내가 멘션되었으나 아직 읽지 않은 메시지가 있는 메시지방은 목록에서 강조 표시됩니다.

# 메시지방에서 내가 멘션된 메시지만 알림 받기

모든 메시지 알림이 아니라 특정 메시지방에서 내가 멘션된 메시지의 알림만 받을 수 있습니다.

### 모바일 앱

1. 원하는 메시지방에 들어가  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '알림 설정'을 탭합니다.
3. 알림받을 메시지 중 '멘션된 메시지'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '알림 설정'을 클릭합니다.
6. '내가 멘션된 메시지만 알림'을 클릭합니다.
7. '확인'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의  ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '알림 설정'을 클릭합니다.
5. '내가 멘션된 메시지만 알림'을 클릭합니다.
6. '확인'을 클릭합니다.

설정 이후에는 내가 멘션된 메시지가 있을 경우에만 알림이 옵니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
