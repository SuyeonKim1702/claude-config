# 템플릿 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방에서 업무 용도에 맞는 템플릿을 선택하여 내용을 작성하고 보낼 수 있습니다.

템플릿은 [새 템플릿 만들기](/ko/use-guides/general/template/create/new-template/)에서 만들 수 있습니다.

### 모바일 앱

1. 메시지 입력창에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_add-1.png) 아이콘을 탭합니다.
2. '템플릿'을 탭합니다.
3. 각 탭에서 작성할 템플릿을 탭하여 내용을 작성합니다.
4. '전송'을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
4. 메시지 입력창에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2019/06/ic20_messageinput_add_default.png) 아이콘을 클릭합니다.
5. '템플릿'을 클릭합니다.
6. 각 탭에서 작성할 템플릿을 클릭하여 내용을 작성합니다.
7. '전송'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
3. 메시지 입력창에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2019/06/ic20_messageinput_add_default.png) 아이콘을 클릭합니다.
4. '템플릿'을 클릭합니다.
5. 각 탭에서 작성할 템플릿을 클릭하여 내용을 작성합니다.
6. '전송'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
