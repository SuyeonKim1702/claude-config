# 답장

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방에서 특정 메시지를 선택하여 답장을 할 수 있습니다.

답장을 하면 어떤 메시지에 대한 답장인지 표시되고, 이 부분을 누르면 해당 메시지로 이동해 대화의 맥락을 쉽게 파악할 수 있습니다.

단, 다음과 같은 경우에는 답장을 사용할 수 없습니다.

- LINE 사용자가 참여한 단체 메시지방
- 통화 메시지 등 일부 메시지

### 모바일 앱

1. 답장하려는 메시지를 길게 누릅니다.
2. '답장'을 선택합니다.
3. 답장 내용을 입력한 후 전송합니다.

또는 답장하려는 메시지를 왼쪽으로 쓸어넘겨 답장할 수도 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 답장하려는 메시지에서 마우스 오른쪽 버튼을 클릭합니다.
5. '답장'을 선택합니다.
6. 답장 내용을 입력한 후 전송합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 답장하려는 메시지에서 마우스 오른쪽 버튼을 클릭합니다.
4. '답장'을 선택합니다.
5. 답장 내용을 입력한 후 전송합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
