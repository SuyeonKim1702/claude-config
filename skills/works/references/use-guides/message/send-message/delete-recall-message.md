# 메시지 회수

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

보낸 메시지를 직접 회수하거나, [조직/그룹 메시지방의 마스터](/ko/use-guides/message/message-room/group-message-room/manage-group-master-and-member/)가 다른 멤버의 메시지를 회수할 수 있습니다.

회수한 메시지는 메시지방의 모든 참여자에게 더 이상 노출되지 않고, 메시지가 회수되었다는 문구가 표시됩니다.

회수 가능 시간 내라면 읽음 확인 여부와 상관 없이 회수할 수 있습니다.

단, 다음 경우는 메시지를 회수할 수 없습니다.

- // start of content for ko en jp

### 모바일 앱

1. 메시지방에서 회수하려는 메시지를 길게 누릅니다.
2. 메시지 위에 나오는 메뉴 중 '회수'를 탭한 후 대화창에서 '확인'을 탭해 메시지를 회수합니다.  
   회수가 가능한 메시지에만 '회수'가 표시됩니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 회수하고자 하는 메시지방을 클릭합니다.
4. 회수하려는 메시지를 마우스 오른쪽 버튼으로 클릭합니다. 혹은 메시지 옆에 마우스를 옮겨두면 나타나는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. 메시지 위에 나오는 메뉴 중 '회수'를 클릭한 후 대화창에서 '확인'을 클릭하여 메시지를 회수합니다.  
   회수가 가능한 메시지만 '회수'가 표시됩니다.

### PC 앱

1. 화면 왼쪽 상단 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 회수할 메시지가 있는 메시지방을 클릭합니다.
3. 회수하려는 메시지를 마우스 오른쪽 버튼으로 클릭합니다. 혹은 메시지 옆에 마우스를 옮겨두면 나타나는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
4. 메시지 위에 나오는 메뉴 중 '회수'를 클릭한 후 대화창에서 '확인'을 클릭하여 메시지를 회수합니다.  
   회수가 가능한 메시지만 '회수'가 표시됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
