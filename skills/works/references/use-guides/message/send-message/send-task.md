# 할 일 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방에서 [할 일을 작성](/ko/use-guides/task/task-writing/new-task/)하여 보낼 수 있습니다.

메시지방 멤버들은 할 일의 상세 정보를 조회하거나 수정할 수 있습니다.

### 모바일 앱

1. 하단의 '메시지'를 탭합니다.
2. 메시지방 목록에서 메시지방을 탭합니다.
3. 메시지 입력창 왼쪽의  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_add_line.png) 아이콘을 탭합니다.
4. '할 일'을 탭합니다.
5. 새 할 일을 씁니다.
6. '저장'을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지 입력창에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2019/06/ic20_messageinput_add_default.png) 아이콘을 클릭합니다.
5. '할 일 쓰기'를 클릭합니다.
6. 새 할 일을 씁니다.
7. '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지 입력창에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2019/06/ic20_messageinput_add_default.png) 아이콘을 클릭합니다.
4. '할 일 쓰기'를 클릭합니다.
5. 새 할 일을 씁니다.
6. '저장'을 클릭합니다.

할 일 쓰기 화면에서 메시지방을 선택할 경우 메시지방 할 일로 등록되며, 해당 메시지방에 알림을 발송할지 여부를 선택할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
