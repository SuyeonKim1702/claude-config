# 메시지 전달

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지로 주고 받은 텍스트와 스티커, 사진과 동영상, 파일 등을 다른 메시지방으로 전달할 수 있습니다.

텍스트를 직접 복사하거나 화면을 캡쳐하지 않고도 전달 기능을 통해 빠르게 공유할 수 있습니다.

'묶어서 전달' 설정을 선택하면 메시지를 보낸 사람, 발송 시간, 메시지방 이름을 포함해 주고 받은 메시지를 하나로 묶어 전달합니다.

'묶어서 전달' 설정을 해제한 경우, 선택한 메시지가 개별적으로 하나씩 발송됩니다. 메시지를 전송한 사용자 정보 등은 포함되지 않으며, 한 번에 최대 5개의 메시지만 선택할 수 있습니다.

묶어서 전달 기능은 LINE 또는 외부 네이버웍스 사용자와의 메시지에서는 사용할 수 없습니다.  
전달할 대상은 한 번에 5개까지 선택할 수 있습니다.

### 모바일 앱

1. 메시지방에서 전달하려고 하는 메시지를 길게 누릅니다.
2. '전달'을 탭합니다. '전달'은 전달이 가능한 메시지에만 노출됩니다.
3. '묶어서 전달' 설정 여부를 선택합니다.
4. 전달할 메시지를 선택하고, '확인'을 탭합니다.
5. 메시지를 전달할 연락처나 메시지방을 선택하고 '확인'을 탭해 메시지를 전달합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방에서 전달하려고 하는 메시지를 마우스 오른쪽 버튼으로 클릭합니다.  
   혹은 메시지 옆에 마우스를 옮겨두면 나타나는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
5. '묶어서 전달' 설정 여부를 선택합니다.
6. 전달할 메시지를 선택하고, '전달'을 클릭합니다.
7. 메시지를 전달할 연락처나 메시지방을 선택하고 '전달'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방에서 전달하려고 하는 메시지를 마우스 오른쪽 버튼으로 클릭합니다.  
   혹은 메시지 옆에 마우스를 옮겨두면 나타나는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
4. 여러 개의 메시지를 마우스로 끌어서 마우스 오른쪽 버튼을 클릭할 수도 있습니다.
5. '묶어서 전달' 설정 여부를 선택합니다.
6. 메시지를 전달할 연락처나 메시지방을 선택하고 '전달하기'를 클릭합니다.

## 사진과 동영상 빠르게 전달하기

### 모바일 앱

1. 메시지방에서 전달하고 싶은 사진이나 동영상의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-icoShareBtn.png) 아이콘을 탭합니다.
2. 연락처 혹은 메시지방을 선택하여 전달합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
