# 연락처 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
네이버웍스 주소록에 저장되어 있는 구성원의 연락처나 모바일 기기에 있는 연락처를 선택하여 메시지로 보낼 수 있습니다.

단, [LINE 사용자](/ko/use-guides/message/message-with-external-user/message-line-user/)나 [다른 회사의 네이버웍스 사용자](/ko/use-guides/message/message-with-external-user/message-with-external-line-works-user/)가 있는 외부 메시지방에는 연락처를 보낼 수 없습니다.

### 모바일 앱

1. 메시지 입력창 왼쪽의 ![](/wp-content/uploads/2024/04/icon-n-act_com_add-1.png) 아이콘을 탭합니다.
2. '연락처'를 탭합니다.
3. 연락처를 보냅니다.
   - NAVER WORKS 주소록에서 선택: 네이버웍스 사내 주소록에서 보내려는 연락처를 선택한 후 오른쪽 상단의 '확인'을 탭합니다.
   - 기기 연락처에서 선택: 모바일 기기의 주소록에서 보내려는 연락처를 선택해 탭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
