# 현재 위치 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
나의 현재 위치를 메시지로 보낼 수 있습니다. 메시지를 받은 사람은 보낸 사람의 위치를 지도 상에서 확인할 수 있습니다.

### 모바일 앱

1. 현재 위치를 보낼 메시지방에서 ![](/wp-content/uploads/2024/04/icon-n-act_com_add-1.png) 아이콘을 탭합니다.
2. '위치 공유'를 탭합니다.
3. 앱 설치 후 위치 공유 기능을 처음 사용하는 경우, 위치 동의를 물어보면 승인합니다. iOS와 안드로이드 기기에서 차이가 있습니다.
4. 지도 첨부에서 '위치 공유'를 탭하여 위치를 보냅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
