# 나와의 메시지

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
나와의 메시지는 나 혼자서만 사용할 수 있는 메시지방입니다.

나만의 메모를 기록하거나, 메시지로 오가는 내용 중 필요한 것들을 모아 보관하거나, 스마트폰과 데스크탑 간의 자료 이동 시에도 이용할 수 있습니다.

## 나와의 메시지방 만들기

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 내 이름이나 프로필 사진을 탭합니다.
3. '나와의 메시지'를 탭해서 나와의 메시지방을 만듭니다.  
   또는, '대화할 사람'에서 본인 연락처를 선택하여 '확인'을 탭하면 나와의 메시지방을 만들 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 왼쪽 메시지방 목록 상단의 '새로 만들기'를 클릭합니다.
4. '사내 구성원과 대화'를 클릭하고 나의 프로필을 선택합니다.
5. '확인'을 클릭해 나와의 메시지방을 만듭니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/05/ic22_contact.png) 아이콘을 클릭합니다.
2. 멤버 목록 중 나의 프로필 사진을 클릭합니다.
3. '나와의 메시지'를 클릭하여 나와의 메시지방을 만듭니다.

이미 메시지방을 만든 적이 있는 경우에는 기존의 메시지방이 열립니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
