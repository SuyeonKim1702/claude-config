# 일반 메시지방

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
일반 메시지방은 누구나 초대하여 대화할 수 있는 메시지방입니다.

1:1 또는 단체 대화가 가능하며, 자유롭게 초대와 탈퇴를 할 수 있습니다.

여러 명이 대화하는 메시지방에서는 노트, 일정, 할 일, 폴더를 사용하여 협업할 수 있습니다.

메일링리스트와 그룹 공개 여부 등의 설정은 [조직 메시지방](/ko/use-guides/message/message-room/team-message-room/) 또는 [그룹 메시지방](/ko/use-guides/message/message-room/group-message-room/group-message-room/)에서만 사용할 수 있습니다.

일반 메시지방이 아닌 조직 및 그룹 메시지방은 메시지방 목록에서 보이는 메시지방의 대표 사진 오른쪽 하단에 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_groups.png) 아이콘이 표시됩니다.

# 일반 메시지방 만들기

연락처 검색 결과나 조직도에서 1:1 또는 단체 메시지방을 만들 수 있습니다.

여러 명의 멤버를 선택하고 메시지방을 만들면 같은 멤버가 이미 참여 중인 메시지방들이 추천됩니다.

기존 메시지방을 선택하면 비슷한 메시지방을 또 만들지 않고도 대화를 이어 나갈 수 있습니다.

### 모바일 앱

1. 메시지에서 ![](/wp-content/uploads/2024/02/icon_plus.png) 아이콘을 탭합니다.
2. '사내 구성원과 대화'와 '외부 사용자와 대화' 중 선택합니다.
3. 대화할 사람의 이름을 검색하거나 조직도에서 이름을 선택합니다.
4. '확인'을 탭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방을 선택하거나 '새 메시지방 만들기'를 탭합니다.
5. 여러 명이 참여하는 새 메시지방에는 이름, 사진, 설명을 입력할 수 있습니다. 기본 정보를 확인한 후 '만들기'를 탭해서 대화를 시작합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록 상단 '새로 만들기'를 클릭합니다.
4. '사내 구성원과 대화'와 '외부 사용자와 대화' 중 선택합니다.
5. 대화할 사람의 이름을 검색하거나 조직도에서 이름을 클릭합니다.
6. '확인'을 클릭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방 오른쪽의 '열기'를 클릭하거나 '새 메시지방 만들기'를 클릭합니다.
7. 여러 명이 참여하는 새 메시지방에는 이름, 사진, 설명을 입력할 수 있습니다. 기본 정보를 확인한 후 '만들기'를 클릭해서 대화를 시작합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록 하단 ![](/wp-content/uploads/2024/02/icon_plus.png) 아이콘을 클릭합니다.
3. '사내 구성원과 대화'와 '외부 사용자와 대화' 중 선택합니다.
4. 대화할 사람의 이름을 검색하거나 조직도에서 이름을 선택합니다.
5. '확인'을 클릭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방 오른쪽의 '열기'를 클릭하거나 '새 메시지방 만들기'를 클릭합니다.
6. 여러 명이 참여하는 새 메시지방에는 이름, 사진, 설명을 입력할 수 있습니다. 기본 정보를 확인한 후 '만들기'를 클릭해서 대화를 시작합니다.

## 대화 상대가 있는 경우 메시지방 만들기

대화 상대가 이미 있는 경우 바로 메시지방을 만들 수도 있습니다.

- 프로필에서 1:1 메시지방 만들기
- 연락처 상세에서 1:1 메시지방 만들기
- 일정 참석자 대상으로 메시지방 만들기
- 메일 수신자 대상으로 메시지방 만들기
- 기존 메시지방에서 새 메시지방 만들기

### 프로필에서 1:1 메시지방 만들기

주소록, 메시지, 메일, 일정, 게시판, 드라이브 등 서비스 곳곳에서 이름 또는 프로필 사진을 누르면 나오는 프로필에서 1:1 메시지방을 만들 수 있습니다.

### 모바일 앱

1. 사람 이름 또는 프로필 사진을 탭합니다.
2. 프로필에서 '메시지'를 탭하여 1:1 메시지방을 만듭니다.

### PC 웹

1. 이름 또는 프로필 사진을 클릭합니다.
2. 프로필에서 '메시지'를 클릭해서 1:1 메시지방을 만듭니다.

### PC 앱

1. 이름 또는 프로필 사진을 클릭합니다.
2. 프로필에서 ![](/wp-content/uploads/2024/05/ic20_task_message_157efb.png) 아이콘을 클릭해서 1:1 메시지방을 만듭니다.

### 연락처 상세에서 1:1 메시지방 만들기

주소록에서 연락처를 찾아서 1:1 메시지방을 만들 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. '주소록'을 탭합니다.
3. 조직도나 연락처를 검색해서 프로필을 탭합니다.
4. '메시지'를 탭하여 메시지방을 만듭니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '주소록'을 클릭합니다.
3. 조직도나 연락처를 검색해서 이름을 클릭합니다.
4. 프로필에서 '메시지'를 클릭해서 메시지방을 만듭니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/05/ic22_contact.png) 아이콘을 클릭합니다.
2. 조직도나 연락처를 검색해서 프로필 사진을 클릭합니다.
3. 메시지 아이콘을 클릭해서 메시지방을 만듭니다.

### 멤버 목록에서 메시지방 만들기

멤버 목록에서 바로 메시지방을 만들 수 있습니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/05/ic22_contact.png) 아이콘을 클릭합니다.
2. 목록 상단 검색창에서 메시지를 보낼 멤버를 검색하여 멤버를 더블 클릭하거나 마우스 오른쪽 버튼으로 클릭하여 '메시지 보내기'를 클릭합니다.

### 일정 참석자 대상으로 메시지방 만들기

회의 참석자 대상으로 캘린더에서 바로 메시지방을 만들수 있습니다.

해당 일정의 제목이 메시지방 제목이 됩니다.

### 모바일 앱

1. 캘린더에서 대화할 상대가 참석하는 일정을 엽니다.
2. 일정에서 '전체 메시지'를 탭합니다.
3. 메시지방에 초대할 멤버를 선택하고 '확인'을 탭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방을 선택하거나 '새 메시지방 만들기'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '캘린더'를 클릭합니다.
3. 대화할 상대가 참석하는 일정을 클릭합니다.
4. '참석' 항목의 참석자 오른쪽의 '메시지'를 클릭합니다.
5. 메시지방에 초대할 멤버를 선택하고 '확인'을 클릭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방 오른쪽의 '열기'를 클릭하거나 '새 메시지방 만들기'를 클릭합니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](/wp-content/uploads/2024/05/common_24_calendar_fill.png) 아이콘을 클릭합니다.
2. PC웹 캘린더 화면으로 이동합니다.
3. 대화할 상대가 참석하는 일정을 클릭합니다.
4. '참석' 항목의 참석자 오른쪽의 '메시지'를 클릭합니다.
5. 메시지방에 초대할 멤버를 선택하고 '확인'을 클릭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방 오른쪽의 '열기'를 클릭하거나 '새 메시지방 만들기'를 클릭합니다.

### 메일을 보낸사람과 받는사람을 포함해 메시지방 만들기

메일을 보내거나 받은 사람과 메시지방을 만들 수 있습니다.

해당 메일의 제목이 메시지방 제목이 됩니다.

### 모바일 앱

1. 대화할 상대가 발신인 또는 수신인인 메일을 탭합니다.
2. '전체 메시지'를 탭합니다.
3. 메시지방에 초대할 멤버를 선택하고 '확인'을 탭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방을 선택하거나 '새 메시지방 만들기'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메일'을 클릭합니다.
3. 대화할 상대가 발신인 또는 수신인인 메일을 클릭합니다.
4. 보낸 사람 메일 주소 오른쪽 '메시지'를 클릭합니다.
5. 메시지방에 초대할 멤버를 선택하고 '확인'을 클릭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방 오른쪽의 '열기'를 클릭하거나 '새 메시지방 만들기'를 클릭합니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](/wp-content/uploads/2024/05/bottomtab_24_mail_fill.png) 아이콘을 클릭합니다.
2. PC웹 메일 화면으로 이동합니다.
3. 대화할 상대가 발신인 또는 수신인인 메일을 클릭합니다.
4. 보낸 사람 메일 주소 오른쪽 '메시지'를 클릭합니다.
5. 메시지방에 초대할 멤버를 선택하고 '확인'을 클릭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방 오른쪽의 '열기'를 클릭하거나 '새 메시지방 만들기'를 클릭합니다.

### 기존 메시지방에서 새 메시지방 만들기

이미 만들어진 메시지방에서 일부 멤버만 선택하거나 추가하여 새 메시지방을 만들 수 있습니다.

나와의 메시지, 1:1 메시지방, 외부 대화상대가 포함된 메시지방에서는 지원하지 않는 기능입니다.

### 모바일 앱

1. 하단의 '메시지'를 탭합니다.
2. 메시지방 목록에서 원하는 멤버들이 모여 있는 메시지방을 탭합니다.
3. 메시지방 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
4. '새 메시지방 만들기'를 탭합니다.
5. 원하는 멤버를 선택합니다.  
   메시지방에 없는 멤버를 추가하려면 '연락처 검색' 창에 멤버의 정보를 입력합니다.
6. 선택한 멤버를 확인하고 '확인'을 탭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방을 선택하거나 '새 메시지방 만들기'를 탭합니다.

2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 원하는 멤버들이 모여 있는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '새 메시지방 만들기'를 클릭합니다.
6. 원하는 멤버를 선택합니다.

7. 선택한 멤버를 확인하고 '확인'을 클릭합니다.  
   선택한 멤버들이 이미 참여 중인 메시지방이 존재한다면, '참여 중인 메시지방' 목록에서 기존 메시지방 오른쪽의 '열기'를 클릭하거나 '새 메시지방 만들기'를 클릭합니다.

# 메시지방에 구성원 초대하기

기존의 메시지방에 또 다른 구성원을 초대할 수 있습니다.

### 모바일 앱

1. 메시지방의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '초대'를 탭합니다.
3. 초대할 구성원을 선택합니다.
4. '확인'을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '초대하기' 메뉴를 클릭합니다.
6. 초대할 구성원을 선택합니다.
7. '초대하기'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '초대하기' 메뉴를 클릭합니다.
5. 초대할 구성원을 선택합니다.
6. '초대하기'를 클릭합니다.

# 메시지방에서 멤버 내보내기

메시지방에 잘못 초대된 멤버나, 대화에 더는 참여할 필요가 없는 멤버를 내보낼 수 있습니다.

내보내진 멤버는 메시지방에서 제외되었다는 서비스 알림을 받습니다.

### 모바일 앱

1. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '대화 중 멤버'를 탭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 탭하고, '내보내기'를 탭합니다.
4. 내보낼 멤버를 선택한 후, 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-message-room-remeove-member.png) 아이콘을 탭하고 '내보내기'를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 상단 프로필 사진을 클릭합니다.
5. 마우스 포인터를 내보낼 멤버 위에 올린 후, 오른쪽의 'x' 아이콘을 클릭합니다.
6. '확인'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 상단 방 이름 또는 프로필 사진을 클릭합니다.
4. 마우스 포인터를 내보낼 멤버 위에 올린 후, 오른쪽의 'x' 아이콘을 클릭합니다.
5. '내보내기'를 클릭합니다.

# 일반 메시지방 나가기

1:1 메시지방이나 단체 메시지방에서 퇴장할 수 있습니다.

메시지방에서 나가면 이전의 대화내용을 볼 수 없습니다. 내가 만든 메시지방에서 나간 이후에도 다른 멤버는 계속 그 메시지방을 사용할 수 있습니다.

## 일반 메시지방에서 나가기

일반 메시지방에서는 직접 메시지방에서 나갈 수 있습니다.

### 모바일 앱

1. 나가려는 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '나가기'를 탭하고 '확인'을 탭해서 메시지방에서 나갑니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 나가려는 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '나가기'를 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 나가려는 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '나가기'를 클릭합니다.

## 일반 메시지방 목록에서 나가기

메시지방 목록에서 바로 메시지방을 나갈 수 있습니다.

### 모바일 앱

1. 메시지방 목록에서 나가려는 메시지방을 왼쪽으로 쓸어 넘깁니다.
2. '나가기'를 탭합니다.
3. 대화창에서 '확인'을 탭합니다.

메시지방 하나를 길게 누른 후 여러 개의 메시지방을 선택하여 한꺼번에 나갈 수도 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 나가고 싶은 메시지방 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
4. '나가기'를 클릭합니다.
5. 나가고 싶은 다른 메시지방이 있다면 함께 선택한 후 '나가기'를 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 나가고 싶은 메시지방을 마우스 오른쪽 버튼으로 클릭합니다.
3. '나가기'를 클릭합니다.
4. 나가고 싶은 다른 메시지방이 있다면 함께 선택한 후 '나가기'를 클릭합니다.
