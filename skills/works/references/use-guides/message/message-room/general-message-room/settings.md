# 일반 메시지방 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

# 메시지방 이름 바꾸기

단체 메시지방에서는 누구나 메시지방 이름을 변경할 수 있습니다.

변경된 이름은 참여자 모두에게 보여집니다.

단, 1:1 메시지방의 이름은 변경할 수 없습니다.

### 모바일 앱

1. 메시지방에 들어가서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. ![](/wp-content/uploads/2024/05/ic12_pen_000000_60.png) 아이콘을 탭합니다.
4. 이름을 입력합니다.
5. '확인'을 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '메시지방 설정'을 클릭합니다.
6. ![](/wp-content/uploads/2024/05/ic12_pen_000000_60.png) 아이콘을 클릭합니다.
7. 이름을 입력합니다.
8. '확인'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '메시지방 설정'을 클릭합니다.
5. 이름을 입력합니다.
6. ![](/wp-content/uploads/2024/04/1562812221696058.png) 아이콘을 클릭합니다.

# 메시지방 대표 사진 설정

단체 메시지방에서는 누구나 대표 사진을 설정할 수 있습니다.

### 모바일 앱

1. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. 사진 아이콘을 탭합니다.
4. 카메라로 찍거나 사진을 선택해 대표 사진을 설정합니다.

### PC 웹

1. 메시지방 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
2. '메시지방 설정'을 클릭합니다.
3. 사진 아이콘을 클릭합니다.
4. 대표 사진을 설정합니다. 이미 대표 사진이 설정되어 있다면, '사진 변경'을 클릭합니다.

### PC 앱

1. 메시지방 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
2. '메시지방 설정'을 클릭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/setting_pcapp_camera_icon.png) 아이콘을 클릭해 대표 사진을 설정합니다.

# 이전 대화 공개 설정

메시지방에 새로 초대한 멤버에게 이전 대화 내용을 공개할 수 있습니다.

### 모바일 앱

1. 메시지방에 들어가서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '이전 대화 공개' 설정을 켭니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '이전 대화 공개'를 클릭합니다.
6. '이전 대화 공개하기'를 선택합니다.
7. '확인'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '이전 대화 공개'를 클릭합니다.
5. '이전 대화 공개하기'를 선택합니다.
6. '확인'을 클릭합니다.

단, 이전 대화 공개 설정 후 새로 초대한 멤버에게만 적용되며, 메시지방의 기존 멤버들에게는 적용되지 않습니다.

# 메시지방 새 창으로 열기

PC앱에서 메시지방을 새 창으로 열 수 있습니다.

필요 시 화면의 항상 위로 고정시킬 수도 있습니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 새 창으로 열 메시지방에서 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
3. '새 창으로 열기'를 클릭합니다.  
   새 창으로 연 메시지방을 가장 위로 고정시키려면 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭한 후 '항상 위'를 클릭합니다.  
   고정을 해제하려면 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭한 후 '항상 위 해제'를 클릭합니다.

# 메시지방 배경 색상 설정

메시지방을 구분하기 위해서 메시지방별로 배경 색상을 설정할 수 있습니다.

배경 색상은 본인에게만 적용되며, 다른 멤버들에게는 적용되지 않습니다.

### 모바일 앱

1. 메시지방에 들어가서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '배경 색상'을 탭합니다.
4. 설정할 배경 색상을 선택해서 메시지방의 배경색을 설정합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '배경 바꾸기'를 클릭합니다.
6. 나만 볼 수 있는 메시지방 배경색을 지정합니다.
7. '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '배경 바꾸기'를 클릭합니다.
5. 나만 볼 수 있는 메시지방 배경색을 지정합니다.
6. '확인'을 클릭합니다.

메시지방을 구분하기 위해서 메시지방별로 알림음을 설정할 수 있습니다.

알림음은 본인 기기에만 적용되며, 다른 멤버에게는 적용되지 않습니다.

### 모바일 앱

1. 메시지방에 들어가서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '알림음'을 탭합니다.
4. 설정할 알림음을 선택합니다.

# 소속 회사/단체

메시지방의 소속 회사/단체를 확인할 수 있습니다.

메시지방을 만든 멤버가 소속된 회사/단체가 표시됩니다.

여러 명이 참여하는 일반 메시지방, 그룹 메시지방은 소속 회사/단체의 공용용량을 사용하며, 노트, 일정, 할 일, 폴더의 파일 용량과 메시지로 주고받은 파일이 공용용량으로 계산됩니다.

메시지방을 만든 후 멤버가 퇴장해도 메시지방의 소속 회사/단체 정보는 변경되지 않습니다.

5. '메시지방 설정'을 클릭한 후 '소속 회사/단체'를 확인합니다.
