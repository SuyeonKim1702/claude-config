# 그룹 마스터 및 멤버 관리

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
최초로 그룹을 만든 사람은 그룹 마스터가 됩니다.

그룹 마스터만 그룹 마스터 또는 그룹 멤버를 추가하거나 삭제할 수 있습니다.

# 그룹 마스터 추가 및 삭제

그룹 마스터는 다른 그룹 마스터를 추가하거나 삭제할 수 있습니다.

그룹 마스터는 메시지, 노트, 일정, 할 일, 폴더, 메일링리스트의 사용 여부를 선택할 수 있고, 멤버를 추가하거나 삭제할 수 있습니다.

단, [외부 연동 권한](/ko/admin-guides/manage-service/message/message-with-external-users/)을 보유한 멤버만 외부 그룹 마스터로 지정할 수 있습니다.

## 그룹 마스터 추가

### 모바일 앱

1. 그룹 메시지 방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 버튼을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_members_setting-1.png) 아이콘을 탭합니다.
3. '그룹 멤버'를 탭합니다.
4. 멤버 중 그룹 마스터로 추가하려는 사람의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
5. '그룹 마스터로 추가'를 탭합니다.
6. '그룹 수정'으로 돌아가 '저장'을 탭합니다.

그룹 멤버에 없는 사람을 그룹 마스터로 추가하려면,  
'그룹 마스터' 탭에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_member_add-1.png) 아이콘을 탭하여 원하는 사람을 그룹 마스터로 추가할 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 클릭합니다.
5. '그룹 관리'를 클릭합니다.
6. '그룹 마스터'의 화면 오른쪽 '주소록'을 클릭합니다.
7. 멤버 중 그룹 마스터로 추가하려는 멤버를 선택합니다.
8. '확인'을 클릭합니다.
9. '저장'을 클릭하여 그룹 마스터로 추가합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '그룹 관리'를 클릭합니다.
5. '그룹 마스터'의 화면 오른쪽 '주소록'을 클릭합니다.
6. 멤버 중 그룹 마스터로 추가하려는 멤버를 선택합니다.
7. '확인'을 클릭합니다.
8. '저장'을 클릭하여 그룹 마스터로 추가합니다.

## 그룹 마스터 삭제

### 모바일 앱

1. 그룹 메시지 방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 버튼을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_members_setting-1.png) 아이콘을 탭합니다.
3. '그룹 멤버'를 탭합니다.
4. 삭제하려는 그룹 마스터의 ![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭합니다.
5. '그룹 마스터에서 삭제' 를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 클릭합니다.
5. '그룹 관리'를 클릭합니다.
6. '그룹 마스터'의 삭제할 멤버의 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_close.png) 아이콘을 클릭합니다.
7. '저장'을 클릭하여 그룹 마스터를 삭제합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '그룹 관리'를 클릭합니다.
5. '그룹 마스터'의 삭제할 멤버의 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_close.png) 아이콘을 클릭합니다.
6. '저장'을 클릭하여 그룹 마스터를 삭제합니다.

본인이 그룹 마스터인 경우에 본인을 그룹 마스터에서 삭제할 수 있습니다.

단, 그룹 마스터가 1명인 경우에는 그룹 마스터를 삭제할 수 없습니다.

그룹은 최소 1명의 그룹 마스터가 필요합니다.

# 멤버 추가 및 삭제

그룹 마스터는 멤버를 추가하거나 삭제할 수 있습니다.

멤버를 삭제하면 해당 멤버는 더 이상 해당 그룹 메시지방을 사용할 수 없습니다.

단, 한 멤버당 참여할 수 있는 그룹 수는 최대 2000개며, 소속 그룹 수가 2000개 이상인 멤버는 그룹 멤버로 추가할 수 없습니다.

외부 그룹에는 외부 연동 권한을 보유한 멤버만 추가할 수 있습니다.

### 모바일 앱

1. 그룹 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 버튼을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_members_setting-1.png) 아이콘을 탭합니다.
3. '그룹 멤버'를 탭합니다.
4. 멤버를 추가하려면, '그룹 멤버'나 '그룹 마스터' 탭에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_member_add-1.png) 아이콘을 탭하여 원하는 사람을 추가합니다.  
   멤버를 삭제하려면, 삭제하려는 사람의![](/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 탭하여 '그룹 멤버에서 삭제'를 탭합니다.
5. '그룹 관리'로 돌아가 '저장' 을 탭합니다.

그룹 메시지방의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 메뉴에서 '초대'를 탭하여 '사내 주소록에서 추가하기'를 탭하면, 더 간편하게 멤버를 추가할 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_more-1.png) 아이콘을 클릭합니다.
5. '그룹 관리'를 클릭합니다.
6. 멤버를 추가하려면 화면 오른쪽의 '주소록'을 클릭하고 멤버 중 그룹 멤버로 추가하려는 멤버를 선택합니다.  
   멤버를 삭제하려면 삭제하려는 멤버 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/02/lw-icon-close.svg) 아이콘을 클릭합니다.
7. '확인'을 클릭합니다.
8. '저장'을 클릭하여 그룹 멤버를 추가합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '그룹 관리'를 클릭합니다.
5. 멤버를 추가하려면 화면 오른쪽의 '주소록'을 클릭하고 멤버 중 그룹 멤버로 추가하려는 멤버를 선택합니다.  
   멤버를 삭제하려면 삭제하려는 멤버 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/02/lw-icon-close.svg) 아이콘을 클릭합니다.
6. '확인'을 클릭합니다.
7. '저장'을 클릭하여 그룹 멤버를 추가합니다.

멤버가 삭제되어도 해당 멤버의 메시지나 일정, 글, 파일들은 남아 있습니다.

# 외부 그룹으로 초대하기

외부 그룹 마스터인 경우, 다른 회사의 구성원을 초대할 수 있습니다.

초대를 받은 다른 회사의 구성원이 네이버웍스 사용자라면 사용 중인 계정으로 가입할 수 있으며, 네이버웍스 사용자가 아니라면 자신의 계정을 생성한 후 초대 받은 외부 그룹에 가입할 수 있습니다.

## 메시지에서 외부 그룹으로 초대하기

### 모바일 앱

1. 멤버를 초대할 외부 그룹 메시지방을 엽니다.
2. ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
3. '초대'를 탭합니다.
4. '그룹 초대 링크 공유하기'를 탭합니다.
5. 초대할 수단을 선택하여 그룹 초대 링크를 공유합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 왼쪽 메시지방 목록에서 멤버를 초대할 외부 그룹 메시지방을 클릭합니다.
4. 오른쪽 상단의 ![](/wp-content/uploads/2024/04/pc-icon-dotdotdot.png) 아이콘을 클릭합니다.
5. '그룹 초대 링크 공유'를 클릭합니다.
6. 초대할 사람의 이메일 주소를 입력하고 '초대 메일 보내기'를 클릭합니다.

## 그룹 관리 화면에서 외부 그룹으로 초대하기

그룹 생성 후, 그룹 관리 화면에 진입 시 외부 그룹 초대 기능을 사용할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. '주소록'을 탭합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_menu.png) 아이콘을 탭합니다.
4. '그룹'을 탭합니다.
5. 멤버를 초대할 그룹을 선택합니다.
6. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_member_setting-1.png) 아이콘을 탭합니다.
7. 초대 링크 메뉴 하단의 파란색 링크 부분을 탭하여 초대 링크를 복사합니다.
8. 초대할 사람에게 해당 링크를 전달합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '주소록'을 클릭합니다.
3. 왼쪽 메뉴에서 '그룹'을 클릭합니다.
4. 멤버를 초대할 그룹을 선택합니다.
5. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_group_information.png) 아이콘을 클릭합니다.
6. 초대 링크 메뉴 오른쪽의 '초대하기' 버튼을 클릭합니다.
7. 그룹 초대 팝업이 뜨면 초대할 사람의 이메일 주소를 입력하고 '초대 메일 보내기'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
