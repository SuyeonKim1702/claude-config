# 그룹 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
그룹 메시지방의 대표 사진을 설정할 수 있습니다.

또한, [그룹 마스터](/ko/use-guides/message/message-room/group-message-room/manage-group-master-and-member/)는 그룹 관리에 필요한 기능을 설정할 수 있습니다.

# 이전 대화 공개 설정

그룹의 마스터는 그룹 메시지방에 새로 초대하는 멤버에게 이전 대화 내용을 공개할 수 있습니다.

단, 이전 대화 공개 설정 후 새로 초대한 멤버에게만 적용되며, 메시지방의 기존 멤버들에게는 적용되지 않습니다.

### 모바일 앱

1. 그룹 메시지방에 들어가서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. 이전 대화 공개 여부를 설정합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '이전 대화 공개'를 클릭합니다.
6. '이전 대화 공개하기'를 클릭합니다.
7. '확인'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '이전 대화 공개'를 클릭합니다.
5. '이전 대화 공개하기'를 클릭합니다.
6. '확인'을 클릭합니다.

# 메시지방 배경 색상 설정

메시지방을 구분하기 위해서 메시지방 별로 배경 색상을 설정할 수 있습니다. 배경 색상은 본인에게만 적용되며, 다른 멤버들에게는 적용되지 않습니다.

### 모바일 앱

1. 그룹 메시지방에 들어가서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '배경 색상'을 탭합니다.
4. 설정할 배경 색상을 선택해서 메시지방의 배경색을 설정합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '배경 바꾸기'를 클릭합니다.
6. 나만 볼 수 있는 메시지방 배경색을 지정합니다.
7. '저장'을 클릭하여 배경색상을 저장합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '배경 바꾸기'를 클릭합니다.
5. 나만 볼 수 있는 메시지방 배경색을 지정합니다.
6. '확인'을 클릭하여 배경색상을 저장합니다.

# 대표 사진 설정

메시지방 목록에서 보이는 메시지방의 대표 사진을 설정할 수 있습니다.

3. 사진 아이콘을 탭합니다.
4. 카메라로 찍거나 사진을 선택해 대표 사진을 설정합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '사진 편집'을 클릭합니다.
6. ![](/wp-content/uploads/2024/05/ic30_profile_camera.png) 아이콘을 클릭하여 대표 사진을 설정합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '메시지방 설정'을 클릭합니다.
5. ![](/wp-content/uploads/2024/04/setting_pcapp_camera_icon.png) 아이콘을 클릭하여 대표 사진을 설정합니다.

대표 사진은 메시지방 이름과 더불어 메시지방을 구별하는 데 도움이 되므로, 그룹의 특성을 잘 대표되는 이미지로 등록하는 것이 좋습니다.

# 그룹 공개 및 비공개 설정

그룹 마스터는 그룹을 공개하거나 비공개로 설정할 수 있습니다.

단, 외부 그룹은 비공개로 설정됩니다.

그룹을 공개하면 주소록의 그룹의 목록에 노출됩니다. 그룹을 비공개로 하면 그룹 마스터가 멤버를 추가해야만 그룹에 들어올 수 있습니다.

### 모바일 앱

1. 그룹 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 버튼을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_members_setting-1.png) 아이콘을 탭합니다.
3. 그룹관리에서 공개그룹으로 하려면 그룹공개를 켜고, 비공개 그룹으로 하려면 그룹공개를 끕니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '그룹 관리'를 클릭합니다.
6. '그룹 수정' 화면으로 이동하여 '공개설정'에서 공개 그룹으로 하려면 '그룹공개'를 선택하고, 비공개 그룹으로 하려면 선택을 해제합니다.
7. '저장'을 클릭하여 그룹 공개 및 비공개를 설정합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '그룹 관리'를 클릭합니다.
5. '그룹 수정' 화면으로 이동하여 '공개설정'에서 공개 그룹으로 하려면 '그룹공개'를 선택하고, 비공개 그룹으로 하려면 선택을 해제합니다.
6. '저장'을 클릭하여 그룹 공개 및 비공개를 설정합니다.

# 메시지, 노트, 일정, 할 일, 폴더, 메일링 리스트 사용 설정

그룹 마스터는 해당 그룹의 메시지, 노트, 일정, 할 일, 폴더, 메일링 리스트의 사용 여부를 설정할 수 있습니다.

메일링 리스트는 네이버웍스 코어 Standard, Standard Plus 상품에서만 사용할 수 있으며, PC웹 주소록과 Admin에서 설정할 수 있습니다.

단, 외부 그룹은 할 일을 사용할 수 없습니다.

### 모바일 앱

1. 그룹 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 버튼을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_members_setting-1.png) 아이콘을 탭합니다.
3. 그룹관리에서 메시지, 노트, 일정, 할 일, 폴더를 켜거나 끕니다.  
   노트, 일정, 할 일, 폴더는 메시지방에서만 사용되므로 메시지 사용을 끄면 노트, 일정, 할 일, 폴더도 사용할 수 없습니다.
4. '저장'을 탭해 설정을 저장합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '그룹 관리'를 클릭합니다.
6. '그룹 수정' 화면으로 이동하여 '메시지방 기능'에서 '메시지', '노트', '일정', 할 일', '폴더', '메일링리스트'의 '사용'을 클릭하여 그룹 사용 기능을 켜거나 '사용 안 함'을 클릭하여 끕니다.
7. '저장'을 클릭하여 사용 여부를 설정합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '그룹 관리'를 클릭합니다.
5. '그룹 수정' 화면으로 이동하여 '메시지방 기능'에서 '메시지', '노트', '일정', 할 일', '폴더', '메일링리스트'의 '사용'을 클릭하여 그룹 사용 기능을 켜거나 '사용 안 함'을 클릭하여 끕니다.
6. '저장'을 클릭하여 사용 여부를 설정합니다.

노트, 일정, 폴더를 사용하다가 끄는 경우, 기존에 등록된 글이나 일정, 파일은 모두 삭제됩니다. 복구할 수 없으니 주의해서 선택해야 합니다.

# 그룹 삭제

그룹 마스터는 그룹을 삭제할 수 있습니다.

### 모바일 앱

1. 그룹 메시지방에서  ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 버튼을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_members_setting-1.png) 아이콘을 탭합니다.
3. 그룹 관리에서  '그룹 삭제' 버튼을 탭해서 그룹을 삭제합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '그룹 관리'를 클릭합니다.
6. '그룹 수정' 화면으로 이동하여 화면 왼쪽 하단의 '삭제'를 클릭합니다.
7. 주의사항을 읽고 '그룹 삭제 시 **모든 대화 이력과 사용중인  
   노트, 일정, 폴더에 저장된 데이터, 그룹 메일 주소가 삭제**됩니다.'를 선택합니다.
8. '삭제'를 클릭하여 그룹을 삭제합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 그룹 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '그룹 관리'를 클릭합니다.
5. '그룹 수정' 화면으로 이동하여 화면 왼쪽 하단의 '삭제'를 클릭합니다.
6. 주의사항을 읽고 '그룹 삭제 시 **모든 대화 이력과 사용중인  
   노트, 일정, 폴더에 저장된 데이터, 그룹 메일 주소가 삭제**됩니다.'를 선택합니다.
7. '삭제'를 클릭하여 그룹을 삭제합니다.

그룹을 삭제하면 사용 중인 메시지, 노트, 일정, 폴더, 메일링 리스트의 모든 데이터가 삭제됩니다.

삭제된 데이터는 복구할 수 없습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
