# 그룹 메시지방

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
업무 목적에 따라 관리해야 하는 메시지방을 그룹으로 만들면, 그룹의 마스터가 멤버와 기능 설정을 관리할 수 있습니다.

그룹 메시지방에서는 필요할 때 [노트](/ko/use-guides/message/note/open-note/), [일정](/ko/use-guides/message/group-schedule/open-group-calendar/), [할 일](/ko/use-guides/message/group-task/), [폴더](?page_id=3018), 메일링리스트를 쓸 수 있습니다.

# 그룹 메시지방 만들기

그룹 메시지방을 이용하기 위해 먼저 [그룹을 만듭니다.](/ko/use-guides/contacts/group/internal-group/)

# [그룹 설정](/ko/use-guides/message/message-room/group-message-room/manage-group/)

그룹 마스터는  노트, 일정, 할 일, 폴더의 사용 여부 등 그룹 설정을 할 수 있습니다.

# [그룹 마스터 및 멤버 관리](/ko/use-guides/message/message-room/group-message-room/manage-group-master-and-member/)

그룹 마스터는 그룹 마스터 및 멤버를 관리할 수 있습니다.

# 그룹 메시지방 나가기

그룹 멤버는 스스로 그룹 메시지방에서 나갈 수 없습니다.

[그룹 마스터가 멤버를 그룹에서 삭제](/ko/use-guides/message/message-room/group-message-room/manage-group-master-and-member/#i-2)해야 그룹 메시지방에서 나가게 됩니다.

나간 멤버는 해당 그룹 메시지방에 입장할 수 없지만, 이미 올렸던 글이나 사진 등은 나간 메시지방에 남습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
