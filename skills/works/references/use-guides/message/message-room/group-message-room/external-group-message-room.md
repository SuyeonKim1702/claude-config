# 외부 그룹 메시지방

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
외부 그룹 메시지방에서는 외부 네이버웍스 사용자와 협업할 수 있습니다.

메시지 이외에 [노트](/ko/use-guides/message/note/open-note/), [일정](/ko/use-guides/message/group-schedule/open-group-calendar/), [폴더](?page_id=3018)를 사용할 수 있습니다.

# 외부 그룹 메시지방 만들기

메시지방을 만들 때 '외부 사용자와 대화'를 선택해서 새로운 [외부 그룹](?page_id=27100) 메시지방을 만들 수 있습니다.

외부 그룹에는 사내 구성원 뿐만 아니라 네이버웍스를 사용하는 외부 사용자도 참여할 수 있습니다.

# [그룹 설정](/ko/use-guides/message/message-room/group-message-room/manage-group/)

그룹 마스터는 노트, 폴더, 일정의 사용 여부 등 그룹 설정을 할 수 있습니다.

# [그룹 마스터 및 멤버 관리](/ko/use-guides/message/message-room/group-message-room/manage-group-master-and-member/)

그룹 마스터는 그룹 마스터 및 멤버를 관리할 수 있습니다.

# 외부 그룹 메시지방 나가기

외부 그룹 멤버는 스스로 외부 그룹 메시지방에서 나갈 수 없습니다.

그룹 마스터가 멤버를 그룹에서 삭제해야 그룹 메시지방에서 나가게 됩니다.

나간 멤버는 해당 그룹의 메시지방에 입장할 수 없지만, 이미 올렸던 글이나 사진 등은 나간 메시지방에 남습니다.

관리자가 특정 멤버의 [외부 연동 권한](/ko/admin-guides/security/external-message-authority/#i-2)을 비활성화하면, 그 멤버는 외부 그룹에서 자동으로 제외되어 메시지방에서 나가게 됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
