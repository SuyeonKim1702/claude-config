# 메시지 Bot

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
[메시지 Bot API](https://developers.worksmobile.com/kr/docs/bot)를 이용해 메시지에서 사내 시스템의 알림을 받거나 사내 시스템과 대화하게 할 수 있습니다.

결재, 정산, 시스템 상태 등의 사내 시스템의 알림을 SMS 대신 메시지로 보낼 수 있어 비용을 절감할 수 있습니다.

근태, 안부확인, 결재승인 등 사내 시스템을 네이버웍스 Bot으로 구현하면 해당 서비스로 이동하지 않고 메시지에서 바로 처리할 수 있습니다.

다음 7가지 타입이 있습니다.

- 텍스트/이미지형
- 버튼형
- 리스트형
- 퍼시스턴트 메뉴형
- 리치 메뉴형
- 케러셀/이미지 케러셀형
- 빠른 응답형

# 메시지방에 Bot 초대하기

메시지방에 Bot을 초대할 수 있습니다.

초대할 수 있는 Bot의 종류는 관리자가 [Admin에서 관리](/ko/admin-guides/manage-service/bot/)할 수 있습니다.

### 모바일 앱

1. 메시지방의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '초대'를 탭합니다.
3. 초대할 Bot을 선택합니다.
4. '확인'을 탭합니다.

### PC 웹

1. 상단의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 ![](https://help.worksmobile.com//wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '초대하기'를 클릭합니다.
6. 초대할 Bot을 선택합니다.
7. '초대하기'를 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '초대하기'를 클릭합니다.
5. 초대할 Bot을 선택합니다.
6. '초대하기'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
