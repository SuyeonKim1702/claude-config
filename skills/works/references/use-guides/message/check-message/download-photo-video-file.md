# 사진, 동영상, 파일 다운로드

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방에서 주고 받은 사진이나 동영상, 파일을 기기에 다운로드할 수 있습니다.

# 사진 다운로드

보관 기간 내의 사진을 기기나 드라이브에 다운로드할 수 있습니다.

### 모바일 앱

1. 메시지방에서 다운로드할 사진을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_save_line.png) 아이콘을 탭합니다.
   - 묶음 사진인 경우 현재 보고 있는 사진만 다운로드할지, 묶음 사진 전체를 다운로드할지 선택할 수 있습니다.
3. 저장할 위치를 선택합니다. 가입한 상품에 따라 일부는 노출되지 않을 수 있습니다.
   - 기기에 저장
   - 폴더 저장/드라이브 저장: 같은 메시지방의 폴더 또는 드라이브에 저장

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 사진을 다운로드할 메시지방을 클릭합니다.
4. 저장할 사진 하단의 '저장'을 클릭합니다.
   - 묶음 사진인 경우 묶음 사진 전체가 다운로드됩니다.
5. 저장할 위치를 선택합니다. 사용환경에 따라 일부는 노출되지 않을 수 있습니다.
   - PC 저장: 기기에 저장
   - 폴더 저장/드라이브 저장: 같은 메시지방의 폴더 드라이브에 저장
   - 노트로 보내기: 같은 메시지방 노트의 새로운 게시글 또는 게시글의 댓글로 보내기

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 사진 다운로드할 메시지방을 클릭합니다.
3. 저장할 사진 하단의 '저장', '드라이브에 저장', '폴더에 저장' 가운데 하나를 클릭합니다.
4. '저장'의 경우 저장 위치와 파일명을 지정하여 저장하고, '드라이브에 저장' 또는 '폴더에 저장'의 경우 원하는 위치를 선택하여 저장합니다.
   - 묶음 사진인 경우 묶음 사진 전체가 다운로드됩니다.

## 사진 편집해서 저장하기

[Admin에서 '모바일앱 파일 다운로드'](/ko/admin-guides/security/file-security/)가 가능하도록 설정되어 있을 경우, 사진을 편집하여 저장할 수 있습니다.

### 모바일 앱

1. 편집할 사진을 탭합니다.
2. '편집'을 탭합니다.
3. 편집이 필요한 사진일 경우 아래 기능을 사용하여 편집한 후 '적용'을 탭합니다.  

   |  |  |
   | --- | --- |
   |  | 사진 회전 |
   |  | 사진 자르기 |
   |  | 사진에 그림 그리기 |

   사진 편집 후 원본으로 돌리고 싶다면 '원본으로'를 탭합니다.
4. ![](/wp-content/uploads/2024/05/common_24_save_line.png) 을 탭해 사진을 저장합니다.

# 파일 다운로드

첨부파일 보관 기간 내의 파일을 기기나 드라이브에 저장할 수 있습니다.

### 모바일 앱

### **폴더 /** 드라이브에 다운로드

1. 메시지방에서 다운로드할 파일을 길게 누릅니다.
2. '폴더' 또는 '드라이브'를 탭해서 저장할 위치를 선택합니다.

### iOS 기기에 파일 다운로드

iOS 11 및 이후 버전에서만 기기에 파일 다운로드가 가능합니다. iOS 11 이전 버전에서는 기기의 특성상 파일을 기기에 저장할 수 없습니다.

단, 파일을 탭했을 때 [관리자의 설정](/ko/admin-guides/security/file-security/)에 따라 모바일 [문서 뷰어](/ko/use-guides/drive/document-viewer/overview/)가 열리며 다운로드가 제한될 수 있습니다.

1. 메시지방에서 다운로드할 첨부파일을 탭합니다.
2. '파일 보기'를 탭합니다.
3. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_share.png) 아이콘을 탭합니다.
4. '파일에 저장'을 탭하여 기기에 저장합니다.

### Android 기기에 파일 다운로드

Android 기기에서 파일을 열면 자동으로 기기에 저장됩니다.

단, [관리자의 설정](/ko/admin-guides/security/file-security/)에 따라 파일을 탭했을 때 모바일 [문서 뷰어](/ko/use-guides/drive/document-viewer/overview/)가 열리며 다운로드가 제한될 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 파일을 다운로드할 메시지방을 클릭합니다.
4. 파일 하단의 '저장'을 클릭합니다.
5. 저장할 위치를 선택합니다. 사용 환경에 따라 일부는 노출되지 않을 수 있습니다.
   - PC 저장: 기기에 저장
   - 폴더 저장/드라이브 저장: 같은 메시지방의 폴더 또는 드라이브에 저장
   - 노트로 보내기: 같은 메시지방 노트의 새로운 게시글 또는 게시글의 댓글로 보내기

### PC 앱

3. 파일 하단의 '저장', '드라이브에 저장', '폴더에 저장' 가운데 하나를 클릭합니다.

# 메시지 파일 보관 기간

메시지방의 사진, 동영상, 파일의 보관 기간은 소속 회사/단체의 [관리자가 설정한 사진/파일 다운로드 기간](/ko/admin-guides/manage-service/message/general-settings/#i-2)에 따릅니다.

관리자는 보관 기간을 7일에서 최대 3년까지 설정할 수 있습니다.

보관 기간이 지난 사진, 동영상, 파일은 메시지방에서 미리보기 및 다운로드를 할 수 없으며, 검색이 되지 않습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
