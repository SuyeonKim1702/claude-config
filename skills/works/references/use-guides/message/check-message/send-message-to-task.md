# 메시지를 할 일에 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지를 할 일로 보내 업무를 관리할 수 있습니다.

할 일로 보낸 메시지는 할 일에 담겨 할 일과 함께 보여지며, [보낸 메시지방으로 이동](/ko/use-guides/task/task-check/check-task/#i-5)할 수도 있습니다.

### 모바일 앱

1. 하단의 '메시지'를 탭합니다.
2. 메시지방 목록에서 메시지방을 탭합니다.
3. 할 일로 보내려는 메시지를 길게 누릅니다.
4. '할 일'을 선택한 후, '확인'을 탭합니다. 할 일로 보낼 다른 메시지가 있다면 모두 선택한 뒤 '확인'을 탭합니다.
5. 새 할 일을 씁니다.
6. '저장'을 탭하여 할 일을 저장합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방에서 전달하려고 하는 메시지에 마우스 오른쪽 버튼을 클릭합니다.
5. '할 일로 보내기'를 클릭합니다. 할 일로 보낼 메시지를 모두 선택한 뒤 '할 일로 보내기'를 클릭합니다.
6. 새 할 일을 씁니다.
7. '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방에서 전달하려고 하는 메시지에 마우스 오른쪽 버튼을 클릭합니다.
4. '할 일로 보내기'를 클릭합니다. 할 일로 보낼 메시지를 모두 선택한 뒤 '할 일로 보내기'를 클릭합니다.
5. 새 할 일을 씁니다.
6. '저장'을 클릭합니다.

할 일 쓰기 화면에서 특정 메시지방을 선택할 경우 메시지방 할 일로 등록되며, 해당 메시지방에 알림을 발송할지 여부를 선택할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
