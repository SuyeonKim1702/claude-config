# 콘텐츠 모아보기

### 모바일 앱

1. 메시지방의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '사진/동영상', '파일', '링크', '템플릿' 중 모아볼 콘텐츠 종류를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 콘텐츠를 모아볼 메시지방을 클릭합니다.
4. 1:1 메시지방에서는 화면 오른쪽에서 콘텐츠를 모아볼 수 있습니다.  
   여러 명이 참여하는 메시지방에서는 화면 오른쪽의 '콘텐츠' 탭을 클릭해 콘텐츠를 모아볼 수 있습니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 콘텐츠를 모아볼 메시지방을 클릭합니다.
3. ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '콘텐츠'를 클릭하여 콘텐츠를 모아볼 수 있습니다.

# 콘텐츠 위치로 이동하기

메시지방 내에 콘텐츠가 존재하는 위치로 이동하여 확인할 수 있습니다.

### 모바일 앱

1. 메시지방의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '사진/동영상', '파일', '링크', '템플릿' 중 콘텐츠 종류를 탭합니다.
3. '메시지로 이동'을 탭하여 선택한 콘텐츠의 메시지방 내 위치로 이동합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 콘텐츠가 있는 메시지방을 클릭합니다.
4. 1:1 메시지방의 경우 화면 오른쪽에서, 여러 명이 참여하는 메시지방의 경우 화면 오른쪽의 '콘텐츠' 탭을 클릭하여 콘텐츠를 확인합니다.
5. 콘텐츠 종류를 선택합니다.
6. 마우스 포인터를 콘텐츠 위에 올려, '메시지로 이동'을 클릭합니다.  
   선택한 콘텐츠의 메시지방 내 위치로 이동합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 콘텐츠가 있는 메시지방을 클릭합니다.
3. ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '콘텐츠'를 클릭합니다.
5. 콘텐츠 종류를 선택합니다.
6. 마우스 포인터를 콘텐츠 위에 올려, '메시지로 이동'을 클릭합니다.  
   선택한 콘텐츠의 메시지방 내 위치로 이동합니다.

# 여러 개의 콘텐츠를 저장하기

여러 개의 콘텐츠를 선택하여 한번에 저장할 수 있습니다.

### 모바일 앱

1. 메시지방의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '사진/동영상', '파일', '템플릿' 중 저장할 콘텐츠 종류를 탭합니다.
3. '선택'을 탭합니다.
4. 저장할 콘텐츠를 선택합니다.
5. ![](/wp-content/uploads/2024/05/common_24_save_line.png) 아이콘을 탭합니다.
6. 저장할 곳을 탭하여 콘텐츠를 저장합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 해당 메시지방을 클릭합니다.
4. 1:1 메시지방의 경우 화면 오른쪽에서 여러 명이 참여하는 메시지방의 경우 화면 오른쪽의 '콘텐츠' 탭을 클릭하여 콘텐츠를 확인합니다.
5. '사진/동영상', '파일',  '템플릿' 중 저장할 콘텐츠 종류를 선택합니다.
6. 마우스 포인터를 저장할 콘텐츠 위에 올려, 동그라미를 체크합니다.
7. '저장'을 클릭합니다.
8. 저장할 장소를 클릭하여 콘텐츠를 저장합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 해당 메시지방을 클릭합니다.
3. ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '콘텐츠'를 클릭합니다.
5. '사진/동영상', '파일', '템플릿' 중 저장할 콘텐츠 종류를 선택합니다.
6. 마우스 포인터를 저장하고 싶은 콘텐츠 위에 올려, 동그라미를 체크합니다.
7. 저장할 위치를 클릭하여 콘텐츠를 저장합니다.

콘텐츠는 20개까지 선택할 수 있습니다.

# 여러 개의 콘텐츠를 보내기

여러 개의 콘텐츠를 선택하여 한번에 메시지로 보낼 수 있습니다.

### 모바일 앱

1. 메시지방의 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '사진/동영상', '파일', '템플릿' 중 보낼 콘텐츠 종류를 탭합니다.
3. '선택'을 탭합니다.
4. 보낼 콘텐츠를 선택합니다.
5. 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_share_iphone.png) 아이콘을 탭합니다.
6. 보낼 위치를 탭합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 해당 메시지방을 클릭합니다.
4. 1:1 메시지방의 경우 화면 오른쪽에서, 여러 명이 참여하는 메시지방의 경우 화면 오른쪽의 '콘텐츠' 탭을 클릭하여 콘텐츠를 확인합니다.
5. '사진/동영상', '파일', '템플릿' 중 전달할 콘텐츠 종류를 선택합니다.
6. 마우스 포인터를 전달할 콘텐츠 위에 올려, 동그라미를 체크합니다.
7. 하단의 '전달'을 클릭합니다.
8. 전달할 위치를 클릭합니다.
9. '전달'을 클릭하여 콘텐츠를 전달합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 해당 메시지방을 클릭합니다.
3. ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '콘텐츠'를 클릭합니다.
5. '사진/동영상', '파일', '템플릿' 중 전달할 콘텐츠 종류를 선택합니다.
6. 마우스 포인터를 전달할 콘텐츠 위에 올려, 동그라미를 체크합니다.
7. '전달'을 클릭합니다.
8. 전달할 위치를 클릭합니다.
9. '전달하기'를 클릭하여 콘텐츠를 전달합니다.

콘텐츠는 20개까지 선택할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
