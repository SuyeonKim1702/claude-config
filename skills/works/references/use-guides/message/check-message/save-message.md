# 메시지 저장

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지 내용을 .txt 형식의 파일로 저장할 수 있습니다.

단, 현재 화면에 표시되고 있는 내용만 저장 가능합니다.

[메시지를 노트에 저장](/ko/use-guides/message/check-message/send-message-to-note/)할 수도 있고, Admin에서 [메시지 감사 로그](/ko/admin-guides/audit/#i-3)를 다운로드해 저장할 수도 있습니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 저장할 메시지방을 클릭합니다.
3. 메시지방 오른쪽 상단 ![](/wp-content/uploads/2024/05/ic16_more_767676.png) 아이콘을 클릭합니다.
4. '메시지 저장'을 클릭합니다.
5. 주의사항을 확인한 후 '확인'을 클릭합니다.
6. 저장 장소와 이름 등을 수정 후 '저장'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
