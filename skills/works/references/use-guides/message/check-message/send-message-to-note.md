# 메시지를 노트에 보내기

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
여러 명이 참여하는 메시지방의 메시지를 노트에 보낼 수 있습니다.

메시지는 [관리자가 설정한 보관 기간](/ko/admin-guides/manage-service/message/general-settings/#i-2)이 지나면 볼 수 없고, 필요한 메시지를 대화에서 매번 찾는 것도 어려울 수 있으므로, 필요한 메시지는 노트에 저장해 두면 유용합니다.

### 모바일 앱

1. 노트에 보낼 메시지를 길게 누릅니다.
2. '노트'를 탭합니다.
3. 보낼 다른 메시지가 있다면 선택한 후 '확인'을 탭합니다.
4. 새 글로 작성할지 기존 글의 댓글로 작성할지 선택합니다.
   - 새 글로 작성하는 경우, '새 게시글로 보내기'를 선택하고, '확인'을 탭합니다. 선택한 메시지가 본문으로 첨부된 상태로 글쓰기 창이 열립니다.
   - 기존 글의 댓글로 작성할 경우, 글을 선택하고 '확인'을 탭하면 선택한 메시지가 해당 글의 댓글에 입력된 상태로 댓글 창이 열립니다.
5. 편집 한 후 글이나 댓글을 저장합니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지방에서 전달하려고 하는 메시지에 마우스 오른쪽 버튼을 클릭합니다.  
   또는 메시지 옆에 마우스를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
5. '노트로 보내기'를 클릭합니다.
6. 저장할 다른 메시지가 있다면 선택한 후 '노트로 보내기'를 클릭합니다.
7. '노트로 보내기'에서 새 글로 작성할지 기존 글의 댓글로 작성할지 선택합니다.
   - 새 글로 작성하는 경우, '새 게시글로 보내기'를 선택하고, '확인'을 클릭합니다. 선택한 메시지가 본문으로 첨부된 상태로 글쓰기 창이 열립니다.
   - 기존 글의 댓글로 작성할 경우, 글을 선택하고, '확인'을 클릭하면 선택한 메시지가 해당 글의 댓글에 입력된 상태로 댓글 창이 열립니다.
8. 새 글로 작성한 경우 '저장'을, 기존 글의 댓글로 작성한 경우 댓글 창에서 메시지의 내용을 확인 후 키보드의 Enter 키를 눌러 노트에 저장합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 원하는 메시지방을 클릭합니다.
3. 메시지방에서 전달하려고 하는 메시지에 마우스 오른쪽 버튼을 클릭합니다.  
   또는 메시지 옆에 마우스를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
4. '노트로 보내기'를 클릭합니다.
5. PC웹 화면으로 이동하여 '노트로 보내기'에서 새 글로 작성할지 기존 글의 댓글로 작성할지 선택합니다.
   - 새 글로 작성하는 경우, '새 게시글로 보내기'를 선택하고, '확인'을 클릭합니다. 선택한 메시지가 본문으로 첨부된 상태로 글쓰기 창이 열립니다.
   - 기존 글의 댓글로 작성할 경우, 글을 선택하고, '확인'을 클릭하면 선택한 메시지가 해당 글의 댓글에 입력된 상태로 댓글 창이 열립니다.
6. 새 글로 작성한 경우 '저장'을, 기존 글의 댓글로 작성한 경우 댓글 창에서 메시지의 내용을 확인 후 키보드의 Enter 키를 눌러 노트에 저장합니다.

한 번에 최대 50개의 메시지를 노트의 새 글이나 댓글로 작성할 수 있으며, 텍스트, 이미지, 파일, 음성 파일, 영상 파일 모두 노트에 저장이 가능합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
