# 템플릿 인쇄

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
[메시지방에서 보낸 템플릿](/ko/use-guides/message/send-message/send-template/)을 작성 내용을 포함해 인쇄할 수 있습니다.

### PC 웹

1. 인쇄할 템플릿 메시지 하단의 '템플릿 자세히 보기'를 클릭하거나, 화면 오른쪽의 '[콘텐츠 > 템플릿](/ko/use-guides/message/check-message/contents/)' 목록에서 인쇄할 템플릿을 클릭합니다.
2. 템플릿 본문 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭하고 '인쇄'를 클릭합니다.

### PC 앱

1. 인쇄할 템플릿 메시지 하단의 '템플릿 자세히 보기'를 클릭하거나, 화면 오른쪽 상단의 '[콘텐츠 > 템플릿](/ko/use-guides/message/check-message/contents/)' 목록에서 인쇄할 템플릿을 클릭합니다.
2. 템플릿 본문 오른쪽 상단의  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭하고 '인쇄'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
