# 메시지 삭제

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
메시지방에서 내가 보낸 메시지나 다른 사람이 보낸 메시지를 삭제할 수 있습니다.

단, 해당 기기에 있는 메시지만 삭제되며, 다른 사용자의 기기와 내 계정으로 사용 중인 다른 기기에서는 삭제되지 않습니다.

삭제한 메시지는 복구할 수 없습니다.

### 모바일 앱

1. 메시지방에서 삭제하려는 메시지를 길게 누릅니다.
2. '삭제'를 탭합니다.
3. 여러 메시지를 삭제할 경우에는 삭제할 메시지들을 선택합니다.
4. '확인'을 탭합니다.
5. '삭제'를 탭해 메시지를 삭제합니다.

### PC 앱

1. 화면 왼쪽 상단의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 메시지방을 클릭합니다.
3. 메시지방에서 삭제하려고 하는 메시지에 마우스 오른쪽 버튼을 클릭합니다.  
   혹은 메시지 옆에 마우스를 옮겨두면  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘이 보이며, 해당 버튼을 클릭합니다.
4. 메뉴에서 '삭제'를 클릭하여 메시지를 삭제합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
