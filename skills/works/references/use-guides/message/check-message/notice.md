# 메시지 공지

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
대화에 참여한 모든 멤버들에게 공지할 내용이 있을 때, 메시지를 공지로 등록하여 메시지방 상단에 노출할 수 있습니다.

메시지방 멤버 누구나 공지를 등록하거나 내릴 수 있고, 공지를 선택하면 메시지방 스크롤이 이동하여 메시지 원문을 확인할 수 있습니다.

단, LINE 사용자와의 메시지방이나 Bot 메시지방 등 일부 메시지방에서는 공지 기능이 제공되지 않습니다.

# **공지 등록하기**

메시지방 멤버는 누구나 본인의 메시지 뿐만 아니라 다른 사람의 메시지도 공지로 등록할 수 있습니다.

### 모바일 앱

1. 공지로 등록할 메시지를 길게 누릅니다.
2. ‘공지’ 를 탭합니다.

### PC 웹

1. 공지로 등록할 메시지에 마우스 오른쪽 버튼을 클릭합니다.  
   또는 메시지 옆에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
2. ‘공지’를 클릭합니다.

### PC 앱

1. 공지로 등록할 메시지에 마우스 오른쪽 버튼을 클릭합니다.  
   또는 메시지 옆에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
2. ‘공지’를 클릭합니다.

# **공지 내리기**

메시지방 멤버는 누구나 메시지방에 등록된 공지를 내릴 수 있습니다.

### 모바일 앱

1. 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 탭하여 펼칩니다.
2. 내리고 싶은 공지의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 탭합니다.
3. ‘공지 내리기’를 탭합니다.

### PC 웹

1. 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 클릭하여 펼칩니다.
2. 내리고 싶은 공지의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
3. ‘공지 내리기’를 클릭합니다.

### PC 앱

1. 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 클릭하여 펼칩니다.
2. 내리고 싶은 공지의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
3. ‘공지 내리기’를 클릭합니다.

# **공지 확인하기**

공지는 최근 등록된 순서대로 최대 5개까지 볼 수 있고, 밀려난 공지는 다시 보이지 않습니다.

공지를 선택하면 메시지방 스크롤이 이동하여 메시지 원문을 확인할 수 있습니다.

단, 이전 대화 내용이 [공개 범위](https://help.worksmobile.com/ko/use-guides/message/message-room/general-message-room/settings#i-3)가 아니어서 원문으로 이동이 불가한 경우 ‘찾을 수 없는 메시지입니다.’ 문구가 뜹니다.

# **공지 관리**

공지 영역에 노출되는 공지와 접힘/펼침 상태는 멤버 개인별로 관리할 수 있습니다.

## **다시 보지 않기**

공지를 다시 보지 않게 설정할 수 있습니다.

### 모바일 앱

1. 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 탭하여 펼칩니다.
2. 다시 보지 않을 공지의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 탭합니다.
3. ‘다시 보지 않기’를 탭합니다.

### PC 웹

1. 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 클릭하여 펼칩니다.
2. 다시 보지 않을 공지의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
3. ‘다시 보지 않기’를 클릭합니다.

### PC 앱

1. 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 클릭하여 펼칩니다.
2. 다시 보지 않을 공지의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭합니다.
3. ‘다시 보지 않기’를 클릭합니다.

또는 공지 창 하단의 '다시 보지 않기'를 눌러 전체 공지를 다시 보지 않게 할 수도 있습니다.

## **공지 접기/펼치기**

메시지방 상단에는 기본적으로 최신 공지 1개만 노출됩니다.

공지 영역을 펼쳐 더 많은 공지를 보거나, 메시지방 오른쪽 상단에 작게 접어둘 수 있습니다.

### 모바일 앱

1. 기본 상태의 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 탭하여 펼칩니다.
2. 공지 영역 하단의 ‘접기’를 탭하여 접습니다.
3. 접힌 상태의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_massagenoti_line_118dff.png) 아이콘을 탭하여 다시 펼칩니다.
4. 공지 영역 오른쪽 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/admin-ajax-reverse.png) 아이콘을 탭하면 다시 기본 상태의 공지 영역으로 돌아갑니다.

### PC 웹

1. 기본 상태의 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 클릭하여 펼칩니다.
2. 공지 영역 하단의 ‘접기’를 클릭하여 접습니다.
3. 접힌 상태의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_massagenoti_line_118dff.png) 아이콘을 클릭하여 다시 펼칩니다.
4. 공지 영역 오른쪽 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/admin-ajax-reverse.png) 아이콘을 클릭하면 다시 기본 상태의 공지 영역으로 돌아갑니다.

### PC 앱

1. 기본 상태의 공지 영역에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic12_arrow_line_open_767676.png) 아이콘을 클릭하여 펼칩니다.
2. 공지 영역 하단의 ‘접기’를 클릭하여 접습니다.
3. 접힌 상태의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic16_massagenoti_line_118dff.png) 아이콘을 클릭하여 다시 펼칩니다.
