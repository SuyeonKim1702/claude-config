# 메시지 읽음 확인

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
내가 보낸 메시지를 메시지방에서 몇 명이 읽었는지 확인할 수 있습니다.

읽은 멤버, 안 읽은 멤버를 확인할 수 있어 업무 대화를 놓치지 않을 수 있습니다.

# 메시지 읽은 멤버수 확인

내가 보낸 메시지 왼쪽에 읽은 멤버 수가 표시됩니다. 1:1 메시지방에서는 '읽음'이라고 표시됩니다.

# 메시지 읽은 멤버 확인

내가 보낸 메시지의 읽은 멤버, 안 읽은 멤버를 명단으로 확인할 수 있습니다.

만약 급한 경우에는 프로필을 탭해서 1:1 메시지를 보내거나 전화를 걸 수도 있습니다. 메시지방에서 [멘션 기능](?page_id=2615)을 이용해서 특정 사람에게 알림이 가도록 메시지를 보내는 방법도 있습니다.

단, [LINE 사용자](/ko/use-guides/message/message-with-external-user/message-line-user/) 또는 [외부 네이버웍스 사용자](/ko/use-guides/message/message-with-external-user/message-with-external-line-works-user/)를 포함해 만들어진 외부 메시지방에서는 읽은 멤버를 확인할 수 없습니다.

### 모바일 앱

1. '읽음'을 탭합니다. 읽은 멤버와 안 읽은 멤버를 선택해서 확인할 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '메시지'를 클릭합니다.
3. 메시지방 목록에서 메시지방을 클릭합니다.
4. 메시지 전송 시간 위에 표시되는 '읽음'을 클릭합니다.  
   읽은 멤버와 안 읽은 멤버를 선택해서 확인할 수 있습니다.

### PC 앱

1. 화면 왼쪽 상단의  ![](/wp-content/uploads/2024/04/icon-message-pc-web-board-button.png) 아이콘을 클릭합니다.
2. 메시지방 목록에서 읽음 확인하고 싶은 메시지방을 클릭합니다.
3. 메시지 전송 시간 위에 표시되는 '읽음'을 클릭합니다.  
   읽은 멤버와 안 읽은 멤버를 선택해서 확인할 수 있습니다.

프로필 사진을 탭해서 1:1 메시지 등의 추가 커뮤니케이션을 할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
