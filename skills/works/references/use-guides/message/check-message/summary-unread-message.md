# 읽지 않은 메시지 요약

읽지 않은 메시지가 30건 이상일 경우 WORKS AI를 활용하여 읽지 않은 메시지 내용을 요약할 수 있습니다.

단, 1:1 메시지방, Bot 메시지방에서는 메시지를 요약할 수 없습니다.

요약 중에 메시지방을 나가면 요약이 중단됩니다.

### 모바일 앱

1. 읽지 않은 메시지가 쌓인 메시지방을 선택합니다.
2. '읽지 않은 메시지 요약하기' 버튼을 탭합니다.
3. 메시지 요약 내용을 확인합니다.

### PC 웹

1. 읽지 않은 메시지가 쌓인 메시지방을 선택합니다.
2. '읽지 않은 메시지 요약하기'를 클릭합니다.
3. 메시지 요약 내용을 확인합니다.

### PC 앱

1. 읽지 않은 메시지가 쌓인 메시지방을 선택합니다.
2. '읽지 않은 메시지 요약하기'를 클릭합니다.
3. 메시지 요약 내용을 확인합니다.

요약 결과 화면에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/03/copy.png) 아이콘을 눌러 요약된 내용을 복사할 수도 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
