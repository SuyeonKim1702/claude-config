# 메시지방 멤버 전체에게 메일 및 일정 초대

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
일반 메시지방, 조직 메시지방, 그룹 메시지방 멤버 모두를 대상으로 한꺼번에 메일을 보내거나 일정에 초대할 수 있습니다.

# 메시지방 멤버 전체에게 메일 보내기

메시지방의 멤버 전체에게 메일을 보낼 때, 일일이 멤버를 찾지 않아도 한꺼번에 수신인으로 추가할 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '메일'을 탭합니다.
3. '확인'을 탭하고 메일을 작성해서 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다
3. 메시지방 목록에서 조직/그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '메일'을 클릭합니다.
6. '메일쓰기 : 메일' 화면으로 이동하여 메일을 작성하고 왼쪽 상단의 '보내기'를 클릭하여 메일을 보냅니다.

# 메시지방 멤버 전체에게 일정 초대하기

메시지방 멤버 전체를 일정에 초대할 때, 일일이 멤버를 찾지 않아도 한꺼번에 참석자로 추가할 수 있습니다.

### 모바일 앱

1. 메시지방에서 ![](/wp-content/uploads/2024/05/common_24_menu_line.png) 아이콘을 탭합니다.
2. '약속 초대'를 탭합니다.
3. '확인'을 탭하고 약속 초대를 보냅니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. ‘메시지’를 클릭합니다
3. 메시지방 목록에서 조직/그룹 메시지방을 클릭합니다.
4. 메시지방 오른쪽 상단의 ![](/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘을 클릭합니다.
5. '약속 초대'를 클릭합니다.
6. '일정 : 캘린더' 화면으로 이동하여 일정 내용을 작성하고 왼쪽 상단의 '저장'을 클릭하여 일정을 초대합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
