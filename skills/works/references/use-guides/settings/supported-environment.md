# 지원 환경

네이버웍스의 정상적인 사용을 지원하는 시스템 환경입니다.

네이버웍스 앱과 드라이브 탐색기는 [다운로드 페이지](https://naver.worksmobile.com/download/)에서 다운로드할 수 있습니다.

서비스 권장 환경 미만에서는 네이버웍스의 정상 동작을 보장할 수 없으며 최신 앱 설치가 제한될 수 있습니다.

# 시스템 요구 사항

|  |  |
| --- | --- |
| 플랫폼 | 시스템 요구 사항 |
| PC웹 | - Google Chrome 최신 버전 - Microsoft Edge 최신 버전 |
| PC앱 | - Windows 10 22H2 및 이후 - Windows 11 22H2 및 이후 - macOS 13.0 및 이후 |
| 모바일앱 | - Android 10.0 및 이후 - iOS 16 및 이후 (iPad 포함) |
| 드라이브 탐색기 | - Windows 10 22H2 및 이후, Windows 11 22H2 및 이후   - 프로세서: AMD 64 기반 64비트 프로세서 2.5GHz 이상 (ARM 기반 설치 불가)   - 메모리: 8GB 이상   - 디스크 포맷 형식: NTFS   - 최소 용량: 3GB   - 권장 용량: 1TB 이상 - macOS 12.0 및 이후   - 디스크 포맷 형식: APFS   - 메모리: 8GB 이상   - 최소 용량: 3GB   - 권장 용량: 1TB 이상 |

# 앱 이용 시 주의사항

안정적인 서비스 이용을 보장하기 위해 앞으로 최신 버전 대비 2개 버전 이전의 앱을 이용하는 사용자를 대상으로 강제 업데이트를 진행합니다.

// start of content for ko en jp

업데이트 정보는 [릴리스 노트](https://naver.worksmobile.com/release-notes/)에서 확인할 수 있습니다.

// end of content for ko en jp

단, 서비스 운영 상황에 따라 필수 업데이트 대상 버전이 변경될 수도 있으며, 업데이트가 자동 진행될 수도 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
