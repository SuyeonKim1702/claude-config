# 모바일 앱 데이터 삭제

기기의 네이버웍스 모바일 앱의 데이터를 원격으로 삭제할 수 있습니다.

원격 삭제 시 네이버웍스 앱이 초기화되며, 다운로드한 파일이 삭제됩니다.

단, Android 기기에서 여러 계정을 사용 중인 경우, 현재 로그인된 계정의 데이터와 파일만 삭제됩니다.

한 번 삭제된 데이터는 복구할 수 없습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '디바이스'를 탭합니다.
4. '최근 3년 간 접속한 기기' 목록에서 원격 관리할 기기를 선택합니다.
5. '모바일앱 데이터 삭제' 버튼을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. '디바이스'를 선택합니다.
3. '최근 3년 간 접속한 기기' 목록에서 원격 관리할 기기를 선택합니다.
4. '모바일앱 데이터 삭제' 버튼을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
