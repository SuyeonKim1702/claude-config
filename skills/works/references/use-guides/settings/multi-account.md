# 여러 계정으로 사용하기

여러 개의 네이버웍스 계정을 가지고 있다면 모바일앱에서 여러 계정을 추가해서 이용할 수 있습니다.

- 추가한 모든 계정의 알림을 받거나, 계정 별로 알림을 다르게 설정할 수 있습니다.
- 추가한 여러 계정 간의 전환이 간단합니다.

이 기능은 모바일앱에서만 사용할 수 있습니다. PC웹 서비스나 PC앱, 드라이브 탐색기에서 다른 계정으로 전환하려면 로그아웃 후 다시 로그인해야 합니다.

# 다른 계정 추가하기

네이버웍스 모바일앱에 최대 5개의 계정을 추가할 수 있습니다.

## 계정 추가하기

첫 번째 계정으로 로그인된 상태에서 두 번째 계정을 추가할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 왼쪽 상단의 소속 회사/단체 이름을 탭합니다.
3. '다른 계정 추가하기'를 탭합니다.
4. 추가할 계정으로 로그인합니다.

# 계정 전환하기

다음 방법으로 추가한 다른 계정으로 전환할 수 있습니다.

- 다른 계정에서 받은 푸시 알림을 선택하기
- 계정 전환 메뉴에서 직접 전환하기

### 모바일 앱

1. '홈'에서 왼쪽 상단의 소속 회사/단체의 이름을 탭하거나 다른 서비스 화면에서 오른쪽 상단의 프로필 아이콘을 탭합니다.
2. 전환할 다른 계정을 탭합니다.  
   만약 전환할 계정이 로그아웃 상태라면 다시 로그인하면서 전환할 수 있습니다.

# 계정을 로그아웃하거나 제거하기

추가한 계정을 더이상 이용하지 않으려면 로그아웃 하거나 기기에서 제거할 수 있습니다.

## 계정 로그아웃하기

지금은 사용하지 않지만 나중에 다시 사용하려면 해당 계정에서 로그아웃합니다.

- 기존에 동기화 했던 데이터가 삭제되지 않습니다.
- 더이상 이 계정의 알림을 받지 않습니다.
- 다음에 다시 계정을 이용하려면 다시 로그인해야 합니다.

### 모바일 앱

1. '홈'에서 왼쪽 상단의 소속 회사/단체의 이름을 탭합니다.
2. 오른쪽 상단에서 '관리'를 선택합니다.
3. 로그아웃할 계정을 켜거나 끕니다.

## 기기에서 계정 제거하기

이 기기에서 계정을 더이상 사용하지 않으려면 계정을 제거합니다.

- 이 기기의 계정 목록에서 계정이 제거되고 동기화했던 데이터가 삭제됩니다.
- 기기에서 계정을 제거하더라도 네이버웍스 계정은 없어지지 않으며, 언제든지 네이버웍스에 다시 로그인해서 사용할 수 있습니다.

### 모바일 앱

1. '홈'에서 왼쪽 상단의 소속 회사/단체의 이름을 탭합니다.
2. 오른쪽 상단에서 '관리'를 선택합니다.
3. 계정 하단의 '이 기기에서 제거'를 탭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
