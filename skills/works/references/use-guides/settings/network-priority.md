# 사용 네트워크 우선 순위 설정

Android 기기에서는 앱에서 사용되는 모든 데이터에 대해 네트워크의 사용 우선 순위를 설정할 수 있습니다.

- Wi-Fi 우선: 모바일 데이터와 Wi-Fi를 모두 사용할 수 있는 환경에서는 Wi-Fi가 우선 사용됩니다.
- 모바일 데이터 우선: 모바일 데이터와 Wi-Fi를 모두 사용할 수 있는 환경에서는 모바일 데이터가 우선 사용됩니다.

기본 설정은 'Wi-Fi 우선'입니다.

설정 변경 후에는 앱을 재실행해야 합니다.

기기의 모든 계정에 설정 내용이 적용됩니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](https://help.worksmobile.com//wp-content/uploads/2024/08/ic32_Setting.png)아이콘을 탭합니다.
3. '사용 네트워크 우선순위' 에서 설정합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
