# 로그인 및 보안

로그인 방법을 선택할 수 있습니다.

네이버웍스에서 제공하는 서비스의 모바일 앱에 로그인하면, 해당 앱들 간 계정이 공유됩니다.

단, iOS에서 클로바노트 앱은 계정이 공유되지 않습니다.

이전에 다른 네이버웍스 관련 모바일 앱에 로그인한 이력이 있다면, 해당 계정으로 로그인할 수 있습니다.

모든 앱에서 로그아웃된 상태라면, 공유 계정으로 로그인 할 때 비밀번호를 한 번 더 입력해야 합니다.

만약 ID, 비밀번호로 로그인하지 않았다면 '다른 방법으로 로그인'을 선택해 사용 중인 로그인 방식으로 다시 로그인해주세요.

계정을 삭제하려면 계정 목록에서 x 아이콘을 눌러 삭제합니다.

네이버웍스 모바일 앱의 홈 화면 및 로그아웃 후 표시되는 계정 목록에서 계정을 삭제해도, 로그인 시 선택하는 앱 간 계정 공유 목록에서는 삭제되지 않습니다.

# ID/비밀번호로 로그인

네이버웍스의 ID와 비밀번호로 로그인할 수 있습니다.

## ID/비밀번호 등록

네이버웍스에 가입할 때 ID를 등록하지 않은 경우, 1회에 한해 직접 ID와 비밀번호를 등록합니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '로그인 및 보안'을 탭한 후 'ID 및 비밀번호 변경'을 탭합니다.
4. 등록할 ID와 비밀번호를 입력하고 '변경하기'를 탭합니다.

### PC 웹

1. 화면 오른쪽 상단의 프로필 사진을 클릭합니다.
2. 메뉴에서 '보안'을 클릭합니다.
3. '로그인 정보'에서 'ID 변경'을 클릭합니다.
4. 등록할 ID와 비밀번호를 입력하고 '변경하기'를 클릭합니다.

[비밀번호는 이후에도 직접 변경](/ko/use-guides/settings/security/change-password/)할 수 있으나, 한번 등록을 마친 ID는 변경하려면 [관리자에게 요청](/ko/admin-guides/manage-members/members/update-member-information/#i-2)해야 합니다.

# 휴대폰 번호로 로그인

미리 휴대폰 번호를 네이버웍스 계정과 연결하면 다음부터는 휴대폰 번호를 통해 일회용 인증번호를 받아서 로그인할 수 있습니다.

이 기능은 관리자가 [로그인 정책](/ko/admin-guides/security/account-security/#i-2)에서 허용한 경우에만 사용할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '로그인 및 보안'에서 '휴대폰 번호로 로그인'을 켭니다.
4. 연결할 휴대폰 번호를 입력하고, '인증'을 탭합니다. 내 개인정보에 등록된 번호를 연결하거나, 새로운 번호로 연결할 수 있습니다.
5. 문자 메시지로 전송된 인증번호를 입력한뒤 '확인'을 탭합니다. 만약 이미 다른 계정과 연결된 휴대폰 번호라면, 다른 계정과의 연결을 해제하고 현재 로그인된 계정으로 연결할 수 있습니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '보안'을 클릭합니다.
3. '로그인 설정' 영역의 '휴대폰 번호로 로그인' 설정에서 '연결하기'를 클릭합니다.
4. 연결할 휴대폰 번호를 입력하고, '인증'을 클릭합니다. 내 개인정보에 등록된 번호를 연결하거나, 새로운 번호로 연결할 수 있습니다.
5. 문자 메시지로 전송된 인증번호를 입력한뒤 '확인'을 클릭합니다. 만약 이미 다른 계정과 연결된 휴대폰 번호라면, 다른 계정과의 연결을 해제하고 현재 로그인된 계정으로 연결할 수 있습니다.

- 개인정보에 등록된 휴대폰 번호를 연결하거나, 새로운 번호로 연결할 수 있습니다.
- 만약 다른 번호로 연결했다면, 이 번호는 네이버웍스 서비스에서 로그인 시 이외에는 노출되거나 사용되지 않습니다(ID/비밀번호 찾기에는 개인정보에 등록된 휴대폰 번호가 사용됩니다).
- 설정을 완료한 다음 로그인 시점에 만약 등록한 휴대폰 번호로 문자 메시지를 받을 수 없는 상황이라면, '서비스 알림 메시지로 인증번호 받기(가명)'를 탭하여 서비스 알림으로 받을 수 있습니다.

# 네이버로 로그인

네이버웍스 계정을 네이버 아이디와 연동하면, 다음부터는 '연결된 네이버 ID로 로그인'을 통해 ID와 비밀번호를 입력하지 않고 쉽게 로그인할 수 있습니다.

이 기능은 관리자가 [로그인 정책](/ko/admin-guides/security/account-security/#i-2)에서 허용한 경우에만 사용할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '로그인 및 보안'에서 '연결된 네이버 ID로 로그인'을 켭니다.
4. 연동할 네이버 아이디로 로그인하고 연동을 완료합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '보안'을 클릭합니다.
3. '로그인 설정' 영역의 '네이버로 로그인' 설정에서 '연결하기'를 클릭합니다.
4. 연동할 네이버 아이디로 로그인하고 연동을 완료합니다.

네이버 아이디는 로그인할 네이버웍스 계정을 인증하기 위해서만 사용됩니다.

- 연동 시 개인 정보 제공 동의 화면에서 개인 이메일 또는 휴대폰 번호 제공에 동의하면 개인 이메일과 휴대폰 번호가 [네이버웍스 프로필](/ko/use-guides/settings/personal/edit-my-profile/)에 저장됩니다.
- 저장된 개인 이메일 또는 휴대폰 번호로 아이디 또는 비밀번호를 찾을 수 있습니다.

# 휴대폰 잠금해제로 로그인

네이버웍스 앱이 설치된 모바일 기기를 신뢰하는 기기로 추가해 모바일 기기의 잠금해제 방식(지문, 패턴, 얼굴인식 등)으로 네이버웍스에 로그인할 수 있습니다.

단, 신뢰하는 기기로 추가한 모바일 기기의 잠금해제 방식(지문, 패턴, 얼굴인식 등)이 전부 비활성화된 경우 휴대폰 잠금해제로 로그인할 수 없습니다.

이 기능은 관리자가 [로그인 정책](/ko/admin-guides/security/account-security/#i-2)에서 허용한 경우에만 사용할 수 있습니다.

## 모바일 기기를 내 신뢰하는 기기로 추가하기

지금 사용하고 있는 모바일 기기를 내 신뢰하는 기기로 추가합니다.

하나의 모바일 기기를 최대 5개의 계정에 신뢰하는 기기로 추가할 수 있습니다.

단, 내 신뢰하는 기기로 추가하기 전에 모바일 기기에 잠금해제 방식(지문, 패턴, 얼굴인식 등)이 설정되어 있어야 합니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '로그인 및 보안'을 탭한 후 '휴대폰 잠금해제로 로그인 (FIDO)'을 켭니다.
4. '추가하기'를 탭한 후, 기기에 설정된 잠금해제 방식(지문, 패턴, 얼굴인식 등)으로 본인 인증을 진행합니다.
5. 기기 이름을 등록하고 '추가하기'를 탭합니다.

## 모바일 기기에서 로그인하기

신뢰하는 기기에서 로그아웃 되었을 때 다시 로그인합니다.

### 모바일 앱

1. 로그인하려는 계정을 탭합니다.
2. 기기에 설정된 잠금해제 방식(지문, 패턴, 얼굴인식 등)으로 로그인합니다.

## PC에서 모바일 기기로 로그인하기

PC웹이나 PC앱, 드라이브 탐색기에 로그인할 때 신뢰하는 모바일 기기로 인증해 로그인할 수 있습니다.

### ID로 로그인하기

### PC

1. 첫 로그인 화면에서 ID를 입력합니다.
2. 비밀번호를 입력하는 화면에서 '내 모바일 기기로 인증'을 클릭합니다. 만약 신뢰하는 기기가 추가되어있지 않다면, 이 버튼이 나타나지 않습니다.
3. 모바일 기기로 발송된 '지금 PC에서 로그인하고 있나요?' 알림을 탭합니다.
4. PC 화면에서 표시되는 두 자리 숫자와 모바일앱에서 표시되는 숫자가 일치하는지 확인한 뒤, 모바일앱에서 'PC에 로그인하기'를 탭해 PC에 로그인합니다.

### 휴대폰 번호로 로그인하기

### PC

1. 첫 로그인 화면에서 휴대폰 번호를 입력합니다.
2. 모바일 기기로 발송된 '지금 PC에서 로그인하고 있나요?' 알림을 탭합니다.
3. PC 화면에서 표시되는 두 자리 숫자와 모바일앱에서 표시되는 숫자가 일치하는지 확인한 뒤, 모바일앱에서 'PC에 로그인하기'를 탭해 PC에 로그인합니다.

# 2단계 인증

2단계 인증이란, 로그인 시 개인정보에 등록된 인증수단을 통해 일회용 인증번호를 받는 추가적인 인증 단계를 거치는 기능입니다.

2단계 인증을 활성화하면 비밀번호 유출이나 외부의 계정 탈취 시도로부터 계정을 안전하게 보호할 수 있습니다.

## 2단계 인증 활성화하기

- 일회용 인증번호를 받을 개인이메일이나 휴대폰 번호 정보를 등록하면 2단계 인증이 활성화됩니다.
- 2단계 인증을 활성화한 다음부터는 로그인 시 매번 2단계 인증을 진행하게 됩니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '로그인 및 보안'을 탭합니다.
4. '2단계 인증'을 켭니다.
5. 2단계 인증에서 사용될 개인이메일과 휴대폰 정보를 입력합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '보안'을 클릭합니다.
3. '2단계 인증'을 켭니다.
4. 2단계 인증에서 사용될 개인이메일과 휴대폰 정보를 입력합니다.

사내 시스템과 개인 정보가 연동된 경우 등 서비스 이용 환경에 따라, 2단계 인증 수단을 직접 등록할 수 없는 경우가 있습니다.

이러한 경우에는 관리자에게 요청하거나 네이버웍스와 연동된 사내 시스템에 개인정보를 등록해야 합니다.

## 2단계 인증 비활성화하기

- 관리자가 2단계 인증을 '선택 - 구성원 설정에 따름 '으로 설정한 경우에는 언제든지 2단계 인증을 비활성화할 수 있습니다.
- [관리자가 2단계 인증을 '필수 - 모든 구성원에게 2단계 인증 적용'으로 설정](/ko/admin-guides/security/account-security/#i-2)한 경우에는 반드시 2단계 인증을 진행해야하며, 2단계 인증을 비활성화할 수 없습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '로그인 및 보안'을 탭합니다.
4. '2단계 인증'을 끕니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '보안'을 클릭합니다.
3. '2단계 인증'을 끕니다.

# 모바일앱 간편로그인

모바일앱에 한번 로그인한 다음 간편로그인을 설정하면 다음부터 ID와 비밀번호를 입력하는 대신 PIN을 입력하거나 생체인증으로 빠르게 로그인할 수 있습니다.

단, 사내 이용 환경에 따라 이 설정이 노출되지 않을 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '로그인 및 보안'을 탭합니다.
4. '간편 로그인'의 'PIN으로 로그인'을 켭니다.
5. 간편 로그인에 사용할 PIN을 설정합니다.
   - PIN이 등록되지 않은 경우, 새로 등록할 PIN을 입력하고 한번 더 확인하면 새 PIN이 등록되면서 간편로그인 설정이 활성화됩니다.
   - 이미 '화면 잠금'이 설정된 상태라면 동일한 PIN을 사용하기 때문에 PIN을 확인하면 간편로그인 설정이 활성화됩니다.
6. PIN 등록이 완료되면 '생체 인증 사용'을 설정할 수 있습니다.
   - '생체 인증 사용'을 켜면 PIN을 입력하는 대신 디바이스에 등록된 생체 인증 수단(지문, 얼굴인식 등 기기에서 제공하는 방식)으로 로그인할 수 있습니다.
   - '생체 인증 사용'을 끄면 PIN입력으로만 로그인할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
