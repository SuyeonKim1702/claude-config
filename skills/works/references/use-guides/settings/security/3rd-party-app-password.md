# 외부 앱 비밀번호

외부 앱 비밀번호란 CalDAV, IMAP, SMTP, POP3 등 외부 앱에 네이버웍스 계정으로 로그인할 때 사용하는 외부 앱 전용 비밀번호입니다.

외부 앱 비밀번호는 무작위의 알파벳과 숫자의 조합으로 생성되며, 외부 앱 비밀번호로 외부 앱에 로그인하면 네이버웍스 계정을 안전하게 보호할 수 있습니다.

계정 탈취를 방지하기 위해, 2023년 2월 16일부터 네이버웍스에서는 외부 앱 비밀번호로만 외부 앱에 로그인할 수 있습니다.

# 외부 앱 비밀번호 생성하기

외부 앱 비밀번호를 사용하는 경우, 외부 앱에 로그인을 하려면 비밀번호를 생성해야 합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '보안'을 선택합니다.
3. '외부 앱 비밀번호 생성하기'를 클릭합니다.
4. '새 비밀번호 생성'을 클릭합니다.
5. 로그인할 외부 앱 이름을 입력하고 '비밀번호 생성'을 클릭합니다.
6. 생성된 비밀번호를 복사하고, 로그인할 외부 앱의 비밀번호 입력창에 입력 후 로그인합니다.

외부 앱 비밀번호는 5개까지 생성할 수 있습니다.

한번 생성한 비밀번호는 여러 번 사용할 수 있지만, 계정 탈취를 방지하기 위해 재사용하지 않는 것을 권장합니다.

외부 앱 비밀번호를 생성하면 메시지로 알림이 전송됩니다. 직접 생성한 외부 앱 비밀번호가 아니라면 즉시 생성된 비밀번호를 삭제하고 네이버웍스 비밀번호를 변경하는 것을 권장합니다.

# 외부 앱 비밀번호 삭제하기

더 이상 사용하지 않는 외부 앱 비밀번호이거나 계정 탈취가 의심되는 경우, 외부 앱 비밀번호를 삭제해야 합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '보안'을 클릭합니다.
3. '외부 앱 비밀번호 생성하기'를 클릭합니다.
4. 삭제하려는 외부 앱 비밀번호의 '관리' 항목에서 '삭제'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
