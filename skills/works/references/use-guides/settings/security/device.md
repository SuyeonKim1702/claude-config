# 디바이스 관리

신뢰하는 기기로 추가한 모바일 기기를 포함해 네이버웍스 모바일앱과 PC앱, 드라이브 탐색기에 접속한 모든 디바이스 정보를 확인하고 관리할 수 있습니다.

# 내 기기 확인하고 관리하기

'[휴대폰 잠금해제로 로그인 (FIDO)](/ko/use-guides/settings/security/login-security-settings/#i-5)'을 활성화해 신뢰하는 모바일 기기 목록을 확인하고 관리할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. 메뉴에서 '디바이스'를 탭합니다.
4. '신뢰하는 기기'에서 내 모바일 기기를 확인합니다. 내 모바일 기기를 관리하려면 기기 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-more-767676.png) 아이콘을 탭합니다.
   - 이름 변경: 기기명을 수정합니다.
   - 기기 삭제: 더 이상 해당 기기에서 생체인증으로 로그인할 수 없도록 신뢰하는 기기를 삭제합니다.
   - 대표 기기로 설정: 해당 기기를 대표 기기로 변경합니다. [내 모바일 기기로 PC웹이나 PC앱, 드라이브 탐색기에서 로그인](/ko/use-guides/settings/security/login-security-settings/#i-5)할 때, 대표 기기에서 로그인 요청을 승인하여 로그인할 수 있습니다.
   - 인증 기록: 해당 기기로 인증한 기록을 확인합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '디바이스'를 클릭합니다.
3. '신뢰하는 기기'에서 내 모바일 기기를 확인합니다. 내 모바일 기기를 관리하려면 기기 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-more-767676.png) 아이콘을 클릭합니다.
   - 이름 변경: 기기명을 수정합니다.
   - 기기 삭제: 더 이상 해당 기기에서 생체인증으로 로그인할 수 없도록 신뢰하는 기기를 삭제합니다.
   - 대표 기기로 설정: 해당 기기를 대표 기기로 변경합니다. [내 모바일 기기로 PC웹이나 PC앱, 드라이브 탐색기에서 로그인](/ko/use-guides/settings/security/login-security-settings/#i-5)할 때, 대표 기기에서 로그인 요청을 승인하여 로그인할 수 있습니다.
   - 인증 기록: 해당 기기로 인증한 기록을 확인합니다.

# 접속한 모든 기기 확인하기

네이버웍스 모바일앱과 PC앱, 드라이브 탐색기에 접속한 모든 디바이스의 정보를 확인할 수 있습니다.

각 디바이스에 설치된 앱이 최신 버전이 아닐 경우, 업데이트 알림을 보내서 최신 버전의 앱으로 업데이트하도록 독려할 수 있습니다.

또한, 업무용 모바일 디바이스를 분실해 정보 탈취가 우려될 때 [모바일 앱 데이터 삭제 기능](/ko/use-guides/settings/delete-mobile-app-data/)으로 원격으로 데이터를 삭제할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. 메뉴에서 '디바이스'를 탭합니다.
4. 특정 디바이스 정보를 확인합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '디바이스'를 클릭합니다.
3. 특정 디바이스 정보를 확인합니다.

'디바이스'에서 확인할 수 있는 항목은 다음과 같습니다.

|  |  |
| --- | --- |
| 모델(디바이스 이름) | - 디바이스의 모델명이 표시됩니다. - 이 정보를 누르면 디바이스의 더 상세한 정보를 확인하고 앱 데이터를 삭제할 수 있습니다. |
| 디바이스 OS(버전) | - 디바이스 운영체제의 종류와 버전이 표시됩니다. |
| Vendor ID | - 디바이스에서 생성하는 고유한 값으로 디바이스에서  노출되지는 않습니다. 디바이스를 초기화하면 변경되지만, 네이버웍스 앱을 삭제하거나 재설치해도 변경되지 않습니다. - 외부 MDM을 사용하는 디바이스에 대해 확인할 수 있습니다. |
| 설치 앱 | - 이 기기에서 마지막으로 접속한 네이버웍스 앱과 버전이 표시됩니다. - 모바일앱이 최신 버전이 아닌 경우, 서비스 알림으로 업데이트 안내를 보낼 수 있습니다. |
| 최초 접속 | - 이 디바이스에서 해당 사용자가 최초 접속한 일자와 시각이 표시됩니다. |
| 최근 접속 | - 이 디바이스에서 해당 사용자가 최근 접속한 일자와 시각이 표시됩니다. - 이 정보는 모바일앱에 다시 로그인한 경우이거나, 마지막으로 접속한 지 24시간이 지난 경우에 다시 실행할 경우에 접속 시각이 업데이트됩니다. |

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
