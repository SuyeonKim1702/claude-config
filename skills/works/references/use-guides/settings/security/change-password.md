# 비밀번호 변경

나의 네이버웍스 ID를 확인하고, 비밀번호를 변경할 수 있습니다.

단, 관리자의 설정에 따라 '로그인 및 보안' 영역이 노출되지 않을 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '로그인 및 보안'을 탭한 후 '비밀번호 변경'을 탭합니다.
4. 현재 비밀번호와 새 비밀번호를 입력한 후 '변경하기'를 탭합니다.

### PC 웹

1. 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '보안'을 클릭합니다.
3. '로그인 정보'에서 '비밀번호' 항목 오른쪽의 '비밀번호 변경'을 클릭합니다.
4. 현재 비밀번호와 새 비밀번호를 입력한 후 '변경하기'를 클릭하여 비밀번호를 변경합니다.

[간편 로그인](/ko/use-guides/settings/security/login-security-settings/#i-5) 시 사용하는 PIN의 변경은 [PIN 변경하기](/ko/use-guides/settings/security/passcode/#i-3)를 참고하세요.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
