# 접속 현황 관리

본인 계정의 최근 90일간의 접속 현황을 확인할 수 있습니다. 접속 현황에서 확인할 수 있는 항목은 다음과 같습니다.

- 접속 시간
- 접속 환경
- 위치(IP 주소)
- 관리

# 접속 현황 확인하기

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '접속현황'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '접속 현황'을 클릭합니다.

# 접속 환경 로그아웃하기

접속 현황에서 본인이 로그인 하지 않은 환경이 있다면 로그아웃 할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '접속현황'을 탭합니다.
4. 로그아웃할 환경 오른쪽의 '로그아웃'을 탭합니다.
5. '확인'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '접속 현황'을 클릭합니다.
3. 로그아웃할 환경 오른쪽의 '로그아웃'을 클릭합니다.
4. '확인'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
