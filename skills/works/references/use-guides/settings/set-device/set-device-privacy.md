# 기기의 사진 접근 허용 설정

네이버웍스 모바일앱에서 사진 및 동영상을 전송하기 위해서는 앱이 사진 라이브러리에 접근하도록 허용해야 합니다.

# iOS 기기의 사진 라이브러리 접근 허용 설정하기

### 모바일 앱

1. 앱에서 사진 및 동영상을 처음 전송하거나 첨부할 때, 사진에 접근할 수 있도록 허용해야 합니다.
   - - iOS 13 및 이전: 팝업창에서 '승인'을 탭합니다.
     - iOS 14 및 이후: 팝업창에서 접근 허용 범위를 '사진 선택...'과 '모든 사진에 대한 접근 허용' 중에서 선택합니다.

# iOS 기기의 사진 라이브러리 접근 권한 다시 설정하기

만약 접근 권한을 다시 설정하고 싶다면 기기의 '설정'에서 변경해야 합니다.

### 모바일 앱

2. '사진 접근 허용'에서 접근 권한을 변경합니다.
3. iOS 14 및이후의 환경에서는 접근 허용 범위를 '선택한 사진'과 '모든 사진' 중에서 선택할 수 있습니다.
