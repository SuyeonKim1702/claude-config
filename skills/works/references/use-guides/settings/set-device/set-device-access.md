# 마이크, 카메라, 화면 기록 접근 허용 설정

네이버웍스 앱에서 음성ㆍ 영상 통화 및 화면공유를 이용하기 위해서는 마이크, 카메라, 화면 기록 접근을 허용해야 합니다.

### 모바일 앱

### iOS 기기의 마이크, 카메라 접근 허용 설정하기

1. 음성ㆍ 영상 통화를 첫 사용 시 접근 권한을 묻는 팝업에서 '확인'을 탭합니다.
2. 만약 '허용 안 함'을 선택한 후 다시 접근 권한을 허용하고 싶다면 기기의 설정에서 허용해야 합니다.
   1. 기기의 설정으로 이동합니다.
   2. 목록에서 '네이버웍스'를 탭합니다.
   3. '네이버웍스 접근 허용'에서 '마이크'와 '카메라'를 ON으로 설정합니다.

### 안드로이드 기기의 마이크, 카메라 접근 허용 설정하기

1. 음성ㆍ 영상 통화를 첫 사용 시 접근 권한을 묻는 팝업에서 '허용'을 탭합니다.
2. 만약 접근 권한을 거부한 후 다시 허용하고 싶다면, 기기의 '설정'에서 변경해야 합니다.
   2. 앱 정보에서 '권한'을 선택합니다.
   3. '마이크', '전화, '카메라'의 액세스를 허용합니다.

### PC 앱

### Windows 기기의 마이크, 카메라 접근 허용 설정하기

2. '마이크', '카메라' 메뉴로 이동합니다.
3. 해당 페이지에서 마이크 및 카메라 액세스를 모두 허용합니다.
4. 앱을 종료한 후 재실행하여 사용합니다.

### Mac 기기의 마이크, 카메라 접근 허용 설정하기

1. 음성ㆍ 영상 통화를 첫 사용 시 접근 권한을 묻는 팝업에서 '확인'을 클릭합니다.
2. 만약 '허용 안 함'을 선택한 후 다시 접근 권한을 허용하고 싶다면 기기의 설정에서 허용해야 합니다.
   2. 마이크와 카메라 목록에서 '네이버웍스'를 ON으로 설정합니다.
   3. 앱을 종료한 후 재실행하여 사용합니다.

### Mac 기기의 화면 기록 허용 설정하기

1. 화면공유 첫 사용 시 접근 권한을 묻는 팝업에서 '시스템 환경설정 열기'를 클릭합니다.
3. 만약 '거부'를 선택한 후 다시 접근 권한을 허용하고 싶다면 기기의 설정에서 허용해야 합니다.
   2. 화면 기록 목록에서 '네이버웍스'를 ON으로 설정합니다.
   3. 앱을 종료한 후 재실행하여 사용합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
