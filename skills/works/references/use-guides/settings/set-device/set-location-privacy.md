# 위치정보 접근 허용 설정

네이버웍스 앱에서 [현재 위치를 보내기](/ko/use-guides/message/send-message/send-current-location/) 위해서는 위치정보 이용 동의를 해야 합니다.

### 모바일 앱

1. 앱에서 위치정보를 처음 전송하거나 첨부할 때, 아래의 화면에서 '동의합니다.'를 탭합니다.

위치를 메시지로 보내거나 일정에 첨부된 지도를 확인하는 등 위치정보를 활용하는 기능을 사용하기 위해서는 먼저 앱이 위치정보에 접근할 수 있도록 허용해야 합니다.

# iOS 기기의 위치 접근 허용 설정하기

### 모바일 앱

1. 위치정보를 사용하는 기능을 처음 실행할 때, 접근 권한 확인을 요청하는 팝업이 표시됩니다.
2. 위치정보 사용을 '한 번 허용'할 지, '앱을 사용하는 동안 허용'할 지 선택합니다.

# 안드로이드 기기의 위치 접근 허용 설정하기

### 모바일 앱

1. 위치 정보를 사용하는 기능을 처음 실행할 때, 접근 권한 확인을 요청하는 아래의팝업에서 '허용'을 탭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
