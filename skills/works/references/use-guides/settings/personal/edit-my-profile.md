# 내 프로필 수정

내 프로필에서 아래 항목을 신규로 등록하거나 수정할 수 있습니다.

- 프로필 사진
- 다국어 이름
- 닉네임
- 사내번호
- 휴대폰
- 개인 이메일
- 근무처
- 담당 업무
- 메신저/SNS
- 생일
- 연결된 연락처 (PC웹에서만 수정 가능)
- 그 외 관리자가 추가한 항목

단, ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-12-private-555555.png) 아이콘이 표시된 이름, 직책/직급, 소속회사/부서, 회사 이메일은 [관리자만 수정](/ko/admin-guides/manage-members/members/update-member-information/)할 수 있습니다.

# 내 프로필 수정하기

내 프로필을 수정할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 내 이름이나 프로필 사진을 탭합니다.
3. '내 프로필' 화면에서 '수정'을 탭합니다.
4. 신규로 등록하거나 수정하고자 하는 항목을 변경합니다.
5. '저장'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '개인정보'를 클릭합니다.
3. '프로필'에서 '내 프로필'을 클릭하여 신규로 등록하거나 수정하고자 하는 항목을 변경합니다.
4. '저장'을 클릭합니다.

# 내 프로필 사진 변경

내 프로필 사진과 상태 등을 확인하고 설정할 수 있습니다.

단, 관리자의 설정에 따라 사진 변경이 불가할 수 있으며, 이 경우 카메라 아이콘이 노출되지 않습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 내 이름이나 프로필 사진을 탭합니다.
3. '내 프로필' 화면에서 '수정'을 탭합니다.
4. ![](/wp-content/uploads/2024/04/icon-change-profile-image.png) 아이콘을 탭합니다.
5. 기기에 있는 사진을 사용할 경우 '사진 선택', 카메라로 바로 촬영할 경우 '사진 촬영'을 탭해서 프로필 사진을 변경합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '개인정보'를 클릭합니다.
3. 프로필 사진의  ![](/wp-content/uploads/2024/04/setting_pcweb_camera_icon.png) 아이콘을 클릭 후 '변경'을 클릭합니다.
4. 변경하고자 하는 프로필 사진을 선택합니다. (프로필 사진은 용량이 10MB 이하인 파일만 등록할 수 있습니다.)
5. 페이지 하단의 '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 하단 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '기본 설정' 탭에서 ![](/wp-content/uploads/2024/04/setting_pcapp_camera_icon.png) 아이콘을 클릭하여 사진을 선택합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
