# 외부 사용자와 대화

LINE 사용자나 외부 네이버웍스 사용자와 대화를 하기 위해 필요한 QR 코드, ID 등의 정보를 확인하고, 대화 상대를 추가할 수 있습니다.

외부 네이버웍스 사용자가 내 휴대폰 번호를 검색할 수 있도록 설정할 수도 있습니다.

단, 이 영역은 관리자가 [LINE 사용자와의 대화 또는 외부 네이버웍스 사용자와의 대화를 허용](/ko/admin-guides/manage-service/message/message-with-external-users/)한 경우에만 노출됩니다.

# 휴대폰 번호로 나를 찾도록 허용하기

외부 네이버웍스 사용자가 내 휴대폰 번호를 검색해 나를 대화 상대로 추가하거나 나에게 메시지를 보낼 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '외부 사용자와 대화'를 탭합니다.
4. '휴대폰 번호로 나를 찾도록 허용'을 켭니다.
   - 계정에 등록된 휴대폰 번호가 없으면 본인 휴대폰 번호 인증을 합니다. 인증을 마치면 '휴대폰 번호로 나를 찾도록 허용'이 켜집니다.

### PC 웹

1. 화면 오른쪽 상단의 내 프로필 이미지를 클릭합니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '휴대폰 번호로 나를 찾도록 허용'을 켭니다.
   - 계정에 등록된 휴대폰 번호가 없으면 본인 휴대폰 번호 인증을 합니다. 인증을 마치면 '휴대폰 번호로 나를 찾도록 허용'이 켜집니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '외부 사용자와 대화' 탭을 클릭합니다.
4. '휴대폰 번호로 나를 찾도록 허용'에서 '허용하러 가기'를 클릭합니다.
5. PC웹으로 이동한 후 '휴대폰 번호로 나를 찾도록 허용'을 켭니다.
   - 계정에 등록된 휴대폰 번호가 없으면 본인 휴대폰 번호 인증을 합니다. 인증을 마치면 '휴대폰 번호로 나를 찾도록 허용'이 켜집니다.

# 내 ID 보내기

LINE 또는 외부 네이버웍스 사용자에게 내 ID를 공유할 수 있습니다.

LINE 또는 외부 네이버웍스 사용자는 공유받은 ID를 검색해 나를 대화 상대로 추가하거나 나에게 메시지를 보낼 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 내 이름이나 프로필 사진을 탭합니다.
3. 'ID'에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_copy.png) 아이콘을 클릭하여 복사합니다.
4. 복사한 ID를 SMS나 메일로 보내서 나를 친구로 추가할 수 있게 합니다.

### PC 웹

1. 화면 오른쪽 상단의 내 프로필 이미지를 클릭합니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '외부 사용자와 대화' 메뉴의 '내 메시지 ID'에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_copy.png) 아이콘을 클릭하여 복사합니다.
4. 복사한 '내 메시지 ID'를 원하는 대화상대에게 보냅니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '외부 사용자와 대화' 탭을 클릭합니다.
4. '내 메시지 ID'에서 ID를 복사하여 원하는 대화 상대에게 보냅니다.

# 대화 상대 추가 URL 보내기

대화 상대 추가 URL을 전송하면 상대방이 URL을 눌러 LINE 또는 네이버웍스에서 나를 대화 상대로 추가할 수 있습니다.

### PC 웹

1. 화면 오른쪽 상단의 내 프로필 이미지를 클릭합니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '외부 사용자와 대화' 메뉴의 '대화 상대 추가 URL'에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_copy.png) 아이콘을 클릭합니다.
4. 복사된 '대화 상대 추가 URL'을 원하는 대화상대에게 보냅니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '외부 사용자와 대화' 탭을 클릭합니다.
4. '대화 상대 추가 URL'에서 URL을 복사하여 원하는 대화 상대에게 보냅니다.

# QR 코드 보내기

내 QR 코드를 보내면, 상대방이 LINE 또는 네이버웍스로 스캔하여 나의 연락처를 추가할 수 있습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 내 이름이나 프로필 사진을 탭합니다.
3. 'QR 코드'를 탭합니다.
4. QR 코드를 바로 보여주거나, '저장'을 탭하여 이미지로 저장하여 보냅니다.

### PC 웹

1. 화면 오른쪽 상단의 내 프로필 이미지를 클릭합니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '외부 사용자와 대화' 메뉴의 QR 코드'에서 'QR 코드 저장'을 클릭하여 PC에 저장하여 보냅니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '외부 사용자와 대화' 탭을 클릭합니다.
4. 'QR 코드'에서 '저장'을 클릭하여 PC에 저장하여 전송합니다.

# 대화 상대 추가하기

LINE 사용자 또는 외부 네이버웍스 사용자에게 초대 링크나 QR코드를 공유해 대화 상대로 추가할 수 있습니다.

## 검색으로 네이버웍스 대화 상대 추가하기

다른 회사의 네이버웍스 사용자의 메시지 ID 또는 휴대폰 번호를 검색하여 대화 상대로 추가하거나, 바로 메시지를 보낼 수 있습니다.

### PC 웹

1. 화면 오른쪽 상단의 내 프로필 이미지를 클릭합니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '외부 사용자와 대화' 메뉴의 'NAVER WORKS 대화 상대 추가'에서 '메시지 ID/휴대폰 번호로 찾기'를 클릭합니다.
4. '메시지 ID' 또는 '휴대폰 번호'를 선택하고 ID나 휴대폰 번호를 검색합니다.
5. '추가'를 클릭하여 외부 사용자를 주소록에 추가합니다.  
   사용자의 ID 또는 휴대폰 번호를 모를 경우 '메일로 초대하기'를 클릭하여 초대 메일을 전송합니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '외부 사용자와 대화' 탭을 클릭합니다.
4. '대화 상대 추가'에서 '메시지 ID/휴대폰 번호로 찾기'를 클릭합니다.
5. '메시지 ID' 또는 '휴대폰 번호'를 선택하고 ID나 휴대폰 번호를 검색합니다.
6. '대화상대로 추가'를 클릭하여 주소록에 추가하거나 '대화'를 클릭하여 바로 상대방과 대화를 시작합니다.

[차단된 대화상대 관리](/ko/use-guides/message/settings/general/message-setting/manage-blocked-user/)에서 차단된 대화상대를 확인하거나 차단을 해제할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
