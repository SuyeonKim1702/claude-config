# 이미지 명함

이미지 명함은 이미지 형태의 프로필입니다. 이미지 명함으로 온라인 상에서 명함을 상대에게 제공할 수 있습니다.

이미지 명함을 활용하면 지인이나 외부 대화 상대와의 대화를 쉽게 시작할 수 있습니다.

'정방형', '가로형', '세로형', '명함' 4가지 유형의 템플릿을 제공합니다.

|  |  |  |
| --- | --- | --- |
| **유형** | **이미지** | **제공 정보** |
| 정방형 |  | 내 프로필 이미지, 이름, 직책, 대표 조직, 회사 로고 또는 회사명, 내 메시지 ID, 대화상대 추가 QR코드 |
| 가로형 |  | 내 프로필 이미지, 이름, 직책, 대표 조직, 회사 로고 또는 회사명, 내 메시지 ID, 대화상대 추가 QR코드 |
| 세로형 |  | 내 프로필 이미지, 이름, 직책, 대표 조직, 회사 로고 또는 회사명, 내 메시지 ID, 대화상대 추가 QR코드 |
| 명함 |  | 이름, 대표 다국어 이름, 직책, 대표 조직, 회사 로고 또는 회사명, 이메일 주소, 휴대폰 번호, 대화상대 추가 QR코드 |

위 정보 중 사용자가 실제 등록한 항목이 표시되나, 회사 로고는 관리자가 미리 등록한 로고가 표시됩니다.

- 정방형, 가로형: [Admin > 모바일앱](/ko/admin-guides/settings/customize/mobile-app/)에 등록된 로고가 표시됩니다.
- 세로형, 명함: [Admin >  PC웹](/ko/admin-guides/settings/customize/pc-web/)에 등록된 로고가 표시됩니다.

단, 이미지 명함은 관리자가 Admin에서 [LINE 사용자 또는 외부 네이버웍스 사용자와의 대화 권한](/ko/admin-guides/manage-service/message/message-with-external-users/)을 허용해야 사용할 수 있습니다.

## 이미지 명함 공유하기

내 이미지 명함을 기기에 저장하거나, 다른 앱으로 공유할 수 있습니다.

단, 공유하기는 모바일앱에서만 가능하며, 관리자가 [모바일앱 파일 다운로드를 제한](/ko/admin-guides/security/file-security/#i)한 경우, 기기에 저장할 수 없습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 내 이름이나 프로필 사진을 탭합니다.
3. '내 프로필' 화면에서 '이미지 명함'을 탭합니다.
4. 정방형, 가로형, 세로형, 명함 중 원하는 유형을 선택합니다.
5. 이미지로 저장하거나 공유합니다.
   - 이미지로 저장: 만들어진 이미지 명함을 파일로 다운로드합니다.
   - 공유하기: 기기에 설치된 다른 앱으로 이미지 명함을 공유합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
