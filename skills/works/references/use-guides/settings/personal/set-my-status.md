# 상태 설정

# 상태

내 프로필에 표시할 상태를 설정할 수 있습니다.

상태를 설정하면 조직도, 프로필 및 메시지방 멤버 목록에 상태 아이콘이 표시됩니다.

단, 관리자가 [상태 표시 사용을 허용](/ko/admin-guides/manage-members/status/)한 경우에만 상태 설정을 할 수 있습니다.

사용자가 직접 설정할 수 있는 상태는 다음의 4가지 종류를 기본으로 제공합니다.

아래 종류 이외에도 [관리자가 회사에 필요한 상태를 직접 추가](/ko/admin-guides/manage-members/status/#i-2)할 수 있습니다.

|  |  |
| --- | --- |
|  | **바쁨**  업무에 집중하고 싶을 때, '바쁨' 상태로 설정할 수 있습니다. |
|  | **자리 비움**  잠시 자리를 비우거나 일시적으로 응답하기 어려울 때, '자리 비움' 상태로 설정할 수 있습니다. |
|  | **퇴근**  업무가 종료되어 업무 중이 아닐 때, '퇴근' 상태로 설정할 수 있습니다. |
|  | **부재**  휴가, 출장, 교육, 병가, 휴직 등의 사유로 부재 시, '부재' 상태로 설정할 수 있습니다. |

사용자가 직접 설정하지 않아도 자동으로 내 프로필에 표시되는 상태는 다음의 3가지 종류가 있습니다.

|  |  |
| --- | --- |
|  | **휴직**관리자가 구성원을 휴직으로 처리한 경우 휴직 기간 동안 자동으로 표시됩니다. |
|  | **일정 있음**  네이버웍스 캘린더와 연동되어 사용자에게 일정이 등록되어 있는 경우 해당 시간대에 자동으로 표시됩니다. |
|  | **미접속**  관리자가 구성원을 추가한 후, 구성원이 전달받은 계정 정보로 아직 한번도 로그인하지 않은 경우 자동으로 표시됩니다. |

# 상태 설정

상태를 설정하면 상대방이 나의 현재 상태를 확인할 수 있습니다.  
상태 정보가 표시되는 영역은 다음과 같습니다.

- 프로필
- 메시지방 목록 (1:1 메시지방에만 표시)
- 1:1 메시지방 상단
- 회사 구성원 목록
- VIP 목록

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.  
   또는 오른쪽 상단의 내 프로필 이미지나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭하여 내 프로필을 엽니다.
2. '상태 설정' 또는 현재 내 상태를 탭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. 오른쪽 상단의 '저장'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '상태'를 클릭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 하단의 내 프로필 이미지를 클릭합니다.
2. '상태'에 마우스 포인터를 올린 후 상태 종류를 선택합니다.

## 상태 메시지 설정

상태를 설정할 때 상태 메시지를 남길 수 있습니다.

상태 메시지를 입력하면, 상대방이 내 프로필에서 내가 남긴 상태 메시지를 확인할 수 있습니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.  
   또는 오른쪽 상단의 내 프로필 이미지나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭하여 내 프로필을 엽니다.
2. '상태 설정' 또는 현재 내 상태를 탭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. 상태 메시지 입력 창에 상태 메시지를 입력합니다.  
   상태 메시지는 최대 50자까지 입력할 수 있습니다.
5. 오른쪽 상단의 '저장'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. '상태'를 클릭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. 상태 메시지 입력 창에 상태 메시지를 입력합니다.  
   상태 메시지는 최대 50자까지 입력할 수 있습니다.
5. '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 하단의 내 프로필 이미지를 클릭합니다.
2. '상태'에 마우스 포인터를 올린 후 '상세 설정'을 클릭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. 상태 메시지 입력 창에 상태 메시지를 입력합니다.  
   상태 메시지는 최대 50자까지 입력할 수 있습니다.
5. '적용'을 클릭합니다.

## 상태 상세 설정

상태 설정 시 '상세 설정'을 통해 상태 설정 기간을 선택하거나 미래의 상태를 예약할 수 있습니다. 상태 설정 기간을 선택하면 해당 기간 동안 내 프로필과 조직도에 상태가 표시되고, 기간이 지나면 자동으로 상태 표시가 사라집니다. 또한 상태를 설정한 기간 동안 메일을 받은 경우, 지정한 내용으로 메일이 자동으로 발송되게 할 수 있습니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.  
   또는 오른쪽 상단의 내 프로필 이미지나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭하여 내 프로필을 엽니다.
2. '상태 설정' 또는 현재 내 상태를 탭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. '상세 설정'을 탭합니다.
5. '상태 설정 기간'을 탭하고 '설정 안 함', '1시간', '2시간', '4시간', '오늘까지', '직접 입력' 중 선택합니다. '직접 입력' 선택 시 시작 시간과 종료 시간을 추가적으로 선택할 수 있습니다.
6. '저장'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. '상태'를 클릭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. '상세 설정'을 클릭합니다.
5. '상태 설정 기간'을 클릭하고 '설정 안 함', '1시간', '2시간', '4시간', '오늘까지', '직접 입력' 중 선택합니다. '직접 입력' 선택 시 시작 시간과 종료 시간을 추가적으로 선택할 수 있습니다.
6. '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 하단의 내 프로필 이미지를 클릭합니다.
2. '상태'에 마우스 포인터를 올린 후 '상세 설정'을 클릭합니다.
3. 상태 종류를 선택하고 '상세 설정'을 클릭합니다.
4. '상태 설정 기간'에서 '설정 안 함', '1시간', '2시간', '4시간', '오늘까지', '직접 입력' 중 선택합니다. '직접 입력' 선택 시 시작 시간과 종료 시간을 추가적으로 선택할 수 있습니다.
5. '적용'을 클릭합니다.

## 자동 응답 메일 설정

메일 서비스를 사용한다면 상태를 설정한 기간 동안 수신한 메일에 자동으로 답장할 수 있습니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.  
   또는 오른쪽 상단의 내 프로필 이미지나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭하여 내 프로필을 엽니다.
2. '상태 설정' 또는 현재 내 상태를 탭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. '상세 설정'을 탭합니다.
5. '메일 자동 답장'의 '사내 메일에 답장' 또는 '외부 메일에 답장'을 켭니다.
6. '전체 회신' 또는 '받는이(to)에 내가 있을 경우만 회신'을 선택합니다.
7. 자동 응답 메일 내용을 입력 후 '저장'을 탭합니다.  
   자동 응답 메일 내용은 300자까지 입력 가능합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. '상태'를 클릭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. '상세 설정'을 클릭합니다.
5. '메일 자동 답장'의 '사내 메일에 답장' 또는 '외부 메일에 답장'을 켭니다.
6. '전체 회신' 또는 '받는이(to)에 내가 있을 경우만 회신'을 선택합니다.
7. 자동 응답 메일 내용을 입력 후 '저장'을 클릭합니다.  
   자동 응답 메일 내용은 300자까지 입력 가능합니다.

### PC 앱

1. 화면 왼쪽 하단의 내 프로필 이미지를 클릭합니다.
2. '상태'에 마우스 포인터를 올린 후 '상세 설정'을 클릭합니다.
3. 상태 종류를 선택하고 '상세 설정'을 클릭합니다.
4. '메일 자동 답장'의 '사내 메일에 답장' 또는 '외부 메일에 답장'을 켭니다.
5. '전체 회신' 또는 '받는이(to)에 내가 있을 경우만 회신'을 선택합니다.

# 예약된 상태

상태 설정 시, 기간을 미래로 설정하여 상태 예약을 할 수 있습니다.

미래의 상태를 예약 후 저장하면, 상태 설정 화면 하단에 '예약된 상태'로 표시됩니다.

단, 현재 상태가 설정되어 있는 상황에서 미래의 상태 예약 설정을 하면, 현재 상태는 '설정 안 함' 상태로 변경됩니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.  
   또는 오른쪽 상단의 내 프로필 이미지나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭하여 내 프로필을 엽니다.
2. '상태 설정' 또는 현재 내 상태를 탭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. '상세 설정'을 탭한 후, '상태 설정 기간'을 '직접 입력'으로 선택합니다.
5. '상태 설정 기간'의 시작 일시를 현재 시간 이후로 설정합니다.
6. 오른쪽 상단의 '저장'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. '상태'를 클릭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. '상세 설정'을 클릭한 후, '상태 설정 기간'을 '직접 입력'으로 선택합니다.
5. '상태 설정 기간'의 시작 일시를 현재 시간 이후로 설정합니다.
6. '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 하단의 내 프로필 이미지를 클릭합니다.
2. '상태'에 마우스 포인터를 올린 후 '상세 설정'을 클릭합니다.
3. 상태 설정 화면에서 상태 종류를 선택합니다.
4. '상세 설정'을 클릭한 후, '상태 설정 기간'을 '직접 입력'으로 선택합니다.
5. '상태 설정 기간'의 시작 일시를 현재 시간 이후로 설정합니다.
6. '적용'을 클릭합니다.

# 예약된 상태 설정 변경

'예약된 상태'에서 상태 종류 및 기간, 상태 메시지, 자동 응답 메일 설정을 변경할 수 있습니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.  
   또는 오른쪽 상단의 내 프로필 이미지나 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭하여 내 프로필을 엽니다.
2. '상태 설정' 또는 현재 내 상태를 탭합니다.
3. 상태 설정 화면 하단의 '예약된 상태'에서 상태를 탭합니다.
4. 예약된 상태 화면에서 상태 종류 및 기간, 상태 메시지, 자동 응답 메일 설정을 변경합니다.
5. 오른쪽 상단의 '저장'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. '상태'를 클릭합니다.
3. 상태 설정 화면 하단의 '예약된 상태'에서 '편집'을 클릭합니다.
4. 상태 종류 및 기간, 상태 메시지, 자동 응답 메일 설정을 변경합니다.
5. '저장'을 클릭합니다.

### PC 앱

1. 화면 왼쪽 하단의 내 프로필 이미지를 클릭합니다.
2. '상태'에 마우스 포인터를 올린 후 '상세 설정'을 클릭합니다.
3. 상태 설정 화면 하단의 '예약된 상태'에서 '편집'을 클릭합니다.
4. 상태 종류 및 기간, 상태 메시지, 자동 응답 메일 설정을 변경합니다.
5. '적용'을 클릭합니다.
