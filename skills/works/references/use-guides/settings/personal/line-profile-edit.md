# LINE 프로필 수정

LINE 대화상대와 연동하여 대화하는 경우, LINE 사용자에게 보이는 내 프로필 정보를 수정할 수 있습니다. LINE 프로필에 등록된 정보는 네이버웍스 프로필 정보와는 별개로 LINE 프로필 화면에만 표시됩니다.

LINE 프로필에서 아래 항목을 수정할 수 있습니다.

- 계정 소개
- 전화
- 홈페이지

단, 프로필 사진, 이름, 그룹명은 수정할 수 없고 네이버웍스에 등록된 구성원 정보로 표시됩니다.  
또한, LINE 연동 권한이 있는 구성원에게만 해당 메뉴가 노출됩니다.

# LINE 프로필 수정하기

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. 내 이름이나 프로필 사진을 탭합니다.
3. 'LINE 프로필'을 탭합니다.
4. '수정'을 탭합니다.
5. 수정하고자 하는 항목을 수정합니다.
6. '저장'을 탭합니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '개인정보'를 클릭합니다.
3. '프로필'에서 'LINE 프로필'을 클릭하여 수정하고자 하는 항목을 수정합니다.
4. '저장'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
