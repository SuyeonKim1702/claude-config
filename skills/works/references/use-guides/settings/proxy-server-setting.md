# 프록시 서버 설정

프록시 서버 정보는 로그인하지 않거나 인터넷이 접속할 수 없는 상태에서도 설정할 수 있습니다.

### PC 앱

1. 앱에서 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.  
   아직 로그인하지 않은 경우, 화면 하단 작업표시줄에서 네이버웍스 앱을 찾아 마우스 오른쪽 버튼으로 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '연결' 탭을 클릭합니다.
4. 설정 방식을 선택합니다.
   - OS 설정을 반영하려면 'Windows 설정 적용'에 체크합니다.
   - 직접 입력하려면 '서버 수동 설정'에 체크 후 아래와 같이 프록시 서버의 정보를 입력합니다.
     - 종류: 사용 중인 네트워크 환경에 맞춰 'HTTP/HTTPS'와 'SOCKS'를 선택합니다.
     - 서버: 프록시 서버의 IP주소를 입력합니다.
     - 포트: 프록시 서버에 접속할 수 있는 포트번호를 입력합니다.
     - 사용자: 프록시 서버의 인증 ID를 입력합니다.
     - 암호: 프록시 서버의 인증 암호를 입력합니다.
5. '적용'을 클릭합니다.
6. 프록시 서버 접속이 완료됨을 확인한 후 '확인'을 클릭합니다.
7. 네이버웍스 PC앱이 재시작됨을 확인합니다.

프록시 서버 수동 설정에 실패한 경우, 오류가 있는 설정 항목이 빨간색으로 표시됩니다. 해당하는 접속 정보를 수정하여 '적용'을 클릭합니다.

빨간색 설정 항목이 없더라도 네이버웍스 서버 접속에 실패할 경우 프록시 서버를 이용할 수 없습니다.

# 프록시 서버 설정 해제

### PC 앱

1. 앱에서 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다
3. '연결' 탭을 클릭합니다.
4. '설정 안 함'을 클릭합니다.
5. '적용'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
