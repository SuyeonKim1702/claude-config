# 시간대 설정

네이버웍스에서 사용하는 시간대를 자신이 원하는 시간대로 설정할 수 있습니다. 모바일앱에서 설정한 시간대는 PC웹에도 동일하게 적용됩니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '시간대 설정'을 탭합니다.
4. 시간대를 원하는 도시로 설정합니다.  
   기기에서 시간대 설정을 한 경우, 동일하게 적용됩니다.

### PC 웹

1. 화면 오른쪽 상단 프로필 사진을 클릭합니다.
2. 메뉴에서 '언어·시간대'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
