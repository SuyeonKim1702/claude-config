# 애플리케이션 정보

애플리케이션의 버전 정보를 확인합니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. '도움말 및 지원'에서 '애플리케이션 정보'를 탭합니다.
   - 현재 버전: 현재 설치된 네이버웍스 앱의 버전입니다. 최신 버전과 다를 경우 보안 강화 및 향상된 기능 사용을 위해 최신 버전으로 업데이트하세요.
   - 최신 버전: 현재 공개된 네이버웍스 앱의 최신 버전입니다.
   - 업데이트 기록: 버전별 업데이트 내역입니다.
   - 이용약관: 네이버웍스의 이용약관입니다.
   - 개인정보 처리방침: 네이버웍스의 개인정보 처리방침입니다.
   - 법적 공지

### PC 앱

1. 화면 왼쪽 하단의 ![](/wp-content/uploads/2024/04/icon-message-pc-app-setting.png) 아이콘을 클릭합니다.
2. '애플리케이션 정보'를 클릭합니다.
   - 현재 버전: 현재 설치된 네이버웍스 앱의 버전입니다. 최신 버전과 다를 경우 보안 강화 및 향상된 기능 사용을 위해 최신 버전으로 업데이트하세요.
   - 최신 버전: 현재 공개된 네이버웍스 앱의 최신 버전입니다.
   - 업데이트 기록: 버전별 업데이트 내역입니다.
   - 이용약관: 네이버웍스의 이용약관입니다.
   - 개인정보 처리방침: 네이버웍스의 개인정보 처리방침입니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
