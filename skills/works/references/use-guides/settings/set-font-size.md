# 글자 크기 설정

글자 크기는 기기에 설정된 글자 크기를 사용할지, 앱 자체로 설정할지 선택할 수 있습니다.

- 기기 설정 사용
- 앱내 글자 크기 설정

## 기기 설정 사용

글자 크기를 기기 설정으로 사용하게 되면 앱에서 글자 크기를 조정하지 않고, 기기에 설정된 크기로 보여줍니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. 글자 크기를 탭합니다.
4. 기기 설정 사용을 켜거나 끕니다.

## 글자 크기 설정

원하는 글자 크기로 설정할 수 있습니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. 글자 크기를 탭합니다.
4. 원하는 글자 크기 옵션을 탭합니다.

## 글자 확대/축소 버튼 표시 설정

메일, 게시판, 노트, 설문에서 글자를 확대/축소하는 버튼이 보이도록 설정할 수 있습니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. 글자 크기를 탭합니다.
4. '글자 확대/축소 버튼 표시'에서 원하는 서비스 설정을 켜거나 끕니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
