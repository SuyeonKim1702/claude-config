# 알림 일시 정지

알림 일시 정지를 설정하면 일시적으로 모든 기기의 알림을 끌 수 있습니다.

# 알림 일시 정지 설정

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 프로필 영역을 탭한 후 내 프로필 화면에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2020/06/list_18_alarm_line.png) 아이콘을 탭합니다.  
   또는 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭한 후 '알림 설정'을 탭합니다.
3. '알림 일시 정지'를 탭합니다.
4. 알림을 일시적으로 끌 시간을 선택합니다.

기기 앱 목록 화면에서 네이버웍스 모바일앱 아이콘을 길게 누른 후 '알림 일시 정지'를 탭해 보다 빠르게 알림 일시 정지 설정을 할 수 있습니다.

### PC 웹

1. 서비스별 PC웹 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_menu_setting_767676.png) 아이콘을 클릭한 후 '알림'을 클릭합니다.  
   또는, 화면 오른쪽 상단 프로필 사진을 클릭해서 '알림 일시 정지' 메뉴를 선택합니다.

### PC 앱

1. 화면 왼쪽 하단의 내 프로필 이미지를 클릭합니다.
2. 마우스 포인터를 '알림 일시 정지' 위에 올립니다.
3. 알림을 일시적으로 끌 시간을 선택합니다.

# 알림 일시 정지 상태 확인

현재 알림 일시 정지 상태인지 확인할 수 있습니다.

### 모바일 앱

1. '홈'의 프로필 영역의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2020/06/list_18_alarmoff_fill-e1709085045267.png) 아이콘을 탭합니다.
2. '알림 일시 정지' 오른쪽에 일시 정지가 해제되는 시각을 확인합니다.  
   다른 서비스 화면에서는 오른쪽 상단의 프로필 사진에 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2020/06/list_18_alarmoff_fill-e1709085045267.png) 아이콘이 표시됩니다.

### PC 웹

1. 서비스별 PC웹 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_menu_setting_767676.png) 아이콘을 클릭한 후 '알림'을 클릭합니다.  
   또는, 화면 오른쪽 상단 프로필 사진을 클릭해서 '알림 일시 정지' 메뉴를 선택합니다.

### PC 앱

1. 오른쪽 상단 프로필 영역의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2020/06/list_18_alarmoff_fill-e1709085045267.png) 아이콘을 클릭합니다.
2. 마우스 포인터를 '알림 일시 정지' 위에 올려 알림 일시 정지 해제 시각을 확인합니다.

# 알림 일시 정지 해제

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. 프로필 영역의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2020/06/list_18_alarmoff_fill-e1709085045267.png) 아이콘을 탭합니다.  
   또는 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭한 후 '알림 설정'을 탭합니다.
3. '알림 일시 정지'를 탭합니다.
4. 상단의 '알림 다시 받기'를 탭합니다.

### PC 웹

1. 서비스별 PC웹 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_menu_setting_767676.png) 아이콘을 클릭한 후 '알림'을 클릭합니다.  
   또는, 화면 오른쪽 상단 프로필 사진을 클릭해서 '알림 일시 정지' 메뉴를 선택합니다.

### PC 앱

1. 오른쪽 상단 프로필 영역의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2020/06/list_18_alarmoff_fill-e1709085045267.png) 아이콘을 클릭합니다.
2. 마우스 포인터를 '알림 일시 정지' 위에 올립니다.
3. 상단의 '알림 다시 받기'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
