# 알림 설정

모바일 앱 알림/팝업 알림 수신 여부나 서비스별 알림을 설정할 수 있습니다.

# 모바일 앱 알림/팝업 알림

새 메시지/메일/서비스 알림이 있을 때 모바일에서 모바일 앱 알림, PC 앱과 PC 웹에서 팝업 알림을 받을지 여부를 설정할 수 있습니다.  
사용하는 환경/기기별로 설정을 다르게 적용할 수 있습니다.

- 메시지 알림: 새 메시지가 오면 모바일 앱 알림/팝업 알림을 받습니다.
- 메일 알림:새 메일이 오면 모바일 앱 알림/팝업 알림을 받습니다.
- 서비스 알림: 새 서비스 알림이 오면 모바일 앱 알림/팝업 알림을 받습니다.  
  WORKS앱과 드라이브 앱을 함께 사용 중일 때, 중복 알림을 받지 않으려면 '드라이브 알림'을 끕니다.

단, 모바일의 경우 iOS나 안드로이드 기기에서 먼저 [기기 알림 설정](/ko/use-guides/settings/set-device/set-device-notifiation/)을 해야 합니다. 설정이 되어 있지 않으면 앱 내에서 알림을 켜도 모바일 앱 알림을 받을 수 없습니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '알림 설정'을 탭합니다.
4. '모바일앱 알림' 항목에서 '알림 허용'을 켜거나 끕니다.
5. '알림 허용'을 켠 상태에서 받고 싶은 알림의 종류를 선택합니다.
6. 알림을 켭니다.
7. 모바일 앱 알림에서 자세한 내용을 표시하지 않으려면 '내용 미리보기'를 끕니다.

### PC 웹

1. 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/02/icon-n-20-menu-setting-767676-1.png) 아이콘을 클릭하여, '환경설정' 화면으로 이동합니다.
2. 왼쪽 메뉴에서 '알림'을 클릭합니다.
3. 알림 설정 화면 중 'PC웹 팝업 알림'을 켜거나 끕니다.  
   브라우저에 권한을 허용하지 않았다면, '알림 설정 하기'를 클릭해 권한을 허용합니다.
4. 'PC웹 팝업 알림'을 켠 상태에서 알림 방식과 받고 싶은 알림의 종류를 선택합니다.
   - 알림의 상세 내용을 알림에 표시할지 여부에 따라 '내용 미리 보기' 또는 '내용 숨기기'를 선택합니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_menu_setting_767676.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '알림 설정' 탭을 클릭합니다.
4. '알림 방법' 항목에서 '팝업 알림'을 선택하거나 선택 해제합니다.
5. '팝업 알림'을 켠 상태에서 알림 방식과 받고 싶은 알림의 종류를 선택합니다.
   - 알림에 표시할 내용에 따라 '팝업 알림' 오른쪽 메뉴에서 '발신자' 또는 '발신자+내용'을 선택합니다.

# 서비스별 알림 설정

서비스별로 어떤 종류의 알림을 받을지 설정할 수 있습니다.

내 계정으로 로그인한 모든 환경/기기에 설정이 동일하게 적용됩니다.

### 모바일 앱

1. 하단의 '홈'을 탭합니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '알림 설정'을 탭합니다.
4. '서비스별 알림'을 탭하고 각 서비스별로 알림을 설정합니다.

### PC 웹

1. 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_menu_setting_767676.png) 아이콘을 클릭하여, '환경설정' 화면으로 이동합니다
2. 왼쪽 메뉴 중 '알림'을 클릭합니다.
3. '서비스별 알림' 아래 각 서비스별로 알림을 설정합니다.

### PC 앱

1. 화면 왼쪽 하단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_menu_setting_767676.png) 아이콘을 클릭합니다.
2. '환경설정'을 클릭합니다.
3. '알림 설정' 탭을 클릭합니다.
4. '알림 방법' 항목에서 '팝업 알림'을 선택한 상태인지 확인합니다.
5. '서비스 알림' 오른쪽의 '서비스별 알림' 버튼을 클릭합니다.
6. PC 웹으로 이동하여 '서비스별 알림' 아래 각 서비스별로 알림을 설정합니다.

서비스별 자세한 알림 설정 방법은 각 서비스에서 제공하는 알림 설정 가이드를 참고해주세요.

- 게시판 알림: 게시판별 알림 설정이 가능합니다. 자세한 내용은 [게시판 알림 설정](/ko/use-guides/board/check-post/board-notification/#i) 가이드를 참고해주세요.
- 메시지 알림: PC 웹 팝업 알림 사용 중에 PC 앱과 모바일 앱의 알림 중복 수신 여부를 확인합니다. 자세한 내용은 [메시지 알림 설정](/ko/use-guides/message/settings/set-message-notification/general-notification-setting/) 가이드를 참고해주세요.
- 메일 알림: 메일함별 또는 즐겨 찾는 연락처로부터 수신된 메일의 알림을 설정합니다. 자세한 내용은 [메일 알림 설정](/ko/use-guides/mail/settings/notification/) 가이드를 참고해주세요.
- 캘린더 알림: 약속 관련 알림 및 캘린더별 미리알림을 설정합니다. 자세한 내용은 [캘린더 알림 설정](/ko/use-guides/calendar/calendar-settings/set-event-alert/) 가이드를 참고해주세요.
- 할 일 알림: 할 일 할당/완료/변경, 리마인드 관련 알림을 설정합니다. 자세한 내용은 [할 일 알림 설정](/ko/use-guides/task/task-setting/notification/) 가이드를 참고해주세요.
- 드라이브 알림: 파일 잠금 해제, 멤버 초대 알림을 설정합니다. 자세한 내용은 [드라이브 알림 설정](/ko/use-guides/drive/settings/drive-notification/) 가이드를 참고해주세요.
- 설문 알림: 생성한 설문과 참여한 설문에 대한 알림을 설정합니다. 자세한 내용은 [설문 알림 설정](/ko/use-guides/form/form-notification/) 가이드를 참고해주세요.
- 경영지원: 결재, 근태, 자원 예약에 대한 알림을 설정합니다. 자세한 내용은 경영지원 각 [메뉴별 알림 설정](/ko/use-guides/support-etc/settings/#i-4) 가이드를 참고해주세요.

# 브라우저별 알림 표시 허용하기

브라우저 알림 표시를 차단한 경우에는 웹 팝업 알림을 이용할 수 없습니다.

환경설정은 화면 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2019/08/setting.jpg) 아이콘을 클릭하여 들어갈 수 있습니다. 

## Chrome

브라우저 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭한 후 '설정 > 개인 정보 보호 및 보안 > 사이트 설정 > 알림'에서 'notify.worksmobile.com'을 '허용' 목록에 추가합니다.

## Microsoft Edge

브라우저 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-434343.png) 아이콘을 클릭한 후 '설정 > 쿠키 및 사이트 권한 > 알림'에서 '허용' 오른쪽의 '추가'를 클릭해 'notify.worksmobile.com'을 추가합니다.

## **Whale**

브라우저 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2019/08/whale_settings-1.png) 아이콘을 클릭한 후 '설정 > 개인정보 보호 > 알림'에서 '알림 전송이 허용됨' 항목에 'notify.worksmobile.com'을 추가합니다.

## **Safari**

브라우저 설정을 따로 하지 않아도 됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
