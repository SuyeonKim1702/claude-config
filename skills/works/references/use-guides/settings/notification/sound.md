# 알림음

모바일앱 알림이 올 때 받을 알림음을 설정할 수 있습니다.

### 모바일 앱

**Android**

1. 하단의 ‘홈’을 탭합니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '알림 설정'을 탭합니다.
4. 모바일앱 알림 종류별로 '메시지 알림', '메일 알림','서비스 알림'을 탭해 각각 알림음을 선택합니다.

**iOS**

1. 하단의 ‘홈’을 탭합니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '알림 설정'을 탭합니다.  
   '사운드' 설정이 꺼져있다면, 설정을 켜서 [OS 설정](/ko/use-guides/settings/set-device/set-device-notifiation/#i)으로 이동합니다. OS 설정에서 '알림'을 탭해 '사운드' 설정을 켠 후 네이버웍스로 다시 돌아옵니다.
4. '알림음'을 탭해 알림음을 선택합니다.
5. '저장'을 탭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
