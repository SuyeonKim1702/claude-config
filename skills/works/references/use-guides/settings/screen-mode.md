# 화면 모드 설정

라이트/다크 모드를 기기의 설정에 따라 자동으로 적용하거나, 직접 선택할 수 있습니다.

## 기기 설정 사용

화면 모드가 기기의 디스플레이 설정에 따라 라이트/다크 모드로 자동 적용됩니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '화면 모드'를 탭합니다.
4. '기기 설정 사용'을 켭니다.

## 화면 모드 설정

원하는 화면 모드를 직접 설정할 수 있습니다.

### 모바일 앱

1. 하단의 ‘홈’을 탭합니다.
2. ![](/wp-content/uploads/2024/05/common_24_setting_line.png) 아이콘을 탭합니다.
3. '화면 모드'를 탭합니다.
4. '기기 설정 사용'을 끕니다.
5. 라이트 모드, 다크 모드 중 원하는 옵션을 탭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
