# 서비스 즐겨찾기 설정

자주 사용하는 서비스를 즐겨찾기 영역에 추가합니다.

서비스는 최대 6개까지 추가할 수 있습니다.

단, 변경한 내용은 반영되기까지 최대 15분 정도 소요될 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '즐겨찾기 설정'을 클릭합니다.
3. 즐겨찾기에 추가하려는 서비스를 순서대로 클릭한 후 '저장'을 누릅니다. '초기화' 버튼을 클릭하면 사용 중인 상품에 따른 초기 설정값으로 돌아갑니다.
4. '저장'을 누르면 화면이 새로고침되며 즐겨찾기가 반영됩니다.

# 즐겨찾기 이동 방식 설정

즐겨찾기 클릭 시 브라우저의 새 탭에서 열지 현재 탭에서 이동할지 설정할 수 있습니다.

### PC 웹

1. 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ic20_viewtype_thumbnail_767676.png) 아이콘을 클릭합니다.
2. '즐겨찾기 설정'을 클릭합니다.
3. '새 창으로 열기'를 켜거나 끕니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
