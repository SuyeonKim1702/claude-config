# 회사 정보

회사 정보를 조회하고 수정합니다.

# 기본 정보 수정

네이버웍스를 이용하는 기업이나 단체의 이름과 전화번호를 확인하거나 추가할 수 있습니다. 이미 등록된 정보를 수정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '회사 정보'를 눌러 '회사 정보' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '기본 정보'로 이동합니다.

## 도메인 또는 그룹명 변경

최초 신청 시 입력한 그룹명 또는 도메인명입니다. 신청에 사용한 도메인 또는 그룹명을 변경할 수 있습니다.

### 도메인 변경하기

도메인을 사용하여 네이버웍스를 신청한 경우, 최고관리자는 도메인을 변경할 수 있습니다. '변경'을 눌러 ['도메인'](/ko/admin-guides/settings/additional-domains/additional-domains/)메뉴로 이동하면, 등록한 보조 도메인을 기본 도메인으로 전환할 수 있습니다. 도메인을 전환하면, 모든 구성원의 계정이 'ID@(전환한 도메인)'으로 변경됩니다.

### 그룹명 변경하기

그룹명을 사용하여 네이버웍스를 신청한 경우, 최고관리자는 그룹명을 변경할 수 있습니다.

그룹명을 변경하면 변경 후 24시간 내에는 그룹명을 다시 변경할 수 없습니다. 또, 모든 구성원이 서비스에서 강제 로그아웃되며, 새로운 그룹명이 포함된 ID로 로그인을 해야 합니다.

구성원의 개인 이메일을 구성원 정보에 등록해 두면 개인 이메일로 변경사실이 통지되어 구성원에게 변경 사실을 공유할 때 편리합니다.

1. 그룹명 오른쪽의 '변경'을 누릅니다.
2. 주의사항을 체크하고, 변경할 그룹명을 입력 후 '그룹명 변경'을 누릅니다. 그룹명은 2~22자의 영문 소문자나 숫자, 하이픈(-)의 조합으로 구성해야 하며, 이미 사용 중인 그룹명은 사용할 수 없습니다. 또한, 'worksmobile', 'lineworks'를 포함한 경우와 'line', 'admin'은 그룹명으로 사용할 수 없습니다.
3. 주의사항을 확인하고 '그룹명 변경'을 누릅니다.
4. '확인'을 누릅니다.

그룹명 변경이 완료되면 최고 관리자의 개인이메일로 메일이 발송되며, 등록한 휴대폰 번호로 SMS가 전송됩니다.

## 기업/단체명 입력

기업이나 단체의 이름을 입력합니다.

화면 하단의 '고급 설정' 내 다국어 설정을 사용하고 적용 정보에서 '기업/단체명'을 선택하면 설정한 언어로 기업/단체명을 추가 입력할 수 있습니다.

1. '기본 정보' 영역의 '기업/단체명' 항목에 회사명을 입력합니다.
   - 1글자 이상 40자까지 입력 가능
   - 일부 특수문자의 경우 사용 불가
   - 대소문자 구별 없이 LINE, NAVER WORKS, WORKS MOBILE, NAVER, NAVER WORKS, NAVER CLOUD 사용 불가
2. 오른쪽 상단의 '저장'을 누릅니다.

## 전화번호 입력

회사의 대표 전화번호를 입력합니다.

1. 1. '기본 정보' 영역의 '전화 번호' 항목에 회사의 대표 전화번호를 입력합니다.
   - 1글자 이상 30자까지 입력 가능
   - 숫자, -, \*, +, P, T, (, ) 입력 가능
   - 오른쪽 상단의 '저장'을 누릅니다.

# 언어·시간대/이용지역 설정

서비스 화면에 기본으로 적용되는 언어와 시간대를 설정합니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '회사 정보'를 눌러 '회사 정보' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '언어·시간대/이용지역'으로 이동합니다.

## 언어 선택

회사의 구성원이 처음 접속했을 때 PC웹 서비스 화면에 기본적으로 노출될 언어를 선택합니다.

단, 모바일앱과 PC앱은 구성원이 디바이스에서 설정한 언어가 노출됩니다.

1. '언어·시간대/이용지역' 영역의 '언어' 항목에 서비스에 기본적으로 표현될 언어를 선택합니다.
   - 한국어, 영어, 일본어, 중국어 간체, 중국어 번체 중 선택 가능
2. 오른쪽 상단의 '저장'을 누릅니다.

[구성원 정보 수정](/ko/admin-guides/manage-members/members/update-member-information/)에서 특정 구성원의 언어를 변경하면 해당 구성원에게는 변경된 언어가 PC웹 서비스 화면에 노출됩니다.

## 시간대/이용지역 선택

서비스를 사용하는 지역과 시간대를 선택합니다.

1. '언어·시간대/이용지역' 영역의 '시간대/이용지역' 항목에 서비스에 기본적으로 적용될 시간대를 검색하여 선택합니다.
2. 국가명 또는 도시명을 검색할 수 있습니다.
3. 오른쪽 상단의 '저장'을 누릅니다.

# 공식 계정 알림 수신 설정

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 신청 후 사용 가능합니다.

네이버웍스 공식 계정의 알림 메시지 수신 여부를 설정합니다.

네이버웍스 공식 계정의 알림은 네이버웍스 서비스의 업데이트 등 소식을 전달합니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '회사 정보'를 눌러 '회사 정보' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '공식 계정 알림'으로 이동합니다.

## 수신 여부 설정

네이버웍스 공식 계정의 알림 메시지 수신 여부를 설정합니다.

1. '공식 계정 알림' 영역의 '알림 설정' 항목에 수신 여부를 선택합니다.
2. 오른쪽 상단의 '저장'을 누릅니다.

## 수신 범위 설정

네이버웍스 공식 계정의 알림을 받기로 했을 때 받는 사람의 범위를 설정합니다.

1. '공식 계정 알림' 영역의 '수신 범위' 항목에 수신 범위를 선택합니다.
   - 관리자만 수신: [관리 권한이 있는 모든 관리자](/ko/admin-guides/security/administrator-authorities/manage-administrator-authorities/)만 알림을 받습니다.
   - 구성원(관리자 포함) 수신: 관리자 포함 회사의 모든 구성원이 알림을 받습니다.
2. 오른쪽 상단의 '저장'을 누릅니다.

# 고급 설정(다국어 설정)

다국어 설정을 사용하면 네이버웍스 서비스 내 정보가 구성원의 사용자 환경 언어로 표시됩니다.

- 기업/단체명
## 다국어 사용하기

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '회사 정보'를 눌러 '회사 정보' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 하단의 '고급 설정'을 누릅니다.
3. '다국어 설정'에서 '사용'을 선택합니다.
4. '다국어 추가' 버튼을 누릅니다.
5. 원하는 순서대로 다국어로 노출할 언어를 선택하고, '확인'을 누릅니다.
6. 왼쪽 하단 또는 오른쪽 상단의 '저장'을 누릅니다.

언어의 순서를 변경하려면 해당 언어를 원하는 위치에 끌어다 놓습니다.

언어를 삭제하려면 언어 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/delete.png) 아이콘을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
