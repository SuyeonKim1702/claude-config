# PC앱 설정

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 신청 후 사용 가능합니다.

PC앱에 노출할 서비스 순서를 변경하고 메뉴를 추가 등록합니다.

# 서비스 순서 설정

네이버웍스 PC앱 메뉴에서 노출할 서비스의 순서를 설정합니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '맞춤 설정'을 눌러 '맞춤 설정' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 'PC앱'을 누릅니다.
3. '서비스 순서'에서 이동할 서비스 아이콘을 끌어서 원하는 위치에 놓습니다. 왼쪽에 놓인 서비스부터 PC앱 메뉴 상단에 노출됩니다. 조정된 위치는 즉시 왼쪽의 미리보기 영역에서 확인할 수 있습니다.
4. 오른쪽 상단의 '저장'을 눌러 적용합니다.

실제 화면에서는 [구성원별 권한](/ko/admin-guides/security/authority/authority/)에 따라 사용할 수 있는 서비스만 표시됩니다.

# 바로가기 메뉴 추가 등록

PC앱 메뉴에 추가로 노출할 바로가기 메뉴를 등록합니다. 메뉴는 35개까지 추가할 수 있으며, 아이콘 이미지 파일은 다음 조건을 충족해야 합니다.

- 사이즈: 가로 54px, 세로 54px
- 1MB 미만
- 파일의 확장자는 .png
- 파일명은 영문, 숫자, 허용된 특수문자( - \_ & % ( ) ' . , )만 사용 가능

1. '바로가기 메뉴' 의 '+ 추가'를 누릅니다.
2. '바로가기 메뉴 추가' 대화창에서 '업로드'를 눌러 아이콘 이미지 파일을 업로드한 후 아이콘 클릭 시 연결할 링크 URL을 입력합니다.
3. 하단의 '확인'을 누릅니다.
4. 오른쪽 상단의 '저장'을 눌러 적용합니다.
   - 아이콘이 2개 이상 추가되면 2번째 아이콘부터는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/ic20_more_767676.png) 아이콘으로 표시됩니다.
   - 아이콘의 순서를 변경하려면 아이콘 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/grabber.png) 를 움직여 메뉴 순서를 변경합니다.

# 등록 내용 초기화

1. 화면 오른쪽 상단의 '초기화'를 누릅니다.
2. '저장'을 눌러 초기 설정값으로 변경합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
