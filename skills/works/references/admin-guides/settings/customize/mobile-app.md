# 모바일앱 설정

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 신청 후 사용 가능합니다.

로딩 이미지와 메뉴를 추가 등록합니다.

# 로딩 이미지 설정

네이버웍스 모바일앱의 로딩화면에 회사의 로고 등 원하는 이미지를 표시합니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '맞춤 설정'을 눌러 '맞춤 설정' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '모바일앱'을 누릅니다.

## 로딩화면 이미지 등록

로딩화면에 노출될 이미지를 등록합니다.

등록할 파일은 다음 조건을 충족해야 합니다.

- 3MB 미만
- 파일의 확장자는 .png
- 파일명은 영문과 숫자, 허용된 특수문자( - \_ & % ( ) ' . , )만 사용 가능
- 다크 모드에는 #222222 배경색과 대비되도록 밝은 색상의 이미지 권장

1. '로딩 이미지' 하단의 '파일 업로드'를 눌러 등록할 파일을 선택합니다. 조건을 충족하는 파일을 선택하면 왼쪽의 미리보기로 적용된 로딩 화면을 확인할 수 있습니다.
   - 등록한 이미지는 가로 600px 기준에 맞춰 사이즈가 자동 조절됨
   - 라이트 모드와 다크 모드에 각각 이미지를 등록할 수 있으며, 하나만 등록해도 모든 모드에 동일하게 적용됨
2. 오른쪽 상단의 '저장'을 눌러 적용합니다.

## 기업명 노출

로딩화면에 노출될 이미지를 등록하면 [기업/단체명](/ko/admin-guides/settings/company-information/#i)을 로딩 화면에 노출할 수 있습니다.

노출되는 기업명의 폰트는 사용자 디바이스의 시스템 폰트가 적용됩니다.

1. '로딩 이미지' 항목의 '기업/단체명 표시'를 선택합니다.
2. 오른쪽 상단의 '저장'을 눌러 적용합니다.

# 모바일앱 메뉴 설정

모바일앱의 '홈'의 메뉴를 추가하고 하단 메뉴의 순서를 변경합니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '맞춤 설정'을 눌러 '맞춤 설정' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '모바일앱'을 누릅니다.

## 바로가기 메뉴 추가 등록

모바일앱의 '홈'에 노출할 바로가기 메뉴를 추가합니다.

메뉴는 35개까지 추가할 수 있으며, 아이콘을 눌러 또 다른 앱을 열거나 웹 사이트로 이동할 수 있습니다.

아이콘 이미지 파일은 다음 조건을 충족해야 합니다.

- 사이즈
  - iOS: 68픽셀 \* 68 픽셀
  - Android: 128픽셀 \* 128픽셀
- 1MB 미만
- 파일의 확장자는 .png
- 파일명은 영문, 숫자, 허용된 특수문자( - \_ & % ( ) ' . , )만 사용 가능

1. '바로가기 메뉴' 영역의 '+ 추가'를 누릅니다.
2. '바로가기 메뉴 추가' 대화창에서 메뉴명과 연결 링크가 포함된 바로가기 메뉴 영역에 대한 내용을 입력합니다.
   - 메뉴명
     - 메뉴명(필수): 노출될 메뉴명입니다. 다국어 입력이 되어있지 않으면 사용자 디바이스의 언어 설정에 관계없이 입력한 메뉴명이 노출됩니다.
     - 다국어: 다국어를 입력하면 네이버웍스 사용자 디바이스의 언어 설정에 맞춰 메뉴명이 노출됩니다.
   - 연결 대상:  아이콘을 눌렀을 때 이동할 대상을 선택합니다.
     - 또 다른 앱: 모바일 기기에 앱이 설치되었다면 해당 앱이, 설치되지 않았다면 설치 페이지가 열립니다. iOS와 Android 가운데 최소한 하나는 선택해야 합니다.  
       OS별 Deep Linking과 관련된 자세한 내용은 헬프센터의 '[모바일앱에서 외부 앱 연동 시 유의사항을 알고 싶어요.](https://help.worksmobile.com//ko/faqs/#considerations-to-add-icon-menu-on-mobile-app/)'에서 확인할 수 있습니다.
       - iOS
         1. '업로드'를 눌러 아이콘 이미지를 선택합니다.
         2. 'Universal Link'를 눌러 Universal Link 또는 URL Scheme 중 하나를 선택한 후 정보를 입력합니다.
         3. install URL을 입력합니다.
       - Android
         1. '업로드'를 눌러 아이콘 이미지를 선택합니다.
         2. 'App Link'를 눌러 App Link 또는 URL Scheme 중 선택한 후 정보를 입력합니다.
         3. Package Name이 있을 경우, 항목을 입력합니다.
         4. install URL을 입력합니다.
     - 모바일 웹 사이트: 모바일 기기의 기본 브라우저에서 웹 사이트가 열립니다.
       1. iOS와 Android 오른쪽의 '업로드'를 눌러 각각의 이미지를 업로드합니다.
       2. Link URL을 입력합니다. Badge URL을 입력하면 연결한 웹 사이트에서 알림 발생 시 아이콘에 뱃지가 표시되도록 설정할 수 있습니다.
3. '바로가기 메뉴 추가' 화면에서 하단의 '확인'을 누릅니다.
4. 오른쪽 상단의 '저장'을 눌러 적용합니다.

8개 이상의 바로가기 메뉴를 등록한 경우, 바로가기 메뉴 목록을 스크롤해 모든 정보를 확인할 수 있습니다.

## 하단 탭 메뉴 설정

네이버웍스에서 제공하는 메시지, 캘린더, 게시판, 주소록, 할 일, 설문, 메일, 드라이브 서비스의 하단 탭 노출 서비스와 순서를 설정합니다.

하단 탭 메뉴에는 홈 아이콘 오른쪽으로 최대 4개 서비스가 노출됩니다.

하단 탭 메뉴에 노출되지 않는 서비스는 홈 화면의 상단부터 순서대로 노출됩니다.

1. '서비스 순서' 영역에서 이동할 서비스 아이콘을 끌어서 원하는 위치에 놓습니다. 조정된 위치는 즉시 왼쪽의 미리보기 영역에서 확인할 수 있습니다.
2. 오른쪽 상단의 '저장'을 눌러 적용합니다.

실제 화면에서는 [구성원별 권한](/ko/admin-guides/security/authority/authority/)에 따라 사용할 수 있는 서비스만 표시됩니다.

# 등록 내용 초기화

1. 화면 오른쪽 상단의 '저장' 버튼 옆 '초기화'를 누르면 네이버웍스 초기 설정 값이 미리보기에 적용됩니다.
   - 등록된 로딩 화면 이미지 없음
   - 추가된 바로가기 메뉴 없음
   - 네이버웍스 서비스 아이콘이 기본 제공 순서로 재정렬됨
2. '저장'을 눌러 완료합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
