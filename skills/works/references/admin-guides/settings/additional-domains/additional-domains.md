# 도메인 관리

보조 도메인을 추가하거나 추가된 보조 도메인을 삭제합니다.

보조 도메인을 활용하면 사용중인 네이버웍스 메일의 도메인이 변경되는 경우 사용중인 메일 도메인과, 변경되는 메일 도메인을 함께 사용할 수 있습니다.

보조 도메인을 추가하면 모든 구성원이 기존 이메일 주소와 함께 '구성원ID@보조 도메인'으로 구성된 새로운 이메일 주소를 가지게 됩니다.

메일 쓰기 화면의 '보내는 사람' 영역에 기본 도메인의 메일 주소와 보조 도메인의 메일 주소를 선택할 수 있습니다.

보조 도메인의 메일 주소로 받은 메일은 기본 도메인의 수신메일함으로 인입됩니다. 보조 도메인의 메일 주소로 받은 메일의 경우에 기본 도메인의 메일 주소로 수신된 메일과 구분되도록 메일 제목 앞에 보조 도메인이 표기됩니다.

# 보조 도메인 추가

네이버웍스 가입 시 등록한 도메인 외에 보유한 다른 도메인을 보조 도메인으로 추가합니다.

최대 10개의 보조 도메인을 추가할 수 있습니다.

보조 도메인으로 네이버웍스 메일을 이용하려면 보조 도메인의 DNS에 MX 레코드 설정이 필요합니다 (네이버웍스 메일을 사용하지 않는다면, TXT/CNAME 설정으로도 가능). 네이버웍스에서 MX 레코드 설정이 확인되면, 기본 도메인 메일 주소 뿐만 아니라 보조 도메인 메일 주소도 발신자의 메일 주소로 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '도메인'을 눌러 '도메인' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단의 '도메인 추가'를 누릅니다.
   - 기본 도메인의 상태가 '사용중'이면 '도메인 추가'가 노출됩니다.
3. 보조 도메인을 입력하고 '완료'를 누릅니다.
   - 네이버웍스에 이미 등록된 도메인은 등록할 수 없습니다.
   - 2차 도메인(예: aa.bb.com에서, bb.com)은 다음 세 가지 경우에 해당하는 경우, 총 5개까지만 보조 도메인으로 등록할 수 있습니다.
     - 다른 회사에서 사용 중인 도메인
     - 다른 회사에 등록된 보조 도메인
     - 같은 그룹사의 하위 도메인
4. MX 레코드 설정을 통해 보조 도메인을 활성화합니다.
   - 네이버웍스 메일을 사용하지 않는다면, TXT/CNAME 설정으로도 가능합니다.

MX 레코드 설정 후 연동이 확인될 때까지는 도메인 관리 사이트에 따라 최대 72시간까지 소요될 수 있습니다.

보조 도메인의 상태가 '사용중'이 되면 모든 구성원에게 보조 도메인의 메일 주소가 생성되며, 생성된 메일 주소로 메일을 받거나 보낼 수 있습니다.

추가된 보조 도메인이 14일 내에 활성화되지 않으면 자동으로 삭제됩니다.

# 보조 도메인 삭제

등록한 보조 도메인을 삭제합니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '도메인'을 눌러 '도메인' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 도메인 목록에서 삭제할 도메인의 '삭제'를 누릅니다.

보조 도메인을 삭제하면 즉시 모든 구성원의 보조 도메인 메일주소가 삭제되어 해당 도메인의 메일 주소로 메일을 받거나 보낼 수 없습니다.

기본 도메인은 삭제할 수 없습니다.

# 기본-보조 도메인 변경

최고관리자는 도메인 소유권을 확인한 보조 도메인을 기본 도메인으로 변경할 수 있습니다.

현재 사용중인 기본 도메인을 변경하고 싶다면, 변경하게 될 도메인을 먼저 보조 도메인으로 추가해야 합니다.

도메인 변경 시 주의사항입니다.

- 구성원 수가 많으면 도메인 변경 소요시간이 길어질 수 있습니다.
- 도메인을 변경하면 변경 후 24시간 내에는 도메인을 다시 변경할 수 없습니다. 또한, 모든 구성원이 서비스에서 강제 로그아웃되며, 새로운 도메인이 포함된 ID로 로그인을 해야 합니다.
- 구성원의 개인 이메일을 [구성원 정보에 등록](/ko/admin-guides/manage-members/members/update-member-information/)해두면 개인 이메일로 변경 사실이 통지되어 구성원에게 변경 사실을 공유할 때 편리합니다.
- 도메인 변경이 완료되면 최고 관리자의 개인 이메일로 메일이 발송되며, 등록한 휴대폰 번호로 SMS가 전송됩니다.

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '도메인'을 눌러 '도메인' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '사용중' 상태의 보조 도메인에 있는 '기본 도메인으로 변경'을 누릅니다.
   - '사용중' 상태가 되기 위해서는 반드시 보조 도메인에 MX레코드를 설정하여 도메인 소유권을 확인해야 합니다.
3. 주의사항을 확인한 뒤 '도메인 변경'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
