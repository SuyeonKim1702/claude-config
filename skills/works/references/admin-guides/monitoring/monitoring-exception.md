# 예외 관리

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

모니터링 대상에서 제외할 구성원의 목록을 관리할 수 있습니다.

예외 관리 목록에 등록된 구성원은 아래의 모니터링 정책에서 제외됩니다.

- 메시지 수신: 예외 관리 목록에 등록된 구성원이 [메시지 모니터링 정책](/ko/admin-guides/monitoring/message/#i)의 '받는 사람'에 포함된 경우, 이 구성원이 받은 메시지는 메시지 모니터링 내역에 남지 않습니다.
- 메시지 발신: 예외 관리 목록에 등록된 구성원이 [메시지 모니터링 정책](/ko/admin-guides/monitoring/message/#i)의 '보낸 사람'에 포함된 경우, 이 구성원이 보낸 메시지는 메시지 모니터링 내역에 남지 않습니다.
- 메일 수신: 예외 관리 목록에 등록된 구성원이 [메일 수신 모니터링 정책](/ko/admin-guides/monitoring/received-mail/#i)의 '받는 사람'에 포함된 경우, 이 구성원이 받은 메일은 메일 수신 모니터링 내역에 남지 않습니다.
- 메일 발신: 예외 관리 목록에 등록된 구성원이 [메일 발신 모니터링 정책](/ko/admin-guides/monitoring/sent-mail/#i)의 '보낸 사람'에 포함된 경우, 이 구성원이 보낸 메일은 메일 발신 모니터링 내역에 남지 않습니다.
- 드라이브: 예외 관리 목록에 등록된 구성원이 [드라이브 모니터링 정책](/ko/admin-guides/monitoring/drive/#i)의 '구성원 지정'에 포함된 경우, 이 구성원의 파일 업로드, 다운로드, 링크 공유 기록은 드라이브 모니터링 내역에 남지 않습니다.

# 예외 관리 목록에 구성원 추가

예외 관리 목록에 구성원을 추가하면 해당 구성원은 모니터링 정책 적용 대상에서 제외됩니다.

구성원은 200명까지 등록할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '모니터링'을 선택해 메뉴를 펼칩니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '예외 관리'를 선택합니다.
3. 오른쪽 상단의 '구성원 추가'를 누릅니다.
4. 추가할 구성원을 선택한 후 '확인'을 누릅니다.

# 예외 관리 목록에서 구성원 삭제

예외 관리 목록에 등록된 구성원을 삭제하는 시점부터 해당 구성원은 모니터링 정책 적용 대상에 포함됩니다.

1. Admin 왼쪽 메뉴에서 '모니터링'을 선택해 메뉴를 펼칩니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '예외 관리'를 선택합니다.
3. 목록에서 삭제할 구성원 오른쪽에 있는 '삭제'를 누릅니다. 여러 구성원을 한 번에 삭제하고 싶다면 삭제할 구성원을 선택하고 '삭제'를 누릅니다.

# 예외 관리 목록에서 구성원 검색

예외 관리 목록에 속한 구성원을 검색할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '모니터링'을 선택해 메뉴를 펼칩니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '예외 관리'를 선택합니다.
3. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 누릅니다.
4. 구성원의 이름 또는 계정을 입력해 검색합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
