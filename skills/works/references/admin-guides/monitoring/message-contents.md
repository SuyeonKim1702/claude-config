# 메시지 내용

메시지 내용을 확인하고 검색합니다.

// start of content for ko en tw

### 모바일 앱

1. Admin 메뉴에서 '메시지 내용'을 탭합니다.
2. '검색'을 눌러 '메시지 내용' 검색 결과를 화면에 노출합니다.

- 메시지: 보낸 메시지의 내용입니다.
- 보낸사람: 메시지를 보낸 구성원의 이름과 메일 주소입니다.
- 받는사람: 메시지를 받는 사람의 이름과 메일 주소입니다.
- 날짜: 메시지를 보낸 날짜와 시각입니다.

### PC 웹

1. Admin 메뉴에서 '메시지 내용'을 클릭합니다.
2. '검색'을 눌러 '메시지 내용' 검색 결과를 화면에 노출합니다.

- 메시지: 보낸 메시지의 내용입니다.
- 보낸사람: 메시지를 보낸 구성원의 이름과 메일 주소입니다.
- 받는사람: 메시지를 받는 사람의 이름과 메일 주소입니다.
- 날짜: 메시지를 보낸 날짜와 시각입니다.
- 채널 ID: 메시지를 주고 받은 메시지방의 ID입니다.

## 메시지 내용 검색

메시지 내용을 검색합니다.

PC 웹에서는 검색 기간으로 검색할 수 있고, 상세를 선택하여 동시에 여러 조건을 지정한 검색도 가능합니다.

// start of content for ko en tw

### 모바일 앱

1. '메시지 내용' 화면에서 검색 아이콘을 탭하여 검색 화면을 펼친 후 조건을 입력합니다.
   - Bot 메시지: Bot이 보낸 메시지를 필터링하는 옵션입니다.
2. '검색'을 탭하여 검색합니다.

### PC 웹

1. '메시지 내용'화면에서 '기간'을 선택합니다. '상세'를 클릭하면 동시에 여러 항목을 지정하여 검색할 수 있습니다.
   - Bot 메시지: Bot이 보낸 메시지를 필터링하는 옵션입니다.
2. '검색'을 클릭하여 메시지 내역을 확인합니다.

## 메시지 내용 로그 다운로드

조회한 로그를 다운로드합니다. 파일 다운로드 링크는 2주일 후 만료됩니다.

1. '메시지 내용'화면에서 검색을 이용해 다운로드할 내용을 화면에 노출합니다.
2. 오른쪽 상단의 '다운로드'를 눌러 다운로드를 요청합니다.
3. 파일 다운로드 준비가 완료되면, 서비스 알림 메시지가 도착합니다. 메시지에서 '파일 다운로드'를 눌러 다운로드합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
