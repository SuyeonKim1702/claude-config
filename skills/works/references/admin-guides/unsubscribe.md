# 서비스 탈퇴

NAVER WORKS 계정을 탈퇴하고 서비스 및 상품을 해지합니다.

# 탈퇴 시 유의사항

탈퇴 시 모든 상품은 해지되며 데이터가 삭제됩니다. 해지 전, 필요한 데이터의 백업을 권장합니다.

백업 방법은 아래와 같습니다.

# 탈퇴할 수 없는 경우

다음의 경우, NAVER WORKS를 탈퇴할 수 없습니다.

- 현재 2개 이상의 기본 상품이 해지되지 않고 사용 중인 경우  
  (예: 네이버웍스 코어 Standard + 네이버웍스 경영지원 결재)
- 그룹사 하위 도메인이 아직 삭제(탈퇴)되지 않은 경우

# 탈퇴 방법

1. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 'NAVER WORKS 탈퇴'를 눌러 'NAVER WORKS 탈퇴' 화면으로 이동합니다.
2. 화면 하단 '설문 후 탈퇴' 버튼을 누릅니다.
3. 설문에 응답합니다. 탈퇴 사유를 선택하고, NAVER WORKS 관련 소식을 메일로 수신하려면 메일 주소를 입력합니다.
4. 비밀번호를 입력한 후 '탈퇴하기' 버튼을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
