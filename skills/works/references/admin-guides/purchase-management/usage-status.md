# 이용 현황

계약 내용에 따라 고객사별로 표시되는 항목이 다릅니다. 이 메뉴는 PC웹에서만 확인할 수 있습니다.

  
현재 사용하는 서비스의 상품을 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구매 및 청구'를 선택해 메뉴를 펼친 후 '이용 현황'을 누릅니다.
2. 이용 중인 상품 현황을 확인합니다.

# 상품 변경

상품의 플랜을 업그레이드하거나, 옵션 상품을 추가 신청할 수 있습니다.

1. 변경하려는 상품 오른쪽의 ![](/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
2. '상품 변경' 버튼을 클릭합니다.
3. 해당 상품의 신청 페이지로 이동하여, 플랜을 변경하거나 옵션 상품을 추가 신청합니다.

단, 네이버웍스 코어 상품의 체험판이 종료된 후 Free 플랜을 신청하는 경우, 정책에 따라 일부 기능이 제한됩니다.

플랜별 기능 차이는 [기능 상세 페이지](https://naver.worksmobile.com/naver-works-function-list/message/)를 참고해주세요.

# 유료 전환

체험판을 사용 중 정식 상품 또는 유료 상품으로 전환할 수 있습니다.

유료 전환 전에 [네이버 클라우드 플랫폼 사이트에서 결제 수단을 미리 등록](/ko/admin-guides/purchase-management/payment-management/)해두어야 합니다.

무료 체험판 기간 동안 유료 상품으로 전환하지 못한 경우 서비스 사용이 정지되며 30일의 유예 기간이 제공됩니다.

유예 기간 동안에는 네이버웍스 서비스 사용이 제한되며, 구성원은 서비스를 사용할 수 없고, 관리자는 Admin만 사용할 수 있습니다.

유예 기간 종료일까지 유료 상품으로 전환하지 않을 시 탈퇴 처리되며 저장했던 데이터가 모두 삭제되니, 기간 내에 전환을 완료해주세요.

1. 유료로 전환하려는 상품을 확인하고 '유료 전환' 버튼을 클릭합니다.
2. 상품별 신청 페이지에서 원하는 상품 및 계약 형태를 선택합니다.
3. 신청 내역 확인 및 약관에 동의한 후 '신청' 버튼을 클릭합니다.

무료 체험판은 모든 상품의 최상위 플랜을 제공하고 있어, 유료 전환 시 원하는 플랜을 선택할 수 있습니다.

단, 네이버웍스 코어 상품의 체험판이 종료된 후 Free 플랜을 신청하는 경우, 정책에 따라 일부 기능이 제한됩니다.

최상위 플랜 외 다른 플랜(Free, Lite 플랜 등)을 신청하는 경우, 정책에 따라 일부 기능이 제한됩니다. 플랜별 기능 차이는 [기능 상세 페이지](https://naver.worksmobile.com/naver-works-function-list/message/)를 참고해주세요.

연간 계약과 월간 계약의 차이는 '[연간 계약과 월간 계약의 차이가 무엇인가요?](https://help.worksmobile.com/ko/faqs/#the-difference-an-annual-contract-and-a-monthly-contract)' 도움말을 참고해주세요.

# 상품 갱신 설정

상품별 계약 갱신 조건을 설정할 수 있습니다.

변경된 갱신 조건은 계약 갱신일부터 적용됩니다.

1. 갱신하려는 상품 오른쪽의 ![](/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
2. '갱신 설정' 버튼을 클릭합니다.
3. 갱신 조건을 설정하고 '저장'을 클릭합니다.  
   '동일한 조건으로 갱신'을 선택할 경우 이전과 같은 조건으로 계약을 갱신합니다.

# 상품 해지

일부 옵션 상품 또는 상품 전체를 해지할 수 있습니다.

## 상품 해지 시 유의사항

상품을 해지하면 각 상품별로 아래의 서비스 데이터가 모두 삭제됩니다.

|  |  |  |
| --- | --- | --- |
| 해지 전 이용 상품 | 플랜 | 삭제되는 서비스 데이터 |
| 코어 | - Standard Plus - Standard - Trial 무료 체험판 | 게시판  메시지  메일  캘린더  드라이브  할 일  설문 |
| - Lite - Free | 게시판  메시지  캘린더  할 일  설문 |
| 드라이브 | - Standard - Lite - Trial 무료 체험판 | 드라이브 |
| 결재 | N/A | 결재  게시판  내 업무 |
| 근태 | N/A | 근태 |
| 급여 | N/A | 급여 |
| 재무 | N/A | 재무 |

단, 전체 상품 중 해지하지 않고 남은 상품에 해당하는 서비스의 권한과 데이터는 삭제되지 않습니다.  
예를 들어, 네이버웍스 코어, 웍스 드라이브, 웍스 결재 상품을 함께 이용 중인 경우, 네이버웍스 코어 상품을 해지해도 드라이브와 게시판 서비스는 삭제되지 않습니다.

이 경우 조직과 그룹 메시지방 데이터(노트/일정/할 일/폴더), 1:1 및 1:N 대화방, 외부 그룹 등이 삭제되며, 공용용량이 조정됩니다.

1. 해지를 원하는 상품 오른쪽의 ![](/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
2. '해지' 버튼을 클릭합니다.
3. 해지를 원하는 상품을 선택합니다.
   - 기본: 이용 중인 옵션 상품을 포함한 상품 전체가 해지됩니다.
   - 옵션: 선택한 옵션 상품만 해지됩니다.
4. 하단의 '해지' 버튼을 클릭하고 해지할 상품을 한 번 더 확인합니다.
5. 내용을 확인한 후, '해지' 버튼을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
