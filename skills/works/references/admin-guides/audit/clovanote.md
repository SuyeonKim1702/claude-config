# 클로바노트

클로바노트 내 노트와 폴더의 생성/삭제/다운로드 내역을 확인합니다.

// start of content for jp

단, AiNote Free 플랜을 이용 중인 경우 오늘 기준 14일 내 로그만 정상적으로 조회할 수 있으며, 감사 로그 다운로드는 AiNote 유료 플랜에서만 사용할 수 있습니다.

// end of content for jp

Admin 왼쪽 메뉴에서 '감사'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.

### 모바일 앱

- 제목: 과업이 발생한 노트 제목이나 폴더명이 표시됩니다. 휴지통의 경우 제목 없이 빈 값으로 표시됩니다.
- 항목: 서비스 내에서 과업이 발생한 항목입니다.
- 과업: 사용자가 수행한 과업입니다. 과업의 종류는 다음과 같습니다.
  - 노트 생성
  - 노트 열람
  - 음성 파일 다운로드
  - 음성 기록 다운로드
  - 메모 다운로드
  - 공유 링크 생성
  - 공유 링크 삭제
  - 휴지통 이동
  - 영구 삭제
  - 휴지통 비우기
- 사용자: 과업을 행한 구성원명과 메일 주소입니다.
- 날짜: 과업을 행한 날짜와 시각입니다.

### PC 웹

- 제목: 과업이 발생한 노트 제목이나 폴더명이 표시됩니다. 휴지통의 경우 제목 없이 빈 값으로 표시됩니다.
- 항목: 서비스 내에서 과업이 발생한 항목입니다.
- 노트 ID: 과업이 발생한 노트의 ID입니다. 폴더와 휴지통은 빈 값으로 표시됩니다.
- 과업: 사용자가 수행한 과업입니다. 과업의 종류는 다음과 같습니다.
  - 노트 생성
  - 노트 열람
  - 음성 파일 다운로드
  - 음성 기록 다운로드
  - 메모 다운로드
  - 공유 링크 생성
  - 공유 링크 삭제
  - 휴지통 이동
  - 영구 삭제
  - 휴지통 비우기
- 사용자: 과업을 행한 구성원명과 메일 주소입니다.
- 날짜: 과업을 행한 날짜와 시각입니다.
- 서비스 타입: 과업을 수행한 사용자 환경입니다. PC 웹과 모바일 앱이 있습니다

## 클로바노트 감사 로그 검색

클로바노트의 감사 로그를 검색합니다.

### 모바일 앱

1. '클로바노트' 화면에서 검색 아이콘을 탭하여 검색 화면을 펼친 후 조건을 입력합니다.
2. '검색'을 탭합니다.

### PC 웹

검색 기간으로 검색할 수 있고, 상세를 선택하여 동시에 여러 조건을 지정한 검색도 가능합니다.

1. '클로바노트' 화면에서 '기간'을 선택합니다. '상세'를 클릭하면 동시에 여러 조건을 지정하여 검색할 수 있습니다.
2. '검색'을 클릭합니다.

## 클로바노트 감사 로그 다운로드

조회한 감사 로그를 다운로드합니다.

1. '클로바노트' 화면에서 검색을 이용해 다운로드할 내용을 화면에 노출합니다.
2. 오른쪽 상단의 '다운로드'를 눌러 다운로드를 요청합니다.
3. 왼쪽 메뉴의 '로그 다운로드'를 눌러 이동합니다.
4. 요청한 로그를 찾아 파일을 다운로드합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
