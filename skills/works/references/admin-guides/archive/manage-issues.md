# 이슈 관리

이 기능은 아카이빙 옵션 상품 신청 후 사용 가능합니다.

사법 기관의 제출 요청 등 이슈가 생길 때 보관 정책에 의해 보관된 데이터에서 해당 이슈와 관련된 내용만 검색해서 결과를 확인하거나 제출하는 경우가 많습니다.

이러한 경우에 관련 내용만 검색되도록 이슈를 만들어 관리할 수 있습니다.

이슈 관리에서는 이슈를 등록하거나 수정하고 삭제할 수 있으며, 이슈에 대한 결과를 확인할 수도 있고, 다운로드할 수 있습니다.

# 이슈 관리 화면 설명

이슈 관리 화면의 각 항목에 대한 설명입니다.

### 모바일 앱

1. Admin 왼쪽 메뉴에서 '아카이빙'을 선택해 메뉴를 펼친 후 '이슈 관리'를 눌러 '이슈 관리' 화면으로 이동합니다.
   - 등록된 이슈의 정보를 제공합니다.
     - 구분: 메일과 메시지로 구분됩니다.
     - 이슈명: 등록된 이슈의 타이틀입니다.
   - 검색: '검색'을 클릭하면 현재 시점을 기준으로 해당 이슈의 검색 조건으로 검색된 결과 화면을 띄웁니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '아카이빙'을 선택해 메뉴를 펼친 후 '이슈 관리'를 눌러 '이슈 관리' 화면으로 이동합니다.
   - 구분: 메일과 메시지로 구분됩니다.
   - 이슈명: 등록된 이슈의 타이틀입니다.
   - 등록자: 이슈를 등록한 구성원의 이름입니다.
   - 마지막 수정일: 이슈의 최종 수정일입니다.
   - 검색: '검색'을 클릭하면 현재 시점을 기준으로 해당 이슈의 검색 조건으로 검색된 결과 화면을 띄웁니다.

# 이슈 등록하기

보관된 데이터에서 검색할 조건을 이슈로 등록합니다.

이후에도 같은 조건으로 검색할 필요가 있을 경우, 이슈로 등록해 놓으면 조건을 다시 입력하지 않고 검색할 수 있습니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '아카이빙'을 선택해 메뉴를 펼친 후 '이슈 관리'를 눌러 '이슈 관리' 화면으로 이동합니다.
2. '이슈 등록'을 누릅니다.
3. '이슈 등록' 페이지에서 필요한 항목을 입력합니다.
   - 이슈 이름: 이슈의 타이틀을 입력합니다.
   - 구분: 메일과 메시지 가운데 검색할 대상을 선택합니다.
   - 검색 대상: 구분에서 메일을 선택했을 경우만 노출됩니다.
     - 전체: 구성원 전체의 메일함을 대상으로 검색합니다.
     - 구성원: 특정 구성원의 메일함을 대상으로 검색합니다.
   - 보낸 사람: 보낸 사람을 지정합니다.
   - 받는 사람: 받는 사람을 지정합니다. 최대 10명까지 지정할 수 있습니다.
   - 내용
     - 전체 검색: 제목, 본문, 첨부파일을 모두 검색합니다.
     - 제목+본문
     - 제목
     - 본문
     - 첨부파일: 파일명과 파일 문서 내용을 검색합니다. 단, 대용량 첨부파일이나 비밀번호가 적용된 첨부파일은 파일명만 검색할 수 있습니다.
   - 보안 등급: 'Admin[> 보안 > 메일 보안 > 보안 등급](/ko/admin-guides/security/mail-security/#i-7)'에서 설정한 등급명을 선택할 수 있습니다.
     - 사용 안 함
     - 일반
     - 대외비: 등급명을 수정했을 경우, 수정한 등급명으로 보일 수 있습니다.
     - 기밀: 등급명을 수정했을 경우, 수정한 등급명으로 보일 수 있습니다.
     - 업무 외
   - 기간 : 이슈를 검색할 기간을 설정합니다. 설정한 시간대를 기준으로 해당  기간의 이슈가 검색됩니다.
4. '저장'을 클릭해 이슈를 등록합니다.

# 이슈 수정하기

### PC 웹

등록한 이슈의 내용을 수정합니다.

1. Admin 왼쪽 메뉴에서 '아카이빙'을 선택해 메뉴를 펼친 후 '이슈 관리'를 눌러 '이슈 관리' 화면으로 이동합니다.
2. 수정할 이슈의 '이슈명' 항목을 클릭합니다.
3. '이슈 수정' 화면에서 필요한 항목을 수정합니다.
4. '저장'을 클릭해 이슈를 수정합니다.

# 이슈 삭제하기

등록한 이슈를 삭제합니다.

'이슈 관리' 화면에서 삭제할 항목을 선택하고 삭제할 수 있습니다.

## 이슈 관리 화면에서 삭제하기

### PC 웹

여러 건의 이슈를 동시에 삭제할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '아카이빙'을 선택해 메뉴를 펼친 후 '이슈 관리'를 눌러 '이슈 관리' 화면으로 이동합니다.
2. '이슈 관리' 화면의 이슈 목록에서 삭제할 이슈를 선택한 후 상단의 '삭제'를 클릭합니다.
3. '확인'을 클릭해 선택한 이슈를 삭제합니다.

# 이슈로 검색한 결과 확인 및 다운로드하기

등록한 이슈의 조건으로 검색한 결과를 확인합니다.

## 이슈의 검색결과 보기

이슈의 검색 조건으로 검색한 결과를 미리 확인합니다.

'이슈 관리' 화면에서 '검색'을 눌러 확인할 수 있습니다.

### 메시지 대화방 검색 결과

구분에서 메시지를 선택한 이슈의 검색 결과입니다.

### 모바일 앱

- 검색 결과로 다음과 같은 정보를 확인할 수 있습니다.
  - 메시지 / 파일명: 대화방에 받거나 보낸 가장 최근의 메시지나 파일명입니다.
  - 보낸 사람: 메시지를 보낸 사용자 이름이 노출됩니다.
  - 수신시각 : 해당 메시지가 수신된 시각입니다.
  - 대화방 이름/멤버: 메시지가 전달된 대화방 이름이 노출되고 대화방 이름이 없을 때에는 사용자의 이름이 노출됩니다.

### PC 웹

- 대화방 이름/멤버: 메시지가 전달된 대화방 이름이 노출되고 대화방 이름이 없을 때에는 사용자의 이름이 노출됩니다.
  - 대화방 이름을 클릭하면 대화방에 포함된 사용자 중 보낸 사람을 제외한 사용자의 이름과 조직/직책을 확인할 수 있습니다.
  - 사용자 이름을 클릭하면 이 사용자가 받은 메시지를 검색하거나 보낸 메시지를 검색할 수 있습니다. 사용자가 구성원일 경우 조직도에서 구성원을 조회할 수 있습니다.
- 보낸 사람: 메시지를 보낸 사용자 이름이 노출됩니다. 사용자 이름을 누르면 이 사용자가 받은 메시지를 검색하거나 보낸 메시지를 검색할 수 있습니다. 사용자가 구성원일 경우 조직도에서 구성원을 조회할 수 있습니다.
- 메시지 / 파일명: 대화방에 받거나 보낸 가장 최근의 메시지나 파일명입니다. 누르면 해당 '대화방 내용 조회' 화면이 노출됩니다.
- 수신시각: 해당 메시지가 수신된 시각입니다.
- 다운로드: 검색 결과를 다운로드 합니다.

### 메일 검색 결과

구분에서 메일를 선택한 이슈의 검색 결과입니다.

### 모바일 앱

- 검색 결과로 다음과 같은 정보를 확인할 수 있습니다.
  - 메일 제목/본문: 메일의 제목과 본문이 노출됩니다.
  - 소유자: 메일을 보관하고 있는 사람입니다.
  - 수신/송신시각: 메일을 보내거나 받은 시각입니다.

### PC 웹

- 소유자: 메일을 보관하고 있는 사람입니다.
- 메일 제목/본문: 메일의 제목과 본문이 노출됩니다. 제목 영역을 클릭하면 메일 내용 확인을 할 수 있는 화면이 노출됩니다.
- 첨부: 메일에 첨부파일이 있을 경우 ![](/wp-content/uploads/2024/04/icon-clip.png) 아이콘이 노출됩니다.
- 수신/송신시각: 메일을 보내거나 받은 시각입니다.
- 다운로드: 건별로 다운로드 합니다. 왼쪽의 ![](/wp-content/uploads/2024/04/icon-message-pc-web-contact-checkbox.png) 아이콘을 클릭한 후 '다운로드'를 클릭합니다.
- 전체 다운로드: 검색 결과 전체를 다운로드 합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
