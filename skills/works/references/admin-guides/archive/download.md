# 다운로드

이 기능은 아카이빙 옵션 상품 신청 후 사용 가능합니다.

요청한 검색결과를 다운로드 받을 수 있습니다.

이슈 관리에서 다운로드 요청한 검색결과가 있을 때 '다운로드' 화면의 목록에 노출됩니다.

요청한 다운로드 항목은 다운로드 가능 시점부터 15일 이후에 삭제됩니다.

### 모바일 앱

1. Admin 왼쪽 메뉴에서 '아카이빙'을 선택해 메뉴를 펼친 후 '다운로드'를 눌러 '다운로드' 화면으로 이동합니다.

- 다운로드 항목의 정보를 다음과 같이 제공합니다.
  - 구분: 메시지 또는 메일
  - 다운로드명: 다운로드를 요청할 때 입력한 이름입니다.
  - 만료시각: 다운로드를 할 수 있는 기간이 종료되는 시각입니다. 만료시각이 지나면 목록에서 자동 삭제됩니다.
  - 처리상태
    - 다운로드: 검색 결과를 파일로 다운로드할 수 있습니다.
    - 준비중: 다운로드 대기 상태입니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '아카이빙'을 선택해 메뉴를 펼친 후 '다운로드'를 눌러 '다운로드' 화면으로 이동합니다.

- 구분: 메시지 또는 메일
- 다운로드명: 다운로드를 요청할 때 입력한 이름입니다.
- 요청자: 다운로드를 요청한 사용자의 이름입니다.
- 용량: 다운로드 파일의 용량입니다.
- 요청시각: 다운로드를 요청한 시각입니다.
- 만료시각: 다운로드를 할 수 있는 기간이 종료되는 시각입니다. 만료시각이 지나면 목록에서 자동 삭제됩니다.
- 처리상태
  - 다운로드: 검색 결과를 파일로 다운로드할 수 있습니다.
  - 준비중: 다운로드 대기 상태입니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
