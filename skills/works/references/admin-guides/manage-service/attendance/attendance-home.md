# 근태 메인

이 기능은 웍스 근태 상품 신청 후 사용 가능합니다.

  
근태 메인에서는 근태 관련 주요 상황과 시간대별 근태 관리 현황을 조회할 수 있으며, 추가된 연차 기준 관리, 연차 확인, 근무 방식 설정, 근무 대상자 추가, 부재 항목 설정, 특별 휴가 설정, 경조 구분 설정, 출장 기준 설정 항목으로 근태 관련 현황을 조회하거나 설정할 수 있습니다. 또한 콘텐츠 영역에 표시되는 카드 또는 정보를 클릭하면 해당 기능을 바로 사용할 수 있습니다.

근태 메인을 사용하려면 근태 서비스에서 화면 왼쪽 상단의 '근태'를 클릭합니다.

근태 메인 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 콘텐츠 영역에 표시할 콘텐츠를 선택  - Getting Started: 근태 서비스를 이용하기 위한 설정 - Dashboard: 근태 관련 요약 정보를 카드 형태로 표시 - Timeline: 근태 서비스 내에서 기록된 로그를 최신 순으로 표시 |
| ② 콘텐츠 영역 | - 선택된 콘텐츠에 따라 근태 관련 조회 및 설정 항목 표시 - 카드 형태로 표시되는 항목 또는 로그 내역을 클릭하면 해당 메뉴로 이동 - 카드의 '영상 보기'를 클릭하면 해당 항목에 대한 안내 영상 재생 - '?'을 클릭하면 해당 항목에 대한 설명을 표시 |

# Getting Started

근태 서비스에 처음으로 접속하면 근태 관련 정보 설정을 안내하는 Getting Started 화면이 나타납니다.

화면 상단에 근태 관리자가 설정할 주요 항목에 대한 안내 영역이 표시됩니다. 안내에 따라 설정을 완료해 주십시오. 이 3가지 항목이 모두 완료된 경우 해당 영역은 보이지 않습니다.

- 근무 방식 추가하기: 우리 회사에 맞는 근무 시간, 휴게 시간을 설정합니다. 새로운 근무 방식을 추가하거나, 기본 제공된 근무 방식을 수정하여 사용할 수 있습니다.
- 대상자 지정하기: 근무 방식을 생성한 후 해당 근무 방식으로 근무할 대상자를 지정합니다.
- 연차 기준 추가하기: 우리 회사의 연차 기준을 설정합니다.

주요 항목 설정을 완료한 후, 근태 서비스를 원활히 사용하기 위해서는 Getting Started 화면의 카드 순서에 따라 모든 항목의 설정을 완료해 주십시오. 카드 형식의 각 설정 항목은 처음에 닫힌 형태로 표시되며, 한 번 클릭하면 카드가 열려 기본 반영 사항이 표시되고, 한 번 더 클릭하면 해당 메뉴로 이동합니다. 

| 영역 | 설명 |
| --- | --- |
| ① 연차 방식 관리 | 연차 발생 기준을 설정할 수 있는 [연차 방식 관리](/ko/admin-guides/manage-service/attendance/manage-absence/#i-3) 메뉴로 이동  - 영상 보기: 클릭하면 설정 안내 영상 확인 가능 |
| ② 연차 확인 | 임직원의 연차를 조회하고 관리할 수 있는 [연차 조회/수정](/ko/admin-guides/manage-service/attendance/attendance-status/#i-6) 메뉴로 이동  - 영상 보기: 클릭하면 설정 안내 영상 확인 가능 |
| ③ 근무 방식 설정 | 근무 방식을 회사 규정에 맞게 설정할 수 있는 [근무 방식 관리](/ko/admin-guides/manage-service/attendance/work-method/#i-2) 메뉴로 이동  - 영상 보기: 클릭하면 설정 안내 영상 확인 가능 |
| ④ 근무 대상자 추가 | 등록된 근무 방식에 해당하는 임직원을 지정하고 관리할 수 있는 [근무 대상자 관리](/ko/admin-guides/manage-service/attendance/work-method/#i-1) 메뉴로 이동  - 영상 보기: 클릭하면 설정 안내 영상 확인 가능 |
| ⑤ 부재 항목 설정 | 회사에서 관리할 부재 일정 항목을 설정할 수 있는 [부재 항목 관리](/ko/admin-guides/manage-service/attendance/manage-absence/#i-1) 메뉴로 이동 |
| ⑥ 특별 휴가 설정 | 회사 규정에 맞게 특별 휴가를 설정할 수 있는 [특별 휴가 관리](/ko/admin-guides/manage-service/attendance/manage-absence/#i-5) 메뉴로 이동 |
| ⑦ 경조 구분 설정 | 경조 휴가 일수와 경조금을 관리할 수 있는 [경조 기준 관리](/ko/admin-guides/manage-service/attendance/manage-absence/#i-6) 메뉴로 이동 |
| ⑧ 출장 기준 설정 | 출장지에 따른 출장비 경비 지급 기준을 설정할 수 있는 [출장 기준 관리](/ko/admin-guides/manage-service/attendance/manage-absence/#i-7) 메뉴로 이동 |
| ⑨ 데이터 이관 | 임직원의 연차 정보를 업로드해 관리할 수 있는 [데이터 불러오기](/ko/admin-guides/manage-service/attendance/attendance-settings/#i-4) 메뉴로 이동 |

# Dashboard

Getting Started 화면의 모든 설정을 완료한 후에는, 근태 서비스에 접속하면 Dashboard 화면이 나타납니다. Dashboard 화면에는 근태 관련 요약 정보가 카드 형태로 표시됩니다.

Dashboard 화면에 표시되는 카드는 다음과 같습니다.

| 영역 | 설명 |
| --- | --- |
| ① 오늘의 부재자 | - 모든 부재 일정 신청 건 중에서 오늘 부재인 건수 표시 - 클릭하면 [부재 일정 현황](/ko/admin-guides/manage-service/attendance/attendance-status/#i-5) 메뉴로 이동하며, 금일 부재 현황 목록 표시 |
| ② 이달의 연차 촉진 대상 | - 이번 달 연차 촉진 대상자 수와 완료자 수를 표시 - 클릭하면 [연차 촉진 대상 관리](/ko/admin-guides/manage-service/attendance/manage-absence/#i-4) 메뉴로 이동하며, 금월 연차 촉진 대상 목록 표시 |
| ③ 이달의 출장비 | - 부재 항목이 '출장'이며 이번 달 첫날부터 말일 사이에 결재가 완료된 부재일정 신청에 대한 출장비의 합을 표시 - 클릭하면 [출장 신청 내역](/ko/admin-guides/manage-service/attendance/attendance-status/#i-9) 메뉴로 이동하며, 금월 출장 신청 목록 표시 |
| ④ 이달의 경조금 | - 부재 항목이 '경조'이며 이번 달 첫날부터 말일 사이에 결재가 완료된 부재일정 신청에 대한 경조비의 합을 표시 - 클릭하면 [경조 신청 내역](/ko/admin-guides/manage-service/attendance/attendance-status/#i-8) 메뉴로 이동하며, 금월 경조 신청 목록 표시 |
| ⑤ 오늘의 지각 | - 오늘 지각한 임직원 수 표시 - 클릭하면 [출퇴근 현황](/ko/admin-guides/manage-service/attendance/attendance-status/#i) 메뉴로 이동하며, 금일 출퇴근 현황 목록 표시 - 지각을 사용하지 않는 근무 방식에 속한 임직원은 집계되지 않음 |
| ⑥ 오늘의 조퇴 | - 오늘 조퇴한 임직원 수 표시 - 클릭하면 [출퇴근 현황](/ko/admin-guides/manage-service/attendance/attendance-status/#i-1) 메뉴로 이동하며, 금일 출퇴근 현황 목록 표시 - 조퇴를 사용하지 않는 근무 방식에 속한 임직원은 집계되지 않음 |
| ⑦ 오늘의 의무 근로 미준수 | - 오늘 의무 근로를 미준수한 임직원 수 표시 - 클릭하면 [출퇴근 현황](/ko/admin-guides/manage-service/attendance/attendance-status/#i-1) 메뉴로 이동하며, 금일 출퇴근 현황 목록 표시 - 의무 근로 준수 여부를 사용하는 근무 방식에 속한 임직원에 대해서만 집계 |
| ⑧ 근무 방식 없는 대상자 | - 근무 방식이 지정되지 않은 임직원 수 표시 - 클릭하면 [근무 대상자 관리](/ko/admin-guides/manage-service/attendance/work-method/#i-1) 메뉴로 이동 |
| ⑨ 근무 마감 내역 | - 가장 최근의 근무 마감 내역 표시 - 클릭하면 '근로 시간 마감' 메뉴의 [마감 내역](/ko/admin-guides/manage-service/attendance/attendance-status/#i-4) 탭으로 이동 |

# Timeline

근태 서비스의 최근 한 달간 사용 로그가 100건까지 최신 순으로 표시됩니다. 각 항목의 링크를 클릭하면 관련 메뉴로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
