# 설정

이 기능은 웍스 근태 상품 신청 후 사용 가능합니다.

  
근태 서비스의 각종 하위 메뉴를 사용 또는 미사용으로 설정하거나 출퇴근 버튼 명칭을 관리하며, 부재 현황 조회 범위와 권한을 설정할 수 있습니다. 또한 웍스 근태 사용 이전, 외부 솔루션의 연차 내역을 이관하여 사용할 수 있습니다.

# 근태 설정

근무 계획 사용 여부, 부재 상태 공유 필수 여부 및 연차 사용 촉진제 적용 여부를 설정할 수 있습니다.

근태 설정 화면은 다음과 같이 구성되어 있습니다.  

| 영역 | 설명 |
| --- | --- |
| ① 근무 계획 사용 여부 | 근무 계획 사용 여부 설정  - 신청 가능 기간: 근무 계획 사용 시 표시 |
| ② 신청 가능 기간 | 근무 계획 사용 여부를 사용으로 선택 시 표시  - 근무 계획 신청은 기준일 전에 (근무를 수행하기 이전에) 작성하는 것이 바람직하지만, 특정 사유로 미리 근무 계획을 작성하지 못했더라도 예외적으로 기준일 이후 계획을 작성할 수 있도록 옵션화 함 - 기준일 당일: 시간(시/분)을 세밀하게 정하여 근무 계획 신청서 작성 가능 - 기준일 당일 이외: 3일 후/7일 후/10일 후/마감전까지 근무 계획 신청서 작성 가능 - 신청 가능 기간 내에 근무 계획 미작성 시 시간 외 근로 시간 집계를 위한 근무 결과 작성 불가 |
| ③ 부재 일정 공유 설정 | 부재 일정 서식의 각 공유 항목의 필수 여부를 설정  - 부재 시간:   - 필수 선택: 선택 시 반일 단위, 시간 단위 부재 일정은 시간 입력 필수 - 캘린더 등록: 캘린더 등록 필수 여부 선택 가능   - 필수 선택: 선택 시 부재 일정 신청 화면에 캘린더 등록은 등록으로 표시되며 변경 불가, 선택하지 않을 경우 부재 일정 신청자가 직접 캘린더 등록 여부를 선택 가능   - 기본 캘린더 자동 선택: 캘린더 솔루션의 기본 캘린더에 등록   - 개인 캘린더 직접 선택: 개인 캘린더를 직접 선택해 등록 - 부재 메시지:   - 필수 선택: 선택 시 부재 일정 신청 화면에 부재 메시지 입력 필수 |
| ⑤ 연차 사용 촉진제 적용 | 연차 사용 촉진제 적용 여부 설정  - 사용 선택 시 '근태 > 부재 > 연차 촉진 대상 관리' 메뉴가 표시되며 연차 종료일 기준의 연차 사용 촉구 시기 관리 가능. '결재 홈 > 문서 작성'에 연차 사용 계획 서식이 표시됨 - 연차 사용 촉진제를 적용할 경우 '근태 > 부재 > 연차 기준 관리'의 연차 이월 여부를 '이월 안 함'으로 설정 권장([연차 방식 관리](/ko/admin-guides/manage-service/attendance/manage-absence/#i-3) 참고) |
| ⑥ 취소 | 설정한 내용을 삭제하고 설정 전의 화면으로 새로 고침 |
| ⑦ 저장 | 설정한 내용 저장 |

연차 사용 촉진제는 근로자의 권리인 연차 유급 휴가 사용을 장려하기 위해 도입된 제도로, 연차 유급 휴가의 사용 기간이 만료되기 전 회사에서 근로자에게 잔여 연차 일수를 알려주고 사용할 수 있도록 촉진하는 제도입니다.

# 출퇴근 버튼 명칭 관리

출퇴근 버튼 명칭을 설정할 수 있습니다.

출퇴근 버튼 명칭 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 출퇴근 버튼 | 근무 시작 시점과 종료 시점을 의미하는 버튼 명칭을 설정  - ON: 근무 시작 시점을 체크하는 버튼명 선택 - OFF: 근무 종료 시점을 체크하는 버튼명 선택 - PAUSE: 근무 시간 중에 휴게를 기록하는 버튼명 선택 |
| ② 취소 | 설정한 내용을 삭제하고 설정 전의 화면으로 새로 고침 |
| ③ 저장 | 설정한 내용 저장 |

# 부재 현황 조회 설정

회사 전체 혹은 사용자 본인 소속 조직(겸직 포함)의 부재 현황을 조회할 수 있도록 조회 권한과 범위를 설정할 수 있습니다.

부재 현황 조회 설정 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 조회 권한 부여 | 부재 현황을 조회할 수 있는 대상과 조회 가능 범위 설정  - 대상: 부서장/부서/사원 중 선택 - 조회 범위: 전체 부서/내 부서/조직장인 부서 중 선택 |
| ② 해제 | 설정한 조회 권한 내용을 해제 |
| ③ 취소 | 설정한 내용을 삭제하고 설정 전의 화면으로 새로 고침 |
| ④ 저장 | 설정한 내용 저장 |

부재 현황 조회 권한을 설정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '근태'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '조회 권한 부여'를 클릭합니다.
   - 대상: 대상 항목에서 조회 권한을 부여할 대상을 선택합니다. 그룹사로 사용 시 원소속 회사를 기준으로 합니다.  

     - 부서장: 우리 회사의 부서장(겸직 부서장 포함, 그룹사 소속의 겸직 사원 불포함)
     - 부서: 선택한 우리 회사의 부서 부서원(하위 조직 및 겸직 부서 포함, 그룹사 소속의 겸직 사원 불포함)
   - 조회 범위: 조회 범위를 선택합니다.
     - 전체 부서: 우리 회사의 전체 부서(그룹사 소속의 겸직 사원 포함)
     - 내 부서: 우리 회사의 내 소속 부서(하위 조직 및 겸직 부서 포함, 그룹사 소속의 겸직 사원 포함 )
     - 부서장인 부서: 조회 대상이 부서장인 경우 표시되는 항목, 겸직 부서를 포함한 내가 부서장인 우리 회사 부서(그룹사 소속의 겸직 사원 포함)
   - 삭제: 설정한 조회 권한을 삭제하려면 클릭하여 삭제할 수 있습니다.
5. '저장'을 클릭합니다.

- 동일 대상자에 부여된 조회 범위 중 가장 큰 범위의 현황을 조회할 수 있습니다.
- 그룹사로 사용 시 조회 대상은 원소속 회사에서 지정하며, 조회 범위는 원소속 회사를 기준으로 각 회사별 적용합니다. 겸직 발령을 받은 사원은 원소속 회사와 겸직 회사의 부재 현황 조회 설정을 함께 적용합니다.

# 데이터 불러오기

사용하던 타 서비스의 연차 정보를 이관할 수 있습니다. 업로드한 내역은 사원의 남은 휴가일에 반영됩니다.

연차 내역을 연중에 이관하는 경우, 사원별 사용한 연차를 반영하여 남은 휴가일을 관리할 수 있습니다.

데이터를 불러오기 위해 다음 절차를 차례대로 진행해 주십시오.

1. 연차 기준 관리 설정
2. 데이터 불러오기

## 1. 연차 기준 관리 설정

사원별 연차 정보를 등록하기 전에 연차 발생 기준과 근속 연수별 연차 일수를 설정해 주십시오.  
연차 발생 기준 설정에 대한 자세한 내용은 [연차 방식 관리](/ko/admin-guides/manage-service/attendance/manage-absence/#i-3)를 참고해 주십시오.

## 2. 데이터 불러오기

### 연차 일수 업로드

올해 연차 사용 일수, 조정 일수를 이관할 수 있습니다. 업로드한 연차 내역은 사원의 올해 남은 휴가일에 반영됩니다.

연차 일수 데이터를 불러오는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '근태'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 올해 사용한 연차 내역을 업로드하려면, 데이터 불러오기 화면에서 Step2의 '업로드'를 클릭해 템플릿 파일을 다운로드합니다
   - 샘플 파일이 엑셀 파일로 다운로드됩니다.
   - 과거에 사용한 부재 내역의 업로드는 Step3의 '과거 부재 내역 업로드'를 참고해 주십시오.
5. 템플릿 파일을 참고하여 연차 정보를 작성한 후 저장합니다.
6. Step2의 '업로드'를 클릭한 후 연차 정보 파일을 선택한 다음 '열기'를 클릭합니다.
7. 연차 정보 일괄 추가 목록에서 업로드한 연차 정보 내용과 처리 상태를 확인합니다.
   - 업로드 실패 시 실패 사유를 확인할 수 있습니다. 오류 원인을 해결한 후 '다시 올리기'를 클릭해 다시 업로드합니다.
   - 업로드 이력 목록에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_detail.png)을 클릭하면 날짜별 업로드 이력을 조회할 수 있습니다.

### 과거 부재 내역 업로드

올해 연도 이전에 사용한 부재 내역을 이관할 수 있습니다. 업로드한 내역은 부재 일정 현황에 반영됩니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '근태'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 데이터 불러오기 화면에서 Step3의 '업로드'를 클릭해 템플릿 파일을 다운로드합니다.
   - 샘플 파일이 엑셀 파일로 다운로드됩니다.
5. 템플릿 파일을 참고하여 부재 정보를 작성한 후 저장합니다.
6. Step3의 '업로드'를 클릭한 후 부재 정보 파일을 선택한 다음 '열기' 버튼을 클릭합니다.
7. 부재 정보 일괄 추가 목록에서 업로드한 부재 내역과 업로드 처리 상태를 확인합니다.
   - 업로드 실패 시 실패 사유를 확인할 수 있습니다. 오류 원인을 해결한 후 '다시 올리기' 버튼을 클릭해 다시 업로드합니다.
   - 업로드 이력 목록에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_detail.png)을 클릭하면 날짜별 업로드 이력을 조회할 수 있습니다.
   - 올해 연차 내역 업로드 시 사용 연차에 반영됩니다.
   - 경조나 출장 내역 업로드 시 '경조 신청 내역', '출장 신청 내역'에 제공하지 않습니다.(경조 구분/출장지 등 정보를 받지 않음)

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
