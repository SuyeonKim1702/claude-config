# 근태 현황

이 기능은 웍스 근태 상품 신청 후 사용 가능합니다.

  
근무 방식을 지정 받은 사원이 출퇴근 체크를 통해 근무 기록을 남깁니다. 사원들이 입력한 근무 기록은 '출퇴근 현황'과 '근로 시간 현황' 메뉴를 통해 확인할 수 있습니다.

# 출퇴근 현황

사원들의 출퇴근 현황을 관리하고 필요 시 특정 일의 출근 및 퇴근 시간을 수정하거나, 휴게 시간 혹은 외부 근무 시간을 추가할 수 있습니다.![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/bs-ko-support-attendance-status-pc_1.png)

출퇴근 현황 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 사원의 출퇴근 현황을 근무 방식, 기간, 사원명 조건에 따라 검색 |
| ② 업로드 | [출퇴근 기록 업로드](#i-1-1) |
| ③ 다운로드 | [출퇴근 기록 다운로드](#i-1-2) |
| ④ 출퇴근 현황 목록 | 사원의 날짜별 출퇴근 현황 확인  - 근무 방식: 조회된 사원의 근무 방식. 당일 인사 발령 시, 출퇴근 기록이 없으면 근무 방식이 기존 방식으로 조회될 수 있음 - 근무 일정: 시차 근무제, 선택 근무제에서 '근무 일정 상신'을 적용받는 사원의 근무 일정 표시 - 출근/퇴근:   - 시각: 출퇴근 ON/OFF를 체크한 시간   - 체크 환경: 출퇴근 체크 환경 표시, 필터 항목(PC, 모바일, 결재, 간주, 관리자 등록)은 표 아래의 참고 박스 확인 - 근무 위치: 근무 위치를 사용하는 경우 항목 표시 및 사용자가 근무 위치를 입력한 경우 근무 위치 반영 - 휴게: 사원의 총 휴게 시간. '회사에서 설정한 휴게 시간 + 사용자가 추가한 시간'이 표시. 근무 방식 설정 시 입력된 휴게 시간이 없거나 사용자가 추가 휴게 시간을 입력하지 않았을 경우 빈 칸으로 표시 - 외부 근무: 사용자의 외부 근무 시간. 근무 방식 설정에서 근로 시간에 합산되는 외부 근무 추가를 허용하지 않거나 사용자가 추가 외부 근무 시간을 입력하지 않았을 경우 빈 칸으로 표시 - 부재: 근로 시간에 합산되는 부재 일정을 신청한 경우 신청한 부재 내역을 시간 단위로 표시. 하루에 다양한 종류의 부재 신청 내역이 있는 경우 기준 근로 시간 (선택 근무제인 경우 표준 근로 시간)까지 합산하여 표시 - 지각: 지각을 사용하는 경우, 사용자가 출근을 체크한 시각이 지각에 해당 시 'v' 표시 - 조퇴: 조퇴를 사용하는 경우, 사용자가 퇴근을 체크한 시각이 조퇴에 당 시 'v' 표시 - 의무 근로 준수: 의무 근로 시간 준수를 사용하는 경우, 사용자의 출근/퇴근 체크 시각이 의무 근로 준수에 해당 시 'O' 표시. 부재 사용 등으로 판단 불가일 경우 '-' 표시. 의무 근로를 체크하지 않은 요일은 빈 칸으로 표시 - 근무 일정 준수: 시차 근무제, 선택 근무제에서 '근무 일정 상신'을 적용받는 사원의 근무 일정 준수 여부 표시, 준수에 해당 시 'O' 표시 - 근무 일정 대비 과부족: 시차 근무제, 선택 근무제에서 '근무 일정 상신'을 적용받는 사원의 근무 일정 대비 근로 시간 과부족 여부  표시, 과부족에 해당 시 'O' 표시 - 지각, 조퇴, 의무 근로 준수 항목은 근무 방식에서 설정된 지각, 조퇴, 의무 근로 준수를 사용하는 경우에만 표시 |
| ⑤ 상세 | 클릭하면 사원의 상세 출퇴근 정보 확인 및 출근, 퇴근, 휴게 시간, 외부 근무 시간 수정  - 단, 마감된 일자의 기록은 수정 불가 - 부재 항목은 사용자가 신청한 근로 시간에 합산되는 부재 일정이 있을 경우 표시 - '초기화'를 클릭하여 당일 출퇴근 기록 초기화 가능([당일 출퇴근 시간 초기화](#i-1-3) 참조) |

출퇴근 체크 환경 표시에 대한 설명:

- PC: 사용자가 PC 화면에서 체크
- 모바일: 사용자가 모바일 앱에서 체크
- WorkTalk: WorkTalk에서 출퇴근을 입력한 경우('현재 시간' 방식으로 입력 시에 표시되며, '직접 입력' 시 환경에 따라 'PC' 또는 '모바일'로 표시됩니다.)
- 결재: 근무 결과 신청 서식을 상신
- 간주: 근무 방식 설정에서 '간주 근무' 선택
- 관리자 등록: 근태 관리자가 출퇴근 시간을 입력
