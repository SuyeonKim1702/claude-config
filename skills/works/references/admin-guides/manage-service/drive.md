# 드라이브

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

  

- [드라이브 일반 설정](https://help.worksmobile.com/ko/admin-guides/manage-service/drive/general-settings/)
- [개인 폴더](https://help.worksmobile.com/ko/admin-guides/manage-service/drive/personal-folder-settings/)
- [드라이브 파일 이전](https://help.worksmobile.com/ko/admin-guides/manage-service/drive/transfer-drive-files/)

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
