# 노트 이전

이 기능은 클로바노트 상품 신청 후 사용 가능합니다.

구성원이 보유한 노트를 다른 구성원에게 이전할 수 있습니다.

구성원이 퇴사하거나 계열사로 이동하기 전, 노트가 삭제되거나 다른 도메인으로 이전되기 전에 현재 도메인에서 노트를 계속 사용할 수 있습니다.

# 노트 이전 설정

구성원이 보유한 노트를 다른 구성원에게 한 번에 이전할 수 있습니다.

이전에는 일정 시간이 소요되며, 완료되면 노트를 받은 구성원의 폴더 목록에 새로운 폴더가 생성되고 해당 폴더에 노트가 저장됩니다.

새 폴더 이름은 ‘TF\_이전 보낸 구성원명’ 형식으로 생성됩니다.

1. Admin 왼쪽 메뉴에서 ‘서비스’를 선택해 메뉴를 펼친 후 ‘Default’를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/06/ham.png) 버튼을 탭하면 메뉴가 나타납니다.
2. ‘노트 이전’을 선택합니다.
3. 이전할 구성원의 이름을 입력합니다.
4. ‘다음’을 클릭합니다.
5. 노트를 받을 구성원의 이름을 입력합니다.
6. ‘노트 이전’을 클릭합니다.

# 파일 이전 히스토리 확인

이전된 노트의 히스토리를 확인할 수 있습니다.

받은 구성원과 보낸 구성원, 이전 날짜, 상태, 이전된 노트 개수를 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 ‘서비스’를 선택해 메뉴를 펼친 후 ‘Default’를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/06/ham.png) 버튼을 탭하면 메뉴가 나타납니다.
2. ‘노트 이전’을 선택합니다.
3. 하단의 ‘이전 히스토리 보기’ 메뉴를 선택합니다.
4. 내용을 확인합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
