# 노트 생성 제한

특정 조직 또는 구성원이 노트를 생성할 수 없도록 제한할 수 있습니다.

제한된 구성원은 노트 열람만 가능하며, 생성, 편집, 공유, 다운로드 등을 할 수 없습니다.

## 노트 생성 제한 추가

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '특정 구성원의 생성 및 편집 제한' 영역 오른쪽의 '관리' 버튼을 누릅니다.
4. 화면 오른쪽 상단의 '구성원 추가'를 누릅니다.
5. 노트 생성을 제한하고자 하는 구성원을 선택한 후 '확인'을 누릅니다. 최대 100명까지 한 번에 추가할 수 있습니다.

## 노트 생성 제한 해제

구성원의 노트 생성 제한을 해제할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '특정 구성원의 생성 및 편집 제한' 영역 오른쪽의 '관리' 버튼을 누릅니다.
4. 제한을 해제하려는 구성원 오른쪽의 '삭제' 버튼을 누릅니다.
5. 주의사항을 확인한 후 '확인'을 누릅니다.

여러 구성원의 노트 생성 제한을 한 번에 해제하려면 구성원 왼쪽 상단의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-16-todo-uncheck.png) 아이콘을 선택한 후 왼쪽 상단의 '삭제'를 누릅니다.

화면 오른쪽 상단의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-12-search-434343.png) 아이콘을 클릭해 특정 구성원을 검색할 수도 있습니다.

# 노트 기능 제한

클로바노트의 주요 기능을 사용할지 여부를 설정합니다.

설정은 모든 구성원에게 일괄 적용됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '모든 구성원의 노트 기능 제한' 목록에서 제한할 기능 왼쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-16-todo-uncheck.png) 아이콘을 선택하거나 선택 해제합니다.
   - 공유 받은 노트의 음성 재생 제한: 공유 받은 노트의 음성 파일 재생을 제한합니다. 단, 직접 생성한 노트의 음원 파일은 구성원 본인에 한해 재생 가능합니다.
   - 음성 기록 다운로드 제한: 노트의 음성을 기록한 텍스트 파일 다운로드를 제한합니다.
   - 메모 다운로드 제한: 노트의 메모 파일 다운로드를 제한합니다.
   - 노트 공유 제한: 노트의 공유를 제한합니다. 제한 설정 전 생성했던 노트 공유 링크도 모두 만료됩니다.
4. '저장'을 누릅니다.

# 노트 자동 삭제 기간 설정

구성원이 생성한 노트를 자동 삭제할지 여부와 기간을 설정합니다.

기간을 설정하면 구성원이 노트를 생성한 일자를 기준으로 해당 기간이 지났을 때 노트가 자동으로 삭제됩니다.

설정은 모든 구성원에게 일괄 적용됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '노트 자동 삭제 기간 설정' 영역을 클릭합니다.
4. 자동 삭제 처리될 기간을 선택합니다.
5. '저장'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
