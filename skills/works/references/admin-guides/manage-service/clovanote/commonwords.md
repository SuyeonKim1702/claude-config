# 자주 쓰는 단어

이 기능은 클로바노트 상품 신청 후 사용 가능합니다.

언어별로 노트에서 자주 쓰는 단어, 고유 명사, 전문 용어를 등록할 수 있습니다.

자주 쓰는 단어를 등록하면 해당 단어들이 포함된 발화의 인식 정확도가 높아집니다.

관리자가 등록한 자주 쓰는 단어(최대 1,000개)는 모든 구성원에게 공통으로 적용되어, 구성원의 '자주 쓰는 단어' 설정에 노출됩니다.

// start of content for jp

단, 관리자의 자주 쓰는 단어 입력 기능은 추후 출시 예정인 AiNote 유료 플랜 신청 후 사용 가능합니다.

// end of content for jp

각 구성원이 등록한 자주 쓰는 단어(최대 1,000개)는 구성원 본인에게만 적용됩니다.

단어는 최소 두 글자 이상이어야 하며, 대소문자를 구분합니다. 영문자와 숫자를 포함해 최대 20자까지 입력 가능합니다.

# 자주 쓰는 단어 등록

// start of content for jp

이 기능은 추후 출시 예정인 AiNote 유료 플랜 신청 후 사용 가능합니다.

// end of content for jp

자주 쓰는 단어를 등록하고 관리합니다.

등록된 단어는 최신순대로 상단 왼쪽에 노출됩니다.

한번 등록된 단어는 수정할 수 없으며, 필요 시 x 아이콘을 눌러 삭제 후 재등록해야 합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '자주 쓰는 단어' 탭을 선택합니다.
3. 한국어, 영어, 일본어, 중국어(간, 번체) 중 자주 쓰는 단어를 등록하고자 하는 언어의 오른쪽 '관리' 메뉴를 누릅니다.
4. '자주 쓰는 단어 추가' 영역에 직접 입력 후 '추가' 버튼을 누르거나, '파일 업로드'를 눌러 파일로 일괄 등록합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
