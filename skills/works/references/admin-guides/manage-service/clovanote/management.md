도메인 내 모든 노트를 검색하고 관리할 수 있습니다.

노트의 제목, 생성자 등의 정보를 열람하고, 필요 시 해당 노트를 삭제하거나 일부 기능을 제한할 수 있습니다.

단, 관리자는 노트의 내용은 볼 수 없습니다.

# 노트 검색

서비스 환경 내에서 생성된 모든 노트를 검색하고 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '노트 관리' 탭을 선택합니다.
3. 노트 검색 목록 오른쪽 상단의 '상세' 버튼을 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-12-search-434343.png) 아이콘을 탭합니다.  
   검색창에 생성 기간으로 검색하거나 상세'를 눌러 항목별로 상세한 검색을 할 수도 있습니다. 상세 검색에서는 노트의 '제목', '생성자', '노트 ID', '공유 URL'을 기준으로 검색할 수 있습니다.
4. '검색'을 누릅니다.

# 노트 삭제

관리자가 노트를 삭제할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '노트 관리' 탭을 선택합니다.
3. 삭제하려는 노트 오른쪽의 '삭제'를 누르거나 왼쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-16-todo-uncheck.png) 아이콘을 선택한 후 왼쪽 상단의 '삭제' 버튼을 누릅니다.
4. 주의 사항을 확인한 후 '삭제'를 누릅니다.

# 노트 관리

관리자가 노트 현황을 확인하고, 개별 노트의 기능을 관리할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '클로바노트'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '노트 관리' 탭을 선택합니다.
3. 목록에서 현황을 확인할 노트 제목을 누릅니다.
4. 현황을 확인하고 개별 기능을 제한하거나 설정을 변경합니다.
5. 하단의 '저장'을 눌러 변경 사항을 저장합니다.

노트 데이터 보관 안내

사용자가 노트를 영구 삭제하면 노트 데이터는 삭제되며 복구할 수 없습니다.

관리자가 특정 구성원을 삭제하는 경우, 해당 구성원이 생성한 모든 노트는 삭제되며 복구할 수 없습니다.

휴지통에 있는 노트는 1년간 보관 후 자동 삭제되어 복구할 수 없습니다.

음성 파일의 보관 기간은 *이용 플랜에 따라 최소 1년부터 최대 5년입니다.*

// end of content for jp

저장 기간 경과 후에는 음성 파일이 삭제되어 재생할 수 없습니다.

보관이 필요한 음성 파일은 보관 기간 경과 전에 다운로드해 백업해 주세요.

음성 파일을 제외한 나머지 데이터는 유지되며, 변환 기록을 포함해 노트 내용을 정상적으로 열람할 수 있습니다.

단, 클로바노트 상품을 해지하는 경우, 해지일로부터 7일 후 모든 노트의 데이터가 영구 삭제됩니다.
