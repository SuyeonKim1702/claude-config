# 경영지원 설정 메인

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
경영지원 설정 메인에서는 회사 정보, 사업장, 회사 휴일 등 필요한 설정 사항을 설정하고 관리할 수 있습니다.

경영지원 설정 메인을 사용하려면 경영지원 설정에서 화면 왼쪽 상단의 '경영지원 설정'을 클릭합니다.

경영지원 설정 메인 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 콘텐츠 영역에 표시할 콘텐츠를 선택  - [Getting Started](#i-1): 경영지원 설정에 필요한 설정을 카드 형태로 표시 - [Timeline](#i-2): 최근 한 달 동안 경영지원 설정에 기록된 로그를 100건까지 최신 순으로 표시 |
| ② 콘텐츠 영역 | 선택된 콘텐츠에 따라 경영지원 설정 항목 표시. 카드 형태로 표시되는 항목 또는 로그 내역을 클릭하면 해당 메뉴로 이동 |

# Getting Started

경영지원 설정에 처음으로 접속하면 경영지원 사용에 필요한 정보 설정을 안내하는 Getting Started 화면이 나타납니다. 각 설정 항목은 닫힌 카드 형태로 표시되며, 카드를 한 번 클릭하면 카드가 열려 기본 반영 사항이 표시되고, 한 번 더 클릭하면 해당 메뉴로 이동합니다.

경영지원을 원활히 사용하기 위해서는 Getting Started 화면의 카드 순서에 따라 모든 항목의 설정을 완료해 주십시오.

| 영역 | 설명 |
| --- | --- |
| ① 회사 정보 관리 | 회사 정보를 확인하고 수정할 수 있는 [회사 정보 관리](/ko/admin-guides/manage-service/support-management/company-info/#i-1) 메뉴로 이동  - 영상 보기: 클릭하면 설정 안내 영상 확인 가능 |
| ② 사업장 관리 | 사업장을 추가하고 관리할 수 있으며, 사업장별 인증서를 등록하고 관리할 수 있는 [사업장 관리](/ko/admin-guides/manage-service/support-management/company-info/#i-2) 메뉴로 이동 |
| ③ 회사 휴일 관리 | 회사만의 휴일을 등록하고 관리할 수 있는 [회사 휴일 관리](/ko/admin-guides/manage-service/support-management/settings/#i-3) 메뉴로 이동 |

# Timeline

경영지원 설정의 최근 한 달간 사용 로그가 100건까지 최신 순으로 표시됩니다. 각 항목의 링크를 클릭하면 관련 메뉴로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
