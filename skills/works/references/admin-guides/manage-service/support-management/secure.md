# 보안

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

​등록된 회사의 데이터 보안을 위해 관리자 메뉴 접근 시 2단계 인증을 실행하거나, 저장된 데이터를 백업할 수 있습니다. 또한 장애 발생 시, 고객지원에서 데이터에 접근하는 것을 제어할 수 있습니다.

# 2단계 인증 관리

서비스별 관리 메뉴 접근 시의 2단계 인증 적용을 관리할 수 있습니다. 2단계 인증을 사용하면 메뉴 접근 시 관리자 인증을 한 번 더 하게 됨으로써 데이터의 보안 및 안전성을 강화할 수 있습니다.

서비스별 2단계 인증을 사용 설정하는 방법은 다음과 같습니다.

1. PC 웹 화면의 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '경영지원 설정'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)를 클릭합니다.

2차 인증(추가 인증) 유지 시간은 30분입니다. 인증 세션 유지 기간 내에 2차 인증이 필요한 다른 관리 메뉴에 접근할 경우 인증이 필요하지 않을 수 있습니다.

# 고객 지원 접근 허용

경영지원 사용 중 발생한 장애를 처리하기 위해 고객지원의 접근을 허용할 수 있습니다.

고객의 정보에 접근할 수 있도록 '동의'를 선택한 후 '저장'을 클릭합니다.

장애 처리가 완료된 후, '동의'를 해제하여 접근을 차단할 수 있습니다.

# 데이터 내보내기

저장된 모든 데이터를 EXCEL, HTML, PDF 파일, 첨부 파일, PNG, GIF 등 형식으로 상시 다운로드할 수 있습니다. 현재까지 생성한 모든 문서와 이용 중인 서비스에 저장된 정보 전체를 일괄 다운로드하여 보관용으로 활용하거나, 이용 중인 다른 솔루션으로 이관할 수 있습니다.

특히, 결재 문서의 경우 완료(거절 및 취소 포함)된 문서를 개별 PDF로 저장할 수 있어 회사의 중요 문서 보관이 용이합니다. 사용 상품에 따른 서비스별 데이터를 백업할 수 있습니다.

- 오랜 기간 사용한 고객은 생성된 데이터가 많아 전체 데이터를 백업하는데 시간이 소요될 수 있습니다. 이 경우 필요한 서비스를 선택하여 백업하거나 기간을 설정하여 백업할 수 있습니다.
- 백업 신청을 한 후 다운로드 준비가 완료되면 안내 메일이 발송되며, 데이터를 바로 다운로드할 수 있습니다.
- 이용 중인 상품이나 서비스에 한하여 백업할 수 있습니다.
- 결재 문서와 회계 전표 중 진행 중인 정보는 다운로드되지 않습니다.
- 문서 및 전표, 거래처에 첨부된 파일도 다운로드됩니다.
- 문서는 문서 작성일을 기준으로 폴더가 생성되어 다운로드되며, 회계 전표는 전표일을 기준으로 폴더가 생성되어 다운로드됩니다.
- 모든 데이터는 회사가 설정한 언어로 다운로드됩니다.
