# 서비스 개요

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
경영지원 설정에서는 기본 설정 항목을 조회하고 관리할 수 있습니다.

경영지원 설정을 사용하려면 PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '경영지원 설정'을 클릭합니다.

경영지원 설정은 권한이 있는 사용자에게만 노출됩니다.

# 경영지원 설정 시나리오

경영지원 설정의 이용 시나리오는 다음과 같습니다.

1. 회사 정보, 사업장 및 거래처 관리
   - 회사 정보 조회, 수정
   - 사업장 추가, 수정, 삭제
   - 사업장별 인증서 추가, 삭제
   - 거래처 추가
   - 이 단계에 대한 사용 가이드 페이지
     - [Getting Started](/ko/admin-guides/manage-service/support-management/supportmanagement-home/#i-1)
     - [회사 정보](/ko/admin-guides/manage-service/support-management/company-info/)
     - [공통 코드 관리](/ko/admin-guides/manage-service/support-management/common-code/)
2. 경영지원 사용 환경 설정
   - 고객지원 접근 권한 설정
   - 서비스 사용 여부 설정
   - 외부 솔루션 연동 설정
   - 홈 화면 관리 및 고정 링크 메뉴 추가
   - 회사 휴일 등록, 수정, 삭제
   - 이 단계에 대한 사용 가이드 페이지
     - [Getting Started](/ko/admin-guides/manage-service/support-management/supportmanagement-home/#i-1)
     - [고객지원 접근 허용](/ko/admin-guides/manage-service/support-management/secure/#i-3)
     - 서비스 사용 관리
     - 서비스 연동 관리
     - 홈 화면 관리
     - 회사 휴일 관리
     - [설정](/ko/admin-guides/manage-service/support-management/settings/)
3. 감사 로그 조회
   - 사용자 및 관리자의 수행 과업 로그 조회
   - 이 단계에 대한 사용 가이드 페이지
     - [감사](/ko/admin-guides/manage-service/support-management/audit/)
4. 경영지원 사용 현황 확인 및 백업
   - 로그인 기록 확인
   - 데이터 백업
   - 이 단계에 대한 사용 가이드 페이지
     - [로그인 현황](/ko/admin-guides/manage-service/support-management/secure/#i-1)
     - [데이터 내보내기](/ko/admin-guides/manage-service/support-management/secure/#i-4)

# **자주 하는 질문을 확인해 보십시오.**

**Q. 경영지원 사용 시 기본 정보를 설정하는 이유가 무엇인가요?**

- 경영지원은 회사 정보를 생성하고 축적하며 동시에 공유하는 시스템입니다. 따라서 정보 보관 전에 어떤 정보를 어떻게 보관할지, 신청한 서비스에 따라 각 서비스에 대한 기준 설정이 필요합니다.

**Q. 사명이 변경된 경우 변경해야 하는 사항이 무엇인가요?**

- 각 메뉴에서 사명 정보가 입력되어 있는 항목을 수정해 주십시오.
- Admin에서 회사명, 회사 로고를 수정해 주십시오.
- 접속 URL은 관리자가 직접 수정할 수 없으며 고객지원에 연락하여 수정할 수 있습니다.

**Q. 매해 반복되는 회사 창립 기념일을 회사 휴일로 설정하려면 어떻게 해야 하나요?**

- [회사 휴일 추가](/ko/admin-guides/manage-service/support-management/settings/#i-3)를 참고해 주십시오.

**Q. 2개 이상의 언어를 사용할 수 있나요?**

- 항목의 값을 다국어로 입력할 수 있는 기능을 제공합니다. 먼저, Admin에서 다국어 사용 여부 및 사용할 다국어를 선택합니다. 한국어, 영어, 일본어, 중국어(간체), 중국어(번체) 5개의 언어를 지원합니다.

- 사용자는 관리자가 설정한 언어 중에서 본인의 화면에 표시할 언어를 선택할 수 있습니다. 단, 기본 언어로 설정된 항목은 반드시 입력해야 합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
