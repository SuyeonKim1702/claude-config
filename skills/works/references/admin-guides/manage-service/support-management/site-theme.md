# 사이트 테마

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
사이트 테마에서는 홈 화면에 표시할 1개의 고정 링크를 관리할 수 있습니다.

## 고정 링크 관리

홈 화면의 우측 상단에 표시할 고정 링크 메뉴를 설정할 수 있습니다.

1. PC웹 화면의 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '경영지원 설정'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요 시, 홈 고정 링크 메뉴를 입력 후 '저장'을 클릭합니다. 홈 고정 링크 메뉴는 PC 웹에서만 제공되며, 홈의 오른쪽 상단에 표시됩니다. 클릭하면 새창으로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
