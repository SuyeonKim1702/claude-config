# 사업장 권한 관리

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
회계 관리자 권한을 사업장별 또는 사원별로 부여하고 관리할 수 있습니다.

사업장 권한 관리 화면은 다음과 같이 구성되어 있습니다.

# 사업장별 권한 부여

사업장 권한을 부여하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '회계를 클릭합니다.
2. 메뉴에서 '사업장 권한 관리'를 클릭합니다.
3. '사업장별' 탭을 클릭합니다.
4. 사업장 목록에서 권한을 부여할 사업장을 선택합니다.
6. '저장'을 클릭합니다.

# 사업장별 권한 삭제

사업장 권한을 해제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '회계를 클릭합니다.
2. 메뉴에서 '사업장 권한 관리'를 클릭합니다.
3. '사업장별' 탭을 클릭합니다.
4. 사업장 목록에서 권한을 삭제할 사업장을 선택합니다.
5. 오른쪽 영역에서 사업장 권한을 삭제할 사원을 선택합니다.
6. '삭제'를 클릭합니다

# **사원별 권한 부여**

사원별 권한을 부여하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '회계를 클릭합니다.
2. 메뉴에서 '사업장 권한 관리'를 클릭합니다.
3. '사원별' 탭을 클릭합니다.
5. 오른쪽 영역에서 권한을 부여할 사업장을 선택합니다.

# 사원별 권한 삭제

사업장 권한을 해제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '회계를 클릭합니다.
2. 메뉴에서 '사업장 권한 관리'를 클릭합니다.
3. '사원별' 탭을 클릭합니다.
4. 사원 목록에서 권한을 삭제할 사원을 선택합니다. 사업장 권한을 부여 받은 사원만 표시됩니다. 사원 추가는 '사업장별' 탭에서 추가해 주세요.
5. 오른쪽 영역에서 권한을 삭제할 사업장을 선택 해제합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
