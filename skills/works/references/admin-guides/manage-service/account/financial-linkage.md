# 금융계좌 연동

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  

- 은행 및 계좌를 연동하는 방법은 [은행/계좌 연동](/ko/admin-guides/manage-service/account/funds-status/#i-1)을 참고해 주십시오.
- 금융 계좌 연동 메뉴는 스크래핑을 통해 연동된 은행으로부터 가져온 데이터를 확인하기 위한 메뉴이며, 실제 금융사와의 입출금 거래는 지원하지 않습니다.
- 은행/계좌 연동은 법인 계좌만 지원합니다.

# 입금 연동 내역

입금 연동 내역 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 입금 내역을 사업장, 거래일, 전표 번호, 예적금 구분, 은행, 계좌 번호 조건에 따라 검색 |
| ② 전표 생성 | 선택한 입금 내역의 전표 생성([연동 입금 전표 생성](#i-1-2) 참고) |
| ③ 전표 삭제 | 클릭하여 현재 선택한 매입 내역의 입금 전표 삭제 |
| ④ 계정 과목 일괄 지정 | 클릭하여 입금 연동 내역의 계정 과목 일괄 지정([계정 과목 일괄 지정](#i-1-1) 참고) |
| ⑤ 다운로드 | 클릭하여 연동된 입금 내역 목록을 엑셀 파일로 다운로드([입금 연동 내역 다운로드](#i-1-4) 참고) |
| ⑥ 연동된 입금 내역 목록 | 입금 내역을 목록으로 확인  - 거래처:을 클릭하여 거래처 선택 - 계정 과목(대변): 기본으로 가수금 계정 표시, 전표를 생성하지 않은 내역은 을 클릭하여 계정 과목 지정 가능, 전표 생성 후 변경 불가 - 전표 번호: 입금 전표를 생성한 경우 전표 번호를 클릭하여 생성된 전표를 확인 가능,입금 전표를 생성하지 않은 경우 을 클릭하여 기존 입금 전표 매핑 가능([입금 내역에 기존 전표 매핑](#i-1-3) 참고) |

## 계정 과목 일괄 지정

입금 전표가 생성되지 않은 연동된 입금 내역을 선택하여 계정 과목을 일괄 지정할 수 있습니다.

계정 과목 일괄 지정 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 계정 과목을 일괄 지정할 입금 내역을 선택한 후 '계정 과목 일괄 지정'을 클릭합니다.
5. 팝업 창에서 일괄로 지정할 계정 과목을 선택 후 '확인'을 클릭합니다.
6. '저장'을 클릭합니다.
   - 선택된 입금 연동 내역의 계정 과목이 일괄 지정됩니다.

계정 과목 일괄 지정 시, 이미 계정 과목을 지정한 내역도 일괄 변경됩니다.

## 연동 입금 전표 생성

연동된 입금 내역으로 새로운 전표를 생성하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 전표를 생성할 입금 내역을 선택한 후 '전표 생성'을 클릭합니다.
5. 알림 팝업 창에서 '예'를 클릭합니다.
   - 해당 입금 연동 내역의 전표가 생성됩니다.

## 입금 내역에 기존 전표 매핑

이미 존재하는 관련 입금 전표를 입금 내역에 매핑하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 전표를 매핑할 입금 내역을 선택한 후 '전표 번호' 열의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_search.png)을 클릭합니다.
5. 관련 전표 조회 팝업 창에서 전표일 및 기타 조건을 설정하여 관련 입금 전표를 검색합니다.
6. 매핑할 기존 입금 전표를 선택한 후 '저장'을 클릭합니다.
7. 알림 팝업 창에서 '예' 버튼을 클릭합니다.
   - 해당 입금 역동 내역에 기존 전표가 매핑됩니다.

## 입금 연동 내역 다운로드

입금 연동 내역을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 입금 연동 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 입금 연동 내역이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 입금 연동 내역이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 Download 폴더에 입금연동내역\_YYYYMMDD\_XXXXXX.xlsx 파일명으로 저장됩니다.

# 계좌 잔액 비교

연동된 각 은행 계좌의 계좌 잔액과 전표 잔액을 확인하고 잔액 간의 차액을 확인할 수 있습니다.

계좌 잔액 비교 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 계좌 잔액 내역을 사업장, 은행, 계좌 번호 조건에 따라 검색 |
| ② 다운로드 | 클릭하여 연동된 계좌 잔액 내역 목록을 엑셀 파일로 다운로드 |
| ③ 연동된 계좌 잔액 내역 목록 | 계좌 잔액 내역을 목록으로 확인, 통화별 합계 금액 표시 |

계좌 잔액, 전표 잔액의 전표 집계 기준의 차이점:

- 계좌 잔액: 스크래핑 한 계좌의 잔액
- 전표 잔액: 전표 잔액의 집계, 관리 항목의 계좌를 기준으로 집계함
