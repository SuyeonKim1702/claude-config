# 부가세 신고

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
부가세 신고에서는 전표 생성 시 입력한 부가세 대급금, 예수금 정보를 기준으로 부가세 신고서 및 부속서류를 사업장별로 작성하고, 신고용 전산매체를 생성하여 부가세 신고를 진행할 수 있습니다.

부가세 신고의 일반적인 진행 절차는 다음과 같습니다.

이 가이드에서는 PC웹 화면에서 부가세 신고 하위 메뉴가 표시되는 순서에 따라 부가세 신고를 진행하는 방법을 설명합니다.

# 부가세 신고 전 준비 사항

부가세 신고를 진행하기 전 네이버웍스 회계 서비스에 등록된 부가세 원장 자료와 국세청 홈택스에 등록된 부가세 자료(전자 세금 계산서, 현금 영수증, 사업용 신용 카드 내역)가 일치하는지 반드시 확인해야 합니다.

[국세청 홈택스 홈페이지](https://www.hometax.go.kr/)에 접속하여 전자세금계산서, 현금영수증, 사업용 신용카드 내역을 확인한 후 아래와 같이 매입/매출 합계표로 정리하면 네이버웍스 회계 서비스의 원장과 홈택스의 자료가 일치하는지 쉽게 검증할 수 있습니다. 또한, 작성한 매입/매출 합계표를 부가세 신고 시의 검증 자료로 활용할 수 있습니다.

# 사업장별 신고 서류 등록

사업장별 신고 서류를 등록하고, 사업장별 신고 서류 목록을 엑셀 파일로 다운로드할 수 있습니다.

부가세 신고 대상 사업장별로 기준 정보를 등록하고 신고 대상 서류를 등록합니다.

사업장별 신고 서류를 등록하는 방법은 다음과 같습니다.

1. [사업장 관리](/ko/admin-guides/manage-service/support-management/company-info/#i-2)를 참고하여 사업장별 부가세 신고 과세자 구분(사업자 단위 과세 적용 사업자, 총괄 납부 사업자, 일반 사업자) 및 부가세 신고서 제출 대상 세무서를 등록합니다.
2. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
3. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
5. 신고 서류를 등록할 사업장을 선택한 후 '검색'을 클릭합니다.
6. 표시되는 부가세 신고 서류 목록에서 등록할 서류의 상태를 '사용 함'으로 설정합니다.
   - 목록 두 번째 행의 입력란에 키워드를 입력하여 표시되는 서류를 필터링할 수 있습니다.
7. '저장'을 클릭한 후 알림 팝업 창에서 '확인'을 클릭합니다.
   - 신고 서류가 등록됩니다.

## 사업장별 신고 서류 목록 다운로드

사업장별 신고 서류 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 신고 서류 목록을 다운로드할 사업장을 선택한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 신고 서류 목록이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 사업장별 신고 서류 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Download` 폴더에 `사업장별신고서류등록_YYYYMMDD_XXXXXX.xlsx` 파일명으로 저장됩니다.

# 부가세 원장 조회/수정

승인된 전표의 부가세 데이터를 확인하고, 부가세 신고를 진행하기 위해 수정이 필요한 항목을 수정합니다.

부가세 원장 조회/수정 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 부가세 데이터를 사업장, 귀속 연도, 기수, 작성일, 거래 구분, 세금 코드, 전자/수기 여부, 신고 여부, 자산 여부, 전표 번호 조건에 따라 검색 |
| ② 다운로드 | 부가세 데이터 목록을 엑셀 파일로 다운로드([부가세 데이터 목록 다운로드](#i-3-2) 참고) |
| ③ 부가세 데이터 목록 | - 부가세 데이터를 목록으로 확인 - 노란색으로 표시된 항목은 클릭하여 수정 가능([부가세 데이터 수정](#i-3-1) 참고) - 거래처를 클릭하여 거래처 정보 확인 가능 |
| ④ 저장 | 클릭하여 수정한 내용 저장 |

## 부가세 데이터 수정

부가세 데이터를 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 표시되는 목록에서 노란색으로 표시된 항목을 수정합니다.
   - 자산 구분 항목은 자산 여부가 'Y'인 경우 수정할 수 있습니다.
   - 불공제 사유 항목은 세금 코드가 '매입-불공'인 경우 수정할 수 있습니다.
   - 영세 매출 관련 세금 코드(매출-영세, 매출-수출, 매출-카드 영세, 매출-현금 영세)인 경우, 영세율 유형, 수출 신고 번호, 선적일, 외화 통화 코드, 환율, 외화 공급 금액을 수정할 수 있습니다.
5. '저장'을 클릭합니다.
   - 수정 내용이 반영됩니다.

# 신고 부속서류 집계

부가세 원장의 자료를 기준으로 신고 부속서류의 데이터를 집계합니다. 필요한 경우 현재 집계를 삭제한 후 데이터를 재집계할 수 있습니다.

- 현금 매출 명세서가 필수인 사업장인 경우 해당 명세서가 항상 자동으로 생성됩니다. 업종 코드 또는 부업종 코드를 기준으로 현금 매출 명세서 필수 여부를 자동으로 파악합니다.
- 수정 신고 시 과세 표준 및 세액의 결정(경정) 청구서, 과세 표준 수정 신고서 및 추가 자진 납부 계산서가 자동으로 생성됩니다.

신고 부속서류 집계 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 집계 대상 서류를 사업장, 귀속 연도, 기수, 신고 구분 조건에 따라 검색 |
| ② 집계 | - 클릭하여 집계 대상 서류의 데이터 집계([데이터 집계](#i-5-1) 참고) - 집계가 완료되면 각 신고 부속서류를 클릭하여 서류 화면으로 이동 가능 |
| ③ 집계 삭제 | 재집계가 필요한 경우 클릭하여 현재 집계 내역 삭제([집계 삭제](#i-5-2) 참고) |
| ④ 집계 서류 목록 | 집계 대상 서류를 목록으로 확인  - 집계 상태: 신고 부속서류 집계 상태에 따라 미집계, 집계 완료로 표시 - 집계 구분: 자동 생성(데이터 수정 불가), 자동/수기(집계된 데이터 수정 가능), 수기 입력(데이터를 자동 생성하지 않고 수기 입력)으로 표시 - 신고 부속 서류 마감: 마감 시 Y, 미마감 시 N으로 표시 - 마감 일시: 마감을 진행한 년, 월, 일, 시, 분 표시 - 마감자: 신고 부속서류 마감자명 표시 - 집계일시: 집계를 진행한 년, 월, 일, 시, 분 표시 - 작업자: 신고 부속서류 집계 작업자명 표시 |

## 데이터 집계

신고 부속서류의 데이터를 집계하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '집계'를 클릭합니다.
5. 집계할 서류 정보를 입력합니다.
6. 신고 부속서류 집계 화면에서 '집계'를 클릭합니다.
   - 신고 부속서류 데이터가 집계되며, 집계 상태가 '집계 완료'로 변경됩니다.

## 집계 삭제

집계한 데이터를 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '집계 삭제'를 클릭합니다.
5. 삭제할 서류의 정보를 입력합니다.
6. '집계 삭제'를 클릭합니다.
   - 집계된 데이터가 삭제되며, 집계 상태가 '미집계'로 변경됩니다.

# 신고 부속서류 집계 상세 조회

신고 부속서류 집계 데이터를 사업장, 거래 구분, 세금 코드, 전자/수기별로 묶어서 집계하고 각 집계 묶음의 상세 내역을 확인할 수 있습니다.

신고 부속서류 집계 상세 조회 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 집계 정보를 사업장, 귀속 연도, 기수, 신고 구분에 따라 검색 |
| ② 다운로드 | 클릭하여 부가세 구분별 집계 정보를 엑셀 파일로 다운로드 |
| ③ 부가세 구분별 집계 | 집계 정보를 부가세별로 구분하여 확인  - 각 집계 정보를 클릭하여 세부 현황에서 선택된 행의 통합된 전표 및 부가세 정보 확인 가능 |
| ④ 세부 현황 | 사업장, 거래 구분, 세금 코드, 전자/수기의 구분이 동일한 데이터를 통합한 상세 내역 확인  각 상세 정보를 클릭하여 정보 조회 가능  - 전표 번호: 전표 상세 내역 확인 - 거래처: 거래처 정보 확인 |

# 부가세 수정 신고

부가세 신고가 마감된 이후에 수정이 필요한 경우, 수정 신고를 위한 신고 부속서류를 준비한 후 해당 서류를 사용해 부가세 신고서 마감 작업을 다시 진행할 수 있습니다. 부가세 신고서 마감은 [부가세 신고서 작성/마감](#i-8)을 참고해 주십시오.

수정 신고용 부속서류는 부가세 신고서의 마감이 완료된 이후에 작성하고 마감할 수 있습니다. 과세 표준 및 세액의 결정(경정) 청구서, 과세 표준 수정 신고서, 추가 자진 납부 계산서가 자동으로 생성됩니다.

부가세 수정 신고를 하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 신고 구분을 '수정 신고'로 선택한 후 '검색'을 클릭합니다.
5. '집계'를 클릭하여 서류를 집계합니다.  

   마감된 서류가 있는 경우 집계되지 않습니다. 집계하기 전 신고 부속서류의 마감 상태를 확인한 후 마감을 취소해 주십시오.

   - 과세 표준 및 세액의 결정(경정) 청구서, 과세 표준 수정 신고서 및 추가 자진 납부 계산서가 목록에 추가됩니다.
6. 집계된 신고 부속서류 내용을 모두 확인 및 수정한 후 '마감 처리'를 클릭합니다.
   - 과세 표준 및 세액의 결정(경정) 청구서, 과세 표준 수정 신고서 및 추가 자진 납부 계산서는 경우에 따라 1개만 작성할 수 있습니다.
     - 납부할 세액이 더 감소한 경우: 과세 표준 및 세액의 결정(경정) 청구서 작성 가능(세금 환급 목적)
     - 납부할 세액이 더 증가한 경우: 과세 표준 수정 신고서 및 추가 자진 납부 계산서 작성 가능(추가 납부 목적)

     과세 표준 및 세액의 결정(경정) 청구서와 과세 표준 수정 신고서 및 추가 자진 납부 계산서는 둘 중 하나만 마감 처리되어 있더라도 부가세 신고서를 생성할 수 있습니다.
8. 신고 구분을 '수정 신고'로 선택한 후 '검색'을 클릭합니다.
9. '부가세 신고서 생성'을 클릭합니다.
10. '마감 처리'를 클릭합니다.
12. 신고 구분을 '수정 신고'로 선택한 후 '검색'을 클릭합니다.
13. '전산 매체 생성'을 클릭합니다.
    - 부가세 수정 신고가 완료됩니다.
