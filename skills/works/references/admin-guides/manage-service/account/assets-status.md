# 고정자산 현황

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
고정 자산 대장, 고정 자산 명세서, 고정 자산 증감 대장을 통해 회사가 보유하고 있는 고정 자산의 전체 현황을 확인할 수 있습니다.

이 가이드에서는 고정 자산의 현황을 확인하는 방법에 대해 설명합니다. 고정 자산의 등록을 비롯한 각종 관리 기능을 사용하는 방법은 [고정 자산 관리](/ko/admin-guides/manage-service/account/fixed-assets/)를 참고해 주십시오.

# 고정 자산 대장

회사가 보유한 고정 자산을 확인할 수 있습니다. 수량, 취득금액, 감가상각 누계액, 자산 상태, 감가상각 계산 여부 등 고정 자산의 상세 현황을 유형/자산별로 파악할 수 있습니다.

고정 자산 대장 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 고정 자산을 사업장, 부서, 감가상각 계산 여부, 자산 상태 및 유형, 고정 자산 번호 및 이름 조건에 따라 검색 |
| ② 다운로드 | 클릭하여 고정 자산 목록을 엑셀 파일로 다운로드 |
| ③ 고정 자산 목록 | 고정 자산을 목록으로 확인 |
| ④ 조회 | 클릭하여 해당 고장자산의 상세 내역 확인 |
| ⑤ 표시 항목 수 | 목록의 한 페이지에 표시할 고정 자산 항목 수 설정 |

## 고정 자산 증가 자료 목록 다운로드

고정 자산 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 고정 자산 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 고정 자산이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 고정 자산 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Download` 폴더에 `고정자산대장_YYYYMMDD_XXXXXX.xlsx` 파일명으로 저장됩니다.

# 고정 자산 명세서

고정 자산 대장의 금액 변동을 확인할 수 있습니다. 고정 자산 증가 및 감소로 인한 장부 금액의 변동 상세 내역을 확인하고, 감가상각 내역을 고정 자산별로 확인할 수 있습니다.

고정 자산 명세서 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 고정 자산 변동 내역을 사업장, 취득 기간, 부서, 자산 유형, 고정 자산 번호 및 이름 조건에 따라 검색 |
| ② 다운로드 | 클릭하여 고정 자산 변동 내역을 엑셀 파일로 다운로드 |
| ③ 고정 자산 변동 내역 | - 고정 자산 변동 내역을 목록으로 확인 - 각 열의 헤더를 클릭하면 화살표가 표시되며, 화살표를 클릭하여 해당 헤더를 기준으로 정렬 가능 |
| ④ 표시 항목 수 | 목록의 한 페이지에 표시할 고정 자산 변동 내역 항목 수 설정 |

4. 필요한 경우, 검색창에서 조회할 고정 자산 조건을 입력한 후 '검색'을 클릭합니다.

다운로드된 파일은 `Download` 폴더에 `고정자산명세서_YYYYMMDD_XXXXXX.xlsx` 파일명으로 저장됩니다.

# 고정 자산 증감 대장

회사가 보유한 고정 자산의 증감 내역을 상세하게 확인할 수 있습니다. 고정 자산의 기초/기말 잔액과 증가/감소 금액을 기간별로 확인할 수 있으며, 검색 설정에 따라 취득원가와 감가상각 누계액을 별도로 표시하여 확인하거나 동시에 확인할 수 있습니다.

고정 자산 증감 대장 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |

| ② 다운로드 | 클릭하여 고정 자산 증감 내역을 엑셀 파일로 다운로드 |
| ③ 고정 자산 증감 내역 | 고정 자산 증감 내역을 목록으로 확인  각 열의 헤더를 클릭하면 화살표가 표시되며, 화살표를 클릭하여 해당 헤더를 기준으로 정렬 가능 |
| ④ 표시 항목 수 | 목록의 한 페이지에 표시할 고정 자산 증감 내역 항목 수 설정 |
