# 고정자산 관리

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
고정 자산 유형 및 자산 등록, 증가 및 감소, 마감, 감가상각 계산 및 전표 조회 등 고정 자산 회계 업무를 수행하고 자산 정보를 관리할 수 있습니다.

이 가이드에서는 고정 자산을 등록하고 관리하는 방법에 대해 설명합니다. 고정 자산의 대장을 비롯한 각종 현황을 확인하는 방법은 [고정 자산 현황](/ko/admin-guides/manage-service/account/assets-status/)을 참고해 주십시오.

고정 자산을 관리하기 위해서는 먼저 고정 자산의 유형을 등록하여 유형별 감가상각 방법, 내용 연수, 잔존 가치, 계정 과목 등을 설정해야 합니다. 이후 설정한 유형에 따라 고정 자산을 등록한 후 관리할 수 있습니다. 고정 자산 관리 메뉴를 사용하기 전부터 보유하고 있던 자산을 '자산 내역 등록' 메뉴를 통해 등록하고, 누락된 내역이 없는지 재무제표의 장부가액과 비교하여 확인한 후 관리할 수 있습니다.

고정 자산의 관리 흐름은 다음과 같습니다.

고정 자산 관리 메뉴를 사용하기 전부터 보유했던 자산 등록

# 고정 자산 유형 등록

회사가 보유한 고정 자산의 유형을 정의하고, 각 유형별로 감가상각 방법, 내용 연수, 잔존 가치, 계정 과목 등을 설정할 수 있습니다. 기본적으로 제공하는 2개 분류에 더하여 최대 5개 분류까지 구분하여 관리할 수 있습니다.

고정 자산 유형 등록 화면은 다음과 같이 구성되어 있습니다.

| **영역** | **설명** |
| --- | --- |
| ① 검색창 | 고정 자산 유형을 자산 유형과 상각 방법 조건에 따라 검색 |
| ② 고정 자산 유형 분류 | 등록된 고정 자산 유형을 가장 상위의 유형부터 단계별로 표시  각 고정 자산 유형 분류를 클릭하여 하위분류 확인 가능  3분류 이하의 유형은 동일한 수준 내에서 표시 순서 변경 가능 |
| ③ 추가 | 새로운 고정 자산 유형 등록([고정 자산 유형 등록](#i-1) 참조) |
| ④ 삭제 | 클릭하여 선택한 고장자산 유형 삭제  고정 자산이 이미 등록되어 있는 유형, 하위 유형이 정의되어 있는 유형, 기본 제공되는 1, 2 분류는 삭제 불가 |
| ⑤ 고정 자산 유형 상세 | 선택한 고정 자산 유형의 상세 정보 표시  각 계정 과목의 내용을 변경한 후 저장하여 내용 수정 가능 |
| ⑥ 저장 | 클릭하여 수정한 내용 저장 |

## 고정 자산 유형 등록

새로운 고정 자산 유형을 등록하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 고정 자산 유형 분류 목록에서 새로운 유형을 등록할 분류를 찾아 선택합니다.
5. '추가'를 클릭합니다.
6. 아래와 같이 등록할 유형의 설정을 입력합니다.
   - 자산 유형: 고정 자산 유형의 이름 입력  
     다국어를 사용하도록 설정한 경우, 'A' 버튼을 다국어 명칭 입력 가능
   - 상각 방법: 자산 감가상각 방법을 정액법, 정률법 중에서 선택
     - 정액법: 고정 자산의 내용 연수 동안 동일한 금액을 감가상각비로 계산
     - 정률법: 장부가액에 일정한 상각률을 곱하여 감가상각비를 계산
     - 토지 자산의 경우 감가상각을 하지 않으므로 상각 방법을 선택 불가
     - 세법상 정액법으로 감가상각하도록 정해진 건물 및 구축물, 일반적으로 정액법으로 상각하는 무형 자산은 정액법으로 고정
   - 내용 연수: 해당 자산의 효용이 지속되는 내용 연수 기간을 최대 99년까지 지정
   - 잔존 가치: 해당 자산의 감가상각이 완료된 후에 남는 자산 가치 입력
   - 감가상각비 계정: 감가상각비 계정 선택
   - 고정 자산 계정 코드와 감가상각 누계액 계정은 1, 2 분류의 경우 자산 계정 과목으로 자동 설정되며, 2 분류의 하위로 생성하는 고정 자산 유형은 2 분류가 사용하는 자산 계정과 감가상각 누계액을 사용
7. '저장'을 클릭한 후 알림 팝업 창에서 '예'를 클릭합니다.
   - 새로운 고정 자산 유형이 등록되어 유형 분류 목록에 표시됩니다. 목록에서 해당 유형을 클릭하여 하위 유형을 생성할 수 있습니다.

# 고정 자산 등록

고정 자산 등록 메뉴에서는 회사가 보유한 고정 자산을 등록할 수 있습니다.

고정 자산 등록 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 고정 자산을 사업장, 부서, 조회 조건, 자산 유형, 고정 자산, 원자산 번호 조건에 따라 검색 |
| ② 추가 | 새로운 고정 자산 등록([고정 자산 추가](#i-2-1) 참조) |
| ③ 삭제 | 클릭하여 선택한 고장자산 삭제 |
| ④ 일괄 추가 | 엑셀 템플릿을 사용하여 고정 자산 일괄 등록([고정 자산 일괄 등록](#i-2-2) 참조) |
| ⑤ 다운로드 | 클릭하여 고정 자산 목록을 엑셀 파일로 다운로드([고정 자산 목록 다운로드](#i-2-3) 참조) |
| ⑥ 자산 분할 | 선택한 고정 자산을 분할하여 새로운 자산을 추가 생성([고정 자산 분할](#i-2-4) 참조) |
| ⑦ 고정 자산 목록 | 등록된 고정 자산을 목록으로 확인  - 자산 번호: 클릭하여 고정 자산 상세 정보 확인 - 등록자: 고정 자산을 등록한 사용자 정보 확인 - 지출 결의서 검색: 지출 서비스 사용 시 항목 표시, 허용/허용 안 함 변경 가능, '허용'의 경우, '자산 취득'으로 지출 결의서 작성 시 해당 고정 자산이 검색 가능함([지출결의서 작성](/ko/use-guides/approval/pc-web/write-approval-document/#i-11) 참조) |

## 고정 자산 추가

새로운 고정 자산을 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '추가'를 클릭합니다.
5. 아래와 같이 추가할 고정 자산의 내용을 입력합니다.
   - 자산 번호: 고정 자산 등록 시 시스템에서 자동으로 채번되며, 수정할 수 없습니다.(연월+일련번호 5자리)
   - 자산명, 사업장, 부서: 고유자산의 이름, 소속 사업장, 관리 부서 입력
   - 자산 유형: 해당 고정 자산이 분류될 유형 선택  
     유형을 선택하면 해당 유형에 설정된 고정 자산 계정, 상각 방법, 내용 연수, 잔존 가치, 감가상각비 계정, 감가상각 누계액 계정 정보가 자동으로 반영되며, 상각률은 내용연수와 상각 방법에 따라 자동으로 계산됨
   - 감가상각 계산 여부: 감가상각 계산 여부 선택  
     토지 등 감가상각비 불가능한 고정 자산 유형은 선택 불가
   - 감가상각비 계정, 감가상각 누계액 계정, 잔존 가치: 고정 자산 유형에 설정된 정보가 연동되어 표시
   - 특별상각 대상: 해당 고정 자산이 특별 상각 대상인 경우 대상을 선택한 후 특별 상각률 입력
   - 지출 결의서 검색: 지출 서비스 사용 시에 표시, '허용'으로 설정의 경우 '자산 취득' 구분으로 지출 결의 신청 시에 해당 고정 자산 검색 가능
   - 적요: 고정 자산의 적요 입력
6. '저장'을 클릭한 후 알림 팝업 창에서 '예' 버튼을 클릭합니다.
   - 고정 자산이 추가됩니다.

## **고정 자산 일괄 등록**

엑셀 템플릿을 사용하여 여러 고정 자산을 일괄 등록하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '일괄 추가'를 클릭합니다.
5. 자산 일괄 추가 팝업 창에서 '템플릿 파일 다운로드'를 클릭하여 템플릿을 다운로드합니다.
6. 템플릿에 맞추어 고정 자산 내용을 입력한 후 파일을 저장합니다.
   - 표시된 항목은 필수 입력 항목입니다.
7. 업로드 팝업 창에서 사업장을 선택한 후 '내 PC에서 첨부하거나, 파일을 끌어오세요. ' 영역을 클릭하여 첨부할 파일을 선택하거나 해당 영역에 파일을 끌어옵니다.
8. '확인'을 클릭합니다.
   - 고정 자산이 일괄 등록됩니다.

## **고정 자산 목록 다운로드**

고정 자산 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 고정 자산 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 고정 자산이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 고정 자산 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 Download 폴더에 '고정자산등록\_YYYYMMDD\_XXXXXX.xlsx' 파일명으로 저장됩니다.

## **고정 자산 분할**

등록된 고정 자산을 분할하여 새로운 자산을 추가로 생성할 수 있습니다. 직전월 고정 자산이 마감되었고 해당 고정 자산의 상태가 '보유'이며, 이전에 분할된 내역이 없는 경우에만 분할할 수 있습니다.

고정 자산을 분할하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 분할할 고정 자산을 선택한 후 '자산 분할'을 클릭합니다.
   - 고정자산 분할 화면으로 이동합니다.
     - 기존 자산: 기존 자산의 증가/감소/감가상각 관련 자료가 있을 경우, 자동으로 연동됩니다.
     - 신규 자산: 신규 자산의 자산 번호는 시스템 채번되며, 수정할 수 없습니다.(연월+일련번호 5자리)
5. 분할하여 생성할 신규 자산의 자산명, 수량, 취득 금액을 입력합니다.
   - 분할할 수량은 기존 자산의 전체 수량보다 작아야 합니다.
   - 신규 자산의 취득 금액을 입력하면 해당 수치만큼 기존 자산의 취득 금액에서 자동으로 차감됩니다.
   - 분할하고자 하는 수량을 입력하면, 기존 자산의 수량/취득 금액/감가상각 누계액 정보가 분할 됩니다.
6. '저장'을 클릭합니다.
   - 해당 고정 자산이 분할됩니다.

# **고정 자산 증가**

새로운 고정 자산을 취득하였거나 지출이 발생한 경우 해당 자료를 등록할 수 있습니다. 또한, 등록된 증가 자료를 통해 고정 자산 전표를 생성하거나 생성된 고정 자산 전표를 삭제할 수 있습니다.

고정 자산 증가 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 고정 자산 자료를 사업장, 취득일, 전표 번호, 부서, 거래처, 구분, 세금 코드, 고정 자산 번호 및 이름 조건에 따라 검색 |
| ② 자료 생성 | 새로운 고정 자산 증가 자료 생성([고정 자산 증가 자료 생성](#i-3-1) 참고) |
| ③ 자료 삭제 | 클릭하여 현재 선택한 고정 자산 증가 자료 삭제([고정 자산 증가 자료 삭제](#i-3-2) 참고) |
| ④ 전표 생성 | 클릭하여 현재 선택한 고정 자산 증가 자료의 전표 생성([고정 자산 증가 전표 생성](#i-3-3) 참고) |
| ⑤ 전표 삭제 | 클릭하여 현재 선택한 고정 자산 증가 자료의 전표 삭제  - 지출 결의서 작성으로 등록된 고정 자산은 전표 삭제 불가([고정 자산 증가 전표 삭제](#i-3-4) 참조), 해당 결재 문서를 삭제해 함 |
| ⑥ 다운로드 | 클릭하여 고정 자산 증가 자료 목록을 엑셀 파일로 다운로드([고정 자산 증가 자료 목록 다운로드](#i-3-5) 참고) |
| ⑦ 고정 자산 자료 목록 | 고정 자산 증가 자료를 목록으로 확인  각 상세 정보를 클릭하여 정보 조회 가능  - 관리 번호: 고정 자산 자료 상세 정보 확인 - 전표 번호: 생성된 고정 자산 전표 확인 - 결재 문서: (지출 서비스 사용 시 항목 표시)지출 결의서 작성으로 등록된 고정 자산의 경우 문서 번호 표시, 클릭하여 문서 내용 확인 가능 - 거래처: 거래처 정보 확인 - 등록자: 자료를 등록한 사용자 정보 확인 |

## **고정 자산 증가 자료 생성**

고정 자산 증가 자료를 생성하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '자료 생성'을 클릭합니다.
5. 아래와 같이 고정 자산 증가 자료 내역을 입력합니다.
   - 취득일: 적격 증빙의 거래일 입력  
     고정 자산 증가 전표 생성 일자에 반영됨
   - 세금 코드: 해당 증가 내역에 적용될 세금 코드 선택
   - 거래처: 증가 내역의 거래처 입력
   - 고정자산: 증가하는 자산 선택
   - 구분: 신규 취득 또는 자본적 지출 선택
   - 수량, 공급가액, 부가세, 수수료: 증가하는 고정 자산 수량 및 금액 정보 입력
   - 프로젝트: 프로젝트 코드 입력
     - 프로젝트 코드는 [프로젝트 코드 관리](/ko/admin-guides/manage-service/support-management/common-code/#i) 메뉴에서 추가 가능
     - 프로젝트 코드는 회사가 프로젝트 코드를 사용하도록 설정한 경우에만 표시
   - 예산: '회계 > 설정' 메뉴에서 예산 사용 여부를 사용으로 설정 시 표시, 예산을 사용하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_search.png)을 클릭하여 사용 가능한 예산 조회
     - 자산의 사업장, 부서(귀속 부서) 및 프로젝트(선택 입력)가 매칭되는 예산만 조회됨
     - 가용 금액을 확인 후 예산 선택
     - 회사가 예산 통제 여부를 '예'로 설정한 경우, 입력한 공급가액+부가세+수수료의 합계 금액이 가용 예산 초과 시 해당 예산을 사용할 수 없음
   - 적요: 증가 내역의 적요 입력
6. '저장'을 클릭한 후 알림 팝업 창에서 '예'를 클릭합니다.
   - 해당 자료로 전표를 생성하려면 '전표 생성'을 클릭합니다.

각 세금 코드에 대한 부가 설명은 다음과 같습니다.

- 매입-과세: 매입세액이 공제되는 세금계산서 수취분 매입(부가세율 10%)
- 매입-면세: 면세와 관련된 계산서 수취분 매입(부가세율 0%)
- 매입-카드 과세: 매입 세액 공제 가능한 카드전표 수취분 매입(부가세율 10%)
- 매입-카드 면세: 신용카드에 의한 면세 매입(부가세율 0%)
- 매입-현금 과세: 매입 세액 공제 가능한 현금 영수증 수취분 매입(부가세율 10%)
- 매입-현금 면세: 현금 영수증에 의한 면세 매입(부가세율 0%)
