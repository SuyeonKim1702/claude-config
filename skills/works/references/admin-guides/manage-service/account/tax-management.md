# 세무 관리

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
세무 관리에서는 직접 등록하거나 스크래핑을 통해 가져온 국세청 홈택스 자료를 확인할 수 있습니다. 또한, 전표 등록 시 입력한 원천세 정보를 조회할 수 있습니다.

국세청 홈택스 자료 스크래핑은 '경영지원 설정 > 회사 정보 > 사업장 관리 > 인증서 관리'에서 회사 인증서를 등록한 후 '홈택스' 항목을 선택하면 자동으로 수행됩니다. 자세한 설명은 [인증서 관리](/ko/admin-guides/manage-service/support-management/company-info/#i-2-5)를 참고해 주십시오.

# 홈택스 자료 등록

보유한 홈택스 전자세금계산서, 계산서, 현금 영수증 데이터가 있는 경우 직접 업로드하여 확인할 수 있습니다.

홈택스 자료 등록 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 등록된 홈택스 자료를 사업장, 기간, 거래처, 거래 구분, 증빙 구분 조건에 따라 검색  - 증빙 구분에 따라 조회되는 데이터 필드 상이(세금계산서/계산서와 현금 영수증의 경우, 노출 필드 상이) |
| ② 업로드 | 증빙 구분이 세금계산서/계산서인 경우 클릭하여 홈택스 자료 등록 |
| ③ 다운로드 | 클릭하여 등록된 홈택스 자료 목록을 엑셀 파일로 다운로드 |
| ④ 홈택스 자료 목록 | 등록된 홈택스 자료를 목록으로 확인  - 작성일(거래일): 부가가치세법상 공급 연월일(부가세 신고 기준 일자) - 발급일: 실제로 세금 계산서 발급 행위가 일어난 날짜(공인 인증서로 전자 서명한 시점) - 전송일: 발행한 전자 세금 계산서를 국세청으로 전송한 날짜(발행일 당일 또는 익일) |

## 홈택스 자료 업로드

홈택스 자료를 업로드하는 방법은 다음과 같습니다.

홈택스 자료는 증빙구분이 세금계산서/계산서인 경우에 한해 업로드할 수 있습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '업로드'를 클릭합니다.
5. 홈택스 자료 업로드 팝업 창에서 홈택스에서 다운로드한 파일을 원본 그대로 업로드하거나, '템플릿 파일 다운로드'를 클릭하여 템플릿 파일을 다운로드한 후 작성합니다.
   - 홈택스에서 다운로드한 파일의 내용은 수정되지 않아야 합니다.
   - 매입/매출 자료는 홈택스에서 제공하는 양식 또는 샘플 파일 다운로드를 통해 받은 엑셀 파일로만 등록할 수 있습니다.
   - 승인 번호는 중복될 수 없습니다. 중복되는 경우, 중복된 자료는 등록되지 않습니다.
6. 홈택스 자료 업로드 팝업 창에서 세무 사업장, 증빙 양식 구분, 업로드 자료 구분을 선택한 후 '내 PC에서 첨부하거나, 파일을 끌어오세요.' 영역을 클릭하여 첨부할 파일을 선택하거나 해당 영역에 파일을 끌어옵니다.
   - 홈택스 자료를 등록할 세무 사업장을 선택합니다.
   - 증빙 양식 구분'을 선택합니다.
   - 업로드 자료 구분을 선택합니다.
7. '확인'을 클릭한 후 알림 팝업 창에서 '예'를 클릭합니다.
   - 홈택스 자료가 등록됩니다.

## 홈택스 자료 목록 다운로드

등록한 홈택스 자료 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '회계'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 홈택스 자료 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 홈택스 자료가 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 홈택스 자료 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 Download 폴더에 '홈택스자료등록\_YYYYMMDD\_XXXXXX.xlsx' 파일명으로 저장됩니다.

# 홈택스 자료 조회

'홈택스 자료 등록' 메뉴에서 등록하거나 매입/매출 등록 시 XML을 통해 가져오거나, 스크래핑을 통해 자동으로 가져온 모든 자료를 조회할 수 있습니다.
