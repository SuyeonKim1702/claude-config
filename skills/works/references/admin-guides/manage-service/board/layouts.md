# 게시판 메인 설정

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

  
PC와 모바일 게시판 메인 화면에 노출될 게시판의 개수, 게시판의 노출 순서 및 각 게시판의 타입을 설정합니다.

# 최신글 설정

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '메인'을 선택합니다.
3. '최신글'을 켭니다.
4. 상단의 '저장'을 누릅니다.

# 게시판 메인 노출 게시판 수

왼쪽을 미리보기를 통해 설정한 게시판 메인 화면의 레이아웃을 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '메인'을 선택합니다.
3. '메인 화면 노출 게시판 수'에서 게시판 메인 화면에 노출시킬 게시판 수를 선택합니다.
   - 최대 4개의 게시판을 노출할 수 있습니다.
4. 노출할 게시판을 선택합니다.
5. 노출할 게시판의 타입을 선택합니다. 게시판형, 미리보기형, 앨범형 가운데 선택할 수 있습니다.
6. 상단의 '저장'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
