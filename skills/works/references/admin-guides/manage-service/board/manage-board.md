# 게시판 관리

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

  
게시판을 추가하고 각 게시판으로 들어가면 보이는 화면 타입을 설정합니다.

게시판의 멤버 권한과 등록된 게시글 및 댓글을 관리할 수 있는 [게시판 마스터](/ko/use-guides/board/manage-board/)를 선정하고, 공개 게시판의 경우에 게시판을 열람할 수 없는 예외를 만들 수 있습니다.

여러 개의 게시판을 하나의 카테고리로 묶을 수 있습니다.

# 게시판 추가, 수정, 삭제하기

## 게시판 추가

게시판명, 게시판 설명, 게시판 공유 범위, 게시판 타입을 설정할 수 있고, 게시판 마스터를 지정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 오른쪽 상단의 '추가'를 클릭한 뒤 '게시판'을 클릭합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/list_16_circleadd_line.png) 버튼을 탭합니다.
4. 게시판 정보를 입력합니다. '추가 설정'을 눌러 추가적인 정보도 입력할 수 있습니다.
   - 게시판명: 게시판의 이름입니다.
   - 게시판 설명: 게시판의 설명입니다.
   - 공유 범위: 추가하는 게시판에 대한 구성원의 권한을 설정합니다. 게시판에 접근할 수 있는 구성원의 권한 형태에 따라 '전체 공개' 게시판과 '멤버 공개' 게시판으로 구분됩니다.
     - 전체 공개: 모든 구성원이 글쓰기와 읽기 권한을 가집니다.
       - 접근 제한 설정: 예외적으로 전체 공개 게시판을 사용할 수 없는 구성원,  사용자 유형, 조직을 지정하거나 해제할 수 있습니다. 단, [게시판 마스터로 지정](/ko/admin-guides/manage-service/board/manage-board/#i-4)된 구성원은 예외 처리되지 않습니다.
       - 관리자, 게시판 마스터만 글쓰기: 선택 시 관리자와 게시판 마스터만 게시글을 작성하도록 설정합니다.
     - 멤버 공개: 지정된 구성원만 글쓰기 또는 읽기 권한을 가집니다.
       - 이름, 부서명, 회사명, 메일주소로 추가할 수 있고, 읽기, 글쓰기/읽기 권한 가운데 선택해 부여할 수 있습니다.
   - 새글 알림 기본 설정: 새 게시글이 있을 때 구성원이 알림을 받을지 말지 기본값을 설정합니다. 설정 후에도 구성원이 임의로 게시판의 새글 알림을 켜거나 끌 수는 있습니다. 단, 기본 설정 자체는 추가 후에는 수정할 수 없습니다.
   - 말머리: 게시글의 카테고리를 구분하기 위한 말머리를 추가합니다. 추가된 말머리는 게시글 작성 시 선택할 수 있습니다.
   - 마스터: 게시판을 관리할 마스터를 지정합니다.
     - 게시판 마스터는 게시판 공유 범위를 설정하고 등록된 게시글 및 댓글을 삭제하거나 수정할 수 있습니다.
     - 게시판 마스터는 읽음 권한이 없어도 게시판에 접근이 가능합니다.
     - Admin 관리자 중 '게시판' 서비스에 권한이 있는 관리자들은 모든 게시판에 마스터 권한을 자동으로 갖게 됩니다.
   - 공지 작성 권한: 공지 게시글을 작성할 수 있는 권한을 설정합니다.
     - 전체: 모든 구성원이 공지 게시글을 작성할 수 있습니다.
     - 관리자 및 게시판 마스터만: 관리자, 게시판 마스터만 공지 게시글을 작성할 수 있습니다.
   - 타입: PC 게시판에 노출될 게시판의 형태입니다.
     - 게시판형, 미리보기형, 앨범형이 있습니다.
   - 게시글 리액션 사용: 리액션 기능의 사용 여부를 설정합니다.
5. 하단의 '추가'을 누릅니다.

## 게시판 수정

게시판의 이름, 설명, 타입, 사용자 범위를 수정할 수 있고, 게시판 마스터도 변경할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 수정할 게시판을 눌러, 정보를 수정합니다.
4. 하단의 '저장'을 누릅니다.

## 게시판 삭제

만들어진 게시판을 삭제할 수 있습니다.

삭제된 게시판의 게시글은 게시판의 휴지통으로 이동하고, 삭제한 관리자만 타 게시판으로 이동시킬 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 삭제할 게시판 왼쪽 ![](/wp-content/uploads/2018/05/ic20_check_off.png) 아이콘을 누릅니다. 또는 '편집'을 눌러 삭제할 게시판을 선택합니다.
4. 왼쪽 상단의 '삭제'를 누릅니다.
5. 주의사항을 확인 후 '삭제'를 누릅니다.

# 게시판 타입 변경하기

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 수정할 게시판을 선택합니다.
4. 게시판 설정 레이어에서 게시글 목록이 노출될 타입을 선택합니다. (게시판형, 미리보기형, 앨범형)
5. 하단의 '저장'을 누릅니다.

# 게시판에 템플릿 적용

관리자 및 게시판 마스터는 게시판에 [템플릿](/ko/use-guides/general/template/first-step/)을 적용할 수 있습니다.

템플릿을 적용하면 게시판에 글을 쓸 때 적용한 템플릿이 바로 나타납니다.

단, 게시물을 공유할 때 게시판에 설정된 템플릿은 공유되지 않습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 수정할 게시판을 선택합니다.
4. '템플릿 선택'을 누릅니다.
5. 템플릿 목록에서 템플릿을 선택합니다.
6. '완료'를 누릅니다.
7. '저장'을 누릅니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 수정할 게시판을 누릅니다.
4. 게시판 설정 화면에서 게시판 마스터 항목 오른쪽에 구성원 이름을 입력하여 추가합니다.
5. 하단의 '저장'을 누릅니다.

# 전체 공개 게시판 설정

## 전체 공개 게시판으로 설정하기

전체 구성원이 사용할 수 있도록 '전체 공개' 게시판으로 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 전체 공개 게시판으로 변경할 게시판을 누릅니다.
4. '공유 범위' 항목에서 '전체 공개'를 선택합니다.
5. 하단의 '저장'을 누릅니다.

## 전체 공개 게시판의 접근 제한 설정

'전체 공개' 게시판 사용을 제한할 멤버, 사용자 유형 또는 조직(부서) 또는 그룹을 지정하여 전체 공개 게시판의 접근을 제한할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 전체 공개 게시판 중 접근을 제한할 게시판 이름을 누릅니다.
4. '공유 범위' 항목에서 '접근 제한 설정'을 누릅니다.
5. 해당 게시판의 접근을 조직, 그룹, 사용자 유형, 구성원 형태로 제한합니다.
   - 조직: '멤버 추가'를 눌러 '주소록'을 선택합니다. ![](/wp-content/uploads/2024/04/icon-n-filter-admin.png) 아이콘을 눌러 '조직만'을 누릅니다. 접근을 제한하려는 조직을 선택해 '확인'을 누릅니다.
   - 그룹:  '멤버 추가'를 눌러 '주소록'을 선택합니다. '그룹' 탭을 누른 후  접근을 제한하려는 그룹을  선택해 '확인'을 누릅니다.
   - 구성원: '멤버 추가'를 눌러 '주소록'을 선택합니다. ![](/wp-content/uploads/2024/04/icon-n-filter-admin.png) 아이콘을 눌러 '구성원만'을 누릅니다.  접근을 제한하려는 구성원을 선택해 '확인'을 눌러 게시판 접근을 제한하려는 구성원을 추가합니다.
   - 사용자 유형: '멤버 추가'를 눌러 '사용자 유형'을 선택합니다. 게시판 접근을 제한하려는 사용자 유형을 선택하고 '추가'를 누릅니다.
6. '추가'를 누른 후 '저장'을 누릅니다.

# 멤버 공개 게시판 권한 부여하기

'공유 범위'를 '멤버 공개'로 설정 시 지정한 멤버, 조직(부서), 또는 그룹만 게시글을 쓰고, 볼 수 있게 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 멤버 공개 게시판으로 변경할 게시판을 누릅니다.
4. '공유 범위' 항목에서 '멤버 공개'를 선택합니다.
5. 게시판 열람 권한을 부여할 멤버를 추가합니다. 구성원 이름, 메일 주소, 조직(부서), 그룹, 또는 회사명으로 검색할 수 있습니다.
6. 추가된 대상에게 글쓰기/읽기와 읽기 권한을 각각 부여합니다.
   - 글쓰기/읽기
   - 읽기
7. '저장'을 누릅니다.

# 카테고리 추가, 수정, 삭제하기

## 카테고리 추가하기

카테고리를 만들어 게시판을 묶음으로 관리할 수 있습니다.

카테고리를 접으면 포함된 게시판이 목록에서 모두 접힌 채 노출되어, 게시판의 갯수가 많을때 게시판 관리에 편리합니다.

게시판 목록에서는 카테고리와 소속된 게시판을 가로줄로 구분합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 오른쪽 상단의 '추가'를 누른 뒤 '카테고리'를 누릅니다.
4. 카테고리명을 입력합니다.
5. '추가'를 누릅니다.
6. 게시판 목록 맨 하단에 추가된 카테고리를 확인합니다.

## 카테고리명 수정하기

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 수정할 카테고리를 눌러, 카테고리명을 수정합니다.
4. 하단의 '저장'을 누릅니다.

## 카테고리 삭제하기

생성한 카테고리를 삭제할 수 있습니다.

삭제 시 카테고리에 묶여있던 게시판들은 풀립니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 삭제할 카테고리 왼쪽의 ![](/wp-content/uploads/2018/05/ic20_check_off.png) 아이콘을 누릅니다. 또는 '편집'을 눌러 삭제할 카테고리를 선택합니다.
4. 왼쪽 상단의 '삭제'를 누릅니다.
5. 주의사항을 확인 후 '삭제'를 누릅니다.

# 게시판 정리하기

게시판을 들여쓰거나 내어쓰는 방법으로 게시판 목록을 시각적으로 정리할 수 있고, 카테고리를 만들어 유사한 게시판의 그룹을 만들 수 있습니다.

## 게시판 들여쓰기

게시판의 목록에서 들여쓰기를 이용하면 게시판 목록의 상하위 관계를 시각적으로 정리할 수 있습니다.

들여쓰기나 내어쓰기를 하더라도 실제 게시판의 계층이 만들어지는 것은 아니고 목록에서 시각적으로 계층을 표현할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록 오른쪽 상단의 '편집'을 누릅니다.
4. 내어쓰기할 게시판 왼쪽의 ![](/wp-content/uploads/2024/04/ico-menu-indent@3x.png) 아이콘을 누릅니다.
5. 왼쪽 상단의 '저장'을 누릅니다.

![](/wp-content/uploads/2024/04/ico-menu-outdent@3x.png) 아이콘을 다시 누르면 들여쓰기가 취소됩니다.

## 게시판을 카테고리 하위에 넣기

카테고리를 생성한 후 게시판을 카테고리 하위로 이동시켜 카테고리 단위로 묶을 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록 오른쪽 상단의 '편집'을 누릅니다.
4. 이동을 원하는 게시판 왼쪽의 ![](/wp-content/uploads/2024/04/ico-18-move-c-5-c-5-c-6@3x.png) 아이콘을 끌어 원하는 카테고리의 이름 위에 놓습니다.
5. 왼쪽 상단의 '저장'을 누릅니다.

## 게시판과 카테고리 순서 변경하기

게시판 목록에서 게시판이나 카테고리의 노출 순서를 변경할 수 있습니다.

카테고리의 순서를 변경하면 해당 카테고리에 포함된 게시판도 카테고리에 포함되어 순서가 변경됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록 오른쪽 상단의 '편집'을 누릅니다.
4. 이동을 원하는 카테고리 또는 게시판 왼쪽의 ![](/wp-content/uploads/2024/04/ico-18-move-c-5-c-5-c-6@3x.png) 아이콘을 끌어 원하는 위치에 놓아서 순서를 변경합니다.
5. 왼쪽 상단의 '저장'을 누릅니다.

## 여러 개의 게시판과 카테고리 순서 변경하기

게시판 목록에서 게시판이나 카테고리를 선택 후 끌어옮겨 노출 순서를 변경할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리'를 선택합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록 오른쪽 상단의 '편집'을 누릅니다.
4. 이동을 원하는 카테고리 또는 게시판의 왼쪽 ![](/wp-content/uploads/2018/05/ic20_check_off.png) 아이콘을 누릅니다.  
   여러 개의 카테고리와 게시판 선택이 가능합니다.
5. ![](/wp-content/uploads/2024/04/ic-16-checkbox-checked@3x.png) 표시된 아이콘 옆의 ![](/wp-content/uploads/2024/04/ico-18-move-c-5-c-5-c-6@3x.png) 아이콘을 끌어 원하는 위치에 놓아서 순서를 변경합니다.
6. 왼쪽 상단의 '저장'을 누릅니다.

# 게시판 보관

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 결재 상품 신청 후 사용 가능합니다.

게시판 목록에서 일시적으로 숨기고 싶은 게시판을 보관할 수 있습니다.

보관한 게시판은 모든 구성원의 화면에서 숨겨지나 삭제되진 않습니다.

게시판은 최대 100개까지 보관할 수 있으나, 카테고리는 보관할 수 없습니다.

## 게시판 보관

게시판 목록에서 특정 게시판을 보관해 숨길 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 누릅니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리' 탭을 클릭합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록에서 보관할 게시판 왼쪽의 ![](/wp-content/uploads/2018/05/ic20_check_off.png) 아이콘을 누릅니다. 또는 '편집'을 눌러 보관할 게시판을 선택합니다.
4. 왼쪽 상단의 '보관' 버튼을 누릅니다.
5. '확인'을 누릅니다.

## 게시판 활성화

보관했던 게시판이 다시 노출되도록 활성화할 수 있습니다.

여러 게시판을 한 번에 활성화하려면 선택한 게시판의 소유 도메인이 모두 동일해야 합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 누릅니다. 모바일에서는 ![](/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시판 관리' 탭을 클릭합니다. 모바일에서는 '관리' 탭을 탭합니다.
3. 게시판 목록 상단의 '보관된 게시판 보기' 버튼을 누릅니다.
4. 활성화할 게시판 왼쪽의 ![](/wp-content/uploads/2018/05/ic20_check_off.png) 아이콘을 누릅니다.
5. 왼쪽 상단의 '활성화' 버튼을 누릅니다.
6. '확인'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
