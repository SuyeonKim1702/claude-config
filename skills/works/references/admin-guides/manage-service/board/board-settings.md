# 게시판 일반 설정

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

  
게시판에서는 전사적으로 공유되어야 할 중요한 이슈를 간편하고 빠르게 공유할 수 있습니다.

일반에서는 게시글에 첨부하는 파일의 용량 설정, 삭제된 게시글의 보관 기간을 설정합니다.

# 파일 1개당 첨부 파일 용량 제한

게시글과 댓글에 첨부하는 파일이나 본문 내 이미지와 동영상의 1개당 용량을 제한합니다. 큰 사이즈의 첨부파일이나 이미지와 동영상으로 인한 공용용량의 무분별한 사용을 관리할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '파일 1개 최대 첨부 용량' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. '파일 1개 최대 첨부 용량' 항목에서 파일 1개의 최대 용량을 선택합니다.
5. 상단의 '저장'을 누릅니다.

# 삭제한 게시글의 휴지통 보관 기간 설정

구성원, 게시판 관리자 또는 관리자가 삭제한 게시글이 휴지통에 보관되는 기간을 설정합니다.

휴지통에 보관되는 기간동안은 삭제된 글의 복구가 가능합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '휴지통 보관 기간' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 보관기간을 선택합니다.
   - 자동삭제를 원치 않을 경우, '자동 삭제 안 함'을 선택합니다.
5. 상단의 '저장'을 누릅니다.

# 게시판 생성 권한 설정

게시판을 관리자만 생성 가능하게 할지 모두가 생성 가능하게 할지 선택합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 누릅니다.
3. '게시판 생성' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 권한을 선택합니다.
5. 상단의 '저장'을 누릅니다.

# 읽은 멤버 확인

접근 권한이 부여된 게시판에서 게시글을 [읽은 멤버를 확인](/ko/use-guides/board/check-post/check-read-member/)할 수 있는 대상의 범위를 지정할 수 있습니다.

기본적으로 작성자, 게시판 마스터, 관리자만 읽은 멤버를 확인할 수 있습니다.

단, [조직도 조회가 제한](/ko/admin-guides/security/limit-visibility/)된 구성원은 [읽은 멤버 명단](/ko/use-guides/board/check-post/check-read-member/#i-2)을 확인할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '읽은 멤버 확인' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 범위를 선택합니다.
5. 상단의 '저장' 버튼을 누릅니다.

# 필독 설정

[게시글을 필독으로 설정](/ko/use-guides/board/post/set-post-as-must-read/)할 수 있는 대상의 범위를 지정할 수 있습니다.

기본적으로 작성자, 게시판 관리자, 관리자만 게시글을 필독으로 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '필독 설정' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 권한 범위를 선택합니다.
5. 상단의 '저장' 버튼을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
