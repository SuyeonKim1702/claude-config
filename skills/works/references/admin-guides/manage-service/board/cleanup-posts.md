# 게시글 정리

게시글을 다른 게시판이나 휴지통으로 이동하거나, 영구 삭제할 수 있습니다.

단, 게시글 정리는 PC웹에서만 할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다.
2. '게시글 정리' 탭을 선택합니다.
3. 정리 기준을 선택합니다. 게시글이 있는 게시판과 기간을 선택하고, 필요 시 작성자명, 첨부파일 수, 댓글 수 기준을 입력합니다.  
   '게시글 조회'를 클릭하면 선택한 기준을 충족하는 게시글의 수가 조회됩니다. 선택한 기준을 충족하는 게시글이 없을 경우 다음 단계로 넘어갈 수 없습니다.  
   게시글 수를 클릭해 게시판 서비스로 이동하여 해당 기준을 충족하는 게시글 목록을 확인할 수도 있습니다.
4. '다음'을 클릭합니다.
5. 게시글들을 정리할 방법을 선택합니다.  

   |  |  |
   | --- | --- |
   | **정리 방법** | **설명** |
   | 게시판으로 이동 | 특정 게시판을 선택하여 선택한 게시판으로 게시글들을 이동합니다. '게시판 선택'을 클릭해 원하는 게시판을 선택할 수 있습니다. |
   | 휴지통으로 이동 | 게시글을 휴지통으로 이동합니다.  휴지통으로 이동 시, [휴지통 보관 기간](/ko/admin-guides/manage-service/board/board-settings/#i-2) 동안 보관 후 영구 삭제됩니다. |
   | 영구 삭제 | 게시글을 즉시 영구 삭제합니다.  영구 삭제한 게시글은 복구할 수 없습니다. |
6. '게시글 정리'를 클릭해 게시글들을 정리합니다. 정리해야 할 게시글 수가 많은 경우 일정 시간이 소요될 수 있으며, 정리가 완료되면 최고 관리자와 작업을 요청한 관리자에게 서비스 알림이 발송됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
