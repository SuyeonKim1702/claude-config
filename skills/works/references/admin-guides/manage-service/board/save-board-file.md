# 게시판 게시글 백업

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

  
게시판에 작성된 모든 게시글과 첨부한 파일을 다운로드 받거나 저장 요청한 이력을 확인합니다.

# 게시글 백업 파일 저장하기

게시판에 작성된 모든 게시글과 첨부된 파일을 다운로드 받습니다. '게시판 게시글 파일 다운로드 준비가 완료되었습니다.'는 제목으로 서비스 알림이 오면 클릭하여 파일을 저장합니다.

단, 아래의 경우에는 다운로드 파일에 포함되지 않습니다.

- 임시 저장 게시글
- 휴지통에 있는 게시글
- 관리자가 차단한 확장자를 가진 파일이 첨부된 게시글
- 공용용량을 초과한 경우
- 댓글 및 댓글 내 첨부파일

하루에 5번까지 게시글 백업요청을 할 수 있으며, 게시글 백업 내역에 '다운로드'가 가능한 게시글 백업 파일이 존재할 경우 해당 파일을 '삭제' 후 게시글 백업을 다시 요청할 수 있습니다.

모바일에서는 백업 파일 요청은 가능하나, '다운로드'를 제공하지 않습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시글 백업' 탭을 선택합니다.
3. '게시판 선택'을 눌러 백업을 진행할 게시판을 선택합니다. 전체 게시판을 선택할 수도 있습니다.
4. '게시글 백업'을 누릅니다.

# 게시글 백업 내역 확인하기

모든 게시글 및 첨부한 파일을 저장 요청한 내역을 확인합니다. 저장 요청 시각, 요청한 관리자, 다운로드 만료일, 저장 대상 회사 및 전체 파일 용량 등 게시글 파일 저장 요청 기록을 확인하거나, 다운로드하거나, 기록을 삭제합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '게시판'을 눌러 '게시판' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '게시글 백업'을 선택합니다.
3. 화면 하단의 게시글 파일 저장 기록 항목을 확인하고 다운로드하거나 기록을 삭제합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
