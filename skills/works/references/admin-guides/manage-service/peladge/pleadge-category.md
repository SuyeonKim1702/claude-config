# 카테고리 관리

이 기능은 서약서 옵션 상품 신청 후 사용 가능합니다.

  
서약서 양식 카테고리를 추가하거나 삭제할 수 있습니다. 또한 카테고리의 표시 순서를 변경하거나 카테고리 내의 양식을 관리할 수도 있습니다.

제공되는 '기본' 카테고리는 삭제하거나 카테고리 이름을 변경할 수 없습니다.

# 카테고리 추가

카테고리를 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업'에서 '서약서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '카테고리 관리'를 클릭합니다.
4. '추가'를 클릭합니다.
5. 추가된 행에 카테고리명을 입력한 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_2.png) 또는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_1.png)을 클릭하여 카테고리 순서를 지정합니다.
6. '저장'을 클릭합니다.

# 카테고리 수정

카테고리를 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업'에서 '서약서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '카테고리 관리'를 클릭합니다.
4. 수정하려는 내용을 지정합니다.
   - 카테고리명을 변경하려면 변경하려는 카테고리명을 수정합니다.
   - 카테고리의 순서를 변경하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_2.png)또는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_1.png)을 클릭하여 순서를 변경합니다.
   - 카테고리 내의 양식 순서를 변경하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_detail.png)을 클릭한 후 '순서 변경' 탭에서 순서를 변경하고 '저장'을 클릭합니다.
   - 카테고리 내의 양식을 다른 카테고리로 이동하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_detail.png)을 클릭한 후 '카테고리 변경' 탭에서 양식을 이동할 카테고리를 선택하고 '저장'을 클릭합니다.
5. '저장'을 클릭합니다.
   - 변경한 내용이 저장됩니다.

# 카테고리 삭제

카테고리를 삭제하는 방법은 다음과 같습니다.

등록된 양식이 있는 카테고리는 삭제할 수 없습니다. [양식 삭제](/ko/admin-guides/manage-service/peladge/manage-pleadgeforms/#i-2-4)를 참고해 양식을 먼저 삭제한 후 카테고리를 삭제해 주십시오.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업'에서 '서약서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '카테고리 관리'를 클릭합니다.
4. 삭제하려는 카테고리 항목의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_delete.png)을 클릭합니다.
   - 카테고리가 삭제됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
