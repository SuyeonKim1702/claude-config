# 서비스 개요

이 기능은 서약서 옵션 상품 신청 후 사용 가능합니다.

  
서약서 서비스에서는 회사의 중요한 규칙이나 지켜야 할 약속에 대해 서명 대상자로부터 서명을 받은 현황을 관리할 수 있습니다. 또한 용도에 맞게 서약서 양식을 직접 생성할 수도 있습니다.

서약서를 사용하려면 PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '서약서'를 클릭합니다.

- 서약서 메뉴는 관리자 권한이 있는 사용자에게만 노출됩니다.

# 서약서 관리자 시나리오

서약서 관리자의 이용 시나리오는 다음과 같습니다.

1. 서약서 서비스 기본 설정
   - 서약서 카테고리 추가, 수정, 삭제
   - 서약서 양식 추가, 수정, 삭제
   - 이 단계에 대한 사용 가이드 페이지
     - [Getting Started](/ko/admin-guides/manage-service/peladge/pleadge-home/#i-1)
     - [카테고리 관리](/ko/admin-guides/manage-service/peladge/pleadge-category/)
     - [양식 관리](/ko/admin-guides/manage-service/peladge/manage-pleadgeforms/)
2. 서약서 발송 및 현황 관리
   - 서약서를 추가하여 발송, 발송한 서약서 내용 확인, 수정, 삭제
   - 서약 현황 확인 및 서약서 회수
   - 이 단계에 대한 사용 가이드 페이지
# **자주 하는 질문을 확인해 보십시오.**

**Q. 회사에서 사용하던 서약서 양식이 있습니다. 네이버웍스 서약서에서도 사용할 수 있나요?**

- 이전에 사용하던 양식을 그대로 옮겨 사용할 수는 없으나, 서약서 양식 만들기 기능을 통해 손쉽게 양식을 만들고 운영할 수 있습니다. 서약서 양식 만들기에 대한 자세한 내용은 [양식 등록](/ko/admin-guides/manage-service/peladge/manage-pleadgeforms/#i-1)을 참고해 주십시오.

**Q. 특정 날짜에 서약서 서명을 요청하려면 어떻게 해야 하나요?**

- 서약서 서명 발송 시 예약 발송을 선택하면 원하는 날짜에 서약서 서명 요청이 전달됩니다.

**Q. 발송한 서명 요청을 수정한 후 다시 발송할 수 있나요?**

- 서명이 완료되지 않은 서약서는 요청을 취소할 수 있습니다. '서약 현황' 메뉴에서 취소할 서약서를 선택한 후 '회수'를 클릭해 주십시오. 회수된 서약서는 내용을 수정한 후 서명을 재요청할 수 있습니다.
- 서명이 완료된 서약서는 수정하거나 회수할 수 없습니다.

**Q. 서명된 서약서를 삭제할 수 있나요?**

- 서명이 완료된 서약서는 서명자 개인이 삭제할 수 없으며, 서약서 관리자가 삭제해야 합니다. 서약서 관리자가 서약서를 삭제 시 서명자 동의 요청 여부를 선택하며, 동의가 필요한 서약서는 서명자에게 삭제 동의 요청이 전송되어 서명자가 삭제를 동의한 후 삭제됩니다.

**Q. 서명된 서약서의 내용을 수정할 수 있나요?**

- 서약서는 회사의 중요 정보로 활용되므로 서명된 서약서 내용은 수정할 수 없습니다. 수정해야 할 내용이 있는 경우, 서약서 내용을 수정한 후 서명을 재요청해 주십시오.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
