# 서약서 발송

이 기능은 서약서 옵션 상품 신청 후 사용 가능합니다.

  
서약서를 발송할 수 있습니다. 또한 미발송, 발송 예약 상태의 서약서는 수정 또는 삭제할 수 있습니다.

서약서 발송 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 추가된 서약서를 상태, 카테고리, 양식명, 서약서명, 기간 조건에 따라 검색 |
| ② 추가 | 새로운 서약서 추가([서약서 추가](#i-1) 참고) |
| ③ 서약서 목록 | 추가된 서약서 조회 및 관리([추가한 서약서 발송](#i-2), [서약서 관리](#i-3) 참고) |

# 서약서 추가

발송할 서약서를 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '서약서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '서약서 발송'을 클릭합니다.
4. '추가'를 클릭합니다.
5. 카테고리 및 양식명을 선택한 후 서약서 이름을 입력합니다.
6. 서약서에 서명할 구성원 또는 부서를 검색해 추가합니다.
   - 하위 조직 포함을 선택하면 선택한 부서의 하위 조직 구성원 전체에게 서약서를 발송합니다.
7. 서약서 양식에 치환 문자가 적용된 경우, '파일 업로드'를 클릭하여 치환 문자 파일을 업로드해 주십시오.
   - '샘플 파일 다운로드'를 클릭하면 치환 문자 업로드 양식을 다운로드할 수 있습니다.
   - 엑셀 업로드 시 치환 문자의 셀 타입을 숫자로 지정한 경우에만 콤마가 적용됩니다.
8. '저장'을 클릭합니다.
   - 추가한 서약서가 저장되어 목록에 표시됩니다.

추가된 서약서는 자동으로 발송되지 않습니다. [추가한 서약서 발송](#i-2)을 참고하여 반드시 수동으로 발송해 주십시오.

# 추가한 서약서 발송

추가한 서약서를 서약 대상자에게 발송하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '서약서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '서약서 발송'을 클릭합니다.
4. 서약서 목록에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_send.png)을 클릭합니다.
5. 설정 팝업 창에서 설정 내용을 지정한 후 '확인'을 클릭합니다.
   - 서약서를 즉시 발송하려면 '즉시 발송'을 선택한 후 서명 만료일을 지정합니다. 서약서가 즉시 발송됩니다.
   - 서약서를 지정한 날짜에 발송하려면 '예약'을 선택한 후 서명 시작일과 서명 만료일을 지정합니다. 지정한 서명 시작일에 서약서가 발송됩니다.

# 서약서 관리

추가한 서약서의 상세 내용을 확인하거나, 수정하거나, 삭제할 수 있습니다.

## 서약서 상세 내용 확인

서약서 상세 내용을 확인하려면 서약서 목록에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_detail.png)을 클릭합니다.

- 서약 현황 화면으로 이동합니다.
- 서약 현황 화면에 대한 자세한 내용은 [서약 현황](/ko/admin-guides/manage-service/peladge/pleadge-status/)을 참고해 주십시오.

## 서약서 수정

서약서 상태가 '미발송' 또는 '발송 예약'인 경우 서약서의 내용을 수정할 수 있습니다.

서약서를 수정하는 방법은 다음과 같습니다.

1. 서약서 목록에서 수정할 항목의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_edit.png)을 클릭합니다.
2. 수정 내용을 적용한 후 '저장'을 클릭합니다.
   - 수정 내용이 저장됩니다.

## 서약서 삭제

서약서 상태가 '미발송' 또는 '발송 예약'인 경우 서약서를 삭제할 수 있습니다.

- 삭제한 서명 요청은 복구할 수 없습니다. 신중하게 삭제해 주십시오.
- 서약서의 상태가 '완료' 또는 '삭제 요청'인 경우 '서약서 발송'에서 삭제할 수 없습니다. '서약 현황' 메뉴에서 개별적으로 삭제해 주십시오.

서약서를 삭제하는 방법은 다음과 같습니다.

1. 서약서 목록에서 삭제할 항목의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_delete.png)을 클릭합니다.
2. 확인 팝업 창의 내용을 확인한 후 '삭제'를 클릭합니다.
   - 선택한 항목이 삭제됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
