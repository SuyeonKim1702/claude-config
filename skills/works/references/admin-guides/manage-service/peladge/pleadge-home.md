# 서약서 메인

이 기능은 서약서 옵션 상품 신청 후 사용 가능합니다.

  
서약서의 주요 진행 상황과 시간대별 서약서 관리 현황, 추가된 서약서 카테고리 및 양식을 확인할 수 있습니다. 또한 콘텐츠 영역에 표시되는 카드 또는 정보를 클릭하면 해당 기능을 바로 사용할 수 있습니다.

서약서 메인을 사용하려면 서약서 서비스에서 화면 왼쪽 상단의 '서약서'를 클릭합니다.

서약서 메인 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 콘텐츠 영역에 표시할 콘텐츠를 선택  - [Dashboard](#i-2): 서약 관련 요약 정보를 카드 형태로 표시 - [Timeline](#i-3): 최근 한 달 동안 서약서 서비스 내에서 기록된 로그를 100건까지 최신 순으로 표시 - [Getting Started](#i-1): 서약서 카테고리 및 양식 관리 링크 표시 |
| ② 콘텐츠 영역 | - 선택된 콘텐츠에 따라 서약서 현황, 로그 내역, 양식 표시 - 카드 형태로 표시되는 항목 또는 로그 내역을 클릭하면 해당 메뉴로 이동 |

# Getting Started

서약서 서비스에 처음으로 접속하면 서약서 관련 정보 설정을 안내하는 Getting Started 화면이 나타납니다. 각 설정 항목은 닫힌 카드 형태로 표시되며, 카드를 한 번 클릭하면 카드가 열려 기본 반영 사항이 표시되고, 한 번 더 클릭하면 해당 메뉴로 이동합니다.  
서약서 서비스를 원활히 사용하기 위해서는 Getting Started 화면의 카드 순서에 따라 모든 항목의 설정을 완료해 주십시오.

| 영역 | 설명 |
| --- | --- |
| ① 카테고리 관리 | 양식 카테고리를 추가하고 관리할 수 있는 [카테고리 관리](/ko/admin-guides/manage-service/peladge/pleadge-category/) 메뉴로 이동 |
| ② 양식 관리 | 서약서 양식을 추가하고 관리할 수 있는 [양식 관리](/ko/admin-guides/manage-service/peladge/manage-pleadgeforms/) 메뉴로 이동 |

# Dashboard

Getting Started 화면의 모든 설정을 완료한 후에는, 서약서 서비스에 접속하면 Dashboard 화면이 나타납니다. Dashboard 화면에는 서약서 관련 요약 정보가 카드 형태로 표시됩니다.

Dashboard 화면에 표시되는 카드는 다음과 같습니다.

| 영역 | 설명 |
| --- | --- |
| ① 진행 서약서 | - 발송된 서약서 중 진행 중인 서약서 건수 표시 - 클릭하면 [서약 현황](/ko/admin-guides/manage-service/peladge/pleadge-status/) 메뉴로 이동하며, 상태가 진행인 서약서 목록 표시 |
| ② 만료 서약서 | - 발송된 서약서 중 서명자가 서명하지 않았고 서명 만료일이 지난 서약서 건수 표시 - 클릭하면 [서약 현황](/ko/admin-guides/manage-service/peladge/pleadge-status/) 메뉴로 이동하며, 상태가 진행인 서약서 중 서명 만료일이 지난 서약서 목록 표시 |
| ③ 예약 서약서 | - 예약 발송으로 발송된 서약서 건수 표시 - 클릭하면 [서약 현황](/ko/admin-guides/manage-service/peladge/pleadge-status/) 메뉴로 이동하며, 상태가 예약인 서약서 목록 표시 |
| ④ 미발송 서약서 | - 아직 발송하지 않은 서약서 건수 표시 - 클릭하면 [서약 현황](/ko/admin-guides/manage-service/peladge/pleadge-status/) 메뉴로 이동하며, 상태가 미발송인 서약서 목록 표시 |

# Timeline

서약서 서비스의 최근 한 달간 사용 로그가 100건까지 최신 순으로 표시됩니다. 각 항목의 링크를 클릭하면 관련 메뉴로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
