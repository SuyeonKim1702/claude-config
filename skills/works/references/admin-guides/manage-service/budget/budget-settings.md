# 설정

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

# 예산 구분 설정

회사의 예산 정책에 따른 예산 구분을 추가할 수 있습니다. 또한 사용하지 않는 예산 구분을 삭제할 수 있습니다.

예산 구분 설정 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 구분 추가 | 새로운 예산 구분 추가([예산 구분 추가](#i-1-1)) |
| ② 예산 구분 목록 | 예산 구분 목록 표시  - 예산 구분 목록은 예산 기준 메뉴의 예산 구분 설정 팝업에 반영 |
| ③ 예산 구분명 | 클릭하여 예산 과목명 수정 가능 |
| ④ 삭제 | 예산 구분 삭제  - 예산 기준 설정에 사용된 예산 과목은 이 비활성화로 표시되며 삭제 불가 |
| ⑤ 취소 | 수정한 내용을 삭제하고 수정 전의 화면으로 새로 고침 |
| ⑥ 저장 | 수정한 내용 저장 |

## 예산 구분 추가

새로운 예산 구분을 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '구분 추가'를 클릭합니다.
5. 추가되는 영역에 예산 구분명을 입력 후 '저장'을 클릭합니다.
   - 예산 구분을 삭제하려면 삭제 아이콘을 클릭합니다.
   - 예산 기준 설정 시 사용된 예산 구분은 삭제할 수 없습니다.
   - 예산 구분명을 마우스로 원하는 순서에 끌어다 놓습니다.
6. 팝업창에서 '확인'을 클릭합니다.  

   예산 기준 메뉴에서 '예산 구분 설정'을 클릭하여 추가 또는 변경한 예산 구분은 '예산 구분 설정' 화면의 예산 구분 목록에 반영됩니다.

# 예산 과목 설정

예산 과목을 관리할 수 있습니다. 또한 예산 과목에 하위 과목을 설정하여 예산을 자세하게 구분하여 관리할 수 있습니다.

예산 과목 설정 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 추가 | 새로운 예산 과목 추가([예산 과목 추가](#i-2-1)) |
| ② 예산 과목 목록 | 예산 과목 목록 표시 |
| ③ 하위 추가 | 클릭하여 예산 과목의 하위 과목 추가  - 최대 4단계까지 추가 가능 |
| ④ 편집 | 예산 과목에 마우스를 올리면 표시, 클릭하여 예산 과목명 수정 |
| ⑤ 삭제 | 예산 과목에 마우스를 올리면 표시, 클릭하여 예산 과목 삭제  - 예산 기준 설정 시 사용된 예산 과목은 삭제 불가 |
| ⑥ 취소 | 수정한 내용을 삭제하고 수정 전의 화면으로 새로 고침 |
| ⑦ 저장 | 수정한 내용 저장 |

## 예산 과목 추가

새로운 예산 과목을 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '추가'를 클릭합니다.
5. 추가되는 영역에 예산 과목명을 입력 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_check.png)을 클릭합니다.
   - 하위 예산 과목을 추가하려면 예산 과목명에 마우스를 올린 후 '하위 추가'를 클릭합니다.
     - 하위 예산 과목명을 입력 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_check.png)을 클릭합니다.
   - 예산 과목 또는 하위 예산 과목을 삭제하려면 예산 과목에 마우스를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_delete6.png)을 클릭합니다.
   - 상위 예산 과목 삭제 시 해당 예산 과목의 하위 과목은 모두 삭제됩니다.
   - 예산 기준 설정 시 사용된 예산 과목 및 하위 예산 과목은 삭제할 수 없습니다.
   - 예산 과목명을 마우스로 끌어다 원하는 순서 및 레벨에 놓습니다.
6. '저장'을 클릭한 후 확인 팝업창에서 '확인'을 클릭합니다.  

   예산 기준 메뉴에서 '예산 과목 설정'을 클릭하여 추가 또는 변경한 예산 과목은 예산 과목 설정 화면의 예산 과목 목록에 반영됩니다.

# 예산 설정

예산의 결산 구분과 예산 단위를 설정할 수 있습니다.

예산 설정 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 결산 구분 | 결산 구분 월 선택  - 회계 서비스를 사용하는 경우 회계 서비스에서 설정한 결산 월로 자동 설정되며 변경 불가([회계 설정](/ko/admin-guides/manage-service/account/accounting_settings/#i-1) 참고) - 회계 서비스를 사용하지 않을 경우 관리자가 별도로 설정 가능 - 예산 수립에서 신규 예산 추가 시, 예산 사용 기간의 기본 값 - 예산 수립, 예산 배정, 각종 보고서에서 조회 조건의 예산 사용 기간 기본 값 - Dashboard에서 모든 항목의 조회 기간 |
| ② 예산 단위 | 예산 단위 선택  - 예산 단위: 보고서 메뉴에 적용되며, 예산 단위 선택 가능 - 통화: 경영지원 신청 시 선택한 기준 통화 표시 |
| ③ 예산 기준 코드 관리 | 예산 기준 추가 시 코드 생성 방식 선택  - 자동 생성: 예산 기준 추가 시 예산 기준 코드 자동 생성 - 사용자 입력: 예산 기준 추가 시 예산 기준 코드를 직접 입력(알페벳 B로 시작 불가) |
| ④ 예산 코드 관리 | 예산 추가 시 예산 코드 생성 방식 선택  - 자동 생성: 예산 추가 시 예산 코드 자동 생성 - 사용자 입력: 예산 추가 시 예산 코드를 직접 입력(알페벳 E로 시작 불가) |

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
