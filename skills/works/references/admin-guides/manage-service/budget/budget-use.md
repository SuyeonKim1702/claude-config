# 예산 사용

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
관련 서비스에서 예산의 사용 현황을 조회할 수 있습니다.

예산 사용 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 영역 | 예산 사용 현황을 사업장, 예산, 귀속 부서, 예산 기준 코드, 예산 구분, 예산 과목, 서비스 구분/항목, 문서 번호, 사용일, 완료 여부, 등록자, 프로젝트 조건으로 검색 |
| ② 다운로드 | 예산 사용 현황을 엑셀 파일로 다운로드([예산 사용 현황 다운로드](#i-0-1) 참조) |
| ③ 예산 사용 현황 목록 | 예산 사용 현황을 목록으로 확인  - 예산 기준: 클릭하여 예산 기준 상세 조회 - 문서 번호:   - 비용: 비용 정산 신청서 결재 문서 번호   - 지출: 지출 결의서 결재 문서 번호   - 회계: 전표 번호 또는 관리 번호 - 완료 여부:   - 비용: 비용 정산 신청서 최종 승인 완료   - 지출: 지출 결의서 최종 승인 완료   - 회계: 전표 승인 완료 또는 관련 전표 생성 완료 |

## 예산 사용 현황 다운로드

예산 사용 현황을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 사용'을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 예산 사용 내역을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 예산 사용 현황이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 예산 사용 현황이 엑셀 파일로 다운로드됩니다.

- 다운로드된 파일은 Download 폴더에 'Budgetuse.xlsx' 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
