# 예산 기준

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
회사의 예산 정책에 따른 예산 기준을 정의할 수 있습니다. 예산 수립 시 이미 정의된 예산 기준(사용)을 조회하여 선택할 수 있습니다. 또한 예산 기준의 변경 이력을 조회할 수 있습니다.

# 예산 기준 관리

예산 기준의 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 예산 기준을 예산 기준 코드, 예산 구분, 예산 과목, 서비스 항목, 사용 여부 및 최종 수정자 조건에 따라 검색 |
| ② 예산 기준 추가 | 새로운 예산 기준 추가([예산 기준 추가](#i-1-1) 참조) |
| ③ 선택 삭제 | 예산 기준을 선택하여 삭제  - 예산 수립에서 사용 중인 예산 기준은 삭제 불가 |
| ④ 복사 | 예산 기준을 복사하여 추가([예산 기준 복사](#i-1-2) 참조) |
| ⑤ 업로드 | 예산 기준 목록을 엑셀 파일로 일괄 업로드([예산 기준 업로드](#i-1-4) 참조) |
| ⑥ 다운로드 | 예산 기준 목록을 엑셀 파일로 다운로드([예산 기준 다운로드](#i-1-5) 참조) |
| ⑦ 예산 구분 설정 | 예산 구분 설정([예산 구분 설정](#i-1-6) 참조) |
| ⑧ 예산 과목 설정 | 예산 과목 설정([예산 과목 설정](#i-1-7) 참조) |
| ⑨ 사용 여부 | 각 예산 기준의 사용 여부 표시  미사용 상태에서는 예산 수립 화면에서 예산 기준 코드 조회 시 노출되지 않음 |
| ⑩ 예산 구분 | 예산 구분 선택 |
| ⑪ 예산 과목 | 예산 과목 선택 |
| ⑫ 서비스 구분 | 맵핑할 서비스 선택  - 비용, 지출 또는 회계 서비스 중 하나 이상의 서비스를 사용하는 경우에만 표시 |
| ⑬ 추가/삭제 | ⑫에서 선택한 서비스 구분에 따라, 맵핑할 서비스 항목 추가 또는 삭제 |
| ⑭ 서비스 항목 | - ⑫에서 선택한 서비스 구분에 따라, 맵핑할 항목 선택   - 회계 서비스의 경우 계정 과목 선택 시 서비스 항목 자동 표시   - 1개 또는 여러 개의 서비스 항목 추가 가능 - 항목 사용 여부: 예산 기준의 사용 여부와는 별도로, 하나의 예산 기준에 등록한 여러 서비스 항목들의 사용 여부를 각각 설정 가능. '사용 안 함'으로 설정한 서비스 항목은 예산 수립/배정 및 관련 서비스(비용/지출/회계)에서 사용할 수 없음 |
| ⑮ 계정 과목 | - 회계 서비스를 사용하는 경우에만 표시 - 비용, 지출 서비스는 각 서비스에서 맵핑된 계정 과목 자동 표시 - 회계 서비스는 맵핑할 계정 과목을 선택 |
| ⑯ 최종 수정자/최종 수정 일시 | 예산 기준의 최종 수정자, 수정 시 표시 |
| ⑰ 저장 | 수정한 내용 저장 |

## 예산 기준 추가

새로운 예산 기준을 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다
4. '관리' 탭을 클릭합니다.
5. '예산 기준 추가'를 클릭합니다.
   - 예산 기준 코드 관리를 '자동 생성'으로 설정한 경우 예산 기준 코드가 자동으로 표시됩니다.
7. 사용 여부를 선택합니다.
8. 예산 구분을 선택합니다.
   - '예산 구분 설정'을 클릭하여 예산 구분을 설정 후 추가할 수 있습니다. [예산 구분 설정](#i-1-6)을 참고해 주십시오.
   - 예산 구분은 '설정 > [예산 구분 설정](/ko/admin-guides/manage-service/budget/budget-settings/#i-1)'에서도 설정할 수 있으며 설정된 예산 구분은 동기화됩니다.
9. 예산 과목을 선택합니다.
   - '예산 과목 설정'을 클릭하여 예산 과목을 설정 후 추가할 수 있습니다. [예산 과목 설정](#i-1-7)을 참고해 주십시오.
   - 예산 과목은 '설정 > [예산 과목 설정](/ko/admin-guides/manage-service/budget/budget-settings/#i-2)'에서도 설정할 수 있으며 설정된 예산 과목은 동기화됩니다.
10. 맵핑할 서비스 구분을 선택합니다.
    - 비용 서비스를 사용하는 경우 '비용', 지출 서비스를 사용하는 경우 '지출', 회계 서비스를 사용하는 경우 '회계'가 표시됩니다. 비용, 지출 및 회계 서비스를 모두 사용하지 않는 경우 서비스 구분 항목은 표시되지 않습니다.
11. 맵핑할 서비스 항목을 선택합니다.
    - 8번 절차에서 선택한 서비스 구분에 따라, 비용 선택 시 비용 서비스에서 설정한 비용 항목, 지출 선택 시 지출 서비스에서 설정한 지출 항목이 표시됩니다.
    - 회계 서비스 선택 시 서비스 항목 선택은 비활성화로 표시되며, 선택한 계정 과목이 자동 표시됩니다.
12. 계정 과목을 선택합니다.
    - 회계 서비스를 사용하는 경우에만 표시됩니다.
    - 서비스 구분에서 회계를 선택한 경우 맵핑할 계정 과목을 선택합니다.
    - 서비스 구분에서 비용, 지출 서비스를 선택한 경우 선택한 비용 항목 또는 지출 항목에 맵핑된 회계 계정 과목이 자동으로 표시됩니다. 해당 계정 과목을 확인합니다. 비용 항목이나 지출 항목의 계정 과목 맵핑은 [비용 항목 관리](/ko/admin-guides/manage-service/expense/settings/#i-2), [지출 항목 관리](/ko/admin-guides/manage-service/expenditure/expenditure-item/)를 참고해 주십시오.
13. 해당 예산 기준에 동일 서비스 구분의 서비스 항목을 추가로 맵핑할 경우 추가 열의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_plus.png)을 클릭 후 9, 10번 절차를 반복합니다. 추가하지 않을 경우 이 절차를 건너뜁니다.
    - 맵핑한 서비스 항목을 삭제하려면 삭제 열의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_delete.png)을 클릭합니다.
14. '저장'을 클릭한 후 확인 팝업창에서 '확인'을 클릭합니다

## 예산 기준 복사

생성한 예산 기준을 복사하여 새로운 예산 기준을 추가할 수 있습니다.

예산 기준을 복사하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다.
4. '관리' 탭을 클릭합니다.
5. 예산 기준 목록에서 복사할 예산 기준을 선택합니다.
6. '복사'를 클릭합니다.
   - 항목이 복사된 상태로 새로운 행이 추가됩니다.
7. [예산 기준 추가](#i-1-1)를 참조하여 필요한 항목을 수정합니다.
8. '저장'을 클릭합니다.
   - 새로운 예산 기준이 생성됩니다.

## 예산 기준 수정

예산 기준을 수정할 수 있습니다.

예산 수립에 사용된 예산 기준은 예산 구분, 예산 과목, 서비스 구분 항목을 수정할 수 없습니다.

예산 기준의 수정 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다
4. '관리' 탭을 클릭합니다.
5. [예산 기준 추가](#i-1-1)를 참조하여 수정할 예산 기준의 필요 항목을 수정합니다.
   - 자동 생성한 예산 기준 코드는 수정할 수 없습니다.
   - 이미 사용한 예산은 예산 기준 항목을 수정할 수 없습니다.
6. '저장'을 클릭합니다.
   - 예산 기준이 수정됩니다.

## 예산 기준 업로드

예산 기준 목록을 엑셀 파일로 일괄 업로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다.
4. '업로드'를 클릭합니다.
5. '샘플 파일 다운로드'를 클릭하여 샘플 파일을 다운로드 받은 후 내용을 작성합니다.
6. '파일 첨부'를 클릭하여 작성한 파일을 업로드합니다.
7. '저장'을 클릭합니다.

## 예산 기준 목록 다운로드

예산 기준 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 예산 기준 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 예산 기준 목록이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 예산 기준 목록이 엑셀 파일로 다운로드됩니다.

- 다운로드된 파일은 Download 폴더에 'base.xlsx' 파일명으로 저장됩니다.

## 예산 구분 설정

회사의 예산 정책에 따라 예산을 큰 단위로 분류하여 설정할 수 있습니다.

예산 구분을 설정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다.
4. '관리' 탭을 클릭합니다.
5. '예산 구분 추가'를 클릭합니다.
6. 예산 구분 설정 팝업창에서 내용을 설정하고 '저장'을 클릭합니다.
   - '구분 추가'를 클릭합니다.
   - 추가된 행에 구분명을 입력합니다.
   - 예산 구분이 추가됩니다.
   - 사용하지 않는 예산 구분을 삭제하려면 예산 구분의 삭제 아이콘을 클릭합니다.
     - 예산 기준에 사용된 예산 구분은 삭제 아이콘이 비활성화로 표시되며 삭제할 수 없습니다.
   - 예산 구분명을 수정하려면 수정할 예산 구분명을 입력합니다.

## 예산 과목 설정

예산 과목을 설정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다.
4. '관리' 탭을 클릭합니다.
5. '예산 과목 추가'를 클릭합니다.
6. 예산 과목 설정 팝업에서 내용을 설정하고 '저장'을 클릭합니다.
   - '추가'를 클릭합니다.
   - 추가된 행에 예산 과목명을 입력 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_check.png)을 클릭합니다.
   - 생성한 예산 과목에 하위 예산 과목을 추가하려면 예산 과목명에 마우스를 올린 후 '하위 추가'를 클릭합니다.
     - 하위 예산 과목명을 입력 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_check.png)을 클릭합니다.
     - 최상위 단계를 포함하여 총 4단계까지 설정할 수 있습니다.
   - 예산 과목 또는 하위 예산 과목을 수정하려면 해당 예산 과목명에 마우스를 올려 편집 아이콘을 클릭합니다.
     - 수정할 예산 과목명을 입력합니다.
   - 사용하지 않는 예산 과목 또는 하위 예산 과목을 삭제하려면 예산 과목명에 마우스를 올려 삭제 아이콘을 클릭합니다.
     - 하위 예산 과목이 있는 예산 과목을 삭제하면 하위 예산 과목까지 모두 삭제됩니다.
     - 예산 기준에 맵핑된 예산 과목은 삭제할 수 없습니다.

# 변경 이력

예산 기준의 변경 이력을 조회할 수 있습니다.

변경 이력 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 예산 기준 코드, 변경 항목, 최종 수정자, 변경 일시 조건에 따라 검색 |
| ② 다운로드 | 예산 기준 변경 이력을 엑셀 파일로 다운로드 |
| ③ 예산 기준 변경 목록 | 예산 기준 변경 목록 표시  - 구분: 등록, 수정 또는 삭제 - 서비스 항목:변경된 서비스 항목 표시 - 변경 항목: 변경 구분이 수정인 경우 변경 항목 및 변경 전, 변경 후 내용 표시 |

## 예산 기준 변경 이력 조회

예산 기준 변경 이력을 조회하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다.
4. '변경 이력' 탭을 클릭합니다.
5. 필요한 경우, 검색창에서 조회할 예산 기준 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 예산 기준 변경 이력이 조회됩니다.

## 예산 기준 변경 이력 다운로드

예산 기준 변경 이력 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예산 기준'을 클릭합니다.
4. '변경 이력'탭을 클릭합니다.
5. 필요한 경우, 검색창에서 조회할 예산 기준 변경 이력 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 예산 기준 변경 이력 목록이 조회됩니다.
6. '다운로드'를 클릭합니다.
   - 예산 기준 변경 이력 목록이 엑셀 파일로 다운로드됩니다.

- 다운로드된 파일은 Download 폴더에 'BaseRevision.xlsx' 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
