# 서비스 개요

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
예산 서비스는 회사 예산의 통합적인 관리를 지원하는 서비스로서, 수립된 예산을 비용/지출/회계 서비스뿐만 아니라 프로젝트 코드까지 조회하고 사용할 수 있으며 사용한 예산을 관리할 수 있는 서비스입니다.

1. '예산 기준' 메뉴에서는 먼저 예산 수립을 위한 기준을 설정하여 예산의 다양한 속성을 정의합니다.
2. '예산 수립' 메뉴에서 이미 정의된 예산 기준을 이용하여 사업장/예산명/귀속 부서/프로젝트 코드/예산 사용 기간,사용 부서,예산 금액을 설정합니다. 또한 예산 전용, 이관 기능을 활용하여 배정 후 잔액 내에서 예산의 귀속 부서 또는 예산 과목을 일괄 변경할 수 있습니다.
3. '예산 배정' 메뉴에서는 수립된 예산 내에서 실제 배정할 금액을 지정하고 다양한 통제 기능을 적용할 수 있습니다.
4. 비용, 지출, 회계 서비스와 같이 관련 서비스에서 예산을 사용한 결과는 다양한 리포트 형태로 제공합니다.

예산 메뉴를 사용하려면 PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '예산'을 클릭합니다.

- 예산 메뉴는 관리자 권한이 있는 사용자에게만 노출됩니다.

# 예산 관리자 시나리오

예산 관리자의 이용 시나리오는 다음과 같습니다.

1. 예산 기본 속성 설정
   - 결산 구분월 및 예산 단위 설정
   - 예산 구분 설정
   - 예산 과목 설정
   - 예산 기준 설정
   - 이 단계에 대한 사용 가이드 페이지
     - [Getting Started](/ko/admin-guides/manage-service/budget/budget-home/#i-1)
     - [설정](/ko/admin-guides/manage-service/budget/budget-settings/)
     - [예산 기준](/ko/admin-guides/manage-service/budget/budget-standard/)
2. 예산 수립, 배정
   - 새로운 예산 추가 및 변경
   - 수립한 예산 배정
   - 예산 전용, 이관
   - 이 단계에 대한 사용 가이드 페이지
3. 예산 현황 조회
   - 예산 수립 현황 조회: 사업장별, 프로젝트별, 예산 과목별
   - 예산 배정 현황 조회: 귀속 부서별, 프로젝트별, 예산 과목별
   - 예산 사용 현황 조회: 사업장별, 월별, 프로젝트별
   - 이 단계에 대한 사용 가이드 페이지
     - [Dashboard](/ko/admin-guides/manage-service/budget/budget-home/#i-2)
     - [보고서](/ko/admin-guides/manage-service/budget/budget-reports/)
4. 사업장 권한 관리
   - 예산 서비스의 사업장 권한 부여 및 해제
   - 이 단계에 대한 사용 가이드 페이지
     - [사업장 권한 관리](/ko/admin-guides/manage-service/budget/unit-authority/)

# **자주 하는 질문을 먼저 확인해 보십시오.**

**Q. 예산 서비스 사용 신청은 어떻게 하나요?**

**Q.예산 수립을 위해 회사내 조직별로 예산 요청을 수렴하고 검토할 수 있는 절차를 제공하나요?**

- 예산 서비스에서는 이미 검토가 완료된 최종 예산을 반영하실 수 있도록 지원합니다.

**Q.사업장별 예산 수립이 가능하나요?**

- 해당 사업장 권한을 부여하시면 사업장별 예산 수립 및 배정이 가능합니다.

**Q. 프로젝트별 예산 수립이 가능하나요?**

- 예산 수립 시 현재 '사용 중'인 프로젝트 코드를 적용할 수 있으며, 이후 프로젝트 별로 예산 수립 및 배정, 사용 현황을 조회해 볼 수 있습니다.

**Q. 예산 수립부터 사용까지의 어떤 프로세스로 진행되나요?**

- '예산 기준 수립 - 예산 수립 - 예산 배정 - 예산 사용(비용/지출/회계 서비스에서 서식 및 전표 생성) - 보고서 조회(예산 및 사용 현황)'입니다.

**Q. '결산 구분'은 어떻게 설정하나요?**

- 회계 서비스를 사용 중일 경우 회계의 결산 구분 설정 값을 그대로 적용하며, 회계를 사용하지 않는 경우 사용자가 별도로 설정 가능합니다. 그러나 한 번 설정한 내용은 이후 변경이 불가능합니다.

**Q. 예산 과목 설정은 몇 단계까지 가능하나요?**

- '장-관-항-목'과 같은 형태를 지원하기 위하여 최대 4단계 구조를 지원합니다.

**Q. 수립하고 배정한 예산은 어디서 어떻게 사용하나요?**

- 비용 서비스를 사용하시면 비용 정산서 작성 시 해당 사용자가 '예산 사용 부서'로 지정된 예산을 조회하여 사용할 수 있으며, 지출 서비스를 사용하시면 지출 결의서 작성 시 해당 사용자가 '예산 사용 부서'로 지정된 예산을 조회하여 사용할 수 있습니다.

**Q. '귀속 부서'와 '사용 부서'의 차이는 무엇인가요?**

- 귀속 부서는 해당 예산 전체에 대한 책임 부서이며 1개의 부서만 선택 가능합니다. 사용 부서는 해당 예산을 사용할 수 있는 부서 또는 개인을 의미합니다.

**Q. 예산을 수립할 수 있는 기간은 어느 정도인가요?**

- 최대 2년의 예산을 수립할 수 있습니다.

**Q. '예산 기준'란 무엇인가요?**

- 예산의 기본 속성을 의미합니다. 예산 수립 및 배정 시 가장 중요한 자료로 사용되므로, 예산의 구조를 충분히 검토하시고 설정하시기 바랍니다.

**Q. '예산 구분'은 무엇인가요?**

- 각 회사의 예산 정책에 따라 예산을 큰 단위로 분류하여 설정할 수 있습니다.

**Q. '프로젝트 코드'는 언제 사용 가능하나요?**

**Q. 예산 '전용'과 '이관'은 무엇인가요?**

- 회사의 중요한 정책의 변경으로 여러 예산의 속성을 일괄 변경하는 것을 의미합니다.
- '예산 이관'은 수립된 예산 중 배정하고 남은 잔액 내에서 다른 귀속 부서로 변경 할 수 있으며, '예산 전용'은 수립된 예산 중 배정하고 남은 잔액 내에서 상위 예산 과목이 동일한 범주 내의 다른 예산 과목으로 변경할 수 있습니다.

- '겸직 포함' 체크시 해당 예산은 사용 부서의 겸직으로 소속되어 있는 구성원도 사용할 수 있습니다.  
  '하위 부서 포함'은 상위 부서에 수립된 예산을 그 하위 조직에서도 사용 가능합니다.

**Q. 예산 기준에서 '사용 여부'는 어떤 의미인가요?**

- 예산 기준에서 '사용 안 함'으로 설정 시 예산 수립 단계에서 노출하지 않습니다.

**Q. 예산 수립/ 배정에서 예산 상태 '오픈/마감'은 어떤 의미인가요?**

- 예산 수립 및 배정에서 예산 상태를 오픈으로 설정한 경우 사용자가 예산 조회 시 노출되며, 마감의 경우 노출되지 않습니다.

**Q. '예산 배정'에서 '예산 통제'는 어떤 의미인가요?**

- '예산 통제'를 '예'로 설정할 경우 배정된 예산을 초과하여 예산을 사용할 경우에 결재 상신 및 회계 전표 생성이 불가능합니다.

**Q. 예산 배정'에서 '이월 여부'는 어떤 의미인가요?**

- 월별 배정된 예산에서 사용하고 남은 잔액을 다음 달 월 배정 금액에 더하는 것을 의미합니다.
- '이월(분기 내)', '이월(반기 내)'는 해당 기간 내에서만 이월되며, 해당 기간 종료 시 잔액을 다음 기간으로 이월하지 않습니다.

**Q. 비용 정산서 및 지출 결의서 사용 시 '가용 예산'은 무엇인가요?**

- 해당 월에 실제 사용 가능한 예산을 의미합니다. 해당 월에 배정된 예산 금액과 이월 금액을 합산하여 '가용 예산'이라고 하며, 사용자는 '가용 예산' 범위 내에서 예산을 사용할 수 있습니다.

**Q. 사용하지 않는 예산을 삭제할 수 있나요?**

- 기본적으로 '설정-예산 기준 설정-예산 수립-예산 배정-예산 사용'의 프로세스로 진행했을 경우, 뒤에서부터 정리되어야 하며, 뒤에 하나라도 남아있을 경우 앞에서 삭제되지 않습니다. 자세한 내용은 아래 설명을 참고해 주십시오.
  1. 제공 서식(비용정산서, 지출결의서)에서 예산을 이용하여 결재가 완료되었을 경우, 해당 문서 삭제
  3. '예산 수립'에서 해당 예산 삭제
  4. '예산 기준'에서 해당 기준 삭제
  5. '설정'의 예산 구분 및 예산 과목 삭제

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
