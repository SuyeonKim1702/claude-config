# 예산 업로드

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
회사 예산을 엑셀 파일로 한 번에 업로드하여 수립 및 배정할 수 있습니다.

## 예산 일괄 업로드 단계

예산 일괄 업로드 단계는 다음과 같습니다.

| 단계 | 설명 |
| --- | --- |
| STEP 1: 예산 사용 기간 지정 | 예산 사용 기간은 최대 24개월까지 입력 가능  - 템플릿 파일 다운로드 시 ‘예산 사용 기간’만큼의 예산 월을 제공하며, 템플릿 파일 내에서 수정할 수 없습니다. |
| STEP 2: 템플릿 다운로드 | 템플릿 파일을 다운로드 후 내용 작성('예산 업로드 템플릿 작성 안내' 참고)  - 템플릿 파일 작성 시, 엑셀 파일 내 예산 사용 기간은 Step1에서 설정한 예산 사용 기간과 일치해야 합니다. 예산 사용 기간이 수정된 경우에는 템플릿 파일을 다시 다운로드 받아서 작성해 주십시오. |
| STEP 3: 첨부 파일 업로드 | 작성한 파일을 업로드 후 저장('업로드 데이터 확인 및 저장' 참고)  - 템플릿 파일 작성 시, 엑셀 파일 내 예산 사용 기간은 Step1에서 설정한 예산 사용 기간과 일치해야 합니다. - 저장 완료 시 '예산 수립/예산 배정' 화면에 반영됩니다. |

예산 업로드 템플릿을 작성하기 전에 아래 데이터를 다운로드 받아 주세요. 데이터 다운로드를 위해 해당 서비스의 담당자 권한을 부여 받거나, 해당 서비스 담당자로부터 파일을 전달 받으세요.

  - 필요 데이터: 이름/부서/로그인 아이디/재직 상태(재직인 사원)
  - 필요 데이터: 부서명, 부서 코드
  - 필요 데이터: 프로젝트 코드명/프로젝트 코드/사용 상태(사용)
  - 필요 데이터: 예산 기준 코드, 사용 여부(사용), 항목 사용 여부(사용)

## 예산 업로드 템플릿 작성 안내

예산 업로드를 위한 템플릿 작성 시 다음 내용을 참고해 주십시오.

- 예산 업로드 템플릿 파일 작성 시, 가급적 다운로드한 파일에서 복사하여 붙여 넣으세요.
- 다운로드한 파일이 아닌 다른 곳에서 복사하여 붙여 넣거나 수기로 입력할 경우, 공백 또는 특수 문자가 포함되지 않도록 주의해 주세요.
- 귀속 부서 및 사용 부서는 부서명/구성원 이름을 정확히 입력하거나, 동일한 부서명/구성원 이름이 존재할 경우, 부서 코드/구성원 로그인 아이디를 입력해 주십시오.
  - 구성원 이름: 회사가 다국어를 사용하는 경우, 사용자의 언어 설정에 따른 언어별 이름이 입력되어 있으면 해당 언어 이름을 입력해야 합니다.
- 각 예산 기준별 배정 금액은 예산 사용 기간에 해당하는 전체 월도별 배정 금액의 합과 동일해야 합니다.
- 부서명/부서 코드와 같이 명칭과 코드가 동시에 존재할 경우 코드를 우선으로 반영합니다.
- 템플릿 파일 작성 시, 엑셀 파일 내 예산 사용 기간은 Step1에서 설정한 예산 사용 기간과 일치해야 합니다.

| 사업장 | 예산 사업장을 선택하세요.  - 하나의 예산에 대한 사업장입니다. 하나의 예산의 가장 첫 행에만 입력하거나, 모든 열을 동일하게 채워도 됩니다. - 공란으로 둘 경우 모든 사업장에 상관없이 예산 사용이 가능합니다. | ABC Corporation |
| 예산명 | 예산명을 수기로 입력하세요.  - 하나의 예산에 대한 예산명입니다. 하나의 예산의 가장 첫 행에만 입력하거나, 모든 열을 동일하게 채워도 됩니다. | 모바일 개선 예산 |
| 귀속 부서명 | 부서명 또는 사원명을 정확하게 입력하세요. 하나의 예산에 대한 책임 부서 또는 책임 구성원입니다.  - (회사가 다국어를 사용하는 경우)사용자의 언어 설정에 따른 언어별 이름이 입력되어 있으면 해당 언어 이름을 입력해야 합니다. | 인사팀 또는 홍길동 |
| 프로젝트 코드명 | (예산에 프로젝트 코드 적용 시)프로젝트 코드명을 정확하게 입력해 주세요.  - 하나의 예산에 대한 프로젝트를 적용합니다. 프로젝트 코드 적용이 필요 없을 경우 비워두어도 됩니다. 하나의 예산의 가장 첫 행에만 입력해도 되며, 모든 열을 동일하게 채워도 됩니다. | 모바일 개선 프로젝트 |
| 예산 기준 순번 | 예산 기준의 순번을 수기로 입력하세요.  - 하나의 예산에 속한 하나의 예산 기준 단위입니다. 하나의 예산 기준에 해당하는 열에 동일한 숫자를 입력해야 합니다. - 예산 기준을 여러 개 추가할 경우 동일한 기준 내의 순번은 동일해야 합니다. | 1, 2, 3 … |
| 예산 기준 코드 | 예산 기준 코드를 입력하세요.  - 하나의 예산에 속한 하나의 예산 기준 코드입니다. - 하나의 예산 기준 코드 내의 모든 열은 동일하게 채워야 합니다. 하나의 예산에 여러 개의 예산 기준 코드를 적용할 경우, 아래의 파란색 참고 박스를 확인하여 작성해 주십시오. | B20230830002 |
| 사용 부서명 | 하나의 예산에 대한 사용 부서 또는 사용할 구성원입니다.  - 사원의 경우, 사원명을 입력하세요. 부서의 경우, 부서명을 입력하세요. - (회사가 다국어를 사용하는 경우)사용자의 언어 설정에 따른 언어별 이름이 입력되어 있으면 해당 언어 이름을 입력해야 합니다. - 하나의 예산 기준에 해당하는 모든 열을 입력해야 합니다. | 홍길동/인사팀 |
| 사용 부서 코드 | - 사원일 경우 사원의 로그인 아이디를 입력, 부서일 경우 부서 코드를 입력하세요. - 사용 부서명 또는 사용 부서 코드 중 하나만 입력해도 되나, 동일 부서명 또는 동일 구성원이 존재할 경우 부서 코드 또는 구성원의 로그인 아이디를 입력하세요. | abc@naver.com 또는 abc202305000030 |
| 겸직 포함 | 겸직 포함 여부를 선택하세요.  - 사용 부서별 겸직 포함 여부를 모두 입력해야 합니다. | 사용 |
| 하위 조직 포함 | 하위 조직 포함 여부를 선택하세요.  - 사용 부서별 하위 조직 포함 여부를 모두 입력해야 합니다. | 사용 |
| 배정 금액 | 배정 금액은 수립할 예산 금액으로 입력해 주세요.  - 하나의 예산(하나의 예산순번)에 배정할 금액을 예산 기준 코드별로 각각 가장 첫 행에 입력합니다. - 월별 배정 금액의 합과 동일한 값이어야 합니다. - 모든 행에 예산을 채우지 말아 주세요. | 1,200,000 |
| 예산 통제 여부 | 예산 통제 여부를 선택하세요.  - 하나의 예산에 속한 하나의 예산 기준 코드에 적용할 예산 통제 여부입니다. 예산 기준 코드별 가장 첫 행에 입력하며, 모든 열을 채우지 말아 주세요. | 사용 |
| 이월 여부 | 이월 여부를 선택하세요.  - 하나의 예산에 속한 하나의 예산 기준 코드에 적용할 이월 여부입니다. - 예산 기준 코드별 가장 첫 행에 입력하며, 모든 열을 채우지 말아 주세요. | 이월 |
| 배정 월별 금액 | 월별 예산 금액을 입력해 주십시오. 월별 예산 금액의 합은 배정 금액과 동일해야 합니다.  - 하나의 예산에 속한 하나의 예산 기준 코드에 적용할 예산 배정 금액입니다. 예산 기준 코드별 각각 가장 첫 행에 입력해 주세요. - 하나의 예산 기준 코드 내의 모든 열을 채우지 말아 주세요. | 100,000 |

# 업로드 데이터 확인 및 저장

예산 업로드 결과를 확인합니다. 작성한 템플릿에 오류가 있는 경우 화면 하단에 오류 메시지가 표시됩니다.

- 오류 메시지 영역의 '다운로드'를 클릭하여 업로드한 파일의 오류 내역을 다운로드할 수 있습니다. 수정 시 오류 메시지를 참고하여 엑셀 파일을 수정합니다.
- 예산 업로드 화면 하단의 '다시 올리기'를 클릭하여 수정한 파일을 다시 업로드합니다. 이 경우, 기존에 업로드한 파일을 덮어씁니다.

화면 하단의 '저장'을 클릭하여 저장합니다.

- 예산의 수립과 배정이 동시에 반영됩니다. 예산 수립, 예산 배정 목록에서 확인합니다

업로드 후 저장한 예산 데이터 삭제:

- 업로드한 예산은 일괄 삭제가 불가능합니다. 따라서 사전에 충분히 확인 후 업로드합니다.
- 업로드한 예산은 기존 예산 서비스의 예산 삭제와 동일한 프로세스로 삭제 가능합니다.
  - 먼저, 예산 배정 화면에서 삭제할 예산을 선택합니다. 화면 하단에 표시되는 월도별 배정 금액과 상단의 배정 금액을 모두 0으로 입력 후 저장합니다.
  - 다음, 예산 수립 화면에서 해당 예산을 삭제합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
