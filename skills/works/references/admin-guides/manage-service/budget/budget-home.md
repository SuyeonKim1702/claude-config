# 예산 메인

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
예산 서비스의 기본 설정 항목 및 예산의 수립, 배정 정보와 현황 관련 통계를 한 눈에 확인할 수 있습니다. 또한 콘텐츠 영역에 표시되는 카드 또는 정보를 클릭하면 해당 기능을 바로 사용할 수 있습니다.

예산 메인을 사용하려면 예산 서비스에서 화면 왼쪽 상단의 '예산'을 클릭합니다.

예산 메인 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 콘텐츠 영역에 표시할 콘텐츠를 선택  - [Dashboard](#i-2): 예산 관련 요약 정보를 카드 형태로 표시 - [Timeline](#i-3): 최근 한 달 동안 예산 서비스 내에서 기록된 로그를 100건까지 최신 순으로 표시 - [Getting Started](#i-1): 예산 서비스 이용에 필요한 설정을 카드 형태로 표시하며, 예산 설정, 예산 과목 설정, 예산 구분 설정, 권한 관리 현황을 표시 |
| ② 콘텐츠 영역 | - 선택된 콘텐츠에 따라 예산 현황, 로그 내역, 예산 서비스 관리 항목 표시 - 카드 형태로 표시되는 항목을 클릭하면 해당 메뉴로 이동 |

# Getting Started

예산 서비스에 처음으로 접속하면 예산 관련 정보 설정을 안내하는 Getting Started 화면이 나타납니다. 각 설정 항목은 닫혀진 카드 형태로 표시되며, 카드를 한 번 클릭하면 카드가 열려 기본 반영 사항이 표시되고, 한 번 더 클릭하면 해당 메뉴로 이동합니다.  
예산 서비스를 원활히 사용하기 위해서는 Getting Started 화면의 카드 순서에 따라 모든 항목의 설정을 완료해 주십시오.

| 영역 | 설명 |
| --- | --- |
| ① 예산 설정 | 예산 서비스를 이용하기 위한 기본 설정을 관리할 수 있는 [예산 설정](/ko/admin-guides/manage-service/budget/budget-settings/#i-3) 메뉴로 이동 |
| ② 예산 과목 설정 | 예산 과목을 등록하는 [예산 과목 설정](/ko/admin-guides/manage-service/budget/budget-settings/#i-2) 메뉴로 이동 |
| ③ 예산 구분 설정 | 예산 구분을 설정하고 관리할 수 있는 [예산 구분 설정](/ko/admin-guides/manage-service/budget/budget-settings/#i-1) 메뉴로 이동 |

# Dashboard

Getting Started 화면의 모든 설정을 완료한 후에는, 예산 서비스에 접속하면 Dashboard 화면이 나타납니다. Dashboard 화면에는 예산 관련 요약 정보가 카드 형태로 표시됩니다.

Dashboard 화면에 표시되는 카드는 다음과 같습니다.

| 영역 | 설명 |
| --- | --- |
| ① 요약 정보 | - 예산 기준, 수립, 배정 및 사용 관련한 요약 정보 표시 |
| ② 예산 보고서 | - 현재까지의 예산 수립, 배정 및 사용 현황을 다양한 형태의 기준으로 요약한 데이터 표시 |

# Timeline

예산 서비스의 최근 한 달간 사용 로그가 100건까지 최신 순으로 표시됩니다. 각 항목의 링크를 클릭하면 관련 메뉴로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
