# 서식

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
결재 서식과 서식의 결재선, 공유자, 사용 권한 등을 관리할 수 있습니다.

부재 일정, 근무 결과 신청, 비용 정산 신청 등 일반적인 업무를 위한 서식이 기본으로 제공되며, 필요한 경우 회사의 업무 목적에 맞는 서식을 직접 생성하여 사용할 수 있습니다.

# 서식 관리

기본으로 제공하는 서식을 확인하고 서식 목록에서 서식의 사용 여부, 진행 문서 수정 여부를 변경할 수 있습니다. 서식명을 클릭해 상세 화면에서 서식을 수정하거나 결재선, 수신/공유 범위 및 사용 권한을 수정할 수 있습니다. 회사가 필요한 서식을 직접 생성할 수 있습니다.

서식 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 서식을 카테고리, 서식명 조건에 따라 검색 |
| ② 서식 만들기 | 새로운 서식 생성([서식 추가](#i-1-1) 참고) |
| ③ 서식 목록 | 기본 제공 및 회사가 생성한 서식 목록 확인 및 관리  - 서식명: 클릭하여 서식 상세 설정 화면에서 수정 가능([서식 추가](#i-1-1) 참고) - 기본적으로 제공하는 서식은 수정이나 삭제할 수 없으며, 일부 서식은 진행 문서 수정 여부만 변경 가능 - 회사가 기본 제공 서식의 사용을 원하지 않는 경우, 서식 상세 설정 화면의 '권한 설정'에서 작성 권한에 대상자를 추가하지 않으면 모든 구성원에게 기본 서식이 보이지 않음 |
| ⑤ 진행 문서 수정 | 결재자가 문서 결재 시에 내용을 수정 가능 또는 불가능하도록 변경  - '결재 문서 수정'을 '사용'으로 설정 시 결재자는 문서 내용을 수정 후 결재 가능([문서 수정 후 동의](/ko/use-guides/approval/pc-web/approve-document/#i-3-7) 참고) - 문서의 수정 항목은 문서 조회 화면의 '정보 변경 이력'에서 확인 가능 - 매입 대금 정산 신청, 거래처 신청, 프로젝트 코드 신청, 급여 증명서 신청 서식은 결재 시 문서를 수정할 수 없음 |
| ⑥ 미리 보기 | 해당 서식의 내용 미리보기  PC 및 모바일 화면 미리 보기를 제공 |
| ⑦ 삭제 | 서식 삭제  - 기본적으로 제공하는 서식은 삭제 불가 - 영림원 연동 서식은 삭제 불가 |

## 서식 추가

새로운 서식을 생성할 수 있습니다.

새로운 서식을 생성하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '결재 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '서식 만들기'를 클릭합니다.  

   서식 만들기는 5가지 설정으로 구성됩니다.(비용 관련 서식은 '위반 결재선 설정'을 추가 제공)
   1. (필수)기본 설정
   2. (필수)서식 만들기
   3. 결재선 설정
   4. 수신·공유 설정
   5. 권한 설정
   - 기본 설정 및 서식 만들기 완료 후 저장 후에 다른 탭으로 이동할 수 있습니다. 단, 서식 수정 시에는 순서 구분 없이 탭 이동 가능합니다.
5. (필수)기본 설정: '기본 설정' 탭에서 서식 기본 정보를 입력합니다.
   - 카테고리: 기존 카테고리를 선택하거나 드롭다운 박스 하단의 '카테고리 추가'를 클릭하여 원하는 카테고리를 생성 후 선택합니다.
   - 서식명: 추가할 서식의 서식명을 입력합니다.
   - 문서 제목: 문서 작성 시 기본으로 제공되는 제목을 사용하도록 설정할 수 있습니다. 사용 설정(ON) 시 기본 제공할 제목을 입력합니다.
     - '문서 제목 수정 불가'에 체크하면 문서 작성 시 사용자는 관리자가 설정한 문서 기본 제목을 수정할 수 없습니다.
   - 서식 하단 설명 영역: 문서 작성에 대한 안내 문구를 서식별로 제공하려면 '기본으로 제공될 서식 하단 설명 작성'을 선택합니다. ([설명 영역 입력 예시](#i-1-12) 참고)
   - 작성 위치: '외부에서 상신' 선택 시 결재 서비스의 기안용 서식이 아닌, 외부 솔루션에서 결재 API 를 통해서만 상신 가능한 서식입니다. 외부 솔루션에서 문서를 작성하는 방법은 [외부 솔루션에서 결재 문서 작성](#i-1-13)을 참고해 주십시오.
     - '외부에서 상신'은 처음 설정한 후 변경할 수 없으므로 신중하게 선택해 주십시오.
     - 작성 완료 후 이동: 외부에서 상신 서식의 경우 표시되는 항목입니다. 외부 솔루션에서 서식 작성 완료 후 이동할 URL을 입력합니다.
     - 외부 솔루션에서 양식 작성 완료 후 문서 번호와 함께 해당 URL로 이동합니다. 입력하지 않으면 문서의 보기 화면으로 이동합니다.  

       영림원 ERP에서 작성하는 것으로 등록된 서식들은 '외부에서 상신'이 체크된 상태에서 비활성화됩니다.
   - 결재 명칭: 결재 관리의 '명칭 관리' 메뉴에서 서식별 결재 구분 명칭, 서식별 처리 버튼 명칭, 서식별 작성 버튼 명칭 중 하나라도 '사용'으로 설정한 경우 결재 명칭 영역이 표시되며([명칭 관리](/ko/admin-guides/manage-service/manage-approval/settings/#i-2) 참고), 서식별로 설정할 수 있습니다. 생성 중인 서식의 결재 구분, 처리 버튼, 작성 버튼을 설정해 주십시오.
     - 결재 구분: 각 결재 구분의 명칭을 선택합니다.([결재 구분 및 결재자 상세 설명](#i-1-10) 참고)
     - 처리 버튼: 결재 처리 버튼의 명칭을 선택합니다.
     - 작성 버튼: 문서 작성 시의 최종 저장 버튼의 명칭을 선택합니다.
   - '저장'을 클릭합니다. 신규 서식을 생성 중이라면 '서식 만들기' 탭을 클릭해 작성합니다.
6. (필수)서식 만들기: '서식 만들기' 탭을 클릭해 서식을 편집합니다.
   - 서식 편집 방법은 [서식 만들기](#i-1-2)를 참고합니다.
     - 서식 편집 후 저장 시 최종 편집 일시 및 편집자명이 표시됩니다.
   - 스마트 에디터: 서식에 스마트 에디터 컴포넌트가 포함된 경우 표시되는 항목입니다. '문서 작성 시 기본으로 제공되는 글양식 사용'을 선택하여 기본 표시 양식을 추가할 수 있습니다. 에디터 영역에 양식을 추가해 주십시오.
     - 문서 작성 위치를 '외부에서 상신'으로 선택한 경우, 기본으로 제공되는 글 양식 수정 기능은 제공되지 않습니다.
     - '파일 첨부'를 클릭해 설명 파일을 첨부할 수 있습니다.
     - 에디터 박스에 안내 문구를 입력합니다.
   - '저장'을 클릭합니다.  

     서식 편집 완료 후 저장 시 서식 관리 목록에서 서식을 조회할 수 있습니다. 서식의 사용 여부를 '사용'으로 설정하면 사용할 수 있습니다.  
     서식의 추가 설정은 탭(결재선 설정, 수신·공유 설정, 권한 설정)을 클릭해 정보를 입력합니다.
7. 결재선 설정: 서식의 결재선을 설정하려면 '결재선 설정' 탭을 클릭하여 결재선을 설정할 수 있습니다.
   - 결재선 설정 탭의 각 항목 디폴트 값은 아래와 같이 제공됩니다. 서식 결재선 설정은 선택 사항으로서, 관리자가 서식의 결재선을 별도로 설정하지 않을 경우 아래 디폴트 값을 그대로 유지해도 무방합니다.
     - 결재선 필수 여부: 필수 아님
     - 관리자가 결재선 설정: OFF
     - 결재선 변경 허용: 허용 안 함
     - 추가 설정
       - 상위 부서장이 없는 경우 생략 안 함
       - 문서 작성 시 서식 소속 회사 구성원만 결재자로 설정 안 함
     - 전결 관리: 사용 안 함
   - 서식의 결재선을 설정하려면 [결재선 설정](#i-1-3)을 참고합니다.
     - 관리자가 설정한 결재선은 서식 작성 시에 기본으로 표시되며 삭제할 수 없습니다.
     - 거래처 신청 서식, 프로젝트 코드 신청 서식의 경우 결재 연동을 사용하고 대표 관리자를 설정한 경우 해당 대표 관리자는 결재선에 자동 포함되며 삭제할 수 없습니다.([경영지원 설정 > 공통 코드 > 공통 코드 관리](/ko/admin-guides/manage-service/support-management/common-code/#i-1))
   - 서식 결재 시 전결 권한을 사용하려면 전결 권한 사용 여부를 '사용'으로 설정합니다.
     - 전결 권한의 설정은 [전결 설정](#i-1-4)을 참고해 주십시오.
     - '저장'을 클릭합니다.
     - 서식에 관리자가 설정한 결재선이 적용됩니다.
8. 결재선 설정(위반): 비용 정산/계획 신청 서식은 '결재선 설정(위반)' 탭이 추가로 표시됩니다.
   - 회사가 감사 항목을 사용하고 위반 결재선으로 결재 상신하도록 설정한 경우 위반 결재선을 설정할 수 있습니다. 비용 관련 서식의 위반 결재선을 설정하는 방법은 [결재선 설정](#i-1-3)을 참고해 주십시오.
   - 사용자가 비용 정산/계획 문서 상신 시 회사가 설정한 감사 항목에 위반하였을 때 위반 결재선이 적용됩니다.  

     비용 감사 서식의 위반 결재선과 관련된 설명은 [Self Check 관리](/ko/admin-guides/manage-service/expense/manage-check/)를 참고해 주십시오.
9. 수신·공유 설정: 서식의 수신 및 공유 정보를 설정하려면 '수신·공유 설정' 탭을 클릭해 정보를 입력할 수 있습니다.
   - 수신·공유 설정 탭의 각 항목의 디폴트 값은 아래와 같이 제공됩니다. 수신·공유 설정은 선택 사항으로서, 관리자가 서식의 수신·공유를 별도로 설정하지 않을 경우 아래 디폴트 값을 그대로 유지해도 무방합니다.
     - 수신 관리:
       - 상세 설정 사용(서식 상세 구분이 있는 경우 표시): 사용 안 함
       - 수신 사용 여부: 사용 안 함
       - 수신 대상 변경 허용: 허용 안 함
     - 공유 관리:
       - 상세 설정 사용(서식 상세 구분이 있는 경우 표시): 사용 안 함
       - 공유 범위: 전체 공유
       - 공유 시점: 완료 된 후 공유
       - 공유 대상 변경 허용: 허용 안 함
   - 서식의 수신, 공유 설정을 변경하려면 각각 [수신 설정](#i-1-5), [공유 설정](#i-1-6)을 참고하여 설정합니다.
     - '저장'을 클릭합니다.
     - 서식에 관리자가 설정한 수신/공유 설정이 적용됩니다.
10. 권한 설정: 서식의 작성 권한, 열람 권한을 설정하려면 '권한 설정' 탭을 클릭해 권한을 설정합니다.
    - 권한 설정 탭의 각 항목 디폴트 값은 아래와 같이 제공됩니다. 권한 설정은 선택 사항으로서, 서식의 권한을 별도로 설정하지 않을 경우 아래 디폴트 값을 그대로 유지해도 무방합니다.
      - 작성 권한: 전체 사원
      - 열람 권한: 권한 부여 대상 없음
    - 서식의 작성 권한 및 열람 권한을 변경하려면 [서식 권한 설정](#i-1-7)을 참고하여 설정해 주십시오.
11. '저장'을 클릭합니다.
    - 서식에 관리자가 설정한 권한이 적용됩니다.

## 서식 만들기

빈 서식에 모든 내용을 직접 구성하여 우리 회사만의 새로운 서식을 생성하거나, 이미 존재하는 다른 서식을 템플릿으로 활용하여 새로운 서식을 생성할 수 있습니다.

'서식 만들기' 탭에서 '서식 편집기'를 클릭합니다.

1. 서식 편집기 화면에서 서식을 추가할 템플릿을 선택합니다.
   - 웍스 결재 제공 템플릿: 웍스 결재에서 기본으로 제공하는 서식을 템플릿으로 사용하여 생성
     - 제공 템플릿 목록에서 템플릿으로 사용할 서식명을 클릭합니다.
     - 서식명에 마우스를 올려 서식에 포함된 컴포넌트를 확인할 수 있습니다.
   - 영림원 전용 템플릿: 영림원과 연동 설정된 회사에게만 노출되는 템플릿을 사용하여 생성
     - 템플릿 목록에서 템플릿으로 사용할 서식명을 클릭합니다.
   - 우리 회사 서식: 회사에서 이미 생성한 서식을 템플릿으로 사용하여 생성
     - 새로운 서식: 빈 서식으로 새롭게 생성하려면 '새로운 서식'을 클릭합니다.
     - 검색 영역에서 카테고리, 서식명을 조건으로 검색할 수 있습니다.
     - 회사 서식 목록에서 템플릿으로 사용할 서식명을 클릭합니다.
     - 그룹사로 이용하는 경우, 검색 조건에 회사가 추가되고 다른 회사에서 생성한 서식을 활용할 수 있습니다.
     - 그룹사에서 서식을 삭제할 경우 목록에 표시되지 않습니다.
2. 아래를 참고하여 컴포넌트 영역에서 컴포넌트를 클릭한 후 서식 공간 영역에 마우스로 끌어다 놓은 다음 상세 설정을 입력합니다.
   - 각 컴포넌트에 대한 상세한 설명은 [컴포넌트 상세 설명](#i-1-8)을 참고해 주십시오.

| 영역 | 설명 |
| --- | --- |
| ① 컴포넌트 영역 | 서식 공간 영역에 추가할 수 있는 컴포넌트 목록 |
| ② 서식 공간 영역 | 컴포넌트를 추가하여 서식 내용을 구성하는 영역  추가한 컴포넌트를 서식 공간 내에서 마우스로 끌어다 이동할 수 있으며, 마우스를 올리면 표시되는 을 클릭하여 삭제 가능  - 하나의 서식에 컴포넌트를 70개까지만 추가 가능 |
| ③ 상세 설정 영역 | 추가한 컴포넌트의 설정을 입력하는 영역  - 공통 설정: 추가한 컴포넌트 중 동일한 종류의 컴포넌트에 공통적으로 적용되는 이름, 설명 문구, 입력 필수 여부   - 결재자가 수정 후 승인한 문서는 수정된 항목이 파란색으로 표시되지만, 컴포넌트의 이름을 입력하지 않으면 수정 항목 확인 불가([문서 수정 후 동의](/ko/use-guides/approval/pc-web/approve-document/#i-3-7) 참고) - 상세 설정: 선택한 개별 컴포넌트의 상세 설정 |
| ④ 임시 저장 | 임시 저장한 서식 목록 |

- 현재 내용을 임시 저장하려면 '임시 저장'을 클릭합니다. 임시 저장한 서식은 오른쪽 상단의 임시 저장 목록에서 확인 및 불러올 수 있습니다.
- 작성한 서식을 미리 확인해 보려면 '미리 보기'를 클릭합니다.
- '서식 저장'을 클릭해 저장합니다.

영림원 전용 템플릿과 영림원 ERP에서 작성하는 것으로 등록된 서식들도 연관 문서, 첨부 파일 등의 컴포넌트를 넣을 수 있습니다.
