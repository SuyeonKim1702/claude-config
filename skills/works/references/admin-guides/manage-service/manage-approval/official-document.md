# 공문서

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
공문서 발신 서식을 관리하고 공문서 신청 내역을 조회할 수 있습니다.

# 공문서 관리

사용자가 신청할 수 있는 공문서 발신 서식을 추가하고 관리할 수 있습니다.

공문서 수신 신청 서식은 회사가 공문서 수·발신을 사용 시 기본으로 제공되는 서식으로, 양식을 편집하거나 추가할 수 없습니다.

공문서 관리 화면은 다음과 같이 구성되어 있습니다.![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/01/workplace-workplace-6-7_manageofficialdocument_ko.png)

| 영역 | 설명 |
| --- | --- |
| ① 공문서 추가 | 새로운 공문서 서식 추가([공문서 서식 추가](#i-1-2) 참조) |
| ② 목록 | 공문서 서식 목록  - 공문서명을 클릭하여 [공문서 서식 수정](#i-1-3) |
| ③ 사용 여부 | 공문서 서식의 사용 여부 설정 |
| ④ 미리 보기 | 클릭하여 공문서 서식 미리보기 |
| ⑤ 삭제 | [공문서 서식 삭제](#i-1-4) |

## 공문서 서식 추가

공문서를 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '결재 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '공문서 추가'를 클릭합니다.
5. 공문서의 서식을 구성합니다.
   - 공문서명: 공문서 서식명을 입력합니다.
   - 공문서 제목: 공문서 기본 제목의 제공 여부를 선택할 수 있습니다. 기본 제목을 제공할 경우 제공할 제목을 입력합니다. '문서 제목 수정 불가'를 선택하면 사용자는 제공되는 기본 제목을 수정할 수 없습니다.
   - 발신 공문 항목: 공문서 발신 신청 서식에 표시할 항목을 설정할 수 있습니다.
     - 발신 정보: 발신 정보를 표시할 경우, 회사 정보 또는 사업장을 선택 후 표시할 상세 항목을 선택합니다. 선택한 발신 정보는 사용자의 공문서 발신 신청 서식 작성 화면에 자동으로 표시되며, 사용자가 수정할 수 없습니다.
       - '직접 입력'을 선택 후 표시할 상세 항목을 선택하면, 사용자가 공문서 발신 신청 서식 작성 시 직접 입력할 수 있습니다.
     - 문서 정보: 표시할 문서 정보를 선택합니다.
       - 문서 번호는 아래 '공문서 번호' 항목의 양식 설정에 따라 결재 완료 시 공문서에 표시됩니다.
       - 시행 일자는 공문서 신청 날짜로 자동 표시되며, 사용자가 수정할 수 있습니다.
     - 수신 정보: 표시할 수신 정보를 선택합니다.
       - 수신처 또는 참조처를 선택하면 수신 세부 정보 또는 참조 세부 정보가 표시됩니다.
     - 기타: 회사 로고 표시 여부를 선택합니다. 'Admin'에 등록된 로고가 있는 경우 회사 로고가 표시됩니다.
   - 양식: 공문서 양식 스타일을 선택합니다. '미리보기'를 클릭하면 양식을 미리 확인할 수 있습니다.
   - 공문 기본 내용: 공문서의 기본 제공 글양식을 설정할 수 있습니다.
   - 공문서 번호: 공문서 번호 양식을 설정합니다.
6. '저장'을 클릭하면 추가한 공문서 서식이 저장됩니다.

## 공문서 서식 수정

공문서 서식을 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '결재 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 수정할 공문서 서식의 제목을 클릭합니다.
5. 수정 내용을 적용한 '저장'을 클릭합니다.
   - 수정된 내용이 저장됩니다.

## 공문서 서식 삭제

공문서 서식을 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '결재 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 수정할 공문서 서식의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_delete6.png)을 클릭합니다.
5. '저장'을 클릭합니다.
   - 공문서 서식이 삭제됩니다.
   - 작성한 공문서가 있는 서식은 삭제할 수 없습니다.

# 공문서 신청 내역

공문서 수·발신 내역을 조회할 수 있습니다. 또한, 조건에 따라 신청 내역을 검색하여 조회할 수 있으며, 공문서 수·발신 신청 목록을 엑셀 파일로 다운로드할 수 있습니다.

## 수신 내역

공문서 수신 신청 내역 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 공문서 수신 신청 내역을 상태, 문서 번호, 보관 부서, 제목, 기간, 작성자, 공문서 번호, 발신처에 따라 검색 |
| ② 다운로드 | 공문서 수신 신청 내역을 엑셀 파일로 다운로드([공문서 신청 목록 다운로드](#i-2-4) 참조) |
| ③ 신청 목록 | 공문서 신청 목록  - 상태: 수신 신청한 공문서의 결재 진행 상태 - 문서번호: 클릭하여 공문서 조회 |

## 발신 내역

공문서 발신 신청 내역 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 공문서 발신 신청 내역을 상태, 문서 번호, 공문서명, 제목, 기간, 작성자, 공문서 번호, 수신처에 따라 검색 |
| ② 다운로드 | 공문서 발신 신청 내역을 엑셀 파일로 다운로드([공문서 신청 목록 다운로드](#i-2-4) 참조) |
| ③ 신청 목록 | 공문서 발신 신청 목록  - 상태: 발신 신청한 공문서의 결재 진행 상태 - 문서 번호: 클릭하여 공문서 조회 - 발급일: 공문서를 인쇄한 최종 날짜(사용자 또는 관리자) |

## 공문서 신청 내역 다운로드

공문서 신청 내역을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '결재 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 다운로드할 탭을 선택합니다.
5. 필요한 경우, 검색창에서 조회할 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 공문서가 조회됩니다.
6. '다운로드'를 클릭합니다.
   - 공문서 신청 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 Downloads 폴더에 'history.xlsx' 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
