# 문서함

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
모든 결재 문서를 확인하고, 이관 문서나 수신/발신 문서를 관리할 수 있습니다.

# 결재 문서 관리

결재와 관련된 모든 문서를 확인할 수 있습니다. 각 문서의 상세 내용과 결재 진행 상태를 확인하고 결재선 변경, 공유 범위 변경 등의 작업을 수행할 수 있습니다.

결재 문서 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 결재 문서를 결재 진행 상태, 서식, 서식 구분, 문서 번호, 대상(작성자/결재자/공유자/기안 부서), 내용, 작성일 조건에 따라 검색  - 서식: 원하는 서식을 하나 이상 선택하여 문서 조회 가능   - 를 클릭하면 회사의 전체 서식 표시   - 검색어를 입력하면 서식 카테고리나 서식명에 검색어가 포함된 경우 포인트 색상으로 표시 - 결재자, 공유자로 검색 시 사원/부서로 검색 가능 - 그룹사로 사용하는 경우 회사로 검색 가능 |
| ② 공유 대상 추가 | 선택한 문서에 일괄 공유 대상 추가('[문서 조회 > 공유 문서 > 공유 대상 추가](/ko/use-guides/approval/pc-web/view-document/#i-1-7)' 참조) |
| ③ 숨기기/보이기 | 선택한 결재 문서를 목록에서 숨김 또는 숨김 취소  - 숨기기: 문서를 선택 후 '숨기기'를 클릭하면 사용자 화면에 문서가 표시되지 않음. 관리자는 결재 문서 관리에서 '숨긴 문서 포함'을 선택하여 조회 가능. 결재 진행 중인 문서는 숨기기 불가 - 보이기: 취소선이 표기된 문서를 선택한 후 '보이기'를 클릭하면 문서가 다시 관리자와 사용자의 화면에 표시 |
| ④ 다운로드 | 결재 문서 목록을 엑셀 파일로 다운로드 |
| ⑤ 숨긴 문서 포함 | 숨긴 결재 문서를 목록에 표시 |
| ⑥ 결재 문서 목록 | 결재 문서를 목록으로 확인하고 각 문서의 상세 내용 확인([결재 문서 상세 확인](#i-1-1) 참조)  작성자를 클릭하면 문서 작성자 정보 확인 가능  그룹사로 사용 시 '회사' 열 표시 |

## 결재 문서 상세 확인

각 결재 문서의 상세 내용을 확인하고 결재선 또는 공유 범위를 변경할 수 있습니다. 필요한 경우 문서를 삭제할 수 있습니다.

결재 문서의 상세 내용을 확인하는 방법은 다음과 같습니다.

1. PC 웹 화면에서 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '결재 관리'를 클릭합니다.
3. 확인할 결재 문서의 제목을 클릭합니다.
   - 결재 문서 팝업 창이 열립니다.

결재 문서 팝업 창은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 문서 내용 | 결재 문서의 내용 표시  - 부재 일정(대체 휴무) 신청 문서: 근무 결과 신청 문서 표시, 클릭하여 내용 조회 가능 - 부재 일정 취소/근무 결과 신청 취소 문서: 취소 대상 문서 표시, 클릭하여 내용 조회 가능 - 첨부 파일: 다운로드하거나 이미지 미리보기 가능 - 연관 문서: 클릭하여 연관 문서 조회   - 연관 문서는 문서 작성자 및 문서 결재자, 공유자까지 모두 열람 가능 |
| ② 알림 끄기/켜기 | 해당 문서와 관련된 알림 켬 또는 끔 |
| ③ 정보 변경 이력 | 결재 진행 중인 문서에 표시되며, 클릭하면 문서의 변경 이력 확인 |
| ④ 결재선 요약 | 결재자가 있는 경우 표 형태로 결재선 및 결재 결과 요약 표시(자세한 내용은 [문서 인쇄](#i-1-2) 참조)  - 서명 사용 시 결재 승인자의 서명을 표시 - 서명을 미등록한 결재자는 기본 제공 서명 표시(반려(지정 반려), 위임의 경우 '반려', '위임' 도장 아이콘 표시) |
| ⑤ 결재선 닫기/보기 | 결재선 및 공유 정보를 숨김 또는 숨김 취소 |
| ⑥ 결재선 | 문서의 결재선 정보 표시  - 보기: 문서의 전체 결재선을 확인 가능, 결재 구분에 '합의' 또는 '병렬 합의'가 포함된 경우 '합의 나누어 보기'를 선택하여 동의(자동 동의 포함)와 합의를 나누어 보기 가능 - 설정: 현재 결재자에 이은 다음 결재자 추가, 사원/부서로 추가할 수 있으며 [결재선 추가](/ko/admin-guides/manage-service/manage-approval/manage-forms/#i-1-3) 참조 - 위임: 대상자를 검색한 후 선택하여 결재 위임 - 부서 결재선:   - 문서 결재 전 결재자에 부서명(<예시>기획팀) 표시   - 문서 결재 후 결재 구분 뒤에 '부서' 표시(<예시>합의 결재-부서), 결재자에 결재한 사원명/부서명(<예시>홍길동/기획팀) 표시 |
| ⑦ 수신처 | (수신처가 설정된 문서)수신처 정보 표시  - 이력: 문서의 수신처 또는 담당자 변경 이력 확인 - 사원/부서 검색: 사원/부서로 검색하여 수신처 추가 가능 - 하위 조직 포함: 부서 수신처 추가 시 선택하면 하위 부서를 수신처에 포함 |
| ⑧ 공유 범위 | 문서의 공유 범위 변경, '전체 공유' 또는 '일부 공유'를 클릭 시 즉시 반영되며 공유 알림 발송  - 전체 공유: 전체 사용자에게 문서 공유 - 일부 공유: 지정된 대상자에게만 문서를 공유하며, 사원/부서로 검색하여 공유 대상을 추가하거나 기존 공유 대상자 삭제 가능   - 부서로 공유 시 '하위 조직 포함'을 선택하면 하위 조직 구성원 포함   - 부서 단위로 공유하면 신규 입사 또는 부서 이동 시 공유 대상을 변경할 필요 없이 문서가 공유됨   - '소속된 구성원 개별 입력'을 선택하면 부서 구성원을 모두 풀어서 표시함 |
| ⑨ 문서 의견 | 문서에 대한 의견을 입력 및 수정한 후 '확인'을 클릭하여 저장 |
| ⑩ 삭제 | 결재가 완료된 문서인 경우 표시([결재 문서 삭제](#i-1-3) 참조) |
| ⑪ 인쇄 | [문서 인쇄](#i-1-2) 참조 |

## 결재 문서 인쇄

문서를 인쇄하는 방법은 다음과 같습니다.

1. PC 웹 화면에서 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '결재 관리'를 클릭합니다.
3. 인쇄할 문서의 제목을 클릭합니다.
   - 결재 문서 팝업 창이 표시됩니다.
4. 결재 문서 팝업 창에서 인쇄 옵션을 선택한 후 '인쇄'를 클릭합니다.
   - 결재선 요약: 문서 상단에 결재 결과를 표로 요약해 인쇄합니다.
     - 합의 결재선: 결재선에 합의 결재선이 포함된 경우에만 표시됩니다. 선택 시 결재선 요약 영역에 합의 결재선을 포함해 인쇄합니다.
     - 서명 포함: 관리자가 서명을 사용으로 설정 시 표시됩니다. 선택 시 결재선 요약 영역에 결재자의 서명을 포함해 인쇄합니다.
       - 문서 작성 시점 기준의 개인별 서명 이미지가 인쇄됩니다. 문서 저장 이후에 변경된 서명 이미지는 해당 문서 인쇄 시에 적용되지 않으며 다음 문서부터 적용됩니다.
       - 서명은 도장 또는 서명 이미지 형식으로 표시될 수 있습니다. 서명 이미지 등록은 [서명 관리](/ko/use-guides/support-home/pc-web/pc-web-screen/#i-3)를 참조해 주십시오.
       - 기본 서명 이미지 표시의 경우: 서명을 등록하지 않은 결재자, 반려(지정 반려)한 결재자, 위임한 결재자
       - 사선(/) 표시의 경우: 전결 동의된 결재자
   - 결재선: 인쇄 화면 하단에 결재자별 결재 시간과 결재 의견을 포함해 인쇄합니다. 선택 해제 시 결재 의견과 공유 의견도 함께 인쇄에서 제외됩니다.
     - 결재 의견: 문서의 결재 의견을 포함해 인쇄합니다.
   - 공유 의견: 인쇄 화면 하단에 공유 의견을 포함해 인쇄합니다.
   - 영수증: 비용 정산 신청 문서의 경우 영수증을 포함해 인쇄할 수 있습니다.
   - 수신 원문: 일반 기안지(수신) 서식으로 작성된 문서에 표시되는 인쇄 옵션입니다. 선택 시 원문서를 포함해 인쇄합니다.
   - 수신처: 수신처가 설정된 서식에 표시되는 인쇄 옵션입니다. 선택 시 수신처 정보를 포함해 인쇄합니다.
   - 문서 인쇄 시 문서 하단에 회사 로고 이미지를 표시해 인쇄할 수 있습니다.
     - 회사 로고 이미지는 Admin의 '회사 로고 및 컬러 관리'에 로고 이미지를 등록한 경우 표시됩니다.

## 결재 문서 삭제

완료된 결재 문서와 문서에 담긴 데이터를 삭제할 수 있습니다.  

마감하기 전의 일반 문서, 증명서, 지출 결의서, 근무 계획 신청, 비용 계획 신청, 기타 사용자 설정 서식으로 작성된 문서는 즉시 삭제할 수 있습니다. 마감된 문서의 경우 다음과 같이 마감 해제 또는 확정 취소 작업을 먼저 수행해야 삭제할 수 있습니다.

- 근로 시간 마감: 기준일의 근무 기록이 확정된 근무 결과 신청 문서는 근무 관리자에게 요청하여 확정을 취소해야 삭제할 수 있습니다.
- 회계 전표 마감: 사용일의 해당 사업장 전표가 마감된 비용 정산 문서, 지출 결의 신청 문서는 회계 관리자에게 요청하여 마감 상태를 '오픈'으로 변경해야 삭제할 수 있습니다.
- 고정 자산 마감: 사용일의 해당 사업장 고정 자산이 마감된 지출 결의 문서는 회계 관리자에게 요청하여 마감 상태를 '오픈'으로 변경해야 삭제할 수 있습니다.

결재 문서를 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면에서 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '결재 관리'를 클릭합니다.
3. 필요한 경우, 검색창에서 조회 조건을 입력한 후 '검색'을 클릭합니다.
   - '숨긴 문서 포함'을 선택하면 숨긴 문서까지 조회합니다.
4. 결재 문서 목록에서, 결재가 완료된 문서 중 삭제할 문서의 제목을 클릭합니다.
5. 결재 문서 팝업 창에서 '삭제'를 클릭합니다.
6. 삭제 사유를 입력한 후 '확인'을 클릭합니다.
   - 문서를 삭제하지 않고 '숨기기'를 클릭하여 목록에서 숨겨지도록 처리할 수도 있습니다.

퇴직한 결재자가 포함된 진행 문서의 삭제:

- 결재가 완료된 문서만 삭제할 수 있습니다. 결재선에 퇴직자가 포함되어 결재 진행이 불가능한 경우, 문서의 상세 페이지에서 미처리 결재자를 결재 관리자에 위임하여 처리할 수 있습니다. 관리자가 위임 받은 문서를 처리하여 결재가 완료되면 해당 문서를 삭제할 수 있습니다.
