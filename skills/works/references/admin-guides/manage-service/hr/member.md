# 구성원

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
회사에 소속된 사원 정보를 조회하고 인사 발령 등 사원을 관리할 수 있습니다. 또한 Reports To를 임명하고 관리할 수 있습니다.

# 사원 관리

사원 정보를 관리할 수 있으며 인사 발령을 처리할 수 있습니다. 또한, 엑셀 파일로 사원의 인사 정보를 일괄로 수정할 수 있으며, 사원 목록을 엑셀 파일로 다운로드할 수 있습니다.

사원 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ② 업로드 | 사원 인사 정보 일괄 수정([사원 인사 정보 일괄 수정](#i-3) 참조) |
| ③ 다운로드 | 사원 목록을 엑셀 파일로 다운로드([사원 목록 다운로드](#i-4) 참조) |
| ④ 사원 목록 | 사원의 계정 상태, 재직 상태, 입사일, 이름, 부서, 사용자 유형, 직급, 직책, 로그인 아이디 표시  - 로그인 아이디는 ID 부분의 앞 두 자리를 제외한 정보를 마스킹 처리 - 재직 상태:   - 발령 사용 시 재직 대기, 재직, 휴직, 퇴직 또는 전적(그룹사로 사용 시)   - 발령을 사용하지 않는 경우 재직 대기, 재직, 휴직 |
| ⑤ 인사 카드 | 사원의 이름을 클릭하거나 을 클릭하여 해당 사원의 상세 정보 확인 및 관리([인사 카드](#i-2) 참조) |
| ⑥ 표시 항목 수 | 목록의 한 페이지에 표시할 사원 수 설정 |

# 인사 카드 조회/수정

사원 목록에서 사원의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/08/workplace_detail2-e1755766583861.png)을 클릭하면 해당 사원의 인사 카드를 조회할 수 있습니다. 인사 카드 화면에서는 사원 정보를 수정하거나 인쇄할 수 있으며, 해당 사원의 인사 발령을 할 수 있습니다.

인사 카드는 다음과 같이 구성되어 있습니다.

| 컨텐츠 탭 | 설명 |
| --- | --- |
| 발령 | 인사 발령 내역 표시 및 개인별 인사 발령 적용  - 발령 사용 시에 탭 표시 - 발령 추가: 입사일 있는 사원에 대해 발령 추가([인사 발령](#i-6) 참조) - 발령 초기화: 입사일 있는 사원에 대해 인사 발령 내역을 초기화('발령 초기화' 참조) |

'설정 > 발령 사용 관리'에서 발령 사용 여부를 '사용'으로 설정한 경우에만 인사 카드에 '발령' 탭이 표시되며, 해당 탭에서도 인사 발령할 수 있습니다.([발령 사용 관리](/ko/admin-guides/manage-service/hr/settings/#i-1) 참조)

## 인사 정보 수정

퇴직 사원은 퇴직일, 로그인 아이디 정보만 수정할 수 있습니다. 단, 퇴직 후 7일 이전에는 로그인 아이디를 수정할 수 없습니다.

인사 카드에 입력된 사원 정보를 수정하는 방법은 다음과 같습니다.

1. PC 웹의 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 사원 목록에서 정보를 수정할 사원의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/08/workplace_detail2-e1755766583861.png)을 클릭합니다.
5. '인사 정보' 탭에서 수정할 내용을 적용합니다.
   - '기본 정보' 영역
     - '사업장' 정보를 수정할 수 있습니다.
   - '추가 정보' 영역
     - 표시 항목은 인사 정보 설정에 따라 다를 수 있습니다.([인사 정보 설정](/ko/admin-guides/manage-service/hr/settings/#i-2) 참고)
     - 급여 서비스를 사용하는 경우 '인적 사항'은 필수 입력 사항입니다.
     - 가족 정보, 경력 정보, 어학 정보, 자격증, 학력 정보를 추가하려면 '추가'를 클릭 후 입력합니다.
       - 추가한 정보를 삭제하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_delete6.png)을 클릭합니다.

## 사원 퇴직

발령을 사용하는 인사 카드 하단의 '퇴직'을 클릭하면 인사 발령으로 이동하여 퇴직 발령할 수 있습니다.([퇴직 발령](#i-6-9) 참조)

- 퇴직 발령을 취소하려면 인사 발령 목록에서 퇴직 발령을 취소합니다.([인사 발령 취소](#i-7) 참조) 퇴직일로부터 7일 경과 후에는 퇴직 발령을 취소할 수 없습니다.

퇴직 사원의 경우, 인사 카드 하단의 '삭제'를 클릭하면 퇴직 사원을 영구 삭제할 수 있습니다. 영구 삭제 시 사원 정보를 복구할 수 없습니다.

### 사원 정보 인쇄

사원 정보를 인쇄하는 방법은 다음과 같습니다.

1. 사원 목록에서 정보를 인쇄할 사원의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/08/workplace_detail2-e1755766583861.png)을 클릭합니다.
2. '확인'을 클릭합니다.
   - 각 콘텐츠 탭 화면 하단에 표시됩니다.
3. 인쇄할 항목을 선택한 후 '확인'을 클릭합니다.
4. 인쇄 팝업 창이 나타나면 인쇄할 항목을 확인한 후 '인쇄'를 클릭합니다.

### 발령 초기화

구성원의 발령 내역에 기본 발령이 제대로 등록되지 않았거나, 발령 기록을 모두 삭제하고 싶은 경우 발령을 초기화할 수 있습니다.

발령 초기화 시 기존 발령 내역이 모두 삭제되므로, 초기화 실행 전 인사 발령 화면에서 발령 내역을 다운로드합니다.

입사일이 없는 사원은 '발령 초기화' 버튼이 활성화되지 않습니다.

발령 내역을 초기화하는 방법은 다음과 같습니다.

1. 사원 목록에서 발령 내역을 초기화할 사원의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2025/08/workplace_detail2-e1755766583861.png)을 클릭합니다.
2. 인사 카드에서 '발령' 탭을 클릭합니다.
3. '발령 초기화'를 클릭합니다.
4. 알림창에서 '확인'을 클릭합니다.
5. '저장'을 클릭합니다.
   - 구성원의 발령 내역이 초기화되며, 기본 발령 내역을 생성합니다.
     - 입사일을 발령일로 하는 부서 이동, 사용자 유형 변경 및 직급/직책 변경의 발령 내역을 생성합니다.
   - 발령 초기화 시점의 구성원의 소속 부서, 사용자 유형, 직급, 직책 정보로 기본 발령을 등록합니다.
   - 발령 초기화 후 발령 내역을 취소할 수 없습니다.
   - 발령 초기화 시 기존 휴직 상태는 재직 상태로 변경됩니다.

# 사원 정보 일괄 수정

기존 등록된 사원의 기본 정보 중 사업장 정보, 추가 정보, 부가 정보를 일괄로 수정할 수 있습니다.

사원 정보를 일괄로 수정하는 방법은 다음과 같습니다.

1. PC 웹의 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '업로드'를 클릭합니다.  

5. 수정이 필요한 정보의 템플릿을 클릭하여 다운로드합니다.
   - 다운로드한 템플릿은 Download 폴더에 '[YYYY-MM-DD]sample update [수정항목명]' 파일명으로 저장됩니다.
   - 추가 정보에 해당하는 템플릿은 관리자가 '인사 정보 설정' 메뉴에서 사용하도록 설정한 항목만 표시됩니다. 템플릿 목록에 없는 경우, [인사 > 설정 > 인사 정보 설정](/ko/admin-guides/manage-service/hr/settings/#i-2)에서 사용 여부를 '사용'으로 변경해 주세요.
     - 급여 서비스를 사용하는 경우 인적 사항의 '국적', '주민등록번호/외국인 등록번호'는 필수 항목으로 관리하며, 가족 정보 입력 시 가족의 '이름'과 '주민등록번호'는 필수로 입력합니다.
       - '기본 공제 대상 여부'는 급여에서 부양 가족의 공제 대상 여부를 판단하는 정보로 사용됩니다.
   - 부가 정보 템플릿은 관리자가 [설정 > 인사 정보 설정](/ko/admin-guides/manage-service/hr/settings/#i-2)에서 부가 정보를 추가한 경우에만 표시됩니다. 일괄 업로드할 부가 정보가 있는 경우, 인사 정보 설정 메뉴에서 부가 정보를 먼저 추가해 주십시오.
6. 템플릿에 수정할 내용을 입력한 후 저장합니다.
   - 모든 템플릿 양식은 그대로 유지하고, 수정할 사원의 정보만 입력합니다.

   정보에 따라 업로드 시 반영되는 방식은 다음과 같습니다.

   - 사업장/인적 사항
     - 기존 정보를 유지하며, 변경한 정보만 반영합니다.
     - 빈 값으로 업로드하는 경우, 기존 정보를 유지합니다. 변경이 필요 없는 항목은 빈 항목으로 유지해 주십시오.
   - 가족 정보/경력 정보/어학 정보/자격증/병역 정보/학력 정보/부가 정보
     - 기존 정보를 모두 표시한 상태로 템플릿 파일을 다운로드합니다.
     - 수정한 내용을 업로드하면 기존 정보를 모두 지운 뒤 변경한 내용으로 반영합니다.
