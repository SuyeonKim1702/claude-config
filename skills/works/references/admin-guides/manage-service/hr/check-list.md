# 입퇴사 체크리스트

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
입퇴사 체크리스트를 등록할 수 있습니다. 업무 목적에 따라 여러 개의 체크리스트를 하나의 그룹으로 묶어 할당하고 관리할 수 있습니다.

# 체크리스트 현황

입·퇴사자가 체크해야 할 업무 리스트를 그룹별로 할당할 수 있으며, 할당된 체크리스트를 입사자별 또는 체크리스트별로 구분하여 확인할 수 있습니다. 또한, 조건에 따라 체크리스트 할당 내역을 검색하여 조회할 수 있으며, 체크리스트 할당 목록을 엑셀 파일로 다운로드할 수 있습니다.

체크리스트 현황 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 체크리스트 할당 내역을 조회할 기준  - 입·퇴사자: 입·퇴사자별로 체크리스트 현황 조회 - 체크리스트별: 체크리스트별로 체크리스트 현황 조회 |
| ② 검색창 | 체크리스트를 구분, 그룹명, 입·퇴사자, 할당 기간, 처리자, 체크리스트명, 업무 담당자, 상태에 따라 검색 |
| ③ 할당 추가 | 입·퇴사자에게 체크리스트 할당([체크리스트 할당](#i-1-1) 참고) |
| ④ 삭제 | 할당된 체크리스트 삭제([할당된 체크리스트 삭제](#i-1-4) 참고) |
| ⑤ 다운로드 | 체크리스트 할당 목록을 엑셀 파일로 다운로드([체크리스트 할당 목록 다운로드](#i-1-3) 참고) |
| ⑥ 체크리스트 현황 | 할당된 전체 체크리스트의 진행 현황 표시  - 진행: 처리 기한 내 진행 중인 체크리스트 수 - 완료: 처리 기한 내 완료된 체크리스트 수 - 지연: 처리 기한 내 완료되지 않아 수행할 수 없는 체크리스트 수 |
| ⑦ 체크리스트 할당 목록 | 할당된 체크리스트 정보 및 진행 현황 확인  - '입·퇴사자' 탭에서 그룹명을 클릭하면 할당된 체크리스트 세부 목록 표시 - '체크리스트별' 탭에서 체크리스트명을 클릭하면 체크리스트 상세 화면 표시 |
| ⑧ 지연 데이터만 조회 | 선택하면 체크리스트 할당 내역 중 지연된 항목만 목록에 표시 |

## 체크리스트 할당

체크리스트를 할당하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '할당 추가'를 클릭합니다.
5. 구분을 선택합니다.
   - 구분에 따라 선택할 수 있는 그룹명이 달라집니다.
6. 할당할 체크리스트의 그룹명을 선택합니다.
   - 체크리스트를 그룹에 추가하고 관리하려면 [체크리스트 그룹 관리](#i-4)를 참고해 주십시오.
   - 그룹에 포함된 체크리스트가 체크리스트명에 표시됩니다.
7. 할당 대상을 검색하여 추가합니다.
   - 사원 또는 부서 단위로 할당할 수 있습니다.
     - 입사 체크리스트 할당의 경우, 입사일이 없는 사원은 할당할 수 없습니다. 또한 부서 단위 할당 시, 입사일이 없는 사원은 할당에서 제외됩니다.
   - 아르바이트 사원을 할당 대상에 추가할 수 있습니다.
   - 할당 대상의 하위 조직을 포함하려면 '하위 조직 포함'을 선택합니다.
   - 처리 기한이 퇴사일을 기준으로 설정된 체크리스트를 할당할 경우, 할당 대상자의 퇴직(예정)일이 등록되어 있어야 합니다.
   - 처리 기한을 퇴사일 이후로 설정한 경우, 해당 사원의 퇴사일 이전에는 체크리스트를 수행할 수 없습니다.
8. '할당'을 클릭합니다.
   - 할당 대상에게 할당된 체크리스트 안내 메일이 발송됩니다.
   - 입퇴사자가 받는 체크리스트 할당 메일에서는 처리자가 입퇴사자인 체크리스트만 노출됩니다.

## 할당된 체크리스트 상세 조회 및 관리

'입·퇴사자' 탭에서 체크리스트 할당 목록의 그룹명을 클릭한 후 체크리스트명을 클릭하거나 '체크리스트별' 탭에서 체크리스크명을 클릭하면 체크리스트 상세 화면으로 이동합니다.

체크리스트 상세 화면에서는 체크리스트 내용을 확인하거나 수정할 수 있습니다. 또한, '리마인더 요청'을 클릭하여 체크리스트 처리자에게 리마인더 알림 메일을 발송할 수 있으며, 댓글을 작성할 수 있습니다.

체크리스트 상세 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 기본 정보 | 체크리스트의 기본 정보 표시  - 입·퇴사자명: 클릭하여 입사자 또는 퇴사자 정보 확인 - 처리자: 체크리스트의 처리자(입퇴사자/부서장/업무 담당자) 표시, 을 클릭하여 수정 가능 - 처리 기한: 체크리스트 처리 기한 표시, 기한 내 처리하지 못한 경우 지연 상태로 표시, 오늘 이후의 날짜로 수정 가능 |
| ② 추가 정보 | 링크, 인사 정보, 서약서, 서식, 첨부 파일, 업무 담당자, 상태 표시  - 링크: 체크리스트에 입력된 링크 표시 - 인사 정보: 내 인사 정보 링크 제공 여부 표시 - 서약서: 첨부된 서약서명 표시 - 서식: 첨부된 서식명 표시 - 파일 첨부: 첨부된 파일을 표시하며, 을 클릭하면 '파일 첨부'와 파일 첨부 영역 표시 - 업무 담당자: 처리자가 업무 담당자이거나 업무 담당자가 설정된 경우 업무 담당자 표시, 사원명을 검색하여 업무 담당자 추가 (그룹사를 사용하는 경우 업무 담당자는 입퇴사자와 동일 그룹사인 구성원만 등록할 수 있습니다.) - 상태: 체크리스트 진행 현황 표시 |
| ③ 처리 이력 | 체크리스트 할당, 수정, 리마인더 요청 및 완료 이력 표시  - 할당 외 처리 이력이 없는 경우 '처리 이력' 미표시 |
| ④ 댓글 | 클릭하여 체크리스트에 대한 댓글 등록  - 댓글 등록 시 알림 메일 발송 - 등록된 댓글이 있는 경우 댓글 수 표시 |
| ⑤ 리마인더 요청 | - 체크리스트 처리자가 입사자/퇴사자 본인이 아니며, 상태가 진행인 경우 표시 - 클릭하면 체크리스트 처리자에게 리마인더 알림 메일 발송 |
| ⑥ 수정 | 클릭하여 체크리스트 내용 수정  - 비활성화된 항목은 을 클릭하여 수정한 후 '수정' 클릭 - 체크리스트를 수정한 이력은 '처리 이력'을 클릭하여 확인 |

지연된 체크리스트는 관리자가 처리 기한을 수정하여 연장할 수 있습니다. 처리 기한이 수정되면 상태가 지연에서 진행으로 변경됩니다.

## 체크리스트 할당 목록 다운로드

체크리스트 할당 목록을 다운로드하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 체크리스트 할당 조건을 입력한 후 '검색'을 클릭합니다
   - 입력한 조건에 해당하는 체크리스트 할당 내역이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 체크리스트 할당 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Download` 폴더에 `[YYYY-MM-DD]checkList_stateByEmp.xlsx` 또는 `[YYYY-MM-DD]checkList_stateByCheckList.xlsx` 파일명으로 저장됩니다.

## 할당된 체크리스트 삭제

할당된 체크리스트를 삭제하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 체크리스트 할당 목록에서 삭제할 항목을 선택합니다.
5. '삭제'를 클릭합니다.
   - '입·퇴사자' 탭에서 삭제할 경우, 선택한 입·퇴사자의 체크리스트가 모두 삭제됩니다.
   - '체크리스트별' 탭에서 삭제할 경우, 선택한 체크리스트만 삭제됩니다.

# 체크리스트 관리

입·퇴사 체크리스트를 등록하고 관리할 수 있습니다. 또한, 조건에 따라 체크리스트를 검색하여 조회할 수 있으며, 체크리스트 목록을 엑셀 파일로 다운로드할 수 있습니다.

체크리스트 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 체크리스트를 구분, 카테고리명, 체크리스트명, 처리자 구분에 따라 검색 |
| ② 추가 | 새로운 체크리스트 추가([체크리스트 추가](#i-2-1) 참고) |
| ③ 다운로드 | 체크리스트 목록을 엑셀 파일로 다운로드([체크리스트 목록 다운로드](#i-3-4) 참고) |
| ④ 체크리스트 목록 | 추가된 체크리스트를 표시  - 체크리스트명을 클릭하여 체크리스트 확인, 수정, 삭제 또는 복사([체크리스트 상세 내용 확인](#i-3), [체크리스트 수정](#i-3-1), [체크리스트 삭제](#i-3-3), [체크리스트 복사](#i-3-2) 참고) |

## 체크리스트 추가

체크리스트를 추가하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '추가'를 클릭합니다.
5. 기본 정보를 입력합니다.
   - 체크리스트를 관리할 카테고리명을 선택합니다. 카테고리에 관한 내용은 [카테고리 관리](#i-5) 메뉴를 참고해 주십시오.
   - 처리자를 선택합니다.
     - 업무 담당자: 업무상 체크리스트를 수행할 담당자이며, 선택 시 추가 정보의 업무 담당자 항목이 필수 항목으로 전환됩니다.
     - 부서장: 입·퇴사자가 소속한 부서의 부서장이 체크리스트를 수행하며, 아르바이트 사원은 근무지 매니저가 부서장으로 적용됩니다.  

       - 상위 부서장이 공석인 경우 차상위 부서장이 처리자로 배정됩니다.
       - 할당된 체크리스트에 대해, 부서장이 퇴직(또는 삭제) 또는 임명 해제 상태인 경우 관리자에게 처리자 변경 안내 메일이 발송됩니다. 관리자는 해당 체크리스트의 처리자를 변경할 수 있습니다.
   - 유효하지 않은 직원이 처리자로 지정된 경우, 관리자에게 안내 메일이 발송됩니다.
6. 추가 정보를 입력합니다.
   - 여러 개의 링크를 등록하려면 '추가'를 클릭합니다.
   - '인사 정보' 링크 제공 여부를 선택합니다.
     - 선택할 경우 사용자의 인사 정보 화면의 링크가 제공되어 인사 정보를 바로 수정할 수 있습니다.
   - 서약서를 추가하려면 '서약서 선택'을 클릭합니다.
     - 서약서 서비스를 사용하는 경우에만 서약서 항목이 표시되며, 양식 내 치환 문자가 포함된 경우 추가할 수 없습니다.
   - 서식을 추가하려면 '서식 선택'을 클릭합니다.
     - 사용 중인 서비스 및 사용 설정에 따라 선택 가능한 서식이 다를 수 있습니다.
   - 파일을 첨부하려면 '파일 첨부'를 클릭하거나 '마우스로 파일을 끌어오세요.' 영역에 파일을 끌어다 놓습니다.
   - 업무 담당자를 추가하려면 추가할 사원명을 검색하여 추가합니다.
     - 체크리스트 처리자로 설정되지 않은 업무 담당자는 해당 체크리스트 업무에 대해 관리하는 역할을 담당합니다.
7. '저장'을 클릭합니다.

기 작성된 체크리스트와 동일한 내용으로 체크리스트를 추가하려면 [체크리스트 복사](#i-3-2)를 참고해 주십시오.

# 체크리스트 상세 내용 확인

체크리스트 상세 내용을 확인하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 체크리스트 목록에서 상세 내용을 확인할 체크리스트명을 클릭합니다.
   - 체크리스트 수정 화면으로 이동합니다.

## 체크리스트 수정

체크리스트를 수정하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 체크리스트 목록에서 수정할 체크리스트의 체크리스트명을 클릭합니다.
5. 수정 내용을 적용한 후 '저장'을 클릭합니다.
   - 파일을 첨부하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_edit.png)을 클릭합니다.
6. 수정 사유를 입력한 후 '확인'을 클릭합니다.
   - 수정한 내용이 반영됩니다.

체크리스트 수정 화면 오른쪽 상단의 '수정 이력'을 클릭하여 체크리스트 최초 생성일과 수정 이력을 확인할 수 있습니다.

## 체크리스트 복사

체크리스트 내용을 복사하여 새로운 체크리스트를 추가하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 체크리스트 목록에서 복사할 체크리스트의 체크리스트명을 클릭합니다.
5. '복사'를 클릭한 후 팝업 창에서 '확인'을 클릭합니다.
   - 복사된 체크리스트와 동일한 내용이 반영된 체크리스트 등록 화면으로 이동합니다.
6. [체크리스트 추가](#i-2-1)를 참고하여 필요한 내용을 추가한 후 '저장'을 클릭합니다.
   - 복사한 내용으로 체크리스트가 추가됩니다.

- 삭제된 서약서, 결재 서식은 복사되지 않습니다.
- 업무 담당자에 퇴사한 사원이 지정된 경우, 업무 담당자가 복사되지 않습니다.

## 체크리스트 삭제

체크리스트를 삭제하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 체크리스트 목록에서 삭제할 체크리스트의 체크리스트명을 클릭합니다.
5. '삭제'를 클릭합니다.
6. 체크리스트 삭제 팝업 창에서 '예'를 클릭합니다.
   - 체크리스트가 삭제됩니다.

체크리스트가 그룹에 포함된 경우, 바로 삭제할 수 없으며 해당 그룹에서 제외한 후 삭제할 수 있습니다.

## 체크리스트 목록 다운로드

체크리스트 목록을 다운로드하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 체크리스트 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 체크리스트가 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 체크리스트 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Download` 폴더에 `[YYYY-MM-DD]checkList_set.xlsx` 파일명으로 저장됩니다.

# 체크리스트 그룹 관리

업무 목적 또는 수행 대상에 따라 체크리스트를 그룹화하여 관리할 수 있습니다.

체크리스트 그룹 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 체크리스트 그룹을 구분, 그룹명에 따라 검색 |
| ② 추가 | 새로운 체크리스트 그룹 추가([체크리스트 그룹 추가](#i-4-1) 참고) |
| ③ 삭제 | 선택한 체크리스트 그룹 삭제([체크리스트 그룹 삭제](#i-4-2) 참고) |
| ④ 다운로드 | 체크리스트 그룹 목록을 엑셀 파일로 다운로드([체크리스트 그룹 목록 다운로드](#i-4-6) 참고) |
| ⑤ 체크리스트 그룹 목록 | 추가된 체크리스트 그룹 표시  - 그룹명을 클릭하여 체크리스트 그룹 확인, 수정, 복사([체크리스트 그룹 상세 내용 확인](#i-4-3), [체크리스트 그룹 수정](#i-4-4), [체크리스트 그룹 복사](#i-4-5) 참고) |

## 체크리스트 그룹 추가

체크리스트 그룹을 추가하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '추가'를 클릭합니다.
5. 체크리스트 정보를 입력합니다.
   - 자동 할당 여부를 '사용'으로 선택하면 할당 대상 부서 항목이 표시됩니다.
   - 할당 대상 부서를 선택합니다. '하위 조직 포함'을 선택하면 할당 대상에 할당 대상 부서의 하위 조직을 포함할 수 있습니다.
   - 입사일이 없는 부서 사원은 자동 할당에서 제외됩니다.
   - 신규 등록 사원은 계정 활성화 날짜('바로 활성화' 포함)에 자동 할당합니다.

   발령 사용 여부를 '사용'으로 설정하고, 구분이 '입사'인 경우에만 자동 할당 여부를 '사용'으로 선택할 수 있습니다.
6. '추가'를 클릭하여 체크리스트를 추가합니다.
   - 체크리스트 선택 팝업 창에서 추가할 체크리스트를 클릭하여 선택한 후 '확인'을 클릭합니다.
   - 카테고리명, 체크리스트명을 검색하여 체크리스트를 조회할 수 있습니다.
7. 선결 체크리스트를 지정합니다.
   - 체크리스트를 선택한 후 '추가'를 클릭하여 해당 체크리스트보다 먼저 수행해야 하는 체크리스트를 선택할 수 있습니다.
   - 동일한 체크리스트를 중복으로 선택할 수 없으며, 연속된 선결 조건으로 순환 고리가 형성되는 체크리스트는 선택할 수 없습니다.
8. 메일 안내 문구를 입력합니다.
   - 작성된 문구가 체크리스트 할당 안내 메일의 제목으로 발송됩니다.
   - 입력하지 않을 경우, 기본 제목으로 메일이 발송됩니다.
9. '저장'을 클릭합니다.
   - 체크리스트 그룹이 추가됩니다.

# 카테고리 관리

입사와 퇴사 체크리스트의 카테고리를 추가하거나 수정 또는 삭제할 수 있습니다. 또한 카테고리의 순서를 변경하거나 카테고리 내 체크리스트를 관리할 수 있습니다.

## 카테고리 추가

카테고리를 추가하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '추가'를 클릭합니다.
5. 추가된 행에 카테고리명을 입력한 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_2.png)또는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_1.png)을 클릭하여 카테고리 순서를 지정합니다.
6. '저장'을 클릭합니다.

## **카테고리 수정**

카테고리를 수정하는 방법은 다음과 같습니다.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 수정하려는 내용을 지정합니다.
   - 카테고리명을 변경하려면 입력된 카테고리명을 삭제한 후 변경할 카테고리명을 입력합니다.
   - 카테고리의 순서를 변경하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_2.png)또는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_1.png)을 클릭합니다.
   - 카테고리 내의 체크리스트 순서를 변경하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_detail.png)을 클릭하고 '순서 변경' 탭에서 순서를 변경한 후 '저장'을 클릭합니다.
   - 카테고리 내의 체크리스트를 다른 카테고리로 이동하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_detail.png)을 클릭하고 '카테고리 변경' 탭에서 체크리스트를 이동할 카테고리를 선택한 후 '저장'을 클릭합니다.
5. '저장'을 클릭합니다.
   - 변경한 내용이 저장됩니다.

## 카테고리 삭제

카테고리를 삭제하는 방법은 다음과 같습니다.

등록된 체크리스트가 있는 카테고리는 삭제할 수 없습니다. [체크리스트 삭제](#i-3-3)를 참고하여 체크리스트를 삭제한 후 카테고리를 삭제해 주십시오.

1. PC웹 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 삭제하려는 카테고리 항목의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_delete.png)을 클릭합니다.
5. '저장'을 클릭합니다.
   - 카테고리가 삭제됩니다.
