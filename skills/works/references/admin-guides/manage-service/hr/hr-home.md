# 인사 메인

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
인사 메인에서는 조직 개편 반영일, 인사 현황, 증명서 발급 요청 건수, 체크리스트 현황을 확인할 수 있으며, 인사 서비스 이용에 필요한 설정 사항들을 관리할 수 있습니다. 또한, 인사 메뉴 내 기록된 로그 내역을 확인할 수 있습니다.

인사 메인을 사용하려면 인사 서비스에서 화면 왼쪽 상단의 '인사'를 클릭합니다.

인사 메인 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 콘텐츠 영역에 표시할 콘텐츠를 선택  - [Dashboard](/ko/admin-guides/manage-service/hr/hr-home/#i-2): 조직 개편 반영일, 인사 현황, 증명서 발급 요청 건수, 체크리스트 현황 등을 카드 형태로 표시 - [Timeline](#i-3): 최근 한 달 동안 인사 서비스 내에서 기록된 로그를 100건까지 최신 순으로 표시 - [Getting Started](#i-1): 인사 관리에 필요한 설정 사항 관리 링크 표시 |
| ② 콘텐츠 영역 | - 선택된 콘텐츠 탭에 따라 로그 내역 또는 관련 메뉴를 카드 형태로 표시 - 카드 형태로 표시되는 항목 또는 로그 내역을 클릭하면 해당 메뉴로 이동 |

- 접근 권한을 부여 받은 메뉴 및 카드만 콘텐츠 탭 또는 영역에 표시됩니다.
- 기록된 로그는 임의로 삭제할 수 없으며, 100건을 초과할 경우 오래된 로그부터 자동 삭제됩니다.

# Getting Started

인사 서비스에 처음으로 접속하면 인사 서비스 관련 정보 설정을 안내하는 Getting Started 화면이 나타납니다. 각 설정 항목은 닫힌 카드 형태로 표시되며, 카드를 한 번 클릭하면 카드가 열려 기본 반영 사항이 표시되고, 한 번 더 클릭하면 해당 메뉴로 이동합니다.  
인사 서비스를 원활히 사용하기 위해서는 Getting Started 화면의 카드 순서에 따라 모든 항목의 설정을 완료해 주십시오.

| 영역 | 설명 |
| --- | --- |
| ① 조직도 구성 | 조직 개편을 할 수 있는 [조직도 관리](/ko/admin-guides/manage-service/hr/org/#i-1) 메뉴로 이동  - 영상 보기: 클릭하여 설정 안내 영상 확인 |
| ② 구성원 인사 정보 설정 | 회사에 필요한 인사 정보를 설정할 수 있는 [인사 정보 설정](/ko/admin-guides/manage-service/hr/settings/#i-2) 메뉴로 이동 |
| ③ 증명서 관리 | 발급 신청이 가능한 증명서를 추가하고 관리할 수 있는 [증명서 관리](/ko/admin-guides/manage-service/hr/certificate/#i-2) 메뉴로 이동 |
| ④ 업무용 계정 추가 | 업무용 계정을 추가할 수 있는 [업무용 계정](/ko/admin-guides/manage-service/hr/business-account/) 메뉴로 이동 |
| ⑤ 근무지 정보 설정 | 아르바이트 사원 관리를 위한 근무지를 추가하고 관리할 수 있는 [근무지 관리](/ko/admin-guides/manage-service/hr/parttime/#i-1) 메뉴로 이동 |
| ⑥ 아르바이트 사원 추가 | 아르바이트 사원을 추가할 수 있는 [아르바이트 사원 관리](/ko/admin-guides/manage-service/hr/parttime/#i-2) 메뉴로 이동 |
| ⑦ 체크리스트 카테고리 관리 | 체크리스트의 카테고리를 설정할 수 있는 [카테고리 관리](/ko/admin-guides/manage-service/hr/check-list/#i-5) 메뉴로 이동 |
| ⑧ 체크리스트 그룹 관리 | 목적에 따라 체크리스트를 관리할 수 있는 [체크리스트 그룹 관리](/ko/admin-guides/manage-service/hr/check-list/#i-4) 메뉴로 이동 |
| ⑨ 체크리스트 관리 | 입·퇴사자별 체크리스트를 추가하고 관리할 수 있는 [체크리스트 관리](/ko/admin-guides/manage-service/hr/check-list/#i-2) 메뉴로 이동 |
| ⑩ 데이터 이관 | 경력증명서 발급에 필요한 퇴직자의 인사 정보를 업로드하여 관리할 수 있는 [데이터 불러오기](/ko/admin-guides/manage-service/hr/settings/#i-8) 메뉴로 이동 |
| ⑪ 환경 설정 | 발령과 업무용 계정 사용 여부, 아르바이트 사용 여부 및 급여 사용 여부를 설정할 수 있는 [설정](/ko/admin-guides/manage-service/hr/settings/) 메뉴로 이동 |

# Dashboard

Getting Started 화면의 모든 설정을 완료한 후에는, 인사 서비스에 접속하면 Dashboard 화면이 나타납니다. Dashboard 화면에는 인사 서비스 관련 요약 정보가 카드 형태로 표시됩니다.

Dashboard 화면에 표시되는 카드는 다음과 같습니다.

| 영역 | 설명 |
| --- | --- |
| ① 인사 정보 미입력 구성원 | - 인사 정보를 입력하지 않은 구성원 수 표시 - 클릭하면 [사원 관리](/ko/admin-guides/manage-service/hr/member/#i-1) 메뉴로 이동 |
| ② 예정된 조직 개편 반영일 | - 예약된 조직 개편의 반영일 표시 - 클릭하면 [조직도 관리](/ko/admin-guides/manage-service/hr/org/#i-1) 메뉴로 이동 |
| ③ 이달의 신규 채용 | - 해당 달에 입사한 사원 인원수 표시 - 클릭하면 [사원 관리](/ko/admin-guides/manage-service/hr/member/#i-1) 메뉴로 이동하며, 해당 달에 입사한 사원 목록 표시 |
| ④ 복직 예정자 | - 복직 예정 건수 표시 - 클릭하면 복직 예정자 팝업창이 나타나며, 복직 예정자 목록 표시 |
| ⑤ 예정된 인사 발령 | - 예정된 인사 발령 건수 표시 - 클릭하면 [인사 발령](/ko/admin-guides/manage-service/hr/member/#i-6) 메뉴로 이동하며, 경영지원 접속일 기준으로 3개월 이전의 인사 발령 목록 표시 |
| ⑥ 증명서 발급 요청 | - 당일 신청한 증명서 발급 신청 건수 - 클릭하면 [증명서 발급 내역](/ko/admin-guides/manage-service/hr/certificate/#i-1) 메뉴로 이동 |
| ⑦ 아르바이트 계약 | - 아르바이트 사원 계약 현황 표시 - 클릭하면 [아르바이트 사원 관리](/ko/admin-guides/manage-service/hr/parttime/#i-2) 메뉴로 이동 |
| ⑧ 아르바이트 급여 | - 정산 확정된 아르바이트 총급여 및 급여 지급일 표시 - 클릭하면 [급여명세서 관리](/ko/admin-guides/manage-service/hr/parttime/#i-3) 메뉴로 이동 |
| ⑨ 업무용 계정 사용 상태 | - 업무용 계정 등록 현황 표시 - 클릭하면 [업무용 계정](/ko/admin-guides/manage-service/hr/business-account/) 메뉴로 이동 |
| ⑩ 체크리스트 현황 | - 체크리스트 현황 표시 - 클릭하면 [체크리스트 현황](/ko/admin-guides/manage-service/hr/check-list/#i-1) 메뉴로 이동 |
| ⑪ 월별 조직원 변동 추이 | - 월별 조직원 인원의 변동 그래프 표시 - 월별 조직원 변동 추이 카드명 클릭 시 [조직도 관리](/ko/admin-guides/manage-service/hr/org/#i-1) 메뉴로 이동 |

# Timeline

인사 서비스의 최근 한 달간 사용 로그가 100건까지 최신 순으로 표시됩니다. 각 항목의 링크를 클릭하면 관련 메뉴로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
