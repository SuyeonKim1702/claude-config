# 증명서

이 기능은 웍스 결재 상품 신청 후 사용 가능합니다.

  
증명서 발급을 신청하고 발급할 수 있으며, 발급할 수 있는 증명서 양식을 추가하고 관리할 수 있습니다.  
기본으로 한글/영문의 재직증명서와 경력증명서를 제공하고 있습니다.

# 증명서 발급/내역

증명서 발급을 신청할 수 있고, 발급 신청된 증명서를 발급할 수 있습니다. 또한, 조건에 따라 발급 신청 내역을 검색하여 조회할 수 있으며, 증명서 발급 신청 목록을 엑셀 파일로 다운로드할 수 있습니다.

증명서 발급/내역 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 증명서 발급 신청 내역을 신청일, 발급 번호, 사원명에 따라 검색 |
| ② 추가 | 새로운 증명서 발급 신청([증명서 발급 신청](#i-1-1) 참고) |
| ③ 다운로드 | 증명서 발급 신청 목록을 엑셀 파일로 다운로드([증명서 발급 신청 목록 다운로드](#i-1-3) 참고) |
| ④ 발급 목록 | 증명서 발급 신청 목록  - 상태:   - 요청 중: 증명서 발급을 신청하였으나 발급 전인 상태   - 완료: 발급이 완료된 상태   - 취소: 증명서 발급 신청이 반려된 상태 - 문서 번호: 클릭하여 발급 문서 조회. 관리자가 발급 시 문서 번호 표시되지 않음 |
| ⑤ 발급 | 증명서 발급 상태에 따라 버튼 표시  - 발급: 증명서 발급([증명서 발급](#i-1-2) 참고) - 재발급: 이미 발급한 증명서를 다시 발급 - 직접 발급: 신청자가 직접 증명서를 발급한 경우 |

## 증명서 발급 신청

증명서 발급을 신청하는 방법은 다음과 같습니다.

1. PC 웹 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '추가'를 클릭합니다.
5. 신청자를 검색하여 선택합니다.
6. 신청할 증명서를 선택합니다.
   - 신청 가능한 증명서를 추가하려면 [증명서 추가](#i-2-1)를 참고해 주십시오.
7. 발급 매수를 입력합니다.
8. 관리자가 설정한 신청자 입력 항목이 있는 경우 입력합니다.
9. 발급 용도를 입력합니다.
10. '발급'을 클릭합니다.
    - 증명서 발급이 요청됩니다. ([증명서 발급](#i-1-2) 참고)

- 경력증명서는 퇴직 사원용으로, 관리자만 신청할 수 있습니다.
- 증명서 관리자 권한이 없는 사용자가 증명서 발급을 신청하려면 [증명서 신청 서식 작성](/ko/use-guides/approval/pc-web/write-approval-document/#i-2-4)을 참고해 주십시오.

## 증명서 발급

증명서를 발급하려면 발급할 증명서의 '발급'을 클릭합니다.

- 발급된 증명서가 PDF 파일로 열립니다. PDF 파일을 인쇄한 후 직인을 찍어 신청자에게 전달해 주십시오.
- 발급된 증명서의 발급 항목에는 '재발급'이 표시됩니다.

## 증명서 발급 신청 목록 다운로드

증명서 발급 신청 목록을 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 증명서 발급 신청 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 증명서 발급 신청 내역이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 증명서 발급 신청 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Downloads` 폴더에 `[YYYY-MM-DD]certificate_history.xlsx` 파일명으로 저장됩니다.

# 증명서 관리

발급 신청할 수 있는 증명서 양식을 추가하고 관리할 수 있습니다.

증명서 관리 메뉴는 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 추가 | 새로운 증명서를 추가([증명서 추가](#i-2-1) 참고) |
| ② 증명서 목록 | 기본 제공되는 증명서 및 추가된 증명서 정보 표시 |
| ③ 순서 변경 | 클릭하여 증명서 순서 변경([증명서 순서 변경](#i-2-2) 참고) |
| ④ 수정 | 클릭하여 증명서 수정([증명서 수정](#i-2-3) 참고) |
| ⑤ 삭제 | 클릭하여 증명서 삭제([증명서 삭제](#i-2-4) 참고) |

## 증명서 추가

증명서를 추가하는 방법은 다음과 같습니다.

1. PC 웹 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '추가'를 클릭합니다.
5. 증명서 기본 정보를 설정합니다.
   - 발급 방법을 원본 신청(직인 미날인)으로 선택한 경우, 관리자가 증명서를 출력하여 회사 직인을 날인한 후 신청자에게 직접 전달해야 합니다.
6. 증명서 서식을 구성합니다.
   - 증명서 제목을 입력합니다.
   - 증명서에 표시할 항목을 내용에서 클릭하여 선택합니다.
     - 인적 정보: 신청자 이름은 필수 표시 항목입니다.
     - 회사 정보: 표시할 회사 정보를 선택합니다.
     - 기타 정보: 직접 입력한 내용을 증명서에 표시하려면 기타 정보의 '추가'를 클릭한 후 내용을 입력합니다.
       - 신청자 입력을 선택하면 증명서 발급 신청 시 신청자가 입력한 내용이 증명서에 표시됩니다. 선택하지 않으면 관리자가 설정한 항목명과 내용이 증명서에 표시됩니다.
     - 발급 용도: 발급 용도를 증명서에 표시하려면 증명서에 표시를 선택합니다.
     - 발급처: 발급처를 선택합니다.
       - '회사'가 기본으로 선택됩니다.
       - '사업장'을 선택할 경우, 신청인이 소속된 사업장 정보가 발급처에 표시됩니다.
     - 대표 직함: 회사 대표의 직함을 입력하면 증명서 하단에 표시됩니다.
     - 회사 로고: 회사 로고 노출 여부를 선택합니다.
       - Admin에서 로고 이미지를 등록한 경우에만 '회사 로고' 항목이 표시됩니다.
     - 증명서 양식 스타일을 선택합니다. '미리 보기'를 클릭하면 추가하는 증명서의 양식을 확인할 수 있습니다.
   - '확인'을 클릭합니다.
7. '저장'을 클릭합니다.
   - 추가한 증명서가 저장됩니다.

## 증명서 순서 변경

증명서 순서를 변경하는 방법은 다음과 같습니다.

1. PC 웹 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 순서 변경 항목의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_2.png) 또는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_change_order_1.png)을 클릭하여 순서를 변경합니다
5. '저장'을 클릭합니다.
   - 변경한 증명서 순서가 저장됩니다.

## 증명서 수정

증명서를 수정하는 방법은 다음과 같습니다.

1. PC 웹 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 수정할 증명서의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_edit.png)을 클릭합니다.
5. 수정 내용을 적용한 후 '확인'을 클릭합니다.
   - '미리 보기'를 클릭하면 수정된 증명서를 확인할 수 있습니다.
6. '저장'을 클릭합니다.
   - 수정된 내용이 저장됩니다.

증명서를 삭제하는 방법은 다음과 같습니다.

1. PC 웹 오른쪽 상단에서 '관리자 메뉴'를 클릭 후, 팝업에서 '인사'를 클릭합니다.

5. '저장'을 클릭합니다.
   - 증명서가 삭제됩니다.
