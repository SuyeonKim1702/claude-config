# 템플릿

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

메시지, 노트, 게시판에서 사용하는 [템플릿](/ko/use-guides/general/template/create/new-template/)의 사용 여부, 전사 공용 템플릿 생성 권한, 이미지 보관 기간, 이미지 파일 용량 제한을 설정합니다.

# 템플릿 사용

메시지, 노트, 게시판에서 사용할 수 있는 템플릿 사용 여부를 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '템플릿'을 눌러 '템플릿' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '사용 여부' 항목에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 사용 여부를 설정합니다.
   - 사용: 메시지, 노트, 게시판에서 템플릿을 사용할 수 있습니다.
   - 사용 안 함 : 메시지, 노트, 게시판에서 템플릿을 사용할 수 없습니다.
3. '저장'을 누릅니다.

# 전사 공용 템플릿 생성 권한

메시지, 노트, 게시판에서 전사 공용 템플릿을 생성할 수 있는 권한을 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '템플릿'을 눌러 '템플릿' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '전사 공용 템플릿 생성 권한' 항목에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 생성 권한을 설정합니다.
   - 구성원이 자유 생성 : 구성원 모두가 자유롭게 전사 공용 템플릿을 생성할 수 있습니다.
   - 관리자만 생성 : 관리자만 전사 공용 템플릿을 생성할 수 있습니다.
3. '저장'을 누릅니다.

# **이미지 보관 기간**

질문에 첨부한 이미지의 보관 기간을 설정합니다.

1. Admin 왼쪽 메뉴에서'서비스'를 선택해 메뉴를 펼친 후 '템플릿'을 눌러 '템플릿' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '이미지 보관 기간' 항목에서 아이콘을 눌러, 보관 기간을 설정합니다.
   - 보관 기간을 1개월, 3개월, 6개월, 1년, 2년, 3년 중 선택합니다.
3. '저장'을 클릭합니다.

# 이미지 파일 1개 최대 첨부 용량

질문에 첨부하는 이미지 파일 1개당 최대 용량을 제한할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '템플릿'을 눌러 '템플릿' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '이미지 파일 1개 최대 첨부 용량' 항목에서 아이콘을 눌러, 용량 제한을 설정합니다.
   - 용량 제한을 2MB, 5MB, 10MB, 15MB 중 선택합니다.
3. '저장'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
