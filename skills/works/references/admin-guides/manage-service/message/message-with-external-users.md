# 외부 사용자와 대화 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
구성원이 외부 대화 상대([LINE 사용자](/ko/use-guides/message/message-with-external-user/message-line-user/), [다른 회사의 네이버웍스 사용자](/ko/use-guides/message/message-with-external-user/message-with-external-line-works-user/))와의 메시지를 주고 받을 수 있도록 관련 기능을 설정할 수 있습니다.

# LINE 사용자와 대화

사내 구성원이 네이버웍스로 외부 LINE 사용자와 메시지를 주고 받을 수 있도록 설정합니다.

실제로 구성원들이 외부 LINE 사용자와 메시지를 주고 받기 위해서는 관리자에 의해 권한을 부여받고, [LINE 사용자가 권한을 부여받은 구성원들을 친구로 추가](/ko/use-guides/contacts/external-contacts/line-user/#i)해야 합니다.

## 사용 여부 설정

사내 구성원이 LINE 사용자와 메시지를 주고 받도록 허용할지 설정합니다.

LINE 사용자와 대화 기능은 네이버웍스 가입 직후 ‘허용 안 함’ 상태로 제공되며, 가입 후 7일이 지나야 ‘허용’으로 변경해 사용할 수 있습니다. 가입 후 7일 동안은 상품을 업그레이드 또는 변경하거나, 신규 상품을 추가하더라도 ‘허용’으로 변경할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 여부' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택한 후 사용 여부를 설정합니다.
4. 설정 변경 후 상단의 '저장'을 선택하여 설정값을 저장합니다.  
   '허용' 시 관리자가 허용한 구성원에 한해 네이버웍스 메시지 환경에서 LINE 사용자와 메시지를 주고받을 수 있습니다. 구성원들에 대한 사용 권한 설정은 하단의 '사용 권한' 항목에서 설정합니다.

'허용'에서 '허용 안 함'으로 변경하면 참여 구성원의 메시지방 목록에서 기존에 만들어진 LINE 사용자와의 메시지방이 삭제되고, 대화 중인 LINE 사용자에게 구성원은 탈퇴한 사용자로 표시됩니다. 기존의 LINE 사용자와의 대화 내역은 Admin의 [감사/로그](/ko/admin-guides/audit/message/)에서 검색 및 조회할 수 있습니다.

1. '사용 여부' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택한 후 사용 여부를 설정합니다.

## 사용 권한

LINE 사용자와 대화할 수 있는 구성원의 권한을 설정합니다.

### 사용 권한 설정

LINE 사용자와 대화할 수 있는 권한을 모든 구성원에게 부여할지, 일부 구성원에게만 부여할지 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 권한' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택한 후 '모든 구성원에게 부여' 또는 '구성원 별 부여' 중 선택합니다. 각 설정별 상세 설명은 아래와 같습니다.  

   |  |  |
   | --- | --- |
   | 설정 | 설명 |
   | 모든 구성원에게 부여 | 관리자가 별도로 권한 관리를 하지 않아도 자동으로 모든 구성원에게 LINE 사용자와 대화할 수 있는 권한이 부여됩니다. |
   | 구성원 별 부여 | 관리자가 지정한 일부 구성원에게만 LINE 사용자와 대화할 수 있는 권한을 부여합니다. 이때, 신규 구성원에게 LINE 사용자와 대화할 수 있는 권한을 자동으로 부여하려면 '신규 구성원에게 연동 권한 자동 부여' 설정을 켭니다. |
4. 설정 변경 후 상단의 '저장'을 눌러 설정값을 저장합니다.

### 연동 사용자 추가

'사용 권한' 설정값을 '구성원 별 부여'로 설정한 경우, 구성원별로 LINE 사용자와 대화할 수 있는 권한을 부여할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 권한' 항목에서 '관리' 버튼을 눌러 '연동 사용자 관리' 페이지로 이동합니다.
4. 연동 사용자 관리 페이지에서 'LINE' 탭을 선택합니다.
5. '추가'를 선택하여 연동할 구성원을 선택한 후 하단의 '확인'을 클릭해 연동할 구성원을 추가합니다.

### 연동 사용자 삭제

'사용 권한' 설정값을 '구성원 별 부여'로 설정한 경우, 구성원별로 LINE 사용자와 대화할 수 있는 권한을 삭제할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 권한' 항목에서 '관리' 버튼을 눌러 '연동 사용자 관리' 페이지로 이동합니다.
4. 연동 사용자 관리 페이지에서 'LINE' 탭을 선택합니다.
5. 삭제할 구성원을 선택한 후 '삭제'를 누릅니다.

### 연동 사용자 URL 재설정

연동 사용자의 외부 대화 상대 추가 URL을 새 URL로 일괄 재설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 누릅니다.
3. '사용 권한' 항목에서 '관리' 버튼을 누릅니다.
4. 'LINE' 탭을 누릅니다.
5. URL을 재설정할 구성원을 선택한 후 'URL 재설정' 버튼을 누르거나, 외부 대화 상대 추가 URL 하단의 '재설정' 버튼을 누릅니다.

## 인사 메시지 자동 발송

네이버웍스 사용자를 친구로 추가한 LINE 친구에게 자동으로 인사 메시지를 전송하도록 설정합니다. 영업 정보 또는 인사 메시지를 직접 편집할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '인사 메시지 자동 발송' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 '인사 메시지 자동 발송' 사용 여부를 설정합니다.
4. 설정 변경 후 상단의 '저장'을 눌러 설정값을 저장합니다.  
   인사 메시지를 수정하고 싶은 경우 인사 메시지 영역 하단의 '수정' 버튼을 클릭하면 '인사 메시지 수정' 레이어가 실행됩니다. 내용을 수정한 후 '저장'을 선택하여 변경된 내용을 저장합니다.

## 신규 구성원에게 연동 권한 자동 부여

신규 추가된 구성원에게 연동 권한을 자동으로 부여할지 설정합니다.

단, 이 설정은 'LINE 사용자와 대화'의 ['사용 권한'이 '구성원 별 부여'로 설정](/ko/admin-guides/manage-service/message/message-with-external-users/#line-user-management)되어 있을 경우에만 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 권한' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 '신규 구성원에게 연동 권한 자동 부여' 설정 여부를 설정합니다.  
   ON으로 설정 시 신규 추가된 구성원에게 연동 권한이 자동으로 부여되고, OFF로 설정 시 연동 권한이 자동으로 부여되지 않습니다.
4. 설정 변경 후 상단의 '저장'을 눌러 설정값을 저장합니다.

## 파일 전송 · 첨부 제한

LINE 연동 시 LINE 사용자가 참여한 메시지방에 파일 공유하는 것을 허용할지 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '파일 전송 · 첨부' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 파일 전송 및 첨부 제한 여부를 설정합니다.  
   'ON'을 선택시 LINE 사용자가 참여한 메시지방의 메시지 입력창에서 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-act_com_add-1.png) 아이콘을 탭하거나 클릭했을때 카메라, 연락처, 드라이브(폴더)아이콘이 비활성화되어 카메라로 촬영해서 보내기, [드라이브(폴더)에 있는 파일 보내기](/ko/use-guides/drive/send-file-or-link/send-as-file/message/)를 할 수 없습니다.
4. 상단의 '저장'을 눌러 설정값을 저장합니다.

# 다른 회사의 네이버웍스 사용자와 대화

사내 구성원이 네이버웍스를 사용하는 다른 회사의 구성원과 메시지를 주고받을 수 있도록 설정합니다.

실제로 구성원이 다른 회사의 네이버웍스 사용자와 메시지를 주고 받기 위해서는 관리자에 의해 권한을 부여받고, 권한을 부여받은 구성원이 [다른 회사의 네이버웍스 사용자를 추가](/ko/use-guides/contacts/external-contacts/external-line-works-user/)해야 합니다.

## 사용 여부 설정

사내 구성원이 다른 회사의 네이버웍스 사용자와 메시지를 주고 받도록 허용할지 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 여부' 항목의 오른쪽에서 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택한 후 사용 여부를 설정합니다.
4. 설정 변경 후 상단의 '저장'을 선택하여 설정값을 저장합니다.  
   '허용' 시 관리자가 허용한 구성원에 한해 네이버웍스 메시지 환경에서 네이버웍스를 사용하는 다른 회사의 사용자와 메시지를 주고받을 수 있습니다. 구성원에 대한 사용 권한 설정은 하단의 '사용 권한' 항목에서 설정합니다.

'허용'에서 '허용 안 함'으로 변경하면 참여 구성원의 메시지방 목록에서 기존에 만들어진 다른 회사의 네이버웍스 사용자와의 메시지방이 삭제되고, 대화 중인 다른 회사의 네이버웍스 사용자에게 구성원은 탈퇴한 사용자로 표시됩니다. 기존의 다른 회사의 네이버웍스 사용자와의 대화 내역은 Admin의 [감사/로그](/ko/admin-guides/audit/)에서 검색 및 조회할 수 있습니다.

## 사용 권한

다른 회사의 네이버웍스 사용자와 대화할 수 있는 구성원의 권한을 설정합니다.

   | 설정 | 설명 |
   | 모든 구성원에게 부여 | 관리자가 별도로 권한 관리를 하지 않아도 자동으로 모든 구성원에게 다른 회사의 네이버웍스 사용자와 대화할 수 있는 권한이 부여됩니다. |
   | 구성원 별 부여 | 관리자가 지정한 일부 구성원에게만 다른 회사의 네이버웍스 사용자와 대화할 수 있는 권한을 부여합니다. 이때, 신규 구성원에게 다른 회사의 네이버웍스 사용자와 대화할 수 있는 권한을 자동으로 부여하려면 '신규 구성원에게 연동 권한 자동 부여' 설정을 켭니다. |
4. 설정 변경 후 상단의 '저장'을 눌러 설정값을 저장합니다.

### 연동 사용자 추가

'사용 권한' 설정값을 '모든 구성원에게 부여'로 설정한 경우, 구성원별로 다른 회사의 네이버웍스 사용자와 대화할 수 있는 권한을 부여할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 권한' 항목에서 '관리' 버튼을 눌러 '연동 사용자 관리' 페이지로 이동합니다.
4. 연동 사용자 관리 페이지에서 '다른 회사의 네이버웍스' 탭을 선택합니다.
5. '추가'를 선택하여 연동할 구성원을 선택한 후 하단의 '확인'을 선택해 연동할 구성원을 추가합니다.

### 연동 사용자 삭제

'사용 권한' 설정값을 '구성원 별 부여'로 설정한 경우, 구성원별로 다른 회사의 네이버웍스 사용자와 대화할 수 있는 권한을 삭제할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 권한' 항목에서 '관리' 버튼을 눌러 '연동 사용자 관리' 페이지로 이동합니다.
4. 연동 사용자 관리 페이지에서 '다른 회사의 네이버웍스' 탭을 선택합니다.
5. 삭제할 구성원을 선택한 후 '삭제'를 누릅니다.

### 연동 사용자 URL 재설정

연동 사용자의 외부 대화 상대 추가 URL을 새 URL로 일괄 재설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 누릅니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 누릅니다.
3. '사용 권한' 항목에서 '관리' 버튼을 누릅니다.
4. '네이버웍스' 탭을 누릅니다.
5. URL을 재설정할 구성원을 선택한 후 'URL 재설정' 버튼을 누르거나, 외부 대화 상대 추가 URL 하단의 '재설정' 버튼을 누릅니다.

## 신규 구성원에게 연동 권한 자동 부여

신규 추가된 구성원에게 연동 권한을 자동으로 부여할지 설정합니다.

단, 이 설정은 '다른 회사의 네이버웍스 사용자와 대화'의 ['사용 권한'이 '구성원 별 부여'로 설정](/ko/admin-guides/manage-service/message/message-with-external-users/#naverworks-user-management)되어 있을 경우에만 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '사용 권한' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택한 후 '신규 구성원에게 연동 권한 자동 부여' 설정 여부를 설정합니다. ON으로 설정 시 신규 추가된 구성원에게 연동 권한이 자동으로 부여되고, OFF로 설정 시 연동 권한이 자동으로 부여되지 않습니다.
4. 설정 변경 후 상단의 '저장'을 눌러 설정값을 저장합니다.

## 파일 전송 · 첨부 제한

다른 회사의 네이버웍스 사용자와 대화 시 다른 회사의 네이버웍스 사용자가 참여한 메시지방에 파일 공유하는 것을 허용할지 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 사용자와 대화'를 선택합니다.
3. '파일 전송 · 첨부 제한' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택한 후 '파일 전송 · 첨부 제한' 여부를 설정합니다.  
   다른 회사의 네이버웍스 사용자와의 메시지방에 파일을 올릴 수 있게 하려면 'OFF'로, 올릴 수 없게 하려면 'ON'을 선택합니다. 'ON'을 선택하면 다음과 같은 제약사항이 발생합니다.
   - 다른 회사의 네이버웍스 사용자가 참여한 메시지방: 메시지 입력창에서 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-act_com_add-1.png) 아이콘을 탭하거나 클릭했을때 카메라, 연락처, 드라이브(폴더), 템플릿 아이콘이 비활성화되어 카메라로 촬영해서 보내기, [드라이브(폴더)에 있는 파일 보내기](?page_id=2757), 템플릿 작성 및 보내기를 할 수 없습니다.
   - 외부 그룹: 외부 그룹 내 노트, 일정, 폴더 메뉴에서 파일을 첨부할 수 없습니다. 또한 노트에 템플릿을 첨부하거나, [게시판에 템플릿을 설정](/ko/use-guides/message/note/manage-note/manage-board/)할 수 없습니다.
4. 상단의 '저장'을 눌러 설정값을 저장합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
