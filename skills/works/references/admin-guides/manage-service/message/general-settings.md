# 메시지 일반 설정

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

메시지 서비스에서 사용할 수 있는 기능을 설정합니다.

# 기능 설정

읽은 멤버 확인, 메시지 회수 가능 시간, 음성ㆍ영상통화 및 화면 공유에 관한 설정을 할 수 있습니다.

## 읽은 멤버 확인

구성원이 [읽은 멤버 확인](/ko/use-guides/message/check-message/read-receipt/) 기능을 사용할 수 있게 할지 선택합니다.

'내가 보낸 메시지를 읽은 멤버 확인' 설정을 켜면 메시지를 보낸 구성원이 메시지방에 참여한 구성원 중 누가 메시지를 읽었는지 확인할 수 있습니다.

'다른 사람이 보낸 메시지를 읽은 멤버 확인' 설정을 켜면 메시지방에 참여한 구성원 모두가 다른 사람이 보낸 메시지를 누가 읽었는지 확인할 수 있습니다.

'내가 보낸 메시지를 읽은 멤버 확인' 설정을 켜야 '다른 사람이 보낸 메시지를 읽은 멤버 확인' 설정이 가능합니다. '다른 사람이 보낸 메시지를 읽은 멤버 확인' 설정만 켤 수는 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '기능'의 '읽은 멤버 확인' 항목에서 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. '내가 보낸 메시지를 읽은 멤버 확인'을 켭니다.  
   '내가 보낸 메시지를 읽은 멤버 확인' 설정을 켜면 '다른 사람이 보낸 메시지를 읽은 멤버 확인'도 설정할 수 있습니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

## 메시지 회수 가능 시간

[메시지 회수](/ko/use-guides/message/send-message/delete-recall-message/)가 가능한 시간을 변경할 수 있습니다.

'1시간'과 '24시간' 중 선택할 수 있으며, 설정한 시간 내에 자신이 보낸 메시지를 회수하거나, 조직/그룹 메시지방에서는 조직장 또는 그룹 마스터가 다른 멤버의 메시지를 회수할 수 있습니다.

메시지를 회수하면 모든 메시지방 멤버가 회수한 메시지의 내용을 볼 수 없게 되지만, 회수한 메시지가 있다는 것은 알 수 있습니다.

단, LINE으로 발송된 메시지는 회수할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '기능'의 '메시지 회수 가능 시간' 항목에서 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 메시지 회수 가능 시간을 선택합니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '기능'의 '음성/영상통화 및 화면 공유' 항목에서 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. '음성/영상통화 및 화면 공유'를 켭니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

메시지방 기능(노트, 캘린더, 할 일, 폴더)의 사용 여부를 설정합니다.

그룹, 조직 메시지방을 포함한 모든 메시지방에 일괄 적용됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '기능'의  '메시지방 기능' 항목에서 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 메시지방 노트, 메시지방 일정, 메시지방 할 일, 메시지방 폴더 중 사용하고자 하는 기능을 선택합니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

# 메시지/파일 관리

메시지 및 첨부한 파일에 대해 설정합니다.

설정한 값은 PC웹, PC앱, 모바일앱에 공통으로 적용됩니다.

단, 기간을 현재 설정보다 짧은 기간으로 변경할 경우, 차분의 과거 데이터는 삭제 처리됩니다.

## 메시지 보관/검색 기간

텍스트 메시지 내용을 보관 및 검색할 수 있는 기간을 설정합니다.

설정한 기간 동안 주고 받은 메시지 내용을 확인 및 검색할 수 있습니다.

설정 기간 이전의 메시지는 자동으로 삭제됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지/파일 관리'의 '메시지 내용 보관/검색 기간' 항목에서 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 기간을 설정합니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

## 사진/파일 보관 기간

메시지방에서 공유한 파일이나 사진을 다운로드 할 수 있는 기간을 설정합니다.

설정값을 변경하면 이전에 업로드한 파일이나 사진에도 변경된 설정값이 적용됩니다.

기존에 설정되어 있던 기간보다 짧게 변경할 경우, 줄어든 보관 기간 동안의 파일이 삭제되고 삭제된 파일만큼 사용 가능한 공용용량이 늘어납니다.

삭제된 파일과 사진은 보관 기간을 연장해도 복구할 수 없습니다.

단, '메시지 내용 보관/검색 기간'보다 길게 설정할 경우, '메시지 내용 보관/검색 기간' 이내의 사진/파일만 확인 및 다운로드할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지/파일 관리'의 '사진/파일 보관 기간' 항목에서 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 기간을 설정합니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

## 파일 1개 최대 첨부 용량

메시지방에 첨부할 수 있는 파일 1개의 최대 용량을 설정합니다.

최대 용량을 작게 설정하면 구성원이 의도치 않게 용량이 큰 파일을 첨부해 공용 용량이 빨리 차는 상황을 방지할 수 있습니다.

설정 용량을 초과한 파일은 전송되지 않습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지/파일 관리'의 '파일 1개 최대 첨부 용량' 항목에서 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 파일 1개의 최대 전송 용량을 설정합니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

# 메시지방 노트

메시지방에서 사용할 수 있는 메시지방 노트의 첨부파일 용량과 삭제한 글이 보관되는 휴지통의 보관 기간을 설정합니다.

## 파일 1개 최대 첨부 용량

노트에 첨부할 수 있는 파일 1개의 최대 용량을 설정합니다.

최대 용량을 작게 설정하면 구성원이 무의식적으로 용량이 큰 파일을 첨부해 공용용량이 빨리 차는 상황을 방지할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지방 노트'의 '*파일 1개 최대 첨부 용량*' 항목에서 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 첨부파일 1개의 최대 용량을 설정합니다.
4. 오른쪽 상단의 '저장'을 누릅니다.

## 휴지통 보관 기간

삭제된 후 휴지통에 보관 중인 글의 보관 기간을 설정합니다. 보관 기간이 지난 글은 영구 삭제되어 복구할 수 없습니다.

휴지통에 보관되어 있어도 공용용량에는 포함되므로 용량을 확보하기 위해서는 휴지통에 보관된 글을 영구 삭제해야 합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지방 노트'의 '휴지통 보관 기간' 항목에서 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 보관 기간을 선택합니다. 자동삭제를 원하지 않는다면 '자동 삭제 안 함'을 선택합니다.
4. 오른쪽 상단의 '저장'을 누릅니다.

## 링크 공유 기능

메시지방 폴더의 폴더나 파일을 [링크로 공유](/ko/use-guides/message/group-folder/send-file-or-link/send-as-link/share-link/)하는 것을 허용할지 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지방 폴더'의 '링크 공유 기능' 항목에서 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 링크 공유 기능 사용 여부를 선택합니다.
   - 링크 공유 허용함: 메시지방 폴더의 하위 폴더나 파일을 링크로 공유할 수 있도록 허용할지 조직장 또는 그룹 마스터가 설정할 수 있습니다.
   - 링크 공유 허용하지 않음: 메시지방 폴더의 하위 폴더나 파일을 링크로 공유할 수 없습니다. 설정 시 기존에 생성된 링크도 접근과 이용이 불가능해집니다.
4. 오른쪽 상단의 '저장'을 누릅니다.

## 링크 접근 권한

링크 접근 권한을 설정합니다.

링크 공유를 허용한 후 링크 접근 권한을 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지방 폴더'의 '링크 접근 권한' 항목에서 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 링크 접근 권한을 설정합니다.
   - 사내 구성원: 네이버웍스에 로그인한 사내 구성원만 링크에 접근할 수 있습니다.
   - 모든 사람(비밀번호 설정 가능): 사내 구성원 이외에 링크를 아는 모든 사람이나 비밀번호를 아는 사람이 링크에 접근할 수 있습니다.
   - 지정한 사람만(보안 OTP 인증): OTP(One Time Password) 인증을 해야만 링크에 접근할 수 있습니다.  
     지정한 사람만 접근할 수 있도록 보안을 강화하여 공유할 경우에 사용하며, 의도치 않은 링크 확산을 방지합니다.
4. 오른쪽 상단의 '저장'을 누릅니다.

## 링크 유효 기간

공유된 링크를 사용할 수 있는 기간을 설정합니다.

유효 기간이 만료된 링크는 기간 만료 페이지로 이동되어 이용이 불가능합니다.

링크 공유를 허용한 후 링크 유효 기간을 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지방 폴더'의 '링크 유효 기간' 항목에서 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 링크 유효 기간을 선택합니다.
4. 오른쪽 상단의 '저장'을 누릅니다.

## 휴지통 보관 기간

삭제된 후 휴지통에 보관 중인 파일의 보관 기간을 설정합니다.

보관 기간이 지난 파일은 영구 삭제되어 복구할 수 없습니다.

휴지통에 보관되어 있어도 공용용량에는 포함되므로 용량을 확보하기 위해서는 휴지통에 보관된 파일을 영구 삭제해야 합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '메시지방 폴더'의 '휴지통 보관 기간' 항목의 오른쪽 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 보관 기간을 선택합니다. 최대 5일/15일/30일/50일/자동 삭제 안 함 중 선택할 수 있습니다.
4. 오른쪽 상단의 '저장'을 누릅니다.

## 버전 이력 파일 보관 기간

파일을 덮어썼을 때 생기는 [버전 이력](/ko/use-guides/message/group-folder/upload-download-view/file-version/)별 파일이 보관되는 기간을 설정합니다.

메시지방 폴더의 버전 이력은 공용용량을 차지합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '고급 설정'을 누르고 '관리' 버튼을 누릅니다.
4. 오른쪽 상단의 '수정'을 선택한 후 '버전 저장 기간'을 선택합니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

## 파일 유형

기본 제공하는 파일 종류 외 버전 이력을 남길 파일의 종류를 추가하거나 삭제합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '메시지'를 눌러 '메시지'로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '고급 설정'을 누르고 '관리' 버튼을 누릅니다.
4. 오른쪽 상단의 '수정'을 선택한 후 버전 이력을 저장할 파일의 확장자를 선택합니다. 선택된 확장자는 파란색으로 표시됩니다.  
   확장자를 직접 입력하려면 '추가된 파일' 하단에 직접 파일 확장자를 입력하고 '추가'를 누릅니다. 직접 추가한 확장자는 'x'를 눌러 제거할 수 있습니다.
5. 오른쪽 상단의 '저장'을 누릅니다.
