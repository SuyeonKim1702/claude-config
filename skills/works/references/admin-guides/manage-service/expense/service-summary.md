# 서비스 개요

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
비용 서비스에서는 각종 비용의 계획 및 정산 현황을 확인하고, 연동한 법인 카드의 사용 내역을 확인할 수 있습니다. 또한, 다양한 정산 현황을 보고서 형태로 확인하고 비용이 투명하게 사용되고 있는지 감사할 수 있습니다. 예산을 사용으로 설정하면 비용 정산 신청 시 예산을 조회 및 사용할 수 있습니다.

비용 메뉴를 사용하려면 PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '비용'을 클릭합니다.

- 비용 메뉴는 관리자 권한이 있는 사용자에게만 노출됩니다.
- 비용 설정에서 활성화한 기능에 따라 표시되는 메뉴가 달라집니다. 비용 설정에 대한 자세한 설명은 [비용 설정](/ko/admin-guides/manage-service/expense/settings/#i-1)을 참고해 주십시오.

# 비용 관리자 시나리오

비용 관리자의 이용 시나리오는 다음과 같습니다.

1. 비용 서비스 기본 설정
   - 비용 서비스 사용 환경 설정
   - 카드사 연동 및 카드 관리
   - 비용 항목 관리
   - 이 단계에 대한 사용 가이드 페이지
     - [Getting Started](/ko/admin-guides/manage-service/expense/expense-home/#i-1)
     - [설정](/ko/admin-guides/manage-service/expense/settings/)
     - [카드 관리](/ko/admin-guides/manage-service/expense/settings/#i-3)
2. 비용 신청 현황 확인, 감사 항목 관리
   - 비용 신청 현황, 카드 사용 현황을 확인하고 보고서 검토
   - 감사 항목 설정 및 위반 사항 확인, 소명 요청 및 답변 관리
   - 이 단계에 대한 사용 가이드 페이지
# **자주 하는 질문을 먼저 확인해 보십시오.**

**Q. 오늘 법인 카드를 등록하여 연동을 하였습니다. 카드 사용 내역은 어느 기간까지 확인 가능하나요?**

- 새벽 및 저녁 2회에 걸쳐 스크래핑을 하여 법인 카드를 연동 설정하신 이후의 카드 사용 내역을 제공해 드립니다. 그러나 취소 또는 누락 건에 대한 데이터 정합성을 높이기 위하여 추가로 주말 스크래핑을 수행하여 한 달 전의 데이터를 수집해 제공해 드립니다.

**Q. 회식비를 부서별로 월 50만원만 사용하고 싶습니다. 어떻게 설정해야 하나요?**

- 아래 절차를 참고하여 설정해 주십시오.([예산 배정](/ko/admin-guides/manage-service/budget/budget-allocation/) 참조)
  3. '회식' 비용 항목과 매칭된 예산의 배정 금액을 각 예산에 600만원으로 입력하고 배정 기준을 균등 금액/월로 선택해 주십시오.
  4. 예산 통제 여부를 'Y'로 선택한 후 저장해 주십시오.

**Q. 비용 정산 시 예산이 부족하다는 메시지가 나와 더 이상 진행하지 못할 경우 어떻게 처리해야 하나요?**

- 아래 3가지 방법 중 하나를 선택하여 처리해 주십시오.
  - 예산 과부족 발생 시 예산 금액 또는 월별 배정 금액을 변경(증액 또는 삭감)할 수 있습니다. 자세한 내용은 [예산 배정](/ko/admin-guides/manage-service/budget/budget-allocation/)을 참조해 주십시오.

**Q. 카드 관리의 법인카드 스크래핑은 카드사 홈페이지의 데이터를 스크래핑하나요?**

- 법인카드 스크래핑은 법인카드가 실제로 등록된 법인카드사 홈페이지의 데이터를 스크래핑합니다. 은행 사이트나 개인 카드 홈페이지와 혼동하지 말아주세요.
- 또한 카드사 연동 설정을 위한 인증서, ID/PW는 실제 법인카드 홈페이지에 등록된 정보와 동일해야 합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
