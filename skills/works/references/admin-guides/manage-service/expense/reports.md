# 보고서

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
예산과 비용 계획에 따른 정산 금액을 통합하여 조회하고 기간별 비용 추이를 차트로 확인할 수 있습니다. 또한, 법인 카드의 사용 현황을 확인하고 각 분류 기준에 따른 정산 현황을 파악할 수 있습니다.

# 비용 현황 통합 조회

예산 과목별 정산 금액과 비용 계획 대비 정산 금액을 확인하고, 정산 세부 내역을 파악하며, 감사 항목에 위반되는 사항이 없는지 조회할 수 있습니다.

비용 현황 통합 조회 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 비용 현황을 비용 귀속, 기간 조건에 따라 검색 |
| ② 다운로드 | 비용 현황 목록을 엑셀 파일로 다운로드([비용 통합 현황 목록 다운로드](#i-1-1) 참고) |
| ③ 비용 현황 목록 | 비용 현황을 목록으로 확인  - 각 행의 '귀속'을 클릭하여 조직도와 사원 상세 정보 확인 - 각 행의 '계획'을 클릭하여 비용 계획 문서 확인 - 각 행의 '합계'를 클릭하여 비용 정산 문서 확인 - 각 행의 Self Check의 감사 위반을 클릭하면 '감사 항목 위반 > 비용 정산 신청' 탭으로 이동, 소명 요청/완료를 클릭하면 위반/소명 관리 메뉴로 이동하여 세부 사항 확인 ([위반/소명 관리](/ko/admin-guides/manage-service/expense/manage-check/#i-2) 참고) - 예산: 해당 기간의 예산 배정 금액 및 이전 기간과의 대비 값 표시(<예시> 월 단위 조회 시 전월 대비 증감액) - 계획: 비용 계획 신청 금액 및 해당 기간에 대한 대비 값 표시(<예시> 월 단위 조회 시 전월 대비 증감액) - 지급: 회계에서 지급 처리된 금액 |

- 예산 금액이 비어있는 경우는 배정된 예산이 없음을 의미합니다.
- 기간을 직접 입력해 조회 시 과거 기간 비교가 어려워 대비 값(양/음수로 표기)은 표시되지 않습니다.

## 비용 통합 현황 목록 다운로드

비용 통합 현황 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '비용'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 비용 통합 현황 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 비용 통합 현황이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 비용 통합 현황 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Download` 폴더에 `TotalStcList.xlsx` 파일명으로 저장됩니다.

# 기간별 비용 추이

차트를 통해 예산, 비용 계획, 정산 금액 추이를 월별로 확인하고, 비용 항목에 따른 정산 추이를 확인할 수 있습니다.

기간별 비용 추이 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 비용 추이 차트를 비용 귀속, 기간 조건에 따라 검색 |
| ② 연간 비용 금액 | 연간 예산, 비용 계획, 정산 금액을 월별 또는 분기별 차트로 표시  - 월 표시는 회계의 결산 구분 값을 따름, 12월 결산일 경우 1~12월까지 표시 - 예산을 사용하지 않을 경우 예산 그래프 표시 안 됨 - 비용 계획을 사용하지 않을 경우 계획 그래프 표시 안 됨 |
| ③ 비용 항목별 정산 추이 | 월별로 각 비용 항목의 정산 금액 표시 |

# 연동카드 사용 현황

연동 카드 사용 현황을 조건에 따라 검색하여 조회할 수 있으며, 이달의 승인, 정산, 정산 제외, 미정산, 정산 불가 금액을 한눈에 확인할 수 있습니다. 또한, 연동 카드 사용 현황 목록을 엑셀 파일로 다운로드할 수 있습니다.

연동카드 사용 현황 화면은 다음과 같이 구성되어 있습니다. 기간을 연도로 선택한 후 검색하면 현황을 월별 보기 또는 분기별 보기를 선택하여 확인할 수 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 금액 합계 | 현재 월의 승인, 정산, 정산 제외, 미정산, 정산 불가 합계액을 표시 |
| ② 검색창 | 법인 카드 사용 내역을 카드 번호, 기간 조건에 따라 검색  - 기간을 연도로 선택한 후 검색하면 월별 카드 사용 현황을 표시 |
| ③ 다운로드 | 법인 카드 사용 현황 목록을 엑셀 파일로 다운로드([연동카드 사용 현황 목록 다운로드](#i-3-1) 참고) |
| ④ 연동카드 사용 현황 목록 | 월별 또는 분기별 법인 카드 사용 현황을 목록으로 확인  - 각 금액을 클릭하여 정산 문서 확인 - 카드사 이름을 클릭하여 [카드 관리](/ko/admin-guides/manage-service/expense/settings/#i-3) 메뉴로 이동 - '비용 > 설정 > 카드 관리' 메뉴에서 삭제된 카드에 정산 내역이 있을 경우 해당 내역을 포함하되 카드에 삭제 표시 - '비용 > 설정 > 카드 관리' 메뉴에서 미사용 설정된 카드에 정산 내역이 있을 경우 해당 내역을 포함하되 카드에 미사용 표시 |

# 사원별 정산 현황

신청자 또는 참석자별로 정산 현황을 확인할 수 있습니다. 또한 조건에 따라 정산 현황을 검색하여 조회할 수 있으며, 사원별 정산 현황 목록을 엑셀 파일로 다운로드할 수 있습니다.

사원별 정산 현황 화면은 다음과 같이 구성되어 있습니다. 기간을 연도로 선택한 후 검색하면 현황이 월별 보기 또는 분기별 보기를 선택하여 확인할 수 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 사원별 정산 현황을 조회할 기준  - 신청자: 정산 신청자별로 정산 현황 표시 - 참석자: 정산 건의 참석자별로 정산 현황 표시 |
| ② 검색창 | 정산 내역을 사원, 기간 조건에 따라 검색  - 기간을 연도로 선택한 후 검색하면 월별로 정산 현황 표시 |
| ③ 다운로드 | 사원별 정산 목록을 엑셀 파일로 다운로드([사원별 정산 현황 목록 다운로드](#i-6-1) 참고) |
| ④ 사원별 정산 현황 목록 | 선택 기간 내 각 사원의 정산 현황을 목록으로 확인  - 정산 신청자 또는 참석자의 이름을 클릭하여 해당 사원의 상세 정보 확인 - 합계 정산 금액을 클릭하여 비용 정산 문서 확인 - 기간 직접 입력으로 신청자 조회 시 Self Check 열의 감사 항목 위반 여부 및 소명 상태를 클릭하여 세부 사항 확인 ([위반/소명 관리](/ko/admin-guides/manage-service/expense/manage-check/#i-2) 참고) - 월별 정산 금액이 가장 많은 사원에 ⑤ 표시 - 연도별 신청자 조회 시 감사 위반이 있는 월에는  표시 |

# 프로젝트별 코드별 정산 현황

프로젝트 코드를 사용한 비용 정산 내역에 대해 프로젝트 및 상세 비용 항목별 정산 현황을 조회할 수 있습니다. 또한 프로젝트별 정산 현황 목록을 엑셀 파일로 다운로드할 수 있습니다.

프로젝트별 코드별 정산 현황 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 정산 내역을 프로젝트 코드, 기간, 비용 항목 조건에 따라 검색 |
| ② 다운로드 | 프로젝트 코드별 정산 목록을 엑셀 파일로 다운로드([프로젝트 코드별 정산 현황 다운로드](#i-7-1) 참고) |
| ③ 프로젝트 코드별 정산 현황 목록 | 선택 기간 내 각 프로젝트의 정산 현황을 목록으로 확인  - 프로젝트 코드 또는 코드명을 클릭하여 해당 프로젝트 코드의 상세 정보 및 수정 내역 확인 가능 - 건수 또는 합계 정산 금액을 클릭하여 비용 정산 문서 확인 |

## 프로젝트 코드별 정산 현황 다운로드

프로젝트 코드별 정산 현황 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '비용'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 프로젝트, 기간, 비용 항목별 정산 조건을 입력한 후 '검색'을 클릭합니다
   - 입력한 조건에 해당하는 프로젝트 코드별 정산 현황이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 프로젝트 코드별 정산 목록이 엑셀 파일로 다운로드됩니다.  

     다운로드된 파일은 `Download` 폴더에 `StcProjectCode.xlsx` 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
