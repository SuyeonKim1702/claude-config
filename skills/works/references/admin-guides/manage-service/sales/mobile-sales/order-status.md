# 주문서 현황

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
모바일 주문서 조회에서는 주문서 작성 메뉴에서 등록한 주문 현황을 조회할 수 있습니다.  
주문 내역을 품목 단위까지 상세히 조회할 수 있으며, 주문 금액 대비 실제 판매 금액을 비교하여 미판매 금액도 조회 가능합니다.

주문서 현황 메뉴를 사용하려면 모바일 홈의 '관리자 메뉴'에서 '영업'을 탭한 후, 메뉴에서 '주문서 현황'을 탭합니다.

주문서 현황 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 사업장, 주문일, 영업 그룹, 주문 담당자, 거래처, 주문 번호, 견적 번호 조건에 따라 검색 |
| ② 미판매 건만 | 선택하면 미판매건 주문서 현황만 조회 |
| ③ 견적서 미참조 건만 | 선택하면 견적서 미참조 주문서 현황만 조회 |
| ④ 주문서 현황 목록 | 입력한 조건에 따라 주문서 현황 목록 표시 |
| ⑤ 더보기 | 주문서의 상세 내용 확인 |

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
