# 현 재고 조회

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
모바일 현재고 조회에서는 관리 중인 창고의 품목별 현재 재고 수량을 조회할 수 있습니다.

현 재고 조회를 사용하려면 모바일 홈의 '관리자 메뉴'에서 '영업'을 탭한 후, 메뉴에서 '현 재고 조회'를 탭합니다.

현재고 조회 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 사업장, 창고, 품목 구분, 품목 그룹, 품목 조건에 따라 검색 |
| ② 현재고 목록 | 입력한 조건에 따라 현재고 수량 목록 표시 |

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
