# 주문서 작성

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
모바일 주문서 작성에서는 거래처로부터 주문받은 내역을 등록하고 관리할 수 있습니다.

주문서를 작성하는 방법은 다음과 같습니다.

1. 모바일 홈의 '관리자 메뉴'에서 '영업'을 탭한 후, 메뉴에서 '주문서 작성'을 탭합니다.
2. 주문서 작성 화면에서 주문 정보를 입력한 후 '저장'을 탭합니다.
   - 사업장: 사업장을 선택합니다.
   - 주문 번호: 주문 번호는 자동으로 채번됩니다. (채번 규칙은 OR + yyyymmdd + 6자리 시퀀스입니다.)
   - 영업 그룹: 등록되어 있는 영업 그룹을 선택합니다.
   - 주문 담당자: 등록되어 있는 주문 담당자를 선택합니다.
   - 거래처: 등록되어 있는 거래처를 선택합니다.
   - 수금처: 수금처를 선택합니다.
   - 주문일: 주문일을 선택합니다. (기본 값은 현재 날짜입니다.)
   - 부가세 유형: 부가세 유형(과세, 비과세, 영세율)을 선택합니다.
   - 통화: 통화를 선택합니다. 선택한 통화에 따라 통화에 해당되는 품목을 추가할 수 있습니다.
   - 단가 확정 여부: 단가 확정 여부를 선택합니다.
   - 진행 상태: 주문 진행 상태를 확인할 수 있습니다.
     - 신규: 신규로 주문서 내용을 입력하는 상태
     - 저장: 주문서 내용을 입력 후 '저장'을 탭한 상태
     - 출고 가능: '저장'을 탭한 상태에서 '승인'을 탭한 상태
     - 출고 중: 판매 관리에서 판매가 진행 중인 상태
     - 출고 완료: 판매 관리에서 판매가 완료된 상태
   - 적요: 주문에 대한 설명을 입력합니다.
   - 주문 품목: '추가'를 탭해 주문서 품목을 추가합니다. 이미 등록된 품목은 추가할 수 없습니다.
     - 품목명: 주문서에 추가할 품목을 선택합니다.
     - 규격: 품목을 선택하면 선택한 품목의 규격이 자동 입력됩니다.
     - 주문 수량: 주문 수량을 입력합니다.
     - 판매 단위: 품목을 선택하면 선택한 품목의 판매 단위가 자동 입력됩니다.
     - 외화단가: 품목과 주문 수량을 입력하면 입력한 품목의 외화단가가 자동 입력됩니다.
     - 외화판매가: 품목과 주문 수량을 입력하면 입력한 품목의 외화판매가가 자동 입력됩니다.
     - 단가: 품목과 주문 수량을 입력하면 입력한 품목의 단가가 자동 입력됩니다. (외화 단가는 환율에 따라 자동 계산됩니다.)
     - 판매가: 품목의 판매가가 자동 입력됩니다.
     - 할인금액: 할인금액을 입력합니다. (할인 금액은 판매가보다 클 수 없습니다.)
     - 공급가: 품목의 공급가가 자동 입력됩니다.
     - 부가세: 부가세는 공급가의 10% 금액으로 자동 입력됩니다.
     - 주문금액: 공급가 + 부가세 가격이 자동 입력됩니다.
     - 납품일: 주문 품목의 납품일을 입력합니다.
3. '승인'을 탭하여 주문서를 승인합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
