# 견적서 현황

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
모바일 견적서 조회에서는 '견적서 작성' 메뉴에서 등록한 견적 현황을 조회할 수 있습니다.

견적서 현황 메뉴를 사용하려면 모바일 홈의 '관리자 메뉴'에서 '영업'을 탭한 후, 메뉴에서 '견적서 현황'을 탭합니다.

견적서 현황 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 사업장, 견적일, 영업 그룹, 견적 담당자, 거래처, 견적명, 품목, 견적 번호 조건에 따라 검색 |
| ② 견적서 현황 목록 | 입력한 조건에 따라 견적서 현황 목록 표시 |
| ③ 더보기 | 견적서의 상세 내용 확인 |

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
