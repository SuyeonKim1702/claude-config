# 주문서 조회

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
모바일 주문서 조회에서는 거래처로부터 주문받은 내역 중 주문서 작성에 등록된 내용을 조회할 수 있습니다.

주문서 조회 메뉴를 사용하려면 모바일 홈의 '관리자 메뉴'에서 '영업'을 탭한 후, 메뉴에서 '주문서 조회'를 탭합니다.

주문서 조회 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 사업장, 주문일, 주문 담당자, 거래처, 주문 번호, 단가 확정 여부 조건에 따라 검색 |
| ② 주문서 목록 | 입력한 조건에 따라 주문서 목록 표시 |
| ③ 진행 상태 | 주문서 진행 상태 표시 |
| ④ 상세 보기 | 주문 번호 또는 '더보기'를 탭하면 주문서의 상세 내용 확인 및 수정, 상신('주문서 상세 보기' 참조) |

# 주문서 상세 보기

주문서 상세 내용을 확인 및 수정, 상신하는 방법은 다음과 같습니다.

1. 모바일 홈의 '관리자 메뉴'에서 '영업'을 탭한 후, 메뉴에서 '주문서 조회'를 탭합니다.
2. 필요한 경우, 검색창에서 조회 조건을 입력한 후 '확인'을 탭합니다.
   - 입력한 조건에 해당하는 견적서가 조회됩니다.
3. 조회된 목록에서, 상세 내용을 확인할 주문서의 주문 번호 또는 '더보기'를 탭합니다
4. 주문 상세 화면에서 주문 내용을 확인합니다.
   - 주문 상세 내역을 수정하려면 항목을 변경한 후 '저장'을 탭합니다.
   - 주문서를 승인하려면 '승인'을 탭합니다. 주문 진행 상태가 '주문 가능' 상태로 변경됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
