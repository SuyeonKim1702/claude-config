# 견적서 작성

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
모바일 견적서 작성에서는 고객으로부터 견적 요청을 받은 품목, 수량, 단가 등 판매 품목에 대해 견적 내역을 거래처에 제시하는 견적서를 등록하고 관리할 수 있습니다.

견적서를 작성하는 방법은 다음과 같습니다.

1. 모바일 홈의 '관리자 메뉴'에서 '영업'을 탭한 후, 메뉴에서 '견적서 작성'을 탭합니다.
2. 견적서 작성 화면에서 견적 정보를 입력합니다.
   - 사업장: 사업장을 선택합니다.
   - 견적 번호: 견적 번호는 자동으로 채번됩니다. (채번 규칙은 ES + yyyymmdd + 6자리 시퀀스입니다.)
   - 영업 그룹: 등록되어 있는 영업 그룹을 선택합니다.
   - 견적 담당자: 등록되어 있는 견적 담당자를 선택합니다.
   - 견적명: 견적서 명칭을 입력합니다.
   - 견적일: 견적일을 선택합니다. (기본 값은 오늘 날짜입니다.)
   - 부가세 유형: 부가세 유형(과세, 비과세, 영세율)을 선택합니다.
   - 통화: 통화를 선택합니다. 선택한 통화에 따라 통화에 해당되는 품목을 추가할 수 있습니다.
   - 진행 상태: 견적 진행 상태를 확인할 수 있습니다.
     - 신규: 신규로 견적서 내용을 입력하는 상태
     - 저장: 견적서 내용을 입력 후 '저장'을 탭한 상태
     - 주문 가능: '저장'을 탭한 후 '승인'을 탭한 상태
   - 지급 조건: 대금 지급 조건을 입력합니다.
   - 파일 첨부: 견적 관련 자료를 첨부합니다.
   - 견적 내용: 견적에 대한 설명을 입력합니다.
   - 견적 품목: '추가'를 탭해 견적서 품목을 추가합니다. 이미 등록된 품목은 추가할 수 없습니다.
     - 품목명: 견적서에 추가할 품목을 선택합니다.
     - 규격: 품목을 선택하면 선택한 품목의 규격이 자동 입력됩니다.
     - 견적 수량: 견적 수량을 입력합니다.
     - 판매 단위: 품목을 선택하면 선택한 품목의 판매 단위가 자동 입력됩니다.
     - 외화단가: 품목과 견적 수량을 입력하면 입력한 품목의 외화단가가 자동 입력됩니다.
     - 외화판매가: 품목과 견적 수량을 입력하면 입력한 품목의 외화판매가가 자동 입력됩니다.
     - 단가: 품목과 견적 수량을 입력하면 입력한 품목의 단가가 자동 입력됩니다. (외화 단가는 환율에 따라 자동 계산됩니다.)
     - 판매가: 품목의 판매가가 자동 입력됩니다.
     - 할인 금액: 할인 금액을 입력합니다. (할인 금액은 판매가보다 클 수 없습니다.)
     - 공급가: 품목의 공급가가 자동 입력됩니다.
     - 부가세: 부가세는 공급가의 10% 금액으로 자동 입력됩니다.
     - 합계액: 공급가 + 부가세 가격이 자동 입력됩니다.
     - 납품일: 품목의 납품일을 입력해 주십시오.
     - 납품 일수: 품목의 납품 일수를 입력합니다.
     - 납품 장소: 품목의 납품 장소를 입력합니다.
     - 인도 조건: 품목의 인도 조건을 입력합니다.
3. '저장'을 탭합니다. 작성한 견적서가 저장됩니다.
   - '승인'을 탭하여 견적서를 승인합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
