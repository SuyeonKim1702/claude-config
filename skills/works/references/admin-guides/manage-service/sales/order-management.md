# 수주 관리

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
수주 관리에서는 견적 내역과 주문 내역을 등록하고 관리할 수 있습니다. 또한 더이상 주문이 진행되지 않는 주문서의 주문 마감 관리를 할 수 있습니다.

# 자주 하는 질문을 확인해 보십시오.

**Q. 견적 요청이란 무엇인가요?**

- 고객이 필요한 제품을 구매하기 위해 구매할 제품의 사양이나 단가 등을 요청하게 되며 이를 견적 요청이라고 합니다.

**Q. 수주 관리 메뉴 처음 화면에서 바로 조회가 안 되는 경우가 있는데, 조회가 되는 기준은 무엇인가요?**

- 로그인한 사용자가 속한 영업그룹이 2개 이상일 때는 접속 시 바로 조회하지 않습니다. 사용자가 속한 영업그룹이 하나라면 화면 접속 시 바로 조회가 됩니다. 만약, 사용자가 속한 영업그룹이 없다면 수주 관리 메뉴를 사용할 수 없습니다.

**Q. 견적서 작성 시 품목 정보가 노출되지 않나요?**

- 품목 정보를 등록했음에도 견적서 작성 시 품목 정보가 노출되지 않는다면, 품목 단가 정보가 등록되어 있는지 확인해 주십시오. 품목별 단가 정보가 등록되어 있지 않을 경우, 품목 정보가 노출 되지 않습니다. 품목 단가 등록은 [영업 > 영업 기준 정보 > 품목 단가 관리](/ko/admin-guides/manage-service/sales/sales-basement/#i-4) 메뉴에서 처리할 수 있습니다.

**Q. 견적서 작성 이후 후속 프로세스를 진행한 경우, 데이터 취소 및 삭제는 어떻게 하나요?**

- 데이터 취소 및 삭제 순서
  1. 회계에서 전표 삭제(회계 서비스 사용 시)
- 문서 삭제 불가 조건
  - 견적서 이후 주문, 판매 등의 프로세스가 진행된 경우에는 바로 삭제할 수 없으며, 위 문서 삭제 순서에 따라 진행할 수 있습니다.
  - 견적서 및 거래명세서 이메일이 이미 발송된 경우에는 문서 삭제를 할 수 없습니다.

**Q. 견적서에 회사 직인을 반영하려면 어떻게 하나요?**

# 견적서 작성

견적 내용을 등록하고 관리할 수 있습니다.

견적서 작성 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 사업장, 견적일, 견적 담당자, 거래처, 견적명, 진행 상태 조건에 따라 검색 |
| ② 견적서 작성 | 견적서 신규 작성 |
| ③ 삭제 | 선택한 견적서 삭제 |

## 견적서 신규 작성

견적서를 작성하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '견적서 작성'을 클릭합니다.
5. 정보를 입력한 후 '추가'를 클릭합니다.
   - 노란색으로 표시된 항목은 필수 입력 항목입니다.
   - 사업장: 사업장을 선택합니다.
   - 견적 번호: 견적 번호는 자동으로 채번됩니다. (채번 규칙: ES + yyyymmdd + 6자리 시퀀스)
   - 견적 담당자: 등록되어 있는 견적 담당자를 선택합니다.
   - 견적명: 견적서 명칭을 입력합니다.
   - 견적일: 견적일을 선택합니다. (기본 값: 오늘 날짜)
   - 부가세 유형: 부가세 유형(과세, 비과세, 영세율)을 선택합니다.
   - 통화: 통화를 선택합니다. 선택한 통화에 따라 통화에 해당되는 품목을 추가할 수 있습니다.
   - 진행 상태: 진행 상태(신규, 저장, 주문 가능)를 선택합니다.
     - 신규: 신규로 견적서 내용을 입력하는 상태
     - 저장: 견적서 내용을 입력한 후 '저장'을 클릭한 상태로 수정 가능
     - 주문 가능: '저장'을 클릭한 후 '승인'을 클릭한 상태로 수정 불가. 입력한 견적서를 메일로 발송 가능
   - 관련 자료 첨부: 견적 관련 자료를 첨부합니다.
6. 품목별 견적 정보를 입력합니다.
   - 품목 코드: 품목 코드 항목을 선택하여 견적서를 작성할 품목을 선택합니다. 품목 코드를 선택하면 품목 그룹, 품목명, 규격, 단위, 판매 단가는 자동으로 입력됩니다.
   - 견적 수량: 입고 수량을 입력합니다.
   - 할인 금액: 추가한 품목에 일괄적으로 할인을 적용하려면 할인을 적용할 품목을 선택 후 '할인 적용'을 클릭합니다. 할인 정보를 입력합니다.(기본 값: 0) 
     - 할인을 적용하려면 적용할 품목들을 선택 후 '할인 적용'을 선택하고 아래 항목을 입력합니다. 

       - 할인 단위를 '퍼센트'로 할 경우 할인율 입력란에 퍼센트 단위로 입력합니다.
       - 할인 단위를 '금액할인'으로 할 경우 할인금액 입력란에 할인가를 입력합니다.
       - 절사 단위, 끝수 처리를 입력합니다.
       - '저장'을 클릭합니다.

     - 할인 적용을 취소하려면 '할인 적용 취소'를 선택 후 '저장'을 클릭합니다.

       - 품목의 할인 금액이 초기화됩니다.
   - 중복된 품목은 저장할 수 없습니다.
   - 추가한 품목을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.
   - 견적서를 등록하려면 항목의 필숫값을 모두 입력하고, 품목이 1개 이상 존재해야 합니다.
7. '저장'을 클릭합니다.
8. 내용을 확인한 후 '저장'을 클릭하면 작성한 견적서가 저장됩니다. '승인'을 클릭하여 견적서를 승인합니다.

## 견적서 삭제

견적서를 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색 창에서 조회할 견적서 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 견적서 현황이 조회됩니다.
5. 삭제할 견적서를 선택합니다.
6. '삭제'를 클릭합니다.

- 견적서 이후 주문, 판매 등의 프로세스가 진행된 경우 문서를 삭제할 수 없습니다.
- 견적서 및 거래명세서 이메일이 이미 발송이 된 경우 문서를 삭제할 수 없습니다.

# 주문서 작성

거래처로부터 주문받은 내역을 등록하거나 승인된 견적서를 바탕으로 주문서를 작성할 수 있습니다. 또한 등록된 주문 품목에 대해 할인 금액을 일괄적으로 적용할 수 있으며, 판매 처리 시 주문서 내용을 참고하여 판매 내역을 등록할 수 있습니다.

주문서 작성 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 사업장, 주문일, 주문 담당자, 거래처, 주문 번호, 진행 상태 조건에 따라 검색 |
| ② 주문서 작성 | 주문서 작성을 위해 등록 페이지로 이동([주문서 작성](#i-5-1) 참조) |
| ③ 삭제 | 선택한 주문서 삭제 |
| ④ 주문서 목록 | 주문서 목록 조회 |

## 주문서 작성

주문서를 작성하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '주문서 작성'을 클릭합니다.
5. 주문 정보를 입력한 후 '추가'를 클릭합니다.
   - 노란색으로 표시된 항목은 필수 입력 항목입니다.
   - 사업장: 사업장을 선택합니다.
   - 주문 번호: 주문 번호는 자동으로 채번됩니다. (채번 규칙: OR +yyyymmdd + 6자리 시퀀스)
   - 주문 담당자: 등록되어 있는 주문 담당자를 선택합니다.
   - 거래처: 등록되어 있는 거래처를 선택합니다.
   - 수금처: 수금처를 선택합니다.
   - 주문일: 주문일을 선택합니다. (기본 값: 현재 날짜)
   - 부가세 유형: 부가세 유형(과세, 비과세, 영세율)을 선택합니다.
   - 통화: 통화를 선택합니다. 선택한 통화에 따라 통화에 해당되는 품목을 추가할 수 있습니다.
   - 단가 확정 여부: 단가 확정 여부를 선택합니다.
   - 진행 상태: 진행 상태(신규, 저장, 출고 가능, 출고)를 선택합니다.
     - 신규: 신규로 주문서 내용을 입력하는 상태
     - 저장: 주문서 내용을 입력한 후 '저장'을 클릭한 상태로 수정 가능
     - 출고 가능: '저장'을 클릭한 후 '승인'을 클릭한 상태로 수정 불가
     - 출고 중: 판매 관리에서 판매가 진행 중인 상태
6. 품목별 주문 정보를 입력합니다.
   - 품목 코드: 품목 코드 항목을 선택하여 주문할 품목을 선택합니다. 품목 코드를 선택하면 품목 그룹, 품목명, 규격, 단위, 판매 단가는 자동으로 입력됩니다.
   - 주문 수량: 주문 수량을 입력합니다.
   - 납기일: 물품 납기일을 입력합니다.
   - 할인 금액: 추가한 품목에 일괄적으로 할인을 적용하려면 할인을 적용할 품목을 선택 후 '할인 적용'을 클릭합니다. 할인 정보를 입력합니다.
     - 할인을 적용하려면 적용할 품목들을 선택 후 '할인 적용'을 선택하고 아래 항목을 입력합니다.
       - 할인 단위를 퍼센트로 할 경우 할인율 입력란에 퍼센트 단위로 입력합니다.
       - 할인 단위를 금액할인으로 할 경우 할인금액 입력란에 할인가를 입력합니다.
       - 절사 단위, 끝수 처리를 입력합니다.
       - '저장'을 클릭합니다.

       - 할인 적용을 취소하려면 '할인 적용 취소'를 선택 후 '저장'을 클릭합니다. 품목의 기존 단가로 변경됩니다.
   - 추가한 내역을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.
7. '저장'을 클릭합니다. 내용을 확인한 후 '저장'을 클릭하면 작성한 주문서가 저장됩니다. '승인'을 클릭하여 주문서를 승인합니다.
   - 주문서 승인이 완료되어야 판매 관리에서 승인된 주문서를 참고하여 판매 내역을 작성할 수 있습니다.

## 견적서를 참조하여 주문서 작성

견적서를 참조하여 주문서를 작성하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '견적서 참조'를 클릭합니다.
   - 승인 완료된 견적서만 참조할 수 있습니다.(견적서 승인은 [견적서 신규 작성](#i-2-1) 참조)
5. 검색 창에서 조회할 견적서 조건을 입력한 후 '검색'을 클릭합니다.
6. 참조할 견적서의 견적 번호를 클릭합니다.
   - 견적서의 품목 및 수량이 자동으로 입력됩니다.
7. 품목별 주문 정보를 수정하거나 입력합니다.
   - 품목 코드: 품목 코드 항목을 선택하여 주문할 품목을 선택합니다. 품목 코드를 선택하면 품목 그룹, 품목명, 규격, 단위, 판매 단가는 자동으로 입력됩니다.
   - 주문 수량: 주문 수량을 입력합니다.
   - 납기일: 물품 납기일을 입력합니다.
   - 할인 금액: 추가한 품목에 일괄적으로 할인을 적용하려면 '할인 적용'을 클릭한 후 할인 정보를 입력하고 '확인'을 클릭합니다.(기본 값: 0)
     - 할인 단위를 퍼센트로 할 경우 할인율 입력란에 퍼센트 단위로 입력합니다.
     - 할인 단위를 금액할인으로 할 경우 할인금액 입력란에 할인가를 입력합니다.
     - 할인 정보를 입력한 후 '확인'을 클릭하면 입력된 할인 내역이 등록된 모든 품목에 일괄 적용됩니다.
   - 추가한 내역을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.
   - 항목을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.
8. '저장'을 클릭합니다.
9. 내용을 확인한 후 '저장' 또는 '승인'을 클릭합니다.
   - '저장'을 클릭하면 작성한 주문서가 저장됩니다.
   - '승인'을 클릭하면 주문서 작성이 완료됩니다.

## 주문서 삭제

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.

   - 입력한 조건에 해당하는 주문서 현황이 조회됩니다.

승인 완료된 주문서는 삭제할 수 없습니다.

# 주문 마감

주문서 관리에서 등록되어 승인된 주문 내역 중 주문 취소 등의 사유로 더 이상 진행되지 않는 내역을 선택하여 마감할 수 있습니다. 마감된 내역은 판매 관리에서 주문서 참조 시 참고되지 않습니다.

주문을 마감하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색 창에서 조회할 주문서 정보를 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 주문서가 조회됩니다.
5. 주문을 마감할 주문서를 선택한 후 '마감 처리'를 클릭합니다.
   - 해당 주문서가 주문 마감 처리됩니다.
   - 마감을 취소하려면, 마감 처리된 주문서를 선택한 후 '마감 취소'를 클릭합니다.

승인 완료된 주문서는 마감 처리할 수 없습니다.
