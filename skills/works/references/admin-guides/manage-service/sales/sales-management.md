# 판매 관리

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
판매 관리에서는 승인된 주문서 내역을 바탕으로 등록된 입고 내역을 첨부하여 판매 내역을 작성하고 관리할 수 있습니다. 매출을 확정하고 회계(사용 시)로 전송하여 전표 처리하거나 세금계산서 발행(전자세금계산서 서비스 사용 시)을 요청할 수 있습니다. 또한 거래명세서를 출력하거나 거래처에 이메일을 전송할 수 있습니다.

# 자주 하는 질문을 먼저 확인해 보십시오.

**Q. 판매 관리 메뉴 처음 화면에서 바로 조회가 안 되는 경우가 있는데, 조회가 되는 기준은 무엇인가요?**

- 로그인한 사용자가 속한 영업그룹이 1개 이상일 때는 접속 시 바로 조회하지 않습니다. 사용자가 속한 영업그룹이 하나라면 화면 접속 시 바로 조회가 됩니다. 만약, 사용자가 속한 영업그룹이 없다면 판매 관리 메뉴를 사용할 수 없습니다.

**Q. 거래명세서에 회사 직인을 반영하려면 어떻게 하나요?**

# 판매 등록

거래처에 판매(출고)된 내역을 등록하고 관리할 수 있습니다.  
승인된 주문서 내역을 근거로 판매 내역을 작성하거나 입고 내역을 불러와 판매 내역을 작성할 수 있으며, 영업 담당자가 직접 판매 내역을 작성 후 등록할 수 있습니다.

일괄 할인 적용 기능을 이용하여 등록된 판매 품목의 할인 금액을 일괄 적용할 수 있습니다. 등록된 판매건별로 거래명세서를 인쇄할 수 있으며, 거래처에 메일을 발송할 수 있습니다.

반품의 경우 판매 형태를 '반품'으로 선택하여 등록하면 반품 처리됩니다. 정상 판매의 경우 판매 내역에 등록된 품목수량만큼 재고가 감소하고, 반품의 경우 재고가 증가됩니다.

판매 등록 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 사업장, 기간, 판매 담당자 또는 영업 그룹, 거래처, 출고 창고, 판매 번호 조건에 따라 검색 |
| ② 판매 등록 | 새로운 판매 내역 등록([판매 내역 등록](#i-2-1) 참조) |
| ③ 삭제 | 선택한 판매 내역 삭제([판매 내역 삭제](#i-2-4) 참조) |
| ④ 다운로드 | 선택한 판매 목록을 엑셀 파일로 다운로드([판매 내역 다운로드](#i-2-7) 참조) |
| ⑤ 판매 내역 목록 | 등록된 판매 내역 목록 조회  - 판매 번호: 클릭하여 판매 내역 조회 가능 - 판매 형태:   - 판매: '판매'로 등록된 판매 내역   - 반품: '반품'으로 등록된 판매 내역 - 확정 여부: 매출 확정 여부 표시([매출 확정](#i-4-1) 참조) |

## 판매 내역 등록

새로운 판매 내역을 등록하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '판매 등록'을 클릭합니다.
5. 판매 정보를 입력한 후 '추가'를 클릭합니다.
   - 노란색으로 표시된 항목은 필수 입력 항목입니다.
   - 사업장: 사업장 항목을 콤보박스에서 선택합니다.
   - 판매 번호: 판매 번호는 자동으로 채번됩니다. (채번 규칙: SL + yyyymmdd + 6자리 시퀀스)
   - 판매 형태: 판매 형태(판매, 반품)를 선택합니다.
   - 판매 담당자: 등록되어 있는 판매 담당자를 선택합니다.
   - 거래처: 등록되어 있는 거래처를 선택합니다.
   - 수금처: 등록되어 있는 거래처에서 수금할 거래처를 선택합니다.
   - 확정 여부: 판매 내역 조회 시 표시되는 항목으로 매출 확정 상태(확정/미확정)가 표시됩니다.
   - 영업 그룹: 판매 내역을 등록할 영업 그룹이며, 목록에서 선택한 영업 그룹이 표시됩니다.
   - 거래처 담당자: (전자세금계산서 서비스 사용 시)거래처 담당자를 입력합니다.
   - 연락처: (전자세금계산서 서비스 사용 시)거래처 담당자의 연락처를 입력합니다.
   - 이메일: (전자세금계산서 서비스 사용 시)거래처 담당자의 이메일을 입력합니다.
   - 작성일: 판매 등록 날짜를 선택합니다.
   - 판매일: 판매 날짜를 선택합니다.(기본 값: 현재 날짜)
   - 출고 창고: 판매 물품이 출고될 출고 창고를 선택합니다.
   - 주문 번호: 주문서참조를 했을 때 주문 번호가 자동으로 입력됩니다.
   - 부가세 유형: 부가세 유형(과세, 비과세, 영세율)을 선택합니다.
   - 수금 형태: 수금 형태(외상 매출, 신용카드, 현금)를 선택합니다.
   - 통화: 통화를 선택합니다. 선택한 통화에 따라 통화에 해당되는 품목을 추가할 수 있습니다.
   - 영수/청구: 영수 또는 청구 구분을 선택합니다.
6. 품목별 판매 정보를 입력합니다.
   - 품목 코드: 품목 코드 항목을 선택하여 판매할 품목을 선택합니다. 품목 코드를 선택하면 품목 그룹, 품목명, 규격, 단위, 판매 단가는 자동으로 입력됩니다.
   - 판매 수량: 판매 수량을 입력합니다.
   - 할인 금액: 추가한 품목에 일괄적으로 할인을 적용하려면 할인을 적용할 품목을 선택 후 '할인 적용'을 클릭합니다. 할인 정보를 입력합니다.(기본 값: 0) 
     - 할인을 적용하려면 할인 적용을 선택 후 아래 항목을 입력합니다. 

       - 할인 단위를 퍼센트로 할 경우 할인율 입력란에 퍼센트 단위로 입력합니다.
       - 할인 단위를 금액할인으로 할 경우 할인금액 입력란에 할인가를 입력합니다.
       - 절사 단위, 끝수 처리를 입력합니다.
       - '저장'을 클릭합니다.

     - 할인 적용을 취소하려면 '할인 적용 취소'를 선택 후 '저장'을 클릭합니다.

       - 품목의 기존 단가로 변경됩니다.
   - 구성품 재고 반영 여부: 선택한 품목이 세트 품목을 사용하면 구성품 재고 반영 여부를 체크합니다.
   - 중복된 품목은 저장할 수 없습니다.
   - 추가한 품목을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.  

     구성품 재고 반영 여부를 체크했을 경우, 품목의 재고는 선택한 품목의 재고가 아닌 품목의 세트 품목 재고가 반영됩니다.
7. '저장'을 클릭합니다.

출고 창고의 재고가 부족한 경우 판매 내역이 저장되지 않습니다. 재고를 확인해 주십시오.

## 주문서 또는 입고 내역을 참조하여 판매 내역 등록

주문서 또는 입고 내역을 참조하여 판매 내역을 등록하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '판매 등록'을 클릭합니다.
5. 입고 내역을 참조하는 경우, 거래처를 입력합니다.
6. '주문서 참조' 또는 '입고 내역 참조'를 클릭합니다.
   - 주문서 참조로 등록하는 경우, 승인 완료된 주문서만 참조할 수 있습니다. (주문서 승인은 [주문서 작성](/ko/admin-guides/manage-service/sales/order-management/#i-5-1) 참조)
   - 입고 내역 참조로 등록하는 경우, 저장 완료된 입고 내역만 참조할 수 있습니다. ([입고 등록](/ko/admin-guides/manage-service/sales/inventory-management/#i-1-1) 참조)
7. 참조할 주문서 또는 입고 내역의 주문 번호 또는 입고 번호를 클릭합니다.
   - 주문서 또는 입고 내역의 품목 및 수량이 자동으로 입력됩니다.
8. 필요한 경우, 품목별 판매 정보를 수정하거나 입력합니다.
   - 품목 코드: 품목 코드 항목을 선택하여 판매할 품목을 선택합니다. 품목 코드를 선택하면 품목 그룹, 품목명, 규격, 단위, 판매 단가는 자동으로 입력됩니다.
   - 판매 수량: 판매 수량을 입력합니다.
   - 할인 금액: 추가한 품목에 일괄적으로 할인을 적용하려면 할인을 적용할 품목을 선택 후 '할인 적용'을 클릭합니다. 할인 정보를 입력합니다.(기본 값: 0) 
     - 할인을 적용하려면 할인 적용을 선택 후 아래 항목을 입력합니다. 

       - 할인 단위를 퍼센트로 할 경우 할인율 입력란에 퍼센트 단위로 입력합니다.
       - 할인 단위를 금액할인으로 할 경우 할인금액 입력란에 할인가를 입력합니다.
       - 절사 단위, 끝수 처리를 입력합니다.
       - '저장'을 클릭합니다.

     - 할인 적용을 취소하려면 '할인 적용 취소'를 선택 후 '저장'을 클릭합니다.

       - 품목의 기존 단가로 변경됩니다.
   - 구성품 재고 반영 여부: 선택한 품목이 세트 품목을 사용하면 구성품 재고 반영 여부를 체크합니다.
   - 중복된 품목은 저장할 수 없습니다.
   - 추가한 품목을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.

   구성품 재고 반영 여부를 체크했을 경우, 품목의 재고는 선택한 품목의 재고가 아닌 품목의 세트 품목 재고가 반영됩니다.

   - 항목을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.
   - 추가한 품목에 일괄적으로 할인을 적용하려면 '할인 적용'을 클릭한 후 할인 정보를 입력하고 '확인'을 클릭합니다.

9. '저장'을 클릭합니다.

출고 창고의 재고가 부족한 경우 판매 내역이 저장되지 않습니다. 재고를 확인해 주십시오.

## 판매 내역 상세 정보 조회/수정

등록된 판매 내역을 조회하거나 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색 창에서 조회할 판매 내역 조건을 입력한 후 '검색'을 클릭합니다.
5. 조회할 판매 내역의 판매 번호를 클릭합니다.
   - '미확정' 상태의 판매 내역은 수정할 수 있습니다. 필요한 내용을 수정해 주십시오.
   - '확정' 상태의 판매 내역은 수정할 수 없으며, [거래명세서를 인쇄](#i-2-5)하거나 거래처 담당자에게 [메일을 발송](#i-2-6)할 수 있습니다.
6. 수정한 내역이 있는 경우 '저장'을 클릭합니다.

## 판매 내역 삭제

판매 내역을 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색 창에서 조회할 판매 내역 조건을 입력한 후 '검색'을 클릭합니다.
5. 삭제할 판매 내역의 체크 박스를 선택합니다.
   - 매출 확정 상태가 '확정'인 판매 내역은 삭제할 수 없습니다.
6. '삭제'를 클릭합니다.

## 거래명세서 인쇄

판매 거래명세서를 인쇄하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색 창에서 조회할 판매 내역 조건을 입력한 후 '검색'을 클릭합니다.
5. 거래명세서를 인쇄할 판매 내역의 판매 번호를 클릭합니다.
6. 거래명세서 '인쇄'를 클릭합니다.

회사 직인이 등록되어 있는 경우, 거래처명세서 인쇄 및 메일 발송 화면에 회사 직인이 표시됩니다.([경영지원 설정 > 회사 정보 > 회사 정보 관리](/ko/admin-guides/manage-service/support-management/company-info/) 메뉴에서 등록)
