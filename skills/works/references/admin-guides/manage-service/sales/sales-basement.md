# 영업 기준 정보

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
영업 기준 정보에서는 품목, 단가, 매장과 같은 영업 관리를 위한 기준 정보를 등록하고 관리할 수 있습니다. 품목 정보, 품목 단가 정보, 거래처별 품목 단가 정보는 엑셀 파일을 업로드해 기존에 관리하던 정보를 일괄 등록할 수도 있습니다.

# 품목 정보 관리

품목 그룹을 생성하여 품목을 해당 품목 그룹별로 관리합니다. 품목 정보 관리 메뉴에서는 품목 그룹을 생성하고, 품목 그룹에 품목을 추가하거나 관리할 수 있습니다. 세트 품목의 경우 구성 품목과 구성 수량을 등록하고 관리할 수 있습니다. 엑셀 파일을 업로드 기능을 이용하면 다량의 품목 정보를 일괄 등록할 수도 있습니다.

품목 정보 관리에 대한 자세한 내용은 [품목 정보 관리](/ko/admin-guides/manage-service/support-management/common-code/#i-4)를 참고해 주십시오.

# 창고 정보

사업장별 창고 정보를 등록하고 관리할 수 있습니다. 창고 수에 제한 없이 사업장의 창고를 등록할 수 있으며, 모든 창고의 분산된 재고 정보를 관리할 수 있습니다.

창고 정보 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 영역설명 |
| --- | --- |
| ① 검색 창 | 사업장, 창고, 사용 여부 조건에 따라 검색 |
| ② 추가 | 새로운 창고 정보 추가([창고 정보 추가](#i-2-1) 참고) |
| ③ 삭제 | 선택한 창고 삭제([창고 삭제](#i-2-3) 참고) |
| ④ 다운로드 | 창고 정보를 엑셀 파일로 다운로드 |
| ⑤ 창고 목록 | 창고 정보 조회  - 창고 코드를 클릭하면 해당 창고의 상세 정보 조회 및 수정 |

## 창고 정보 추가

새로운 창고 정보를 추가하는 방법은 다음과 같습니다.

[설정](/ko/admin-guides/manage-service/sales/settings/) 메뉴에서 매장 관리 사용을 'Y'로 설정한 경우, 매장 정보를 등록하면 해당 매장 코드로 창고 정보가 자동 생성됩니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '추가'를 클릭합니다.
5. 창고 상세 정보를 입력합니다.
   - 노란색으로 표시된 항목은 필수 입력 항목입니다.
   - 창고 코드는 영문 대소문자와 숫자 조합으로 생성할 수 있으며 중복되지 않아야 합니다.
   - 매장 항목은 창고 유형에서 '매장 창고'를 선택한 경우에만 입력할 수 있습니다.
   - 사용 여부를 '미사용'으로 선택하는 경우, 영업 서비스 내에서 해당 창고가 조회되지 않습니다.
6. '저장'을 클릭합니다.

## 창고 정보 수정

창고 정보를 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 수정하려는 창고의 창고 코드를 클릭합니다.
5. 창고 상세 정보를 수정합니다.
   - 사업장과 창고 코드를 제외한 모든 항목을 수정할 수 있습니다. 단, 매장 정보 등록 시 자동 생성된 창고의 경우 사용 여부만 변경할 수 있습니다.
   - 창고 코드와 매장 코드가 다를 경우, 창고 유형을 매장 창고로 선택하면 보조 창고로 인식합니다.
   - 창고 코드와 매장 코드가 동일한 경우, 필수 매장 창고로 인지하여 정보를 수정할 수 없습니다.
6. '저장'을 클릭합니다.

## 창고 삭제

창고를 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 삭제할 창고를 선택합니다.
5. '삭제'를 클릭합니다.
6. 알림 팝업 창에서 '확인'을 클릭합니다.
   - 선택한 창고가 삭제됩니다.

아래의 경우 창고를 삭제할 수 없습니다.

- 창고에 재고 데이터가 존재하는 경우
- 해당 창고로 등록된 판매 데이터가 존재하는 경우
  - 판매 데이터 삭제 후 창고 삭제 가능
