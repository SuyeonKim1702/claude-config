# 재고 관리

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
재고 관리에서는 입⋅출고 관리뿐만 아니라 창고 간 재고 이동 관리, 전산 재고와 실물 재고를 확인하고 일치시킬 수 있는 재고 실사, 현재고 조회, 재고 수불 현황 조회, 월 단위 재고 마감 관리를 할 수 있습니다.

# 입고 관리

입고 내역을 등록하고 입고 내역을 엑셀 파일로 다운로드할 수 있습니다.

입고 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 사업장, 입고일, 입고 담당자, 입고 창고, 입고 유형 또는 입고 번호 조건에 따라 검색 |
| ② 입고 등록 | 입고 내역 등록([입고 등록](#i-1-1) 참고) |
| ③ 입고 삭제 | 선택한 입고 내역 삭제([입고 삭제](#i-1-3) 참고) |
| ④ 다운로드 | 입고 관리 목록을 엑셀 파일로 다운로드([입고 내역 다운로드](#i-1-2) 참고) |
| ⑤ 입고 내역 목록 | 입고 내역 목록 조회  - 입고 번호를 클릭하면 해당 입고 내역 상세 정보 조회 |

## 입고 등록

입고 내역을 등록하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '입고 등록'을 클릭합니다
   - 노란색으로 표시된 항목은 필수 입력 항목입니다.
   - 입고 번호: 입고 번호는 자동으로 채번됩니다. (채번 규칙: PI + yyyymmdd + 6자리 시퀀스)
   - 사업장: 사업장 항목을 선택합니다.
   - 입고 창고: 선택한 사업장의 창고를 선택합니다. 입고 창고를 선택하지 않으면 품목을 추가할 수 없습니다.
   - 입고일: 당일 포함 과거 일자만 선택합니다.
   - 입고 유형: 입고 유형을 선택합니다.
     - 입고 유형 종류: 입고 유형(생산 입고, 구매 입고, 기타 입고)을 선택합니다.
   - 입고 담당자: 입고 담당자를 선택합니다.입고 정보를 입력한 후 '추가'를 클릭합니다.  

5. 입고 품목 정보를 입력합니다.
   - 품목 코드: 품목 코드 항목을 선택하여 입고할 품목을 선택합니다. 품목 코드를 선택하면 품목 그룹, 품목명, 규격, 재고 수량, 재고 단위는 자동으로 입력됩니다.
   - 입고 수량: 입고 수량을 입력해 주십시오(필수 항목). 입고 수량은 정수 및 양수만으로 입력합니다.
   - 중복된 품목은 저장할 수 없습니다.
   - 추가한 품목을 삭제하려면 '삭제'를 클릭합니다.
6. '저장'을 클릭합니다.

## 입고 내역 다운로드

입고 내역을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 검색 창에서 조회할 입고 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 입고 내역이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 입고 내역이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Downloads` 폴더에 `입고 관리_YYYYMMDDXXXXXX.xlsx` 파일명으로 저장됩니다.

## 입고 삭제

입고 내역을 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 삭제할 입고 자료를 선택합니다.
5. '입고 삭제'를 클릭합니다.
6. 알림 팝업 창에서 '확인'을 클릭합니다.
   - 입고 자료가 삭제됩니다.

아래의 경우 입고 자료를 삭제할 수 없습니다.

- 마감된 월의 입고 자료

# 출고 관리

판매하는 품목의 출고 내역 정보를 등록하거나 조회할 수 있습니다. 정상적인 판매에 의한 출고뿐만 아니라 손망처리, 폐기 등 출고 처리가 필요한 경우 기타 출고 처리를 통해 재고를 관리할 수 있으며, 재고는 출고 수량만큼 감소합니다.

출고 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 사업장, 출고일, 출고 담당자, 출고 창고, 출고 유형, 출고 번호 조건에 따라 검색 |
| ② 출고 등록 | 새로운 출고 내역 등록([출고 등록](#i-2-1) 참고) |
| ③ 출고 삭제 | 선택한 출고 내역 삭제([출고 삭제](#i-2-3) 참고) |
| ④ 다운로드 | 출고 관리 목록을 엑셀 파일로 다운로드([출고 내역 다운로드](#i-2-2) 참고) |
| ⑤ 출고 내역 목록 | 출고 내역 목록 조회  - 출고 번호를 클릭하면 해당 출고 내역 상세 정보 조회 |

## 출고 등록

출고 내역을 등록하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '출고 등록'을 클릭합니다.
   - 노란색으로 표시된 항목은 필수 입력 항목입니다.
   - 출고 번호: 출고 번호는 자동으로 채번됩니다. (채번 규칙: OC + yyyymmdd + 6자리 시퀀스)
   - 사업장: 사업장을 선택합니다.
   - 출고 창고: 선택한 사업장의 창고를 선택합니다. 출고 창고를 선택하지 않으면 품목을 추가할 수 없습니다.
   - 출고일: 당일 포함 과거 일자만 선택합니다.
   - 출고 유형: 출고 유형(폐기, 손망 처리, 기타)을 선택합니다.
   - 출고 담당자: 출고 담당자를 선택합니다. 출고 정보를 입력한 후 '추가'를 클릭합니다.

1. - 품목 코드: 품목 코드 항목을 선택하여 출고할 품목을 선택합니다. 품목 코드를 선택하면 품목 그룹, 품목명, 규격, 재고 수량, 재고 단위는 자동으로 입력됩니다.
   - 출고 수량: 출고 수량을 입력합니다(필수 항목). 출고 수량은 정수 및 양수만으로 입력합니다.
     - 이 메뉴에서 출고 수량은 마이너스 재고가 허용되지 않습니다. 단, 판매 등록 시에는 마이너스 재고를 허용하고 있습니다.
   - 중복된 품목은 저장할 수 없습니다.
   - 추가한 품목을 삭제하려면 '삭제'를 클릭합니다.

6. '저장'을 클릭합니다.

# 재고 이동 관리

사업장 간 또는 창고 간 재고가 이동된 결과를 등록하고 내역을 관리할 수 있습니다. 이동 결과를 등록하면 입고 및 출고 창고에 내역이 각각 반영됩니다. 입출고 유형은 입⋅출고 사업장이 다른 경우 사업장 이동으로, 같은 사업장이지만 창고만 다른 경우 창고 이동으로 구분합니다.

재고 이동 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 재고 이동 번호, 이동일, 담당자, 출고 사업장, 출고 창고, 입고 사업장, 입고 창고 조건에 따라 검색 |
| ② 재고 이동 등록 | 재고 이동 등록([재고 이동 등록](#i-3-1) 참고) |
| ③ 재고 이동 삭제 | 선택한 재고 이동 내역 삭제([재고 이동 삭제](#i-3-3) 참고) |
| ④ 다운로드 | 재고 이동 목록을 엑셀 파일로 다운로드([재고 이동 내역 다운로드](#i-3-2) 참고) |
| ⑤ 재고 이동 내역 목록 | 재고 이동 내역 목록 조회  - 재고 이동 번호를 클릭하면 해당 재고 이동 내역 상세 정보 조회 |

## 재고 이동 등록

재고 이동 내역을 등록하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '재고 이동 등록'을 클릭합니다.
   - 노란색으로 표시된 항목은 필수 입력 항목입니다.
   - 재고 이동 번호: 재고 이동 번호는 자동으로 채번됩니다.(채번 규칙: WT + yyyymmdd + 6자리 시퀀스)
   - 담당자: 재고 이동을 담당할 당담자를 선택합니다.
   - 이동일: 당일 포함 과거 날짜만 선택합니다.
   - 출고 사업장: 재고를 출고하기 위한 사업장을 선택합니다.
   - 출고 창고: 출고 사업장에 속해있는 출고 창고를 선택해 주십시오. 출고 창고를 선택하지 않으면 품목을 추가할 수 없습니다.
   - 입고 사업장: 재고를 입고하기 위한 사업장을 선택합니다.
   - 입고 창고: 입고 사업장에 속해 있는 입고 창고를 선택합니다. 재고 이동 정보를 입력한 후 '추가'를 클릭합니다.  

5. 재고 이동 품목 정보를 입력합니다.
   - 품목 코드: 품목 코드 항목을 선택하여 입고할 품목을 선택합니다. 품목 코드를 선택하면 품목 그룹, 품목명, 규격, 재고 수량, 재고 단위는 자동으로 입력됩니다.
   - 이동 수량: 이동 수량을 입력합니다. 이동 수량은 재고 수량보다 많을 수 없습니다. 이동 수량은 정수 및 양수만으로 입력합니다.
   - 중복된 품목은 저장할 수 없습니다.
   - 추가한 품목을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.
6. '저장'을 클릭합니다.

# 재고 실사 관리

창고의 재고에 대해 회사 내부 규정에 따른 정기 실사, 품목의 중요도에 따라 순환 실사, 실재고 수량 관리를 위한 수시 실사에 대한 결과를 등록하고 관리할 수 있습니다.

재고 실사 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 창 | 사업장, 재고 실사일, 재고 실사 담당자, 재고 실사 창고, 재고 실사 번호, 재고 실사 유형, 재고 실사 상태 조건에 따라 검색 |
| ② 재고 실사 등록 | 재고 실사 등록([재고 실사 등록](#i-4-1) 참고) |
| ③ 재고 실사 삭제 | 선택한 재고 실사 내역 삭제([재고 실사 삭제](#i-4-3) 참고) |
| ④ 다운로드 | 재고 실사 목록을 엑셀 파일로 다운로드([재고 실사 내역 다운로드](#i-4-2) 참고) |
| ⑤ 재고 실사 내역 목록 | 재고 실사 내역 목록 조회  - 재고 실사 번호를 클릭하면 해당 재고 실사 내역 상세 정보 조회 |

- 재고 실사 유형은 다음과 같이 구분됩니다.
  - 정기 실사: 재고 조사를 연, 분기, 반기 또는 월 단위로 한 번씩 실행하는 재고 조사(예: 매달 네 번째 주 금요일 정기 실사)
  - 순환 실사: 실제 재고 수량과 장부상의 재고량을 일치시키기 위해 주기적으로 실제 수량을 조사하는 재고 조사
  - 수시 실사: 특정 품목에 대한 불시 확인이 필요할 때 수시로 조사하는 재고 조사
- 재고 실사 상태는 저장 상태에 따라 자동으로 입력되며 다음과 같이 구분됩니다.
  - 실사 중: 임시 저장 상태
  - 실사 완료: 실사 완료(저장), 재고 조정 이전 단계
  - 조정 완료: 재고 조정이 완료되어 반영된 단계

## 재고 실사 등록

재고 실사 내역을 등록하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. '재고 실사 등록'을 클릭합니다.
5. 재고 실사 정보를 입력한 후 '추가'를 클릭합니다.
   - 노란색으로 표시된 항목은 필수 입력 항목입니다.
   - 재고 실사 번호: 재고 실사 번호는 자동으로 채번됩니다. (채번 규칙: WT + yyyymmdd + 6자리 시퀀스)
   - 사업장: 사업장을 선택합니다.
   - 재고 실사 창고: 재고 실사 창고를 선택합니다. 재고 실사 창고를 선택하지 않으면 품목을 추가할 수 없습니다.
   - 재고 실사명: 재고 실사명을 입력합니다.
   - 재고 실사일: 재고 실사일을 입력합니다.
   - 재고 실사 유형: 재고 실사 유형을 선택합니다.
     - 정기 실사: 재고 조사를 일년에 한 번 또는 분기, 반기, 월에 한 번씩 실행하는 재고 조사 (예: 매달 4째주 금요일 정기실사)
     - 순환 실사: 실제 재고 수량과 장부상의 재고량을 일치시키기 위하여 주기적으로 재고를 실제로 조사하는 재고 조사
     - 수시 실사: 특정 품목에 대한 불시 확인이 필요할 때 수시로 조사하는 재고 조사
   - 재고 실사 담당자: 재고 실사 담당자를 콤보박스에서 선택합니다.
   - 재고 실사 상태: 재고 실사 상태는 저장 상태에 따라 자동으로 입력됩니다.
     - 실사 중: 임시 저장 상태
     - 실사 완료: 실사 완료(저장), 재고 조정 이전 단계
     - 반영 완료: 재고 조정이 완료되어 재고 반영이 된 단계.  

6. 재고 실사 품목 정보를 입력합니다.
   - 품목코드: 품목코드 항목을 선택하여 입고할 품목을 선택합니다. 품목코드를 선택하면 품목 그룹, 품목명, 규격, 재고 수량, 재고 단위는 자동으로 입력됩니다.
   - 실사 수량: 실제 품목 수량을 입력해 주십시오. 실사 수량은 정수 및 양수만으로 입력합니다.
   - 차이 수량: 차이 수량은 '실사 수량 - 전산 수량'을 의미합니다.
   - 중복된 품목은 저장할 수 없습니다.
   - 추가한 품목을 삭제하려면 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.
   - 재고 실사 내역 목록에서 '재고 실사 상태가 실사 완료'로 표시됩니다.
   - 재고 실사가 끝나지 않았다면 '임시 저장'을 클릭합니다. 재고 실사 내역 목록에서 재고 실사 상태가 '실사 중'으로 표시됩니다.
   - 이후 재고 실사가 완료되면 재고 조정을 완료하고 '재고 반영'을 클릭합니다. 재고 실사 내역 목록에서 재고 실사 상태가 '조정 완료'로 표시됩니다.

## 재고 실사 내역 다운로드

재고 실사 내역을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 검색 창에서 조회할 재고 실사 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 재고 실사 내역이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 재고 실사 내역이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Downloads` 폴더에 `재고 실사 관리_YYYYMMDDXXXXXX.xlsx` 파일명으로 저장됩니다.

## 재고 실사 삭제

재고 실사 자료를 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '영업'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 삭제할 재고 실사 자료를 선택합니다.
5. '재고 실사 삭제'를 클릭합니다.
6. 알림 팝업 창에서 '확인'을 클릭합니다.
   - 재고 실사 자료가 삭제됩니다.

아래의 경우 재고 실사 자료를 삭제할 수 없습니다.

- 마감된 월의 재고 실사 자료
