# 설정

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
매장 관리 사용 여부, 마이너스 재고 허용 여부, 마이너스 재고 허용 시 경고 메시지 여부, 단가 설정 등을 설정할 수 있습니다.

영업 설정 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 매장 관리 사용 | 매장 관리 사용을 '사용 함'으로 선택한 경우, 영업 기준 정보에서 매장 정보 메뉴 관리  - 운영 중인 직영 매장이 있는 경우 '사용 함' 선택 |
| ② 마이너스 재고 허용 여부 | 마이너스 재고 허용 여부를 '사용 함'으로 선택하는 경우, 판매 관리에서 실물은 존재하지만 전산상 재고가 없는 상황이 발생하면 일단 판매 처리할 수 있도록 마이너스를 허용 |
| ③ 마이너스 재고 허용 시 경고 메시지 여부 | 마이너스 재고 허용 시 경고 메시지 여부를 '사용 함'으로 선택하는 경우, 재고가 마이너스가 되면 경고 메시지 표시 |
| ⑤ 견적 유효 기간 | 견적 유효 기간을 입력한 경우, 작성한 견적서는 작성일로부터 해당 유효 기간 내에만 사용 가능  견적서 작성 및 견적서 상세 정보 조회 화면, 견적서 이메일 발송 시 유효 기간이 표시됨 |
| ⑥ 취소 | 설정한 내용을 삭제하고 설정 전의 화면으로 새로 고침 |
| ⑦ 저장 | 설정한 내용 저장 |

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
