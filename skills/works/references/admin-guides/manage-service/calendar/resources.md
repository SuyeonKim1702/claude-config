# 설비 관리

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
회의실, 화상회의 장비 등 설비를 등록하고 관리할 수 있습니다.

등록된 설비는 구성원이 [일정을 생성](/ko/use-guides/calendar/create-event/new-event/#i-6)하거나 [설비 예약](/ko/use-guides/calendar/reserve-resource/view-availability/) 시 사용할 수 있습니다.

# 설비 추가

구성원들에게 필요한 설비를 추가할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '설비 '를 선택합니다.
3. 오른쪽의 '설비 추가'를 누릅니다.
4. 카테고리, 설비명, 수용 인원, 설명, 사용 설정, 사용 회사 등을 입력합니다.
   - 카테고리 : 설비가 포함될 카테고리를 선택할 수 있습니다. 카테고리를 켜면 생성된 카테고리 목록이 노출됩니다.
   - 설비명: 설비의 이름을 입력합니다.
   - 수용 인원: 수용 인원을 입력하면 구성원이 설비를 예약할 때 참고 정보로 활용할 수 있습니다.
   - 설명: 설명을 입력하면 구성원이 설비를 예약할 때 참고 정보로 활용할 수 있습니다.
   - 사용 설정: 사용 설정을 켜면 구성원이 일정을 생성하거나 설비를 예약할 때 해당 설비를 볼 수 있습니다.
   - 사용 회사: 그룹사인 경우 해당 설비를 사용할 수 있는 회사를 설정할 수 있습니다. 그룹사가 아닐 경우 본인 회사만 사용 회사로 노출됩니다.
5. 오른쪽의 '저장'을 눌러 설비를 저장합니다.

# 설비 정보 수정

설비정보를 확인하고 수정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '설비 '를 선택합니다.
3. 수정하고자 하는 설비가 포함된 카테고리를 누른 뒤 설비를 누릅니다.  
   카테고리가 설정되지 않은 설비는 최상단의 회사명 카테고리에서 확인할 수 있습니다.
4. 오른쪽의 '수정'을 누릅니다.
5. 수정하고자 하는 정보를 선택해 수정합니다.
6. 오른쪽의 '저장'을 눌러 수정된 설비 정보를 저장합니다.

# 설비 삭제

등록된 설비를 삭제할 수 있습니다.

설비를 삭제하면 설비 예약자에게 알림이 발송되고, 일정은 유지되나 예약되었던 설비는 삭제됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '설비 '를 선택합니다.
3. 삭제하고자 하는 설비의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/checkbox16_default.png) 아이콘을 누릅니다.
4. 상단의 '삭제'를 누릅니다.
5. '확인'을 눌러 삭제합니다.

# 설비 예약 관리

설비의 예약 현황을 확인하고, 구성원의 설비 예약을 취소할 수 있습니다.

설비 예약을 삭제하더라도, 설비가 예약된 일정 자체가 삭제되는 것은 아닙니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다.
2. '설비 '를 선택합니다.
3. 예약을 관리하고자 하는 설비가 포함된 카테고리를 누른 뒤 설비 목록의 '예약 관리'를 누릅니다.  
   카테고리가 설정되지 않은 설비는 최상단의 회사명 카테고리에서 확인할 수 있습니다.
4. 날짜 별로 예약 일시, 제목, 예약자 등 예약 현황을 확인합니다.
5. 취소가 필요한 예약은 설비의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/checkbox16_default.png) 아이콘을 누른 뒤 '삭제'를 누릅니다.

# 카테고리 관리

카테고리를 등록하여 여러 개의 설비를 묶어서 사용할 수 있습니다. 카테고리를 이동, 삭제하거나 하위 카테고리를 생성할 수 있습니다.

단, 기본으로 설정되어 있는 회사명 카테고리는 수정하거나 삭제할 수 없습니다.

## 카테고리 추가

### 모바일 앱

1. Admin 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 아이콘을 탭합니다.
2. '서비스'를 탭합니다.
3. '캘린더'를 탭하여 '설비 ' 화면으로 이동합니다.
4. 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/arrow_right_black-3.png) 아이콘을 탭하여 카테고리 목록으로 이동합니다.
5. '카테고리 추가'를 탭합니다.
6. 카테고리명을 입력합니다.
7. 오른쪽 상단의 '저장'을 탭합니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다.
2. '설비 '를 선택합니다.
3. 왼쪽 카테고리 목록 상단의 '카테고리 추가'를 클릭합니다.
4. 카테고리명을 입력합니다.
5. 오른쪽 하단의 '저장'을 클릭합니다.

## 카테고리 수정

등록된 카테고리명을 수정할 수 있습니다. 카테고리명은 중복으로 사용할 수 있습니다.

### 모바일 앱

1. Admin 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 아이콘을 탭합니다.
2. '서비스'를 탭합니다.
3. '캘린더'를 탭하여 '설비 ' 화면으로 이동합니다.
4. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 탭한 뒤, '수정'을 탭합니다.
5. 카테고리명을 수정합니다.
6. 오른쪽 상단의 '저장'을 탭합니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다.
2. '설비 '를 선택합니다.
3. 왼쪽 카테고리 목록에서 수정하고자 하는 카테고리 위에 마우스 포인터를 올립니다.
4. 카테고리명 오른쪽의![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 클릭한 뒤 '수정'을 클릭합니다.
5. 카테고리명을 수정합니다.
6. 오른쪽 하단의 '저장'을 클릭합니다.

카테고리 목록에서 카테고리를 이동할 수 있습니다.

### 모바일 앱

1. Admin 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 아이콘을 탭합니다.

3. '캘린더'를 탭하여 '설비 ' 화면으로 이동합니다.
4. 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/arrow_right_black-3.png) 아이콘을 탭하여 카테고리 목록으로 이동합니다.
5. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ico-18-move-c-5-c-5-c-6@3x.png) 아이콘을 탭해 카테고리 목록에서 이동하고자 하는 카테고리를 필요한 위치로 끌어다 놓습니다.
