# 캘린더 일반

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
캘린더 서비스의 일반적인 사용 기능을 설정할 수 있습니다.

# 일정 표시 시간

새롭게 추가되는 구성원 캘린더의 개인 일간, 주간 일정에서 보이는 시간의 기본값을 설정할 수 있습니다. 기존 구성원들에게는 변경 내용이 저장되지 않습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 ' 탭을 선택합니다.
3. '일정 표시 시간' 항목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. 표시 시작 시간과 표시 종료 시간을 설정합니다.
5. 오른쪽 상단의 '저장'을 눌러 설정값을 저장합니다.

# 설비 예약

설비 예약 사용 여부를 설정합니다. 설비 기능을 사용하면 구성원이 [일정을 만들 때](?page_id=3952) 설비를 포함해서 일정을 만들 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 ' 탭을 선택합니다.
3. '설비 예약' 항목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. '설비 예약'을 켭니다.
5. 오른쪽 상단의 '저장'을 눌러 선택값을 저장합니다.

# 회사 캘린더

[회사 캘린더](/ko/admin-guides/manage-service/calendar/manage-company-calendar/) 사용 여부를 설정합니다. 회사 캘린더는 관리자가 만드는 캘린더로 관리자만 일정을 등록할 수 있고, 모든 구성원이 볼 수 있습니다. 회사 캘린더의 이름은 [회사 정보 관리에 입력한 기업 및 단체명](/ko/admin-guides/settings/company-information/)입니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 ' 탭을 선택합니다.
3. '회사 캘린더' 항목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. '회사 캘린더'를 켭니다.
5. 오른쪽 상단의 '저장'을 눌러 선택값을 저장합니다.

# 구성원 일정 보기

[구성원 일정 보기](/ko/use-guides/calendar/check-event/view-other-peoples-calendars/) 기능의 사용 여부를 설정합니다. 구성원 일정 보기를 사용하면 구성원이 다른 구성원의 일정도 볼 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 ' 탭을 선택합니다.
3. '구성원 일정 보기' 항목 오른쪽  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. '구성원 일정 보기'를 켭니다.
5. 오른쪽 상단의 '저장'을 눌러 선택값을 저장합니다.

## 구성원 일정 보기의 공개 범위 설정

'구성원 일정 보기' 항목이 켜져 있을 때 나타나는 항목입니다. 다른 구성원의 일정에 노출될 정보의 범위를 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 ' 탭을 선택합니다.
3. '구성원 일정 보기' 항목 오른쪽  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. '사내 구성원(동일 도메인)이 조회 시' 항목에서 공개 범위를 선택합니다.
   - 일정 상세 정보 조회: 다른 구성원 일정의 시간 정보뿐 아니라 일정의 제목과 장소 등의 상세 정보까지 노출합니다.
     - 예외 관리: 일정의 상세 정보를 노출하지 않을 구성원이나 부서를 지정합니다.
   - 일정의 일시만 조회: 다른 구성원 일정에서 시간 정보만 노출합니다.
5. 오른쪽 상단의 '저장'을 눌러 선택값을 저장합니다.

# 일정 내보내기

'[일정 내보내기](/ko/use-guides/calendar/create-event/event-import-export/)'를 사용하면 캘린더의 일정을 내보내 백업할 수 있습니다.

일정 내보내기를 사용으로 설정하면 캘린더의 일정을 내보내기 기능으로 백업할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 ' 탭을 선택합니다.
3. '일정 내보내기' 항목 오른쪽  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
4. '일정 내보내기'를 켭니다.
5. 오른쪽 상단의 '저장'을 눌러 선택값을 저장합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 ' 탭을 선택합니다.
3. '외부 공유' 항목 오른쪽  ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.

5. 오른쪽 상단의 '저장'을 누릅니다.
