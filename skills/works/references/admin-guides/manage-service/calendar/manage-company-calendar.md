# 회사 캘린더 관리

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

  
회사 캘린더에 일정을 등록하고 관리할 수 있습니다.

회사 캘린더는 회사에 소속된 구성원 모두에게 노출되는 캘린더로 구성원 모두가 알아야 하는 일정을 등록해 사용할 수 있습니다.

회사 캘린더에는 관리자만 일정을 등록할 수 있으며 [사용 설정](/ko/admin-guides/manage-service/calendar/calendar-settings/#i-3) 상태가 '사용'일 때에만 구성원에게 노출됩니다.

# 회사 캘린더 일정 확인

회사 캘린더에 등록 일정을 연도 또는 월 별로 볼 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '회사 캘린더 '를 선택합니다.
3. 왼쪽 상단에서 확인하고 싶은 연도나 월을 눌러 일정 목록을 확인합니다.
4. 상세 정보를 확인하고 싶은 일정이 있다면 일정 목록에서 누릅니다.
5. 선택한 일정의 상세 일정 정보를 확인합니다.

# 회사 캘린더 일정 쓰기

회사 캘린더에 일정을 작성할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '회사 캘린더 '를 선택합니다.
3. 오른쪽 상단의 '일정 쓰기 '를 누릅니다.
4. 일정의 [제목, 종일 일정으로 설정 여부, 일시, 장소, 메모](/ko/use-guides/calendar/create-event/new-event/) 등을 입력합니다.
5. 오른쪽의 '저장'을 눌러 일정을 저장합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
