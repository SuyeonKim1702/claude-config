# 캘린더 관리

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

구성원이 공유 받은 다른 사람의 기본 캘린더, 공유 캘린더, 구독 캘린더를 관리할 수 있습니다.

관리자는 구성원의 이름으로 해당 구성원이 포함된 캘린더를 찾고, 캘린더 멤버 관리를 하거나 공유 캘린더를 삭제합니다.

# 관리할 캘린더 목록 찾기

[캘린더를 공유](/ko/use-guides/calendar/calendar/share-calendar/)받은 구성원의 이름을 이용하여 캘린더를 찾을 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '캘린더 관리'를 선택합니다.
3. 검색창에 구성원 이름을 입력해 구성원이 포함된 캘린더 목록을 불러옵니다.

# 캘린더 멤버 관리

관리자가 캘린더 멤버의 권한을 변경하거나 캘린더 공유 멤버를 추가할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '캘린더 관리'를 선택합니다.
3. 검색 창에 구성원 이름을 입력해 불러온 캘린더 목록에서 '캘린더 설정'을 누릅니다.
4. '캘린더 설정' 창에서 멤버의 권한을 변경 하거나, 구성원을 검색하여 멤버로 추가합니다.
5. 하단의 '저장'을 눌러 변경 사항을 저장합니다.

# 캘린더 삭제

캘린더를 관리자가 삭제할 수 있습니다. 캘린더를 삭제하면 참여자가 모두 탈퇴 처리되고, 캘린더에 등록한 모든 일정이 삭제되어 복구가 불가능합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '캘린더'를 눌러 '캘린더'로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '캘린더 관리'를 선택합니다.
3. 검색 창에 구성원 이름을 입력해 불러온 캘린더 목록에서 '캘린더 설정'을 누릅니다.
4. '캘린더 설정' 창 하단에 있는 '삭제'를 누릅니다.
   - 공유 캘린더나 구독 캘린더인 경우에만 '삭제' 버튼 노출
   - 기본 캘린더인 경우에는 '삭제' 버튼 노출되지 않음
5. '확인'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
