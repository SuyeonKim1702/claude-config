# 공간 · 업무 기기 · 차량

이 기능은 자원 예약 옵션 상품 신청 후 사용 가능합니다.

  
자원의 예약 현황을 조회하고 관리자가 대신하여 공간, 업무 기기, 차량 자원을 예약하거나 예약 내역을 수정 또는 취소할 수 있습니다.

# 공간

회의실 예약 내역을 조회하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 조회할 날짜를 선택합니다.
   - 기본적으로 오늘 날짜가 표시됩니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_calendar.png)을 클릭하여 조회할 날짜를 선택하거나, 좌/우 화살표를 클릭하여 조회할 수 있습니다.
5. 조회할 회의실의 사옥, 위치, 정원수를 선택합니다.
   - 위치가 등록된 경우에만 위치를 선택할 수 있습니다. 위치 등록은 [위치 관리](/ko/admin-guides/manage-service/manage-reservation/manage-location/)를 참고해 주십시오.
6. 회의실의 예약 현황이 타임테이블에 조회됩니다.
   - '예약 중' 시간대에 마우스를 올리거나 클릭하면 예약 정보를 확인할 수 있습니다.
   - 자세한 예약 내용은 '상세 보기'를 클릭하여 확인합니다.
     - 예약 상세 내용이 조회됩니다.
     - 예약 내용의 수정이 필요한 경우 [회의실 예약 수정](#i-1-2), 예약 취소는 [회의실 예약 취소](#i-1-1)를 참고해 주십시오.
   - 예약의 수정 및 삭제는 해당 자원에 대한 '예약 운영 권한'이 있는 관리자만 가능합니다.
   - 캘린더에서 해당 회의실의 예약 현황을 조회하려면 회의실명에 마우스를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_calendar.png)을 클릭하면 조회할 수 있습니다.

## 회의실 예약 취소

관리자가 회의실 예약을 취소하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 예약을 취소할 날짜와 사옥, 위치 및 회의실명을 선택합니다.
   - 위치가 등록된 경우에만 위치를 선택할 수 있습니다. 위치 등록은 [위치 관리](/ko/admin-guides/manage-service/manage-reservation/manage-location/)를 참고해 주십시오.
5. 조회된 회의실의 예약 내역 중 취소할 예약을 클릭합니다.
6. 회의실 예약 정보 팝업에서 '예약 취소'를 클릭합니다.
   - 회의실 다중 예약 취소는 예약 상세 정보 팝업에서 '전체 취소' 또는 '예약 취소'를 클릭합니다. 전체 취소의 경우 오늘 이후의 예약을 전체 취소하며, 예약 취소의 경우 해당 일의 예약만 취소합니다.
7. 팝업에서 '확인'을 클릭합니다.
   - 예약 내역이 취소됩니다.

## 회의실 예약 수정

관리자가 회의실 예약을 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 예약을 수정할 날짜와 사옥, 위치 및 정원수를 선택합니다.
   - 위치가 등록된 경우에만 위치를 선택할 수 있습니다. 위치 등록은 [위치 관리](/ko/admin-guides/manage-service/manage-reservation/manage-location/)를 참고해 주십시오.
5. 조회된 회의실의 예약 내역 중 수정할 예약을 클릭합니다.
6. 회의실 예약 정보 팝업에서 '예약 수정'을 클릭합니다.
7. 수정할 내용을 적용합니다.
   - 예약 일시를 변경하려면 변경할 시간을 선택합니다. 해당 회의실의 다른 예약과 중복되는 일시로 변경할 수 없습니다.
   - 회의실을 변경하려면 '회의실 변경'을 클릭합니다.
     - 회의실 변경 팝업에서 변경할 회의실을 선택합니다. 변경할 회의실에 동일 예약 일시의 다른 예약이 존재하는 경우 회의실을 변경할 수 없습니다.
     - '변경'을 클릭합니다.
8. 회의실 예약 수정 팝업에서 '수정'을 클릭합니다.
   - 예약 내역이 수정됩니다.

## 관리자가 대신하여 회의실 예약

관리자가 회의 요청자를 대신하여 회의실을 예약하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 예약할 날짜를 선택합니다.
   - 기본적으로 오늘 날짜가 표시됩니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_calendar.png)을 클릭하여 예약할 날짜를 선택하거나, 좌/우 화살표를 클릭하여 원하는 날짜를 선택합니다.
   - 오늘부터 관리자가 설정한 예약 최대 가능일까지 선택할 수 있습니다.
   - 자원별 사용 제한일은 선택할 수 없습니다.
5. 사옥, 위치 및 정원수를 선택합니다.
   - 관리자가 위치를 등록한 경우에만 위치를 선택할 수 있습니다.
   - 조건에 맞는 회의실이 조회됩니다.
6. 예약할 회의실의 타임테이블에서 예약 가능 시간 중 예약할 시간을 클릭합니다.
   - '예약 중' 시간대는 예약할 수 없습니다. 해당 영역에 마우스를 올리면 예약 정보를 확인할 수 있습니다.
   - 현재 시간 이전 시간으로는 예약할 수 없습니다.
   - 자원 오픈 시간보다 이전 시간 또는 23시 59분 이후 시간으로 예약할 수 없습니다.
   - 일반 예약의 경우, 예약 제한 횟수가 설정된 회의실은 관리자가 설정한 예약 횟수 초과 시 예약할 수 없습니다.
7. 회의실 예약 팝업에서 예약 정보를 입력합니다.
   - 일반 예약의 경우 [회의실 일반 예약](/ko/use-guides/resource-reserve/pc-web/reservation-use/#i-1-1)을 참고하여 '일반 예약' 탭에 입력, 장기 예약이나 반복되는 요일 예약 등 다중 예약의 경우 [회의실 다중 예약](/ko/use-guides/resource-reserve/pc-web/reservation-use/#i-1-1)을 참고하여 '다중 예약' 탭에 정보를 입력합니다.
   - 메모를 입력합니다.
     - 관리자가 등록한 예약 메모가 있는 경우 표시됩니다.
   - 회의 요청자를 검색하여 입력합니다.
     - 요청자를 입력하지 않을 경우 대신 예약하는 관리자의 이름으로 예약됩니다.
     - 회의실 예약 완료 시 요청자에게 안내 메일이 발송됩니다.
   - '예약'을 클릭합니다.

# 업무 기기

업무 기기 예약 내역을 조회하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 조회할 날짜와 사옥을 선택합니다.
   - 기본적으로 오늘 날짜가 표시됩니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_calendar.png)을 클릭하여 조회할 날짜를 선택하거나, 좌/우 화살표를 클릭하여 조회할 수 있습니다.
   - 오늘부터 관리자가 설정한 예약 최대 가능일까지 선택할 수 있습니다.
   - 자원별 사용 제한일은 선택할 수 없습니다.
5. 사옥을 선택합니다.
   - 조건에 맞는 업무 기기 중 사용 권한이 있는 기기만 조회됩니다.
6. 예약할 업무 기기의 타임테이블에서 예약 가능 시간 중 예약할 시간을 클릭합니다.
   - '예약 중' 시간대는 예약할 수 없습니다. 해당 영역에 마우스를 올리면 예약 정보를 확인할 수 있습니다.
     - 예약 내용의 수정이 필요한 경우 [업무 기기 예약 수정](#i-2-2), 예약 취소는 [업무기기 예약 취소](#i-2-1)를 참고해 주십시오.
   - 해당 업무 기기의 예약 현황을 캘린더에서 조회하려면 업무 기기명에 마우스를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_calendar.png)을 클릭하면 조회할 수 있습니다.

## 업무 기기 예약 취소

관리자가 업무 기기 예약을 취소하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 예약을 취소할 날짜와 사옥을 선택합니다.
5. 조회된 업무 기기의 예약 내역 중 취소할 예약을 클릭합니다.
6. 업무 기기 예약 정보 팝업에서 '예약 취소'를 클릭합니다.
   - 다중 예약된 업무 기기 취소의 경우 예약 상세 정보 팝업에서 '전체 취소' 또는 '예약 취소'를 클릭합니다. 전체 취소의 경우 오늘 이후의 예약을 전체 취소하며, 예약 취소의 경우 해당 일의 예약만 취소합니다.
7. 팝업에서 '확인'을 클릭합니다.
   - 예약 내역이 취소됩니다.

## 업무 기기 예약 수정

관리자가 업무 기기 예약을 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '업무 기기'의 기기 종류를 클릭합니다.
4. 예약을 수정할 날짜와 사옥을 선택합니다.
5. 조회된 업무 기기의 예약 내역 중 수정할 예약을 클릭합니다.
6. 업무 기기의 예약 정보 팝업에서 '예약 수정'을 클릭합니다.
7. 수정할 내용을 적용합니다.
   - 예약 내용을 변경하려면 변경할 내용을 입력합니다.
   - 예약 일시를 변경하려면 변경할 시간을 선택합니다.
   - 업무 기기를 변경하려면 '업무 기기 변경'을 클릭합니다. (업무 기기의 종류에 따라 버튼명이 다르게 표시될 수 있습니다.)
     - 업무 기기 변경 팝업에서 변경할 업무 기기를 선택합니다.
     - 이미 예약 중인 업무 기기로 변경할 수 없습니다.
     - '변경'을 클릭합니다.
8. 업무 기기 예약 수정 팝업에서 '수정'을 클릭합니다.
   - 예약 내역이 수정됩니다.

## 관리자가 대신하여 업무 기기 예약

     - 자원 관리자가 등록한 예약 메모가 있는 경우 표시됩니다.
   - 예약 요청자를 검색하여 입력합니다.
   - '예약'을 클릭합니다.

# 차량

차량 예약 내역을 조회하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 조회할 날짜와 사옥을 선택합니다.
   - 기본적으로 오늘 날짜가 표시됩니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_calendar.png)을 클릭하여 조회할 날짜를 선택하거나, 좌/우 화살표를 클릭하여 조회할 수 있습니다.
5. 차량의 예약 내역이 타임테이블에 조회됩니다.
   - '예약 중' 시간대에 마우스를 올리거나 클릭하여 예약 정보를 확인할 수 있습니다. 자세한 예약 내용은 회의실 예약 정보 팝업에서 '상세 보기'를 클릭합니다.
     - 차량의 예약 상세 내용이 조회됩니다.
     - 예약 내용 수정은 [예약 수정](#i-4-4), 예약 취소는 [예약 취소](#i-4-5)를 참고해 주십시오.
   - 해당 차량의 예약 현황을 캘린더에서 조회하려면 차량명에 마우스를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_calendar.png)을 클릭하면 조회할 수 있습니다.

## 차량 예약 취소

관리자가 차량 예약을 취소하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 예약을 취소할 날짜와 사옥을 선택합니다.
5. 조회된 차량의 예약 내역 중 취소할 예약을 클릭합니다.
6. 차량 예약 정보 팝업에서 '예약 취소'를 클릭합니다.
7. 팝업에서 '확인'을 클릭합니다.
   - 예약 내역이 취소됩니다.

## 차량 예약 수정

관리자가 차량 예약을 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 예약을 수정할 날짜와 사옥을 선택합니다.
5. 조회된 차량의 예약 내역 중 수정할 예약을 클릭합니다.
6. 차량 예약 정보 팝업에서 '예약 수정'을 클릭합니다.
7. 수정할 내용을 적용합니다.
   - 예약 일시를 변경하려면 변경할 시간을 선택합니다.
   - '변경'을 클릭합니다.
8. 차량 예약 수정 팝업에서 '수정'을 클릭합니다.
   - 예약 내역이 수정됩니다.

## 관리자가 대신하여 차량 예약

관리자가 차량 예약 요청자를 대신하여 업무 기기를 예약하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 예약할 날짜를 선택합니다.
   - 기본적으로 오늘 날짜가 표시됩니다.
   - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_calendar.png)을 클릭하여 예약할 날짜를 선택하거나, 좌우 화살표를 클릭하여 원하는 날짜를 선택합니다.
   - 오늘부터 관리자가 설정한 예약 최대 가능일까지 선택할 수 있습니다.
   - 관리자가 설정한 자원별 사용 제한일은 선택할 수 없습니다.
   - 사옥을 선택합니다.
     - 조건에 맞는 차량이 조회됩니다.
5. 예약할 차량의 타임테이블에서 예약 가능 시간 중 예약할 시간을 클릭합니다.
   - '예약 중' 시간대는 예약할 수 없습니다. 해당 영역에 마우스를 올리면 예약 정보를 확인할 수 있습니다.
   - 현재 시간 이전 시간으로는 예약할 수 없습니다.
   - 자원 오픈 시간보다 이전 시간 또는 23시 59분 이후 시간으로 예약할 수 없습니다.
   - 예약 제한 횟수가 설정된 차량은 관리자가 설정한 예약 횟수 초과 시 예약할 수 없습니다.
6. 차량 예약 팝업에서 예약 정보를 입력합니다.
   - [차량 예약](/ko/use-guides/resource-reserve/pc-web/reservation-use/#i-3-1)을 참고하여 '일반 예약'에 입력합니다.
   - 메모를 입력합니다.
     - 자원 관리자가 등록한 예약 메모가 있는 경우 표시됩니다.
   - 예약 요청자를 검색하여 입력합니다.
   - '예약'을 클릭합니다.
