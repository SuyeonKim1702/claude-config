# 예약 조회

이 기능은 자원 예약 옵션 상품 신청 후 사용 가능합니다.

  
자원의 예약 현황을 조회할 수 있습니다. 또한 조건에 따라 예약 현황을 검색하거나 예약 목록을 엑셀 파일로 다운로드할 수 있습니다.

예약 조회 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 예약 현황을 예약 기간, 사옥명, 위치, 타입, 서비스, 자원, 예약자 또는 로그인 아이디 조건에 따라 검색 |
| ② 다운로드 | 예약 현황 목록을 엑셀 파일로 다운로드([예약 현황 목록 다운로드](#i-1) 참고) |
| ③ 자원 예약 목록 | 자원 예약 목록 표시  - 예약자: 클릭하여 예약자 정보 조회 - GMT 정보: 자원이 속한 사옥의 GMT 정보 표시 - 구분: 일반 또는 다중 예약 |

# 예약 목록 다운로드

예약 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '예약 조회'를 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 예약 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 예약 현황이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 예약 현황 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Downloads` 폴더에 `reserve-YYMMDD_HHMMSS.xlsx` 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
