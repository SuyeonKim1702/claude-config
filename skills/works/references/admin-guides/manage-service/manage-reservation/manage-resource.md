# 자원 관리

이 기능은 자원 예약 옵션 상품 신청 후 사용 가능합니다.

  
회사의 공용 자원을 등록하고 정보를 관리할 수 있습니다. 또한, 조건에 따라 자원을 검색해 조회할 수 있으며, 자원 목록을 엑셀 파일로 다운로드할 수 있습니다.

# 자원 관리

자원 관리 메뉴는 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 사옥, 위치, 사용 여부, 자원 타입, 서비스에 따라 검색 |
| ② 자원 등록 | 신규 자원 추가 |
| ③ 다운로드 | 자원 목록을 엑셀 파일로 다운로드 |
| ④ 자원 목록 | 등록된 자원 정보 표시 |
| ⑤ 수정 | 클릭하여 자원 정보 조회 및 수정 |
| ⑥ 복사 | 자원 정보를 복사하여 추가 |

## 자원 추가

자원을 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '자원 관리'를 클릭합니다.
4. '자원 등록'을 클릭합니다.
5. 자원의 기본 정보를 입력합니다.
   - 사옥을 선택합니다. 사옥 등록은 [사옥 관리](/ko/admin-guides/manage-service/manage-reservation/manage-office/)를 참고해 주십시오.
   - 필요할 경우, 자원 위치를 선택합니다. 위치 등록은 [위치 관리](/ko/admin-guides/manage-service/manage-reservation/manage-location/)를 참고해 주십시오.
   - 자원 타입을 선택합니다.
     - 자원 타입은 공간, 업무 기기 및 차량이 제공됩니다.
   - 선택한 자원 타입에 대한 상세 서비스를 선택합니다.
   - 자원명을 입력합니다.
   - 자원에 대한 설명 문구를 입력합니다. 사용자가 자원명에 마우스를 올리면 표시되는 설명 문구입니다.
   - 예약 메모를 입력합니다. 사용자 자원 예약 화면의 '메모' 영역에 표시될 문구입니다. 사용자가 입력할 메모의 양식이나 예시를 안내할 수 있습니다.
   - 자원 목록에 노출될 우선 순위를 입력합니다.
   - 자원의 사용 여부를 선택합니다.
     - 자원을 사용으로 설정하더라도, 자원의 사용 권한을 부여 받은 사용자만 예약할 수 있습니다.
   - 예약 내용 공개 여부를 선택합니다.
     - '공개 안 함'으로 설정 시 사원은 자원 예약 홈 및 공간, 업무기기, 차량의 예약 화면에서 이미 점유된 시간만 확인할 수 있으며 예약 상세 정보를 조회할 수 없습니다.
6. 자원의 예약 정보를 입력합니다.
   - 다중 예약을 '사용'으로 설정 시 여러 날짜에 걸친 장기 예약이나 요일별 반복 예약을 적용할 수 있습니다.
     - 자원 타입이 차량인 경우 다중 예약 사용은 비활성화로 표시됩니다.
   - 예약 가능 시간을 입력합니다. 사용자는 매일 해당 시간에 예약할 수 있습니다.
   - 예약 제한 횟수 사용 여부를 선택합니다.
     - 사용자가 하나의 자원을 예약할 수 있는 횟수를 의미하며, 예약 제한 횟수를 사용으로 선택 시 일/주/월 단위로 설정할 수 있습니다.
     - 다중 예약을 사용하는 경우 예약 제한 횟수 사용 설정을 할 수 없습니다.
   - 예약 기준 시간을 입력합니다.
     - 자원 예약의 시간 단위입니다. 예약 기준 시간을 20분으로 설정한 경우, 20분, 40분, 60분 등 20분을 단위로 예약할 수 있습니다. 단 최소 예약 가능 시간보다 작은 시간으로 예약할 수 없습니다.
   - 자원 최소 예약 가능 시간 및 최대 예약 가능 시간을 설정합니다.
     - 자원 최소 예약 가능 시간, 최대 예약 가능 시간은 해당 자원을 1회 신청 시 사용할 수 있는 최소 또는 최대 예약 시간을 의미합니다.
   - 자원 사용 가능 시간 중 회의실 또는 기기 수리 등으로 특정 기간 동안 예약을 제한할 경우, 자원 사용 제한 여부를 사용으로 설정할 수 있습니다.
     - 자원 사용을 제한할 경우 '추가'를 클릭 후 제한 시작일 및 종료일과 시간을 선택합니다.
     - 필요 시 안내 문구를 입력합니다.
7. 선택한 자원 타입에 따른 정보를 입력합니다.
   - 회의실은 정원 수를 입력합니다.
   - 업무기기는 대표 이미지를 등록합니다.
   - 차량은 해당 영역이 표시되지 않습니다.
8. 자원의 사용/관리/예약 운영/예약 메일 알림 권한을 부여합니다.
   - 자원 사용 권한: 자원을 사용할 수 있는 권한
   - 자원 관리 권한: 자원 관리, 위치 관리, 사옥 관리 가능한 권한
   - 예약 운영 권한: 자원의 예약 현황을 조회, 관리자 모드로 사원을 대신하여 예약/수정 가능한 권한
   - 예약 메일 알림: 자원이 예약된 경우 메일 알림을 수신할 담당자
   - 사원, 부서, 그룹 단위로 권한을 부여할 수 있습니다.
   - 그룹으로 부여 시 '그룹 관리'를 클릭하여 조회되는 그룹을 선택하거나 신규 그룹을 생성 후 부여할 수 있습니다. 추가한 권한 그룹은 타 서비스에서도 활용할 수 있습니다.  

9. 필요할 경우, 기타 정보를 입력합니다.
   - 자원의 관리 목적으로 메모를 작성할 수 있습니다.
   - 관리자 메모는 사용자에게 노출되지 않습니다.
10. '저장'을 클릭합니다.
    - 자원의 추가를 취소하려면 '취소'를 클릭합니다.

## 자원 정보 조회/수정

자원 목록에서 조회할 자원의 '수정'을 클릭하면 상세 화면에 자원 상세 정보가 표시됩니다. 이 화면에서 자원의 정보를 조회하고 수정하거나 초기화할 수 있습니다.

1. 조회 화면에서 수정할 내용을 적용합니다. '자원 추가'를 참고해 주십시오.
   - 조회 시 자원 서비스별 예약 개수가 표시됩니다.
   - 사옥, 위치, 타입, 서비스는 수정할 수 없습니다.
2. '저장'을 클릭합니다.
   - 수정된 내용이 저장됩니다.
   - 수정된 내용을 초기화하려면 '초기화'를 클릭합니다.
   - 사용 여부가 '미사용'으로 설정된 자원 또는 예약된 내역이 없는 자원은 '삭제'가 표시됩니다. 필요하지 않은 자원은 클릭하여 삭제할 수 있습니다.

## 자원을 복사하여 추가

등록한 자원 정보를 복사하여 자원을 빠르게 등록할 수 있습니다.

1. 자원 목록에서 복사할 자원의 '복사'를 클릭합니다.
   - 복사된 자원 정보와 동일한 정보가 반영된 자원 추가 화면으로 이동합니다.
2. [자원 추가](#i-1-1)를 참고하여 정보를 수정하거나 나머지 정보를 입력합니다.
3. '저장'을 클릭합니다.
   - 자원이 등록됩니다.

## 자원 목록 다운로드

자원 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '자원 관리'를 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 자원 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 자원이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 자원 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Downloads` 폴더에 `resource_YYYYMMDD_HHMMSS.xlsx` 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
