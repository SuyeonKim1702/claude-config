# 자원 예약 관리 메인

이 기능은 자원 예약 옵션 상품 신청 후 사용 가능합니다.

  
사옥, 위치 및 자원의 등록 현황을 확인할 수 있으며, 자원 예약 서비스 이용에 필요한 설정 사항들을 관리할 수 있습니다. 또한, 자원 예약 관리 메뉴 내 기록된 로그 내역을 확인할 수 있습니다.

자원 예약 관리 메인을 사용하려면 자원 예약 서비스에서 화면 왼쪽 상단의 '자원 예약 관리'를 클릭합니다.

자원 예약 관리 메인 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 콘텐츠 영역에 표시할 콘텐츠를 선택  - [Timeline](#i-2): 최근 한 달 동안 예약 서비스 내에서 기록된 로그를 100건까지 최신 순으로 표시 - [Getting Started](/ko/admin-guides/manage-service/manage-reservation/reservation-management-home/#i): 자원 예약 관리에 필요한 설정 항목 표시 |
| ② 콘텐츠 영역 | - 선택된 콘텐츠에 따라 사옥, 위치 및 자원의 등록 현황, 로그 내역 표시 - 카드 형태로 표시되는 항목 또는 로그 내역을 클릭하면 해당 메뉴로 이동 |

# Getting Started

자원 예약 서비스에 처음으로 접속하면 예약 관련 정보 설정을 안내하는 Getting Started 화면이 나타납니다. 각 설정 항목은 닫힌 카드 형태로 표시되며, 카드를 한 번 클릭하면 카드가 열려 기본 반영 사항이 표시되고, 한 번 더 클릭하면 해당 메뉴로 이동합니다.  
자원 예약 서비스를 원활히 사용하기 위해서는 Getting Started 화면의 카드 순서에 따라 모든 항목의 설정을 완료합니다.

| 영역 | 설명 |
| --- | --- |
| ① 사옥 관리 | 사옥을 추가하고 관리할 수 있는 [사옥 관리](/ko/admin-guides/manage-service/manage-reservation/manage-office/) 메뉴로 이동 |
| ② 위치 관리 | 위치를 추가하고 관리할 수 있는 [위치 관리](/ko/admin-guides/manage-service/manage-reservation/manage-location/) 메뉴로 이동 |
| ③ 자원 관리 | 자원을 추가하고 관리할 수 있는 [자원 관리](/ko/admin-guides/manage-service/manage-reservation/manage-resource/) 메뉴로 이동 |

# Timeline

자원 예약 관리의 최근 한 달간 사용 로그가 100건까지 최신 순으로 표시됩니다. 각 항목의 링크를 클릭하면 관련 메뉴로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
