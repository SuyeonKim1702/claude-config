# 서비스 개요

이 기능은 자원 예약 옵션 상품 신청 후 사용 가능합니다.

  
자원 예약 관리에서는 회사의 공용 자원을 편리하게 예약할 수 있도록 자원을 등록하고 예약 현황을 관리할 수 있습니다.

자원 예약 관리를 사용하려면 PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.

- 자원 예약 관리는 관리자 권한이 있는 사용자에게만 노출됩니다.

# 자원 예약 관리자 시나리오

자원 예약 관리자의 이용 시나리오는 다음과 같습니다.

1. 자원의 예약 현황 조회
   - 예약 조회
   - 이 단계에 대한 사용 가이드 페이지
     - [예약 조회](/ko/admin-guides/manage-service/manage-reservation/view-reservation/)
2. 자원 관리
   - 회사의 공용 자원을 추가, 조회, 수정, 삭제
   - 이 단계에 대한 사용 가이드 페이지
     - [자원 관리](/ko/admin-guides/manage-service/manage-reservation/manage-resource/)
3. 사옥, 위치 관리
   - 사옥, 위치 추가, 조회, 수정, 삭제
   - 이 단계에 대한 사용 가이드 페이지
   - [위치 관리](/ko/admin-guides/manage-service/manage-reservation/manage-location/)
   - [사옥 관리](/ko/admin-guides/manage-service/manage-reservation/manage-office/)
4. 공간, 업무 기기, 차량 관리
   - 공간, 업무 기기, 차량 등 자원 예약 관리
   - 이 단계에 대한 사용 가이드 페이지
   - [공간, 업무 기기, 차량 관리](/ko/admin-guides/manage-service/manage-reservation/reserve/)

# **자주 사용하는 질문을 확인하세요.**

**Q. 예약 서비스는 누구나 이용할 수 있나요?**

- 자원 예약 서비스를 신청한 고객은 예약 서비스를 사용할 수 있습니다. 자원 등록 개수/사용 횟수에 제한 없이 편리하게 자원을 예약하고 운영할 수 있습니다.

**Q. 회사 외부에서도 자원을 예약할 수 있나요?**

- 모바일 앱에서 자원을 예약할 수 있는 기능이 제공됩니다. 모바일 환경에서 언제 어디서나 편리하게 자원을 예약, 변경 또는 취소할 수 있습니다.

**Q. 예약 정보를 다른 서비스에서 확인할 수 있나요?**

- 외부 솔루션을 연동하여 외부 메시지 솔루션에서 예약 알림을 수신하거나 캘린더에 등록할 수 있습니다. 구글챗, Slack 알림 또는 구글 캘린더 등 다양한 방법으로 개인화된 알림을 최적화 하여 사용할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
