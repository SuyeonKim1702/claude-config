# 사옥 관리

이 기능은 자원 예약 옵션 상품 신청 후 사용 가능합니다.

  
사옥을 등록하고 관리할 수 있습니다. 또한, 사옥 목록을 엑셀 파일로 다운로드할 수 있습니다. 사옥이 등록된 후 자원을 등록할 수 있으므로, 자원을 등록하기 전에 사옥을 먼저 등록해 주십시오.

사옥 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색 | 사옥명, 사용 여부 조건으로 사옥 검색 |
| ② 사옥 등록 | 새로운 사옥을 추가([사옥 추가](#i-1) 참고) |
| ③ 다운로드 | 사옥 목록을 엑셀 파일로 다운로드 |
| ④ 사옥 목록 | 사옥 목록 표시  - 사옥명을 클릭하여 사옥 정보 수정([사옥 조회/수정](#i-2) 참고) - 자원 추가 시 등록한 타임존이 회사 타임존과 다른 경우, 사옥명 옆에 타임존 표시 |

# 사옥 추가

사옥을 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '사옥 관리'를 클릭합니다.
4. '사옥 등록'을 클릭합니다.
5. 등록할 사옥 정보를 입력합니다.
   - 사옥명을 입력합니다.
   - 국가를 입력합니다.
   - 타임존을 선택합니다.
   - 목록에서 표시될 우선 순위를 입력합니다.
   - 사용 여부를 선택합니다.
6. '저장'을 클릭합니다.
   - 사옥이 추가됩니다.

# 사옥 수정

사옥 정보를 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '사옥 관리'를 클릭합니다.
4. 조회할 사옥명을 클릭합니다.
5. 팝업에서 수정할 내용을 적용합니다.
6. '수정'을 클릭합니다.
   - 사옥 정보가 수정됩니다.

사옥에 자원 예약이 존재하는 경우, 사옥명/국가/타임존 정보는 수정할 수 없습니다. 예약 내역이 있는 사옥은 우선 순위, 사용 여부만 수정할 수 있습니다.

# 사옥 삭제

사옥 정보를 삭제하는 방법은 다음과 같습니다.

미사용 사옥만 '삭제'가 표시됩니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '자원 예약 관리'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '사옥 관리'를 클릭합니다.
4. 삭제할 사옥명을 클릭합니다.
5. 팝업에서 '삭제'를 클릭합니다.
   - 사옥 정보가 삭제됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
