# 휴폐업 거래처 조회

이 기능은 전자 세금 계산서 옵션 상품 신청 후 사용 가능합니다.

  
거래처의 휴폐업 상태를 조회할 수 있습니다.

휴폐업 거래처 조회 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '휴페업 거래처 조회'를 클릭합니다.
4. 조회할 거래처의 사업자등록번호를 입력합니다.
5. '검색'을 클릭합니다
   - 거래처의 휴폐업 조회 결과가 표시됩니다. 휴폐업 거래처인 경우 휴폐업 일자가 조회됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
