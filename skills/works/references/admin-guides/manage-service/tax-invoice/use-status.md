# 사용 현황

이 기능은 전자 세금 계산서 옵션 상품 신청 후 사용 가능합니다.

  
전자세금계산서의 신청 상태, 해당 월의 이용 건수 및 사용 내역을 확인할 수 있습니다.

- 전자세금계산서 서비스는 네이버웍스 결재 상품을 유료로 사용하는 경우에만 신청 가능합니다.
- 전자세금계산서 발행 건 당 100원이며, 이용 요금은 네이버웍스 이용 요금과 합산하여 월별 청구됩니다.
  - 과금 기준: 발행된 전자세금계산서는 건당 요금이 부과되며, 발행 취소한 건도 과금 대상에 포함됩니다.

# 사용 관리

전자세금계산서 요금 및 신청 내용을 확인하고 월 이용 건수를 조회할 수 있습니다. 사용하지 않을 경우 사용을 해지할 수 있습니다.

사용 관리 탭 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 전자세금계산서 요금 안내 | - 전자세금계산서 사용 요금 정보 표시 - 네이버클라우드의 '요금 계산 바로가기'에서 청구 예상 요금 확인 가능 |
| ② 전자세금계산서 신청 내용 | - 전자세금계산서 신청 : 신청 완료 상태 표시, '신청/해지하러 가기'를 클릭하여 해지([이용 해지](#i-2-1) 참고) - 신청 일시: 전자세금계산서 신청 일시 표시 |
| ③ 전자세금계산서 이용 내용 | 연/월별 사용 건수 표시 |

# 사용 현황

전자세금계산서 사용 내역을 확인하거나 전자세금계산서 사용 현황을 엑셀 파일로 다운로드할 수 있습니다.

사용 현황 탭 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 전자세금계산서 사용 내역을 사업장, 연/월, 사용자, 관리번호에 따라 검색 |
| ② 다운로드 | 전자세금계산서 사용 내역을 엑셀 파일로 다운로드([사용 내역 다운로드](#i-2-2) 참고) |
| ③ 전자세금계산서 사용 목록 | 전자세금계산서 사용 목록 표시  - 사업장: 발행 사업장 표시 - 서비스: 발행 서비스 표시, 전자세금계산서 서비스 내 발행의 경우 표시 안 함 - 사용자: 발행한 사용자 표시 - 관리 번호: 클릭하여 계산서 상세 내용 확인 |
| ④ 표시 항목 수 | 목록의 한 페이지에 표시할 계산서 수 설정 |

## 이용 해지

전자세금계산서 이용을 해지하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '사용 현황'을 클릭합니다.
4. '신청/해지하러 가기'를 클릭합니다.
   - 전자세금계산서 서비스를 해지하기 전 다음 절차를 먼저 진행해 주십시오.

- 전자세금계산서를 '사용 안 함'으로 변경하는 겨우 기존 데이터가 삭제되지 않습니다. 추후 다시 '사용'으로 설정 변경 시 과거 데이터를 조회 할 수 있습니다.
- 전자세금계산서 서비스를 사용하지 않더라도 네이버웍스 전자세금계산서 서비스에서 발행한 내역은 스마트빌 홈페이지에 접속해서 확인할 수 있습니다.

## 사용 내역 다운로드

전자세금계산서 사용 현황을 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '사용 현황'을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 전자세금계산서 사용 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 전자세금계산서 사용 현황이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 사용 현황이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Download` 폴더에 `EnvMeteringLog.xlsx` 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
