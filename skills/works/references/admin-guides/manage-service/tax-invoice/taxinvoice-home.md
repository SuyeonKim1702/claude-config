# 전자세금계산서 메인

이 기능은 전자 세금 계산서 옵션 상품 신청 후 사용 가능합니다.

  
전자세금계산서 메인에서는 전자세금계산서 환경 설정, 스마트빌 연동 설정 및 이달의 발행 내역, 작성 중인 세금계산서 등 정보를 통계 형태로 확인하고 전자세금계산서 서비스에 기록된 로그 내역을 확인할 수 있습니다. 또한 콘텐츠 영역에 표시되는 카드 또는 정보를 클릭하여 해당 기능을 바로 사용할 수 있습니다.

전자세금계산서 메인을 사용하려면 전자세금계산서 서비스에서 화면 왼쪽 상단의 '전자세금계산서'를 클릭합니다.

전자세금계산서 메인 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 콘텐츠 탭 | 콘텐츠 영역에 표시할 콘텐츠를 선택  - [Dashboard](#i-2): 이 달의 발행, 작성 중인 문서, 계정 과목 미지정 현황을 카드 형태로 표시 - [Timeline](#i-3): 최근 한 달 동안 전자세금계산서 서비스 내에서 기록된 로그를 100건까지 최신 순으로 표시 - [Getting Started](#i-1): 전자세금계산서 서비스 이용에 필요한 설정을 카드 형태로 표시 |
| ② 콘텐츠 영역 | - 선택된 콘텐츠에 따라 전자세금계산서 통계, 로그 내역, 전자세금계산서 서비스 관리 항목 표시 - 카드 형태로 표시되는 항목 또는 로그 내역을 클릭하면 해당 메뉴로 이동 |

# Getting Started

전자세금계산서 서비스에 처음으로 접속하면 전자세금계산서 관련 정보 설정을 안내하는 Getting Started 화면이 나타납니다. 각 설정 항목은 닫힌 카드 형태로 표시되며, 카드를 한 번 클릭하면 카드가 열려 기본 반영 사항이 표시되고, 한 번 더 클릭하면 해당 메뉴로 이동합니다.  
전자세금계산서 서비스를 원활히 사용하기 위해서는 Getting Started 화면의 카드 순서에 따라 모든 항목의 설정을 완료해 주십시오.

| 영역 | 설명 |
| --- | --- |
| ① 전자세금계산서 환경 설정 | 전자세금계산서 기본 환경을 설정하는 [전자세금계산서 설정](/ko/admin-guides/manage-service/tax-invoice/tax-settings/#i-1) 메뉴로 이동 |
| ② 스마트빌 연동 설정 | [스마트빌 연동 설정](/ko/admin-guides/manage-service/tax-invoice/tax-settings/#i-2) 메뉴로 이동하여 전자세금계산서 발행을 위해 스마트빌 계정을 연동, 네이버웍스 경영지원 설정의 인증서 전송 |

# Dashboard

Getting Started 화면의 모든 설정을 완료한 후에는, 전자세금계산서 서비스에 접속하면 Dashboard 화면이 나타납니다. Dashboard 화면에는 전자세금계산서 관련 요약 정보가 카드 형태로 표시됩니다.

Dashboard 화면에 표시되는 카드는 다음과 같습니다.

| 영역 | 설명 |
| --- | --- |
| ① 이 달의 발행 | - 이 달의 전자세금계산서 발행 건수 표시 - 클릭하면 [발행 내역 조회](/ko/admin-guides/manage-service/tax-invoice/issuance-manegement/#i-8) 메뉴로 이동 |
| ② 작성 중인 문서 | - 작성 중인 작성세금계산서 건수 표시 - 클릭하면 [신규 발행](/ko/admin-guides/manage-service/tax-invoice/issuance-manegement/#i-1) 메뉴로 이동 |
| ③ 계정 과목 미지정 | - 계정 과목이 미지정된 세금계산서 건수 표시 - 클릭하면 [계정 과목 지정](/ko/admin-guides/manage-service/tax-invoice/account-transfer/#i-1) 메뉴로 이동 |
| ④ 정발행 | - 발행 요청: 서비스별 발행 요청 건수 표시, 클릭하면 [발행 요청 관리](/ko/admin-guides/manage-service/tax-invoice/issuance-manegement/#i-7) 메뉴로 이동 - 이메일 전송: 이메일 전송 건수 표시, 클릭하면 [이메일 전송](/ko/admin-guides/manage-service/tax-invoice/transfer-management/#i-2) 메뉴로 이동 - 국세청 전송: 국세청 전송 건수 표시, 클릭하면 [국세청 전송 관리](/ko/admin-guides/manage-service/tax-invoice/transfer-management/#i-1) 메뉴로 이동 - 회계 전송: 회계 전송 건수 표시, 클릭하면 [회계 전송](/ko/admin-guides/manage-service/tax-invoice/account-transfer/#i-2) 메뉴로 이동 |
| ⑤ 역발행 | - 발행 요청: 서비스별 발행 요청 건수 표시, 클릭하면 [발행 요청 관리](/ko/admin-guides/manage-service/tax-invoice/issuance-manegement/#i-7) 메뉴로 이동 - 이메일 전송: 이메일 전송 건수 표시, 클릭하면 [이메일 전송](/ko/admin-guides/manage-service/tax-invoice/transfer-management/#i-3) 메뉴로 이동 - 국세청 전송: 국세청 전송 건수 표시, 클릭하면 [국세청 전송 관리](/ko/admin-guides/manage-service/tax-invoice/transfer-management/#i-1) 메뉴로 이동 - 회계 전송: 회계 전송 건수 표시, 클릭하면 [회계 전송](/ko/admin-guides/manage-service/tax-invoice/account-transfer/#i-4) 메뉴로 이동 |
| ⑥ 위수탁 | - 이메일 전송: 이메일 전송 건수 표시, 클릭하면 [이메일 전송](/ko/admin-guides/manage-service/tax-invoice/transfer-management/#i-3) 메뉴로 이동 - 국세청 전송: 국세청 전송 건수 표시, 클릭하면 [국세청 전송 관리](/ko/admin-guides/manage-service/tax-invoice/transfer-management/#i-1) 메뉴로 이동 |

# Timeline

전자세금계산서 서비스의 최근 한 달간 사용 로그가 100건까지 최신 순으로 표시됩니다. 각 항목의 링크를 클릭하면 관련 메뉴로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
