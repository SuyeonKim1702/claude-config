# 보고서

이 기능은 전자 세금 계산서 옵션 상품 신청 후 사용 가능합니다.

  
보고서에서는 사업장, 거래처 또는 기간별 발행한 매입/매출/위수탁 전자세금계산서를 조회할 수 있습니다.

# 매출/매입 합계표

사업장별 발행한 매출/매입 계산서의 합계 금액을 조회합니다.

매출/매입 합계표 메뉴에서는 발행 완료 및 국세청 전송 완료된 계산서에 대한 데이터만 집계합니다.(위수탁 발행 계산서 제외)

매출/매입 합계표 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 컨텐츠 탭 | 클릭하여 계산서 발행 형태별 내역 표시 |
| ② 검색창 | 계산서를 사업장, 관리 번호, 공급자/공급받는 자, 작성일 조건에 따라 검색 |
| ④ 다운로드 | 매입/매출 계산서 목록을 엑셀 파일로 다운로드 ([매입/매출 합계 다운로드](#i-1-1) 참고) |
| ⑤ 인쇄 | 매입/매출 계산서 합계표 인쇄 ([매입/매출 합계표 인쇄](#i-1-2) 참고) |
| ⑥ 매입/매출 내역 | 해당 사업장에서 발행한 거래처별 매입/매출 금액 및 합계 금액 표시  - 건수: 클릭하여 거래처별 매입/매출 계산서 조회 ([거래처별 조회](#i-2) 참고) |

## 매출/매입 합계 다운로드

사업장별 발행한 매입 또는 매출 계산서 합계 내역을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 전자세금계산서 목록이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 해당 사업장으로 발행한 매출/매입 계산서 합계 내역이 엑셀 파일로 다운로드됩니다.

- 다운로드된 파일은 `Download` 폴더에 `ReportSummarySheet.xlsx` 파일명으로 저장됩니다.

## 매출/매입 합계 인쇄

매입 또는 매출 합계 내역을 인쇄하는 방법은 다음과 같습니다.

매출/매입 합계표 인쇄 양식은 부가세 신고를 위한 국세청 '매입처별/매출처별 세금계산서 합계표' 신고 서류 양식과는 다릅니다. 내부 열람 목적으로 활용하시기 바랍니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 매입/매출 계산서 합계 목록이 조회됩니다.
5. '인쇄'를 클릭합니다.
   - 해당 사업장으로 발행한 매입/매출 계산서 합계 내역이 인쇄됩니다.

# 거래처별 조회

각 거래처별 발행한 매입/매출 계산서 및 합계 금액을 조회할 수 있습니다.

거래처별 조회 메뉴에서는 발행 완료 및 국세청 전송 완료된 계산서에 대한 데이터만 집계합니다.(위수탁 발행 계산서 제외)

거래처별 조회 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 컨텐츠 탭 | 클릭하여 계산서 발행 형태별 목록 표시 |
| ② 검색창 | 계산서를 사업장, 관리 번호, 공급자/공급받는 자, 작성일 조건에 따라 검색 |
| ④ 다운로드 | 거래처별 발행된 계산서 목록을 엑셀 파일로 다운로드 ([거래처별 계산서 목록 다운로드](#i-2-1) 참고) |
| ⑤ 계산서 목록 | 거래처별 계산서 목록 및 합계 금액 표시  - 공급자/공급받는 자: 클릭하여 거래처 정보 상세 조회 - 관리 번호: 클릭하여 계산서 상세 내역 조회 |

# 기간별 통합 조회

거래처별 매입/매출 및 위수탁 발행한 전자세금계산서를 기간별로 조회할 수 있습니다.

기간별 통합 조회 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 계산서를 사업장, 계산서 유형, 발행 형태, 기간에 따라 검색 |
| ② 다운로드 | 기간별 매입/매출/위수탁 발행 목록을 엑셀 파일로 다운로드 ([기간별 계산서 목록 다운로드](#i-3-1) 참고) |
| ③ 계산서 목록 | 조회한 기간에 대한 계산서 목록 및 합계 금액 표시  - 건수: 클릭하여 계산서 상세 내역 조회 |
