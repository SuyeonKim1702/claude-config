# 발행 관리

이 기능은 전자 세금 계산서 옵션 상품 신청 후 사용 가능합니다.

  
전자세금계산서를 발행 형태별로 추가할 수 있습니다. 추가한 전자세금계산서를 발행할 수 있습니다.

사업장 권한이 없는 경우 전자세금계산서 신규 발행이 불가능합니다. 관리자에게 사업장 권한 추가를 문의해 주십시오.([사업장 권한 관리](/ko/admin-guides/manage-service/tax-invoice/unit-authority/) 참고)

신규 발행 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 컨텐츠 탭 | 클릭하여 전자세금계산서 발행 형태별 목록 표시 |
| ② 검색창 | 계산서를 사업장, 계산서 유형, 관리 번호, 공급자/공급받는 자, 작성일 조건에 따라 검색 |
| ③ 추가 | 클릭하여 새로운 전자세금계산서 추가([전자세금계산서 추가](#i-1) 참고)  - 사업장 권한이 없는 경우 비활성화 |
| ④ 선택 발행 | 클릭하여 선택한 전자세금계산서 발행([전자세금계산서 발행](#i-4) 참고) |
| ⑤ 선택 삭제 | 클릭하여 선택한 전자세금계산서 삭제([전자세금계산서 삭제](#i-3) 참고) |
| ⑥ 업로드 | 전자세금계산서 업로드([전자세금계산서 업로드 발행](#i-5) 참고) |
| ⑦ 다운로드 | 전자세금계산서 목록을 엑셀 파일로 다운로드([전자세금계산서 목록 다운로드](#i-6) 참고) |
| ⑧ 전자세금계산서 목록 | 전자세금계산서 목록 표시  - 관리 번호: 클릭하여 전자세금계산서 내용 조회 - 공급받는 자/사업자등록번호: 정발행 및 위수탁 목록에 표시 - 공급자/사업자등록번호: 역발행 및 위수탁 목록에 표시 |
| ⑨ 표시 항목 수 | 목록의 한 페이지에 표시할 전자세금계산서 수 설정 |

# 신규 전자세금계산서 추가

전자세금계산서를 추가하는 방법은 다음과 같습니다.  

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 전자세금계산서 발행 형태에 따른 탭을 클릭합니다.
5. '추가'를 클릭합니다.
6. 사업장을 선택합니다.
   - 권한이 있는 사업장이 1개인 경우 자동으로 표시됩니다.
7. 과세 유형을 선택합니다.
   - '면세'를 선택한 경우 전자계산서로 표시됩니다.
8. 발행 형태를 선택합니다.
9. 발행 형태에 따라 공급자 영역, 공급받는 자 영역 또는 위수탁사업자 정보를 입력합니다.  

   거래처가 등록되지 않은 경우 '경영지원 설정 > 공통 코드 > 거래처 관리' 메뉴에서 거래처를 먼저 등록합니다. ([거래처 관리](/ko/admin-guides/manage-service/support-management/common-code/#i-2) 참고)

   - 등록 번호에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_search.png)를 클릭하여 거래처를 선택합니다.
     - 선택한 거래처 정보가 자동 표시되며, 필수 정보를 확인합니다. 필요할 경우 수정합니다.
     - 등록된 거래처 담당자가 1명 이상인 경우 담당자를 선택합니다.

   - 정발행 시 공급자 영역, 역발행 시 공급받는 자 영역, 위수탁 발행 시 위수탁사업자 영역에 사업장 정보가 자동 표시됩니다. 정보 확인 후 필요한 경우 수정해 주십시오.
   - 공급자, 공급받는 자, 위수탁사업자의 사업자등록번호가 모두 상이해야 발행할 수 있습니다.
10. 작성일: 전자세금계산서의 작성일을 입력합니다.
    - 오늘 이후의 날짜를 선택할 수 없습니다.
11. 품목 정보: '추가'를 클릭하여 품목 정보를 추가합니다.
    - 월/일: 입력한 작성일이 자동 표시됩니다. 발행일을 확인합니다.
      - 발행 월은 수정할 수 없습니다. 필요할 경우 발행 일은 원하는 날짜로 수정 가능합니다.
    - 품목: 입력 방식을 선택 후 정보를 입력합니다
      - 직접 입력: 품목명, 규격을 직접 입력합니다.
      - 품목 검색: ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_search.png)을 클릭하여 등록된 품목을 선택할 수 있습니다. 품목 선택 시 규격이 자동 표시됩니다. 품목 정보의 등록은 [경영지원 설정 > 공통 코드 > 품목 정보 관리](/ko/admin-guides/manage-service/support-management/common-code/#i-4) 메뉴를 참고해 주십시오.
    - 수량: 필요할 경우, 품목의 수량을 입력합니다.
    - 단가: 필요할 경우, 품목의 단가를 입력합니다.
    - 공급가액: 세액 제외한 공급가액을 입력합니다.
    - 세액: 공급가액에 따른 세액이 표시됩니다. 필요할 경우 수정합니다.
    - 비고: 필요할 경우, 비고를 입력합니다.
    - 1개 이상의 품목을 입력하려면 '추가'를 클릭하여 행을 추가 후 품목 정보를 입력합니다.
    - 입력한 품목을 삭제하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_search.png)을 클릭하여 삭제합니다.
12. 추가한 품목의 합계 금액을 구분(현금/수표/어음 또는 외상)에 맞게 입력한 후 영수/청구를 선택합니다.
13. 필요한 경우 '파일 첨부'를 클릭하여 증빙 서류를 등록합니다.
14. '저장'을 클릭합니다.

# 전자세금계산서 수정

발행 전 작성 중인 전자세금계산서는 내용을 수정할 수 있습니다.

전자세금계산서를 수정하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 수정할 전자세금계산서 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 목록이 조회됩니다.
5. 수정할 전자세금계산서의 관리 번호를 클릭합니다.
6. 수정 내용을 적용한 후 '저장'을 클릭합니다.
   - 수정한 내용이 저장됩니다.

# 전자세금계산서 삭제

발행 전 작성 중인 전자세금계산서는 삭제할 수 있습니다.

- 삭제한 전자세금계산서는 복구할 수 없습니다.

전자세금계산서를 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 필요한 경우, 검색창에서 수정할 전자세금계산서 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 목록이 조회됩니다.
5. 삭제할 전자세금계산서를 선택한 후 '선택 삭제'를 클릭합니다.
6. 확인 팝업 창의 내용을 확인한 후 '삭제'를 클릭합니다.
   - 선택한 전자세금계산서가 삭제됩니다.

각 전자세금계산서의 상세 내역을 먼저 확인하고자 할 경우, 관리 번호를 클릭하여 상세 내역을 확인한 후 해당 화면에서 '숨김'을 클릭하여 숨김 처리할 수 있습니다. 숨김 처리 시 목록에서 조회되지 않습니다.

# 전자세금계산서 발행

전자세금계산서를 발행하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '전자세금계산서'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
4. 발행할 전자세금계산서를 선택 후 '선택 발행'을 클릭합니다.
   - 각 전자세금계산서의 상세 내역을 먼저 확인하고자 할 경우, 관리 번호를 클릭하여 상세 내역을 확인한 후 해당 화면에서 '발행'을 클릭합니다.
5. '확인'을 클릭합니다.

전자세금계산서 발행 시 주의사항:

- 선택한 사업장이 아래 조건에 부합되어야 전자세금계산서를 발행할 수 있습니다.
- 해당 사업장이 스마트빌 계정과 연동되어야 합니다.
- 해당 사업장에 유효한 전자세금계산서 인증서가 등록되어야 합니다.([경영지원 설정 > 사업장 관리 > 인증서 관리](/ko/admin-guides/manage-service/support-management/company-info/#i-2-5) 참고)
- 네이버웍스 경영지원 설정의 인증서 관리에 등록한 해당 인증서를 스마트빌로 전송하거나, 스마트빌에 해당 인증서가 등록되어 있어야 합니다.([스마트빌 연동 설정](/ko/admin-guides/manage-service/tax-invoice/tax-settings/#i-2) 참고)
- 네이버웍스 경영지원 설정과 스마트빌의 인증서가 일치하지 않는 경우 전자세금계산서를 발행할 수 없습니다. 다음 가이드에 따라 설정해 주십시오.
  2. 사용할 인증서를 다시 등록 후 '전자세금계산서' 항목에 체크합니다.
  3. '전자세금계산서 > 설정 > 스마트빌 연동 설정'에서 등록된 인증서를 스마트빌로 전송해 주십시오.([스마트빌 연동 설정](/ko/admin-guides/manage-service/tax-invoice/tax-settings/#i-2) 참고)
- 발행할 거래처가 '휴페업' 상태인 경우 휴폐업 거래처 알림 팝업이 열립니다. 내용을 확인 후, 필요한 경우 발행해 주십시오.
- 역발행은 스마트빌에 회원 가입된 거래처에만 가능합니다.
- 과금 기준: 발행된 전자세금계산서는 건당 요금이 부과되며, 발행 취소한 건도 과금 대상에 포함됩니다.
