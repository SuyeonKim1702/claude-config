# 개인 폴더

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

  
드라이브에서 개인 폴더의 사용 여부를 설정합니다.

# 개인 폴더 사용 설정 변경

개인 폴더의 사용 여부를 설정합니다.

'사용'으로 변경 시 구성원들이 드라이브에서 개인 폴더를 사용할 수 있습니다.

'사용 안 함'으로 변경 시 설정이 변경된 시점부터 1개월 후 모든 파일은 삭제되고 복구되지 않습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 눌러 '드라이브' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '개인 폴더'를 선택합니다.
3. '개인 폴더'의 '폴더 사용' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 사용 여부를 설정합니다.
4. 상단의 '저장'을 누릅니다.

# 용량 한도 설정 변경

구성원이 사용하는 개인 폴더의 최대 용량 한도를 설정합니다.

사용된 개인 폴더 용량은 공용용량에서 차감됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 눌러 '드라이브' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '개인 폴더'를 선택합니다.
3. '개인 폴더'의 '용량 한도' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 용량 한도를 설정합니다.
4. 상단의 '저장'을 누릅니다.

# 용량 한도 예외 설정

개인 폴더의 용량 한도와 관계 없이 자유롭게 용량을 사용할 수 있는 구성원을 예외로 추가할 수 있습니다.

다만, 공용용량이 초과한 경우에는 용량 사용이 제한됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 눌러 '드라이브' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '개인 폴더'를 선택합니다.
3. '개인 폴더'의 '용량 한도' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, '예외 관리'를 선택하여 '용량 한도 예외' 화면으로 이동합니다.
4. '용량 한도 예외' 화면 상단의 '예외 추가' 버튼을 누릅니다.
5. 예외로 추가할 구성원을 선택한 후 '확인' 버튼을 누릅니다.  
   구성원을 삭제하려면 목록에서 선택해 왼쪽 상단의 '삭제'를 누른 후 '확인' 버튼을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
