# 드라이브 파일 이전

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

구성원의 개인 폴더 및 메시지방 폴더의 파일을 다른 구성원의 개인 폴더, 메시지방 폴더 및 공용  드라이브로 이전합니다.

구성원의 퇴사나 조직 또는 그룹의 변경 등으로 이전 구성원이 폴더를 사용할 수 없을 때 이전을 통해 기존의 폴더 내 파일을 계속 사용할 수 있습니다.

# 파일 이전 설정

구성원의 개인 폴더 또는 메시지방 폴더의 파일을 다른 폴더 또는 드라이브로 한번에 이전할 수 있습니다.

이전에는 일정 시간이 소요되며, 이전이 완료되면 이전한 폴더에서는 파일이 삭제됩니다.

이전 후 새 폴더 이름은 'TF\_' 로 시작하게 됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 눌러 '드라이브' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '파일 이전'을 선택합니다.
3. '이전할 대상 선택'에서 구성원의 개인 폴더 또는 메시지방 폴더 중 이전할 대상을 선택하고 이름을 입력합니다.
4. '다음'을 누릅니다.
5. '이전 받을 대상 선택'에서 구성원의 개인 폴더, 메시지방 폴더, 공용 드라이브 중 이전할 위치를 선택하고 이름을 입력합니다.
6. '파일 이전'을 누릅니다.

# 파일 이전 히스토리

이전된 파일의 히스토리 목록입니다.

이전 받은 구성원 개인 폴더, 메시지방 폴더와 이전된 구성원 개인 폴더, 메시지방 폴더, 공용 드라이브, 이전 받은 용량, 이전 날짜, 이전 상태를 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 눌러 '드라이브' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '파일 이전'을 선택합니다.
3. 하단의 '이전 히스토리 보기' 메뉴를 선택 합니다.
4. '이전 히스토리' 영역을 확인합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
