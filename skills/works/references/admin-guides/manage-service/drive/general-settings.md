# 드라이브 일반 설정

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

  
드라이브 서비스의 공용 폴더, 개인 폴더, 메시지방 폴더의 링크 설정과 파일의 버전 이력 관리에 대해 설정합니다.

# 파일 요청

'파일 요청' 기능의 사용 여부를 설정합니다.

이 기능을 사용하면 링크를 공유하여 외부 사용자로부터 파일을 업로드 받을 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '파일 요청' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 사용 여부를 설정합니다.
4. 상단의 '저장' 버튼을 누르고 '확인'을 누릅니다.

# 공용 폴더 생성 권한

공용 폴더 생성 권한을 설정할 수 있으며, 아래의 '관리'~~를~~ 클릭해 모든 공용 폴더를 관리할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '공용 폴더 생성 권한' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 권한 범위를 설정합니다.
   - 구성원이 자유 생성 : 모든 구성원이 공용 폴더를 생성할 수 있습니다.
4. 상단의 '저장' 버튼을 누르고 '확인'을 누릅니다.

# 링크 공유/유효 기간 설정

드라이브의 링크 공유 기능의 허용 여부와 링크 유효 기간을 설정합니다.

'링크 공유/유효 기간'의 설정값은 공용 폴더, 개인 폴더, 메시지방 폴더에 공통으로 적용됩니다.

## **링크 공유 기능**

링크 공유 기능의 허용 여부를 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '링크 유효 기간' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 허용 여부를 설정합니다.
   - 링크 공유 허용함: 드라이브의 하위 폴더나 파일을 링크로 공유할 수 있습니다.
   - 링크 공유 허용하지 않음: 드라이브의 하위 폴더나 파일을 링크로 공유할 수 없습니다.  
     링크 공유 기능을 끄면, 기존에 생성된 링크로 접근 및 이용이 불가능합니다.
4. 상단의 '저장'을 누르고 '확인'을 누릅니다.

## **링크 유효 기간**

링크의 유효 기간을 설정합니다. 유효 기간이 만료된 링크는 기간 만료 페이지로 이동되어 이용이 불가능합니다.

단, 링크 유효 기간을 설정하려면 링크 공유 기능을 허용해야 합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '링크 유효 기간' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 공유된 링크로 접근할 수 있는 기간을 설정합니다. 최대 7일/30일/90일/180일/ 제한 안 함 중 선택할 수 있습니다.
4. 상단의 '저장'을 누르고 '확인'을 누릅니다.

# **링크 접근 권한**

링크에 대한 접근 권한을 설정합니다. 링크 접근 권한은 공용 폴더, 개인 폴더, 메시지방 폴더 각 항목별로 설정할 수 있습니다.

단, 링크 접근 권한을 설정하려면 링크 공유 기능을 허용해야 합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '링크 접근 권한'에서 각 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 링크의 접근 권한을 설정합니다.
   - '사내 구성원': 네이버웍스에 로그인한 내부의 구성원만 링크에 접근할 수 있습니다.
   - '모든 사람(비밀번호 설정 가능)': 링크를 아는 모든 사람이 접근할 수 있습니다. 구성원이 링크 공유 시 비밀번호로 한 번 더 인증할 수 있도록 추가적으로 설정할 수 있습니다.
   - '지정한 사람만(보안 OTP 인증)': OTP(One Time Password) 인증을 통해 접근할 수 있습니다. 지정한 사람만 링크에 접근할 수 있도록 보안을 강화하여 공유할 경우에 사용하며, 링크 확산을 방지합니다.
4. 상단의 '저장'을 누르고 '확인'을 누릅니다.

# **휴지통 보관 기간**

드라이브에서 삭제한 파일이 휴지통에 보관되는 기간을 설정합니다. 휴지통에 보관되는 기간 동안은 삭제된 파일의 복구가 가능합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. '휴지통 보관 기간' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 보관 기간을 설정합니다. 최대 5일/15일/30일/50일/자동 삭제 안 함 중 선택할 수 있습니다.
4. 상단의 '저장'을 누르고 '확인'을 누릅니다.

# 파일 버전 이력 관리

파일을 덮어쓸 때 버전 이력으로 저장되는 기간을 설정하고 버전으로 저장되는 파일의 종류를 추가하거나 제외합니다.

버전이 저장되어 버전 이력으로 남게 되면, 저장된 버전은 구성원 폴더(개인 폴더)인 경우는 개인 용량을 팀/그룹 폴더인 경우는 공용 용량을 차지합니다.

## 버전 저장 기간 설정

버전 이력으로 저장되는 기간을 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반' 탭을 선택합니다.
3. 하단의 '고급 설정'을 눌러, '설정'을 누릅니다.
4. '수정'을 누릅니다.
5. '버전 저장 기간' 항목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택하여 버전 저장 기간을 선택합니다. 1개월/3개월/6개월/12개월/영구 보존 중에 선택할 수 있습니다.
6. 상단의 '저장'을 누르고 '확인'을 누릅니다.

## 버전 저장 파일 종류 설정

기본 제공하는 파일의 종류 외 버전 이력을 저장할 파일의 종류를 추가합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '드라이브'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. 하단의 '고급 설정'을 눌러, '설정'을 누릅니다.
4. 문서 파일/이미지 파일 확장자를 선택하여, 버전 이력을 저장할 파일의 확장자를 설정합니다. 설정된 기본 확장자를 선택 시, 버전 이력 저장 목록에서 제외됩니다.
   - 확장자를 직접 입력하려면 '추가된 파일' 하단에 확장자를 입력하고 '추가'를 누릅니다.
   - 추가된 확장자를 삭제하려면 'x'를 누릅니다.
5. 상단의 '저장'을 누르고 '확인'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
