# 공개된 멤버가 없는 연락처

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

  
공개된 멤버가 없는 연락처는 구성원이 퇴사했거나 조직이 삭제되어 회사 내 어느 구성원에게도 공개되어 있지 않은 외부 연락처입니다.

Admin에서 공개된 멤버가 없는 연락처의 공개 대상을 다시 지정하거나 필요 없는 경우 연락처를 삭제할 수 있습니다.

# 공개된 멤버가 없는 연락처 보기

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-filter-admin.png) 아이콘을 누른 후 '공개된 멤버가 없는 연락처'를 누릅니다.

# 공개된 멤버가 없는 연락처의 공개 대상 변경

관리자가 공개된 멤버가 없는 연락처의 공개 대상을 변경해 연락처를 특정 구성원 또는 회사 전체 구성원에게 공유할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-filter-admin.png) 아이콘을 누른 후 '공개된 멤버가 없는 연락처'를 누릅니다.
4. 수정하려는 연락처를 누르고 '수정'을 누릅니다.
5. 공개 범위를 '전체 공개' 또는 '멤버 공개'로 변경합니다. '멤버 공개'가 선택된 상태에서는 공개 범위에 구성원, 조직, 그룹을 추가하거나 삭제할 수 있습니다.
6. '저장'을 눌러 설정값을 저장합니다.

# 공개된 멤버가 없는 연락처 삭제

관리자가 연락처를 삭제할 수 있습니다. 삭제한 연락처는 영구 삭제되고 복원이 불가능합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-filter-admin.png) 아이콘을 누른 후 '공개된 멤버가 없는 연락처'를 누릅니다.
4. 삭제하려는 연락처를 누르고 '삭제'를 누른 후 '확인'을 누릅니다. 여러 연락처를 한번에 삭제하려면 목록에서 왼쪽의 체크박스로 선택한 후, 상단의 '삭제'를 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
