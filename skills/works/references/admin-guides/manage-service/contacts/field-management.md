# 외부 연락처 항목 관리

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

외부 연락처에 항목을 추가할 수 있습니다.

추가한 항목의 순서를 변경하거나, 수정 또는 삭제할 수 있습니다.

Admin에서 추가한 항목은 주소록에서 외부 연락처를 추가할 때 노출됩니다.

# 항목 추가

외부 연락처에 새로운 항목을 추가합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 오른쪽 상단의 '항목 관리' 버튼을 클릭합니다.
4. 오른쪽 상단의 '항목 추가'를 클릭합니다.
5. '항목명'을 입력하고, '타입'을 선택합니다. '타입'은 '텍스트'와 '링크' 중 선택할 수 있습니다. 항목을 추가한 후에는 수정할 수 없습니다.
   - 텍스트: 텍스트 형태의 항목을 연락처 항목으로 표시할 수 있습니다. 20자까지 입력할 수 있습니다.
   - 링크: 외부 사이트 링크를 연락처 항목으로 표시할 수 있습니다. 링크 텍스트를 설정할 수 있으며, URL은 반드시 입력해야 합니다.
6. '입력 개수' 및 propertyName을 설정합니다. 항목을 추가한 후에는 수정할 수 없습니다.
   - 입력 개수: '단일값' 또는 '다중값' 중 선택할 수 있습니다. '다중값'을 선택하면 해당 항목에 최대 10개의 값을 입력할 수 있습니다.
   - propertyName: 외부 시스템과 연동할 때 사용할 값을 입력합니다. 입력하지 않으면 임의의 값이 생성됩니다.
7. '저장'을 클릭합니다.

# 항목 순서 변경

추가된 항목이 외부 연락처에 노출되는 순서를 변경합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 오른쪽 상단의 '항목 관리' 버튼을 클릭합니다.
4. 목록에서 항목 왼쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/grabber.png) 아이콘을 끌어 원하는 순서로 이동합니다.

# 항목명 수정

추가된 항목 내용 중 항목명을 수정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 오른쪽 상단의 '항목 관리' 버튼을 클릭합니다.
4. 수정할 항목 오른쪽의 '수정'을 클릭해 '항목명'을 변경합니다.

항목명 외 다른 설정은 항목을 추가한 이후에는 변경할 수 없습니다.

# 항목 삭제

추가된 항목을 삭제합니다.

항목을 삭제하면 모든 연락처 정보에서 해당 항목이 삭제되며, 복원할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 오른쪽 상단의 '항목 관리' 버튼을 클릭합니다.
4. 목록에서 삭제할 항목 오른쪽의 '삭제'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
