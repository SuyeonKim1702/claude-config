# 외부 연락처 관리

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

  
[외부 연락처(링크도 수정)](/ko/use-guides/contacts/external-contacts/view/)는 회사 구성원과 같이 공유하고 관리하는 연락처들을 모아두는 주소록입니다.

관리자는 외부 연락처 관리 탭에서 외부 연락처 검색, 마스터 변경, 공개 범위 변경 및 삭제를 할 수 있습니다.

# 외부 연락처 검색

관리자가 연락처를 검색하여 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 연락처 목록 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-12-search-434343.png) 아이콘을 누르면 검색창이 노출됩니다.  
   검색창에 찾고자 하는 연락처 이름을 검색하거나, '상세'를 눌러 검색 항목별로 상세한 검색을 할 수도 있습니다. 상세 검색에서는 '연락처 종류', '연락처 공개 대상', '연락처 마스터', '연락처 이름', '연락처 회사 ・ 소속'을 기준으로 검색할 수 있습니다.

   |  |  |
   | --- | --- |
   | 연락처 종류 | 연락처의 종류(LINE 연락처, 다른 회사 NAVER WORKS 연락처)를 구분하여 검색합니다. |
   | 연락처 공개 대상 | 검색창에 입력된 구성원이 볼 수 있는 연락처를 검색합니다. |
   | 연락처 마스터 | 검색창에 입력된 구성원이 연락처 마스터인 연락처를 검색합니다. |
   | 연락처 이름 | 연락처의 이름을 검색합니다. |
   | 연락처 회사 ・ 소속 | 연락처의 회사/소속을 검색합니다. |
4. '검색'을 누릅니다.

# 외부 연락처 필터링

외부 연락처 목록에서 연락처를 필터링하여 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 연락처 목록 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-filter-admin.png) 아이콘을 누릅니다.
4. '전체', 'LINE 연락처', '다른 회사 NAVER WORKS 연락처', '공개된 멤버가 없는 연락처', '마스터가 없는 연락처', '휴지통 연락처' 중 선택하여 연락처 목록을 필터링합니다.

# 외부 연락처의 마스터 변경

연락처 마스터는 연락처를 최초로 등록한 구성원을 의미합니다. 연락처 마스터가 삭제된 경우, 다른 구성원으로 연락처 마스터를 변경할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 연락처 목록에서 마스터를 변경하고자 하는 연락처를 선택합니다.
4. '연락처 정보' 창에서 '수정'을 누릅니다.
5. '연락처 정보 수정' 창에서 새 연락처 마스터를 검색해 선택합니다. 연락처 마스터는 1명만 지정할 수 있으며, 연락처 공개 범위가 '멤버 공개'로 설정되어 있을 경우 연락처 마스터는 연락처 공개 대상에 자동으로 포함됩니다.
6. '저장'을 누릅니다.

# 외부 연락처의 마스터 일괄 변경

연락처 마스터가 동일할 경우, 다수의 연락처의 마스터를 한 번에 변경할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 연락처 목록 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-12-search-434343.png) 아이콘을 누른 후, '상세'를 누릅니다.
4. '연락처 마스터' 항목의 검색창에 변경하고자 하는 연락처의 마스터를 입력한 후, '검색' 버튼을 누릅니다.
5. 마스터를 변경하고자 하는 연락처 왼쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-todo-uncheck.png) 아이콘을 선택합니다.
6. 목록 상단의 '마스터 변경'을 누릅니다.
7. '연락처 마스터 변경' 창에서 새로운 마스터를 검색하여 선택한 후 '저장'을 누릅니다.

# 외부 연락처의 공개 범위 변경

관리자가 연락처의 공개 범위를 변경하거나 공개 멤버를 추가할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 연락처 목록에서 공개 범위를 변경할 연락처를 선택합니다.
4. '연락처 정보' 창에서 '수정'을 누릅니다.
5. '연락처 정보 수정' 창에서 공개 범위를 '전체 공개' 또는 '멤버 공개'를 누릅니다.  
   '멤버 공개'를 선택한 경우, 구성원, 조직 또는 그룹을 추가하거나 삭제할 수 있습니다.
6. '저장'을 누릅니다.

# 외부 연락처 삭제

LINE/외부 네이버웍스 사용자는 삭제할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 연락처 목록에서 삭제할 연락처를 선택한 후 목록 상단의 '삭제'를 누릅니다.  
   또는, 삭제할 연락처를 누르고 '연락처 정보' 창에서 '삭제'를 눌러 개별로 연락처를 삭제할 수도 있습니다.

# 외부 연락처 정리

불필요한 연락처('공개된 멤버가 없는 연락처', '마스터가 없는 연락처', '휴지통 연락처')를 한번에 삭제할 수 있습니다.

삭제가 완료되면 관리자에게 서비스 알림이 발송됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 연락처 관리' 탭을 선택합니다.
3. 연락처 목록 상단에 '연락처 정리'를 누릅니다.
4. '공개된 멤버가 없는 연락처', '마스터가 없는 연락처', '휴지통 연락처' 중 삭제하고자 하는 연락처 종류를 선택합니다.
5. '휴지통으로 이동' 또는 '영구 삭제'를 누릅니다. 단, '휴지통 연락처'를 선택한 경우, '휴지통으로 이동'을 누를 수 없습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
