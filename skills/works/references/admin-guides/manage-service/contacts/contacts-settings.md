# 주소록 일반 설정

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

  
주소록 서비스의 일반적인 사용 기능을 설정합니다.

# 외부 연락처

외부 연락처의 회사 태그, MY 태그, 공개범위, 휴지통 보관기간을 설정합니다.

## 회사 태그 생성 권한

회사 태그를 관리할 수 있는 권한을 구성원에게도 부여할지 설정합니다.

'구성원이 자유 생성 ' 선택 시, 회사 내 모든 구성원이 자유롭게 [회사 태그를 추가, 수정, 삭제](/ko/use-guides/contacts/external-contacts/add-edit-delete-tag/)할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '회사 태그 생성 권한' 항목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 권한을 설정합니다.
4. 오른쪽 상단의 '저장'을 눌러 설정값을 저장합니다.

## MY 태그 사용

회사 구성원 본인만 볼 수 있는 MY 태그를 사용하게 할지 설정합니다.

설정을 '허용'에서 '허용 안 함'으로 변경 시, 구성원들이 기존에 사용하고 있던 MY 태그 정보는 모두 삭제됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. 'MY 태그 사용' 항목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 사용 여부를 설정합니다.
4. 오른쪽 상단의 '저장'을 눌러 설정값을 저장합니다.

## 연락처 기본 공개 범위

연락처를 추가할 때 기본으로 선택되는 공개 범위 옵션을 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '연락처 기본 공개 범위' 항목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 설정합니다.
4. 오른쪽 상단의 '저장'을 눌러 설정값을 저장합니다.

## 휴지통 보관 기간

구성원이 삭제한 연락처가 휴지통에 보관되는 기간을 설정합니다.

휴지통에 보관되는 기간 동안은 [연락처를 복원](/ko/use-guides/contacts/external-contacts/add-edit-delete/#i-7)할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '휴지통 보관 기간' 항목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 기간을 설정합니다.
4. 오른쪽 상단의 '저장'을 눌러 설정값을 저장합니다.

## *외부 연락처 저장·내보내기 권한*

PC 웹 주소록에서 외부 연락처의 저장 및 내보내기 기능을 사용할 수 있는 대상의 범위를 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '외부 연락처 저장·내보내기 권한*' 항*목 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누른 후 대상의 범위를 선택합니다.
4. 오른쪽 상단의 '저장'을 눌러 설정값을 저장합니다.

# 그룹

그룹을 생성할 수 있는 권한을 부여할 대상을 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '그룹 생성 권한' 항목의 오른쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 선택 후 관리자만 그룹을 생성하게 하려면 '관리자만 생성'을, 구성원도 생성할 수 있게 하려면 '구성원이 자유 생성'을 선택합니다.
4. 설정 변경 후 상단의 '저장'을 눌러 설정값을 저장합니다.

# 회사 구성원

겸직/휴직 구성원 수 표시 여부를 설정합니다.

## 겸직/휴직 구성원 수 표시

겸직을 맡고 있는 구성원 수와 휴직 중인 구성원의 수를 조직도 상단에 표시할지 설정합니다.

단, PC웹에서 겸직 및 휴직 구성원 수가 표시되는 조직도는 화면 오른쪽 상단의 구성원 검색 시 노출되며, 노출되는 조직도 오른쪽 상단의 '하위조직포함'을 선택해야 합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '주소록'을 눌러 '주소록' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반'을 선택합니다.
3. '겸직/휴직 구성원수 표시' 항목 오른쪽에 있는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 사용 여부를 설정합니다.
4. 오른쪽 상단의 '저장'을 눌러 설정값을 저장합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
