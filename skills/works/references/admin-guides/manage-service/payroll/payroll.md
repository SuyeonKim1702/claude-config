# 급여

이 기능은 웍스 급여 상품 신청 후 사용 가능합니다.

  
임직원의 매월 급여 및 상여를 계산하고 지급할 수 있습니다.

인사 및 근태 서비스와 연동하여 급여를 계산하고, 마감된 결과는 회계 서비스로 전표를 전송할 수 있습니다.  
국세청 홈택스 및 위택스에 원천세 신고를 할 수 있도록 생성 파일을 제공합니다.  
근로소득자의 간이지급명세서를 조회하여 국세청 홈택스에 신고할 파일과 4대보험 취득/상실 신고에 대한 자료도 제공합니다.

# 급여일 관리

급여 계산을 위해 급여일을 설정하고, 여러 개의 차수를 관리할 수 있습니다.  
여러 개의 차수 설정은 '[급여 > 설정 > 급여 설정](/ko/admin-guides/manage-service/payroll/payroll-settings/#i-1)' 메뉴에서 할 수 있습니다.

급여일 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 급여일을 기준 연도에 따라 검색 |
| ② 추가 | [급여일 추가](/ko/admin-guides/manage-service/payroll/payroll/#i-1-1) |
| ③ 다운로드 | 급여일 목록 다운로드 |
| ④ 급여일 목록 | 급여일 확인 및 수정, 삭제 |
| ⑤ 수정 | [선택한 급여일 항목 수정](#i-1-2) |
| ⑥ 삭제 | [선택한 급여일 항목 삭제](#i-1-4) |

## 급여일 추가

급여일을 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '급여'를 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
   - 당해 연도의 모든 급여일이 기본으로 설정되어 조회됩니다.
4. '추가'를 클릭합니다.
5. 급여일 추가 팝업 창에서 급여일 정보를 입력한 후 '저장'을 클릭합니다.  

   - 급여월: 급여월을 선택합니다.
   - 차수: 차수를 입력합니다. 숫자 1부터 99까지 입력할 수 있습니다. 보통 1차수부터 시작합니다.
   - 급여명세서 공개: 급여명세서 공개 여부에 따라 '공개' 또는 '공개 안 함'으로 선택합니다.
     - 작업 중에는 '공개 안 함'으로 설정하여 급여 작업을 하고, 급여 마감, 전표 생성 후 급여 명세서를 발송하는 때에 맞추어 '공개'로 변경하면 사원들이 급여명세서를 확인할 수 있습니다.
   - 상태: 다음 3가지의 상태로 설정할 수 있습니다.
     - 미도래: 지급일이 정해져 있는 급여일을 미리 설정할 수 있습니다.
     - 오픈: 현재 작업 중인 급여에 대한 상태를 설정합니다. 보통 이번 달 급여 작업 시 오픈 상태로 작업을 시작합니다.
     - 마감: 급여가 끝난 후 마감된 상태를 설정합니다. 급여일이 마감인 경우 조회는 할 수 있으나, 수정 및 입력은 할 수 없습니다.
   - 아르바이트용: 추가할 급여일이 아르바이트 전용 급여일인 경우 '예'로 설정합니다.
   - 급여 명세서 메모: 급여명세서 하단의 '※공지사항'으로 임직원들에게 안내할 사항을 입력할 수 있습니다.  

     임직원이 '내 업무'에서 급여명세서를 조회하려면 급여 담당자는 아래 두 가지를 설정해야 합니다.
     1. 급여명세서 공개 항목을 '공개'로 설정합니다.
     2. 상태 항목을 '마감'으로 설정합니다.

연말정산 시 추가 소득에 대한 차수는 12월 99차입니다.

- 12월 99차에 입력하는 지급액은 실제 지급을 하지 않고 과세처리로 원천징수만 했을 경우 연말정산을 위해 소득합산용으로 입력합니다.
