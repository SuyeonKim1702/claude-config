# 지출 결의 신청 현황

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
지출 결의서 서식으로 작성된 지출 결의 신청 현황을 확인할 수 있습니다. 또한, 조건에 따라 지출 결의 신청 현황을 검색하여 조회할 수 있으며, 지출 결의 신청 목록을 엑셀 파일로 다운로드할 수 있습니다.

지출 결의 신청 현황 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 지출 결의 신청 현황을 상태, 부서, 예산명, 기간, 작성자, 지출 항목, 거래처, 증빙 유형, 문서 번호, 문서 제목에 따라 검색 |
| ② 다운로드 | 지출 결의 신청 현황 목록을 엑셀 파일로 다운로드 |
| ③ 지출 결의 신청 현황 목록 | 신청한 지출 결의 목록 확인, 문서 번호를 클릭하면 지출 결의서의 상세 내용 확인  - 프로젝트 코드: 회사가 사용으로 설정한 경우 항목 표시([경영지원 설정 > 공통 코드 > 공통 코드 관리](/ko/admin-guides/manage-service/support-management/common-code/#i-1) 메뉴에서 설정), 지출 신청자가 프로젝트 코드를 입력해 신청한 경우 각 행의 프로젝트 코드를 클릭하여 관련 정보 확인 가능 |

# 지출 결의 신청 현황 목록 다운로드

지출 결의 신청 현황 목록을 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '지출 결의 신청 현황'을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 지출 결의 신청 현황 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 지출 결의 신청 현황이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 지출 결의 신청 현황 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Downloads` 폴더에 `ReportApplyList.xlsx` 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
