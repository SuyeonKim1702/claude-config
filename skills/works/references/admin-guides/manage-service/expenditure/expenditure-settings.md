# 설정

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
지출 예산 사용 여부, 지출 품의 사용 여부 및 지출 품의, 지출 결의 상세 내용 사용 여부를 설정할 수 있습니다.

설정을 변경하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '설정'을 클릭합니다.
4. 기본 설정 및 서식 필수 항목 설정을 선택합니다.
   - 지출 예산 사용 여부는 예산 서비스를 사용하는 경우에 표시됩니다. 지출 예산 사용 여부를 '사용'으로 선택한 경우, 지출 품의, 지출 결의 신청 시 예산을 조회하고 사용할 수 있습니다. 또한, 지출 품의, 지출 결의 신청 현황 목록에 예산 관련 항목이 추가됩니다.
   - 지출 품의서 사용 여부를 '사용'으로 선택한 경우, 지출할 품의서를 사전에 작성하여 지출 결의서 작성 시 활용할 수 있습니다.
   - 지출 품의서 상세 내용 사용 여부를 '사용'으로 선택한 경우, 지출 품의 신청 시 각 내역의 상세 내용을 필수로 입력해야 합니다.
   - 지출 결의서 상세 내용 사용 여부를 '사용'으로 선택한 경우, 지출 결의 신청 시 각 내역의 상세 내용을 필수로 입력해야 합니다.
   - 지출 결의서 증빙 첨부를 '사용'으로 선택한 경우, 지출 결의서 작성 시 증빙 파일을 첨부할 수 있습니다.
5. '저장'을 클릭합니다.
   - 변경한 사항이 반영됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
