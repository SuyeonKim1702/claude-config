# 프로젝트 코드별 정산 현황

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
프로젝트 코드를 사용하여 정산한 지출결의 신청 건에 대해, 프로젝트 및 상세 지출 항목별 지출결의 현황을 조회할 수 있습니다. 또한 프로젝트별 정산 현황 목록을 엑셀 파일로 다운로드할 수 있습니다.

프로젝트 코드별 정산 현황 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 프로젝트 코드별 정산 현황을 프로젝트 코드, 기간, 지출 항목 조건에 따라 검색 |
| ② 다운로드 | 프로젝트 코드별 정산 현황을 엑셀 파일로 다운로드 |
| ③ 프로젝트 코드별 정산 현황 목록 | 프로젝트 코드별 정산 현황을 목록으로 확인  - '진행중 포함'을 선택하면 결재 진행 중인 지출 결의 문서를 포함하여 조회되며, 진행 중인 건수 및 금액 표시, '진행중 포함'을 선택하지 않으면 완료된 건수 및 금액만 표시 - 각 행의 프로젝트 코드, 프로젝트명을 클릭하여 프로젝트 코드의 상세 정보와 수정 내역 확인 - 각 행의 신청 건수, 사용 합계 금액을 클릭하여 지출 결의 문서 및 내역 확인 |

# 프로젝트 코드별 정산 현황 다운로드

프로젝트 코드별 정산 현황 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '프로젝트 코드별 정산 현황'을 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 자원 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 프로젝트 코드별 정산 현황이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 프로젝트 코드별 정산 현황이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Downloads` 폴더에 `ReportProjectCodeList.xlsx` 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
