# 지출 항목 관리

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
지출 항목을 코드, 항목명, 계정 과목에 따라 추가하고 관리할 수 있습니다.

지출 항목 관리의 정보는 예산 서비스 사용 시 예산 기준 설정에 사용됩니다.

지출 항목 관리 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 코드 또는 명칭을 입력하여 지출 항목 검색 |
| ② 추가 | 새로운 지출 항목을 추가([지출 항목 추가](#i-1) 참고) |
| ③ 삭제 | 지출 항목을 삭제([지출 항목 삭제](#i-2) 참고) |
| ④ 업로드 | 지출 항목을 엑셀 파일로 일괄 업로드([지출 항목 일괄 업로드](#i-3) 참고) |
| ⑤ 지출 항목 목록 | - 지출 항목별 지출 항목 코드, 지출 항목명, 계정 과목 확인 - 을 클릭하여 지출 항목 순서 변경([지출 항목 순서 변경](#i-4) 참고) - 지출 항목 사용 여부 변경([지출 항목 사용 여부 변경](#i-5) 참고) |
| ⑥ 순서 변경 | 지출 항목의 순서를 변경([지출 항목 순서 변경](#i-4) 참고) |
| ⑦ 사용 | 지출 항목의 사용 여부 표시 및 변경([지출 항목 사용 여부 변경](#i-5) 참고) |

# 지출 항목 추가

지출 항목을 추가하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. '추가'를 클릭합니다.
4. 지출 항목 코드와 지출 항목명을 입력합니다.
5. 회계 서비스와 연동할 계정 과목의 코드 또는 명칭을 입력하여 검색한 후 선택합니다.
   - 사용자가 지출 결의서 작성 시 자동으로 해당 계정 과목으로 전표가 생성됩니다.
6. '저장'을 클릭합니다.

# 지출 항목 삭제

해당 지출 항목으로 예산이 배정되었거나 정산 처리된 지출 항목은 삭제할 수 없습니다.

지출 항목을 삭제하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '지출 항목 관리'를 클릭합니다.
4. 삭제할 항목을 선택한 후 '삭제'를 클릭합니다.
5. '저장'을 클릭합니다.

# 지출 항목 일괄 업로드

지출 서비스를 처음 사용할 때, 기존에 사용하던 지출 항목을 회계 계정 등과 함께 업로드하거나 지출 항목을 일괄로 추가할 때 사용합니다.

지출 항목을 일괄 업로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '지출 항목 관리'를 클릭합니다.
4. '업로드'를 클릭합니다.
5. 업로드 팝업 창에서 '샘플 파일 다운로드'를 클릭한 후 다운로드한 샘플 파일을 작성합니다.
   - 다운로드한 샘플 파일은 `Download` 폴더의 `costItem-sample.xlsx` 파일명으로 저장됩니다.
6. 업로드 팝업 창에서 '파일 첨부'를 클릭한 후 작성한 샘플 파일을 클릭합니다.
7. '확인'을 클릭합니다.
   - 추가한 지출 항목이 지출 항목 목록에 추가됩니다.
8. '저장'을 클릭합니다.

# 지출 항목 순서 변경

지출 항목의 순서를 변경하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. '지출 항목 관리'를 클릭합니다.
4. '순서 변경'에서 화살표를 클릭해 원하는 순서로 변경합니다.
5. '저장'을 클릭합니다.
   - 변경된 순서가 저장됩니다.

# 지출 항목 사용 여부 변경

지출 항목 사용 여부를 변경하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. '지출 항목 관리'를 클릭합니다.
4. '사용' 열에서 '사용' 또는 '미사용'을 클릭합니다.
5. '저장'을 클릭합니다.
   - 변경된 사용 여부가 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
