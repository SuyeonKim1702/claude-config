# 지출 현황 통합 조회

이 기능은 웍스 재무 상품 신청 후 사용 가능합니다.

  
지출결의 신청의 통합 내역을 조회할 수 있습니다.

지출품의서를 참조하여 지출결의를 신청한 경우, 지출품의 대비 지출결의 합계 금액과 증빙 유형별 금액을 조회할 수 있습니다. 지출품의서를 참조하지 않은 경우, 지출결의서 작성 결과만 조회할 수 있습니다.

지출 현황 통합 조회 화면은 다음과 같이 구성되어 있습니다.

| 영역 | 설명 |
| --- | --- |
| ① 검색창 | 지출 현황을 귀속 부서, 기간, 지출 항목 조건에 따라 검색 |
| ② 다운로드 | 지출 현황 목록을 엑셀 파일로 다운로드 |
| ③ 지출 현황 통합 조회 목록 | 지출품의 및 지출결의 현황을 목록으로 확인  - 각 행의 '귀속 부서'를 클릭하여 조직도와 사원 상세 정보 확인 - 각 행의 '품의 금액'을 클릭하여 지출품의 신청 현황 메뉴로 이동하여, 선택한 귀속 부서/기간/지출 항목에 해당하는 지출품의 내역 확인 - 각 행의 '합계'를 클릭하여 지출결의 문서 확인 및 상세 내역 확인 |

# 지출 현황 목록 다운로드

지출 현황 목록을 엑셀 파일로 다운로드하는 방법은 다음과 같습니다.

1. PC 웹 화면 오른쪽 상단의 '관리자 메뉴'를 클릭 후, 팝업에서 '지출'을 클릭합니다.
2. 화면 왼쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2023/12/workplace_mmenu.png)을 클릭합니다.
3. 메뉴에서 '지출 현황 통합 조회'를 클릭합니다.
4. 필요한 경우, 검색창에서 조회할 자원 조건을 입력한 후 '검색'을 클릭합니다.
   - 입력한 조건에 해당하는 지출 현황 목록이 조회됩니다.
5. '다운로드'를 클릭합니다.
   - 지출 현황 목록이 엑셀 파일로 다운로드됩니다.

다운로드된 파일은 `Downloads` 폴더에 `ReportTotalList.xlsx` 파일명으로 저장됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
