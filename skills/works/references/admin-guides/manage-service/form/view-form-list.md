# 설문 목록 조회

구성원이 작성한 설문 조회 및 모니터링하고 관리할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '설문 목록'을 선택하여, 조회합니다.
   - 제목/생성일: 설문의 제목과 작성된 날짜입니다.
   - 진행 상태: 설문이 진행중인지 종료되었는지 여부입니다.
   - 작성자: 작성자의 이름 및 계정(ID@그룹명 또는 ID@도메인)입니다.
   - 응답자 수: 설문에 응답한 구성원의 숫자입니다. 응답자 수/설문 대상자 수의 형태로 표시됩니다. 사내/외부 사용자 여부도 표시됩니다.
   - 공동 관리: 설문의 공동 관리 여부를 표시합니다.
   - 결과 공개: 설문 결과를 구성원에게 공개할지 설문 작성자가 설정한 값이 표시됩니다.
     - 사용: 설문 결과 공개
     - 사용 안 함: 설문 결과 비공개
   - 응답자 정보: 설문 응답자 정보를 익명으로 받을지, 실명으로 받을지 설문 작성자가 설정한 값이 표시됩니다.

# 설문 삭제

이 기능은 유료 플랜에서만 사용할 수 있습니다.

구성원이 작성한 설문을 관리자가 삭제합니다. 설문을 삭제하면 더 이상 설문에 접근할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '설문 목록'을 선택합니다.
3. 삭제할 설문을 선택한 후 '삭제'를 누릅니다.

# 검색 조건 설정

이 기능은 유료 플랜에서만 사용할 수 있습니다.

검색 조건을 설정해 조건에 맞는 설문을 빠르게 확인합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '설문 목록'을 선택합니다.
3. 오른쪽 상단 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-20-search-767676.png) 아이콘을 클릭합니다.
4. 검색 조건에 필요한 값을 선택하거나 입력합니다.
   - 전체: 검색 대상을 선택합니다. 전체, 설문 제목, 작성자 가운데 선택할 수 있습니다. 설문 제목이나 작성자를 선택하면 검색어를 입력합니다.
   - 기간: 검색하려는 설문이 작성된 기간을 입력합니다.
5. '검색'을 클릭하여 조건에 맞는 설문을 검색합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
