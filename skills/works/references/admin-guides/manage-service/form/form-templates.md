# 설문 템플릿 관리

이 기능은 유료 플랜에서만 사용할 수 있습니다.

설문 템플릿을 추가, 삭제 또는 수정할 수 있고, 템플릿으로 설문을 쉽게 만들 수 있습니다.

일반 회사에서 많이 사용하는 설문 템플릿 12가지를 기본 제공합니다.

# 설문 템플릿 추가

설문 템플릿은 최대 100개까지 추가할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '템플릿'을 선택합니다.
3. 오른쪽 상단의 '템플릿 추가 '를 눌러 템플릿 작성 화면으로 이동합니다.
4. 템플릿 내용을 구성한 후 '완료'를 누릅니다.
5. 설문 템플릿 목록에서 추가한 템플릿을 사용함으로 선택 후 변경된 설정을 저장합니다.

# 설문 템플릿 수정

설문 템플릿의 내용을 수정합니다. 단,기본 제공되는 설문 템플릿은 수정할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '템플릿'을 선택합니다.
3. 템플릿 목록에서 수정할 템플릿을 클릭하면, 템플릿 수정 화면으로 이동합니다.
4. 내용을 수정한 후 하단의 '완료'를 누릅니다.

# 설문 템플릿 삭제

설문 템플릿을 삭제합니다. 기본 제공되는 설문 템플릿도 삭제 가능하며, 삭제한 템플릿은 복구할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '템플릿'을 선택합니다.
3. 템플릿 목록에서 삭제할 템플릿을 선택 후 왼쪽 상단의 '삭제'를 누릅니다.
4. 확인창에서 '확인'를 누릅니다.

# 설문 템플릿 순서 변경

설문 템플릿의 목록 노출 순서를 변경합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '템플릿'을 선택합니다.
3. 오른쪽 상단에 있는 순서 변경 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-16-align-767676.png) 아이콘을 누릅니다.
4. 템플릿 목록에서 순서 변경할 템플릿을 누른 채로 이동하여 순서를 변경합니다.
5. '저장'을 누릅니다.
6. 확인창에서 '확인'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
