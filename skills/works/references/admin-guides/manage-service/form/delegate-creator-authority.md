# 설문 작성자 권한 위임

설문의 수정, 결과 확인, 설문 결과 공유 등 설문 작성자의 권한을 다른 구성원에게 위임합니다.

작성자 권한이 위임되면 기존 설문 작성자는 설문을 더 이상 확인할 수 없고, 위임받은 구성원의 '내가 만든 설문' 목록에 위임받은 설문이 노출됩니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '작성자 위임 '을 선택합니다.
3. 설문 작성자의 이름을 입력하여 검색하고 선택합니다.
4. 대상 설문 목록을 선택하고 '다음'을 누릅니다.
5. 위임 받을 대상자의 이름을 입력하여 검색하고 선택합니다.
6. 하단의 '권한 위임'을 눌러 선택한 설문의 작성자 권한을 위임합니다.

# 작성자 권한 위임 히스토리 확인

작성자 권한을 위임한 설문의 목록입니다. 권한 위임과 관련한 정보를 확인합니다.

'위임 히스토리 보기'를 눌러 '위임 히스토리 보기' 화면으로 이동합니다.

- 제목: 권한을 위임한 설문의 제목입니다.
- 위임 받은 구성원: 새로 권한을 위임 받은 구성원의 이름과 소속 조직이 표시됩니다.
- 위임 날짜: 위임이 완료된 날짜입니다.
- 상태: 위임과 관련된 상태가 표시됩니다.
- 위임 완료: 위임이 완료된 상태입니다.
- 위임 실패: 네트워크 오류 등의 원인으로 위임되지 못한 상태입니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
