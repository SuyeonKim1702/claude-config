# 설문 일반 설정

이 기능은 유료 플랜에서만 사용할 수 있습니다.

설문 작성자가 설문을 만들고, 결과를 공유할 때 필요한 설정을 제어할 수 있습니다.

설정 변경 후 작성된 설문부터 변경된 내용이 적용됩니다.

# 설문 대상

구성원이 설문을 만들때 선택하게 되는 설문의 대상을 관리자가 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 '을 선택합니다.
3. '설문 대상' 항목 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 적절한 값을 선택합니다.
   - 전체+멤버: 설문 작성자가 전체와 멤버중 설문의 대상자를 선택할 수 있게 합니다.
   - 전체: 설문 작성자가 전체 대상의 설문만 만들 수 있습니다. 멤버를 선택해서 설문할 수 없습니다.
   - 멤버: 설문 작성자가 선택한 멤버 대상의 설문만 만들 수 있습니다.
4. '저장'을 눌러 선택값을 저장합니다.

# 응답자 정보

구성원이 설문을 만들때 선택하게 되는 응답자 정보를 관리자가 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 '을 선택합니다.
3. '응답자 정보' 항목 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 적절한 값을 선택합니다.
   - 실명(기본)+익명: 설문 응답자가 실명으로 답변하게 할지 익명으로 답변하게 할지 설문 작성자가 선택할 수 있습니다. 새 설문 생성 시 실명을 기본 설정값으로 설정합니다.
   - 실명+익명(기본): 설문 응답자가 실명으로 답변하게 할지 익명으로 답변하게 할지 설문 작성자가 선택할 수 있습니다. 새 설문 생성 시 익명을 기본 설정값으로 설정합니다.
   - 실명: 설문 응답자가 실명으로 응답하게 합니다. 설문 작성자가 변경할 수 없습니다.
   - 익명: 설문 응답자가 익명으로 응답하게 합니다. 설문 작성자가 변경할 수 없습니다.
4. '저장'을 눌러 선택값을 저장합니다.

# 응답자에게 결과 공유

설문 작성자가 설문을 만들때 설문설정에서 응답자에게 요약 결과 또는 상세결과 공개를 할 수 있도록 할지 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 '을 선택합니다.
3. '응답자에게 결과 공유' 항목 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 적절한 값을 선택합니다.
   - 허용: 설문 작성자가 설문을 만들때 응답자에게 설문 결과 공개를 허용합니다.
   - 허용 안 함: 설문의 작성자는 응답자에게 설문의 결과를 공개할 수 없습니다.
4. '저장'을 눌러 선택값을 저장합니다.

# 중복 응답 허용

설문 작성자가 설문을 만들때 중복 응답이 가능한 설문을 만들지 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 '을 선택합니다.
3. '중복 응답' 항목 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 적절한 값을 선택합니다.
   - 허용: 설문 작성자가 설문을 만들때 설문설정에서 응답자가 같은 설문에 대해 중복 응답을 하게 할지, 못하게 할지 선택할 수 있게 허용합니다.
   - 허용 안 함: 설문의 작성자는 중복 응답을 허용할 수 없습니다.
4. '저장'을 눌러 선택값을 저장합니다.

# 응답 수정 허용

제출한 응답을 수정할 수 있게 할지를 설문 작성자가 설문을 만들 때 설정합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 '을 선택합니다.
3. '응답 수정' 항목 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 적절한 값을 선택합니다.
   - 허용: 설문 작성자가 설문을 만들때 설문설정에서 응답자가 제출한 응답을 수정하게 할지, 못하게 할지 선택할 수 있게 허용합니다.
   - 허용 안 함: 설문의 작성자는 응답 수정을 허용할 수 없습니다.
4. '저장'을 눌러 선택값을 저장합니다.

# 설문 결과 화면 공유

설문 작성자에게 설문 결과 화면을 공유할 수 있게 허용할지 설정하고, 공유 방법을 선택합니다.

1. Admin 왼쪽 메뉴에서 '서비스'를 선택해 메뉴를 펼친 후 '설문'을 눌러 '설문' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/2020-07-03-164523.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '일반 '을 선택합니다.
3. '결과화면 공유' 항목 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러, 적절한 값을 선택합니다.
   - 허용: 설문 작성자가 설문 결과 화면을 공유할 수 있도록 허용합니다. 링크 공유와 csv파일 공유 중 1개 이상을 선택할 수 있습니다.
   - 허용 안 함: 설문 작성자가 설문 결과 화면을 공유할 수 없습니다.
4. '저장'을 눌러 선택값을 저장합니다.
