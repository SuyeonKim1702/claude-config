# 설치된 앱 관리

Apps에서 설치된 앱의 목록을 모아서 볼 수 있습니다.

# 설치된 앱 목록 보기

관리자는 사내에 설치된 앱의 목록을 보고 관리할 수 있습니다.

1. Admin 왼쪽 메뉴에서 'Apps'를 선택해 메뉴를 펼친 후, '설치된 앱 관리'를 눌러 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 설치된 앱의 정보를 확인하거나, 앱을 관리할 수 있습니다.

# 앱 요금제/라이선스 수 변경

구매한 라이선스 단위로 요금이 부과되는 앱인 경우, 관리자는 앱의 요금제를 변경할 수 있습니다.

1. Admin 왼쪽 메뉴에서 'Apps'를 선택해 메뉴를 펼친 후, '설치된 앱 관리'를 눌러 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 요금제를 변경할 앱을 누릅니다.
3. '이용 현황' 메뉴에서 변경하기 버튼을 눌러 요금제/라이선스 수 변경 화면으로 이동합니다.
4. 변경할 요금제를 선택하고, 라이선스 수를 입력합니다.
5. 하단의 '저장'을 누릅니다.

# 앱 사용 구성원 수정

관리자는 앱을 사용할 구성원의 목록을 수정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 'Apps'를 선택해 메뉴를 펼친 후, '설치된 앱 관리'를 눌러 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 사용 가능 구성원을 수정하려는 앱을 누릅니다.
3. '이용 권한'에서 앱을 사용하는 구성원을 추가하거나 제외합니다.
4. 하단의 '저장'을 누릅니다.

# 앱 삭제

관리자는 Apps에 추가된 앱을 삭제할 수 있습니다.

앱을 삭제하는 경우, 구성원들이 더 이상 네이버웍스에서 앱을 사용할 수 없게 됩니다.

1. Admin 왼쪽 메뉴에서 'Apps'를 선택해 메뉴를 펼친 후, '설치된 앱 관리'를 눌러 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 제거하려는 앱을 누릅니다.
3. '제거하기'를 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
