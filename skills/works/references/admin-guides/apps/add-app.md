# 앱 추가

Apps는 네이버웍스와 연동할 수 있는 외부 솔루션들을 모아 놓은 공간입니다.

Apps에서 업무 효율을 높일 수 있는 다양한 외부 솔루션들을 네이버웍스와 연동하여 업무를 효율화하거나 업무 프로세스를 간소화할 수 있습니다.

# 앱 정보

Apps에서 특정 앱을 선택해 앱 정보를 확인할 수 있습니다.

|  |  |
| --- | --- |
| 과금 정보 | - 무료: 네이버웍스에서 과금되지 않는 앱입니다. - 유료: 네이버웍스에서 과금되는 앱입니다. 요금이 네이버웍스 청구서로 통합 청구되며, 개발사에게 전달됩니다. |
| 앱 타입 | - 메시지봇: 메시지방에서 대화할 수 있는 봇 형태의 앱입니다. - 메시지 애드온: 메시지방에서 화면을 열어 현재 메시지방으로 메시지를 전송할 수 있는 앱입니다. - 외부 연동 앱: 네이버웍스 계정으로 로그인할 수 있는 앱입니다. |
| 호환 서비스 | - 이 앱과 연계되는 네이버웍스의 서비스입니다. |

# 앱 추가

관리자는 구성원들이 네이버웍스에서 앱을 사용할 수 있도록 앱을 추가할 수 있습니다.

앱 상세 페이지에서는 3가지의 상태로 앱이 노출됩니다.

|  |  |
| --- | --- |
| 추가하기 | 관리자가 앱을 추가하기 전 상태로, 앱을 추가해야 사용 가능합니다. |
| 사용불가 | 이용 중인 상품이 해당 앱을 지원하지 않아 사용 불가능한 상태입니다. |
| 관리하기 | 이미 관리자가 앱 추가를 완료한 앱입니다. |

이 중 '추가하기' 상태인 앱만 추가할 수 있습니다.

1. Admin 왼쪽 메뉴에서 'Apps'를 선택합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 네이버웍스에서 추가를 원하는 앱을 찾아 누릅니다.
3. 앱 상세페이지 화면에서 '추가하기'를 누릅니다.
4. 앱의 접근 권한을 확인하고 이용 약관에 동의한 뒤 '다음'을 누릅니다.
5. 앱을 사용할 구성원을 선택합니다.
6. '저장'을 누릅니다.  
   선택한 구성원들이 네이버웍스에서 앱을 사용할 수 있는 상태가 됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
