# 앱 신청 내역

구성원이 이용 권한 또는 추가를 요청한 앱의 신청 내역을 볼 수 있습니다.

관리자는 구성원이 신청한 앱을 승인한 후 이용 권한을 부여하거나, 새로 앱을 추가할 수 있습니다.

신청 내역은 승인 또는 거절하지 않으면 계속 유지됩니다.

구성원은 앱이 승인되거나 거절된 이후에도 다시 신청할 수 있습니다.

# 이미 추가된 앱의 권한 신청 관리

이미 네이버웍스에 추가된 앱에 대해 구성원이 신청한 이용 권한을 승인하거나 거절할 수 있습니다.

1. Admin 왼쪽 메뉴에서 'Apps'를 선택해 메뉴를 펼친 후, '앱 신청 내역'을 누릅니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 이미 네이버웍스에 추가된 앱은 '모두 승인' 또는 '모두 거절' 버튼이 노출됩니다. 모든 권한을 한번에 승인/거절하거나, 앱을 눌러 신청한 구성원의 목록을 확인하고 개별적으로 승인하거나 거절합니다.
   - 모두 승인: 이 앱의 이용 권한을 신청한 모든 구성원에게 권한을 부여합니다.
   - 모두 거절: 이 앱의 이용 권한을 신청한 모든 구성원에게 권한을 부여하지 않습니다.

# 아직 추가되지 않은 앱의 신청 관리

아직 네이버웍스에 추가되지 않은 앱에 대해 구성원의 요청대로 앱을 추가하거나 거절할 수 있습니다.

1. Admin 왼쪽 메뉴에서 'Apps'를 선택해 메뉴를 펼친 후, '앱 신청 내역'를 눌러 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 아직 네이버웍스에 추가되지 않은 앱은 '미설치' 및 '모두 거절' 버튼이 노출됩니다. 앱을 누른 뒤 구성원 목록에서 신청을 구성원별로 거절할 수도 있습니다.
   - 앱 추가: 앱을 누른 뒤 오른쪽 상단의 '앱 상세' 버튼을 누른 후, [상세 페이지에서 앱을 추가](/ko/admin-guides/apps/add-app/)합니다.
   - 모두 거절: 앱 추가 신청을 모두 거절합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
