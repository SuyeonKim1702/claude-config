# 모바일 외부 메일 연동

네이버웍스 모바일앱으로 네이버웍스가 아닌 Google Workspace, Microsoft 365 등 다른 메일 서비스의 메일을 수신할 수 있습니다.

기존에 사용하던 메일 서비스를 모바일 환경에서 네이버웍스 하나의 앱으로 메시지 등 다른 서비스와 함께 사용하려고 할 때 유용합니다.

모바일 외부 메일 연동을 사용할 수 있는 조건은 다음과 같습니다.

- Lite 상품을 도메인으로 이용하는 경우
- Standard 또는 Standard Plus 상품으로 네이버웍스 메일을 사용하지 않는 경우
- 네이버웍스 계정과 사용하던 외부 메일 서비스의 메일 주소가 동일한 경우
  - 네이버웍스의 ID 계정이 “member1@companyA.com”인 구성원이 외부 메일을 연동하는 경우 동일한 외부 메일주소(member1@companyA.com)로만 연동 가능
  - ID 부분이 다르거나 (member2@companyA.com) 또는 도메인 부분이 다른 (member1@companyB.com) 외부 메일 주소로는 연동할 수 없음
- 네이버웍스에서 제공하는 도메인(by-works.com)을 [기본 도메인](/ko/admin-guides/settings/company-information/#i)으로 사용하지 않는 경우

사용하려면 다음과 같이 설정합니다.

### 모바일 앱

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '서비스 권한'을 눌러 '서비스 권한' 화면으로 이동합니다. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '기본권한' 오른쪽~~![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-more-line-434343.png)~~ 아이콘을 눌러, '권한 수정'을 선택합니다.
3. '외부 메일 연동'을 '사용'으로 설정합니다.
4. 활성화된 '서버 설정' 버튼을 선택합니다.
5. 서버 설정 방법을 선택 후 '저장'을 선택합니다.
   - Google Workspace, Microsoft 365 메일 서비스를 이용하는 경우: 해당 서비스를 탭해 선택합니다.
   - 이 외의 메일 서비스를 이용하는 경우: '수동 설정'을 탭해 수동 설정 정보를 입력합니다.
     - 수신 서버(IMAP): 받은 메일 서버의 정보를 입력합니다.
     - 발송 서버(SMTP): 발신 메일 서버의 정보를 입력합니다.
6. 설정 완료 후 페이지 상단의 '저장'을 탭해 변경 사항을 저장합니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '서비스 권한'을 눌러 '서비스 권한' 화면으로 이동합니다.
2. 권한템플릿 목록에서 '기본권한'을 선택합니다.
3. '권한 설정'에서 '권한 수정'을 클릭해 수정 모드로 변경합니다.
4. '외부 메일 연동'을 '사용'으로 설정합니다.
5. 활성화된 '서버 설정' 버튼을 클릭합니다.
6. 서버 설정 방법을 선택 후 '저장'을 클릭합니다.
   - Google Workspace, Microsoft 365 메일 서비스를 이용하는 경우: 해당 서비스를 클릭해 선택합니다.
   - 이 외의 메일 서비스를 이용하는 경우: '수동 설정'을 클릭해 수동 설정 정보를 입력합니다.
     - 수신 서버(IMAP): 받은 메일 서버의 정보를 입력합니다.
     - 발송 서버(SMTP): 발신 메일 서버의 정보를 입력합니다.
7. 설정 완료 후 페이지 상단의 '저장'을 클릭해 변경 사항을 저장합니다.

설정을 마친 다음, 모바일앱에 로그인한 구성원이 메일 화면에서 '외부 메일 연동'을 탭해 로그인 정보를 입력하면 외부 메일과 연동이 완료됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
