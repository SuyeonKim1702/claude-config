# 클로바노트 공유 링크 관리

이 기능은 클로바노트 상품 신청 후 사용 가능합니다.

회사 구성원이 공유하고 있는 노트를 조회하고, 공유 링크를 삭제할 수 있습니다.

# 공유 링크 확인

구성원이 공유하고 있는 노트의 링크 및 관련 정보를 확인할 수 있습니다.

Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '{{clovanote}} 공유 링크 관리'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.

기본값으로 오늘 생성된 공유 링크 목록이 노출됩니다.

- 노트 제목: 공유된 노트의 이름입니다. 제목을 클릭해 상세 정보를 확인할 수 있습니다.
- 공유 링크: 노트에 접근하기 위한 링크입니다. 제목을 클릭한 후, 상세 정보에서 전체 링크 주소를 확인할 수 있습니다.
- 생성자: 공유 링크를 만든 구성원의 이름입니다.
- 접근 권한: 공유 링크에 접근할 수 있는 권한으로 '링크를 공유 받은 모든 사용자', '접근 권한을 가진 모든 사용자' 중 구성원이 링크 생성 시 선택한 값이 표시됩니다.
- 생성일: 현재 공유되어 있는 노트 링크가 생성된 날짜와 시간입니다.
- 관리: 관리자는 링크를 삭제할 수 있습니다. 공유 링크가 삭제되더라도, 노트 자체는 삭제되지 않습니다.

# 공유 링크 삭제

공유 중인 링크를 삭제합니다.

공유 링크가 삭제되더라도, 노트 자체는 삭제되지 않습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '{{clovanote}} 공유 링크 관리'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 삭제할 노트 공유 링크 오른쪽의 '관리' 항목에서 '삭제'를 누릅니다.
3. 주의 사항을 확인 후 '삭제'를 선택합니다.

# 검색 조건 설정

검색 조건을 입력하여 공유 링크를 빠르게 확인합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '{{clovanote}} 공유 링크 관리'를 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단의 '상세'를 눌러 조건에 필요한 값을 선택하거나 입력합니다.
   - 생성일: 공유 노트의 링크가 생성된 기간을 선택합니다.
   - 노트 제목: 공유 노트의 제목을 입력합니다.
   - 링크 생성자: 생성자의 이름을 입력합니다.
   - 접근 권한: '링크를 공유 받은 모든 사용자, '접근 권한을 가진 모든 사용자' 중 선택할 수 있습니다.
   - 공유 링크: 공유 노트의 링크를 입력합니다.
3. '검색'을 눌러 조건에 맞는 공유 링크를 검색합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
