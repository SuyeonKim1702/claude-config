# 최고 관리자 권한의 위임

최고 관리자 권한을 다른 구성원에게 위임합니다.

위임 요청은 이메일을 통해 전달됩니다.

// start of content for ko en jp

메일을 사용하지 않는 Free, Lite 상품의 경우 최고 관리자 권한을 위임받을 구성원의 개인 이메일 주소가 등록되어 있어야만 위임할 수 있습니다.

// end of content for ko en jp

// start of content for tw

메일을 사용하지 않는 상품의 경우 최고 관리자 권한을 위임받을 구성원의 개인 이메일 주소가 등록되어 있어야만 위임할 수 있습니다.

// end of content for tw

개인 이메일 주소는 [구성원 정보 수정](/ko/admin-guides/manage-members/members/update-member-information/)을 통해서 등록할 수 있습니다.

최고 관리자 권한 위임 요청을 받은 구성원이 요청을 승낙하면 위임이 완료되며, 최고 관리자 권한은 단  1명에게만 부여되므로 기존 최고 관리자의 권한은 사라집니다.

# 최고관리자 권한 위임하기

최고 관리자 권한을 다른 구성원에게 위임합니다.

위임 받을 구성원이 14일 내에 위임을 수락하면 위임이 완료됩니다.

// start of content for ko en jp

권한을 위임하면 최고 관리자의 권한 항목 값이 '권한 위임'에서 '위임 대기'로 변경됩니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 관리자 권한 목록에서 '최고관리자'를 선택합니다.
3. '권한 설정'과 '관리자' 중 '관리자'를 선택합니다.
4. '위임'을 눌러 '권한 위임' 화면으로 이동합니다.
5. 이름 또는 ID로 검색해 최고관리자 권한을 위임할 구성원을 선택합니다. 메일을 사용하지 않는 Free, Lite 상품을 사용하고 있다면 위임 받을 구성원의 개인 이메일 주소가 등록되어 있어야 합니다.
6. 위임할 구성원을 선택했으면 '다음'을 눌러 위임 요청을 보냅니다. 위임 받을 구성원에게 위임 내용이 메일로 발송됩니다.

# 최고관리자 권한 위임 취소하기

위임 요청 후 위임을 받은 구성원이 수락하기 전까지는 위임 요청을 취소할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 관리자 권한 목록에서 '최고관리자'를 선택합니다.
3. '권한 설정'과 '관리자' 중 '관리자'를 선택합니다.
4. '위임 대기'를 눌러 '권한 위임' 화면으로 이동합니다.
5. 최고관리자 권한 위임을 취소할지 확인하는 화면이 나타나면 '위임 취소'을 선택해서 위임 취소를 완료합니다.

# 최고관리자 권한 수락하기

위임 받을 구성원이 권한 위임 요청을 수락합니다.

위임 받을 구성원이 등록한 메일로 위임 요청 메일이 전달됩니다.

// start of content for ko en jp

수락 기한인 14일 내 위임 요청이 수락되지 않으면 관리자 권한 위임은 자동 취소됩니다.

1. 위임 받을 구성원은 네이버웍스 메일에서 위임 안내 메일을 확인합니다. 메일을 사용하지 않는 Free, Lite 상품인 경우 등록된 개인 이메일로 발송된 위임 안내 메일을 확인합니다.
2. 이메일 하단의 '관리자 권한 수락하기'를 클릭합니다.
3. 관리자 필수 입력정보인 개인 이메일이 등록되지 않았다면 개인 이메일을 등록하고, 휴대폰 번호를 이용해 본인 인증을 합니다. 입력한 개인 이메일과 인증된 휴대폰 번호를 구성원 정보로 자동 저장됩니다.
4. '관리자 위임 수락'를 클릭합니다.
