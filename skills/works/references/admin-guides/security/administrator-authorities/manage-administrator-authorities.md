# 관리자 권한 관리

이 기능은 유료 플랜에서만 사용할 수 있습니다.

네이버웍스에서 Admin에 접속하고 관리 기능을 이용하기 위해서는 관리자로 지정되어야 합니다.

관리자는 설정된 관리자 권한에 따라 각각 다른 메뉴에 접근할 수 있습니다.

기본으로 최고관리자, 부관리자, 운영 관리자의 관리자 권한이 제공되고, 그 외 새로운 관리자 권한을 만들어서 권한을 맞춤으로 설정할 수 있습니다.

2023년 2월 16일 이후에 개설한 도메인부터는 IT관리자, 인사관리자 권한을 제공하지 않습니다.

- 최고관리자: Admin의 모든 메뉴에 접근할 수 있으며, 네이버웍스에서 탈퇴하는 등 관리자에게 허용된 모든 권한을 가지고 있습니다.
  - 모든 네이버웍스에는 1명의 최고관리자가 지정되어 있어야 합니다.
  - 처음 네이버웍스를 신청한 관리자가 기본적으로 최고관리자가 됩니다.
  - 최고관리자 본인은 다른 구성원에게 최고관리자 권한을 위임할 수 있습니다.
- 부관리자, 운영 관리자: 기본적으로 제공하는 이 관리자 권한에 대해서는 권한 설정을 변경할 수 없습니다. 각 관리자 권한에 따라 아래 메뉴에 접근할 수 있습니다.

  # 관리자 권한 생성, 수정, 삭제

  기본적으로 제공하는 관리자 권한인 최고관리자, 부관리자, 운영 관리자 이외에 새로운 어드민 권한을 만들어 관리할 수 있습니다.

  새로 생성한 권한은 어떤 메뉴에 접근을 허용할지 설정할 수 있습니다.

  ## 새 관리자 권한 만들기

  새로운 관리자 권한을 만들어서 특정 메뉴에만 접근할 수 있도록 설정할 수 있습니다.

  1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
  2. 관리자 권한 목록에서 '권한 추가'를 누릅니다.
  3. '권한 추가' 창에서 권한 이름을 입력하고, '확인'을 선택하면 목록에서 새로운 관리자 권한이 생성됩니다.

  새로 추가된 관리자 권한은 모든 메뉴에 접근할 수 있는 상태로 생성됩니다.

  ## 관리자 권한명 수정하기

  관리자 권한의 이름을 수정합니다. 기본으로 제공되는 최고관리자, 부관리자, 운영 관리자 권한의 이름은 수정할 수 없습니다.

  ### 모바일 앱

  1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다. ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
  2. 관리자 권한 목록에서, 수정할 관리자 권한을 눌러 선택합니다.
  3. 관리자 권한명의 오른쪽 옆에 위치한 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 누른 후 '권한명 수정'을 선택합니다.
  4. '권한명 수정' 창에서 변경할 이름을 입력하고 '확인'을 누릅니다.

  ### PC 웹

  1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다.
  2. 화면 왼쪽의 관리자 권한 목록에서, 수정할 관리자 권한 위에 마우스 포인터를 올립니다.
  3. 화면 오른쪽에서 관리자 권한의 오른쪽에 나타나는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 누른 후 '권한명 수정'을 선택합니다.
  4. '권한명 수정' 창에서 변경할 이름을 입력하고 '확인'을 누릅니다.

  ## 관리자 권한 삭제하기

  생성한 관리자 권한을 삭제합니다.

  기본으로 제공되는 최고관리자, 부관리자, 운영 관리자 권한은 삭제할 수 없습니다.

  ### 모바일 앱

  1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다. ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
  2. 관리자 권한 목록에서, 삭제할 관리자 권한을 눌러 선택합니다.
  3. 관리자 권한명의 오른쪽 옆에 위치한 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 누른 후 '권한 삭제'를 선택합니다.

  ### PC 웹

  1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다.
  2. 화면 왼쪽의 관리자 권한 목록에서, 삭제할 관리자 권한 위에 마우스 포인터를 올립니다.
  3. 관리자 권한의 오른쪽에 나타나는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 누른 후 '권한 삭제'를 선택합니다.

  # 관리자 권한 설정하기

  각 관리자 권한별로 해당하는 관리자가 접근할 수 있는 메뉴 권한을 변경할 수 있습니다.

  // start of content for ko

  직접 생성한 관리자 권한에 한해서만 권한 설정을 변경할 수 있습니다.

  1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
  2. 관리자 권한 목록에서, 수정할 관리자 권한을 눌러 선택합니다.  
     근태, 결재 등 경영지원 상품을 이용 중이라면, 수정할 Admin 메뉴를 드롭다운 메뉴에서 한번 더 선택합니다.
  3. '권한 설정'을 선택하고 오른쪽의 '권한 수정'을 누릅니다.
  4. 접근을 허용할 Admin 메뉴를 선택합니다.
     - 메뉴별로 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/02/아래-버튼.png) 아이콘을 눌러서 하위 메뉴 단위까지 상세하게 설정할 수 있습니다.
     - 특정 메뉴에 접근하지 못하게 하려면 선택을 해제합니다.
     - 경영지원 상품의 경우, 이용 가능한 모든 메뉴가 제공되며 일부 메뉴는 서비스 내 설정에 따라 권한이 제한됩니다.
  5. 변경할 메뉴를 모두 설정한 뒤, 상단의 '저장' 버튼을 누릅니다.

  # 관리자 권한 부여 및 해제

  구성원에게 관리자 권한을 부여하거나 부여된 권한을 해제합니다.

  - 한 구성원은 여러 개의 관리자 권한을 가질 수 있습니다.
  - 부여된 여러 개의 관리자 권한에서 허용된 모든 메뉴에 접근할 수 있습니다.

  ## 관리자로 지정하기

  관리자 권한을 선택한 뒤, 이 권한을 부여받을 구성원을 관리자 목록에 추가합니다.

  1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
  2. 관리자 권한 목록에서, 부여할 관리자 권한을 눌러, 이 관리자 권한을 관리할 수 있는 화면으로 이동합니다.
  3. '관리자'를 누릅니다.
  4. 오른쪽의 '관리자 추가'를 누릅니다.
  5. 구성원 선택 창이 뜨면 이 관리자 권한으로 지정할 구성원을 선택한 뒤 '확인'을 누릅니다.

  ## 관리자 권한 해제하기

  이 관리자 권한을 가진 관리자 중에서 이 관리자 권한을 회수합니다.

  이 구성원에게 지정된 다른 관리자 권한은 회수되지 않습니다.

  1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '관리자 권한'을 눌러 '관리자 권한' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
  2. 화면 왼쪽의 관리자 권한 목록에서, 부여할 관리자 권한을 눌러 선택합니다.
  3. '관리자'를 누릅니다.
  4. 관리자 목록에서 이 관리자 권한을 회수할 구성원에 대하여 '해제'를 누릅니다.
  5. 관리자 권한을 해제할지 묻는 창이 뜨면 '확인'을 누릅니다.
