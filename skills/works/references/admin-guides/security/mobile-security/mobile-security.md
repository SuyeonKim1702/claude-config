# 모바일 보안

이 기능은 유료 플랜에서만 사용할 수 있습니다.

네이버웍스 모바일 앱을 안전하게 사용할 수 있도록 회사 차원의 보안 정책을 설정합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '모바일 보안'을 눌러 '모바일 보안' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 각 항목별로 오른쪽의 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 설정을 변경합니다.
3. 상단의 '저장'을 눌러 변경한 내용을 저장합니다.

# 화면 잠금

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품, 웍스 결재 상품 신청 후 사용 가능합니다.

구성원이 네이버웍스 모바일 앱을 실행할 때 화면을 잠그도록 설정합니다.

|  |  |
| --- | --- |
| 상세 설정 | 옵션 |
| 필수 여부 | 관리자가 이 기능을 '필수'로 설정한 경우는 모든 구성원에게 화면 잠금이 적용되며, '선택'으로 설정한 경우는 구성원의 선택에 따라 잠금 여부를 설정할 수 있습니다. '필수'로 설정을 변경하면 다음 모바일 앱 실행 시 반드시 PIN을 등록해야 합니다.   - 필수 - 모든 구성원이 반드시 화면 잠금 설정 - 선택 - 구성원 설정에 따름 |
| PIN 형식 | 관리자가 선택한 형식으로 PIN을 설정하도록 지정할 수 있습니다. PIN 형식이 변경되면 다음 모바일 앱 실행 시 새 PIN을 등록해야 합니다.   - 숫자 4자 - 숫자 또는 영문 6자 - 숫자 또는 영문 8자 |
| 해제 시도 횟수 제한 | 잘못된 PIN을 입력했을 때 로그아웃되지 않고 다시 시도할 수 있는 최대 횟수를 지정할 수 있습니다. 설정을 변경한 뒤 다음 잠금 해제 시 부터 적용됩니다.  - 3회 ~ 10회 실패 시 모바일 앱 로그아웃 |

# 기능 제한

## 로그인 유지 기간

구성원이 모바일 앱에 로그인 한 뒤, 얼마 동안 모바일 앱을 이용하지 않으면 자동으로 로그아웃될지 설정할 수 있습니다.

설정을 변경한 뒤, 구성원이 다음 번 모바일 앱에 로그인한 시점부터 적용됩니다.

## 데이터 저장 기간

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 신청 후 사용 가능합니다.

모바일 앱에서 데이터를 저장하고 조회할 수 있는 기간을 설정합니다.

|  |  |
| --- | --- |
| 옵션 | 내용 |
| 제한 안 함 | 전체 기간의 데이터를 조회할 수 있습니다. |
| 최근 3/ 7/ 30일 내의 데이터만 조회 | - 메시지: 설정한 기간 이내의 대화내역만 조회할 수 있습니다. 메시지 검색에서도 설정한 기간 이내의 대화만 대상으로 검색됩니다. - 노트: 설정한 기간 이내의 게시물만 노트에 남고 조회할 수 있습니다. - 게시판: 설정한 기간 이내에 등록된 게시물만 목록에 남고 조회할 수 있습니다. - 캘린더: 설정한 기간 내의 개인 및 구성원 일정과 할 일만 볼 수 있습니다. - 메일: 설정한 기간 내의 메일을 조회하고 검색할 수 있습니다. - 다운로드한 파일: 설정한 기간 이내에 다운로드한 파일만 구성원의 기기에 남습니다. |

## 이미지/텍스트 복사 제한

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 신청 후 사용 가능합니다.

모바일 앱 내의 이미지/텍스트를 클립보드에 복사하지 못하도록 설정할 수 있습니다.

|  |  |
| --- | --- |
| 상세 설정 | 옵션 |
| 제한 안 함 | 모바일 앱에서 이미지/텍스트를 클립보드에 복사할 수 있습니다. |
| 제한 | 모바일 앱의 읽기 영역에서는 이미지와 텍스트를 클립보드에 복사할 수 없으며, 게시글이나 메일 작성 시에는 쓰기 영역에서도 복사할 수 없습니다. 단, 메시지를 작성할 때 또는 OS에서 기본으로 제공하는 뷰어 및  편집기에서는 이미지/텍스트 복사가 제한되지 않습니다. |

모바일 앱에서 파일 업로드를 제한하는 기능은 '[보안 > 파일 보안](/ko/admin-guides/security/file-security/)'에서 확인할 수 있습니다.

# 모바일 화면 캡처 제한

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품, 웍스 결재 상품 신청 후 사용 가능합니다.

모바일 기기로 사내 보안 정보가 유출되는 것을 방지하기 위해, 모바일에서 네이버웍스 앱 화면을 캡처하는 것을 제한합니다.

모바일 앱 화면 캡처 제한은 네이버웍스 앱을 실행 중일 때만 적용됩니다.

|  |  |
| --- | --- |
| 옵션 | 내용 |
| iOS: 화면 캡처 시 감사 로그 저장, Android: 화면 캡처 차단 | - iOS: 모바일 앱 화면을 캡처할 수 있으나, 사용자에게 캡처 화면을 삭제하라는 경고가 노출되며, 동시에 캡처 상세 이력(캡처한 사용자, 디바이스, 캡처한 화면 원본 이미지 등)이 ['감사 > 모바일 화면 캡처'](/ko/admin-guides/audit/mobile-capture/)에 저장됩니다. - Android: 모바일 앱 화면을 캡처하려는 경우, 경고 안내가 노출되면서 화면을 캡처할 수 없으며, 이력도 기록되지 않습니다. 외부 앱으로 캡처할 경우, 캡처 화면이 검은색 빈 화면으로 저장됩니다. |
| iOS, Android: 화면 캡처 시 감사 로그 저장 | - iOS & Android 공통: 모바일 앱 화면을 캡처할 수 있으나, 사용자에게 캡처 화면을 삭제하라는 경고가 노출되며, 동시에 캡처 상세 이력(캡처한 사용자, 디바이스, 캡처한 화면 원본 이미지 등)이 ['감사 > 모바일 화면 캡처'](/ko/admin-guides/audit/mobile-capture/)에 저장됩니다.   - Android의 경우 이미지 캡처 탐지는 Android 14.0 이상부터 가능합니다. 14.0 미만 버전은 지원하지 않습니다.   - Android의 경우 동영상 캡처 탐지는 Andriod 15.0 이상부터 드라이브 앱에서만 가능합니다. 네이버웍스 모바일 앱에서는 지원하지 않습니다. |

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
