# 파일 보안

이 기능은 유료 플랜에서만 사용할 수 있습니다.

네이버웍스 서비스 내에서 사내 규정에 따라 파일을 안전하게 공유할 수 있도록 회사 차원의 파일 보안 정책을 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '파일 보안'을 눌러 '파일 보안' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 각 항목별로 오른쪽의 버튼을 눌러 설정을 변경합니다.
3. 상단의 '저장'을 눌러 변경한 내용을 저장합니다.

# 모바일앱 파일 다운로드 제한

네이버웍스 모바일앱에서 기기로 파일을 다운로드하지 못하도록 제한할 수 있습니다. 각 서비스별로 파일 다운로드를 제한할지 설정할 수 있습니다.

파일 다운로드를 제한하는 경우,

- 첨부/전송된 문서 파일을 뷰어로 미리볼 수 있지만, 다운로드하는 버튼이 나타나지 않습니다.
- 이미지/동영상 파일을 뷰어로 볼 수 있지만, 다운로드하는 버튼이 나타나지 않습니다.
- 인 앱 브라우저, 서비스 알림과 같은 공통 영역에서도 파일 다운로드가 제한됩니다.

이 기능은 '[보안 > 서비스 권한](/ko/admin-guides/security/authority/authority/)'에서 각 권한 템플릿별로 적용 여부를 다르게 설정할 수 있습니다.

# 모바일앱 파일 업로드 제한

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품, 웍스 결재 상품 신청 후 사용 가능합니다.

모바일 기기에서 네이버웍스 모바일앱으로 파일을 업로드하지 못하도록 제한할 수 있습니다.

- 모바일앱에서 기기의 파일을 첨부할 수 있는 파일/영상/사진 선택 버튼 및 사진/영상 촬영 버튼이 나타나지 않습니다.
- 모바일앱의 주소록에서 기기의 사진을 선택하는 버튼이 나타나지 않습니다.
- 외부 앱에서 파일을 공유하려 할 때, 파일을 보낼 서비스 중 네이버웍스 모바일앱이 나타나지 않습니다.

# 워터마크 표시

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품, 웍스 결재 상품 신청 후 사용 가능합니다.

'워터마크' 설정을 켜면 모바일앱에서 이미지, 문서, 동영상을 볼 때 네이버웍스 모바일 문서 뷰어로 열리고, 워터마크가 표시됩니다.

워터마크를 통해 사내 정보 유출 시 출처를 확인할 수 있습니다.

- 이미지, 문서, 동영상을 보는 사람의 회사명, 계정, 문서 뷰어가 열린 일시가 워터마크로 표시됩니다.
- 업로드/편집 시에는 워터마크가 표시되지 않습니다.
- 드라이브 링크 공유(누구나, 비밀번호, 보안 OTP) 등을 통해 로그인하지 않고 문서 뷰어를 여는 경우, 워터마크가 표시되지 않습니다.

# 파일 확장자 차단

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품, 웍스 결재 상품 신청 후 사용 가능합니다.

악성코드 유포에 활용될 수 있는 특정 확장자의 파일을 네이버웍스 서비스 내에서 업로드 · 전송하거나 다운로드하지 못하도록 설정할 수 있습니다.

- 차단을 적용할 서비스를 지정할 수 있습니다.
- 차단할 확장자 형식을 지정할 수 있습니다.

저장 후 서비스에 반영까지 최대 1시간이 소요될 수 있습니다.

## 차단 적용할 서비스 설정하기

선택한 서비스에서 '차단할 확장자'로 지정된 형식의 파일은 공유하지 못하게 됩니다.

|  |  |
| --- | --- |
| 상세 설정 | 내용 |
| 메시지 | - 차단된 형식의 파일을 메시지로 전송할 수 없습니다. - 차단된 형식의 파일을 메시지방에서 저장할 수 없습니다. - 차단된 형식의 파일을 다른 메시지방으로 전달할 수 없습니다. |
| 게시판 | - 차단된 형식의 파일을 노트에 첨부할 수 없습니다. - 차단된 형식의 파일이 첨부된 노트를 수정할 수 없습니다. 수정하려면 첨부된 파일을 제거해야 합니다. - 차단된 형식의 첨부파일은 저장할 수 없습니다. |
| 메일 | - 차단된 형식의 파일이 첨부된 메일을 수신하지 않습니다. 단, 모니터링의 IP관리에서 [수신허용 IP로 등록](/ko/admin-guides/monitoring/received-mail/#_IP)된 IP에서 보낸 메일은 차단된 형식의 파일이 첨부되어 있어도 수신합니다. - 차단된 형식의 파일을 메일이나 메모에 첨부할 수 없습니다. - 이미 수신한 메일에 차단된 형식의 파일이 첨부된 경우, 파일을 열거나 저장할 수 없습니다. - 이미 수신한 메일이나 임시저장한 메일, 메모, 보낸 메일에 차단된 형식의 파일이 첨부된 경우, 이 메일을 전달하거나 재작성 시 차단된 파일이 제거됩니다.   IP 관리에서 수신허용 IP 주소로 등록된 IP 주소에서 보내진 메일은 차단된 형식의 파일이 첨부되어 있어도 수신됩니다. |
| 캘린더 (메시지방 일정) | - 차단된 형식의 파일을 업로드할 수 없습니다. - 차단된 형식의 파일을 다운로드할 수 없습니다. |
| 드라이브 (메시지방 폴더) | - 차단된 형식의 파일을 업로드할 수 없습니다. - 차단된 형식의 파일을 다운로드할 수 없습니다. - 차단된 형식의 파일을 열거나 복사할 수 없습니다. - 다른 네이버웍스 서비스에 첨부된 차단된 형식의 파일을 드라이브에 저장할 수 없습니다.   드라이브의 파일을 다른 서비스로 첨부하거나 드라이브에 파일 저장하는 등 서비스간 연동 시, 양쪽 서비스 중 하나라도 제한되어있으면 파일이 차단됩니다. |
| 설문 | - 차단된 형식의 파일을 업로드할 수 없습니다. - 차단된 형식의 파일을 다운로드할 수 없습니다. |
| 결재 | - 차단된 형식의 파일을 업로드할 수 없습니다. - 차단된 형식의 파일을 다운로드할 수 없습니다. |
| 영업 | - 차단된 형식의 파일을 업로드할 수 없습니다. - 차단된 형식의 파일을 다운로드할 수 없습니다. |

## 차단할 파일 형식 관리하기

- 악성코드나 바이러스의 유입과 확산을 막기 위해 차단을 권장하는 기본 확장자가 미리 등록되어 있습니다.
- 기본으로 등록된 확장자는 직접 해제해 제외할 수 있습니다.
- 그 외 사내 정책에 따라 차단할 파일 확장자를 최대 200개까지 추가할 수 있습니다.

배치 파일과 실행파일, 라이브러리는 확장자와 콘텐츠를 함께 체크해, 확장자가 달라도 콘텐츠가 동일하다면 파일을 차단하여 더 강력한 보안을 제공합니다.

- DOS 배치파일 : .bat, .cmd
- (Windows 32/64bit) PE 실행파일 : .exe, .scr
- (Windows 32/64bit) PE 라이브러리 : .cpl, .dll, .ocx

그 외 확장자는 파일명에 입력된 확장자를 기준으로 차단됩니다.

- .com
- .js
- .vbs
- .wsf
- .hta
- 직접 입력한 확장자

# 외부 대화방 파일 전송 · 첨부 제한

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 신청 후 사용 가능합니다.

사내 구성원끼리의 대화방이 아닌, 외부의 대화상대와의 대화방이나 그룹에서는 파일을 전송·첨부하지 못하도록 제한할 수 있습니다.

|  |  |
| --- | --- |
| 상세 설정 | 옵션 |
| 외부 LINE 대화방 | 이 설정을 켜면, LINE 사용자가 참여한 메시지방에서 파일을 공유하지 못하게 됩니다.  - 메시지 입력창에서  아이콘을 눌렀을 때 카메라, 연락처, 드라이브(폴더)아이콘이 비활성화되어 카메라로 촬영해서 보내기, 연락처 보내기, [드라이브(폴더)에 있는 파일 보내기](/ko/use-guides/message/send-message/send-folder-drive/)를 할 수 없습니다. |
| 외부 네이버웍스 대화방 | 이 설정을 켜면, 다른 회사의 네이버웍스 사용자가 참여한 메시지방이나 외부 그룹에 파일 공유하지 못하게 됩니다.  - 메시지 입력창에서  아이콘을 눌렀을 때 카메라, 연락처, 드라이브(폴더) 아이콘이 비활성화되어 카메라로 촬영해서 보내기, 연락처 보내기, [드라이브(폴더)에 있는 파일 보내기](/ko/use-guides/message/send-message/send-folder-drive/)를 할 수 없습니다. - 외부 그룹 내 노트, 일정, 폴더 메뉴에서 파일을 첨부할 수 없습니다. |

이 설정은 '[서비스 > 메시지 > 외부 사용자와 대화](/ko/admin-guides/manage-service/message/message-with-external-users/)' 화면에서도 변경할 수 있습니다.

# 파일 로그

'[Admin > 감사 > 파일](/ko/admin-guides/audit/files/)'에서 구성원이 네이버웍스에서 파일을 다운로드 받거나 조회한 기록을 확인할 수 있도록 설정합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
