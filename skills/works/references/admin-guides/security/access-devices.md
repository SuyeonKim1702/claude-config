# 기기 현황

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품, 웍스 결재 상품 신청 후 사용 가능합니다.

최근 3년간 네이버웍스 모바일앱, PC앱, 드라이브앱, 드라이브 탐색기로 접속한 디바이스 정보를 확인하고, 다운로드할 수 있습니다.

각 디바이스에 설치된 앱이 최신 버전이 아닐 경우, 업데이트 알림을 보내서 최신 버전의 앱으로 업데이트하도록 독려할 수 있습니다.

# 디바이스의 목록 확인하기

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '기기 현황'을 눌러 '기기 현황' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 'NAVER WORKS 모바일 앱', 'PC' 또는 '드라이브 모바일앱'을 누릅니다.  
   'NAVER WORKS 모바일 앱' 또는 'PC'를 누릅니다.
   - NAVER WORKS 모바일 앱: 모바일앱에 접속한 기기의 목록이 표시됩니다.
   - PC: PC앱 또는 드라이브 탐색기에 접속한 기기의 목록이 표시됩니다.
   - 드라이브 모바일앱: 드라이브앱에 접속한 기기의 목록이 표시됩니다.

각 항목에 대한 자세한 설명입니다.

|  |  |
| --- | --- |
| 디바이스 | - 디바이스의 모델명이 표시됩니다. - 이 정보를 누르면 디바이스의 더 상세한 정보를 확인하고, 앱 데이터를 삭제할 수 있습니다. |
| Vendor ID | - 디바이스에서 생성하는 고유한 값으로 디바이스에서 노출되지는 않습니다. 디바이스를 초기화하면 변경되지만, 네이버웍스 앱을 삭제하거나 재설치해도 변경되지 않습니다. - 외부 MDM을 사용하는 디바이스에 대해 확인할 수 있습니다. |
| 사용자 | - 이 디바이스에서 모바일앱에 접속한 구성원의 이름과 ID가 표시됩니다. - 사용자의 정보를 누르면 이 구성원의 더 자세한 정보를 확인하고, 이 구성원을 관리할 수 있는 화면으로 이동할 수 있습니다. |
| OS | - 디바이스 운영체제의 종류와 버전이 표시됩니다. |
| 설치 앱 | - 이 기기에서 마지막으로 접속한 네이버웍스 앱과 버전이 표시됩니다. - 모바일앱이 최신 버전이 아닌 경우, 서비스 알림으로 업데이트 안내를 보낼 수 있습니다. |
| 최근 접속 | - 이 디바이스에서 해당 사용자가 최근 접속한 일자와 시각이 표시됩니다. - 이 정보는 모바일앱에 다시 로그인한 경우이거나, 마지막으로 접속한 지 24시간이 지난 경우에 다시 실행할 경우에 접속 시각이 업데이트됩니다. |

# 특정 조건의 디바이스를 검색하기

검색 조건에 해당하는 디바이스의 목록을 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '기기 현황'을 눌러 '기기 현황' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 디바이스 리스트 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-search-767676.png) 아이콘을 선택합니다.
3. 검색 조건에 필요한 값을 선택하거나 입력합니다.
   - 사용자: 특정 구성원이 접속한 디바이스를 검색할 수 있습니다.
   - 디바이스: 디바이스의 모델명으로 검색할 수 있습니다.
   - OS:  iOS 디바이스 또는 Android 디바이스를 모아볼 수 있습니다.
   - OS 버전: 디바이스 운영 체제의 버전으로 검색할 수 있습니다.
   - 설치 앱: 설치한 앱의 종류별로 검색할 수 있습니다.
   - 앱 버전: 특정 네이버웍스 앱 버전 이하의 버전이 설치된 디바이스를 검색할 수 있습니다.
   - 마지막 접속: 마지막 접속일이 최근 30일 이내, 6개월 이내, 1년 이내인 디바이스를 검색하거나, 모든 기간에 접속한 디바이스 가운데 검색할 수 있습니다.
4. '검색'을 누릅니다.

# 업데이트 알림 보내기

Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '기기 현황'을 눌러 '기기 현황' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.

## 한 명에게 업데이트 알림 보내기

특정 디바이스의 사용자에게 업데이트 알림을 보낼 수 있습니다. 새로운 버전의 네이버웍스가 출시되었을 때 업데이트 알림 보내기 기능을 활용하면 편리합니다.

1. 디바이스 목록에서 업데이트 알림을 보낼 기기를 선택합니다. 상세 검색을 이용해서 특정 버전 이하의 앱이 설치된 디바이스를 모아볼 수 있습니다.
2. '업데이트 필요'를 누릅니다.
3. '확인'을 선택하면 이 디바이스를 사용하는 구성원에게 알림을 보냅니다. 'NAVER WORKS 모바일앱' 탭에서 알림을 보내면 모바일앱 업데이트 알림이 발송되고,  'PC' 탭에서 알림을 보내면 PC앱 업데이트 알림이 발송됩니다.

## 여러 명에게 업데이트 알림 보내기

여러 디바이스의 사용자를 선택해서 업데이트 알림을 보낼 수 있습니다.

1. 디바이스 목록에서 업데이트 알림을 보낼 디바이스를 선택합니다. 최대 100개까지 선택할 수 있습니다.
2. 목록 상단의 '업데이트 안내' 버튼을 누릅니다.
3. '확인'을 선택하면 이 디바이스를 사용하는 구성원에게 알림을 보냅니다. 'NAVER WORKS 모바일앱' 탭에서 알림을 보내면 모바일앱 업데이트 알림이 발송되고,  'PC' 탭에서 알림을 보내면 PC앱 업데이트 알림이 발송됩니다.

# 디바이스 목록을 파일로 다운로드하기

네이버웍스에 접속한 기기 정보 검색 결과를 .csv 파일로 다운로드합니다. 검색 결과가 5,000건 이상인 경우에는 파일을 서비스 알림으로 전달합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '기기 현황'을 눌러 '기기 현황' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 필요한 조건으로 검색한 후 오른쪽의 '다운로드'를 눌러 검색 결과를 다운로드합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
