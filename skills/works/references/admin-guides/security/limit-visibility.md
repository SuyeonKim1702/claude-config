# 조직도 조회 제한

이 기능은 유료 플랜에서만 사용할 수 있습니다.

네이버웍스에서 모든 구성원은 조직도를 열고 다른 조직의 구성원을 확인하거나 특정 구성원의 프로필을 열람할 수 있지만,  조직도에서 다른 조직과 다른 조직의 구성원을 보지 못하도록 제한할 수 있습니다.

제한하면 조회 가능한 대상 외 구성원의 프로필 및 일정을 열람할 수 없으며, 네이버웍스의 주소록에서 검색하더라도 조회 가능한 대상만 볼 수 있습니다.

구성원, 사용자 유형, 조직 등의 조회자 단위로 제한을 추가할 수 있습니다.

|  |  |
| --- | --- |
| 제한 대상 | 내용 |
| 구성원 | 추가된 해당 구성원에게만 제한 설정이 적용됩니다. |
| 사용자 유형 | 이 사용자 유형에 해당하는 모든 구성원에게 제한 설정이 적용됩니다. |
| 조직 | 이 조직에 소속된 모든 구성원에게 제한 설정이 적용됩니다. |

구성원, 사용자 유형, 조직순으로 우선 순위가 설정됩니다. 즉, 구성원으로서 적용된 제한이 조직으로 적용된 제한보다 더 우선됩니다.

제한 대상을 추가하면서 해당 대상이 조회할 수 있는 범위를 다음과 같이 설정할 수 있습니다.

|  |  |
| --- | --- |
| 조회 가능 범위 | 내용 |
| 소속된 조직만 | 본인이 소속된 조직만 조직도에서 볼 수 있습니다. |
| 소속된 조직과 지정한 조직 | 본인이 소속된 조직과, 관리자가 지정한 조직(최대 200개)를 조직도에서 볼 수 있습니다.  이 옵션을 선택하면 소속된 조직 외에 볼 수 있도록 허용할 조직을 선택해야 합니다. |
| 조회 불가 | 본인만 조직도에서 볼 수 있습니다. |

# 제한 추가하기

구성원, 사용자 유형, 조직 단위로 조직도를 제한해서 조회하게 설정합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '조직도 조회 제한'을 누릅니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '추가'를 누릅니다.
3. 구성원, 조직, 사용자 유형 중 어떤 단위로 제한을 추가할지 선택합니다.
4. 추가할 구성원/조직/사용자 유형을 선택하고 '확인'을 누릅니다.
5. 조회 가능할 수 있는 조직도 범위를 선택합니다.
   - 조회 불가: 어떤 조직도도 조회할 수 없습니다.
   - 소속된 조직만: 제한할 구성원이 소속된 조직의 조직도만 조회할 수 있습니다.
   - 소속된 조직과 지정한 조직: 제한할 구성원이 소속된 조직과 관리자가 지정한 조직만 조회할 수 있습니다.
6. '소속된 조직과 지정한 조직'으로 설정하려면 '조직 선택' 버튼을 눌러 조회할 수 있도록 설정할 다른 조직을 선택한 뒤 '확인'을 누릅니다.
7. 모든 설정을 완료하고 '저장' 버튼을 눌러 저장합니다.

# 제한 수정하기

설정해 놓은 조직도 조회 제한 범위를 변경 할 수 있습니다.

### 모바일 앱

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '조직도 조회 제한을 탭합니다. ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 구성원, 조직, 사용자 유형 중 수정할 내용이 포함된 단위를 선택합니다.
3. 목록에서 수정이 필요한 항목을 선택합니다.
4. 목록 상단에서 '수정'을 탭합니다.
5. 조회 가능할 수 있는 조직도 범위를 선택합니다.
6. 하단의 '저장'을 탭해 변경한 내용을 저장합니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '보안'을 클릭하여 메뉴를 펼친 후 '조직도 조회 제한'을 클릭합니다.
2. 구성원, 조직, 사용자 유형 중 수정할 내용이 포함된 단위를 선택합니다.
3. 목록에서 수정이 필요한 항목 오른쪽에서 '수정'을 클릭합니다. 또는 목록에서 수정이 필요한 항목 여러개를 선택한 뒤, 목록 상단에서 '수정'을 클릭해 여러 항목을 동시에 수정할 수 있습니다.
4. 조회 가능할 수 있는 조직도 범위를 선택합니다.
5. 하단의 '저장'을 클릭해 변경한 내용을 저장합니다.

# 제한 삭제하기

설정해 놓은 조직도 조회 제한을 해제합니다.

제한을 삭제하더라도, 다른 단위(구성원/조직/사용자 유형)로 제한되어 있다면 우선순위에 따라 제한이 적용됩니다.

### 모바일 앱

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '조직도 조회 제한'을 탭합니다. ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 구성원, 조직, 사용자 유형 중 삭제할 내용이 포함된 단위를 선택합니다.
3. 목록에서 삭제가 필요한 항목을 선택합니다.
4. 목록 상단에서 '삭제'을 탭합니다.
5. 삭제 의사를 확인하는 화면이 나타나면 '확인'을 탭해 삭제합니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '보안'을 클릭하여 메뉴를 펼친 후 '조직도 조회 제한'을 클릭합니다.
2. 구성원, 조직, 사용자 유형 중 삭제할 내용이 포함된 단위를 선택합니다.
3. 목록에서 삭제할 항목 오른쪽에서 '삭제'를 클릭합니다. 또는 목록에서 좌측의 체크박스에 체크하여 삭제할 항목 여러 개를 선택할 수도 있습니다.
4. 목록 상단에서 '삭제'를 클릭합니다.
5. 삭제 의사를 확인하는 화면이 나타나면 '확인'을 클릭하여 삭제합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
