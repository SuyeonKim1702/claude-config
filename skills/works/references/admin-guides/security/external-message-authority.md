# 외부 사용자와 대화 권한

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

서비스 메뉴에서 제공하는 [외부 사용자와 대화](/ko/admin-guides/manage-service/message/message-with-external-users/) 기능 중 연동 사용자 관리를 보안 메뉴에서도 동일하게 설정할 수 있습니다.

# LINE 연동 권한

'LINE 사용자와 대화' 기능을 사용으로 설정했을 때, 연동하여 사용할 수 있는 구성원의 현황을 확인하고 연동 사용자를 추가하거나 삭제합니다.

## LINE 연동 사용자 추가

외부 LINE 사용자와 메시지를 주고 받을 수 있는 구성원을 추가합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '외부 사용자와 대화 권한'을 눌러 '외부 사용자와 대화 권한' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 'LINE'을 선택합니다.
3. '추가'를 선택하여 연동할 구성원을 선택한 후 하단의 '확인'을 눌러 연동할 구성원을 추가합니다.

## LINE 연동 사용자 삭제

외부 LINE 사용자와 메시지를 주고 받을 수 있도록 연동된 구성원을 연동 사용자에서 삭제합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '외부 사용자와 대화 권한'을 눌러 '외부 사용자와 대화 권한' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 'LINE'을 선택합니다.
3. 삭제할 구성원을 선택한 후 '삭제'를 눌러 연동 사용자에서 삭제합니다.

# 다른 회사의 네이버웍스 연동 권한

'다른 회사의 네이버 웍스 사용자와 대화' 기능을 사용으로 설정했을 때, 사용할 수 있는 구성원의 현황을 확인하고 연동 사용자를 추가하거나 삭제합니다.

## 다른 회사의 네이버웍스 연동 사용자 추가

네이버웍스를 사용하는 다른 회사의 구성원과 메시지를 주고 받을 수 있는 구성원을 추가합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후
2. '외부 사용자와 대화 권한'을 눌러 '외부 사용자와 대화 권한' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
3. '다른 회사의 네이버 웍스'을 선택합니다.
4. '추가'를 선택하여 연동할 구성원을 선택한 후 하단의 '확인'을 눌러 연동할 구성원을 추가합니다.

## 다른 회사의 네이버웍스 사용자 삭제

네이버웍스를 사용하는 다른 회사의 구성원과 메시지를 주고 받을 수 있도록 연동된 구성원을 연동 사용자에서 삭제합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '외부 사용자와 대화 권한'을 눌러 '외부 사용자와 대화 권한' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '다른 회사의 네이버 웍스'을 선택합니다.
3. 삭제할 구성원을 선택한 후 '삭제'를 눌러 연동 사용자에서 삭제합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
