# 메일 보안

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 신청 후 사용 가능합니다.

네이버웍스의 메일을 안전하게 사용하기 위해서, 회사 차원에서 메일 서비스에 공통적으로 적용할 메일 보안 정책을 설정할 수 있습니다.

# 외부 메일 가져오기

각 구성원들이 '[외부 메일 가져오기](/ko/use-guides/mail/settings/import-external-mail/)' 기능을 사용하도록 허용할지 설정할 수 있습니다.

단, '[보안 > 네트워크 보안](/ko/admin-guides/security/network-security/)'에서 '지정한 IP에서만 접속을 허용'으로 설정했다면 외부 메일을 가져올 수 없습니다.

기존에 사용하던 다른 메일 서비스가 있다면, 이 기능을 통해 기존 메일 주소로 수신되는 메일과 네이버웍스 메일 주소로 수신되는 메일을 모두 네이버웍스 메일 서비스에서 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '메일 보안'을 눌러 '메일 보안' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 메일 가져오기' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 수정할 수 있는 상태로 전환합니다.
3. '외부 메일 가져오기' 항목을 설정합니다.
   - 기능 ON: 모든구성원들이 '외부 메일 가져오기' 기능을 사용할 수 있습니다.
   - 기능 OFF: 모든 구성원들이 '외부 메일 가져오기' 기능을 사용하지 못하게 됩니다.  
     이미 이 기능이 설정된 경우, 설정된 정보가 모두 삭제됩니다.
4. 상단의 '저장'을 누릅니다.

# **메일 백업/PC에 저장하기**

구성원들이 메일함 백업 및 메일 원문 PC에 저장하기 기능을 사용하도록 허용할지 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '메일 보안'을 눌러 '메일 보안' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '메일 백업/PC에 저장하기' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 수정할 수 있는 상태로 전환합니다.
3. '메일 백업/PC에 저장하기' 항목을 설정합니다.
   - 기능 ON: 모든 구성원들이 '메일 백업/PC에 저장하기' 기능을 사용할 수 있습니다.
   - 기능 OFF: 모든 구성원들이 '메일 백업/PC에 저장하기' 기능을 사용하지 못합니다.
4. 상단의 '저장'을 누릅니다.

# **다운로드 기한이 있는 파일 첨부**

메일에서 '첨부 가능 용량' 보다 큰 파일을 보낼 수 있도록 허용할지 설정할 수 있습니다.

해당 설정을 허용하는 경우, 첨부 가능한 용량보다 큰 파일은 7일간 100회까지만 다운로드 할 수 있는 파일로 첨부됩니다.

메일의 '첨부 가능 용량'은 '[Admin > 서비스 > 메일 > 일반](/ko/admin-guides/manage-service/mail/mail-settings/)' 화면에서 설정할 수 있습니다.

'다운로드 기한이 있는 파일'을 보낼 수 있도록 허용하는 경우, 첨부 할 수 있는 최대 개수 및 용량은 '[Admin > 서비스 > 메일 > 일반](/ko/admin-guides/manage-service/mail/mail-settings/)' 화면에서 설정할 수 있습니다.

2. '다운로드 기한이 있는 파일' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 수정할 수 있는 상태로 전환합니다.

   - 기능 OFF: '첨부 가능 용량'보다 큰 파일을 첨부할 수 없습니다.

# 드라이브 보안 링크

이 기능은 네이버웍스 코어 상품의 Standard Plus 플랜 또는 웍스 드라이브 상품의 Standard Plus 플랜 신청 후 사용 가능합니다.

메일에서 첨부파일을 '드라이브 보안 링크'로 전환하여 보낼 수 있도록 허용할지 설정할 수 있습니다.

설정을 허용하는 경우, 아래 설정도 모두 함께 허용으로 변경됩니다.

메일을 받더라도 [OTP(일회용 비밀번호) 인증](/ko/use-guides/drive/send-file-or-link/send-as-link/otp-link-authenticate/)을 해야 첨부파일을 확인할 수 있어 보안을 강화할 수 있습니다.

'첨부 가능 링크 개수'와 '외부로 발신 시 보안 링크 필수 여부'를 설정할 수도 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '메일 보안'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '드라이브 보안 링크' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 수정할 수 있는 상태로 전환합니다.
3. '드라이브 보안 링크'을 허용할지 여부를 선택합니다.
4. '첨부 가능 링크 개수'를 입력합니다.
5. '외부로 발신 시 보안 링크 필수 여부'를 어떻게 적용할지 선택합니다.
   - 필수 - 보안 링크로만 첨부 가능: 외부 주소로 메일을 보낼 때는 드라이브 보안 링크로만 파일을 첨부할 수 있습니다.
   - 선택 - 구성원이 선택 가능: 외부 주소로 메일을 보낼 때, 구성원이 원하면 드라이브 보안 링크로 파일을 첨부할 수 있습니다.
6. 상단의 '저장'을 누릅니다.

# 외부 수신 시 이미지/링크 차단

사내가 아닌 외부에서 보낸 메일을 받았을 때, 메일 본문의 이미지나 링크를 차단할지 설정합니다.

모든 구성원에게 차단을 적용할 수도 있고, 구성원이 각자 적용하도록 선택할 수도 있습니다.

이미지/링크를 차단한 경우에도 구성원이 직접 '이미지/링크 보기'를 누르면 첨부된 이미지와 링크를 볼 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '메일 보안'을 눌러 '메일 보안' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부 수신 시 이미지/링크 차단' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
3. '외부 수신 시 이미지/링크 차단'을 어떻게 적용할지 선택합니다.
   - 필수 - 모든 구성원의 메일에서 이미지/링크를 차단: 모든 구성원의 계정에서 외부 수신 시 이미지/링크가 기본적으로 차단되도록 설정합니다.
   - 선택 - 각 구성원 환경설정을 따름: 각 구성원이 자율적으로 외부 수신 시 이미지/링크를 차단할지 설정할 수 있습니다.
4. 상단의 '저장'을 누릅니다.

# 외부로 발신 시 주의 배너 표시

구성원이 사내가 아닌 외부 메일 주소로 메일을 발송할 때, 메일 발송 화면에 보안 경고 배너를 노출하도록 설정합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '메일 보안'을 눌러 '메일 보안' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부로 발신 시 주의 배너 표시' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 누릅니다.
3. '외부로 발신 시 주의 배너 표시' 적용 여부를 선택합니다.
4. 상단의 '저장' 버튼을 누릅니다.

# 외부로 발신 시 주의 팝업

구성원이 사내가 아닌 외부 메일 주소로 메일을 발송할 때, 발송 직전에 보안 주의 팝업을 노출하도록 설정합니다.

외부 메일 주소로 발송하는 모든 메일에 대해 보안 주의 팝업을 노출하거나, 특정 조건에만 노출하도록 설정할 수 있습니다.

단, 보내는 사람이 외부 메일 주소가 포함된 메일링리스트(DL)일 때는 보안 주의 팝업이 노출되지 않습니다.

## 주의 팝업 노출 설정

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '메일 보안'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부로 발신 시 주의 팝업 노출' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 수정할 수 있는 상태로 전환합니다.
3. '외부로 발신 시 주의 팝업 노출' 설정을 켭니다.
4. 팝업을 항상 노출할지, 조건에 따라 노출할지 선택합니다.
   - 항상 적용: 외부 메일 주소로 발송하는 모든 메일에 대해 주의 팝업을 노출합니다.
   - 조건 적용: 외부 메일 주소로 발송하는 메일 중, 아래 조건에 해당하는 경우에만 주의 팝업을 노출합니다.
     - 첨부파일이 있는 경우만: 첨부파일이 있는 경우에만 주의 팝업을 노출합니다.
     - '개인별' 발송에 체크하지 않은 경우만: '개인별' 발송 설정을 선택하지 않은 경우에만 주의 팝업을 노출합니다.
     - 첨부파일이 있고, ‘개인별’ 발송에 체크하지 않은 경우만: 첨부파일이 있고, '개인별' 발송 설정을 선택하지 않은 경우에만 주의 팝업을 노출합니다.
5. 상단의 '저장'을 누릅니다.

## 팝업 경고 문구 설정

외부 메일 주소로 메일을 발송하기 전 노출될 주의 팝업의 문구를 설정합니다.

문구를 설정하지 않을 경우, 자동으로 영어로 작성된 기본 문구가 노출되니 도메인의 정책에 맞는 문구를 반드시 입력해주세요.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '메일 보안'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '외부로 발신 시 주의 팝업 노출' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 수정할 수 있는 상태로 전환합니다.
3. '외부로 발신 시 주의 팝업 노출' 설정을 켭니다.
4. 'PC 경고 문구 설정' 또는 '모바일 경고 문구 설정'을 누른 후 문구를 입력합니다.  
   이때, '기본 문구' 탭의 문구는 아래 경우에 노출됩니다.
   - 구성원이 '환경설정'의 ['](/ko/use-guides/message/settings/general/general-setting/#i-3)개인정보 > 언어 설정'에서 설정한 다국어가 'Admin > 회사 정보 > 다국어 설정'에 추가되지 않은 다국어인 경우
   - '경고 문구 설정'의 다국어 탭의 내용이 따로 쓰여있지 않은 경우
5. '저장'을 눌러 문구를 저장하고 'X' 아이콘을 눌러 창을 닫습니다.
6. 상단의 '저장'을 누릅니다.

# 보안 등급 설정

네이버웍스 메일의 보안 등급은 '일반', '대외비', '기밀', '업무 외' 등급으로 나누어지며, 관리자는 보안 등급별로 사용 여부를 선택할 수 있습니다.

구성원은 현재 사용 중인 보안 등급 중 하나를 지정해 메일을 발송할 수 있습니다.

- 일반: 메일의 유효기간이 설정되지 않으며 항상 재전송할 수 있습니다. 보안 등급을 지정하지 않은 메일은 '일반' 등급으로 적용됩니다.
- 대외비, 기밀: 보안상 중요한 메일을 보낼 때 사용할 수 있는 보안 등급입니다. 회사 정책에 맞춰 등급명과 등급별 설정을 변경할 수 있습니다.
  - 등급명: '대외비'와 '기밀' 등급 옆 '수정'을 클릭해 보안 등급명을 변경할 수 있습니다.
    - 수정 후에는 Admin과 서비스에서 변경된 보안 등급명으로 표시됩니다.
    - 감사 로그에는 변경된 보안 등급명을 구분할 수 있도록, 변경한 등급명 앞에 Restricted(대외비) 또는 Confidential(기밀)을 함께 기재하고 있습니다.
  - 등급별 설정: '유효기간', '재전송 금지', '유효기간 경과시 보낸 메일도 삭제'에 대한 설정을 할 수 있습니다.
- 업무 외: 업무 이외의 목적으로 메일을 보낼 때 사용할 수 있는 보안 등급입니다.
  - '업무 외' 등급으로 지정한 메일은 발신 모니터링 정책 대상에서 제외되어, 해당 메일은 모니터링되지 않습니다. 단, 감사, 아카이빙 범위에는 포함됩니다.
  - '업무 외' 등급만 단독으로 사용할 수 없으며, 반드시 하나 이상의 다른 등급과 함께 사용해야 합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '메일 보안'을 눌러 '메일 보안' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '보안 등급'을 선택합니다.
3. '보안 등급' 항목 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 수정할 수 있는 상태로 전환합니다.
4. 각 보안 등급의 사용 여부를 선택합니다.
   - 기능 ON: 구성원이 메일을 보낼 때 해당 보안 등급을 지정할 수 있습니다.
   - 기능 OFF: 구성원이 메일을 보낼 때 해당 보안 등급을 지정할 수 없으며, 모든 보안 등급이 OFF일 경우 '일반' 등급의 메일로 취급됩니다.
5. '대외비'와 '기밀' 등급을 켰다면 '대외비' 메일과 '기밀' 메일의 세부 설정을 변경할 수 있습니다.
   - 유효기간: 메일을 확인할 수 있는 기한입니다. 유효기간이 지난 메일은 삭제됩니다. 하루, 일주일, 1개월, 3개월 중 선택할 수 있습니다.
   - 재전송 금지: 메일을 받은 사람이 받은 메일을 다른 사람에게 전송할 수 없도록 설정합니다.
   - 유효기간 경과시 보낸 메일도 삭제: 유효기간 경과 시 메일을 보낸 사람의 '보낸메일함'에서도 해당 메일이 삭제됩니다.
6. 상단의 '저장'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
