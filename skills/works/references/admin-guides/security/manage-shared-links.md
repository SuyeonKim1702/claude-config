# 드라이브(폴더) 링크 관리

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

회사 구성원이 공유하고 있는 링크의 목록을 조회하고, 상세 정보를 확인하고, 삭제합니다.

# 공유 중인 링크 확인

구성원이 공유하고 있는 링크와 관련 정보를 확인하고, 상세한 링크 정보를 확인하거나, 삭제합니다.

Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '드라이브 링크 관리'를 눌러 '드라이브 링크 관리' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.

기본 조건은 오늘 생성된 공유 링크 전체입니다.

- 파일/폴더명: 링크로 공유된 파일 또는 폴더의 이름입니다.
- 생성자: 공유 링크를 만든 구성원의 이름입니다.
- 생성일: 공유 링크를 생성한 날짜입니다.
- 접근 권한: 공유 링크에 접근할 수 있는 권한으로 누구나 접근, 회사ID로 접근, 비밀번호로 접근, OTP로 접근 가운데 구성원이 공유 링크 생성할 때 선택한 값이 표시됩니다.
- 정보: '상세보기'를 클릭하면 '링크 상세 보기' 창이 뜹니다. 공유 링크에 접근 정보를 포함하여 공유 링크의 상세한 정보를 확인할 수 있습니다.
- 삭제: 관리자가 구성원이 생성한 공유 링크를 삭제할 수 있습니다.

# 공유 중인 링크 삭제

공유 중인 링크를 삭제합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '드라이브 링크 관리'를 눌러 '드라이브 링크 관리' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 삭제할 링크 오른쪽의 '삭제'를 선택합니다.
3. 주의 사항을 확인 후 '확인'을 선택합니다.

# 검색 조건 설정

조건에 맞는 공유 링크를 검색 조건을 설정해 빠르게 확인합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '드라이브 링크 관리'를 눌러 '드라이브 링크 관리' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '검색'을 눌러 조건에 맞는 공유 링크를 검색합니다.
   - 오른쪽 상단의 '상세'를 누르면 조건에 필요한 값을 선택하거나 입력할 수 있습니다.
     - 기간: 검색하려는 공유 링크가 생성된 기간을 선택합니다.
     - 파일/폴더명: 검색할 파일, 폴더명을 입력합니다.
     - 링크 생성자: 생성자의 이름을 입력합니다.
     - 접근 권한: 누구나 접근, 회사ID로 접근, 비밀번호로 접근, OTP로 접근 가운데 선택할 수 있습니다.

# 파일로 다운로드

공유 링크의 목록과 정보를 .xls 파일 형식으로 다운로드 받습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '드라이브 링크 관리'를 눌러 '드라이브 링크 관리' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 파일로 다운받을 링크의 '상세보기'를 선택합니다.
3. '링크 상세 보기' 페이지에서 '다운로드'를 선택합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
