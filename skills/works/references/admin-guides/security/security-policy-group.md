# 보안 정책 그룹

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

보안 정책 그룹은 여러 구성원을 하나의 모니터링 그룹으로 묶어 한 번에 관리할 수 있는 기능입니다.

메일 수신, 메일 발신, 드라이브 모니터링에서 보안 정책 그룹을 정책 조건으로 지정할 수 있으며, 이 경우 보안 정책 그룹 내 구성원 추가 또는 삭제 시 각각의 모니터링 정책에서 일일이 추가하거나 삭제할 필요가 없습니다.

보안 정책 그룹은 50개까지 생성할 수 있으며, 보안 정책 그룹별로 200명까지 구성원을 추가할 수 있습니다.

# 보안 정책 그룹 생성, 수정, 삭제

## 보안 정책 그룹 생성

보안 정책 그룹은 50개까지 생성할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '보안 정책 그룹'을 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 왼쪽 상단의 '그룹 추가'를 누르고 그룹명을 입력합니다.
3. '확인'을 누릅니다.

## 보안 정책 그룹 이름 수정

생성한 보안 정책 그룹의 이름을 수정할 수 있습니다.

### 모바일 앱

1. Admin에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다. '보안'을 선택해 메뉴를 펼친 후 '보안 정책 그룹'을 선택합니다.
2. 이름을 수정할 보안 정책 그룹을 탭합니다.
3. 보안 정책 그룹 이름 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-more-767676.png) 아이콘을 탭하고 '그룹명 수정'을 탭합니다.
4. 이름 변경 후 '확인'을 탭합니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '보안 정책 그룹'을 선택합니다.
2. 이름을 수정할 보안 정책 그룹 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-more-767676.png) 아이콘을 클릭하고 '그룹명 수정'을 선택합니다.
3. 이름 변경 후 '확인'을 클릭합니다.

## 보안 정책 그룹 삭제

생성한 보안 정책 그룹을 목록에서 삭제할 수 있습니다. 모니터링 조건으로 설정되어있는 보안 정책 그룹을 삭제하면 해당 모니터링 정책에서도 삭제되어 모니터링이 중단됩니다.

### 모바일 앱

1. Admin에서 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다. '보안'을 선택해 메뉴를 펼친 후 '보안 정책 그룹'을 선택합니다.
2. 삭제할 보안 정책 그룹을 탭합니다.
3. 보안 정책 그룹 이름 오른쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-more-767676.png) 아이콘을 탭하고 '그룹 삭제'를 탭합니다.
4. '확인'을 탭합니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '보안 정책 그룹'을 선택합니다.
2. 삭제할 보안 정책 그룹 위에 마우스 포인터를 올린 후 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-16-more-767676.png) 아이콘을 클릭하고 '그룹 삭제'를 선택합니다.
3. '확인'을 클릭합니다.

# 보안 정책 그룹 내 구성원 추가, 삭제

보안 정책 그룹에 구성원을 추가/삭제하면 관련 모니터링 정책의 조건에도 자동으로 반영됩니다.

## 보안 정책 그룹 내 구성원 추가

한 그룹 내에 구성원은 200명까지 추가할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '보안 정책 그룹'을 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 구성원을 추가할 보안 정책 그룹을 선택합니다.
3. 오른쪽 상단의 '구성원 추가'를 누릅니다.
4. 추가할 구성원을 선택하고 '확인'을 누릅니다.

## 보안 정책 그룹 내 구성원 삭제

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '보안 정책 그룹'을 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 구성원을 삭제할 보안 정책 그룹을 선택합니다.
3. 삭제할 구성원 오른쪽의 '삭제'를 누릅니다. 여러 구성원을 한 번에 삭제하고 싶다면 삭제할 구성원을 선택하고 '삭제'를 누릅니다.
4. '확인'을 누릅니다.

# 구성원 검색

구성원을 검색해 특정 구성원이 속해 있는 모든 보안 정책 그룹을 조회할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '보안 정책 그룹'을 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-act_com_search-1.png) 아이콘을 누릅니다.
3. 구성원의 이름 또는 계정을 검색합니다.
