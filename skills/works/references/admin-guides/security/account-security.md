# 계정 보안

네이버웍스에 로그인하기 위한 비밀번호 정책과 비밀번호를 여러 번 잘못 입력할 경우에 대한 정책을 설정할 수 있습니다.

ID/비밀번호를 입력하지 않고 로그인할 수 있는 다른 로그인 방식을 허용할지 설정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '계정 보안'을 눌러 '계정 보안' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 각 항목별로 오른쪽의 ![](/wp-content/uploads/2024/04/icon-n-path.png) 아이콘을 눌러 설정을 변경합니다.
3. 상단의 '저장'을 눌러 변경한 내용을 저장합니다.

# 비밀번호 정책

이 기능은 유료 플랜에서만 사용할 수 있습니다.

비밀번호를 입력해서 로그인하는 구성원의 계정을 안전하게 보호할 수 있도록 비밀번호를 등록/만료 및 비밀번호를 잘못 입력할 경우에 대한 정책을 설정할 수 있습니다.

이 설정을 변경해도 구성원은 서비스에서 로그아웃되지 않습니다.

설정 변경한 이후에 구성원들이 새 비밀번호를 등록할 때 새 정책에 따라 비밀번호를 만들게 됩니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '계정 보안'을 눌러 '계정 보안' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 각 항목별로 오른쪽의 버튼을 누르면 설정을 변경할 수 있습니다.
3. 상단의 '저장'을 눌러 변경한 내용을 저장합니다.

각 항목에 대한 자세한 설명입니다.

|  |  |
| --- | --- |
| 상세 설정 | 옵션 |
| 비밀번호 형식 | 구성원이 다음번에 새 비밀번호를 등록할 때 관리자가 지정한 형식으로 비밀번호를 생성해야 합니다.  - 영문, 숫자 필수로 포함: 반드시 영문자와 숫자를 포함해야 합니다. - 영문, 숫자, 특수문자 필수로 포함: 반드시 영문자와 숫자, 그리고 특수문자를 포함해야 합니다.   영문은 대소문자를 구별합니다. |
| 비밀번호 길이 | 구성원이 사용할 비밀번호의 최소 길이를 설정합니다.  - 설정할 수 있는 비밀번호의 길이는 영문기준 8~20자입니다. |
| 비밀번호 만료 | 구성원이 비밀번호를 설정한 후 어느 정도의 기간이 지난 뒤 새로운 비밀번호를 등록하게 할지 설정합니다.  - 만료 안 함: 구성원이 직접 비밀번호를 변경하지 않는 이상, 기존 비밀번호를 계속 사용할 수 있습니다. - 30일/60일/ 90일/ 180일/ 365일 이후 비밀번호 만료: 비밀번호가 만료된 구성원은 다음 로그인 시 새로운 비밀번호를 등록해야 합니다. |
| 비밀번호 재사용 제한 | 최근 사용한 비밀번호 재사용 제한  - 제한 안 함: 최근 사용한 비밀번호를 다시 사용할 수 있습니다. - 최근 1개/2개/3개/4개/5개 비밀번호 사용 불가 |
| 로그인 시도 횟수 제한 | 연속으로 여러 번 비밀번호를 잘못 입력하면 계정이 일시정지 되도록 설정  - 계정이 일시정지된 구성원은 모든 네이버웍스 서비스에서 로그아웃됩니다. - 일시정지된 구성원은 직접 비밀번호를 재설정하거나 관리자가 일시정지 상태를 해제해 주어야 다시 네이버웍스에 로그인할 수 있습니다.   - 구성원: 로그인 화면의 '비밀번호 찾기'에서 '휴대폰 번호로 찾기' 또는 '개인 이메일로 찾기'를 눌러 비밀번호를 재설정합니다.   - 관리자: Admin의 '구성원' 메뉴에서 해당 구성원을 검색한 후 일시정지 항목에서 '해제'를 누릅니다. |
| 비밀번호 일괄 재설정 | 전체 구성원의 비밀번호를 한번에 재설정할 수 있습니다. 재설정 대상은 비밀번호를 등록한 '사용중' 상태의 구성원입니다. 단, 최고관리자 및 간편 로그인 수단으로 가입한 후 ID/비밀번호를 등록하지 않은 구성원은 제외됩니다.  재설정 방법은 두 가지입니다.   - 강제 변경: 모든 구성원의 비밀번호를 동일한 새 비밀번호로 재설정합니다. 모든 구성원이 강제로 로그아웃되어 새 비밀번호로 로그인해야 합니다. - 변경 요청: 모든 구성원에게 비밀번호 변경 요청을 보냅니다.   구성원이 비밀번호를 반드시 변경해야 한다면 보안을 위해 비밀번호를 먼저 강제 변경한 후 다시 변경 요청을 보내주세요. |

# 로그인 정책

이 기능은 유료 플랜에서만 사용할 수 있습니다.

보안과 관련해 회사 차원의 로그인 정책을 설정합니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '계정 보안'을 눌러 '계정 보안' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 각 항목별로 오른쪽의 버튼을 누르면 설정을 변경할 수 있습니다.
3. 상단의 '저장'을 눌러 변경한 내용을 저장합니다.

## 2단계 인증

2단계 인증이란 계정을 안전하게 보호하기 위해 구성원이 로그인 시점에 거치는 추가적인 인증 단계입니다.

관리자는 모든 구성원이 2단계 인증 활성화하도록 '필수 - 모든 구성원에게 2단계 인증 적용'으로 설정하거나, 구성원이 선택적으로 2단계 인증을 활성화하도록 설정할 수 있습니다.

- 2단계 인증이 활성화된 경우, 구성원은 개인정보에 등록된 개인 이메일 또는 휴대폰 번호(문자 메시지)로 일회용 인증번호를 받을 수 있습니다. 단, 휴대폰 번호로 로그인한 구성원은 개인 이메일로만 2단계 인증을 위한 인증 번호를 받을 수 있습니다.
- 각 구성원은 2단계 인증을 활성화하는 시점에 본인의 개인정보에 등록된 개인 이메일 및 휴대폰 번호를 확인하게 됩니다. 만약 2단계 인증을 활성화한 이후에 개인정보의 이메일 또는 휴대폰 번호가 변경되면 변경된 개인 이메일이나 휴대폰 번호로 인증번호를 받게 됩니다.
- 근태, 결재 등 경영지원 상품을 이용 중이라면, ‘경영지원 설정 2단계 인증 관리 바로가기' 버튼을 선택해 서비스별로 2단계 인증을 하도록 설정할 수 있습니다.

|  |  |
| --- | --- |
| 기능 | 옵션 |
| 2단계 인증 (2-step verification) | - 필수: 모든 구성원이 반드시 2단계 인증을 활성화해야 합니다. 설정을 변경한 뒤, 구성원이 다음 로그인하는 시점에 2단계 인증 수단을 등록하게 되며, 그다음부터는 2단계 인증을 거쳐야 로그인할 수 있습니다. - 선택: 구성원이 선택적으로 2단계 인증을 사용하거나, 사용하지 않을 수 있습니다. [구성원이 직접 2단계 인증을 활성화](/ko/use-guides/settings/security/login-security-settings/#i-6)한 경우에만 로그인 시점에 2단계 인증을 진행합니다. |
| 신뢰하는 기기에서 2단계 인증 건너뛰기 | - 이 설정은 아래의 '휴대폰 잠금해제로 로그인 (FIDO)' 기능에 따라, 구성원이 한번 로그인한 다음 신뢰하는 기기로 추가한 모바일 기기에서만 적용됩니다. - 신뢰하는 모바일 기기에서는 다음 번에 생체인증으로 네이버웍스 앱에 로그인할 때 2단계 인증을 거치지 않게 됩니다. |

이 경우, 2단계 인증을 필수로 설정하면 개인 이메일과 휴대폰 번호가 등록되지 않은 구성원은 로그인할 수 없게 되므로 사전에 모든 구성원의 휴대폰 번호 또는 개인 이메일이 등록되어 있는지 확인해야 합니다.

## PC 앱/드라이브 탐색기 로그인 유지

구성원이 PC앱 또는 드라이브 탐색기에 로그인할 때 로그인을 유지할지 설정할 수 있습니다.

관리자가 이 기능을 켜고 구성원이 로그인 유지 기능을 사용하면 PC앱 또는 드라이브 탐색기를 재시작해도 별도의 로그인 없이 PC앱 또는 드라이브 탐색기를 이용할 수 있습니다.

|  |  |
| --- | --- |
| 기능 | 옵션 |
| PC 앱/드라이브 탐색기 로그인 유지 | - 허용: 모든 구성원이 PC앱 또는 드라이브 탐색기에서 로그인 유지 기능을 사용할 수 있습니다. - 허용 안 함: 구성원이 더 이상 PC앱 또는 드라이브 탐색기에서 로그인 유지 기능을 사용할 수 없습니다. 모든 구성원에게 설정 변경에 대한 서비스 알림 메시지가 전송됩니다. 이미 로그인 유지 기능을 사용하던 구성원을 포함해 도메인 내 모든 구성원은 다음 로그인부터 직접 로그인해야 합니다. 환경설정에서 해당 설정이 노출되지 않으며, PC앱 로그인 화면에서 '로그인 유지'를 설정하더라도 로그인이 유지되지 않습니다. |

## 유휴 세션 제한 시간

아무런 동작을 하지 않는 상태로 특정 시간을 보냈을 때 자동으로 로그아웃되도록 설정합니다.

재로그인하면 다시 정상적으로 서비스를 사용할 수 있습니다.

단, PC웹/모바일웹에서 '제한 안 함'으로 설정한 경우 실제 유휴 세션 제한 시간은 24시간으로 설정되며, PC웹에서 로그인할 때 '로그인 유지'를 선택한 경우 최대 30일까지 연장됩니다.

| 모바일앱 | - 30일/180일/1년 |

# 로그인 수단

구성원이 ID와 비밀번호를 입력하는 대신 간편 로그인 방식으로 로그인할 수 있도록 설정할 수 있습니다.

## 휴대폰 번호로 로그인

구성원들이 네이버웍스에 ID와 비밀번호를 입력하는 대신, 휴대폰 번호로 일회용 인증번호를 받아 로그인할 수 있습니다.

먼저 구성원이 직접 처음 1회 휴대폰 번호를 인증해서 휴대폰 번호와 네이버웍스 계정을 연결한 다음부터 이 방법으로 로그인할 수 있습니다.

- 만약 등록한 휴대폰 번호로 문자 메시지를 받을 수 없는 상황이라면 서비스 알림 메시지로 인증번호를 받을 수 있습니다.
- 구성원은 개인정보에 등록된 휴대폰 번호를 연결하거나, 새로운 번호로 연결할 수 있습니다.
- 개인정보에 등록된 번호와 다른 번호로 연결했다면, 이 번호는 네이버웍스 서비스에서 로그인 시 이외에는 노출되거나 사용되지 않습니다. (ID/비밀번호 찾기에는 개인정보에 등록된 휴대폰 번호가 사용됩니다.)

|  |  |
| --- | --- |
| 기능 | 옵션 |
| 휴대폰 번호로 로그인 | - 허용: 이 기능을 켜면 구성원이 휴대폰 번호로 인증 번호를 받아 로그인을 할 수 있도록 허용합니다. 모든 구성원이 이 기능을 사용할 수 있게 되었다는 알림을 받게 됩니다. - 허용 안 함: 이 기능을 끄면 구성원이 더이상 휴대폰 번호로 인증 번호를 받아 로그인할 수 없도록 제한합니다. 이미 이 기능을 설정하여 사용하고 있는 구성원은 다음 로그인부터 '휴대폰 번호로 로그인' 기능을 사용할 수 없게 됩니다. |

## 네이버로 로그인

구성원들이 네이버웍스에 ID와 비밀번호를 입력하는 대신, 개인의 네이버 아이디로 로그인할 수 있습니다.

먼저 구성원이 직접 [네이버웍스 계정과 네이버 아이디를 연결](/ko/use-guides/settings/security/login-security-settings/#i-3)한 다음부터 이 방법으로 로그인할 수 있습니다.

- 연동한 네이버 아이디는 로그인에만 사용되고, 네이버웍스 서비스에서 노출되지 않습니다.
- 관리자는 보안 관리를 위해 [특정 구성원의 '네이버로 로그인' 설정을 해제](/ko/admin-guides/manage-members/members/members-information/#i-2)할 수도 있습니다.

|  |  |
| --- | --- |
| 기능 | 옵션 |

## 모바일앱 간편 로그인

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 결재 상품 신청 후 사용 가능합니다.

구성원들이 네이버웍스에 ID와 비밀번호를 입력하는 대신, 모바일앱에서 등록한 PIN 또는 생체인증 수단으로 로그인할 수 있습니다.

먼저 구성원이 직접 간편 로그인을 이용하도록 설정하고, PIN을 등록한 다음부터 이 방법으로 로그인할 수 있습니다.

단, 사내 이용 환경에 따라 이 설정이 노출되지 않을 수 있습니다.

|  |  |
| --- | --- |
| 기능 | 옵션 |
| 모바일앱 간편 로그인 | - 허용: 이 기능을 켜면 모든 구성원이 모바일앱 간편 로그인 기능을 활성화할 수 있도록 허용합니다. 모든 구성원이 이 기능을 사용할 수 있게 되었다는 알림을 받게 됩니다. - 허용 안 함: 이 기능을 끄면 구성원이 더 이상 모바일앱 간편 로그인 기능을 켤 수 없도록 제한합니다. 이미 이 기능을 설정하여 사용하고 있는 구성원은 다음 로그인부터 다른 방법으로 로그인해야 합니다. |

## 휴대폰 잠금해제로 로그인(FIDO)

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 결재 상품 신청 후 사용 가능합니다.

구성원들이 기기에 설정한 생체인증 등의 휴대폰 잠금해제 방식으로 로그인할 수 있습니다.

먼저 구성원이 네이버웍스 앱이 설치된 자신의 기기를 신뢰하는 기기로 등록하고, '[휴대폰 잠금해제로 로그인](/ko/use-guides/settings/security/login-security-settings/#i-5)' 설정을 켜면 이 방법으로 로그인할 수 있습니다.

- 구성원이 모바일앱에 휴대폰 잠금해제로 로그인하는 경우 [2단계 인증](/ko/use-guides/settings/security/login-security-settings/#i-6)은 진행되지 않습니다.
- 구성원이 PC웹, PC앱, 드라이브 탐색기에 휴대폰 잠금해제로 로그인할 때, ID 또는 신뢰하는 기기의 휴대폰 번호를 입력하면 해당 기기에서 로그인 인증 요청을 승인해 로그인할 수 있습니다.  
  관리자는 이때 [2단계 인증](/ko/use-guides/settings/security/login-security-settings/#i-6)의 진행 여부를 설정할 수 있습니다.
- 관리자는 보안 관리를 위해 구성원들이 등록한 기기명을 확인하고, 특정 기기의 등록을 해제할 수 있습니다.

이 기능을 사용 가능한 상품 및 플랜을 사용하다가 이후 해지하더라도 관련 설정은 계속 노출됩니다. 설정이 보이지 않도록 하려면 '허용 안 함'으로 설정을 변경해야 합니다.

|  |  |
| --- | --- |
| 기능 | 옵션 |
| 휴대폰 잠금해제로 로그인 (FIDO) | - 허용: 구성원이 이 기능을 사용하여 로그인할 수 있도록 허용합니다. 모든 구성원이 이 기능을 사용할 수 있게 되었다는 알림을 받게 됩니다. - 허용 안 함: 구성원이 이 기능을 사용하여 로그인을 할 수 없도록 제한합니다. 이미 이 기능을 설정하여 사용하고 있는 구성원은 신뢰하는 기기가 모두 삭제되며, 다음 로그인부터 다른 방법으로 로그인해야 합니다. 이미 이 기능을 사용하고 있는 구성원은 알림을 받게 됩니다. |
