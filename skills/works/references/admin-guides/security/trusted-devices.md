# 신뢰하는 기기

이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 결재 상품 신청 후 사용 가능합니다.

구성원이 ['휴대폰 잠금해제로 로그인'](/ko/use-guides/settings/security/login-security-settings/#i-5)을 사용하기 위해 내 신뢰하는 기기의 전체 목록을 확인하고, 필요 시 기기를 목록에서 삭제할 수 있습니다.

['구성원' > '구성원'](/ko/admin-guides/manage-members/members/members-information/#i-4)에서 각 구성원별로 신뢰하는 기기를 확인하고 관리할 수도 있습니다.

단, 이 메뉴는 '보안' > '계정 보안'에서 ['휴대폰 잠금해제로 로그인 (FIDO)' 설정이 허용](/ko/admin-guides/security/account-security/#i-3-4)인 경우에만 노출됩니다.

# 신뢰하는 기기의 목록 확인하기

구성원이 ['휴대폰 잠금해제로 로그인'을 설정](/ko/use-guides/settings/security/login-security-settings/#i-5)할 때 신뢰하는 기기의 목록을 확인할 수 있습니다.

Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '신뢰하는 기기'를 눌러 '신뢰하는 기기' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.

# 특정 신뢰하는 기기 삭제하기

구성원이 더 이상 특정 모바일 기기에서 [휴대폰 잠금해제로 로그인](/ko/use-guides/settings/security/login-security-settings/#i-5)을 사용할 수 없도록 기기를 삭제할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '보안'을 선택해 메뉴를 펼친 후 '신뢰하는 기기'를 눌러 '신뢰하는 기기' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 삭제할 기기를 선택한 뒤 왼쪽 상단의 '기기 삭제'를 누릅니다.
3. 안내문을 확인하고 '확인'을 누릅니다.

기기를 삭제한 이후에도, 구성원이 이 기기를 다시 신뢰하는 기기로 추가하거나, 다른 로그인 수단으로 이 기기에서 다시 로그인할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
