# 구성원 추가

회사의 네이버웍스 계정에 구성원을 추가합니다.

처음 네이버웍스를 사용하거나 새로운 구성원이 입사하면 아래 방법으로 구성원을 추가할 수 있습니다.

- 관리자가 Admin에서 직접 구성원을 추가
- 네이버웍스 초대 링크를 추가될 구성원에게 공유해 구성원이 직접가입

이중 관리자가 Admin에서 구성원을 추가하는 방법에는 1명씩 추가하는 '구성원 추가'와 최대 100명까지 한꺼번에 추가할 수 있는 '구성원 일괄 추가'가 있습니다.

단, Free 상품은 최대 30명까지 구성원을 추가할 수 있습니다.

구성원이 네이버웍스를 사용하기 위해서는

- 관리자가 Admin에서 직접 구성원을 추가한 경우에는 추가된 구성원이 비밀번호를 변경하거나 생성해야 하고,
- 구성원이 초대 링크를 통해 직접 가입한 경우에는 설정에 따라 관리자가 구성원의 가입 요청을 승인해야 할 수도 있습니다.

# 구성원 개별 추가

구성원을 한 명씩 추가합니다.

관리자는 성, 이름, ID, 비밀번호의 입력만으로도 구성원을 추가할 수 있습니다.

구성원을 추가한 후 추가된 해당 구성원에게 로그인 정보를 전달합니다.

추가된 구성원은 전달받은 정보를 바탕으로 암호를 변경하거나 설정한 후 로그인합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 화면 상단의 '구성원 추가'를 선택하여 '구성원 추가' 창을 띄웁니다.
3. 항목을 입력하거나 수정한 후 '추가'를 눌러 구성원 개별 추가를 완료합니다.  
   모바일에서는 상단의 '명함 스캔하기'를 탭하고 [명함을 스캔](/ko/use-guides/contacts/external-contacts/add-edit-delete/#i-1-2)해 정보를 입력할 수도 있습니다.

'구성원 추가' 창에서 입력할 수 있는 항목 및 항목별 설명은 아래와 같습니다.

- 프로필 사진: 추가될 구성원의 사진을 등록합니다.
  - 권장하는 사진의 크기는 가로 640px, 세로 640px이며, 10MB 미만의 파일만 등록할 수 있습니다.
- 성과 이름(필수): 구성원의 성과 이름을 입력합니다.
  - 80자까지 입력할 수 있습니다.
  - 기본 환경에서 성/이름에 대한 다국어 설정을 체크한 경우 설정한 언어로 입력할 영역이 노출됩니다.
- 닉네임
  - 프로필에서 이름 옆에 표시됩니다. 호칭을 닉네임으로 부르는 회사에서 활용하면 좋습니다.

- ID(필수)
  - 2~40자의 영어 소문자, 숫자, 점(.), 하이픈(-), 밑줄(\_)을 사용할 수 있습니다. 단, 점(.)은 ID의 맨 앞이나 맨 뒤에 사용할 수 없고, 연속으로 사용할 수 없습니다.  
    메일을 사용하는 네이버웍스 코어 Standard 또는 Standard Plus 상품에서는 ID@도메인이 구성원의 대표 이메일 주소가 됩니다.  
    이 대표 이메일 주소는 외부 메시지 연동 시 '메시지 ID'로도 사용됩니다.
  - 동일한 ID가 있으면 등록되지 않습니다.
- 조직/직책
  - 구성원이 소속될 조직도 상 [조직](/ko/admin-guides/manage-members/organizations/add-organizations/)을 선택합니다.
  - ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-board-admin-plus.png) 아이콘을 눌러 '소속 조직 추가' 창을 띄운 후 '+조직 추가'를 누르면 부서를 추가할 수 있습니다.
  - 부서를 선택 후 해당 부서에서의 [직책](/ko/admin-guides/manage-members/position-level-user-type/#i)을 선택합니다.
  - 조직장일 경우 '조직장'을 체크하여 선택합니다.
  - 소속 부서는 30개까지 선택할 수 있습니다. 대표 조직은 자동완성, 목록 노출 시 우선 노출되며 조직 이름 왼쪽에 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon_check_share_member.png) 아이콘이 표시됩니다. 여러 부서에 소속된 경우 대표 부서를 변경하려면 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-member.png) 아이콘을 클릭합니다.
- 사용자 유형/직급
  - 적절한 [사용자 유형](/ko/admin-guides/manage-members/position-level-user-type/#i-3)을 선택합니다.
  - 회사에서의 직급을 선택합니다. [직급 설정](/ko/admin-guides/manage-members/position-level-user-type/)에서 설정한 값이 표시됩니다.
- 비밀번호(필수): [비밀번호 생성](/ko/admin-guides/manage-members/members/create-password/) 방법을 선택합니다. '보안 > 계정 보안 > [비밀번호 정책](/ko/admin-guides/security/account-security/#i)'에서 비밀번호의 난이도, 최소 길이 등 구성원 비밀번호의 안전한 관리를 위한 회사의 정책을 설정할 수 있습니다.
- 계정 활성화: 계정 활성화 시점을 선택합니다. '바로 활성화' 선택 시, 구성원이 즉시 네이버웍스를 이용할 수 있습니다. 계정 활성화 시점을 설정하려면 '활성화 예약'을 선택해 날짜를 기입해주세요. 해당 날짜부터 구성원이 네이버웍스를 이용할 수 있습니다.
- 사내 번호: 사내 전화번호를 입력합니다.
- 휴대폰: 휴대폰 번호를 입력합니다.
  - 구성원이 ID, 비밀번호를 잊어버렸을 때, 본인인지 확인하기 위해서 필요합니다.
- 보조 이메일: ID를 사용하는 기본 이메일 외 이메일을 추가합니다. 보조 이메일로는 메일을 보낼 수 없으며, 주소록 자동완성 기능도 제공하지 않습니다. 2~40자의 영어 소문자, 숫자, 점(.), 하이픈(-), 밑줄(\_)을 사용할 수 있습니다. 단, 점(.)은 ID의 맨 앞이나 맨 뒤에 사용할 수 없고, 연속으로 사용할 수 없습니다.
  - 동일한 ID가 있으면 등록되지 않습니다.
  - 최대 10개까지 등록할 수 있습니다.
- 개인 이메일: 개인 메일 주소입니다. 비밀번호 생성이나 변경 등 회사메일 주소를 사용할 수 없는 경우 개인메일로 관련메일이 발송됩니다. 메일을 사용할 수 없는 네이버웍스 코어 Free 또는 Lite 상품을 사용할 경우에도 개인 메일은 입력할 수 있습니다.
  - 최고 관리자의 경우 구성원 정보 수정 시 반드시 입력해야 합니다.
  - 알파벳은 소문자로 입력합니다. 메일 주소의 알파벳이 대문자로 되어 있더라도 소문자로 입력해야 합니다.
- 사용 언어
  - 추가된 구성원이 처음 로그인 할 때 서비스에 표시될 언어를 설정합니다. 한국어, 일본어, 영어, 중국어 간체, 중국어 번체 가운데 선택할 수 있습니다
- 근무처: 근무처를 입력합니다.
- 담당 업무: 담당 업무를 입력합니다.
  - 입력된 담당 업무는 프로필에 노출되어 구성원이 담당하는 업무를 파악하는데 도움이 됩니다. 담당 업무로 구성원 검색 또한 가능합니다.
- 메신저/SNS: 메신저나 SNS의 ID를 입력합니다. '항목 관리' 화면에서 서비스 내 '표시'를 선택한 경우 구성원의 프로필에 표시됩니다.
- 생일: 생일을 입력합니다. '항목 관리' 화면에서 서비스 내 '표시'를 선택한 경우 구성원의 프로필에 표시됩니다.
- 입사일: 입사일을 입력합니다. '항목 관리' 화면에서 서비스 내 '표시'를 선택한 경우 구성원의 프로필에 표시됩니다.
- 사원 번호: 사원 번호를 입력합니다. '항목 관리' 화면에서 서비스 내 '표시'를 선택한 경우 구성원의 프로필에 표시됩니다.
- 연결된 연락처
  - 비서와 같은 관련자의 연락처를 링크합니다.
  - PC웹 구성원 정보에 표시됩니다.
- 사용자 추가 항목: '항목 관리' 화면에서 추가된 회사 내 고유의 항목에 정보를 입력합니다. 총 4가지 타입으로 나뉩니다.
  - 텍스트 타입: 최대 100자의 텍스트를 입력할 수 있습니다.
  - 날짜 타입: 날짜를 입력할 수 있습니다.
  - 링크 타입: 링크를 입력할 수 있습니다. URL 정보에는 http://, https://, ws://, wss:// 등의 프로토콜이 필요합니다. 링크 텍스트는 최대 100자, URL은 최대 300자를 입력할 수 있습니다.
  - 정수 타입: -2,147,483,648부터 2,147,483,647 사이의 숫자를 입력할 수 있습니다. 소수점은 입력할 수 없습니다.

# 구성원 일괄 추가

구성원을 100명까지 한꺼번에 추가합니다.

처음 네이버웍스를 사용하는 경우나 대규모 인원이 한꺼번에 추가될 경우에 유용합니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '구성원'을 클릭해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다.
2. 오른쪽 상단의 '구성원 일괄 추가'를 클릭합니다.
3. [비밀번호 생성](/ko/admin-guides/manage-members/members/create-password/) 방법을 선택합니다.
4. '샘플 다운로드'를 클릭하여 정보를 입력할 샘플파일을 다운로드합니다. 샘플파일은 .csv와 .xlsx 형식을 제공합니다.
5. 다운로드 받은 샘플파일에 필요한 항목을 입력한 후 저장합니다. 필수 입력 항목이 아닌 경우에 입력할 데이터가 없는 항목은 빈칸으로 두고 저장합니다. 샘플파일 입력에 대한 자세한 내용은 [구성원 일괄 추가/수정 샘플파일 작성 가이드](/ko/admin-guides/manage-members/members/bulk-add-modify-member-sample-file-guide/#i)를 참고해주세요.
6. '파일 선택'을 클릭하여 저장한 파일을 선택합니다.
7. '다음'을 클릭하여 선택한 파일을 등록합니다.
   - 파일을 등록하면 아래 미리보기 화면에 추가할 구성원 목록을 확인할 수 있습니다. 또한 미리보기 화면에서 추가할 구성원 정보의 수정도 가능합니다.
8. 하단의 '일괄 추가'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
