# 비밀번호 생성

[구성원 추가](/ko/admin-guides/manage-members/members/add-members/) 시 비밀번호 생성방법입니다.

# 자동으로 생성

시스템에서 자동으로 임시 비밀번호를 생성합니다.

해당 구성원이 최초로 로그인할 때 임시 비밀번호를 반드시 변경하도록 요청할지 선택할 수 있습니다.

ID와 임시 비밀번호를 전달받은 구성원이 로그인하면 '미접속' 상태에서 서비스를 이용할 수 있는 '사용중' 상태로 전환됩니다.

1. 구성원 개별 추가 시, 비밀번호 '자동으로 생성'을 선택합니다.
2. 구성원이 처음 로그인할 때 비밀번호를 변경하도록 요청할지 옵션을 설정합니다.
3. 추가할 구성원의 정보를 입력한 후 구성원 추가 대화창에서 '추가'를 누릅니다.
4. '구성원 추가 완료' 창에서 '인쇄하기' 또는 '메일 보내기'를 클릭해, 접속하기 위해 필요한 계정 정보를 추가된 구성원에게 전달합니다. 인쇄물이나 보낸 이메일에는 비밀번호가 표시됩니다.

# 관리자가 등록

관리자가 구성원의 ID와 임시 비밀번호를 직접 생성합니다.

해당 구성원이 최초로 로그인할 때 임시 비밀번호를 반드시 변경하도록 요청할지 선택할 수 있습니다.

ID와 임시 비밀번호를 전달받은 구성원이 로그인하면 '미접속' 상태에서 서비스를 이용할 수 있는 '사용중' 상태로 전환됩니다.

1. 구성원 개별 추가 시, 비밀번호 '관리자가 등록'을 선택합니다.
2. 구성원이 처음 로그인할 때 비밀번호를 변경하도록 요청할지 옵션을 설정합니다.
3. 구성원의 비밀번호를 입력한 후 구성원 추가 대화창에서 '추가'를  누릅니다.
4. '구성원 추가 완료' 창에서 '인쇄하기' 또는 '메일 보내기'를 클릭해 접속하기 위해 필요한 정보를 추가된 구성원에게 전달합니다. 인쇄물이나 보낸 이메일에는 비밀번호가 표시됩니다.

# 구성원이 등록

구성원이 초대 메일의 비밀번호 생성 링크를 통해 직접 비밀번호를 설정하는 방법입니다. 추가될 구성원의 성, 이름, ID를 입력하여 구성원을 추가하면 구성원 개인 메일로 본인 인증 및 비밀번호 등록 안내가 담긴 초대메일이 자동으로 전송됩니다.

초대메일을 받은 구성원이 메일의 '비밀번호 등록하기' 링크로 비밀번호를 생성하면 '미접속' 상태에서 서비스를 이용할 수 있는 '사용중' 상태로 전환됩니다.

1. 구성원 개별 추가 시, 비밀번호 '구성원이 등록'을 선택합니다.
2. 서비스 초대 메일을 보낼 구성원의 이메일 주소를 입력합니다. 입력한 이메일 주소를 개인 이메일로 저장여부를 선택합니다.
3. 추가할 구성원의 정보를 입력한 후 구성원 추가 대화창에서 '추가'를 누릅니다.

구성원은 초대 메일 발송일로부터 30일 이내에 비밀번호를 등록해야 합니다. 30일을 초과해 초대 메일이 만료된 경우 관리자가 초대 메일을 재전송해야 합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
