# 소속 조직 추가와 변경

부서 이동 등 구성원이 소속된 조직이 변경되었을 때, 구성원이 소속된 조직을 변경합니다.

# 구성원 목록에서 추가 및 변경

### 모바일 앱

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽의 구성원 목록에서 프로필 사진 왼쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-contact-checkbox.png) 아이콘을 선택하여 이동할 구성원을 선택합니다.
3. 선택 후 구성원 목록 상단의 '조직 변경'을 눌러 '소속 조직 추가' 창을 띄웁니다.
4. 이동할 조직을 선택합니다.
   - '이동'을 누르면 선택한 구성원에 대한 조직 변경이 완료됩니다. 이때 기존 소속 조직 정보는 모두 삭제되고, 선택한 조직만 소속 조직으로 추가됩니다.
   - '복사'를 누르면 기존 소속 조직을 유지한 상태로 선택한 조직이 추가됩니다.

### PC 웹

조직 변경 창에서 이동

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다.
2. 오른쪽의 구성원 목록에서 프로필 사진 왼쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-contact-checkbox.png) 아이콘을 선택하여 이동할 구성원을 선택합니다.
3. 선택 후 구성원 목록 상단의 '조직 변경'을 클릭해 '소속 조직 추가' 창을 띄웁니다.
4. 이동할 조직을 선택합니다.
   - '이동'을 클릭하면 선택한 구성원에 대한 조직 변경이 완료됩니다. 이때 기존 소속 조직 정보는 모두 삭제되고, 선택한 조직만 소속 조직으로 추가됩니다.
   - '복사'를 클릭하면 기존 소속 조직을 유지한 상태로 선택한 조직이 추가됩니다.

마우스로 끌어서 이동

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다.
2. 오른쪽의 구성원 목록에서, 조직 변경을 하고자하는 구성원 위에 마우스 포인터를 올립니다.
3. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/grabber.png) 아이콘을 선택한 상태에서 마우스로 끌어 왼쪽의 조직도상 원하는 위치로 이동합니다.
   - 단, 이 경우 기존 소속 조직 정보는 모두 삭제되고 선택한 조직만 소속 조직으로 추가됩니다.

# 구성원 정보 수정에서 추가 및 변경

[구성원 정보 수정](/ko/admin-guides/manage-members/members/update-member-information/)을 통해 소속 조직을 변경합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
