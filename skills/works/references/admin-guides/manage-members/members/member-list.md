# 구성원 화면 설명

네이버웍스 내 현재 구성원 정보를 확인 및 관리하고, 신규 입사자나 퇴사자가 발생할 경우 구성원을 추가 또는 삭제할 수 있습니다.

구성원 화면은 크게 왼쪽의 조직도 영역과 오른쪽의 구성원 목록 영역으로 구성되어 있습니다.

# 조직도 영역

왼쪽 조직도 영역에서는 사내 조직 구성을 확인할 수 있습니다. 특정 조직을 선택하면, 해당 조직에 속한 구성원을 오른쪽 구성원 목록 영역에서 확인 가능합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼칩니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '구성원'을 눌러 '구성원' 화면으로 이동합니다. 왼쪽 영역이 조직도 영역입니다.

PC에서는 구성원 메뉴에서도 직접 조직을 편집, 추가 및 삭제할 수 있습니다.

### PC 웹

마우스 포인터를 조직 위에 올리면 조직 왼쪽에는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/grabber.png) 아이콘이, 오른쪽에는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘이 표시되는데, 이때 오른쪽의 아이콘을 클릭하면 '조직명 수정', '하위 조직 추가', '조직 삭제' 옵션이 표시됩니다.

- 조직명 수정: 선택한 조직명이 수정 가능한 상태로 전환되며, 편집 후 다른 영역을 클릭하면 입력한 조직명으로 저장됩니다.
- 하위 조직 추가: 선택한 조직의 하위 조직이 새롭게 추가되며, 조직명을 입력 후 다른 영역을 클릭하면 입력한 조직명으로 하위 조직이 추가됩니다.
- 조직 삭제: 선택한 조직을 삭제합니다. 조직 내 멤버가 없는 조직만 삭제할 수 있으며, 상위 조직을 삭제할 경우 하위 조직도 함께 삭제되니 유의해 주세요.
- 조직 이동: ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/grabber.png) 아이콘을 선택한 상태에서 마우스로 끌어 원하는 위치로 이동하면, 조직 위치가 변경됩니다.

# 구성원 목록 영역

조직도에서 선택된 조직에 속한 구성원의 사진과 이름, 조직, 직급, 상태 정보를 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼칩니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '구성원'을 눌러 '구성원' 화면으로 이동합니다. 오른쪽 영역이 구성원 목록 영역입니다.

Admin에서 구성원 상태는 아래 5가지로 분류하고 있으며, 각 상태별로 필요한 관리 기능을 목록 내에서 직접 수행할 수 있습니다.

- 사용 대기: 계정은 생성되었으나, 활성화되지 않아 로그인할 수 없는 상태입니다.
- 사용중: 서비스를 정상적으로 사용중인 상태이며, 만약 부재중일 경우 상태 현황에 부재 사유와 기간이 표시됩니다.
- 미접속: 추가된 구성원이 아직 첫 로그인을 완료하지 않은 상태입니다. 관리자가 구성원을 추가할 때 전송되는 초대메일 또는 계정 정보 메일 내 링크를 눌러 직접 신규 비밀번호 등록까지 완료해야 '사용중' 상태로 전환됩니다. 미접속 상태에서도 사용중 상태와 동일하게 서비스와 용량 등을 제공하니, 등록을 완료할 수 있도록 안내해 주세요. 30일이 지나면 발송한 초대메일과 발급한 임시 비밀번호가 만료되며, 이 경우 '초대메일 재발송' 또는 '비밀번호 재설정'을 통해 관리자가 다시 초대메일을 발송하거나 임시 비밀번호를 설정해 주어야 합니다.
- 일시정지: 서비스를 이용할 수 없는 정지된 상태입니다. 일시정지는 보안 상의 이유로 관리자가 직접 지정하거나, 구성원이 로그인 시 일정 횟수 이상 비밀번호를 잘못 입력한 경우 자동으로 적용됩니다. 비밀번호 입력 횟수는 Admin '보안' 메뉴에서 설정할 수 있습니다. 일시정지된 구성원이 다시 정상적으로 서비스를 사용하려면, 관리자가 직접 '해제' 버튼을 눌러 '사용중' 상태로 변경해 주어야 합니다.
- 삭제: 퇴사, 계열사 이동 등의 사유로 계정이 삭제된 상태입니다. 삭제 후 7일간 '삭제' 상태로 Admin에 표시되며, 이 기간동안 삭제 '취소' 버튼을 누르면 삭제 전 '사용중' 상태로 계정을 복원할 수 있습니다.

구성원 목록 상단의 필터 아이콘을 누르면, 위에 정의된 계정 상태에 따라 구성원을 필터링해서 볼 수 있습니다.

목록에서 구성원을 선택하면, 상단에  '조직 변경', '삭제', '비밀번호 재설정' 버튼이 표시되어 선택한 구성원을 일괄 관리할 수 있습니다.

- 조직 변경: 조직 변경은 다음의 2가지 기능이 제공됩니다.
  - 소속 조직이 유지된 상태에서 선택한 조직을 새로운 겸직 조직으로 추가하는 '추가' 기능
  - 기존 소속 조직에서 모두 빠지고 선택한 조직만 소속 조직으로 남게되는 '변경' 기능
- 계정 삭제: 구성원을 삭제할 경우 해당 멤버는 서비스에서 강제 로그아웃되며, 더 이상 네이버웍스를 이용할 수 없게 됩니다. 구성원 삭제 후 7일 내 삭제를 취소할 수 있으며, 7일 후에는 완전히 삭제되어 복원이 불가합니다. 중요한 데이터는 반드시 백업한 뒤에 구성원을 삭제해주세요.
- 비밀번호 재설정: '[비밀번호 일괄 재설정](/ko/admin-guides/manage-members/members/password-bulk-reset/)' 가이드를 참고해주세요.

## 구성원 검색

구성원 목록 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-search-767676.png) 아이콘을 누르면, 구성원을 검색할 수 있는 검색창이 표시됩니다.

검색창에 찾고자 하는 구성원 정보를 입력해 찾을 수 있으며, 계정 상태/이름/조직/직급/ID 등의 상세 조건으로 찾고 싶은 경우 '상세'를 눌러 세부적인 검색을 할 수 있습니다. '비밀번호 만료된 구성원만 보기'를 선택하면 비밀번호가 만료된 구성원만 필터링해서 조회할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
