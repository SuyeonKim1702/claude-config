# 미니 프로필 개요

미니 프로필은 구성원 정보를 확인하고 업무를 신속히 처리할 수 있는 Admin용 프로필입니다.

회사 내 동명이인이나 프로필 사진을 등록하지 않은 경우, Admin에서 이름만으로는 어떤 구성원인지 파악하기 어렵습니다.

Admin 서비스 이용 중 표시되는 구성원이 누구인지 쉽게 파악하고, 구성원 관련 업무를 신속히 처리할 수 있도록 구성원 정보와 비밀번호 변경, 휴직 설정 등의 기능을 담아 미니 프로필로 제공합니다.

Admin 내 구성원이 표시되는 영역을 누르면, 아래와 같은 미니 프로필이 표시됩니다.

미니 프로필은 관리자 누구나 볼 수 있으며, 구성원을 식별할 수 있도록 아래 정보를 제공합니다.

- 사진
- 이름(다국어 이름)
- 직급
- 대표 조직/직책
- 구성원 상태(사용 대기, 사용중, 미접속, 일시정지, 삭제)

구성원 정보뿐 아니라, 구성원 상태에 따라 비밀번호 변경, 일시 정지, 앱 사용 차단, 휴직 설정, ID 변경, 계정 삭제 등의 기능을 제공합니다.

이 기능들은 Admin '구성원' 메뉴에 권한이 있는 관리자만 실행할 수 있으며, 구성원 관리를 위해 자주 사용하는 기능에 손쉽게 접근하고 긴급한 보안 이슈 발생 시 신속히 대처할 수 있도록 도와줍니다. 각 기능은 Admin '구성원 정보'에서도 동일하게 제공하고 있으며, 미니 프로필 하단의 '관리'를 누르면 해당 '구성원 정보' 화면으로 바로 이동할 수 있습니다.

단, '구성원' 메뉴에 권한이 있는 관리자만 '구성원 정보' 화면에 접근할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
