# 구성원 삭제

퇴사 등 해당 구성원이 더 이상 회사에서 일하지 않을 경우 구성원을 삭제합니다.

구성원을 삭제하면 해당 구성원 계정으로는 더 이상 서비스를 이용할 수 없게 됩니다.

구성원을 삭제한지 7일 이내에는 삭제를 취소할 수 있지만, 삭제 후 7일이 지나면 구성원의 데이터가 완전히 삭제되어 복원할 수 없습니다.

구성원 삭제 시, 해당 구성원은 참여 중이던 모든 메시지방에서 자동으로 퇴장하며, 부재 설정 정보도 영구적으로 삭제됩니다.  
삭제를 취소하더라도 ‘이전 대화 공개’ 설정이 켜진 그룹/조직 메시지방을 제외한 메시지방에선 이전 대화 내용을 확인할 수 없으며, 부재 설정 정보도 복구되지 않습니다.  
단, 다른 구성원들의 화면에서는 해당 구성원의 대화 내용이 그대로 유지됩니다.

구성원 삭제 후 7일이 지나기 전에 구성원의 데이터를 완전히 삭제 하고싶은 경우, 구성원을 영구 삭제 할 수 있습니다. 구성원이 영구 삭제된 즉시, 구성원의 데이터가 완전히 삭제되어 복원할 수 없습니다.

# '구성원 정보' 화면에서 구성원 삭제

구성원 정보 화면에서 구성원을 삭제합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 삭제할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. 오른쪽 상단의 '관리'를 눌러 '계정 삭제'를 선택합니다. 모바일에서는 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '계정 삭제'를 탭합니다.
4. 주의사항을 확인한 후 '삭제'를 눌러 구성원을 삭제합니다.

# 구성원 목록에서 구성원 삭제

여러명의 구성원을 한꺼번에 삭제합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 구성원 목록에서 프로필 사진 왼쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-contact-checkbox.png) 아이콘을 체크하여 삭제할 구성원을 선택합니다.
3. 구성원 목록 상단의 '삭제'를 누릅니다.
4. 주의사항을 확인한 후 '삭제'를 눌러 선택한 구성원을 삭제합니다.

# '구성원 정보' 화면에서 구성원 영구 삭제

구성원 삭제 후 7일이 지나기 전에 구성원을 영구 삭제합니다.

단, 이미 삭제한 구성원만 영구 삭제가 가능합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 삭제할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. 오른쪽 상단의 '관리'를 눌러 '계정 영구 삭제'를 선택합니다. 모바일에서는 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '계정 영구 삭제'를 탭합니다.
4. 주의사항을 확인한 후 '예'를 눌러 구성원을 삭제합니다.

# 구성원 목록에서 구성원 영구 삭제

구성원 삭제 후 7일이 지나기 전에 구성원을 영구 삭제합니다.

단, 이미 삭제한 구성원만 영구 삭제가 가능합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 구성원 목록에서 영구 삭제하고자 하는 구성원의 계정 상태에서 '영구 삭제'를 누릅니다.
3. 주의사항을 확인한 후 '예'를 눌러 구성원을 영구 삭제합니다.
