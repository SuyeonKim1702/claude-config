# 비밀번호 일괄 재설정

구성원 여러 명의 비밀번호를 한번에 재설정할 수 있습니다.

재설정 방법은 두 가지입니다.

- 강제 변경: 모든 구성원의 비밀번호를 동일한 새 비밀번호로 재설정합니다.
- 변경 요청: 모든 구성원에게 비밀번호 변경 요청을 보냅니다.

단, 강제 변경을 선택할 시 아래 내용을 주의해야 합니다.

- 구성원이 비밀번호를 반드시 변경해야 한다면 보안을 위해 비밀번호를 먼저 강제 변경한 후 다시 변경 요청을 보내주세요.
- 간편 로그인 수단으로 가입한 후 ID/비밀번호를 등록하지 않은 구성원은 강제 변경 대상에서 제외됩니다.
- 구성원은 최대 5분 내로 모두 로그아웃되며, 관리자가 설정한 비밀번호로 로그인해야 합니다.
- 구성원에게 새 비밀번호를 직접 전달하거나, 구성원 개인이 비밀번호를 찾아 재설정하도록 안내해야 합니다.
- 단, 휴대폰 번호나 개인 이메일이 없는 구성원은 비밀번호를 찾을 수 없습니다.

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 누릅니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 구성원 목록에서 비밀번호를 재설정할 구성원을 선택합니다.
3. 화면 상단의 '비밀번호 재설정' 버튼을 누릅니다.
4. 재설정 안내창의 유의사항을 읽고 '확인' 버튼을 누릅니다.
5. '강제 변경'과 '변경 요청' 중 비밀번호 변경 방식을 선택하여 비밀번호를 재설정합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
