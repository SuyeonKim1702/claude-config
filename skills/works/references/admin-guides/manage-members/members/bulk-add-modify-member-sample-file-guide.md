# 구성원 일괄 추가/수정 샘플파일 입력 가이드

[구성원 일괄 추가](/ko/admin-guides/manage-members/members/add-members/#i-2)도 개별 추가와 마찬가지로 비밀번호를 자동으로 생성할 것인지, 관리자 또는 구성원이 직접 등록할 것인지 고르게 됩니다. 선택한 비밀번호 생성 방식에 해당하는 샘플 파일을 다운로드 받아주세요.

다운로드 받은 파일에서 일괄 추가 시 등록할 수 있는 구성원 정보 항목과 샘플 데이터를 확인할 수 있습니다. 첫 행에 등록 가능한 구성원 정보 항목명이 표시되어 있고, 각 항목에 해당하는 데이터를 최대 100명까지 등록할 수 있습니다. 안정적인 데이터 등록을 위해 100명으로 제한하고 있으니, 필요 시 100명 단위로 나누어 등록해 주세요.

선택한 비밀번호 생성 방식에 따라 필수로 입력해야 하는 항목은 아래와 같으며, 파일 항목명이 빨간색으로 강조되어 있습니다.

- 자동으로 생성: 성, 이름, ID, (SSO 사용 중일 경우)External Key
- 관리자가 등록: 성, 이름, ID, 비밀번호, (SSO 사용 중일 경우)External Key
- 구성원이 직접 등록: 성, 이름, ID, 개인 이메일, (SSO 사용 중일 경우)External Key

필수 항목은 구성원을 등록하기 위해 반드시 입력해야 하며, 위 항목 외에는 각 항목별 입력 규칙에 맞춰 선택적으로 추가하면 됩니다.

필수 입력 항목의 상세는 '[Admin > 항목 관리](/ko/admin-guides/manage-members/members/field-management/)'에서 설정할 수 있습니다.

## 구성원 항목별 입력 규칙

**- 성/이름**

구성원의 성과 이름을 정의합니다. 성과 이름은 합쳐서 80자까지 입력할 수 있으며, 특수문자는 ! @ & ( ) - \_ + [ ] { } , . /를 허용합니다. 성과 이름 모두 필수로 입력해야 하는 항목입니다.

**- ID**

구성원의 ID를 정의합니다. 네이버웍스 코어 Standard 또는 Standard Plus 상품을 사용할 경우, 메일 서비스에서 해당 ID로 메일을 수발신할 수 있습니다. ID는 반드시 입력해야 하는 필수값이며, 이미 사용중인 ID는 사용할 수 없습니다. 그 외 아래의 입력 규칙을 갖고 있습니다.

- ID는 2~40문자의 영문자(소문자), 숫자, 점(.), 하이픈, 언더바(\_)를 사용
- ID 시작 문자는 영문자(소문자), 숫자가 가능함
- 점(.)은 ID 시작과 마지막에 적용할 수 없으며, 연속된 형태(..)로 사용할 수 없음

**- 비밀번호**

관리자가 비밀번호를 생성하는 방식으로 멤버를 추가할 경우, 비밀번호 입력 필드가 표시됩니다. 비밀번호는 최소 8자에서 20자 사이의 영문, 숫자, 특수문자로 입력 가능한 것이 기본 정책이며, 회사별로 '[Admin > 보안 > 계정 보안> 비밀번호 정책](/ko/admin-guides/security/account-security/#i)' 에서 비밀번호 글자 수와 입력 가능한 문자를 변경해서 정의할 수 있습니다. 비밀번호는 보안 정보이니, 각별히 유의해서 취급해 주세요.

**- 개인 이메일**

구성원이 비밀번호를 생성하는 방식으로 멤버를 추가할 경우, 구성원이 직접 본인의 비밀번호를 등록할 수 있도록 네이버웍스 초대 메일을 보내야하기 때문에 개인이메일은 반드시 입력해야 하는 정보입니다. 관리자가 비밀번호를 등록하거나 자동으로 생성되는 방식이라면, 선택적으로 입력 가능합니다. 개인 이메일은 SMTP 메일 규약을 준수하며, 'ID@Domain' 형태로 정확히 입력해주어야 합니다.

\* SMTP 규약(2020년 05월 기준): local part는 영문 소문자, 숫자, 마침표(.), 하이픈(-), 언더바(\_), 느낌표(!) 조합으로 1~40자까지 허용하며, Domain을 포함해 최대 255자까지 입력할 수 있음

**- External Key**

SSO를 사용할 경우, 회사에서 기존에 사용하던 ID/PW를 사용해 로그인 할 수 있도록 External Key가 지원됩니다. 각 회사 계정별로 네이버웍스 구성원 정보를 서로 연결시켜 주어야 하는데, 이때 활용되는 식별 정보가 External Key입니다. 정의한 External Key를 토대로 기존 회사 계정과 네이버웍스 구성원 정보가 맵핑되어, 기존에 사용하던 회사 계정으로 로그인해 네이버웍스를 사용할 수 있게 됩니다. External Key는 중복될 수 없으며, 특수문자 %와 \는 허용하지 않습니다.

**- 보조 이메일**

네이버웍스 코어 Standard 또는 Standard Plus 상품에서 보조 이메일 기능을 제공합니다. ID 외에 업무용 이메일이 추가로 필요한 경우, 개인당 보조 이메일을 최대 10개까지 추가할 수 있습니다. 추가한 보조 이메일은 LINE 연동/ 네이버웍스 연동 ID로 지정할 수 있습니다. 입력 규칙은 ID와 동일합니다.

**- 닉네임**

프로필에 표시할 닉네임 정보이며, 최대 100자까지 입력할 수 있습니다. 특수문자는 ! @ & ( ) - \_ + [ ] { } , . /를 허용합니다.

**- 사용자 유형**

**- 직급**

**- 조직**

구성원의 대표 조직을 지정합니다. 조직의 정확한 위치를 알 수 있도록 1depth부터 차례대로 조직명을 입력해야 합니다. 예를 들어 3depth 조직을 대표 조직으로 지정하고 싶을 경우, 각 조직명을 '|'로 구분해 '1depth조직명|2depth조직명|3depth조직명'으로 입력해 주세요.

**- 직책**

**- 사내 번호**

회사에서 사용하는 전화 번호를 정의합니다. 숫자와 -, \*, #, +, P, T를 조합해 최대 100자까지 입력할 수 있습니다.

**- 휴대폰/국가코드**

구성원의 휴대폰 국가코드를 지정합니다. 국가코드는 숫자 앞에 '+'를 붙여야 합니다. 예를 들어 한국일 경우 '+82' 형태가 됩니다.

**- 휴대폰/번호**

구성원의 휴대폰 번호는 국가코드를 제외하고, 숫자와 -, \*, #, +, P, T를 조합해 최대 100자까지 입력할 수 있습니다. 계정정보 분실 시 휴대폰 인증을 통해 ID와 비밀번호를 찾을 수 있으니, 정확히 입력해 주세요.

**- 사용 언어**

구성원이 주로 사용하는 언어를 정의합니다. 네이버웍스에서 제공하는 5가지 언어(한국어, 일본어, 영어, 중국어(간체), 중국어(번체)) 중 하나를 선택할 수 있으며, 처음 서비스 진입 시 지정한 언어로 보여지게 됩니다. 입력 시 Korean, English, Japanese, Chinese(CHN), Chinese(TWN) 중 하나로 입력해 주세요.

**- 근무처**

구성원의 근무처 정보이며, 최대 100자까지 입력할 수 있습니다.

**- 담당 업무**

구성원이 주로 담당하는 업무이며, 최대 100자까지 입력할 수 있습니다.

**- 메신저·SNS/서비스명**

구성원이 주로 사용하는 메신저·SNS 정보를 등록할 수 있습니다. 서비스명은 라인, Facebook, Twitter를 기본으로 제공하고, 그 외 서비스명을 입력하면 '직접 입력'으로 처리됩니다.

**- 메신저·SNS/ID**

메신저·SNS 서비스에서 사용하는 ID이며, 최대 100자까지 입력할 수 있습니다.

**- 입사일**

입사한 날짜를 'YYYYMMDD' 순으로 숫자로 입력합니다.

**- 사원 번호**

구성원의 사원 번호 정보입니다. 최대 20자까지 입력할 수 있습니다.

**- 계정 활성화 날짜**

계정을 활성화할 날짜를 'YYYYMMDD' 순으로 숫자로 입력합니다. 활성화 날짜가 과거 또는 오늘인 경우 계정이 바로 활성화되며, 미래인 경우 해당 날짜에 활성화됩니다.

# 구성원 일괄 수정 - 다운로드 파일 입력 가이드

구성원 일괄 수정에서 기존 구성원 정보를 최대 100명까지 한번에 수정할 수 있습니다. 구성원 일괄 수정에서 수정하고자 하는 구성원이 속한 조직의 멤버 정보를 다운로드 받아, 편집해 주세요.

파일 내 ID는 수정하고자 하는 구성원이 누구인지 식별하는 정보로 활용되므로 수정할 수 없으며, 성과 이름은 반드시 입력해야 하는 정보입니다. 수정하지 않는 항목은 기존 정보가 그대로 입력되어 있어야 합니다. 만약 파일에서 빈 칸으로 두고 일괄 수정할 경우, 기존에 등록된 정보는 삭제 처리되니 유의해 주세요.

**-****휴대폰****/****국가코드**

국가코드 숫자 앞에 '+'를 붙여, 휴대폰 국가코드를 입력해 주세요. 예를 들어 한국일 경우 '+82' 형태가 됩니다.

**-****담당****업무**

구성원이 주로 담당하는 업무이며, 최대 100자까지 입력할 수 있습니다.

**-****메신저****·SNS/****서비스명**

주로 사용하는 메신저·SNS 서비스명이며, 라인, Facebook, Twitter를 기본으로 제공합니다. 3가지를 제외한 서비스명을 입력한 경우 '직접 입력'으로 처리됩니다.
