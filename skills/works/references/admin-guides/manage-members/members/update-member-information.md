# 구성원 정보 수정

관리자를 포함한 구성원의 정보를 수정합니다.

구성원 정보의 각 항목에 대한 설명은 [구성원 추가](/ko/admin-guides/manage-members/members/add-members/)를 참고합니다.

# 개별 구성원 정보 수정

특정 구성원을 선택하여 정보를 수정할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 수정할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. 오른쪽 상단의 '구성원 수정'을 눌러 '구성원 수정' 창을 띄운 후 필요한 정보를 수정합니다.
4. '저장'을 눌러 구성원 정보를 수정합니다.

# 개별 구성원 ID 변경

특정 구성원의 ID를 변경합니다.

ID가 변경된 구성원은 변경 후 5분 내에 모든 서비스에서 로그아웃되며 서비스를 사용하기 위해서는 변경된 ID로 다시 로그인해야 합니다.

연동 중인 메시지 ID와 대화상대 추가 URL, 내 QR 코드도 변경된 ID로 적용되어 기존에 사용하던 ID는 사용할 수 없습니다.

## 기존 ID 변경

겸직 회사에서 동일한 ID를 사용하고 있다면, 겸직 회사의 ID도 함께 변경됩니다.

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 변경할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. 오른쪽 상단의 '구성원 수정'을 눌러 '구성원 수정' 창을 띄웁니다.
4. ID 항목 오른쪽의 '변경'을 누르고 '확인'을 누릅니다.
5. 변경할 새 ID를 입력하고 '변경하기'를 누릅니다.

## 임시 ID 변경

임시 ID를 변경하는 경우 네이버웍스 계정을 정식으로 만드는 것이므로 비밀번호도 함께 등록해야 합니다.

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 변경할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. 오른쪽 상단의 '구성원 수정'을 눌러 '구성원 수정' 창을 띄웁니다.
4. ID 항목 오른쪽의 '변경'을 누르고 '확인'을 누릅니다.
5. 변경할 새 ID와 비밀번호를 입력하고 '변경하기'를 누릅니다.

# 비밀번호 변경하기

구성원의 비밀번호를 변경하거나 구성원에게 비밀번호를 변경할 것을 요청합니다.

비밀번호가 탈취되어 구성원 본인이 아닌 다른 사람이 계정을 악용할 수 있는 경우에, 비밀번호를 변경하여 회사의 정보를 보호할 수 있습니다.

비밀번호를 변경하면 5분 내 연결된 모든 네이버웍스서비스에서 로그아웃되고, 변경한 비밀번호로 다시 로그인하여야 합니다.

비밀번호가 노출되었다고 판단되거나, 구성원이 비밀번호를 잊어버린 경우 관리자가 구성원의 비밀번호를 강제로 변경할 수도 있습니다.

또는, 구성원이 비밀번호를 오랫동안 변경하지 않은 경우에 구성원에게 비밀번호 변경을 요청할 수 있습니다.

## 비밀번호 강제 변경

구성원의 비밀번호를 관리자가 변경합니다.

구성원이 비밀번호를 잊어버려서 로그인할 수 없는 경우에 기존 비밀번호를 몰라도 새로운 비밀번호를 생성할 수 있습니다.

변경 후 최대 5분 내 연결된 모든 네이버웍스 서비스에서 로그아웃됩니다.

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 변경할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. '보안 설정'에서 비밀번호의 '변경'을 누릅니다.
4. '강제 변경' 탭을 클릭합니다.
5. 구성원의 새로운 비밀번호를 입력합니다.
6. '변경하기'를 클릭합니다.

변경된 비밀번호를 해당 구성원에게 알려주어야 해당 구성원이 정상적으로 접속할 수 있습니다.

## 비밀번호 변경 요청

구성원에게 비밀번호 변경을 요청합니다.

구성원에게 메시지로 비밀번호를 변경하도록 요청하는 안내 메시지가 전달됩니다. 안내 메시지의 링크를 통해 비밀번호를 변경하거나 다음 로그인 시 반드시 비밀번호를 변경해야만 서비스를 계속 이용할 수 있습니다.

구성원이 비밀번호를 변경하면 바로 로그인 창으로 연결되어 새로운 비밀번호로 접속해야 서비스를 이용할 수 있습니다.

변경 후 최대 5분 내 연결된 모든 네이버웍스 서비스에서 로그아웃됩니다.

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 변경할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. '보안 설정'에서 비밀번호의 '변경'을 누릅니다.
4. '변경 요청' 탭을 클릭합니다.
5. '요청하기'를 클릭합니다.

# 운영 관리자 권한 부여/회수

구성원 정보 수정 화면에서 운영 관리자 권한을 부여하거나 회수할 수 있습니다.

운영 관리자는 최고 관리자, 부관리자와 함께 구성원 관리, 서비스 관리 등 네이버웍스 운영에 필요한 설정을 함께 관리할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 변경할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. 오른쪽 상단의 '구성원 수정'을 누릅니다.
4. '운영 관리자 권한' 항목 오른쪽의 '부여' 또는 '회수'를 누릅니다.

# 계정 활성화 날짜 변경

구성원 정보 수정 화면에서 계정 활성화 날짜를 변경할 수 있습니다.

계정이 활성화되기 이전에만 날짜 변경이 가능합니다.

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 누릅니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 변경할 구성원을 선택하여 '구성원 정보' 화면으로 이동합니다.
3. 오른쪽 상단의 '구성원 수정'을 누릅니다.
4. '계정 활성화' 항목 오른쪽의 '변경'을 누릅니다.
5. '바로 활성화' 또는 '활성화 예약'을 선택합니다.
6. '저장'을 누릅니다.

# 구성원 정보 일괄 수정

구성원 정보를 다운로드 받아 수정한 후 다시 등록하여 여러 구성원 정보를 한꺼번에 수정합니다.

단, 구성원 일괄 수정 시 기존 구성원 정보에 덮어쓰기됩니다. 빈칸으로 두고 저장하면 기존 정보가 삭제됩니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '구성원' 메뉴 선택 후, '구성원'을 클릭합니다.
2. 오른쪽 상단의 '관리'를 클릭해 '구성원 일괄 수정'을 선택합니다.
3. '구성원 정보 다운로드(CSV)' 또는 '구성원 정보 다운로드(XLSX)'를 클릭하여 현재 구성원의 정보를 다운로드합니다.
4. 다운로드한 파일을 열어 필요한 부분을 수정 후 저장합니다.
5. '파일 선택'를 클릭하여 저장한 파일을 선택합니다.
6. '다음'을 클릭하여 선택한 파일을 등록합니다.
7. 파일을 등록하면 아래 미리보기 화면에서 수정된 구성원 목록을 확인할 수 있습니다. 또한 미리보기 화면에서 구성원 정보의 수정도 가능합니다.
8. 하단의 '일괄 수정'을 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
