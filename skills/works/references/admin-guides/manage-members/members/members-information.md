# 구성원 정보 화면 설명

Admin '구성원' 메뉴의 구성원 목록에서 특정 멤버를 선택하거나 미니 프로필 하단의 '관리'를 누르면, 해당 구성원의 상세 정보 화면으로 이동합니다.

등록된 프로필 정보 외에 서비스 이용 현황이나 보안 상태를 한눈에 파악하고 구성원과 관련된 과업을 한 곳에서 효율적으로 관리할 수 있도록, 아래 정보 및 기능을 통합 제공합니다.

# 구성원 정보

Admin에 등록된 구성원의 프로필 정보(소속, 연락처 등)를 확인하고, 이를 수정할 수 있습니다.

- 소속 조직이 여러 개일 경우, 상단의 ![](/wp-content/uploads/2024/04/1596004083.png) 아이콘으로 강조된 조직이 대표 조직입니다.
- 구성원 정보 상단 '구성원 수정'을 눌러, 등록된 프로필 세부 정보를 수정할 수 있습니다.

# 보안 설정

서비스를 보다 안전하게 사용할 수 있도록 구성원의 보안 관련 현황을 보여주고, 필요한 보안 기능을 설정할 수 있게 합니다.

- 비밀번호: 마지막으로 비밀번호를 변경한 날짜가 표시되며, 비밀번호 변경이 필요할 경우 관리자가 직접 변경하거나 구성원이 바꾸도록 변경 요청을 보낼 수 있습니다. 비밀번호가 만료된 구성원은 '비밀번호 만료'가 표시됩니다.
- 간편 로그인: 구성원이 [휴대폰 번호 또는 네이버 아이디, LINE으로 로그인](/ko/use-guides/settings/security/login-security-settings/)하고 있는지 여부를 확인하고, 간편 로그인 연동을 직접 해제할 수 있습니다.
- 구성원 검색 시 자동완성: 설정을 켜면 구성원 검색창에 키워드 입력 시 구성원(원직/겸직)을 검색할 수 있고, 검색 시 자동완성 기능이 지원됩니다. 설정을 끄면 구성원을 검색할 수 없습니다.
- 구성원별 조직 보안 설정
  - 조직도에서 해당 구성원이 소속된 조직을 선택했을 때, 조직 멤버로 표시될지 여부를 정할 수 있습니다. 해당 구성원이 특정 조직에 소속된 것을 다른 사내 직원들이 알 수 없도록 하려면 조직 '비공개'로 설정합니다.
  - 조직 단위로 제공되는 기능에 대한 권한 여부를 설정합니다. '조직으로 공유 대상 지정 시 포함'이 '포함 안 함'일 경우 조직 메시지방/노트/일정/폴더를 이용할 수 없으며 조직 단위로 권한이 부여된 설문과 게시판에 접근할 수 없습니다.
- 2단계 인증
  - 구성원이 [2단계 인증](/ko/use-guides/settings/security/two-factor-authentication/)을 사용하고 있는지 확인할 수 있습니다.
  - [2단계 인증 설정](/ko/admin-guides/security/account-security/#i-2)에서 '선택 - 구성원 설정에 따름'을 선택한 경우, '사용' 또는 '해제'를 눌러 이 구성원이 2단계 인증을 사용하게 할지 설정할 수 있습니다.
  - [2단계 인증 설정](/ko/admin-guides/security/account-security/#i-2)에서 '필수 - 모든 구성원에게 2단계 인증 적용'을 선택한 경우, '초기화'를 눌러 이 구성원의 2단계 인증 수단을 초기화하고 다시 입력하도록 할 수 있습니다.

# 사용 용량

구성원이 사용중인 드라이브와 메일의 사용 용량을 볼 수 있습니다.

# 디바이스

최근 3년간 네이버웍스 모바일앱, PC앱, 드라이브앱, 드라이브 탐색기에 접속한 구성원의 모든 디바이스 정보를 확인할 수 있습니다.

- 설치한 앱이 최신이 아닐 경우 설치 앱 하단에 '업데이트 필요'라고 나오며, 이 링크를 누르면 앱을 업데이트 하도록 구성원에게 서비스 알림을 보낼 수 있습니다.
- MDM에 등록된 업무용 모바일 디바이스를 분실해 정보 탈취가 우려되거나 퇴사하게 될 경우, 디바이스 상세 정보에서 '앱 데이터 삭제'를 실행할 수 있습니다. 단, 한번 삭제된 데이터는 되돌릴 수 없습니다.

# 접속 현황

최근 90일간 네이버웍스 서비스 접속 기록이 표시됩니다.

- 앱 접속과 웹 접속을 분리해 제공합니다.
- 접속 환경과 위치를 파악해, 필요 시 '로그아웃'을 눌러 접속을 강제로 종료시킬 수 있습니다.

그 밖에도, 구성원 상태에 따라 화면 오른쪽 상단의 '관리'를 통해 앱 사용 차단, 휴직 설정, 계정 삭제, 일시 정지 등의 기능을 제공합니다.

# 앱 사용 차단

구성원 정보 화면 오른쪽 상단의 '관리'를 눌러 '앱 사용 차단'을 선택합니다. 모바일에서는 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '앱 사용 차단'을 탭합니다.

구성원의 네이버웍스 모바일앱 및 PC앱 접속을 차단합니다. (PC웹은 정상적으로 이용 가능)

앱 사용이 차단된 구성원의 경우 '관리'를 누르면 '앱 사용 차단 해제'가 나타납니다.

# 계열사 이동

이 기능은 그룹사 기능을 활성화한 후 사용 가능합니다.

구성원의 소속 계열사를 변경합니다.

구성원 정보 화면 오른쪽 상단의 '관리'를 눌러 '계열사 이동'을 선택합니다. 모바일에서는 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '계열사 이동'을 탭합니다. 신규 소속 계열사를 선택하고 '저장'을 누릅니다.

계열사 이동 후 기존 조직의 메시지방에서는 모두 퇴장하게 되며, 5분 내 모든 기기에서 로그아웃됩니다.

외부 사용자와 연동되어 있는 경우, 연동에 필요한 메시지 ID, 초대 링크, QR 코드가 변경됩니다.

이동 후 계열사에서 이용 중인 상품이 기존과 다르다면 아래와 같이 데이터가 삭제될 수 있으므로 주의하시기 바랍니다.

# 휴직 설정

구성원 정보 화면 오른쪽 상단의 '관리'를 눌러 '휴직 설정'을 선택합니다. 모바일에서는 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '휴직 설정'을 탭합니다.

관리자는 구성원의 휴직 여부에 따라 Admin에서 휴직으로 설정할 수 있으며 휴직 기간도 설정할 수 있습니다.

휴직으로 설정하면 구성원의 프로필과 조직도에 휴직 정보가 표시됩니다.

휴직 기간을 설정했을 경우 기간이 지나면 자동으로 휴직 표시는 사라집니다.

# 계정 삭제

구성원 정보 화면 오른쪽 상단의 '관리'를 눌러 '계정 삭제'를 선택해 해당 [구성원을 삭제](/ko/admin-guides/manage-members/members/delete-members/)합니다. 모바일에서는 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '계정 삭제'를 탭합니다.

삭제 후 7일이 안된 구성원의 경우 '관리'를 누르고 '삭제 취소'를 선택하면, 삭제를 취소할 수 있습니다.

# 일시 정지

구성원 정보 화면 오른쪽 상단의 '관리'를 눌러 '일시 정지'를 선택합니다. 모바일에서는 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '일시 정지'를 탭합니다.

구성원의 네이버웍스 사용을 일시정지합니다.

일시 정지 시 해당 구성원의 접속이 제한되며, 구성원의 부재 설정 정보도 초기화됩니다.

단, 일시 정지가 되어도 해당 구성원의 계정과 데이터는 삭제되지 않고 유지됩니다.

계정이 일시 정지된 구성원의 경우 '관리' 클릭 시 '일시정지 해제'가 노출됩니다.

# 초대메일 재발송

비밀번호 생성 방식이 '구성원이 등록'인 구성원에 대해 초대메일을 재발송할 수 있습니다.

구성원 정보 화면 오른쪽 상단의 '관리'를 눌러 '초대메일 재발송'을 선택합니다. 모바일에서는 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '초대메일 재발송'을 탭합니다.

개인 이메일이 등록되지 않은 구성원이라면, 개인 이메일 정보를 입력한 후 해당 메일로 초대메일을 재발송합니다.

# 비밀번호 재설정

비밀번호 생성 방식이 '자동으로 생성' 혹은 '관리자가 등록'인 구성원에 대해 [비밀번호를 재설정](/ko/admin-guides/manage-members/members/add-expired-members/)할 수 있습니다.

구성원 정보 화면 오른쪽 상단의 '관리'를 눌러 '비밀번호 재설정'을 선택합니다. 모바일에서는 오른쪽 상단의 ![](/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '비밀번호 재설정'을 탭합니다.

비밀번호를 재생성한 후, 구성원에게 메일을 변경된 내용을 알려주거나, 인쇄물로 출력할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
