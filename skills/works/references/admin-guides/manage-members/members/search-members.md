# 구성원 통합 검색

다양한 방법으로 구성원을 검색할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단에 있는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-search-767676.png) 아이콘을 선택해 검색창으로 진입합니다.
3. 검색창에서 다음의 키워드를 이용해 검색합니다.
   - 이름/다국어 이름
   - ID
   - 조직명
   - 보조이메일
   - 사내 번호
   - 휴대폰

보다 상세한 검색이 필요하다면, 검색창 오른쪽의 '상세'를 선택합니다. 아래의 조건을 조합하여 구성원을 검색할 수 있습니다.

- 계정 상태
- 이름
- 조직
- 직급 ([직급을 사용하도록 설정](/ko/admin-guides/manage-members/position-level-user-type/#i-2)한 경우에 한 함)
- ID
- 보조 이메일
- 사용자 유형 ([사용자 유형을 사용하도록 설정](/ko/admin-guides/manage-members/position-level-user-type/#i-3)한 경우에 한 함)

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
