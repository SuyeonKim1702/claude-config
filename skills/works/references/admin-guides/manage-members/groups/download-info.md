# 그룹 정보 다운로드

등록된 모든 그룹의 정보를 CSV 또는 XLSX 형식으로 다운로드할 수 있습니다.

다운로드 준비가 완료되면 서비스 알림 메시지가 발송되며, 다운로드 링크는 5일 후 만료됩니다.

### PC 웹

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후, '그룹'을 클릭합니다.
2. 화면 오른쪽 상단의 '더보기'를 클릭하고 '그룹 정보 다운로드'를 클릭합니다.
3. 파일 형식을 선택합니다.
   - CSV: 쉼표로 항목을 구분한 텍스트 파일로, Outlook 등에서 사용할 수 있습니다.
   - XLSX: 엑셀 파일 형식으로 다양한 편집이 가능하며, 대부분의 주소록 서비스에서 사용할 수 있습니다.
4. 다운로드할 정보를 선택합니다.
   - 설정 정보: 그룹의 기본 정보(그룹명, 설명, 마스터, 공개 여부 등)입니다.
     - 사용 용량 포함: 공용용량과 메시지방 기능별 사용 용량 정보를 포함합니다.
     - 최근 메시지방 사용일 포함: 마지막으로 그룹 메시지방이 사용된 날짜 정보를 포함합니다.
   - 멤버 정보: 그룹 멤버 목록과 각 멤버의 마스터 여부 정보입니다.
5. 다운로드할 그룹을 선택합니다.
   - 전체 그룹: 등록된 모든 그룹의 정보를 다운로드합니다.
   - 지정한 그룹: 원하는 그룹을 선택해 다운로드합니다. 그룹은 1,000개까지만 선택 가능합니다.
6. '다운로드'를 클릭한 후 '확인'을 클릭합니다.
7. 수신된 서비스 알림 메시지에서 '파일 다운로드'를 클릭합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
