# 삭제한 그룹 데이터 복원

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

그룹 삭제 후 30일 이내에 해당 그룹에서 사용하던 메시지, 노트, 일정, 할 일, 폴더, 메일링리스트 데이터를 삭제 전 상태로 복원할 수 있습니다.

# 내부 그룹

삭제한 내부 그룹의 메시지, 노트, 일정, 할 일, 폴더, 메일링리스트 데이터를 삭제 전 상태로 복원합니다.

단, 다른 그룹에서 같은 그룹명과 메일링리스트를 이미 사용하고 있다면 다른 이름으로 변경해 복원할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원을 선택해 메뉴를 펼친 후 '그룹'을 눌러 '그룹' 화면으로 이동합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 그룹 목록 상단의 '그룹 데이터 복원'을 클릭합니다. 모바일에서는 오른쪽 상단의  ![](/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '그룹 데이터 복원'을 탭합니다.
3. 그룹 목록에서 데이터 복원이 필요한 그룹의 '복원'을 누릅니다.  
   그룹명과 메일링리스트가 중복일 경우 수정합니다.
4. '데이터 복원'을 누릅니다.
5. '확인'을 누릅니다.

# 외부 그룹

삭제한 외부 그룹의 메시지, 노트, 일정, 폴더 데이터를 삭제 전 상태로 복원합니다.

# 그룹 데이터 복원 내역 확인

하단의 '상세 보기'를 눌러 '데이터 복원 내역' 창에서 복원한 기능별 용량 등 상세 복원 내역을 확인할 수 있습니다.

데이터 복원이 완료되면 '복원' 버튼이 '복원 완료'로 변경되어 더 이상 데이터를 복원할 수 없게 됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
