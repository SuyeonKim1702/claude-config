# 그룹 추가

그룹을 만듭니다.

그룹은 소속 [조직](/ko/admin-guides/manage-members/organizations/)과는 별개로 프로젝트 및 업무 단위로 묶인 구성원 그룹입니다. 그룹은 구성원 구성에 따라 ['(내부)그룹'](/ko/use-guides/contacts/group/internal-group/)과 ['외부 그룹'](/ko/use-guides/contacts/group/external-group/)으로 구분합니다.

- 내부 그룹: 우리 회사의 직원으로만 구성된 그룹
- 외부 그룹: 우리 회사를 포함해 여러 회사의 직원으로 구성된 그룹

그룹에 속한 구성원을 멤버라고 하며, 그룹은 멤버와 이 멤버를 관리하는 그룹의 마스터로 구성되어 있습니다.

그룹은 내부 그룹과 외부 그룹 모두 포함해 유료 플랜에서는 최대 100,000개, 무료 플랜에서는 최대 1,000개까지 등록할 수 있습니다.

# 내부 그룹 추가

우리 회사의 직원으로만 구성된 그룹을 만듭니다.

관리자는 'Admin > 서비스 > 주소록 > 일반'에서 ['그룹 생성 권한'](/ko/admin-guides/manage-service/contacts/contacts-settings/#i-2) 항목을 통해 관리자만 그룹을 생성할 수 있게 할지, 모든 [구성원이 그룹을 생성](/ko/use-guides/contacts/group/internal-group/#i-2)할 수 있게 할지 선택할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '그룹'을 눌러 '그룹' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단의 '그룹 추가'를 눌러 '그룹'을 선택합니다.
3. 항목을 입력한 후 '추가'를 눌러 그룹 추가를 완료합니다. '저장 후 계속 추가'를 선택하면 계속해서 그룹을 추가할 수 있습니다.

'그룹 추가' 창에서 입력할 수 있는 항목 및 항목별 설명은 아래와 같습니다.

- 그룹 기본 정보
  - 대표 사진: 카메라 아이콘을 눌러 그룹 메시지방의 사진을 설정합니다.
  - 그룹명(필수): 100자 이내로 입력합니다. 특수 문자의 입력에 제한은 없습니다.
  - 설명: 그룹에 대한 설명을 입력합니다.
  - 마스터(필수): 그룹 마스터를 지정합니다. 그룹 마스터는 여러 명을 지정할 수도 있습니다. 관리자가 권한을 제한하지 않을 경우 그룹 마스터는 노트, 폴더, 일정 등의 사용 여부, 그룹명 변경, 공개 여부 등의 그룹 설정을 할 수 있고, 멤버를 추가 또는 삭제하는 등 그룹 멤버 관리를 할 수 있습니다.
  - 멤버(필수): 그룹을 사용할 멤버를 선택합니다. 구성원, 조직, 그룹을 선택할 수 있습니다.
- 메시지방 기능
  - 메시지: 그룹 멤버가 참여하는 메시지방 사용 여부를 선택합니다. 메시지 사용을 켜야만 해당 그룹의 노트, 일정, 할 일, 폴더를 사용할 수 있습니다.
    - 노트: 노트 사용 여부를 선택합니다. 노트를 사용하면 해당 그룹 내 사진 또는 파일을 포함한 게시물을 공유할 수 있습니다.
    - 일정: 일정 사용 여부를 선택합니다. 일정을 사용하면 캘린더에 그룹의 일정을 등록하고 공유할 수 있습니다.
    - 할 일: 할 일 사용 여부를 선택합니다. 할 일을 사용하면 그룹 멤버와 할 일을 공유하고 관리할 수 있습니다.
    - 폴더: 폴더 사용 여부를 선택합니다. 폴더를 사용하면 파일을 폴더에 올려서 그룹 멤버간 공유하거나 다운로드 받을 수 있습니다.
  - 데이터 삭제: 메시지방 기능(노트, 일정, 폴더)의 데이터를 삭제할 수 있습니다. 데이터가 있는 경우에만 삭제할 수 있으며, 삭제 시 복구할 수 없습니다.
  - 메일링리스트: 메일링 리스트 사용 여부를 선택하고 주소를 입력합니다. 메일링 리스트를 사용하면 해당 메일링 리스트로 메일을 수신했을 때, 그룹의 모든 멤버가 메일을 받게 됩니다. 그룹 멤버가 추가되더라도 메일링 리스트는 수정할 필요가 없어서 편리합니다. 영문 소문자, 숫자, 마침표(.), 하이픈(-), 밑줄(\_), 느낌표(!), 샵(#)을 조합하여 2~40자의 범위 내에서 만들 수 있습니다. 단, 구성원, 그룹, 조직의 메일을 포함하여 사용중인 메일 주소를 중복하여 사용할 수 없습니다.
- 고급 설정
  - 마스터에게 메시지방 기능 권한 부여 : 설정을 켜면, 메시지방 기능(메시지, 노트, 일정, 할 일, 폴더, 메일링 리스트 등) 사용 여부를 해당 그룹의 마스터가 설정할 수 있도록 합니다.
  - 모든 멤버에게 그룹 알림 보내기: 설정을 켜면 그룹명이 변경되거나 그룹이 삭제되었을 때 해당 그룹의 모든 멤버에게 알림을 보냅니다. 그룹에 새롭게 초대되거나 그룹에서 제외된 멤버에게도 초대와 제외 관련 알림을 보냅니다.
  - 그룹 공개: 그룹의 공개 여부를 선택합니다. 그룹은 공개 여부에 따라 공개 그룹과 비공개 그룹으로 나뉘어 지고, 비공개 그룹은 그룹 멤버가 아닌 사용자의 전체 그룹 목록이나 검색, 자동완성 결과에 보이지 않습니다. '그룹 공개' 설정을 켜면 공개 그룹이 됩니다.
  - 메일링 리스트 설정:
    - 외부 도메인 메일 수신을 수신 차단하면, 외부 도메인에서 발송한 메일의 수신을 차단하고 사내 메일 주소에서 발송된 메일만 받습니다.
    - 보내는 메일 주소로 사용할 수 있는 구성원을 제한하면, '멤버 지정'을 통해 권한을 부여받은 구성원만 메일링 리스트의 메일 주소를 보내는 메일 주소로 사용할 수 있습니다. 권한을 부여받은 구성원이 메일링 리스트를 보내는 메일 주소로 메일을 보내게 되면, 다른 구성원은 자동으로 숨은 참조로 보낸 메일을 받아보게 됩니다. 이 메일을 받은 사람이 회신을 하면 그룹의 모든 구성원이 메일을 받게 됩니다. 관리자는 ['Admin > 감사 > 메일'](/ko/admin-guides/audit/#i-10)에서 실제 메일을 보낸 구성원을 확인할 수 있습니다.
    - 메일 주소로 메일을 보낼 수 있는 구성원을 제한하면, '멤버 지정'을 통해 권한을 부여받은 구성원만 메일링 리스트의 메일 주소로 메일을 보낼 수 있습니다.
    - 그룹 메일링 리스트에 고객/거래처 연락처 또는 외부 메일을 외부 연락처로 등록할 수 있습니다.

# 외부 그룹 추가

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

우리 회사를 포함해 여러 회사의 직원으로 구성된 그룹을 만듭니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '그룹'을 눌러 '그룹' 화면으로 이동합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단의 '그룹 추가'를 눌러 '외부 그룹'을 선택합니다.
3. 항목을 입력한 후 '추가'를 선택하여 그룹 추가를 완료합니다.
   - '저장 후 계속 추가'를 선택하면 계속해서 그룹을 추가할 수 있습니다.
   - 생성된 외부 그룹 앞에는 외부 그룹을 의미하는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-group_external.png) 아이콘과 비공개 상태를 의미하는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon_group_hidden.png) 아이콘이 표시됩니다.

'외부 그룹 추가' 창에서 입력할 수 있는 항목 및 항목별 설명은 아래와 같습니다.

- 그룹 기본정보
  - 대표 사진: 카메라 아이콘을 눌러 그룹 메시지방의 사진을 설정합니다.
  - 그룹명(필수): 100자 이내로 입력합니다. 특수 문자의 입력에 제한은 없습니다.
  - 설명: 그룹에 대한 설명을 입력합니다.
  - 마스터(필수): 그룹 마스터를 지정합니다. 그룹 마스터는 ['다른 회사의 네이버웍스 연동'](/ko/admin-guides/manage-service/message/message-with-external-users/#_LINE_WORKS) 권한이 있어야 하며, 여러 명을 지정할 수 있습니다. 관리자가 권한을 제한하지 않을 경우 그룹 마스터는 노트, 폴더, 일정 등의 사용 여부, 그룹명 변경 등의 그룹 설정을 할 수 있고, 멤버를 추가 또는 삭제하는 등 그룹 멤버 관리를 할 수 있습니다.
  - 멤버: 외부 그룹을 사용할 멤버를 지정합니다. ['다른 회사의 네이버웍스 연동'](/ko/admin-guides/manage-service/message/message-with-external-users/#_LINE_WORKS) 권한이 있는 사내 구성원을 멤버로 지정할 수 있습니다. 우리 회사가 아닌 다른 회사에 소속된 외부 대화상대는 여기서 바로 그룹 멤버로 지정할 수 없고, 외부 그룹이 생성되면 생성된 해당 그룹의 멤버가 주소록에서 추가해야 합니다.
- 메시지방 기능
  - 메시지: 외부 그룹 멤버가 참여하는 메시지방 사용 여부를 선택합니다. 메시지 사용을 켜야만 해당 그룹의 노트, 일정, 폴더를 사용할 수 있습니다.
    - 노트: 노트 사용 여부를 선택합니다. 노트를 사용하면  해당 그룹 내 사진 또는 파일을 포함한 게시물을 공유할 수 있습니다.
    - 일정: 일정 사용 여부를 선택합니다. 일정을 사용하면 캘린더에 그룹의 일정을 등록하고 공유할 수 있습니다.
    - 폴더: 폴더 사용 여부를 선택합니다. 폴더를 사용하면 파일을 폴더에 올려서 그룹 멤버간 공유하거나 다운로드 받을 수 있습니다.
  - 고급 설정
    - 마스터에게 메시지방 기능 권한 부여: 설정을 켜면, 그룹 사용 기능(메시지, 노트, 일정, 폴더 등) 사용 여부를 해당 그룹의 마스터가 설정할 수 있도록 합니다.
    - 모든 멤버에게 그룹 알림 보내기: 설정을 켜면 그룹명이 변경되거나 그룹이 삭제되었을 때 해당 그룹의 모든 멤버에게 알림을 보냅니다. 그룹에 새롭게 초대되거나 그룹에서 제외된 멤버에게도 초대와 제외 관련 알림을 보냅니다.
    - 그룹 공개: 외부그룹은 비공개 설정만 가능하고, 공개로 설정할 수 없습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
