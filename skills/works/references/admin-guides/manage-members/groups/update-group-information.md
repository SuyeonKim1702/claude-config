# 그룹 정보 수정

등록된 그룹의 정보를 수정합니다.

그룹 정보의 각 항목에 대한 설명은 [내부 그룹 추가](/ko/admin-guides/manage-members/groups/create-groups/#i) 또는 [외부 그룹 추가](/ko/admin-guides/manage-members/groups/create-groups/#i-2)를 참고합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '그룹'을 눌러 '그룹' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 그룹 목록에서 수정할 그룹명을 선택합니다. 수정할 그룹이 내부 그룹이면 '그룹 정보' 창이, 외부 그룹이면 '외부 그룹 정보' 창이 뜹니다. 단, 그룹 목록에는 외부 그룹 중 우리 회사에서 생성한 외부 그룹만 노출됩니다.
   - '그룹 정보'를 선택한 상태에서 '수정'을 선택하면, 그룹 추가 시 입력한 정보를 수정할 수 있습니다.
   - '멤버'를 선택한 상태에서 '마스터 변경'을 선택하면, 마스터를 수정할 수 있습니다.
3. 필요한 정보를 수정한 후 '저장'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
