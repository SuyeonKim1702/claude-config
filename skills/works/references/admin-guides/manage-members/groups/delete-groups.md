# 그룹 삭제

그룹을 선택하여 삭제합니다.

그룹에 멤버가 남아있더라도 삭제할 수 있습니다.

그룹 삭제 시 모든 대화 이력과 사용중인 노트, 일정, 폴더에 저장된 데이터, 그룹 메일 주소가 삭제됩니다.

외부 그룹을 삭제하면 해당 그룹을 사용중인 다른 모든 회사에서도 삭제됩니다.

# 그룹 화면에서 그룹 삭제

'그룹' 화면의 목록에서 그룹을 삭제합니다. 한꺼번에 여러 그룹을 삭제할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '그룹'을 눌러 '그룹' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 그룹 목록에서 삭제할 그룹 왼쪽 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-contact-checkbox.png) 아이콘을 체크합니다.
3. 상단의 '삭제'를 누릅니다.
4. 주의사항을 확인 후 '확인'을 누릅니다.

# 그룹 수정창에서 그룹 삭제

그룹 수정창에서 그룹을 삭제합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '그룹'을 눌러 '그룹' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 그룹화면의 그룹 목록에서 삭제할 그룹을 눌러 '그룹 정보' 창 또는 '외부 그룹 정보' 창을 띄웁니다.
3. '그룹 삭제'를 누릅니다.
4. 주의사항을 확인 후 '확인'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
