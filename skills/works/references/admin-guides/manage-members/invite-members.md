# 구성원 초대

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

링크나 QR코드로 구성원을 초대하는 기능의 사용 여부를 설정합니다.

초대를 받아 가입을 한 사람에 대해 최종적으로 관리자가 가입승인을 할지 설정합니다.

# 구성원 초대 기능 사용 여부 설정

링크나 QR코드를 통한 초대 기능의 사용 여부를 설정합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원 초대'를 눌러 '구성원 초대' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '구성원 초대'를 선택합니다.
3. 오른쪽 상단의 '수정'을 선택합니다.
4. '링크/QR 코드 초대' 기능의 사용여부를 설정합니다. '링크/QR 코드 초대'를 켜면, '초대권한' 항목에서 구성원 초대 권한을 가질 구성원의 범위를 설정할 수 있습니다.
   - 모든 구성원이 초대 가능: 모든 구성원이 다른 구성원을 초대할 수 있는 권한을 가집니다.
   - 관리자만 초대 가능: [관리자 권한](/ko/admin-guides/security/administrator-authorities/manage-administrator-authorities/)에서 구성원 초대 권한이 있는 관리자만 다른 구성원을 초대할 수 있는 권한을 가집니다.
5. 오른쪽 상단의 '저장'을 누릅니다.

# 자동 승인 설정

초대를 받아 가입을 한 사람에 대해 최종적으로 관리자가 가입 승인을 할지 설정합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원 초대'를 눌러 '구성원 초대' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '구성원 초대'를 선택합니다.
3. 오른쪽 상단의 '수정'을 선택합니다.
4. 자동 승인의 설정값을 선택합니다.
   - '자동 승인'을 켜면, 초대를 받아 가입을 신청한 사람이 관리자의 승인 없이 서비스를 이용할 수 있습니다.
   - '자동 승인'을 끄면, 가입을 신청한 사람은 관리자의 승인 후, 서비스를 이용할 수 있습니다.
5. 페이지 오른쪽 상단의 '저장'을 누릅니다.

# 가입 신청자 확인 및 승인

가입 신청자의 정보를 확인하고 가입을 승인하거나 거절합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원 초대'를 눌러 '구성원 초대' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '가입 신청자'를 선택합니다.
3. 가입 신청자의 정보를 확인하고 승인하거나 거절합니다. 가입 신청자에게 전달되는 내용은 다음과 같습니다.
   - 승인한 경우: [회사명]가입이 승인되었습니다. 네이버웍스 서비스를 시작해보세요.
   - 거절한 경우: [회사명]가입이 거절되었습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
