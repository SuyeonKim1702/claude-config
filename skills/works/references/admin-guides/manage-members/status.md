# 상태

이 기능은 유료 플랜에서만 사용할 수 있습니다.

구성원이 [상태를 표시](/ko/use-guides/settings/personal/set-my-status/)할 수 있게 할지 설정하고, 표시할 상태를 편집, 추가 또는 삭제할 수 있습니다.

상태 표시를 가능하게 하면, 구성원이 바쁨, 자리 비움, 퇴근, 부재 등 자신의 상태를 설정할 수 있으며, 설정된 상태는 프로필 및 메시지방 멤버 목록에 아이콘으로 표시됩니다.

# 상태 표시 사용 여부 설정

구성원이 상태를 표시할 수 있게 할지 설정합니다.

'상태 표시' 설정을 끄면 구성원들이 기존에 설정했던 상태 정보는 모두 삭제됩니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '상태'를 눌러 '상태' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 상태를 사용하고 있지 않은 경우, '상태 사용하기'를 누릅니다. 상태를 사용하지 않으려면 '상태 표시' 설정을 끄고 '확인'을 누릅니다.
3. 오른쪽 상단의 '저장'을 누른 후 '확인'을 누릅니다.

# 상태 추가

기본 제공되는 바쁨, 자리 비움, 퇴근, 부재 상태 외에 재택, 휴가, 출장 등 필요한 상태를 직접 추가할 수 있습니다. 상태는 10개까지 추가할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '상태'를 눌러 '상태' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 화면 상단의 '수정'을 누른 후 목록 하단에 '상태 추가'를 누릅니다.
3. 추가하고자 하는 상태의 상세 내용을 입력하고, 순서를 조정합니다.
   - 순서: 구성원이 [상태 설정](/ko/use-guides/settings/personal/set-my-status/#i-2) 시 노출되는 순서입니다. 상태 왼쪽의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/ico-18-move-c-5-c-5-c-6@3x.png) 아이콘을 누른후 원하는 위치에 끌어다 놓습니다.
   - 아이콘: 구성원의 프로필 이미지 왼쪽 하단에 표시되는 아이콘입니다. 기본 제공되는 아이콘 중 선택합니다.
   - 상태: 구성원의 프로필 상세에 표시되는 상태의 이름입니다.
   - 다국어: 다국어 설정을 할 경우 언어별로 노출될 상태의 이름입니다. '[기본환경 >회사 정보 > 다국어 설정](/ko/admin-guides/settings/company-information/#i-4)'에서 적용 정보의 '상태' 항목을 체크한 경우 입력할 수 있습니다.
4. 오른쪽 상단의 '저장'을 누른 후 '확인'을 누릅니다.

# 상태 수정

각 상태를 수정할 수 있습니다.

단, 기본 제공되는 바쁨, 자리 비움, 퇴근, 부재 상태는 아이콘, 상태명, 다국어 항목을 편집할 수 없습니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '상태'를 눌러 '상태' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 화면 상단의 '수정'을 누른 후 상태의 상세 내용을 수정하거나 순서를 변경합니다.
3. 오른쪽 상단의 '저장'을 누른 후 '확인'을 누릅니다.

# 상태 사용 여부 설정

기본 제공되는 바쁨, 자리 비움, 퇴근, 부재 상태의 사용 여부를 설정합니다. 설정을 끄면 기존에 해당 상태로 설정했던 구성원들의 상태가 '설정 안 함'으로 변경됩니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '상태'를 눌러 '상태' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 화면 상단의 '수정'을 누른 후 기본 제공되는 상태의 '관리' 항목을 켜거나 끕니다.
3. 오른쪽 상단의 '저장'을 누른 후 '확인'을 누릅니다.

# 상태 삭제

기본 제공되는 바쁨, 자리 비움, 퇴근, 부재 상태 외에 직접 추가한 상태를 삭제합니다. 삭제를 하면 기존에 해당 상태로 설정했던 구성원들의 상태가 '설정 안 함'으로 변경됩니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '상태'를 눌러 '상태' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 화면 상단의 '수정'을 누르고 삭제하려는 상태의 '관리' 항목에서 '삭제'를 누릅니다.
3. 오른쪽 상단의 '저장'을 누른 후 '확인'을 누릅니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
