# 조직 삭제

조직을 선택하여 삭제합니다.

# 조직 화면에서 조직 삭제

'조직' 화면의 조직 목록에서 조직을 삭제합니다.

삭제하려는 조직이 하위 조직을 포함하고 있다면 하위 조직도 함께 삭제됩니다.

단, 삭제될 조직에 구성원이 남아있으면 해당 조직을 삭제할 수 없습니다. 조직을 삭제하려면 남아있는 [구성원을 모두 다른 조직으로 이동](/ko/admin-guides/manage-members/members/add-change-members-teams/)한 후 조직을 삭제해야 합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '조직'을 눌러 '조직' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 삭제하고자 하는 조직 왼쪽에 있는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-message-pc-web-contact-checkbox.png) 아이콘을 누른 후, 상단의 '삭제'를 누릅니다. 복수의 조직을 선택하여 삭제할 수도 있습니다.

# 구성원 화면에서 조직 삭제

구성원 화면의 조직도에서 조직을 삭제합니다.

삭제하려는 조직이 하위 조직을 포함하고 있다면 하위 조직도 삭제됩니다.

단, 삭제될 조직에 구성원이 있으면 해당 조직을 삭제할 수 없습니다.

삭제할 조직에 구성원이 남아있는 경우 [구성원을 모두 다른 조직으로 이동](/ko/admin-guides/manage-members/members/add-change-members-teams/)한 후 조직을 삭제해야 합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 왼쪽에 보이는 조직도에서, 삭제하고자 하는 조직의 오른쪽에 보이는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-more-767676_2.png) 아이콘을 누른 후 '조직 삭제'를 선택합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
