# 삭제한 조직 데이터 복원

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

조직 삭제 후 30일 이내에 해당 조직에서 사용하던 메시지, 노트, 일정, 폴더 데이터를 아래 방법으로 복원할 수 있습니다.

- 삭제한 조직의 데이터를 사용 중인 다른 조직으로 옮겨서 복원
- 삭제한 조직의 데이터를 가진 새로운 그룹을 만들어서 복원

조직 데이터는 한번 복원한 후에는 다른 조직이나 새 그룹으로 다시 복원할 수 없습니다.

# 다른 조직으로 데이터 복원

삭제한 조직의 메시지, 노트, 일정, 폴더 데이터를 사용 중인 다른 조직에 덮어 쓰는 방식으로 복원합니다.

사용 중인 다른 조직에서 사용하던 메시지, 노트, 일정 데이터는 데이터 복원 이후에는 삭제되어 확인할 수 없습니다.

단, 할 일 데이터의 경우, 사용 중인 다른 조직의 할 일 요청자 또는 담당자는 데이터 복원 이후에도 [내 할 일 목록](/ko/use-guides/task/task-list/my-task/)에서 할 일을 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '조직'을 눌러 '조직' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단의 '관리'를 눌러 '조직 데이터 복원'을 선택합니다. 모바일에서는 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '조직 데이터 복원'을 탭합니다.
3. 조직 목록에서 데이터 복원이 필요한 조직의 '복원'을 누릅니다.
4. '다른 조직으로 데이터 복원'을 선택하고 하단의 '다음'을 누릅니다.
5. 데이터를 이전 받을 기존 조직을 선택합니다.
6. 하단의 주의사항을 모두 선택한 후 '다음'을 누릅니다.
7. '데이터 복원'을 누릅니다.
8. '확인'을 누릅니다.

# 새 그룹으로 데이터 복원

삭제한 조직의 데이터를 가진 새로운 내부 그룹을 만들어서 복원합니다.

새 그룹 생성 시 아래 정보가 표시되며 직접 수정할 수 있습니다.

- 그룹명 : 삭제한 조직의 조직명이 자동으로 입력되며, 이를 직접 수정할 수 있습니다.
- 마스터/멤버 : 삭제한 조직의 마지막 조직장이 그룹 마스터이자 그룹 멤버로 자동 추가되며, 이를 수정할 수 있습니다. 삭제한 조직의 조직장이 없었을 경우 직접 지정해주어야 합니다.
- 메일링리스트 : 새로운 메일링리스트가 자동으로 추가되며, 메일 주소는 직접 변경할 수 있습니다. 메일링리스트 사용을 원치 않으면 '사용 안 함'으로 변경할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '조직'을 눌러 '조직' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단의 '관리'를 눌러 '조직 데이터 복원'을 선택합니다. 모바일에서는 오른쪽 상단의 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/05/common_24_more_line.png) 아이콘을 탭한 후 '조직 데이터 복원'을 탭합니다.
3. 조직 목록에서 데이터 복원이 필요한 조직의 '복원'을 누릅니다.
4. '새 그룹으로 데이터 복원'을 선택한 후 하단의 '다음'을 누릅니다.
5. 자동으로 입력된 그룹 정보를 확인한 후, 필요 시 그룹명, 마스터, 멤버, 메일링리스트를 수정한 뒤 하단의 '다음'을 누릅니다.
6. 새 그룹으로 데이터를 복원할지 확인하는 확인 창의 '그룹으로 전환'을 눌러 데이터 복원을 완료합니다.

# 조직 데이터 복원 내역 확인

데이터 복원이 완료되면 리스트 내 '복원' 버튼이 '복원 완료'로 변경되어 더 이상 데이터를 복원할 수 없게 됩니다.

복원 완료 하단의 '상세 보기'를 누르면 '데이터 복원 내역' 창이 표시되어, 선택했던 복원 방법과 복원한 기능별 용량 등 상세 복원 내역을 확인할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
