# 조직 이동

해당 조직의 상위 조직이 변경되는 경우 등 조직의 구성원이나 세부 설정값은 변경되지 않고 조직의 소속 위치만 이동합니다.

# 조직 화면에서 조직 이동

조직 화면에서 조직을 이동합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '조직'을 눌러 '조직' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 조직목록에서 오른쪽 상단에 있는 '조직 이동'을 선택하여 선택 모드로 진입합니다.
3. 이동할 조직의 왼쪽에 표시된 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/grabber.png) 아이콘을 눌러 필요한 위치에 끌어다 놓습니다. 조직을 같은 계층으로 이동하려면 파란선이 표시될 때, 하위 계층으로 이동하려면 파란색 테두리가 표시될 때 희망하는 위치에 놓습니다.
4. '저장'을 선택하여 선택한 조직의 이동을 완료합니다.

# 구성원 화면에서 조직 이동

구성원 화면의 조직도에서 조직을 이동합니다.

1. Admin 왼쪽 메뉴에서 '구성원'을 선택해 메뉴를 펼친 후 '구성원'을 눌러 '구성원' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 조직도에서 이동하고자 하는 조직을 선택한 상태에서 위아래로 움직여 다른 조직의 하위 조직으로 위치시키거나, 조직도상의 순서를 변경합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
