# 조직 일괄 추가 샘플 파일 입력 가이드

이 기능은 유료 플랜에서만 사용할 수 있습니다.

['조직 일괄 추가'](/ko/admin-guides/manage-members/organizations/add-organizations/#i-2) 메뉴에서 한번에 최대 1000개의 조직을 추가할 수 있습니다.

여러 조직을 한번에 추가하고 싶다면, 조직 추가 샘플 파일을 다운로드 받은 후 아래 사항에 유의하여 편집해 주세요.

- 조직명
  - 조직명은 최대 100자까지 입력할 수 있으며, 특수문자는 ! @ & ( ) - \_ + [ ] { } , . /만 가능합니다.
- 조직 Depth
  - 조직도 내 정확한 위치에 조직을 등록하기 위해, 상위 조직부터 추가하고자 하는 조직까지 차례대로 Depth에 맞춰 입력해야 합니다. 상위 조직을 빈칸으로 두거나, 유효하지 않은 상위 조직명을 입력할 경우 에러로 처리됩니다.
  - 조직은 최대 15 Depth까지 지원하며, 이를 초과할 경우 파일 등록 시 15 Depth까지만 반영되어 미리보기에 표시됩니다.
- 그 외
  - 샘플 파일의 셀을 병합하거나 임의로 편집할 경우 파일을 읽어오지 못할 수 있습니다. 제공된 포맷으로 조직명을 정확히 입력 후 등록해 주세요.
  - 마지막 Depth까지 조직명이 동일한 조직이 이미 존재할 경우, 중복 에러로 처리됩니다.
  - 1000행을 초과할 경우 파일이 등록되지 않으니, 1000행 이하로 편집해 주세요.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
