# 외부 연락처

이 기능은 네이버웍스 코어 또는 웍스 결재 상품 신청 후 사용 가능합니다.

등록된 외부 연락처 연락처의 수를 확인합니다.

기간을 지정해 조회할 수 있고, 엑셀 파일로 다운로드할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '주소록'을 선택하고 '외부 연락처' 탭으로 이동합니다.

## 외부 연락처 현황 (실시간 기준)

연락처 현황을 실시간으로 확인할 수 있습니다.

- 사용 중: 현재 주소록에 등록된 외부 연락처와 휴지통 내 외부 연락처의 합
- 사용 가능: 추가로 등록할 수 있는 연락처 개수
- 전체: 등록할 수 있는 외부 연락처의 최대 개수

## 외부 연락처 유형별 사용 현황

주소록에 등록된 외부 연락처 연락처의 유형별로 통계를 확인할 수 있습니다.

외부 연락처 연락처의 유형에는 'LINE 연락처', '다른 회사의 NAVER WORKS 연락처', '일반 연락처'가 있습니다.

## 날짜별 연락처 사용 현황

날짜별로 등록된 연락처 현황을 확인할 수 있습니다.

- 전체: 전체를 선택하면 각 항목이 누적되어 차트에 표시됩니다.
- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: 화면 오른쪽 상단의 '다운로드'를 누르면 날짜별 연락처 사용 현황 데이터를 엑셀 파일로 다운로드합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
