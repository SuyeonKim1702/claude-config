# 계정

구성원 수 현황과 사용중, 일시정지, 미접속, 삭제 등 상태별 구성원 수를 확인합니다.

기간을 지정해 조회할 수도 있고, 엑셀 파일로 다운로드할 수도 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 통계 메뉴 하위의 '계정'을 눌러 '계정' 화면으로 이동합니다.

# 사용자 유형별 현황

사용자 유형 현황에 대한 통계입니다.

'사용자 유형' 설정이 '사용'으로 설정되어 있는 경우에만 제공되며, 사용자 유형별로 지정된 구성원 수의 통계를 볼 수 있습니다.

순서상 1~4번째는 사용자가 입력한 사용자 유형을 그대로 보여주며, 5번째 이후부터는 '기타'로 모두 합산되어 보여줍니다.

## 구성원 수 현황

구성원 수 현황에 대한 통계입니다.

구성원 수는 사용 여부와 관계 없이 일시정지, 미접속, 삭제 등 모든 상태의 구성원의 수를 더한 것입니다.

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: '다운로드'를 누르면 화면의 데이터를 엑셀 파일로 다운로드 할 수 있습니다.

## 상태별 구성원 수

상태별 구성원 수를 확인합니다.

- 전체: 전체를 선택하면 각 항목이 누적되어 차트에 표시됩니다. 전체에서 아래 각 항목이 차지하는 비중을 쉽게 확인할 수 있습니다.

|  |  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 사용 중 | | 네이버웍스를 사용하고 있는 구성원 | | | | | | | |
| 사용 대기 | | 계정은 생성되었으나, 활성화되지 않아 로그인할 수 없는 구성원 | | | | | | | |
| 일시 정지 | | 관리자에 의해서나 비밀번호 입력 횟수 초과 등으로 사용이 일시 정지된 구성원 | | | | | | | |
| 미접속 | | 관리자가 구성원을 추가했으나 아직 네이버웍스에 접속하지 않은 구성원 | | | | | | | |
| 삭제 | | 삭제된지 7일이 지나지 않은 구성원 | | | | | | | |

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: '다운로드'를 누르면 화면의 데이터를 엑셀 파일로 다운로드 할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
