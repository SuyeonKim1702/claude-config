# 액티브 유저

액티브 유저 수와 앱의 액티브 유저 수를 확인합니다.

기간을 지정해 조회할 수도 있고, 엑셀파일로 다운로드할 수도 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 통계 메뉴 하위의 '액티브 유저'를 눌러 '액티브 유저' 화면으로 이동합니다.

## 액티브 유저 수

액티브 유저 수 현황에 대한 통계입니다.

액티브 유저 수는 서비스를 이용중인 구성원 수로 해당 날짜에 네이버웍스 서비스에 로그인한 기록이 있는 유저 또는 앱 사용 기록이 있는 유저의 수입니다.

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: '다운로드'를 누르면 화면의 데이터를 엑셀 파일로 다운로드 합니다.

## 앱(APP) 액티브 유저 수

앱 액티브 유저 수 현황에 대한 통계입니다.

앱 액티브 유저 수는 해당 날짜에 모바일 앱에 로그인한 기록이 있는 유저 또는 앱 사용 기록이 있는 유저의 수입니다.

차트에 마우스를 올리면 특정일의 앱 액티브 유저 수를 확인할 수 있습니다.

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: '다운로드'를 누르면 화면의 데이터를 엑셀 파일로 다운로드 합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
