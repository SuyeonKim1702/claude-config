# 사용 현황

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

드라이브의 사용 현황 통계를 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택해 메뉴를 펼친 후 '드라이브'를 눌러 '드라이브' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '사용 현황'을 선택합니다.

## 파일 현황

공용 폴더, 메시지방 폴더, 개인 폴더를 포함한 전체 드라이브의 파일 현황 통계가 전일 기준으로 집계됩니다.

- 사용 중 용량: 공용 폴더, 메시지방 폴더, 개인 폴더에 업로드된 전체 파일의 용량
- 저장 된 파일 수: 공용 폴더, 메시지방 폴더, 개인 폴더에 업로드된 전체 파일 수

## 폴더별 사용 현황

공용 폴더, 메시지방 폴더, 개인 폴더 각각에서 사용 중인 저장 공간에 대한 통계가 전일 기준으로 집계됩니다.  
공용 폴더와 메시지방 폴더의 사용 용량은 [공용 용량](/ko/admin-guides/statistics/shared-storage/)에 포함됩니다.

|  |  |
| --- | --- |
| **항목** | **설명** |
| 공용 폴더 | 공용 폴더에서 사용한 용량의 총합 |
| 메시지방 폴더 | 메시지방 폴더에서 사용한 용량의 총합 |
| 개인 폴더 | 개인 폴더에서 사용한 용량의 총합 |
| 폴더 및 파일 | 업로드된 폴더나 파일이 차지한 용량 |
| 버전 이력 | [파일의 버전 이력](/ko/use-guides/drive/upload-download-view/revision/)이 차지한 용량 |
| 휴지통 | [삭제한 파일 및 폴더](/ko/use-guides/drive/manage-file-folder/manage/#i-5)가 차지한 용량으로, 휴지통에서 파일/폴더를 영구 삭제하면 휴지통에서 사용 중인 용량이 그만큼 줄어듭니다. |

파일 유형별 사용 현황

공용 폴더, 메시지방 폴더, 개인 폴더 전체에 대한 파일 유형별 사용 현황 통계입니다.

파일 유형(문서 파일, 사진 파일, 동영상 파일, 음성 파일, 압축 파일, 기타)별로 드라이브 저장 공간에서 차지한 용량과 비율을 확인할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
