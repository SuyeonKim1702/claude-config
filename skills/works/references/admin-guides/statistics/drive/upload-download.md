# 업로드/다운로드

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 신청 후 사용 가능합니다.

드라이브에서 업로드 또는 다운로드한 파일의 수를 확인합니다.

기간을 지정해 조회할 수도 있고, 엑셀 파일로 다운로드할 수도 있습니다.

## 업로드/다운로드

업로드 또는 다운로드한 파일의 수와 파일의 용량을 확인합니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '드라이브'를 선택하여 '업로드/다운로드' 화면으로 이동합니다.

## 파일 수

파일 수는 회사, 조직, 그룹 단위로 공유하는 폴더와 각 개인의 폴더에서 업로드 또는 다운로드한 모든 파일의 수입니다. 집계는 업로드 혹은 다운로드의 횟수로 표시되며 예를 들어, 하나의 파일을 2번 다운로드하면 2로 집계됩니다. 또한 차트에 마우스를 올리면 특정일의 업로드하거나 다운로드한 파일 수를 확인할 수 있습니다.

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: '다운로드'를 누르면 화면의 데이터를 엑셀 파일로 다운로드 합니다.

## 파일 용량

파일 용량은 드라이브에 업로드하거나 다운로드한 파일의 용량입니다. 회사, 조직, 그룹 단위로 공유하는 폴더와 각 개인의 폴더에서 업로드 또는 다운로드한 모든 파일의 용량입니다. 차트에 마우스를 올리면 특정일의 업로드하거나 다운로드한 파일 용량을 확인할 수 있습니다.

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: '다운로드'를 누르면 화면의 데이터를 엑셀 파일로 다운로드합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
