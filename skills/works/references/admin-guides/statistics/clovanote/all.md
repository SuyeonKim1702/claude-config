클로바노트의 월간 음성 인식 시간 및 요약 횟수, 노트 생성 수를 확인합니다.

노트 생성 수는 기간을 지정해 조회할 수 있고, 전체 이용 현황을 엑셀 파일로 다운로드할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '클로바노트'를 선택하고 '전체' 탭으로 이동합니다.

# 월간 사용 현황

음성 인식의 월간 사용 시간 현황과 월간 사용 요약 현황을 실시간으로 확인할 수 있습니다.

- 월간 사용 시간 현황: 음성 인식을 사용한 월 누적 시간
- 남은 시간: 월 기준 음성 인식 가능한 총 시간 중 앞으로 이용 가능한 시간
- 총 제공 시간: 월 기준 음성 인식 가능한 총 시간
  - 모든 구성원이 함께 나눠쓸 수 있는 전체 시간입니다.
  - 상품 사용 시작 및 종료 시점에 따라 일할 계산되어 지급됩니다.
- 사용한 횟수: 월 기준 노트 요약을 사용한 누적 횟수
- 남은 횟수: 월 기준 노트 앞으로 노트 요약이 가능한 횟수
- 총 제공 요약 횟수: 월 기준 노트 요약이 가능한 총 횟수
