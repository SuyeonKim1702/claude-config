# 구성원별

이 기능은 클로바노트 상품 신청 후 사용 가능합니다.

구성원별로 클로바노트의 변환 노트 수, 사용 요약 횟수, 사용 시간을 확인합니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. '클로바노트'를 선택하고 '구성원별' 탭으로 이동합니다.
3. 조직명 또는 특정 기간을 선택하거나, 화면 오른쪽 상단의 ![](/wp-content/uploads/2024/04/icon-n-12-search-434343.png) 아이콘을 클릭한 후 구성원별로 검색해 월간 사용 현황을 확인할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
