# 설문

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

생성한 전체 설문의 수를 확인합니다.

기간을 지정해 조회할 수도 있고, 엑셀파일로 다운로드할 수도 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://help.worksmobile.com//wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 통계 메뉴 하위의 '설문'을 눌러 '설문' 화면으로 이동합니다.

## 설문 생성 수

설문 생성 수 현황에 대한 통계입니다.

설문 생성 수는 [생성하기를 눌러](/ko/use-guides/form/write-survey/add-form/) 생성된 설문의 수입니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
