# AI어시스턴트

이 기능은 AI 스튜디오 상품 신청 후 사용 가능합니다.

AI 스튜디오의 상태별 AI어시스턴트 수를 확인합니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 'AI 스튜디오'를 선택하고 'AI어시스턴트' 탭으로 이동합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
