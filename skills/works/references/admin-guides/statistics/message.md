# 메시지

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

메시지의 수를 확인합니다.

기간을 지정해 조회할 수도 있고, 엑셀 파일로 다운로드할 수도 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 통계 메뉴 하위의 '메시지'를 눌러 '메시지' 화면으로 이동합니다.

## 메시지 수

메시지 수 현황에 대한 통계입니다.

메시지 수는 텍스트, 스티커/이모티콘, Bot, 파일, 음성 통화, 영상 통화, 그룹 통화, 화면 공유를 모두 포함해 보낸 메시지의 수입니다.

단, Bot 메시지의 경우에는 수신인의 수를 메시지 수로 집계합니다.

- 전체: 전체를 선택하면 각 항목이 누적되어 차트에 표시됩니다. 전체에서 아래 각 항목이 차지하는 비중을 쉽게 확인할 수 있습니다.

|  |  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 텍스트 | | 일반 텍스트 메시지와 조직/그룹 메시지방의 알림 메시지 수 | | | | | | | |
| 스티커/이모티콘 | | 스티커나 이모티콘을 보낸 횟수 | | | | | | | |
| Bot | | Bot과 대화한 메시지 수 | | | | | | | |
| 파일 | | 메시지방에서 공유한 파일 수 | | | | | | | |
| 음성 통화 | | 음성 통화를 한 횟수. 그룹콜 제외 | | | | | | | |
| 영상 통화 | | 영상 통화를 한 횟수. 그룹 영상통화, 화면 공유 제외 | | | | | | | |
| 그룹 통화 | | 그룹 음성 통화 및 영상 통화 수 | | | | | | | |
| 화면 공유 | | 화면 공유 수 | | | | | | | |

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: '다운로드'를 누르면 화면의 데이터를 엑셀 파일로 다운로드 합니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
