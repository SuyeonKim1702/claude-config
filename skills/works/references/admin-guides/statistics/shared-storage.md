# 공용용량

공용용량의 사용 현황을 한 눈에 확인합니다.

공용용량은 회사에서 구성원이 공동으로 사용하는 공간으로 공용용량에 포함되는 항목은 아래와 같습니다.

특정 상품을 구매해 추가받은 공용용량은 이용 중인 상품의 기존 공용용량과 합산됩니다.

- 메시지에서 주고 받은 첨부파일
- 조직/그룹 메시지, 일정, 폴더, 노트에 업로드 또는 공유한 파일
- 게시판 게시글 또는 댓글에 첨부한 파일
- 설문에 첨부한 파일
- 캘린더 일정에 첨부한 파일
- 할 일에 첨부한 파일
- 템플릿에 첨부한 파일
- 드라이브에 업로드한 파일
- 경영지원 서비스 내에서 첨부한 파일
- AI 스튜디오 서비스 내에서 AI어시스턴트에 업로드한 파일

각 구성원의 메일함 용량은 공용용량에 포함되지 않습니다. 공용용량은 상품에 따라 다르게 제공되며 상품별 공용용량에 대한 내용은 [상품 페이지](https://naver.worksmobile.com/pricing/)에서 확인할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 통계 메뉴 하위의 '공용용량'을 눌러 '공용용량' 화면으로 이동합니다.

# 공용용량 사용량

제공되는 공용용량과 현재 사용중인 공용용량을 한 눈에 확인할 수 있습니다.

- 사용 가능: 제공되는 전체 공용용량에서 사용중 용량을 차감한 용량입니다. 앞으로 사용할 수 있는 공용용량입니다.
- 전체 용량: 제공되는 공용용량입니다. 가입한 상품과 구성원 수에 따라 제공용량이 달라집니다.
- 사용 중: 제공되는 공용용량 중 현재 사용중인 공용용량입니다.  
  조직/그룹을 삭제하더라도 메시지 파일의 용량은 삭제 후 복원을 할 수 있는 30일간은 공용용량에서 차감되지 않습니다.단, 조직/그룹 삭제 후 '[조직 데이터 복원](https://help.worksmobile.com/ko/admin-guides/manage-members/organizations/recover-deleted-organization-data/)' 또는 '[그룹 데이터 복원](/ko/admin-guides/manage-members/groups/restore-deleted-group-data/)'에서 '영구 삭제' 시 조직/그룹의 데이터가 완전 삭제되어 즉시 용량이 확보됩니다.

공용용량 사용률이 50%, 60%, 70%, 80%, 90%, 100%에 도달할 때마다 관리자 권한을 가진 모든 구성원에게 메시지 알림이 전송됩니다.

공용용량 사용률이 100%에 도달하면 공용용량을 사용하는 기능의 사용이 제한됩니다.

# 사용 현황

제공된 공용용량 중 현재 사용중인 용량입니다.

기간을 지정해 조회할 수도 있고, 엑셀 파일로 다운로드할 수도 있습니다.

- 사용 중 용량: 제공되는 공용용량 중 현재 사용중인 공용용량입니다. 차트에서는 오른쪽 축을 기준으로 하며 단위는 GB입니다.
- 사용 가능 용량: 제공되는 전체 공용용량에서 사용중 용량을 차감한 용량입니다. 앞으로 사용할 수 있는 공용용량입니다. 차트에서는 오른쪽 축을 기준으로 하며 단위는 GB입니다.
- 사용률(%): 전체 용량 가운데 사용 중 용량의 비율입니다. 차트에서는 왼쪽 축을 기준으로 하며 단위는 %입니다.

# 서비스별 용량 - 전체

서비스별로 사용하고 있는 공용용량입니다.

기간을 지정해 조회할 수도 있고, 엑셀 파일로 다운로드할 수도 있습니다.

- 전체: 전체를 선택하면 각 항목이 누적되어 차트에 표시됩니다. 전체에서 아래 각 항목이 차지하는 비중을 쉽게 확인할 수 있습니다.

|  |  |
| --- | --- |
| 게시판 | 게시판 게시글 및 댓글에 첨부한 파일. 게시글과 댓글 자체는 포함되지 않음. |
| 메시지(노트, 폴더 포함) | 일반 메시지방 또는 조직/그룹 메시지방에서 첨부한 파일의 용량과 메시지방 노트와 폴더에 업로드한 파일 용량의 합 |
| 캘린더 | 일정에 첨부한 파일 용량의 합 |
| 설문 | 설문에 첨부한 파일 용량의 합 |
| 할 일 | 할 일에 첨부한 파일 용량의 합 |
| 템플릿 | 템플릿에 첨부한 파일 용량의 합 |
| 드라이브 | 드라이브에 업로드한 파일 용량의 합 |
| 경영지원 | 경영지원 서비스 내에서 첨부한 파일 용량의 합 |
| AI 스튜디오 | AI어시스턴트에 업로드한 파일 용량의 합 |

- 공용용량 관리 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.
- 다운로드: '다운로드'를 누르면 화면의 데이터를 엑셀 파일로 다운로드합니다.

공용용량을 초과했을 때 공용용량을 사용하는 서비스별로 파일을 보관하는 기간을 변경하거나 조직/그룹 관리 기능을 통해 공용용량을 확보할 수 있습니다.

## 서비스별 공용 용량 관리

### **메시지**

| 파일 종류 | 삭제 방법/시점 |
| --- | --- |
| 메시지방 폴더 내 파일 | 메시지방 폴더에서 파일을 삭제합니다. 삭제한 파일은 휴지통으로 이동 후 Admin에서 설정한 '휴지통 보관 기간'이 지나면 완전히 삭제됩니다. |

## **조직/그룹 관리를 통한 용량 관리**

더 이상 사용하지 않는 조직 또는 그룹을 삭제하거나 조직/그룹 메시지방 기능의 데이터를 삭제해 공용 용량을 확보할 수 있습니다.

### 조직 또는 그룹 **삭제**

삭제를 하더라도 데이터 보관 정책에 의해 30일 간 데이터를 보관하고 있어 30일 이후에 용량이 확보됩니다.

### **조직 또는 그룹 메시지방 기능의 데이터 삭제하기**

2. 창 하단의 '수정'을 누른 후 메시지방 기능의 '데이터 삭제'를 눌러 데이터를 삭제하면 삭제한 용량만큼 공용용량이 확보됩니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
