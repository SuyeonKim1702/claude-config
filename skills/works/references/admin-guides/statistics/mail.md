# 메일

이 기능은 네이버웍스 코어 상품의 Standard, Standard Plus 플랜 신청 후 사용 가능합니다.

내부 및 외부에서 보내거나 받은 메일의 수를 확인합니다.

기간을 지정해 조회할 수도 있고, 엑셀 파일로 다운로드할 수도 있습니다.

다운로드의 경우, 보내거나 받은 메일 수를 선택하여 다운로드하거나, '구성원 메일 사용량'을 선택한 후 구성원 전체의 메일 사용량 데이터를 다운로드할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 통계 메뉴 하위의 '메일'을 눌러 '메일' 화면으로 이동합니다.

## 보낸 메일 수

보낸 메일 수 현황에 대한 통계입니다.

보낸 메일 수는 외부로 보낸 메일과 내부로 보낸 메일을 모두 포함한 보낸 메일의 수입니다.

- 전체: 전체를 선택하면 각 항목이 누적되어 차트에 표시됩니다. 전체에서 아래 각 항목이 차지하는 비중을 쉽게 확인할 수 있습니다.

|  |  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 외부 메일 | | 회사 외부로 보낸 메일 수 | | | | | | | |
| 내부 메일 | | 회사에 소속된 구성원에게 보낸 메일 수로 그룹사에 소속된 구성원에게 보낸 메일 수도 포함 | | | | | | | |

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.

## 받은 메일 수

받은 메일 수 현황에 대한 통계입니다.

받은 메일 수는 외부에서 받은 메일과 내부에서 받은 메일을 모두 포함한 받은 메일의 수입니다.

- 전체: 전체를 선택하면 각 항목이 누적되어 차트에 표시됩니다. 전체에서 아래 각 항목이 차지하는 비중을 쉽게 확인할 수 있습니다.

|  |  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 외부 메일 | | 회사 외부에서 받은 메일 수 | | | | | | | |
| 내부 메일 | | 회사에 소속된 구성원에게 받은 메일 수로 그룹사에 소속된 구성원에게 받은 메일 수 포함 | | | | | | | |

- 조회: 특정 기간을 선택하거나 직접 조회 기간을 입력한 후 '조회'를 눌러 통계를 확인합니다. 최근 1년의 데이터만 조회할 수 있습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
