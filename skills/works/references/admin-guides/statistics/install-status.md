# 설치 현황

이 기능은 네이버웍스 코어 상품 신청 후 사용 가능합니다.

각 조직 별로 구성원들의 네이버웍스 모바일앱과 PC앱 설치 현황을 파악하고, 앱 설치를 독려하는 안내메일을 보낼 수 있습니다.

- 모바일앱 설치 여부
  - 이 구성원이 네이버웍스 모바일앱에 접속한 이력이 있다면, 가장 최근에 접속한 모바일 기기 정보와 접속 시각이 나타납니다.
  - 이 구성원이 네이버웍스 모바일앱에 접속한 이력이 없다면 '미설치'로 나타납니다. 만약 서비스 권한 설정으로 모바일앱을 사용하지 못하도록 제한하였다면, '권한 없음'으로 표시됩니다.
- PC앱 설치 여부
  - 이 구성원이 네이버웍스 PC앱에 접속한 이력이 있다면, 가장 최근에 접속한 PC 기기 정보와 접속 시각이 나타납니다.
  - 이 구성원이 네이버웍스 PC앱에 접속한 이력이 없다면 '미설치'로 나타납니다. 만약 서비스 권한 설정으로 PC앱을 사용하지 못하도록 제한하였다면, '권한 없음'으로 표시됩니다.

# 설치 안내 보내기

아직 네이버웍스를 설치하지 않은 구성원에게 다음과 같은 설치 안내 메일을 보낼 수 있습니다.

구성원 정보에 개인 이메일주소가 입력되어 있고 사용권한이 있지만 모바일앱과 PC앱 중 미설치된 앱이 있는 구성원의 경우만 선택이 가능합니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택해 메뉴를 펼친 후 '설치 현황'을 눌러 '설치 현황' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 목록에서 설치 안내를 보낼 구성원을 찾아 선택합니다. ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-n-20-search-767676.png) 아이콘을 눌러 특정 구성원을 검색할 수 있습니다.
3. 목록 상단의 '설치 안내 보내기'를 누릅니다.
4. 설치안내를 보낼지 확인하는 화면에서 '확인'을 눌러 선택한 구성원의 개인 이메일로 설치 안내 메일을 발송합니다.

## 설치하지 않은 모든 구성원에게 설치 안내 보내기

구성원 정보에 개인 이메일주소가 입력되어 있고 사용권한이 있지만 모바일앱과 PC앱 중 미설치된 앱이 있는 모든 구성원에게 설치 안내 메일이 보내집니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택해 메뉴를 펼친 후 '설치 현황'을 눌러 '설치 현황' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 목록 상단의 '설치 안내 전체 보내기'를 누릅니다.
3. 설치안내를 보낼지 확인하는 화면에서 '확인'을 눌러 설치 안내 메일을 발송합니다.

# 설치 안내 이력

최근 20회의 설치 안내 보낸 기록을 확인하고, 각 기록 별로 안내 받은 구성원과 이메일 주소를 파일로 다운로드할 수 있습니다.

1. Admin 왼쪽 메뉴에서 '통계'를 선택해 메뉴를 펼친 후 '설치 현황'을 눌러 '설치 현황' 화면으로 이동합니다. 모바일에서는 ![](https://gsaas-help.pstatic.net/wp-content/uploads/2024/04/icon-admin-mobile-menu.png) 버튼을 탭하면 메뉴가 나타납니다.
2. 오른쪽 상단의 '설치 안내 보낸 이력'을 누릅니다.

- 날짜: 설치 안내를 보낸 시각이 표시됩니다.
- 보낸 관리자: 설치 안내를 보낸 관리자가 표시됩니다.
- 수신인: 설치 안내를 받은 구성원 숫자가 표시됩니다.
- 상세: '다운로드'를 클릭하면 수신인의 이름, 계정, 앱 설치여부, 개인 이메일 정보를 .csv 파일로 다운로드 받습니다.

공공용 네이버웍스에서는 일부 기능이 제한됩니다. 자세한 내용은 [네이버웍스 기능 소개서](https://gov-naverworks.com/naver-works-function-list/)를 확인해주세요.
